C#

C# est un langage de programmation orienté objet développé par Microsoft. Il est utilisé pour créer des applications Windows, des applications Web et des jeux. C# est un langage puissant et polyvalent, offrant une syntaxe simple et une grande compatibilité avec les autres technologies Microsoft.

Excel

Excel est un logiciel de tableur développé par Microsoft. Il permet de créer, organiser et analyser des données sous forme de feuilles de calcul. Excel offre des fonctionnalités avancées telles que des formules, des graphiques et des outils de visualisation des données.

F#

F# est un langage de programmation fonctionnel et orienté objet développé par Microsoft. Il est conçu pour être utilisé dans le développement d'applications robustes et évolutives, en mettant l'accent sur la concision du code, la sécurité des types et la programmation fonctionnelle.

Html

HTML (HyperText Markup Language) est un langage de balisage utilisé pour structurer le contenu d'une page web. Il permet de définir la structure et le formatage du texte, les liens hypertexte, les images et autres éléments multimédias, facilitant ainsi la création et la présentation de sites web.

Java

Java est un langage de programmation polyvalent et orienté objet, utilisé pour développer des applications logicielles pour diverses plates-formes. Il est connu pour sa portabilité, sa sécurité et sa facilité d'utilisation, et est largement utilisé dans le développement d'applications Web, mobiles et d'entreprise.

Commencer le cours

Javascript

JavaScript est un langage de programmation de haut niveau, interprété et orienté objet. Il est principalement utilisé pour créer des pages web interactives en ajoutant des fonctionnalités dynamiques et des effets visuels. Il peut également être utilisé pour développer des applications côté serveur et des applications mobiles.

Python

Python est un langage de programmation moderne, polyvalent et en forte demande sur le marché du travail. Simple à apprendre, mais puissant dans son exécution, il est le choix idéal pour les débutants comme pour les développeurs expérimentés.

Scala

Scala est un langage de programmation polyvalent qui combine les fonctionnalités de l'orienté objet et de la programmation fonctionnelle. Il est conçu pour être concis, expressif et sûr, et fonctionne sur la machine virtuelle Java, ce qui lui permet d'interagir facilement avec les bibliothèques Java existantes.

docker

Docker est une plateforme de virtualisation légère qui permet d'emballer et de distribuer des applications dans des conteneurs, offrant ainsi une isolation et une portabilité optimales. Il simplifie le déploiement et la gestion des applications, en permettant une exécution cohérente sur différents environnements.
