Installation de Sinatra

Pour commencer à utiliser Sinatra, vous devez d'abord l'installer sur votre machine. Voici les étapes à suivre :

  1. Ouvrez votre terminal ou votre invite de commande.
  2. Assurez-vous d'avoir Ruby installé sur votre machine. Vous pouvez vérifier cela en exécutant la commande ruby -v. Si Ruby n'est pas installé, vous pouvez le télécharger et l'installer à partir du site officiel de Ruby.
  3. Une fois Ruby installé, vous pouvez installer Sinatra en exécutant la commande suivante :
gem install sinatra

Cette commande téléchargera et installera la dernière version de Sinatra sur votre machine.

Configuration de l'environnement de développement

Avant de commencer à développer des applications Sinatra, vous devez configurer votre environnement de développement. Voici les étapes à suivre :

  1. Ouvrez votre éditeur de code préféré.
  2. Créez un nouveau dossier pour votre projet Sinatra.
  3. Ouvrez une nouvelle fenêtre de terminal ou d'invite de commande et naviguez jusqu'au dossier de votre projet.
  4. Créez un nouveau fichier Ruby avec l'extension .rb, par exemple app.rb.
  5. Ouvrez le fichier Ruby dans votre éditeur de code.

Votre environnement de développement est maintenant prêt à être utilisé pour développer des applications Sinatra.

Création d'une application Sinatra de base

Une fois que vous avez installé Sinatra et configuré votre environnement de développement, vous pouvez commencer à créer une application Sinatra de base. Voici les étapes à suivre :

  1. Ouvrez le fichier Ruby que vous avez créé précédemment.
  2. Commencez par exiger la gemme Sinatra en ajoutant la ligne suivante en haut de votre fichier :
require 'sinatra'

Cette ligne permet à votre application d'utiliser les fonctionnalités de Sinatra.

  1. Définissez une route racine en ajoutant le code suivant :
get '/' do
  'Hello, world!'
end

Ce code définit une route racine qui renvoie simplement la chaîne de caractères "Hello, world!".

  1. Enregistrez votre fichier Ruby.
  2. Exécutez votre application en exécutant la commande suivante dans votre terminal ou votre invite de commande :
ruby app.rb

Cette commande démarre votre application Sinatra et la rend accessible à l'adresse http://localhost:4567.

Vous pouvez maintenant ouvrir votre navigateur et accéder à http://localhost:4567 pour voir le message "Hello, world!" affiché par votre application Sinatra.

Félicitations ! Vous avez créé avec succès une application Sinatra de base.
