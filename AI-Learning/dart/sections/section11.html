Réalisation d'une application complète en Dart

Dans cette dernière section, nous allons mettre en pratique tout ce que nous avons appris jusqu'à présent en réalisant une application complète en Dart. Cette application sera un gestionnaire de tâches simple, où les utilisateurs pourront créer, modifier et supprimer des tâches.

Pour commencer, nous allons créer une classe Task qui représentera une tâche dans notre application. Voici à quoi ressemblera notre classe :

class Task {
  String title;
  bool isCompleted;

  Task(this.title, this.isCompleted);
}

Nous avons défini deux propriétés pour notre classe Task : le titre de la tâche (title) et un indicateur pour savoir si la tâche est terminée ou non (isCompleted). Nous avons également créé un constructeur qui nous permettra de créer des instances de la classe Task en spécifiant le titre et l'état initial de la tâche.

Maintenant que nous avons notre classe Task, nous allons créer une liste de tâches dans notre application. Voici comment nous pouvons déclarer et initialiser notre liste :

List<Task> tasks = [
  Task('Faire les courses', false),
  Task('Faire du sport', true),
  Task('Lire un livre', false)
];

Nous avons créé une liste tasks qui contient trois instances de la classe Task. Chaque instance représente une tâche avec un titre et un état initial.

Ensuite, nous allons créer une fonction pour afficher toutes les tâches de notre liste. Voici comment nous pouvons le faire :

void displayTasks() {
  for (var task in tasks) {
    print('Titre: ${task.title}');
    print('Terminée: ${task.isCompleted}');
    print('----------------------');
  }
}

Cette fonction parcourt la liste des tâches et affiche le titre et l'état de chaque tâche. Nous utilisons la syntaxe ${} pour insérer les valeurs des propriétés dans la chaîne de caractères.

Maintenant, nous allons ajouter une fonction pour ajouter une nouvelle tâche à notre liste :

void addTask(String title) {
  tasks.add(Task(title, false));
}

Cette fonction prend un titre en paramètre et crée une nouvelle instance de la classe Task avec le titre spécifié et un état initial de non terminé. Ensuite, elle ajoute cette nouvelle tâche à la liste des tâches.

Ensuite, nous allons ajouter une fonction pour marquer une tâche comme terminée :

void completeTask(int index) {
  tasks[index].isCompleted = true;
}

Cette fonction prend un index en paramètre et modifie l'état de la tâche correspondante dans la liste des tâches pour la marquer comme terminée.

Enfin, nous allons ajouter une fonction pour supprimer une tâche de notre liste :

void deleteTask(int index) {
  tasks.removeAt(index);
}

Cette fonction prend un index en paramètre et supprime la tâche correspondante de la liste des tâches.

Maintenant que nous avons toutes ces fonctions, nous pouvons les utiliser pour créer une interface utilisateur pour notre application. Nous pourrions utiliser une bibliothèque de widgets comme Flutter pour cela, mais pour simplifier, nous allons simplement utiliser la console pour afficher et interagir avec notre application.

Voici comment nous pouvons créer une boucle principale pour notre application :

void main() {
  bool isRunning = true;

  while (isRunning) {
    print('Que voulez-vous faire ?');
    print('1. Afficher les tâches');
    print('2. Ajouter une tâche');
    print('3. Marquer une tâche comme terminée');
    print('4. Supprimer une tâche');
    print('5. Quitter');

    String choice = stdin.readLineSync();

    switch (choice) {
      case '1':
        displayTasks();
        break;
      case '2':
        print('Entrez le titre de la tâche :');
        String title = stdin.readLineSync();
        addTask(title);
        break;
      case '3':
        print('Entrez l\'index de la tâche à marquer comme terminée :');
        int index = int.parse(stdin.readLineSync());
        completeTask(index);
        break;
      case '4':
        print('Entrez l\'index de la tâche à supprimer :');
        int index = int.parse(stdin.readLineSync());
        deleteTask(index);
        break;
      case '5':
        isRunning = false;
        break;
      default:
        print('Choix invalide');
    }
  }
}

Cette boucle principale affiche un menu à l'utilisateur et lui demande de choisir une option. En fonction de son choix, elle appelle la fonction appropriée pour effectuer l'action correspondante. L'utilisateur peut également choisir l'option 5 pour quitter l'application.

Voilà, nous avons maintenant une application de gestion de tâches fonctionnelle en Dart ! Vous pouvez l'exécuter et l'essayer vous-même en utilisant l'interpréteur Dart ou en la compilant en JavaScript et en l'exécutant dans un navigateur.
