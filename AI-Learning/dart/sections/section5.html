Listes

Une liste est une collection ordonnée d'éléments. En Dart, les listes peuvent contenir des éléments de n'importe quel type de données, y compris des types primitifs tels que les entiers et les chaînes de caractères, ainsi que des objets personnalisés.

Pour créer une liste en Dart, vous pouvez utiliser la syntaxe suivante :

List<Type> nomDeLaListe = [élément1, élément2, ..., élémentN];

Par exemple, pour créer une liste de nombres entiers, vous pouvez utiliser :

List<int> nombres = [1, 2, 3, 4, 5];

Vous pouvez accéder aux éléments d'une liste en utilisant leur index. L'index du premier élément est 0, le deuxième élément a un index de 1, et ainsi de suite. Pour accéder à un élément spécifique, vous pouvez utiliser la syntaxe suivante :

nomDeLaListe[index];

Par exemple, pour accéder au troisième élément de la liste "nombres", vous pouvez utiliser :

int troisiemeNombre = nombres[2];

Les listes en Dart sont également modifiables. Vous pouvez ajouter des éléments à une liste existante en utilisant la méthode "add". Par exemple :

nombres.add(6);

Cela ajoutera le nombre 6 à la fin de la liste "nombres".

Maps

Une map est une collection d'associations clé-valeur. Chaque élément de la map est composé d'une clé et d'une valeur correspondante. En Dart, les clés et les valeurs peuvent être de n'importe quel type de données.

Pour créer une map en Dart, vous pouvez utiliser la syntaxe suivante :

Map<TypeDeLaCle, TypeDeLaValeur> nomDeLaMap = {
  cle1: valeur1,
  cle2: valeur2,
  ...
  cleN: valeurN
};

Par exemple, pour créer une map de noms et d'âges, vous pouvez utiliser :

Map<String, int> personnes = {
  'Alice': 25,
  'Bob': 30,
  'Charlie': 35
};

Vous pouvez accéder à la valeur associée à une clé spécifique en utilisant la syntaxe suivante :

nomDeLaMap[cle];

Par exemple, pour accéder à l'âge de Bob dans la map "personnes", vous pouvez utiliser :

int ageDeBob = personnes['Bob'];

Les maps en Dart sont également modifiables. Vous pouvez ajouter de nouvelles associations clé-valeur en utilisant la syntaxe suivante :

nomDeLaMap[cle] = valeur;

Par exemple, pour ajouter une nouvelle personne à la map "personnes", vous pouvez utiliser :

personnes['Dave'] = 40;

Sets

Un set est une collection d'éléments uniques, sans ordre spécifique. En Dart, les sets peuvent contenir des éléments de n'importe quel type de données, mais chaque élément doit être unique.

Pour créer un set en Dart, vous pouvez utiliser la syntaxe suivante :

Set<Type> nomDuSet = {élément1, élément2, ..., élémentN};

Par exemple, pour créer un set de nombres, vous pouvez utiliser :

Set<int> nombresUniques = {1, 2, 3, 4, 5};

Les sets en Dart sont également modifiables. Vous pouvez ajouter de nouveaux éléments à un set existant en utilisant la méthode "add". Par exemple :

nombresUniques.add(6);

Cela ajoutera le nombre 6 au set "nombresUniques". Si vous essayez d'ajouter un élément qui est déjà présent dans le set, il ne sera pas ajouté car les sets ne peuvent pas contenir d'éléments en double.

Vous pouvez également supprimer des éléments d'un set en utilisant la méthode "remove". Par exemple :

nombresUniques.remove(5);

Cela supprimera le nombre 5 du set "nombresUniques".
