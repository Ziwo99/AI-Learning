Structure d'un programme Dart

Un programme Dart est composé de plusieurs éléments, notamment des fonctions, des variables et des classes. Voici une structure de base d'un programme Dart :

void main() {
    // Code principal du programme
}

La fonction main() est le point d'entrée du programme. C'est là que le code principal du programme est exécuté.

Variables et types de données

En Dart, les variables sont utilisées pour stocker des valeurs. Voici comment déclarer une variable en Dart :

int age = 25;
String name = "John";
double height = 1.75;
bool isStudent = true;

En Dart, il existe plusieurs types de données, tels que int pour les entiers, String pour les chaînes de caractères, double pour les nombres à virgule flottante, et bool pour les valeurs booléennes (vrai ou faux).

Opérateurs

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques exemples d'opérateurs en Dart :

// Opérateurs arithmétiques
int sum = 10 + 5;
int difference = 10 - 5;
double product = 10 * 5;
double quotient = 10 / 5;
int remainder = 10 % 5;

// Opérateurs de comparaison
bool isEqual = (10 == 5);
bool isNotEqual = (10 != 5);
bool isGreater = (10 > 5);
bool isLess = (10 < 5);
bool isGreaterOrEqual = (10 >= 5);
bool isLessOrEqual = (10 <= 5);

// Opérateurs logiques
bool isTrue = true;
bool isFalse = false;
bool andResult = isTrue && isFalse;
bool orResult = isTrue || isFalse;
bool notResult = !isTrue;

Structures de contrôle (boucles et conditions)

Les structures de contrôle sont utilisées pour contrôler le flux d'exécution du programme. Voici quelques exemples de structures de contrôle en Dart :

// Boucle for
for (int i = 0; i < 10; i++) {
    print(i);
}

// Boucle while
int i = 0;
while (i < 10) {
    print(i);
    i++;
}

// Condition if
int age = 25;
if (age >= 18) {
    print("Vous êtes majeur.");
} else {
    print("Vous êtes mineur.");
}

// Condition switch
String day = "Monday";
switch (day) {
    case "Monday":
        print("C'est lundi.");
        break;
    case "Tuesday":
        print("C'est mardi.");
        break;
    default:
        print("C'est un autre jour de la semaine.");
        break;
}

Fonctions

Les fonctions sont utilisées pour regrouper du code et le réutiliser. Voici comment déclarer une fonction en Dart :

void sayHello() {
    print("Bonjour !");
}

String getFullName(String firstName, String lastName) {
    return firstName + " " + lastName;
}

int calculateSum(int a, int b) {
    return a + b;
}

Les fonctions peuvent avoir des paramètres et renvoyer des valeurs. Elles peuvent également être déclarées avec des types de données spécifiques pour les paramètres et les valeurs de retour.

Classes et objets

Les classes et les objets sont utilisés pour créer des structures de données et des fonctionnalités plus complexes. Voici comment déclarer une classe en Dart :

class Person {
    String name;
    int age;

    void sayHello() {
        print("Bonjour, je m'appelle $name et j'ai $age ans.");
    }
}

// Création d'un objet de la classe Person
Person person = Person();
person.name = "John";
person.age = 25;
person.sayHello();

Les classes peuvent avoir des propriétés (variables) et des méthodes (fonctions). Les objets sont créés à partir d'une classe et peuvent accéder aux propriétés et aux méthodes de cette classe.
