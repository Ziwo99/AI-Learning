Lecture et écriture de fichiers

La manipulation de fichiers est une tâche courante dans le développement d'applications Dart. Dart fournit des fonctionnalités intégrées pour lire et écrire des fichiers de manière synchrone et asynchrone.

Pour lire un fichier de manière synchrone, vous pouvez utiliser la classe File du package dart:io. Voici un exemple :

import 'dart:io';

void main() {
  var file = File('mon_fichier.txt');
  var contents = file.readAsStringSync();
  print(contents);
}

Dans cet exemple, nous importons le package dart:io qui fournit des fonctionnalités d'entrée/sortie. Nous créons ensuite une instance de la classe File en spécifiant le chemin du fichier à lire. Ensuite, nous utilisons la méthode readAsStringSync() pour lire le contenu du fichier de manière synchrone et stocker le résultat dans la variable contents. Enfin, nous affichons le contenu du fichier à l'aide de la fonction print().

Pour écrire dans un fichier de manière synchrone, vous pouvez utiliser la méthode writeAsStringSync() de la classe File. Voici un exemple :

import 'dart:io';

void main() {
  var file = File('mon_fichier.txt');
  file.writeAsStringSync('Contenu à écrire dans le fichier');
}

Dans cet exemple, nous utilisons la méthode writeAsStringSync() pour écrire le contenu spécifié dans le fichier. Si le fichier n'existe pas, il sera créé. Si le fichier existe déjà, son contenu sera remplacé par le nouveau contenu.

Pour lire un fichier de manière asynchrone, vous pouvez utiliser la méthode readAsString() de la classe File. Voici un exemple :

import 'dart:io';

void main() async {
  var file = File('mon_fichier.txt');
  var contents = await file.readAsString();
  print(contents);
}

Dans cet exemple, nous utilisons le mot-clé async pour déclarer la fonction main() comme une fonction asynchrone. Ensuite, nous utilisons le mot-clé await pour attendre que la méthode readAsString() se termine avant de continuer l'exécution du programme.

Pour écrire dans un fichier de manière asynchrone, vous pouvez utiliser la méthode writeAsString() de la classe File. Voici un exemple :

import 'dart:io';

void main() async {
  var file = File('mon_fichier.txt');
  await file.writeAsString('Contenu à écrire dans le fichier');
}

Dans cet exemple, nous utilisons le mot-clé await pour attendre que la méthode writeAsString() se termine avant de continuer l'exécution du programme.

Gestion des répertoires

En plus de la manipulation de fichiers, Dart permet également de gérer les répertoires. Vous pouvez créer, supprimer et lister les fichiers et répertoires d'un répertoire spécifique.

Pour créer un répertoire, vous pouvez utiliser la méthode create() de la classe Directory. Voici un exemple :

import 'dart:io';

void main() {
  var directory = Directory('mon_repertoire');
  directory.createSync();
}

Dans cet exemple, nous créons une instance de la classe Directory en spécifiant le chemin du répertoire à créer. Ensuite, nous utilisons la méthode createSync() pour créer le répertoire de manière synchrone.

Pour supprimer un répertoire, vous pouvez utiliser la méthode delete() de la classe Directory. Voici un exemple :

import 'dart:io';

void main() {
  var directory = Directory('mon_repertoire');
  directory.deleteSync(recursive: true);
}

Dans cet exemple, nous utilisons la méthode deleteSync() pour supprimer le répertoire spécifié de manière synchrone. L'argument recursive: true indique à Dart de supprimer récursivement tous les fichiers et sous-répertoires contenus dans le répertoire.

Pour lister les fichiers et répertoires d'un répertoire, vous pouvez utiliser la méthode list() de la classe Directory. Voici un exemple :

import 'dart:io';

void main() {
  var directory = Directory('mon_repertoire');
  var contents = directory.listSync();
  
  for (var entity in contents) {
    print(entity.path);
  }
}

Dans cet exemple, nous utilisons la méthode listSync() pour obtenir une liste des fichiers et répertoires contenus dans le répertoire spécifié. Ensuite, nous parcourons la liste à l'aide d'une boucle for et affichons le chemin de chaque entité à l'aide de la fonction print().

Il est également possible de gérer les répertoires de manière asynchrone en utilisant les méthodes create(), delete() et list() avec le mot-clé await.
