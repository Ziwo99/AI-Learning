Utilisation de Git avec GitHub

GitHub est une plateforme de développement collaboratif basée sur Git. Elle offre des fonctionnalités avancées pour la gestion de projets, la collaboration et le suivi des problèmes. Voici comment utiliser Git avec GitHub :

Cloner un dépôt GitHub

Pour cloner un dépôt GitHub, utilisez la commande suivante : 

git clone https://github.com/utilisateur/nom-du-depot.git

Cela créera une copie locale du dépôt sur votre machine.

Ajouter et supprimer des dépôts distants

Pour ajouter un dépôt distant à votre projet Git, utilisez la commande suivante : 

git remote add origin https://github.com/utilisateur/nom-du-depot.git

Cela ajoutera un alias "origin" pour le dépôt distant spécifié. Vous pouvez ensuite pousser vos modifications vers ce dépôt en utilisant la commande "git push origin".

Pour supprimer un dépôt distant de votre projet Git, utilisez la commande suivante : 

git remote remove origin

Cela supprimera l'alias "origin" du dépôt distant.

Synchronisation avec des dépôts distants

Pour récupérer les dernières modifications d'un dépôt distant dans votre dépôt local, utilisez la commande suivante : 

git pull origin

Cela fusionnera les modifications du dépôt distant dans votre branche locale.

Pour pousser vos modifications locales vers un dépôt distant, utilisez la commande suivante : 

git push origin

Cela enverra vos modifications vers le dépôt distant spécifié.

Utilisation des branches distantes

Les branches distantes sont des branches qui existent dans le dépôt distant. Pour afficher les branches distantes disponibles, utilisez la commande suivante : 

git branch -r

Pour créer une branche locale à partir d'une branche distante, utilisez la commande suivante : 

git checkout -b nom-de-branche origin/nom-de-branche

Cela créera une nouvelle branche locale basée sur la branche distante spécifiée.
