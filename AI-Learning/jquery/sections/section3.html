Modification du contenu HTML

La manipulation du contenu HTML est l'une des fonctionnalités les plus puissantes de jQuery. Elle permet de modifier le contenu d'un élément HTML de manière dynamique.

Pour modifier le contenu HTML d'un élément avec jQuery, on utilise la méthode html(). Cette méthode permet de récupérer le contenu HTML d'un élément ou de le modifier.

Voici un exemple qui montre comment modifier le contenu HTML d'un élément avec jQuery :

$(document).ready(function(){
  // Sélection de l'élément avec l'ID "myElement"
  var element = $("#myElement");
  
  // Modification du contenu HTML de l'élément
  element.html("Nouveau contenu");
});

Modification des attributs

En plus de la modification du contenu HTML, jQuery permet également de modifier les attributs des éléments HTML. Cela peut être utile pour changer des attributs tels que l'URL d'une image, la valeur d'un champ de formulaire, etc.

Pour modifier un attribut avec jQuery, on utilise la méthode attr(). Cette méthode permet de récupérer la valeur d'un attribut ou de le modifier.

Voici un exemple qui montre comment modifier un attribut avec jQuery :

$(document).ready(function(){
  // Sélection de l'élément avec l'ID "myImage"
  var image = $("#myImage");
  
  // Modification de l'attribut "src" de l'image
  image.attr("src", "nouvelle_image.jpg");
});

Ajout et suppression d'éléments

jQuery permet également d'ajouter et de supprimer des éléments HTML de manière dynamique. Cela peut être utile pour ajouter de nouveaux éléments à une liste, supprimer des éléments indésirables, etc.

Pour ajouter un nouvel élément avec jQuery, on utilise la méthode append(). Cette méthode permet d'ajouter un élément à la fin d'un autre élément.

Voici un exemple qui montre comment ajouter un nouvel élément avec jQuery :

$(document).ready(function(){
  // Sélection de l'élément avec l'ID "myList"
  var list = $("#myList");
  
  // Ajout d'un nouvel élément à la liste
  list.append("
  • Nouvel élément
    • ");
});

    Pour supprimer un élément avec jQuery, on utilise la méthode remove(). Cette méthode permet de supprimer un élément de la page.

    Voici un exemple qui montre comment supprimer un élément avec jQuery :

    $(document).ready(function(){
  // Sélection de l'élément avec l'ID "myElement"
  var element = $("#myElement");
  
  // Suppression de l'élément
  element.remove();
});

    Manipulation des classes CSS

    La manipulation des classes CSS est une autre fonctionnalité importante de jQuery. Elle permet de modifier les classes d'un élément HTML, ce qui peut être utile pour ajouter ou supprimer des styles CSS dynamiquement.

    Pour ajouter une classe à un élément avec jQuery, on utilise la méthode addClass(). Cette méthode permet d'ajouter une ou plusieurs classes à un élément.

    Voici un exemple qui montre comment ajouter une classe à un élément avec jQuery :

    $(document).ready(function(){
  // Sélection de l'élément avec l'ID "myElement"
  var element = $("#myElement");
  
  // Ajout de la classe "highlight" à l'élément
  element.addClass("highlight");
});

    Pour supprimer une classe d'un élément avec jQuery, on utilise la méthode removeClass(). Cette méthode permet de supprimer une ou plusieurs classes d'un élément.

    Voici un exemple qui montre comment supprimer une classe d'un élément avec jQuery :

    $(document).ready(function(){
  // Sélection de l'élément avec l'ID "myElement"
  var element = $("#myElement");
  
  // Suppression de la classe "highlight" de l'élément
  element.removeClass("highlight");
});

    Il est également possible de basculer une classe d'un élément avec jQuery en utilisant la méthode toggleClass(). Cette méthode permet d'ajouter une classe si elle est absente, ou de la supprimer si elle est présente.

    Voici un exemple qui montre comment basculer une classe d'un élément avec jQuery :

    $(document).ready(function(){
  // Sélection de l'élément avec l'ID "myElement"
  var element = $("#myElement");
  
  // Basculement de la classe "highlight" de l'élément
  element.toggleClass("highlight");
});
