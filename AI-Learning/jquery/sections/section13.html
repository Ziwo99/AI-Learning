Section 13 : Projet pratique : Création d'une application web interactive avec jQuery

  • Revenir à la fiche du cours

Analyse des besoins du projet

L'objectif de ce projet est de créer une application web interactive en utilisant jQuery. L'application sera un gestionnaire de tâches qui permettra aux utilisateurs de créer, modifier et supprimer des tâches.

Pour répondre aux besoins du projet, nous aurons besoin des fonctionnalités suivantes :

  • Création d'une nouvelle tâche
  • Affichage de la liste des tâches existantes
  • Modification d'une tâche existante
  • Suppression d'une tâche existante

Conception de l'interface utilisateur

Pour concevoir l'interface utilisateur de notre application, nous aurons besoin des éléments HTML suivants :

  • Un formulaire pour ajouter une nouvelle tâche, comprenant un champ de saisie et un bouton "Ajouter"
  • Une liste pour afficher les tâches existantes
  • Pour chaque tâche, un élément de liste contenant le nom de la tâche et des boutons "Modifier" et "Supprimer"

Voici un exemple de structure HTML pour notre interface utilisateur :

<form id="task-form">
  <input type="text" id="task-input" placeholder="Ajouter une tâche">
  <button id="add-task-button">Ajouter</button>
</form>

<ul id="task-list">
  <li class="task-item">
    <span class="task-name">Tâche 1</span>
    <button class="edit-task-button">Modifier</button>
    <button class="delete-task-button">Supprimer</button>
  </li>
  <li class="task-item">
    <span class="task-name">Tâche 2</span>
    <button class="edit-task-button">Modifier</button>
    <button class="delete-task-button">Supprimer</button>
  </li>
  <li class="task-item">
    <span class="task-name">Tâche 3</span>
    <button class="edit-task-button">Modifier</button>
    <button class="delete-task-button">Supprimer</button>
  </li>
</ul>

Implémentation des fonctionnalités avec jQuery

Pour implémenter les fonctionnalités de notre application, nous utiliserons les méthodes et les événements de jQuery. Voici les étapes pour chaque fonctionnalité :

Création d'une nouvelle tâche

Lorsque l'utilisateur remplit le formulaire et clique sur le bouton "Ajouter", nous devons ajouter une nouvelle tâche à la liste. Voici comment nous pouvons le faire :

$('#add-task-button').click(function() {
  var taskName = $('#task-input').val();
  var taskItem = '<li class="task-item"><span class="task-name">' + taskName + '</span><button class="edit-task-button">Modifier</button><button class="delete-task-button">Supprimer</button></li>';
  $('#task-list').append(taskItem);
  $('#task-input').val('');
});

Affichage de la liste des tâches existantes

Une fois que nous avons ajouté une nouvelle tâche à la liste, nous devons également afficher toutes les tâches existantes. Voici comment nous pouvons le faire :

$(document).ready(function() {
  // Récupérer les tâches existantes depuis une source de données (par exemple, une API)
  var tasks = ['Tâche 1', 'Tâche 2', 'Tâche 3'];

  // Parcourir les tâches et les ajouter à la liste
  for (var i = 0; i < tasks.length; i++) {
    var taskItem = '<li class="task-item"><span class="task-name">' + tasks[i] + '</span><button class="edit-task-button">Modifier</button><button class="delete-task-button">Supprimer</button></li>';
    $('#task-list').append(taskItem);
  }
});

Modification d'une tâche existante

Lorsque l'utilisateur clique sur le bouton "Modifier" d'une tâche, nous devons permettre à l'utilisateur de modifier le nom de la tâche. Voici comment nous pouvons le faire :

$(document).on('click', '.edit-task-button', function() {
  var taskItem = $(this).closest('.task-item');
  var taskName = taskItem.find('.task-name').text();
  var newTaskName = prompt('Modifier la tâche', taskName);

  if (newTaskName !== null) {
    taskItem.find('.task-name').text(newTaskName);
  }
});

Suppression d'une tâche existante

Lorsque l'utilisateur clique sur le bouton "Supprimer" d'une tâche, nous devons supprimer cette tâche de la liste. Voici comment nous pouvons le faire :

$(document).on('click', '.delete-task-button', function() {
  var taskItem = $(this).closest('.task-item');
  taskItem.remove();
});

Test et débogage de l'application

Pour tester notre application, nous devons nous assurer que toutes les fonctionnalités fonctionnent correctement. Voici quelques étapes de test que nous pouvons suivre :

  • Ajouter une nouvelle tâche et vérifier si elle est ajoutée à la liste
  • Modifier le nom d'une tâche existante et vérifier si le nom est mis à jour dans la liste
  • Supprimer une tâche existante et vérifier si elle est supprimée de la liste

Si des problèmes sont rencontrés lors des tests, nous pouvons utiliser les outils de débogage de navigateur pour identifier et résoudre les problèmes.
