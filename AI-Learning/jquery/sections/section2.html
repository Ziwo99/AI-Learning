Sélection d'éléments HTML

Les sélecteurs jQuery permettent de sélectionner des éléments HTML spécifiques dans une page web. Pour sélectionner un élément HTML, on utilise le symbole dollar ($) suivi de parenthèses contenant une chaîne de caractères représentant le sélecteur.

Par exemple, pour sélectionner tous les éléments de paragraphe (balise <p>) dans une page, on utilise le sélecteur "p" :

$("p")

Cela retournera un objet jQuery contenant tous les éléments de paragraphe de la page.

Utilisation des sélecteurs CSS

jQuery utilise les sélecteurs CSS pour sélectionner des éléments HTML. Les sélecteurs CSS permettent de cibler des éléments en fonction de leur balise, de leur classe, de leur ID, de leurs attributs, etc.

Par exemple, pour sélectionner tous les éléments de classe "example" dans une page, on utilise le sélecteur ".example" :

$(".example")

Cela retournera un objet jQuery contenant tous les éléments de classe "example" de la page.

Sélection d'éléments par attribut

Il est également possible de sélectionner des éléments en fonction de leurs attributs. On utilise le sélecteur "[attribut=valeur]" pour cela.

Par exemple, pour sélectionner tous les liens (<a>) ayant l'attribut "href" égal à "https://www.example.com", on utilise le sélecteur 'a[href="https://www.example.com"]' :

$('a[href="https://www.example.com"]')

Cela retournera un objet jQuery contenant tous les liens ayant l'attribut "href" égal à "https://www.example.com".

Sélection d'éléments par position

Il est possible de sélectionner des éléments en fonction de leur position dans la structure HTML. On utilise les sélecteurs ":first", ":last", ":even" et ":odd" pour cela.

Par exemple, pour sélectionner le premier élément de paragraphe (<p>) dans une page, on utilise le sélecteur "p:first" :

$("p:first")

Cela retournera un objet jQuery contenant le premier élément de paragraphe de la page.
