Création d'un tableau

Pour créer un tableau dans Trello, suivez les étapes suivantes :

  1. Connectez-vous à votre compte Trello.
  2. Cliquez sur le bouton "Créer un tableau" situé sur votre page d'accueil.
  3. Donnez un nom à votre tableau et ajoutez une description si nécessaire.
  4. Choisissez si vous souhaitez que votre tableau soit public ou privé.
  5. Cliquez sur le bouton "Créer" pour finaliser la création de votre tableau.

Personnalisation d'un tableau

Une fois que vous avez créé un tableau, vous pouvez le personnaliser selon vos besoins. Voici quelques options de personnalisation :

  • Changer la couleur d'arrière-plan du tableau : Cliquez sur le bouton "Modifier le fond" en haut à droite du tableau et choisissez une couleur.
  • Ajouter une image d'arrière-plan : Cliquez sur le bouton "Modifier le fond" et sélectionnez "Ajouter une image" pour télécharger une image depuis votre ordinateur.
  • Ajouter une couverture à une liste : Cliquez sur le bouton "Modifier la couverture" en haut à gauche de la liste et choisissez une image ou une couleur.
  • Ajouter des étiquettes personnalisées : Cliquez sur le bouton "Étiquettes" en haut à droite du tableau et choisissez "Créer une étiquette" pour ajouter une étiquette avec une couleur et un nom personnalisés.

Ajout et gestion des listes dans un tableau

Les listes sont utilisées pour organiser les cartes dans un tableau. Voici comment ajouter et gérer des listes :

  • Ajouter une liste : Cliquez sur le bouton "Ajouter une liste" à droite du tableau, donnez un nom à la liste et appuyez sur "Entrée" pour la créer.
  • Renommer une liste : Cliquez sur le nom de la liste, modifiez-le et appuyez sur "Entrée" pour enregistrer les modifications.
  • Déplacer une liste : Cliquez sur le titre de la liste et faites-la glisser vers l'endroit souhaité dans le tableau.
  • Archiver une liste : Cliquez sur le bouton "..." à droite du titre de la liste et sélectionnez "Archiver cette liste" pour la masquer du tableau.

Utilisation des cartes dans un tableau

Les cartes sont utilisées pour représenter les tâches, les idées ou les éléments à suivre dans un tableau. Voici comment utiliser les cartes :

  • Ajouter une carte : Cliquez sur le bouton "Ajouter une carte" dans une liste, donnez un nom à la carte et appuyez sur "Entrée" pour la créer.
  • Renommer une carte : Cliquez sur le nom de la carte, modifiez-le et appuyez sur "Entrée" pour enregistrer les modifications.
  • Déplacer une carte : Cliquez sur le titre de la carte et faites-la glisser vers une autre liste ou position dans le tableau.
  • Ajouter une description à une carte : Cliquez sur le bouton "Ajouter une description" dans une carte et saisissez le texte souhaité.
  • Ajouter des membres à une carte : Cliquez sur le bouton "Ajouter des membres" dans une carte et sélectionnez les membres à ajouter.

Utilisation des étiquettes et des filtres

Les étiquettes sont utilisées pour catégoriser et filtrer les cartes dans un tableau. Voici comment utiliser les étiquettes et les filtres :

  • Ajouter une étiquette à une carte : Cliquez sur le bouton "Étiquettes" dans une carte et sélectionnez une étiquette à ajouter.
  • Filtrer les cartes par étiquette : Cliquez sur le bouton "Filtres" en haut à droite du tableau et sélectionnez une étiquette pour afficher uniquement les cartes correspondantes.
