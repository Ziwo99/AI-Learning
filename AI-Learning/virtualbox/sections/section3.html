Démarrage et arrêt d'une machine virtuelle

Pour démarrer une machine virtuelle dans VirtualBox, suivez les étapes suivantes :

  1. Ouvrez VirtualBox sur votre ordinateur.
  2. Sélectionnez la machine virtuelle que vous souhaitez démarrer dans la liste des machines disponibles.
  3. Cliquez sur le bouton "Démarrer" pour lancer la machine virtuelle.
  4. Attendez que la machine virtuelle démarre et affiche l'écran d'accueil de l'OS invité.

Pour arrêter une machine virtuelle, suivez ces étapes :

  1. Sélectionnez la machine virtuelle que vous souhaitez arrêter dans la liste des machines disponibles.
  2. Cliquez sur le bouton "Arrêter" pour éteindre la machine virtuelle.
  3. Attendez que la machine virtuelle s'arrête complètement.

Gestion des snapshots

Les snapshots sont des instantanés de l'état d'une machine virtuelle à un moment donné. Ils permettent de sauvegarder l'état actuel de la machine virtuelle et de revenir à cet état ultérieurement si nécessaire. Voici comment gérer les snapshots dans VirtualBox :

  1. Sélectionnez la machine virtuelle pour laquelle vous souhaitez créer un snapshot.
  2. Cliquez sur le menu "Machine" dans la barre de menu de VirtualBox.
  3. Sélectionnez "Prendre un snapshot" pour créer un nouveau snapshot.
  4. Donnez un nom et une description au snapshot, puis cliquez sur "OK".
  5. Pour restaurer un snapshot, sélectionnez la machine virtuelle, cliquez sur le menu "Machine" et choisissez "Restaurer le snapshot".
  6. Sélectionnez le snapshot que vous souhaitez restaurer et cliquez sur "Restaurer".

Partage de fichiers entre l'hôte et la machine virtuelle

VirtualBox permet de partager des fichiers entre l'hôte (votre ordinateur physique) et la machine virtuelle. Voici comment configurer le partage de fichiers :

  1. Dans VirtualBox, sélectionnez la machine virtuelle pour laquelle vous souhaitez configurer le partage de fichiers.
  2. Cliquez sur le menu "Périphériques" dans la barre de menu de VirtualBox.
  3. Sélectionnez "Insérer l'image CD des additions invité" pour monter le CD d'additions invité dans la machine virtuelle.
  4. Sur la machine virtuelle, ouvrez le CD d'additions invité et exécutez le programme d'installation.
  5. Redémarrez la machine virtuelle après l'installation des additions invité.
  6. Une fois la machine virtuelle redémarrée, vous pouvez accéder aux fichiers partagés en utilisant l'explorateur de fichiers de l'OS invité.

Utilisation des périphériques USB dans VirtualBox

VirtualBox permet d'utiliser des périphériques USB dans une machine virtuelle. Voici comment configurer l'utilisation des périphériques USB :

  1. Dans VirtualBox, sélectionnez la machine virtuelle pour laquelle vous souhaitez configurer l'utilisation des périphériques USB.
  2. Cliquez sur le menu "Périphériques" dans la barre de menu de VirtualBox.
  3. Sélectionnez le périphérique USB que vous souhaitez utiliser dans la liste des périphériques disponibles.
  4. Si le périphérique USB n'apparaît pas dans la liste, assurez-vous qu'il est connecté à votre ordinateur et qu'il est reconnu par le système d'exploitation hôte.
  5. Une fois le périphérique USB sélectionné, il sera disponible dans la machine virtuelle.
