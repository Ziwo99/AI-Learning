Section 4 : Les collections en Kotlin

Section 4 : Les collections en Kotlin

Listes

Les listes sont des collections ordonnées d'éléments en Kotlin. Elles peuvent contenir des éléments de différents types et peuvent être modifiées (ajout, suppression, modification). Pour créer une liste, vous pouvez utiliser la fonction listOf() en spécifiant les éléments à inclure entre parenthèses. Voici un exemple :

val fruits = listOf("pomme", "banane", "orange")

Vous pouvez accéder aux éléments d'une liste en utilisant l'opérateur d'indexation []. Par exemple, pour accéder au premier élément de la liste fruits, vous pouvez utiliser fruits[0]. Les listes en Kotlin sont immuables, ce qui signifie que vous ne pouvez pas modifier directement les éléments d'une liste. Si vous souhaitez modifier une liste, vous devez utiliser une liste mutable.

Ensembles

Les ensembles en Kotlin sont des collections qui ne contiennent pas d'éléments en double et n'ont pas d'ordre spécifique. Pour créer un ensemble, vous pouvez utiliser la fonction setOf() en spécifiant les éléments à inclure entre parenthèses. Voici un exemple :

val nombres = setOf(1, 2, 3, 4, 5)

Les ensembles en Kotlin sont immuables, ce qui signifie que vous ne pouvez pas ajouter ou supprimer des éléments directement à partir d'un ensemble. Si vous souhaitez modifier un ensemble, vous devez utiliser un ensemble mutable.

Maps

Les maps en Kotlin sont des collections qui associent des clés à des valeurs. Chaque clé doit être unique dans une map. Pour créer une map, vous pouvez utiliser la fonction mapOf() en spécifiant les paires clé-valeur entre parenthèses. Voici un exemple :

val capitales = mapOf("France" to "Paris", "Espagne" to "Madrid", "Italie" to "Rome")

Vous pouvez accéder à une valeur dans une map en utilisant l'opérateur d'indexation [] avec la clé correspondante. Par exemple, pour accéder à la capitale de la France dans la map capitales, vous pouvez utiliser capitales["France"].

Manipulation des collections

En Kotlin, il existe de nombreuses fonctions utiles pour manipuler les collections. Voici quelques-unes des fonctions les plus couramment utilisées :

  • filter() : permet de filtrer les éléments d'une collection en fonction d'un prédicat.
  • map() : permet de transformer chaque élément d'une collection en un autre élément.
  • reduce() : permet de réduire une collection à une seule valeur en appliquant une opération sur chaque élément.
  • sorted() : permet de trier les éléments d'une collection selon un ordre spécifié.

Voici un exemple d'utilisation de ces fonctions :

val nombres = listOf(1, 2, 3, 4, 5)

val nombresPairs = nombres.filter { it % 2 == 0 }

val carres = nombres.map { it * it }

val somme = nombres.reduce { acc, nombre -> acc + nombre }

val nombresTries = nombres.sorted()

En utilisant ces fonctions, vous pouvez effectuer des opérations complexes sur les collections de manière concise et lisible.
