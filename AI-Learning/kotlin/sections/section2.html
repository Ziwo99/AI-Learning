Section 2 : Les bases de Kotlin

Section 2 : Les bases de Kotlin

Syntaxe de base

La syntaxe de base de Kotlin est similaire à celle d'autres langages de programmation tels que Java. Voici un exemple de code Kotlin :

fun main() {
    println("Hello, Kotlin!")
}

Variables et types de données

En Kotlin, vous pouvez déclarer des variables en utilisant le mot-clé var pour les variables mutables ou val pour les variables immuables. Voici un exemple :

var age: Int = 25
val name: String = "John"

Opérateurs

Kotlin prend en charge les opérateurs arithmétiques (+, -, *, /), les opérateurs de comparaison (==, !=, <, >, <=, >=) et les opérateurs logiques (&&, ||, !). Voici un exemple :

val a = 10
val b = 5
val sum = a + b
val isGreater = a > b
val logicalAnd = (a > 0) && (b > 0)

Structures de contrôle (boucles, conditions)

En Kotlin, vous pouvez utiliser des boucles for et while ainsi que des instructions conditionnelles if et when. Voici un exemple :

val numbers = listOf(1, 2, 3, 4, 5)

for (number in numbers) {
    println(number)
}

var i = 0
while (i < 5) {
    println(i)
    i++
}

val x = 10
val y = 5

if (x > y) {
    println("x is greater than y")
} else {
    println("x is not greater than y")
}

val z = 3

when (z) {
    1 -> println("z is 1")
    2 -> println("z is 2")
    else -> println("z is neither 1 nor 2")
}

Fonctions et lambdas

En Kotlin, vous pouvez définir des fonctions en utilisant le mot-clé fun. Les lambdas sont également pris en charge. Voici un exemple :

fun addNumbers(a: Int, b: Int): Int {
    return a + b
}

val sum = addNumbers(5, 10)

val multiply: (Int, Int) -> Int = { x, y -> x * y }
val product = multiply(3, 4)

Nullabilité

En Kotlin, vous pouvez spécifier si une variable peut être nulle en utilisant le type de données suivi du symbole ?. Voici un exemple :

var nullableString: String? = null

if (nullableString != null) {
    println(nullableString.length)
} else {
    println("nullableString is null")
}

val length = nullableString?.length
