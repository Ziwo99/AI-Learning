Réalisation d'un projet de gestion de tâches

Dans cette sous-section, nous allons créer une application de gestion de tâches en utilisant Kotlin. Cette application permettra aux utilisateurs de créer, modifier et supprimer des tâches, ainsi que de les marquer comme terminées.

Voici un exemple de code pour la classe Task :

class Task(val id: Int, var title: String, var description: String, var isCompleted: Boolean) {
    fun markAsCompleted() {
        isCompleted = true
    }
}

La classe Task représente une tâche avec un identifiant, un titre, une description et un indicateur de statut de complétion. La méthode markAsCompleted() permet de marquer une tâche comme terminée en mettant à jour la valeur de isCompleted.

Ensuite, nous allons créer une classe TaskManager pour gérer les tâches :

class TaskManager {
    private val tasks = mutableListOf()

    fun addTask(task: Task) {
        tasks.add(task)
    }

    fun removeTask(task: Task) {
        tasks.remove(task)
    }

    fun getTasks(): List {
        return tasks
    }
}

La classe TaskManager utilise une liste mutable de tâches pour stocker les tâches ajoutées. Les méthodes addTask(), removeTask() et getTasks() permettent respectivement d'ajouter une tâche, de supprimer une tâche et de récupérer la liste complète des tâches.

Enfin, nous allons créer une classe Main pour exécuter notre application :

fun main() {
    val task1 = Task(1, "Faire les courses", "Acheter du lait et du pain", false)
    val task2 = Task(2, "Faire du sport", "Courir pendant 30 minutes", false)

    val taskManager = TaskManager()
    taskManager.addTask(task1)
    taskManager.addTask(task2)

    val tasks = taskManager.getTasks()
    for (task in tasks) {
        println("ID: ${task.id}")
        println("Titre: ${task.title}")
        println("Description: ${task.description}")
        println("Terminée: ${task.isCompleted}")
        println()
    }
}

Ce code crée deux tâches, les ajoute au gestionnaire de tâches, puis affiche les détails de chaque tâche.

Développement d'une application de chat en temps réel

Dans cette sous-section, nous allons développer une application de chat en temps réel en utilisant Kotlin. Cette application permettra aux utilisateurs de s'envoyer des messages instantanés.

Voici un exemple de code pour la classe Message :

class Message(val sender: String, val content: String, val timestamp: Long) {
    fun displayMessage() {
        println("[$timestamp] $sender: $content")
    }
}

La classe Message représente un message avec un expéditeur, un contenu et un horodatage. La méthode displayMessage() affiche le message dans la console avec le format [horodatage] expéditeur: contenu.

Ensuite, nous allons créer une classe ChatRoom pour gérer les messages :

class ChatRoom {
    private val messages = mutableListOf()

    fun sendMessage(message: Message) {
        messages.add(message)
    }

    fun displayMessages() {
        for (message in messages) {
            message.displayMessage()
        }
    }
}

La classe ChatRoom utilise une liste mutable de messages pour stocker les messages envoyés. Les méthodes sendMessage() et displayMessages() permettent respectivement d'envoyer un message et d'afficher tous les messages dans la console.

Enfin, nous allons créer une classe Main pour exécuter notre application :

fun main() {
    val message1 = Message("Alice", "Bonjour tout le monde!", System.currentTimeMillis())
    val message2 = Message("Bob", "Salut Alice!", System.currentTimeMillis())

    val chatRoom = ChatRoom()
    chatRoom.sendMessage(message1)
    chatRoom.sendMessage(message2)

    chatRoom.displayMessages()
}

Ce code crée deux messages, les envoie dans la salle de chat, puis affiche tous les messages dans la console.
