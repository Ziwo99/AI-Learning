Création et validation de formulaires

Les formulaires sont un élément essentiel de toute application web, car ils permettent aux utilisateurs d'interagir avec le contenu et de soumettre des données. Dans Angular, la création et la validation de formulaires sont simplifiées grâce à l'utilisation du module FormsModule.

Pour créer un formulaire dans Angular, nous devons d'abord importer le module FormsModule dans notre module principal. Pour ce faire, nous ajoutons l'importation suivante :

import { FormsModule } from '@angular/forms';

@NgModule({
  imports: [
    FormsModule
  ],
  // ...
})
export class AppModule { }

Une fois que nous avons importé le module FormsModule, nous pouvons utiliser les directives fournies par Angular pour créer et valider nos formulaires.

Pour créer un formulaire, nous utilisons la balise <form> et nous lui attribuons une variable de référence à l'aide de la directive ngForm. Par exemple :

<form #myForm="ngForm">
  
</form>

Nous pouvons ensuite ajouter des champs de formulaire à l'intérieur de la balise <form>. Par exemple, pour ajouter un champ de saisie de texte, nous utilisons la balise <input> :

<form #myForm="ngForm">
  <input type="text" name="firstName" [(ngModel)]="firstName">
</form>

Dans cet exemple, nous avons ajouté un champ de saisie de texte avec le nom "firstName" et nous avons lié sa valeur à la variable "firstName" à l'aide de la directive ngModel.

Pour valider les données saisies par l'utilisateur, nous pouvons utiliser les directives de validation fournies par Angular. Par exemple, pour rendre le champ "firstName" obligatoire, nous pouvons ajouter l'attribut required à la balise <input> :

<form #myForm="ngForm">
  <input type="text" name="firstName" [(ngModel)]="firstName" required>
</form>

Angular fournira automatiquement des classes CSS pour indiquer si un champ est valide ou non. Nous pouvons également accéder aux états de validation du formulaire et des champs individuels à l'aide de la variable de référence "myForm". Par exemple, pour vérifier si le formulaire est valide, nous pouvons utiliser la propriété "valid" :

<form #myForm="ngForm">
  <input type="text" name="firstName" [(ngModel)]="firstName" required>
  <button type="submit" [disabled]="!myForm.valid">Submit</button>
</form>

Dans cet exemple, le bouton de soumission sera désactivé tant que le formulaire n'est pas valide.

Utilisation des contrôles de formulaire

Angular fournit une variété de contrôles de formulaire prédéfinis que nous pouvons utiliser pour faciliter la saisie des données par l'utilisateur. Ces contrôles incluent des champs de saisie de texte, des cases à cocher, des boutons radio, des listes déroulantes, etc.

Pour utiliser un contrôle de formulaire prédéfini, nous devons simplement ajouter la balise appropriée à notre formulaire et lui attribuer un nom unique à l'aide de l'attribut "name". Par exemple, pour ajouter une case à cocher, nous utilisons la balise <input> avec le type "checkbox" :

<form #myForm="ngForm">
  <input type="checkbox" name="agree" [(ngModel)]="agree"> I agree to the terms and conditions
</form>

Dans cet exemple, nous avons ajouté une case à cocher avec le nom "agree" et nous avons lié sa valeur à la variable "agree" à l'aide de la directive ngModel.

Nous pouvons également utiliser des contrôles de formulaire personnalisés en créant nos propres composants. Pour ce faire, nous devons créer un composant avec un modèle de formulaire et utiliser la directive ngModel pour lier les valeurs du composant au formulaire principal.

Gestion des soumissions de formulaire

Lorsque l'utilisateur soumet un formulaire, nous pouvons capturer cet événement en utilisant la directive ngSubmit. Pour ce faire, nous ajoutons l'attribut "ngSubmit" à la balise <form> et nous lui attribuons une méthode à appeler lors de la soumission du formulaire. Par exemple :

<form #myForm="ngForm" (ngSubmit)="onSubmit()">
  
</form>

Dans cet exemple, nous avons ajouté l'attribut "ngSubmit" à la balise <form> et nous avons lié cet événement à la méthode "onSubmit()" de notre composant.

La méthode "onSubmit()" sera appelée lorsque l'utilisateur soumettra le formulaire. À l'intérieur de cette méthode, nous pouvons effectuer des actions telles que l'envoi des données du formulaire à un serveur, la validation supplémentaire des données, etc.

Il est également possible d'annuler la soumission du formulaire en utilisant la méthode "preventDefault()" de l'objet Event. Par exemple :

onSubmit(event: Event) {
  event.preventDefault();
  // Autres actions à effectuer lors de la soumission du formulaire
}

Dans cet exemple, nous avons utilisé la méthode "preventDefault()" pour empêcher le comportement par défaut de la soumission du formulaire.

En résumé, dans cette section, nous avons appris à créer et à valider des formulaires dans Angular en utilisant le module FormsModule. Nous avons également exploré l'utilisation des contrôles de formulaire prédéfinis et la gestion des soumissions de formulaire. Les formulaires sont un élément essentiel de toute application web, et Angular facilite grandement leur création et leur gestion.
