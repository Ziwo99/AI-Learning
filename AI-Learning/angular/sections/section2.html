Prérequis pour l'installation d'Angular

Pour pouvoir installer et utiliser Angular, vous devez vous assurer d'avoir les éléments suivants :

  • Node.js : Angular nécessite Node.js pour exécuter les commandes et les outils nécessaires. Vous pouvez télécharger Node.js à partir du site officiel et suivre les instructions d'installation spécifiques à votre système d'exploitation.
  • npm : npm est le gestionnaire de paquets de Node.js. Il est utilisé pour installer les dépendances nécessaires à Angular. npm est généralement installé automatiquement avec Node.js, mais vous pouvez vérifier sa version en exécutant la commande npm -v dans votre terminal.

Installation d'Angular CLI

Angular CLI (Command Line Interface) est un outil en ligne de commande qui facilite la création, la gestion et le déploiement de projets Angular. Pour l'installer, suivez les étapes suivantes :

  1. Ouvrez votre terminal ou votre invite de commande.
  2. Exécutez la commande suivante pour installer Angular CLI de manière globale sur votre système : npm install -g @angular/cli
  3. Une fois l'installation terminée, vous pouvez vérifier la version d'Angular CLI en exécutant la commande ng --version.

Configuration initiale d'un projet Angular

Une fois Angular CLI installé, vous pouvez créer un nouveau projet Angular en suivant ces étapes :

  1. Ouvrez votre terminal ou votre invite de commande.
  2. Naviguez vers le répertoire dans lequel vous souhaitez créer votre projet.
  3. Exécutez la commande suivante pour créer un nouveau projet Angular : ng new nom-du-projet (remplacez "nom-du-projet" par le nom que vous souhaitez donner à votre projet).
  4. Angular CLI vous posera quelques questions pour configurer votre projet, telles que le style CSS à utiliser et si vous souhaitez ajouter des tests unitaires. Répondez aux questions en fonction de vos besoins.
  5. Une fois la configuration terminée, Angular CLI créera le projet et installera les dépendances nécessaires.

Vous pouvez maintenant vous rendre dans le répertoire de votre projet et lancer l'application en exécutant la commande ng serve. Cela démarrera un serveur de développement et vous pourrez accéder à votre application dans votre navigateur à l'adresse http://localhost:4200.
