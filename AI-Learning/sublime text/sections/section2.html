Section 2 : Fonctionnalités de base de Sublime Text

Création et ouverture de fichiers

Pour créer un nouveau fichier dans Sublime Text, vous pouvez utiliser la combinaison de touches Ctrl + N ou sélectionner File dans la barre de menu, puis choisir New File. Vous pouvez également cliquer avec le bouton droit de la souris dans la zone de travail et sélectionner New File.

Pour ouvrir un fichier existant, vous pouvez utiliser la combinaison de touches Ctrl + O ou sélectionner File dans la barre de menu, puis choisir Open File. Vous pouvez également cliquer avec le bouton droit de la souris dans la zone de travail et sélectionner Open File.

Gestion des projets

Sublime Text vous permet de gérer vos projets de manière efficace. Pour créer un nouveau projet, vous pouvez sélectionner Project dans la barre de menu, puis choisir New Project. Vous pouvez également utiliser la combinaison de touches Ctrl + Alt + N.

Pour ajouter des fichiers à votre projet, vous pouvez sélectionner Project dans la barre de menu, puis choisir Add Folder to Project. Vous pouvez également faire glisser et déposer les fichiers directement dans la barre latérale.

Pour ouvrir un projet existant, vous pouvez sélectionner Project dans la barre de menu, puis choisir Open Project. Vous pouvez également utiliser la combinaison de touches Ctrl + Alt + O.

Navigation dans le code

Sublime Text offre de nombreuses fonctionnalités de navigation pour vous aider à vous déplacer rapidement dans votre code. Voici quelques-unes des fonctionnalités les plus utiles :

  • Go to Definition : Cette fonctionnalité vous permet de naviguer rapidement vers la définition d'une fonction ou d'une variable. Vous pouvez l'utiliser en cliquant avec le bouton droit de la souris sur le nom de la fonction ou de la variable, puis en sélectionnant Go to Definition. Vous pouvez également utiliser la combinaison de touches Ctrl + Click.
  • Go to Line : Cette fonctionnalité vous permet de naviguer directement vers une ligne spécifique dans votre fichier. Vous pouvez l'utiliser en sélectionnant Goto dans la barre de menu, puis en choisissant Goto Line. Vous pouvez également utiliser la combinaison de touches Ctrl + G.
  • Go to Symbol : Cette fonctionnalité vous permet de naviguer vers une fonction ou une classe spécifique dans votre fichier. Vous pouvez l'utiliser en sélectionnant Goto dans la barre de menu, puis en choisissant Goto Symbol. Vous pouvez également utiliser la combinaison de touches Ctrl + R.

Sélection et manipulation du texte

Sublime Text offre de nombreuses fonctionnalités pour sélectionner et manipuler le texte de manière efficace. Voici quelques-unes des fonctionnalités les plus utiles :

  • Multiple Sélection : Cette fonctionnalité vous permet de sélectionner plusieurs occurrences d'un mot ou d'une expression dans votre fichier. Vous pouvez l'utiliser en sélectionnant le mot ou l'expression, puis en utilisant la combinaison de touches Ctrl + D pour sélectionner les occurrences suivantes. Vous pouvez également utiliser Ctrl + Shift + L pour sélectionner toutes les occurrences en une seule fois.
  • Indentation : Sublime Text vous permet d'indenter facilement votre code. Vous pouvez indenter une ligne en utilisant la combinaison de touches Tab. Vous pouvez également indenter plusieurs lignes en les sélectionnant, puis en utilisant la combinaison de touches Tab.
  • Duplication : Vous pouvez dupliquer une ligne en utilisant la combinaison de touches Ctrl + Shift + D. Vous pouvez également dupliquer plusieurs lignes en les sélectionnant, puis en utilisant la combinaison de touches Ctrl + Shift + D.

Recherche et remplacement

Sublime Text offre de puissantes fonctionnalités de recherche et de remplacement. Voici quelques-unes des fonctionnalités les plus utiles :

  • Find : Vous pouvez rechercher un mot ou une expression dans votre fichier en utilisant la combinaison de touches Ctrl + F. Vous pouvez également sélectionner Edit dans la barre de menu, puis choisir Find.
  • Replace : Vous pouvez remplacer un mot ou une expression par un autre dans votre fichier en utilisant la combinaison de touches Ctrl + H. Vous pouvez également sélectionner Edit dans la barre de menu, puis choisir Replace.
  • Find in Files : Cette fonctionnalité vous permet de rechercher un mot ou une expression dans tous les fichiers d'un projet. Vous pouvez l'utiliser en sélectionnant Find dans la barre de menu, puis en choisissant Find in Files. Vous pouvez également utiliser la combinaison de touches Ctrl + Shift + F.

Utilisation des raccourcis clavier

Sublime Text offre de nombreux raccourcis clavier pour vous aider à travailler plus rapidement. Voici quelques-uns des raccourcis clavier les plus utiles :

  • Ctrl + S : Enregistre le fichier en cours.
  • Ctrl + Z : Annule la dernière action.
  • Ctrl + Y : Rétablit la dernière action annulée.
  • Ctrl + X : Coupe la sélection en cours.
  • Ctrl + C : Copie la sélection en cours.
  • Ctrl + V : Colle le contenu du presse-papiers.

Ce sont seulement quelques-unes des fonctionnalités de base de Sublime Text. La prochaine section de cette formation e-learning abordera la personnalisation de Sublime Text.
