Utilisation des commandes de base

Sublime Text offre un large éventail de commandes de base pour faciliter l'édition de code. Voici quelques-unes des commandes les plus couramment utilisées :

Ctrl + X

Cette commande permet de couper la sélection actuelle et de la copier dans le presse-papiers.

Ctrl + C

Cette commande permet de copier la sélection actuelle dans le presse-papiers sans la couper.

Ctrl + V

Cette commande permet de coller le contenu du presse-papiers à l'emplacement du curseur.

Ctrl + Z

Cette commande permet d'annuler la dernière action effectuée.

Ctrl + Y

Cette commande permet de rétablir la dernière action annulée.

Utilisation des commandes avancées

En plus des commandes de base, Sublime Text propose également des commandes avancées pour une édition de code plus efficace :

Ctrl + D

Cette commande permet de sélectionner la prochaine occurrence du mot actuellement sélectionné.

Ctrl + Shift + D

Cette commande permet de dupliquer la ligne ou la sélection actuelle.

Ctrl + /

Cette commande permet de commenter ou décommenter la ligne ou la sélection actuelle.

Ctrl + Shift + L

Cette commande permet de sélectionner toutes les occurrences du mot actuellement sélectionné.

Utilisation des expressions régulières

Sublime Text prend en charge l'utilisation des expressions régulières pour effectuer des recherches et des remplacements avancés :

Ctrl + F

Cette commande permet d'ouvrir la barre de recherche pour trouver du texte dans le fichier actuel.

Ctrl + H

Cette commande permet d'ouvrir la barre de recherche et de remplacement pour trouver et remplacer du texte dans le fichier actuel.

Ctrl + Shift + F

Cette commande permet d'ouvrir la barre de recherche dans tous les fichiers d'un projet pour trouver du texte.

Utilisation des macros

Les macros permettent d'enregistrer une séquence d'actions et de les rejouer automatiquement :

Ctrl + Shift + P

Cette commande permet d'ouvrir la palette de commandes.

Ctrl + Q

Cette commande permet d'enregistrer une macro.

Ctrl + Shift + Q

Cette commande permet de rejouer la dernière macro enregistrée.

Il est important de noter que les commandes avancées, les expressions régulières et les macros peuvent grandement améliorer votre productivité lors de l'édition de code avec Sublime Text. Prenez le temps de les explorer et de les maîtriser pour optimiser votre flux de travail.
