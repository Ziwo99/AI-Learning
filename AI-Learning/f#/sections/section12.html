Exemples de code avancés

Voici quelques exemples de code avancés en F# :

let rec fibonacci n =
    match n with
    | 0 -> 0
    | 1 -> 1
    | _ -> fibonacci (n - 1) + fibonacci (n - 2)

let result = fibonacci 10
printfn "The 10th Fibonacci number is %d" result
type Shape =
    | Circle of float
    | Rectangle of float * float

let calculateArea shape =
    match shape with
    | Circle radius -> Math.PI * radius * radius
    | Rectangle width height -> width * height

let circleArea = calculateArea (Circle 5.0)
let rectangleArea = calculateArea (Rectangle 3.0 4.0)

printfn "Area of circle: %f" circleArea
printfn "Area of rectangle: %f" rectangleArea

Utilisez ces exemples pour approfondir votre compréhension de F# et pour résoudre des problèmes plus complexes.
