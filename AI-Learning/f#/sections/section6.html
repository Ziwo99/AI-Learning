Section 6: Manipulation de données en F#

Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes lors de la manipulation de données en F#. Pour lire un fichier, vous pouvez utiliser la fonction System.IO.File.ReadAllText qui lit tout le contenu d'un fichier et le retourne sous forme de chaîne de caractères. Voici un exemple de lecture d'un fichier :

let content = System.IO.File.ReadAllText("chemin/vers/monfichier.txt")
printfn "%s" content

Pour écrire dans un fichier, vous pouvez utiliser la fonction System.IO.File.WriteAllText qui prend en paramètre le chemin du fichier et le contenu à écrire. Voici un exemple d'écriture dans un fichier :

let content = "Contenu à écrire dans le fichier"
System.IO.File.WriteAllText("chemin/vers/monfichier.txt", content)

Accès aux bases de données

F# offre plusieurs options pour accéder aux bases de données, notamment en utilisant des bibliothèques telles que SqlClient pour les bases de données SQL ou MongoDB.Driver pour les bases de données MongoDB. Voici un exemple d'accès à une base de données SQL :

open System.Data.SqlClient

let connectionString = "chaîne de connexion à la base de données"
let query = "SELECT * FROM MaTable"

use connection = new SqlConnection(connectionString)
let command = new SqlCommand(query, connection)

connection.Open()
let reader = command.ExecuteReader()

while reader.Read() do
    let value = reader.GetString(0)
    printfn "%s" value

reader.Close()

Traitement de données JSON

Le traitement de données JSON est courant dans les applications modernes. En F#, vous pouvez utiliser la bibliothèque Newtonsoft.Json pour manipuler des données JSON. Voici un exemple de manipulation de données JSON :

open Newtonsoft.Json

type Person = { Name: string; Age: int }

let json = "{ \"Name\": \"John\", \"Age\": 30 }"
let person = JsonConvert.DeserializeObject<Person>(json)

printfn "Name: %s" person.Name
printfn "Age: %d" person.Age

Traitement de données XML

Le traitement de données XML est également courant dans les applications. En F#, vous pouvez utiliser la bibliothèque System.Xml pour manipuler des données XML. Voici un exemple de manipulation de données XML :

open System.Xml

let xml = "John30"
let doc = new XmlDocument()
doc.LoadXml(xml)

let name = doc.SelectSingleNode("//Name").InnerText
let age = doc.SelectSingleNode("//Age").InnerText

printfn "Name: %s" name
printfn "Age: %s" age

Manipulation de données CSV

La manipulation de données CSV est courante lors du traitement de données tabulaires. En F#, vous pouvez utiliser la bibliothèque FSharp.Data pour manipuler des données CSV. Voici un exemple de manipulation de données CSV :

open FSharp.Data

type MyCsv = CsvProvider<"chemin/vers/monfichier.csv">

let data = MyCsv.Load("chemin/vers/monfichier.csv")

for row in data.Rows do
    printfn "Name: %s" row.Name
    printfn "Age: %s" row.Age
