Section 11: Projets pratiques

Projet 1: Création d'une application de gestion de tâches

Dans ce projet, nous allons créer une application de gestion de tâches en utilisant F#. Cette application permettra aux utilisateurs de créer, modifier et supprimer des tâches, ainsi que de les marquer comme terminées.

Voici un exemple de code pour la création d'une tâche :

type Task = {
    id: int
    title: string
    description: string
    completed: bool
}

let createTask id title description =
    { id = id; title = title; description = description; completed = false }

La fonction createTask prend en paramètre l'identifiant, le titre et la description de la tâche, et renvoie une instance de la structure Task avec la propriété completed initialisée à false.

Vous pouvez ensuite utiliser cette fonction pour créer de nouvelles tâches dans votre application.

Projet 2: Développement d'un site web de vente en ligne

Dans ce projet, nous allons développer un site web de vente en ligne en utilisant F#. Ce site permettra aux utilisateurs de parcourir des produits, de les ajouter à leur panier et de passer des commandes.

Voici un exemple de code pour la récupération des produits :

type Product = {
    id: int
    name: string
    price: decimal
}

let getProducts () =
    [
        { id = 1; name = "Product 1"; price = 10.99M }
        { id = 2; name = "Product 2"; price = 19.99M }
        { id = 3; name = "Product 3"; price = 7.99M }
    ]

La fonction getProducts renvoie une liste de produits disponibles dans le site. Chaque produit est représenté par une structure Product avec un identifiant, un nom et un prix.

Vous pouvez utiliser cette fonction pour afficher les produits sur votre site et permettre aux utilisateurs de les ajouter à leur panier.

Projet 3: Création d'un jeu en F#

Dans ce projet, nous allons créer un jeu en utilisant F#. Ce jeu sera un jeu de devinettes où les joueurs doivent deviner un nombre aléatoire.

Voici un exemple de code pour la génération d'un nombre aléatoire :

open System

let generateRandomNumber () =
    let random = Random()
    random.Next(1, 101)

La fonction generateRandomNumber utilise la classe Random du namespace System pour générer un nombre aléatoire entre 1 et 100.

Vous pouvez utiliser cette fonction pour générer un nombre aléatoire au début du jeu et demander aux joueurs de deviner ce nombre.
