Section 2: Les bases de F#

Syntaxe de base

La syntaxe de base de F# suit les conventions des langages de programmation fonctionnels. Voici un exemple de code F# utilisant la syntaxe de base : 

let message = "Hello, World!"
printfn "%s" message

Types de données

F# prend en charge différents types de données, tels que les entiers, les nombres à virgule flottante, les chaînes de caractères, les booléens, etc. Voici un exemple montrant comment déclarer et utiliser différents types de données en F# : 

let age : int = 25
let pi : float = 3.14159
let name : string = "John Doe"
let isStudent : bool = true

Variables et constantes

En F#, vous pouvez déclarer des variables et des constantes. Les variables peuvent être modifiées, tandis que les constantes sont immuables. Voici un exemple montrant comment déclarer et utiliser des variables et des constantes en F# : 

let mutable count = 0
count <- count + 1

let pi = 3.14159
// pi <- 3.14 // Erreur : les constantes ne peuvent pas être modifiées

Opérateurs

F# prend en charge différents opérateurs pour effectuer des opérations mathématiques, logiques et de comparaison. Voici quelques exemples d'opérateurs en F# : 

let sum = 5 + 3
let isGreater = 10 > 5
let result = (3 * 2) + 4 / 2

Structures de contrôle

Les structures de contrôle en F# permettent de contrôler le flux d'exécution du programme. Voici quelques exemples de structures de contrôle en F# : 

if condition then
    printfn "Condition is true"
else
    printfn "Condition is false"

for i in 1..5 do
    printfn "%d" i

while condition do
    printfn "Looping"

Ceci conclut la section sur les bases de F#. Vous avez maintenant une compréhension de base de la syntaxe, des types de données, des variables et constantes, des opérateurs et des structures de contrôle en F#. Continuez à pratiquer et à explorer pour approfondir vos connaissances.
