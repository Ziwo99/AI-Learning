Section 2: Les bases de F#

Syntaxe de base de F#

La syntaxe de base de F# est similaire à celle des autres langages de programmation fonctionnelle. Voici un exemple de code F# simple :

let message = "Bonjour, monde!"
printfn "%s" message

Dans cet exemple, nous déclarons une variable "message" qui contient la chaîne de caractères "Bonjour, monde!". Ensuite, nous utilisons la fonction printfn pour afficher le contenu de la variable.

Types de données en F#

F# est un langage fortement typé, ce qui signifie que chaque valeur a un type spécifique. Voici quelques types de données couramment utilisés en F# :

  • int : représente les nombres entiers
  • float : représente les nombres à virgule flottante
  • bool : représente les valeurs booléennes (true ou false)
  • string : représente les chaînes de caractères
  • list : représente une liste d'éléments
  • tuple : représente un ensemble ordonné d'éléments de différents types

Voici un exemple de déclaration de variables avec différents types :

let age : int = 25
let pi : float = 3.14159
let isTrue : bool = true
let name : string = "John Doe"
let numbers : int list = [1; 2; 3; 4; 5]
let person : string * int = ("John Doe", 25)

Variables et constantes

En F#, vous pouvez déclarer des variables et des constantes. Les variables peuvent être modifiées après leur déclaration, tandis que les constantes ne peuvent pas être modifiées une fois qu'elles ont été assignées.

Voici un exemple de déclaration de variable et de constante :

let mutable x = 10 // variable mutable
let y = 5 // constante

x <- x + y // modification de la variable mutable
y <- y + 1 // erreur : impossible de modifier une constante

Opérateurs en F#

F# prend en charge les opérateurs couramment utilisés, tels que les opérateurs arithmétiques (+, -, *, /), les opérateurs de comparaison (>, <, =, <>), les opérateurs logiques (&&, ||, !) et bien d'autres.

Voici un exemple d'utilisation d'opérateurs en F# :

let x = 10
let y = 5

let sum = x + y
let isGreater = x > y
let isTrue = true && false

Structures de contrôle (boucles, conditions)

En F#, vous pouvez utiliser des structures de contrôle telles que les boucles et les conditions pour contrôler le flux d'exécution de votre programme.

Voici un exemple d'utilisation d'une boucle for et d'une condition if en F# :

for i in 1..10 do
    if i % 2 = 0 then
        printfn "%d est pair" i
    else
        printfn "%d est impair" i

Fonctions en F#

Les fonctions sont un élément clé de la programmation en F#. Vous pouvez définir vos propres fonctions et les appeler dans votre code.

Voici un exemple de déclaration et d'appel de fonction en F# :

let add x y = x + y

let result = add 5 10
printfn "Le résultat est %d" result

Dans cet exemple, nous définissons une fonction "add" qui prend deux paramètres (x et y) et renvoie leur somme. Ensuite, nous appelons cette fonction avec les arguments 5 et 10, et affichons le résultat.
