Section 5: Programmation orientée objet en F#

Introduction à la programmation orientée objet

La programmation orientée objet (POO) est un paradigme de programmation qui permet de structurer le code en utilisant des objets qui interagissent les uns avec les autres. En F#, la POO est prise en charge grâce à des concepts tels que les classes, l'héritage, le polymorphisme, les interfaces et les modules.

Définition de classes

En F#, une classe est une structure de données qui définit les propriétés et les comportements d'un objet. Les classes sont déclarées à l'aide du mot-clé type suivi du nom de la classe et des membres de la classe.

type Person = 
    class
        member val Name = "" with get, set
        member val Age = 0 with get, set
        member this.SayHello() = printfn "Hello, my name is %s!" this.Name
    end

Héritage et polymorphisme

En F#, l'héritage permet à une classe d'hériter des membres d'une autre classe. Le polymorphisme permet à une classe dérivée de substituer ou d'étendre les membres d'une classe de base. Cela permet de réutiliser du code et de créer des hiérarchies de classes.

type Animal() =
    abstract member MakeSound : unit -> unit

type Dog() =
    inherit Animal()
    override this.MakeSound() = printfn "Woof!"

type Cat() =
    inherit Animal()
    override this.MakeSound() = printfn "Meow!"

let dog = Dog()
let cat = Cat()

dog.MakeSound()
cat.MakeSound()

Interfaces

Les interfaces en F# permettent de définir un contrat que les classes doivent respecter. Une classe peut implémenter plusieurs interfaces, ce qui lui permet de fournir différentes fonctionnalités.

type IShape =
    abstract member GetArea : unit -> float

type Rectangle(width: float, height: float) =
    interface IShape with
        member this.GetArea() = width * height

type Circle(radius: float) =
    interface IShape with
        member this.GetArea() = System.Math.PI * radius * radius

let rectangle = Rectangle(5.0, 3.0)
let circle = Circle(2.0)

printfn "Rectangle area: %f" (rectangle.GetArea())
printfn "Circle area: %f" (circle.GetArea())

Modules

Les modules en F# permettent d'organiser et de regrouper des types, des fonctions et des valeurs connexes. Ils facilitent la gestion de la complexité et la réutilisation du code.

module Math =
    let add x y = x + y
    let subtract x y = x - y

let result1 = Math.add 5 3
let result2 = Math.subtract 10 7

printfn "Addition: %d" result1
printfn "Subtraction: %d" result2
