Section 5: Manipulation avancée des données en F#

Accès aux bases de données

F# offre une prise en charge intégrée pour l'accès aux bases de données relationnelles. Vous pouvez utiliser des bibliothèques telles que FSharp.Data.SqlClient ou FSharp.Data.TypeProviders pour interagir avec des bases de données SQL Server. Ces bibliothèques fournissent des types de données statiquement typés qui reflètent la structure de la base de données, ce qui facilite la manipulation des données.

Voici un exemple d'utilisation de FSharp.Data.SqlClient pour exécuter une requête SQL et récupérer les résultats : 

open FSharp.Data.SqlClient

let connectionString = "Data Source=(localdb)\\MSSQLLocalDB;Initial Catalog=MyDatabase;Integrated Security=True"
let query = "SELECT * FROM Customers"

use connection = new SqlConnection(connectionString)
let results = connection.ExecuteQuery(query)

for row in results do
    printfn "%s %s" row.FirstName row.LastName

Utilisation de LINQ en F#

F# prend en charge l'utilisation de LINQ (Language Integrated Query) pour interroger et manipuler des collections de données. Vous pouvez utiliser les opérateurs LINQ tels que where, select, groupBy, etc. pour effectuer des opérations de filtrage, de tri et de regroupement sur les données.

Voici un exemple d'utilisation de LINQ pour filtrer une liste de personnes en fonction de leur âge : 

let people = [
    { Name = "Alice"; Age = 25 }
    { Name = "Bob"; Age = 30 }
    { Name = "Charlie"; Age = 35 }
]

let filteredPeople = people |> List.filter (fun person -> person.Age > 30)

for person in filteredPeople do
    printfn "%s" person.Name

Traitement de fichiers et de flux de données

En F#, vous pouvez facilement traiter des fichiers et des flux de données à l'aide des fonctions intégrées de manipulation de fichiers. Vous pouvez lire et écrire des fichiers texte, CSV, JSON, XML, etc. en utilisant des bibliothèques telles que FSharp.Data ou FSharp.Data.Json.

Voici un exemple de lecture d'un fichier CSV et d'affichage de son contenu : 

open FSharp.Data

type Person = CsvProvider<"path/to/file.csv">

let people = Person.Load("path/to/file.csv").Rows

for person in people do
    printfn "%s %s" person.FirstName person.LastName

Manipulation de données XML et JSON

F# offre une prise en charge native pour la manipulation de données XML et JSON. Vous pouvez utiliser des bibliothèques telles que FSharp.Data ou FSharp.Data.Json pour lire, écrire et manipuler des données XML et JSON.

Voici un exemple de lecture d'un fichier XML et d'affichage de son contenu : 

open FSharp.Data

type Person = XmlProvider<"path/to/file.xml">

let people = Person.Load("path/to/file.xml").Items

for person in people do
    printfn "%s %s" person.FirstName person.LastName
