Section 2 : Structure d'une page HTML

Section 2 : Structure d'une page HTML

La balise <html>

La balise <html> est la balise racine de tout document HTML. Elle enveloppe l'ensemble du contenu de la page.

<html>
    <!-- Le contenu de la page -->
</html>

La balise <head>

La balise <head> est utilisée pour définir les informations d'en-tête de la page HTML. Elle contient des éléments tels que le titre de la page, les liens vers des fichiers CSS ou JavaScript, les méta-informations, etc.

<head>
    <title>Titre de la page</title>
    <link rel="stylesheet" href="styles.css">
    <script src="script.js"></script>
</head>

La balise <body>

La balise <body> contient tout le contenu visible de la page HTML, tel que le texte, les images, les liens, etc. C'est à l'intérieur de cette balise que vous allez structurer le contenu de votre page.

<body>
    <h1>Titre principal</h1>
    <p>Ceci est un paragraphe.</p>
    <img src="image.jpg" alt="Image">
    <a href="https://www.example.com">Lien vers example.com</a>
</body>

Les balises de titre

Les balises de titre sont utilisées pour structurer le contenu de la page en différents niveaux de titres. Elles vont de <h1> (le titre principal) à <h6> (le titre le plus bas).

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-sous-titre</h3>

Les balises de paragraphe

Les balises de paragraphe sont utilisées pour structurer le texte en paragraphes distincts.

<p>Ceci est un paragraphe.</p>
<p>Ceci est un autre paragraphe.</p>

Les balises de liste

Les balises de liste sont utilisées pour créer des listes ordonnées ou non ordonnées.

Liste ordonnée :

<ol>
    <li>Élément 1</li>
    <li>Élément 2</li>
    <li>Élément 3</li>
</ol>

Liste non ordonnée :

<ul>
    <li>Élément 1</li>
    <li>Élément 2</li>
    <li>Élément 3</li>
</ul>
Ceci conclut la formation sur la structure d'une page HTML. N'hésitez pas à pratiquer et à explorer davantage pour approfondir vos connaissances.
