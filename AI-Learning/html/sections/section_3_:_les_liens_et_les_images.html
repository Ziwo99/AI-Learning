Section 3 : Les liens et les images

Section 3: Les liens et les images

Les balises de lien

Les balises de lien permettent de créer des liens hypertexte vers d'autres pages web ou des ressources externes. La balise <a> est utilisée pour créer un lien. Voici un exemple de code HTML pour créer un lien :

<a href="https://www.example.com">Cliquez ici pour visiter notre site web</a>

Dans cet exemple, le texte "Cliquez ici pour visiter notre site web" sera affiché comme un lien cliquable. Lorsque l'utilisateur clique sur ce lien, il sera redirigé vers la page web spécifiée dans l'attribut href.

Les balises d'image

Les balises d'image permettent d'insérer des images dans une page web. La balise <img> est utilisée pour insérer une image. Voici un exemple de code HTML pour insérer une image :

<img src="image.jpg" alt="Description de l'image">

Dans cet exemple, l'image "image.jpg" sera affichée sur la page web. L'attribut alt est utilisé pour fournir une description textuelle de l'image, qui sera affichée si l'image ne peut pas être chargée ou si l'utilisateur utilise un lecteur d'écran.

Les attributs des liens et des images

Les liens et les images peuvent avoir des attributs supplémentaires pour spécifier leur comportement ou leur apparence. Voici quelques attributs couramment utilisés :

  • target : spécifie où ouvrir le lien (par exemple, dans une nouvelle fenêtre ou un nouvel onglet).
  • title : affiche une info-bulle lorsque l'utilisateur survole le lien ou l'image.
  • width et height : spécifient la largeur et la hauteur de l'image.

Voici un exemple de code HTML utilisant ces attributs :

<a href="https://www.example.com" target="_blank" title="Visitez notre site web">Cliquez ici</a>

<img src="image.jpg" alt="Description de l'image" width="500" height="300">

Dans cet exemple, le lien s'ouvrira dans une nouvelle fenêtre ou un nouvel onglet, une info-bulle avec le texte "Visitez notre site web" sera affichée lorsque l'utilisateur survole le lien, et l'image aura une largeur de 500 pixels et une hauteur de 300 pixels.
