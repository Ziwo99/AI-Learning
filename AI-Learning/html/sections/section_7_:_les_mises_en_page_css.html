Section 7 : Les mises en page CSS

Les boîtes CSS

Les boîtes CSS sont des éléments fondamentaux pour la mise en page d'une page web. Elles permettent de structurer et de positionner les différents éléments de contenu. Une boîte CSS est composée de quatre parties principales : le contenu, le padding, la bordure et la marge.

Le contenu représente le texte ou les éléments visuels à l'intérieur de la boîte. Le padding est l'espace entre le contenu et la bordure. La bordure est une ligne qui entoure la boîte. Enfin, la marge est l'espace entre la boîte et les autres éléments de la page.

Voici un exemple de code HTML utilisant les balises appropriées pour créer une boîte CSS :

<div class="boite">
    <p>Contenu de la boîte</p>
</div>

Les modèles de boîtes CSS

Les modèles de boîtes CSS permettent de contrôler la taille et le positionnement des boîtes. Il existe deux modèles de boîtes principaux : le modèle de boîte en bloc et le modèle de boîte en ligne.

Le modèle de boîte en bloc est utilisé par défaut pour les éléments de type "block". Il occupe toute la largeur disponible et crée un nouveau bloc en dessous de l'élément précédent.

Le modèle de boîte en ligne est utilisé par défaut pour les éléments de type "inline". Il occupe uniquement l'espace nécessaire pour afficher son contenu et ne crée pas de nouveau bloc en dessous de l'élément précédent.

Voici un exemple de code HTML utilisant les balises appropriées pour créer une boîte CSS avec le modèle de boîte en bloc :

<div class="boite-bloc">
    <p>Contenu de la boîte en bloc</p>
</div>

Et voici un exemple de code HTML utilisant les balises appropriées pour créer une boîte CSS avec le modèle de boîte en ligne :

<span class="boite-ligne">Contenu de la boîte en ligne</span>

Les propriétés de positionnement CSS

Les propriétés de positionnement CSS permettent de contrôler la position des éléments sur la page. Il existe plusieurs propriétés de positionnement, notamment :

  • position: static; : positionnement par défaut, l'élément suit le flux normal du document.
  • position: relative; : positionnement relatif à sa position normale. Les propriétés top, right, bottom et left peuvent être utilisées pour ajuster la position.
  • position: absolute; : positionnement absolu par rapport à son premier parent positionné. Les propriétés top, right, bottom et left peuvent être utilisées pour ajuster la position.
  • position: fixed; : positionnement fixe par rapport à la fenêtre du navigateur. L'élément reste à la même position même lorsque la page est défilée.

Voici un exemple de code HTML utilisant les balises appropriées pour créer un élément avec une position relative :

<div class="element-relatif">
    <p>Contenu de l'élément relatif</p>
</div>

Les propriétés de mise en page CSS

Les propriétés de mise en page CSS permettent de contrôler la disposition des éléments sur la page. Il existe plusieurs propriétés de mise en page, notamment :

  • display: block; : l'élément est affiché comme un bloc, occupant toute la largeur disponible.
  • display: inline; : l'élément est affiché en ligne, occupant uniquement l'espace nécessaire pour afficher son contenu.
  • display: inline-block; : l'élément est affiché en ligne, mais peut avoir une largeur et une hauteur définies.
  • float: left; : l'élément est déplacé vers la gauche et les autres éléments s'alignent autour de lui.
  • clear: both; : empêche les éléments suivants de flotter à gauche ou à droite.

Voici un exemple de code HTML utilisant les balises appropriées pour créer un élément avec une mise en page en bloc :

<div class="element-bloc">
    <p>Contenu de l'élément en bloc</p>
</div>
