Section 6 : Les styles CSS

Introduction aux styles CSS

Les styles CSS (Cascading Style Sheets) sont utilisés pour définir l'apparence et la mise en page des éléments HTML. Ils permettent de contrôler les couleurs, les polices, les marges, les bordures et bien plus encore. Les styles CSS peuvent être appliqués directement à un élément HTML ou être définis dans une feuille de style externe.

Les sélecteurs CSS

Les sélecteurs CSS sont utilisés pour cibler spécifiquement les éléments HTML auxquels vous souhaitez appliquer des styles. Il existe différents types de sélecteurs CSS, tels que les sélecteurs d'éléments, les sélecteurs de classe, les sélecteurs d'ID, les sélecteurs d'attributs, etc. Les sélecteurs CSS permettent de sélectionner un ou plusieurs éléments HTML en fonction de critères spécifiques.

p {
    color: blue;
}

#myElement {
    font-size: 20px;
}

.myClass {
    background-color: yellow;
}

Les propriétés CSS

Les propriétés CSS sont utilisées pour définir les différentes caractéristiques d'un élément HTML. Elles permettent de contrôler des aspects tels que la couleur du texte, la taille de la police, la marge, la bordure, etc. Il existe de nombreuses propriétés CSS disponibles, chacune ayant un effet spécifique sur l'apparence de l'élément.

p {
    color: blue;
    font-size: 16px;
    margin-top: 10px;
    border: 1px solid black;
}

Les classes et les ID CSS

Les classes et les ID CSS sont utilisés pour appliquer des styles spécifiques à des éléments HTML. Les classes permettent de regrouper plusieurs éléments et de leur appliquer un style commun, tandis que les ID sont utilisés pour cibler un élément spécifique. Les classes et les ID peuvent être utilisés en conjonction avec des sélecteurs CSS pour appliquer des styles précis.

.myClass {
    color: red;
}

#myElement {
    font-size: 20px;
}
Ceci conclut la section sur les styles CSS. Vous avez maintenant une compréhension de base des styles CSS et de leur utilisation pour personnaliser l'apparence des éléments HTML.
