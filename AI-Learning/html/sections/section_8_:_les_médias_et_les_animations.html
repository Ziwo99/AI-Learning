Section 8 : Les médias et les animations

Section 8 : Les médias et les animations

Les balises audio et vidéo

Les balises audio et vidéo permettent d'intégrer des fichiers audio et vidéo dans une page HTML. Voici comment les utiliser :

Pour intégrer un fichier audio, utilisez la balise <audio> :

<audio src="chemin_vers_le_fichier_audio.mp3">
    Votre navigateur ne supporte pas l'élément audio.
</audio>

Pour intégrer une vidéo, utilisez la balise <video> :

<video src="chemin_vers_le_fichier_video.mp4" controls>
    Votre navigateur ne supporte pas l'élément vidéo.
</video>

Assurez-vous de spécifier le chemin vers le fichier audio ou vidéo dans l'attribut src. Vous pouvez également ajouter l'attribut controls pour afficher les contrôles de lecture.

Les animations CSS

Les animations CSS permettent de créer des effets visuels dynamiques sur une page HTML. Voici comment les utiliser :

Pour créer une animation CSS, vous devez définir une règle d'animation dans votre fichier CSS :

@keyframes nom_de_l_animation {
    0% { propriété: valeur_initiale; }
    100% { propriété: valeur_finale; }
}

Ensuite, vous pouvez appliquer cette animation à un élément HTML en utilisant la propriété animation :

element {
    animation: nom_de_l_animation durée timing-function nombre_d_iterations direction;
}

Assurez-vous de remplacer nom_de_l_animation par le nom que vous avez donné à votre animation, et de spécifier la durée, la fonction de temporisation, le nombre d'itérations et la direction souhaités.

Les transitions CSS

Les transitions CSS permettent de créer des effets de transition fluides entre les états d'un élément HTML. Voici comment les utiliser :

Pour créer une transition CSS, vous devez définir une règle de transition dans votre fichier CSS :

element {
    transition: propriété durée timing-function;
}

Ensuite, vous pouvez spécifier les états de transition en utilisant les pseudo-classes :hover, :active, :focus, etc. :

element:hover {
    propriété: valeur;
}

Assurez-vous de remplacer propriété par la propriété CSS que vous souhaitez animer, et de spécifier la durée et la fonction de temporisation souhaitées.

Les transformations CSS

Les transformations CSS permettent de modifier la position, la taille, la rotation et l'échelle d'un élément HTML. Voici comment les utiliser :

Pour appliquer une transformation CSS à un élément, utilisez la propriété transform :

element {
    transform: valeur_transformation;
}

Voici quelques exemples de transformations couramment utilisées :

element {
    transform: translate(x, y); /* Déplace l'élément selon les coordonnées x et y */
    transform: rotate(angle); /* Fait tourner l'élément selon l'angle spécifié */
    transform: scale(x, y); /* Redimensionne l'élément selon les facteurs x et y */
}

Assurez-vous de remplacer valeur_transformation par la transformation que vous souhaitez appliquer, en utilisant les valeurs appropriées.
