Réalisation d'un site web complet en HTML

Dans cette sous-section, nous allons apprendre à créer un site web complet en utilisant uniquement HTML. Nous allons commencer par la structure de base d'une page HTML, puis nous ajouterons des éléments tels que des titres, des paragraphes, des images, des liens et des listes.

Voici un exemple de structure de base d'une page HTML :

<!DOCTYPE html>
<html>
<head>
    <title>Mon premier site web</title>
</head>
<body>
    <h1>Bienvenue sur mon site web</h1>
    <p>Ceci est un paragraphe de texte.</p>
    <img src="image.jpg" alt="Image">
    <a href="https://www.example.com">Ceci est un lien</a>
    <ul>
        <li>Premier élément de liste</li>
        <li>Deuxième élément de liste</li>
        <li>Troisième élément de liste</li>
    </ul>
</body>
</html>

Vous pouvez personnaliser cette structure de base en ajoutant vos propres titres, paragraphes, images, liens et listes pour créer votre site web.

Voici un exemple de code HTML pour créer une liste numérotée :

<ol>
    <li>Premier élément de liste</li>
    <li>Deuxième élément de liste</li>
    <li>Troisième élément de liste</li>
</ol>

Et voici un exemple de code HTML pour créer une liste à puces :

<ul>
    <li>Premier élément de liste</li>
    <li>Deuxième élément de liste</li>
    <li>Troisième élément de liste</li>
</ul>

En utilisant ces balises et en personnalisant le contenu, vous pouvez créer un site web complet en HTML.
