Section 1: Introduction à HTML

Qu'est-ce que HTML ?

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises et des éléments.

Les balises HTML sont des éléments qui définissent la structure et le sens du contenu. Elles sont entourées de chevrons angulaires (< et >) et sont souvent utilisées par paires, avec une balise d'ouverture et une balise de fermeture.

Voici un exemple de balise HTML :

<p>Ceci est un paragraphe.</p>

Les bases de la structure HTML

Une page HTML est composée de plusieurs éléments de base :

  • Doctype : La déclaration doctype indique au navigateur quel type de document HTML est utilisé.
  • Balise <html> : La balise <html> est la racine de tout document HTML.
  • Balise <head> : La balise <head> contient des informations sur la page, telles que le titre, les métadonnées et les liens vers des fichiers externes.
  • Balise <body> : La balise <body> contient le contenu visible de la page, tel que le texte, les images et les liens.

Voici un exemple de structure HTML de base :

<!DOCTYPE html>
<html>
  <head>
    <title>Titre de la page</title>
  </head>
  <body>
    <h1>Titre principal</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

Les balises HTML essentielles

Il existe de nombreuses balises HTML qui permettent de structurer et de formater le contenu d'une page. Voici quelques-unes des balises les plus couramment utilisées :

  • <h1> à <h6> : Les balises de titre sont utilisées pour définir les niveaux de titre d'une page.
  • <p> : La balise de paragraphe est utilisée pour définir un paragraphe de texte.
  • <a> : La balise de lien est utilisée pour créer des liens hypertexte vers d'autres pages ou ressources.
  • <img> : La balise d'image est utilisée pour afficher des images sur une page.
  • <ul> et <li> : Les balises de liste non ordonnée et de liste ordonnée sont utilisées pour créer des listes à puces ou numérotées.

Voici un exemple d'utilisation de certaines de ces balises :

<h1>Titre principal</h1>
<p>Ceci est un paragraphe.</p>
<a href="https://www.example.com">Lien vers Example.com</a>
<img src="image.jpg" alt="Image">
<ul>
  <li>Élément de liste 1</li>
  <li>Élément de liste 2</li>
</ul>

Les attributs HTML

Les attributs HTML sont utilisés pour fournir des informations supplémentaires sur les balises. Ils sont spécifiés à l'intérieur de la balise d'ouverture et peuvent modifier le comportement ou l'apparence de l'élément.

Voici quelques exemples d'attributs couramment utilisés :

  • href : L'attribut href est utilisé avec la balise <a> pour spécifier l'URL de destination du lien.
  • src : L'attribut src est utilisé avec la balise <img> pour spécifier l'emplacement de l'image.
  • alt : L'attribut alt est utilisé avec la balise <img> pour fournir un texte alternatif qui sera affiché si l'image ne peut pas être chargée.

Voici un exemple d'utilisation d'attributs :

<a href="https://www.example.com" target="_blank">Lien vers Example.com</a>
<img src="image.jpg" alt="Image d'exemple">

Ceci conclut la section d'introduction à HTML. Vous avez maintenant une compréhension de base de la structure HTML, des balises essentielles et des attributs couramment utilisés.
