Section 4 : Les tableaux

La structure d'un tableau

Un tableau est un élément HTML qui permet d'organiser des données sous forme de lignes et de colonnes. La structure d'un tableau se compose des balises <table> pour définir le tableau lui-même, <tr> pour définir une ligne et <td> pour définir une cellule.

Voici un exemple de structure de tableau :

<table>
  <tr>
    <td>Cellule 1</td>
    <td>Cellule 2</td>
  </tr>
  <tr>
    <td>Cellule 3</td>
    <td>Cellule 4</td>
  </tr>
</table>

Les balises de tableau

Les balises de tableau permettent de définir la structure et le contenu d'un tableau.

Voici les principales balises de tableau :

  • <table> : Définit le tableau
  • <tr> : Définit une ligne
  • <td> : Définit une cellule
  • <th> : Définit une cellule d'en-tête
  • <caption> : Définit le titre du tableau

Voici un exemple d'utilisation des balises de tableau :

<table>
  <caption>Titre du tableau</caption>
  <tr>
    <th>En-tête 1</th>
    <th>En-tête 2</th>
  </tr>
  <tr>
    <td>Cellule 1</td>
    <td>Cellule 2</td>
  </tr>
</table>

Les attributs des tableaux

Les attributs permettent de spécifier des informations supplémentaires sur les tableaux.

Voici quelques attributs couramment utilisés pour les tableaux :

  • border : Définit l'épaisseur de la bordure du tableau
  • width : Définit la largeur du tableau
  • height : Définit la hauteur du tableau
  • cellspacing : Définit l'espace entre les cellules
  • cellpadding : Définit l'espace à l'intérieur des cellules

Voici un exemple d'utilisation des attributs de tableau :

<table border="1" width="500" cellspacing="10" cellpadding="5">
  <tr>
    <td>Cellule 1</td>
    <td>Cellule 2</td>
  </tr>
  <tr>
    <td>Cellule 3</td>
    <td>Cellule 4</td>
  </tr>
</table>
