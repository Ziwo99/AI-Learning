Section 5 : Les formulaires

La structure d'un formulaire

Un formulaire HTML est utilisé pour collecter des informations auprès des utilisateurs. Il est composé de plusieurs éléments, notamment les balises <form>, <input>, <label> et <button>.

Voici un exemple de structure de base d'un formulaire :

<form action="traitement.php" method="post">
    <label for="nom">Nom :</label>
    <input type="text" id="nom" name="nom"><br>

    <label for="email">Email :</label>
    <input type="email" id="email" name="email"><br>

    <button type="submit">Envoyer</button>
</form>

Les balises de formulaire

Les balises de formulaire sont utilisées pour structurer et organiser les différents éléments d'un formulaire. Les balises les plus couramment utilisées sont :

  • <form> : Définit un formulaire
  • <input> : Crée un champ de saisie
  • <label> : Associe un libellé à un champ de saisie
  • <button> : Crée un bouton
  • <select> : Crée une liste déroulante
  • <textarea> : Crée une zone de texte multiligne

Les types de champs de formulaire

Les types de champs de formulaire définissent le type de données que l'utilisateur peut saisir. Voici quelques exemples de types de champs couramment utilisés :

  • Texte : <input type="text">
  • Email : <input type="email">
  • Mot de passe : <input type="password">
  • Nombre : <input type="number">
  • Case à cocher : <input type="checkbox">
  • Bouton radio : <input type="radio">

Les attributs des champs de formulaire

Les attributs des champs de formulaire permettent de spécifier des informations supplémentaires sur les champs. Voici quelques exemples d'attributs couramment utilisés :

  • id : Identifie de manière unique un champ de saisie
  • name : Spécifie le nom du champ de saisie
  • value : Définit la valeur par défaut d'un champ de saisie
  • required : Indique que le champ de saisie est obligatoire
  • disabled : Désactive un champ de saisie

Il existe de nombreux autres attributs que vous pouvez utiliser pour personnaliser vos champs de formulaire en fonction de vos besoins spécifiques.
