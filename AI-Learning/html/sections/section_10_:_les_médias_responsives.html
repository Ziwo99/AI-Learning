Section 10 : Les médias responsives

Section 10 : Les médias responsives

Les médias queries

Les médias queries sont des outils puissants pour rendre les sites web réactifs et adaptables à différents appareils et tailles d'écran. Ils permettent de définir des règles CSS spécifiques qui s'appliquent uniquement lorsque certaines conditions sont remplies.

Voici un exemple de média query qui applique des styles CSS différents lorsque la largeur de l'écran est inférieure à 600 pixels :

@media screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Dans cet exemple, la règle CSS à l'intérieur du média query sera appliquée uniquement lorsque la largeur de l'écran est inférieure à 600 pixels. Cela permet de modifier le style de la page pour une meilleure expérience utilisateur sur les petits écrans.

Les images responsives

Les images responsives sont des images qui s'adaptent automatiquement à la taille de l'écran sur lequel elles sont affichées. Cela permet d'éviter les problèmes de mise en page et de chargement excessif des images sur les appareils mobiles.

Voici un exemple de code HTML pour une image responsive :

<img src="image.jpg" alt="Une image" class="img-responsive">

Dans cet exemple, la classe "img-responsive" est utilisée pour appliquer des styles CSS qui permettent à l'image de s'adapter à la taille de son conteneur.

Les mises en page responsives

Les mises en page responsives sont des mises en page qui s'adaptent automatiquement à la taille de l'écran sur lequel elles sont affichées. Cela permet d'optimiser l'affichage du contenu et d'améliorer l'expérience utilisateur sur tous les appareils.

Voici un exemple de code HTML pour une mise en page responsive :

<div class="container">
    <div class="row">
        <div class="col-md-6">
            <p>Contenu de la colonne 1</p>
        </div>
        <div class="col-md-6">
            <p>Contenu de la colonne 2</p>
        </div>
    </div>
</div>

Dans cet exemple, la classe "container" est utilisée pour créer un conteneur qui s'adapte à la taille de l'écran. Les classes "row" et "col-md-6" sont utilisées pour créer une mise en page en colonnes qui s'adapte également à la taille de l'écran.

En utilisant les médias queries, les images responsives et les mises en page responsives, vous pouvez créer des sites web qui offrent une expérience utilisateur optimale sur tous les appareils et tailles d'écran.
