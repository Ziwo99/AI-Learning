Les pointeurs de fonctions

Les pointeurs de fonctions en C permettent de stocker l'adresse d'une fonction dans une variable. Cela offre une grande flexibilité et permet d'appeler des fonctions de manière dynamique. Voici un exemple d'utilisation des pointeurs de fonctions : 


#include 

void sayHello() {
    printf("Hello World!\n");
}

void sayGoodbye() {
    printf("Goodbye World!\n");
}

int main() {
    void (*functionPtr)(); // Déclaration d'un pointeur de fonction

    functionPtr = sayHello; // Assignation de l'adresse de la fonction sayHello
    functionPtr(); // Appel de la fonction sayHello via le pointeur de fonction

    functionPtr = sayGoodbye; // Assignation de l'adresse de la fonction sayGoodbye
    functionPtr(); // Appel de la fonction sayGoodbye via le pointeur de fonction

    return 0;
}

Les allocations dynamiques de mémoire

Les allocations dynamiques de mémoire en C permettent de réserver de l'espace mémoire pendant l'exécution du programme. Cela est utile lorsque la taille des données à stocker est inconnue à la compilation ou lorsque la taille peut varier. Voici un exemple d'utilisation des allocations dynamiques de mémoire : 


#include 
#include 

int main() {
    int size;
    int *dynamicArray;

    printf("Enter the size of the array: ");
    scanf("%d", &size);

    dynamicArray = (int*)malloc(size * sizeof(int)); // Allocation dynamique de mémoire

    if (dynamicArray == NULL) {
        printf("Memory allocation failed.\n");
        return 1;
    }

    for (int i = 0; i < size; i++) {
        dynamicArray[i] = i; // Assignation de valeurs à l'array dynamique
    }

    for (int i = 0; i < size; i++) {
        printf("%d ", dynamicArray[i]); // Affichage des valeurs de l'array dynamique
    }

    free(dynamicArray); // Libération de la mémoire allouée

    return 0;
}

Les bibliothèques et les modules en C

Les bibliothèques et les modules en C sont des fichiers contenant des fonctions et des définitions qui peuvent être utilisées dans d'autres programmes. Ils permettent de réutiliser du code et de faciliter le développement. Voici un exemple d'utilisation d'une bibliothèque en C : 


#include 
#include "myLibrary.h" // Inclusion de la bibliothèque personnalisée

int main() {
    int result = add(2, 3); // Appel de la fonction add() de la bibliothèque

    printf("The result is: %d\n", result);

    return 0;
}

Les threads et la programmation concurrente

Les threads en C permettent d'exécuter plusieurs tâches en parallèle, ce qui améliore les performances et la réactivité des programmes. La programmation concurrente consiste à gérer l'exécution de plusieurs threads de manière synchronisée. Voici un exemple d'utilisation des threads en C : 


#include 
#include 

void* printMessage(void* arg) {
    char* message = (char*)arg;
    printf("%s\n", message);
    return NULL;
}

int main() {
    pthread_t thread1, thread2;

    char* message1 = "Hello from Thread 1!";
    char* message2 = "Hello from Thread 2!";

    pthread_create(&thread1, NULL, printMessage, (void*)message1); // Création du thread 1
    pthread_create(&thread2, NULL, printMessage, (void*)message2); // Création du thread 2

    pthread_join(thread1, NULL); // Attente de la fin du thread 1
    pthread_join(thread2, NULL); // Attente de la fin du thread 2

    return 0;
}
