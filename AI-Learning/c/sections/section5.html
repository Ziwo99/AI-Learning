Section 5 : Les structures de données

Définition et utilisation des structures

Les structures de données sont des types de données personnalisés qui permettent de regrouper plusieurs variables de types différents sous un même nom. Elles sont utilisées pour représenter des objets complexes et peuvent être définies en utilisant le mot-clé struct.

Voici un exemple de définition d'une structure en C :

struct Person {
    char name[50];
    int age;
    float height;
};

Une fois la structure définie, nous pouvons créer des variables de ce type :

struct Person person1;
struct Person person2;

Nous pouvons ensuite accéder aux membres de la structure en utilisant l'opérateur . :

person1.age = 25;
person2.age = 30;

Les unions et les énumérations

Les unions sont des types de données qui permettent de stocker différentes variables de types différents dans la même zone mémoire. Seule une des variables peut être utilisée à la fois. Les unions sont définies en utilisant le mot-clé union.

Voici un exemple de définition d'une union en C :

union Data {
    int i;
    float f;
    char str[20];
};

Les énumérations sont des types de données qui permettent de définir un ensemble de valeurs constantes. Elles sont définies en utilisant le mot-clé enum.

Voici un exemple de définition d'une énumération en C :

enum Color {
    RED,
    GREEN,
    BLUE
};

Les listes chaînées

Les listes chaînées sont des structures de données linéaires qui permettent de stocker une séquence d'éléments. Chaque élément de la liste est constitué d'une valeur et d'un pointeur vers l'élément suivant. La liste est représentée par un pointeur vers le premier élément.

Voici un exemple de définition d'une liste chaînée en C :

struct Node {
    int data;
    struct Node* next;
};

struct Node* head = NULL;

Pour ajouter un nouvel élément à la liste, nous devons allouer de la mémoire pour le nouvel élément et mettre à jour les pointeurs :

struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
newNode->data = 10;
newNode->next = head;
head = newNode;

Les arbres binaires

Les arbres binaires sont des structures de données hiérarchiques qui permettent de stocker des éléments de manière organisée. Chaque élément de l'arbre est appelé un nœud et peut avoir jusqu'à deux nœuds enfants : un nœud gauche et un nœud droit.

Voici un exemple de définition d'un nœud d'arbre binaire en C :

struct Node {
    int data;
    struct Node* left;
    struct Node* right;
};

struct Node* root = NULL;

Pour insérer un nouvel élément dans l'arbre, nous devons parcourir l'arbre en utilisant des règles spécifiques et trouver l'emplacement approprié pour l'insertion.

Voici un exemple d'insertion d'un nouvel élément dans un arbre binaire :

struct Node* insert(struct Node* node, int data) {
    if (node == NULL) {
        struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
        newNode->data = data;
        newNode->left = NULL;
        newNode->right = NULL;
        return newNode;
    }
    if (data < node->data) {
        node->left = insert(node->left, data);
    } else if (data > node->data) {
        node->right = insert(node->right, data);
    }
    return node;
}

Ces exemples illustrent quelques-unes des structures de données les plus couramment utilisées en C. Elles sont essentielles pour la manipulation et l'organisation des données dans les programmes C.
