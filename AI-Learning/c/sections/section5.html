Préprocesseur et macros

Le préprocesseur en C est une étape de compilation qui se déroule avant la compilation proprement dite. Il permet de manipuler le code source avant qu'il ne soit compilé. Une des fonctionnalités les plus utilisées du préprocesseur est l'utilisation de macros.

Une macro est une séquence de code qui est définie à l'aide de la directive #define. Elle peut être utilisée pour définir des constantes, des fonctions inline ou pour effectuer des opérations de substitution de texte.

Voici un exemple de définition d'une macro pour calculer le carré d'un nombre :

#define SQUARE(x) ((x) * (x))

int main() {
    int a = 5;
    int b = SQUARE(a);
    return 0;
}

Dans cet exemple, la macro SQUARE est définie pour calculer le carré d'un nombre. Lorsque la macro est utilisée dans le code, elle est remplacée par l'expression ((x) * (x)). Ainsi, l'appel SQUARE(a) sera remplacé par ((a) * (a)), ce qui équivaut à 5 * 5.

Gestion de la mémoire et optimisation

La gestion de la mémoire est un aspect important de la programmation en C. En C, la mémoire est gérée manuellement, ce qui signifie que le programmeur est responsable de l'allocation et de la libération de la mémoire.

La gestion de la mémoire peut être réalisée à l'aide des fonctions malloc, calloc, realloc et free. La fonction malloc permet d'allouer de la mémoire dynamiquement, la fonction calloc permet d'allouer de la mémoire et de l'initialiser à zéro, la fonction realloc permet de redimensionner une zone de mémoire déjà allouée et la fonction free permet de libérer la mémoire allouée.

Il est important de libérer la mémoire allouée une fois qu'elle n'est plus nécessaire, afin d'éviter les fuites de mémoire. Une fuite de mémoire se produit lorsque de la mémoire allouée n'est pas libérée, ce qui peut entraîner une utilisation excessive de la mémoire et des problèmes de performance.

En ce qui concerne l'optimisation, il est important d'optimiser le code pour améliorer les performances et réduire la consommation de mémoire. Cela peut être réalisé en utilisant des algorithmes efficaces, en évitant les opérations coûteuses et en optimisant l'utilisation de la mémoire.

Manipulation de chaînes de caractères

La manipulation de chaînes de caractères est une tâche courante en programmation. En C, les chaînes de caractères sont représentées comme des tableaux de caractères terminés par un caractère nul ('\0').

La bibliothèque standard C fournit plusieurs fonctions pour manipuler les chaînes de caractères, telles que strlen, strcpy, strcat, etc.

Voici un exemple d'utilisation de la fonction strlen pour calculer la longueur d'une chaîne de caractères :

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, world!";
    int length = strlen(str);
    printf("Length of the string: %d\n", length);
    return 0;
}

Dans cet exemple, la fonction strlen est utilisée pour calculer la longueur de la chaîne de caractères str. La longueur est ensuite affichée à l'aide de la fonction printf.

Fonctions avancées (pointeurs de fonctions, fonctions récursives)

En C, il est possible de manipuler des pointeurs vers des fonctions. Un pointeur de fonction est une variable qui peut contenir l'adresse d'une fonction. Cela permet d'appeler une fonction à travers un pointeur de fonction.

Voici un exemple d'utilisation d'un pointeur de fonction :

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int main() {
    int (*operation)(int, int);
    operation = add;
    int result = operation(5, 3);
    printf("Result: %d\n", result);
    return 0;
}

Dans cet exemple, un pointeur de fonction nommé operation est déclaré. Il est ensuite assigné à la fonction add. Ensuite, le pointeur de fonction est utilisé pour appeler la fonction add avec les arguments 5 et 3. Le résultat est ensuite affiché à l'aide de la fonction printf.

Les fonctions récursives sont des fonctions qui s'appellent elles-mêmes. Elles sont souvent utilisées pour résoudre des problèmes qui peuvent être décomposés en sous-problèmes plus petits.

Voici un exemple d'une fonction récursive pour calculer le factoriel d'un nombre :

#include <stdio.h>

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int result = factorial(5);
    printf("Factorial: %d\n", result);
    return 0;
}

Dans cet exemple, la fonction factorial est définie de manière récursive. Si l'argument n est égal à zéro, la fonction renvoie 1. Sinon, la fonction renvoie n multiplié par le factoriel de n-1. Ainsi, factorial(5) renvoie 5 * 4 * 3 * 2 * 1, ce qui équivaut à 120.

Programmation orientée objet en C

La programmation orientée objet (POO) est un paradigme de programmation qui permet de modéliser le monde réel en utilisant des objets qui interagissent les uns avec les autres. Bien que le langage C ne soit pas nativement orienté objet, il est possible de mettre en œuvre des concepts de la POO en utilisant des structures et des fonctions.

Voici un exemple de mise en œuvre d'une classe en utilisant des structures en C :

#include <stdio.h>

typedef struct {
    int x;
    int y;
} Point;

void printPoint(Point p) {
    printf("Point: (%d, %d)\n", p.x, p.y);
}

int main() {
    Point p;
    p.x = 5;
    p.y = 3;
    printPoint(p);
    return 0;
}

Dans cet exemple, une structure Point est définie pour représenter un point dans un plan cartésien. La fonction printPoint est définie pour afficher les coordonnées d'un point. Dans la fonction main, un objet de type Point est créé et initialisé avec les valeurs 5 et 3. Ensuite, la fonction printPoint est appelée pour afficher les coordonnées du point.

Bien que cette approche ne soit pas aussi puissante que la programmation orientée objet dans d'autres langages, elle permet de mettre en œuvre certains concepts de base de la POO en C.
