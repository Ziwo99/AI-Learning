Section 3 : Les tableaux et les chaînes de caractères

Déclaration et utilisation des tableaux

Un tableau en langage C est une structure de données qui permet de stocker une collection d'éléments de même type. Pour déclarer un tableau, on utilise la syntaxe suivante :

type nom_tableau[taille];

où "type" représente le type des éléments du tableau, "nom_tableau" est le nom que vous donnez au tableau et "taille" est le nombre d'éléments que le tableau peut contenir.

Par exemple, pour déclarer un tableau d'entiers de taille 5, vous pouvez utiliser la déclaration suivante :

int monTableau[5];

Pour accéder aux éléments d'un tableau, on utilise l'indice de l'élément. Les indices commencent à 0 pour le premier élément et vont jusqu'à taille-1 pour le dernier élément. Par exemple, pour accéder au troisième élément d'un tableau, vous pouvez utiliser l'expression suivante :

int troisiemeElement = monTableau[2];

Manipulation des tableaux (tri, recherche, etc.)

En langage C, il existe plusieurs opérations courantes que l'on peut effectuer sur les tableaux, telles que le tri, la recherche d'un élément, l'insertion d'un nouvel élément, etc.

Pour trier un tableau, vous pouvez utiliser l'algorithme de tri le plus adapté à votre situation, comme le tri à bulles, le tri par insertion ou le tri rapide. Voici un exemple de tri à bulles : 

void bubbleSort(int tableau[], int taille) {
    int i, j;
    for (i = 0; i < taille-1; i++) {
        for (j = 0; j < taille-i-1; j++) {
            if (tableau[j] > tableau[j+1]) {
                int temp = tableau[j];
                tableau[j] = tableau[j+1];
                tableau[j+1] = temp;
            }
        }
    }
}

Pour rechercher un élément dans un tableau, vous pouvez utiliser une boucle pour parcourir tous les éléments du tableau et comparer chaque élément avec la valeur recherchée. Voici un exemple de recherche d'un élément :

int search(int tableau[], int taille, int valeur) {
    int i;
    for (i = 0; i < taille; i++) {
        if (tableau[i] == valeur) {
            return i;
        }
    }
    return -1; // Retourne -1 si l'élément n'est pas trouvé
}

Les chaînes de caractères en C

En langage C, une chaîne de caractères est une séquence de caractères terminée par le caractère nul ('\0'). Les chaînes de caractères sont généralement utilisées pour représenter du texte.

Pour déclarer une chaîne de caractères, vous pouvez utiliser la syntaxe suivante :

char nom_chaine[taille];

où "nom_chaine" est le nom que vous donnez à la chaîne de caractères et "taille" est le nombre de caractères que la chaîne peut contenir.

Par exemple, pour déclarer une chaîne de caractères de taille 10, vous pouvez utiliser la déclaration suivante :

char maChaine[10];

Les chaînes de caractères peuvent être initialisées lors de leur déclaration en utilisant des guillemets doubles :

char maChaine[] = "Bonjour";

Vous pouvez accéder aux caractères individuels d'une chaîne de caractères en utilisant l'indice du caractère. Par exemple, pour accéder au troisième caractère d'une chaîne de caractères, vous pouvez utiliser l'expression suivante :

char troisiemeCaractere = maChaine[2];

Fonctions de manipulation des chaînes de caractères

En langage C, il existe plusieurs fonctions prédéfinies qui permettent de manipuler les chaînes de caractères. Voici quelques-unes des fonctions les plus couramment utilisées :

  • strlen(chaine) : retourne la longueur de la chaîne de caractères
  • strcpy(destination, source) : copie la chaîne de caractères source dans la chaîne de caractères destination
  • strcat(destination, source) : concatène la chaîne de caractères source à la fin de la chaîne de caractères destination
  • strcmp(chaine1, chaine2) : compare les chaînes de caractères chaine1 et chaine2 et retourne un entier négatif, nul ou positif selon que chaine1 est inférieure, égale ou supérieure à chaine2

Voici un exemple d'utilisation de ces fonctions :

#include <string.h>

int main() {
    char chaine1[20] = "Bonjour";
    char chaine2[20] = "Monde";

    int longueur = strlen(chaine1);
    printf("Longueur de chaine1 : %d\n", longueur);

    strcpy(chaine1, chaine2);
    printf("chaine1 après copie : %s\n", chaine1);

    strcat(chaine1, chaine2);
    printf("chaine1 après concaténation : %s\n", chaine1);

    int comparaison = strcmp(chaine1, chaine2);
    printf("Résultat de la comparaison : %d\n", comparaison);

    return 0;
}

Ce code affichera :

Longueur de chaine1 : 7
chaine1 après copie : Monde
chaine1 après concaténation : MondeMonde
Résultat de la comparaison : 0
