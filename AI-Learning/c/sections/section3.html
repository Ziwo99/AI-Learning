Structures et unions

Les structures et les unions sont des types de données avancés qui permettent de regrouper plusieurs variables de types différents en une seule entité. Une structure est une collection de variables appelées membres, tandis qu'une union est une collection de variables partageant le même espace mémoire.

Les structures sont déclarées à l'aide du mot-clé struct suivi du nom de la structure et de la liste des membres entre des accolades. Par exemple :

struct Person {
    char name[50];
    int age;
    float height;
};

Une fois déclarée, une structure peut être utilisée pour créer des variables :

struct Person person1;
person1.age = 25;
strcpy(person1.name, "John Doe");
person1.height = 1.75;

Les unions fonctionnent de manière similaire, mais les membres partagent le même espace mémoire. Cela signifie que seule une des variables membres peut être utilisée à la fois. Les unions sont déclarées de la même manière que les structures, mais avec le mot-clé union.

Enumérations

Les énumérations sont des types de données qui permettent de définir un ensemble de valeurs constantes nommées. Chaque valeur de l'énumération est appelée une énumération.

Une énumération est déclarée à l'aide du mot-clé enum suivi du nom de l'énumération et de la liste des énumérations entre des accolades. Par exemple :

enum DaysOfWeek {
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
};

Une fois déclarée, une énumération peut être utilisée pour créer des variables :

enum DaysOfWeek today = Tuesday;

Les énumérations sont souvent utilisées pour représenter des ensembles de valeurs prédéfinies, comme les jours de la semaine ou les mois de l'année.

Allocation dynamique de mémoire

L'allocation dynamique de mémoire permet de réserver de l'espace mémoire pendant l'exécution d'un programme. Cela permet de créer des structures de données flexibles dont la taille peut varier en fonction des besoins.

En langage C, l'allocation dynamique de mémoire se fait à l'aide des fonctions malloc, calloc et realloc. La fonction malloc alloue un bloc de mémoire de la taille spécifiée, la fonction calloc alloue un bloc de mémoire initialisé à zéro, et la fonction realloc permet de redimensionner un bloc de mémoire déjà alloué.

Voici un exemple d'allocation dynamique de mémoire :

int* numbers = (int*)malloc(5 * sizeof(int));
if (numbers != NULL) {
    numbers[0] = 1;
    numbers[1] = 2;
    numbers[2] = 3;
    numbers[3] = 4;
    numbers[4] = 5;
}

Il est important de libérer la mémoire allouée dynamiquement à l'aide de la fonction free une fois que vous avez terminé de l'utiliser :

free(numbers);

Listes chaînées

Les listes chaînées sont des structures de données linéaires qui permettent de stocker et d'accéder à des données de manière séquentielle. Chaque élément de la liste, appelé nœud, contient une valeur et un pointeur vers le nœud suivant.

En langage C, les listes chaînées sont généralement implémentées à l'aide de structures. Voici un exemple de structure de nœud pour une liste chaînée d'entiers :

struct Node {
    int data;
    struct Node* next;
};

Pour créer une liste chaînée, il faut déclarer un pointeur vers le premier nœud de la liste :

struct Node* head = NULL;

Ensuite, on peut ajouter des nœuds à la liste en allouant dynamiquement de la mémoire :

struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
newNode->data = 1;
newNode->next = NULL;
head = newNode;

Pour parcourir une liste chaînée, on peut utiliser une boucle while en utilisant le pointeur next pour passer d'un nœud à l'autre.

Arbres et graphes

Les arbres et les graphes sont des structures de données non linéaires qui permettent de représenter des relations hiérarchiques entre des éléments.

Un arbre est une structure de données composée de nœuds, où chaque nœud peut avoir un nombre quelconque de nœuds enfants. Le premier nœud de l'arbre est appelé la racine.

Un graphe est une structure de données composée de nœuds, où chaque nœud peut être relié à un ou plusieurs autres nœuds par des arêtes. Les graphes peuvent être orientés (les arêtes ont une direction) ou non orientés.

En langage C, les arbres et les graphes sont généralement implémentés à l'aide de structures et de pointeurs. Les opérations sur les arbres et les graphes peuvent être complexes et nécessitent souvent l'utilisation de récursion.

Voici un exemple de structure de nœud pour un arbre binaire :

struct TreeNode {
    int data;
    struct TreeNode* left;
    struct TreeNode* right;
};

Les opérations sur les arbres et les graphes incluent l'insertion, la suppression, la recherche, le parcours et bien d'autres.
