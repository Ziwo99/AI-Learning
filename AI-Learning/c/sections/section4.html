Section 4 : Les fonctions et les pointeurs

Définition et utilisation des fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent de diviser un programme en parties plus petites et plus gérables. Une fonction est définie en utilisant le mot-clé void suivi du nom de la fonction, suivi de parenthèses qui peuvent contenir des paramètres.

Voici un exemple de définition de fonction :

void printHello() {
    printf("Bonjour !\n");
}

Pour utiliser une fonction, il suffit d'appeler son nom suivi de parenthèses. Par exemple :

printHello();

Cela affichera "Bonjour !" à l'écran.

Passage de paramètres par valeur et par référence

Les fonctions peuvent également prendre des paramètres, qui sont des valeurs ou des adresses mémoire passées à la fonction pour qu'elle les utilise. Il existe deux façons de passer des paramètres à une fonction : par valeur et par référence.

Lorsque les paramètres sont passés par valeur, une copie de la valeur est créée et utilisée dans la fonction. Cela signifie que toute modification apportée aux paramètres à l'intérieur de la fonction n'affectera pas les variables d'origine. Voici un exemple :

void addTwoNumbers(int a, int b) {
    int sum = a + b;
    printf("La somme est : %d\n", sum);
}

int main() {
    int x = 5;
    int y = 3;
    addTwoNumbers(x, y);
    return 0;
}

Cela affichera "La somme est : 8" à l'écran.

Lorsque les paramètres sont passés par référence, l'adresse mémoire des variables est passée à la fonction. Cela signifie que toute modification apportée aux paramètres à l'intérieur de la fonction affectera également les variables d'origine. Voici un exemple :

void increment(int *num) {
    (*num)++;
}

int main() {
    int x = 5;
    increment(&x);
    printf("La valeur de x est : %d\n", x);
    return 0;
}

Cela affichera "La valeur de x est : 6" à l'écran.

Les pointeurs et les adresses mémoire

Un pointeur est une variable qui contient l'adresse mémoire d'une autre variable. Il est utilisé pour accéder et manipuler directement la valeur de cette variable. Les pointeurs sont largement utilisés en C, notamment pour la manipulation des tableaux et des chaînes de caractères.

Voici comment déclarer un pointeur :

int *ptr;

Le symbole * indique que la variable ptr est un pointeur vers un entier.

Voici comment obtenir l'adresse mémoire d'une variable :

int num = 10;
int *ptr = #

Le symbole & est utilisé pour obtenir l'adresse mémoire de la variable num et l'assigner au pointeur ptr.

Voici comment accéder à la valeur d'une variable à l'aide d'un pointeur :

int num = 10;
int *ptr = #
printf("La valeur de num est : %d\n", *ptr);

Cela affichera "La valeur de num est : 10" à l'écran.

Utilisation des pointeurs pour la manipulation des tableaux et des chaînes de caractères

Les pointeurs sont souvent utilisés pour manipuler des tableaux et des chaînes de caractères en C. En utilisant des pointeurs, il est possible d'accéder et de modifier les éléments d'un tableau ou les caractères d'une chaîne de caractères.

Voici un exemple d'utilisation d'un pointeur pour accéder aux éléments d'un tableau :

int numbers[] = {1, 2, 3, 4, 5};
int *ptr = numbers;
printf("Le premier élément du tableau est : %d\n", *ptr);

Cela affichera "Le premier élément du tableau est : 1" à l'écran.

Voici un exemple d'utilisation d'un pointeur pour accéder aux caractères d'une chaîne de caractères :

char *message = "Bonjour";
printf("Le premier caractère de la chaîne est : %c\n", *message);

Cela affichera "Le premier caractère de la chaîne est : B" à l'écran.
