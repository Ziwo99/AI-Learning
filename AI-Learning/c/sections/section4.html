Ouverture, lecture et écriture de fichiers

La manipulation de fichiers est une tâche courante en programmation C. Pour travailler avec des fichiers, vous devez d'abord les ouvrir en utilisant la fonction fopen(). Cette fonction prend deux paramètres : le nom du fichier et le mode d'ouverture.

Voici les modes d'ouverture les plus couramment utilisés :

  • "r" : ouvre un fichier en mode lecture. Le fichier doit exister.
  • "w" : ouvre un fichier en mode écriture. Si le fichier existe, son contenu est supprimé. Sinon, un nouveau fichier est créé.
  • "a" : ouvre un fichier en mode ajout. Si le fichier existe, les données sont ajoutées à la fin. Sinon, un nouveau fichier est créé.

Voici un exemple d'ouverture d'un fichier en mode lecture :

FILE *file = fopen("exemple.txt", "r");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier.");
    return 1;
}

Une fois le fichier ouvert, vous pouvez lire son contenu en utilisant des fonctions telles que fscanf() ou fgets(). Voici un exemple de lecture d'une ligne à partir d'un fichier :

char ligne[100];
if (fgets(ligne, 100, file) != NULL) {
    printf("Contenu du fichier : %s", ligne);
}

Pour écrire dans un fichier, vous pouvez utiliser des fonctions telles que fprintf() ou fputs(). Voici un exemple d'écriture dans un fichier :

fprintf(file, "Bonjour, monde !");

Une fois que vous avez terminé de travailler avec un fichier, vous devez le fermer en utilisant la fonction fclose(). Cela libère les ressources associées au fichier.

fclose(file);

Manipulation de fichiers binaires

En plus de manipuler des fichiers texte, vous pouvez également manipuler des fichiers binaires en C. Les fichiers binaires contiennent des données brutes, telles que des images, des sons ou des vidéos.

Pour lire ou écrire des fichiers binaires, vous pouvez utiliser les fonctions fread() et fwrite(). Voici un exemple de lecture d'un fichier binaire :

FILE *file = fopen("image.bin", "rb");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier.");
    return 1;
}

unsigned char buffer[1024];
size_t bytesRead = fread(buffer, sizeof(unsigned char), 1024, file);
fclose(file);

La fonction fread() lit un certain nombre d'éléments de taille fixe à partir du fichier et les stocke dans un tampon. Dans cet exemple, nous lisons 1024 octets du fichier binaire "image.bin" dans le tampon "buffer". La taille de chaque élément est spécifiée avec l'opérateur sizeof().

Voici un exemple d'écriture dans un fichier binaire :

FILE *file = fopen("image.bin", "wb");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier.");
    return 1;
}

unsigned char buffer[1024];
// Remplir le tampon avec des données à écrire

size_t bytesWritten = fwrite(buffer, sizeof(unsigned char), 1024, file);
fclose(file);

Gestion des erreurs et exceptions

Lors de la manipulation de fichiers, il est important de gérer les erreurs qui peuvent survenir. Par exemple, si vous essayez d'ouvrir un fichier qui n'existe pas, vous devez être en mesure de détecter cette erreur et de prendre les mesures appropriées.

La fonction fopen() renvoie un pointeur NULL si elle échoue. Vous pouvez utiliser cette valeur de retour pour vérifier si l'ouverture du fichier a réussi ou non. Voici un exemple :

FILE *file = fopen("exemple.txt", "r");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier.");
    return 1;
}

De plus, vous pouvez utiliser la fonction feof() pour vérifier si la fin du fichier a été atteinte lors de la lecture. Voici un exemple :

while (!feof(file)) {
    // Lire le contenu du fichier
}

Enfin, n'oubliez pas de gérer les erreurs lors de l'écriture dans un fichier. Par exemple, si le disque est plein, l'écriture peut échouer. Vous pouvez vérifier si l'écriture a réussi en comparant le nombre d'éléments écrits avec le nombre d'éléments que vous avez essayé d'écrire.

size_t bytesWritten = fwrite(buffer, sizeof(unsigned char), 1024, file);
if (bytesWritten != 1024) {
    printf("Erreur lors de l'écriture dans le fichier.");
    return 1;
}

En résumé, la manipulation de fichiers en C nécessite d'ouvrir, lire, écrire et fermer des fichiers en utilisant les fonctions appropriées. Assurez-vous de gérer les erreurs qui peuvent survenir et de vérifier le succès des opérations de lecture et d'écriture.
