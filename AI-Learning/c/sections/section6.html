Manipulation des fichiers en C

La manipulation des fichiers en C est une fonctionnalité essentielle pour lire et écrire des données à partir de fichiers. Pour manipuler des fichiers en C, vous devez utiliser la bibliothèque standard stdio.h.

Pour ouvrir un fichier en lecture, vous pouvez utiliser la fonction fopen avec le mode d'ouverture "r". Par exemple :

FILE *file = fopen("mon_fichier.txt", "r");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier en lecture.");
    return 1;
}

Une fois le fichier ouvert, vous pouvez utiliser les fonctions de la bibliothèque stdio.h pour lire les données à partir du fichier. Par exemple, vous pouvez utiliser la fonction fscanf pour lire des entiers :

int num;
while (fscanf(file, "%d", &num) != EOF) {
    printf("%d\n", num);
}

Pour fermer un fichier, vous pouvez utiliser la fonction fclose :

fclose(file);

Lecture et écriture de données dans un fichier

En plus de la lecture, vous pouvez également écrire des données dans un fichier en utilisant les fonctions de la bibliothèque stdio.h. Pour ouvrir un fichier en écriture, vous pouvez utiliser la fonction fopen avec le mode d'ouverture "w". Par exemple :

FILE *file = fopen("mon_fichier.txt", "w");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier en écriture.");
    return 1;
}

Une fois le fichier ouvert, vous pouvez utiliser les fonctions de la bibliothèque stdio.h pour écrire des données dans le fichier. Par exemple, vous pouvez utiliser la fonction fprintf pour écrire une chaîne de caractères :

fprintf(file, "Bonjour, monde !");

Pour fermer un fichier, vous pouvez utiliser la fonction fclose :

fclose(file);

Gestion des erreurs lors de la manipulation des fichiers

Lors de la manipulation des fichiers en C, il est important de gérer les erreurs qui peuvent survenir. Vous pouvez utiliser la fonction feof pour vérifier si la fin du fichier a été atteinte. Par exemple :

while (!feof(file)) {
    // Lire les données du fichier
}

De plus, vous pouvez utiliser la fonction ferror pour vérifier si une erreur s'est produite lors de la manipulation du fichier. Par exemple :

if (ferror(file)) {
    printf("Une erreur s'est produite lors de la manipulation du fichier.");
    return 1;
}

Il est également recommandé de vérifier si le fichier a été ouvert avec succès en vérifiant si le pointeur de fichier est NULL. Par exemple :

FILE *file = fopen("mon_fichier.txt", "r");
if (file == NULL) {
    printf("Impossible d'ouvrir le fichier en lecture.");
    return 1;
}

En gérant correctement les erreurs lors de la manipulation des fichiers, vous pouvez vous assurer que votre programme fonctionne de manière fiable et éviter les problèmes potentiels.
