Structure d'un programme C

Un programme C est composé de fonctions qui sont appelées pour exécuter des tâches spécifiques. Chaque programme C doit avoir une fonction principale appelée main. Voici un exemple de structure de base d'un programme C :

#include <stdio.h>

int main() {
    // Instructions du programme
    return 0;
}

La ligne #include <stdio.h> est une directive de préprocesseur qui permet d'inclure le fichier d'en-tête stdio.h dans le programme. Ce fichier d'en-tête contient les déclarations des fonctions d'entrée/sortie standard.

Variables et types de données

En C, les variables sont utilisées pour stocker des données. Chaque variable a un type de données qui détermine la taille de la mémoire allouée pour stocker la valeur de la variable. Voici quelques types de données couramment utilisés en C :

  • int : pour stocker des nombres entiers.
  • float : pour stocker des nombres décimaux à virgule flottante.
  • char : pour stocker des caractères.
  • double : pour stocker des nombres décimaux à virgule flottante avec une précision accrue.

Voici un exemple de déclaration et d'initialisation de variables en C :

int age = 25;
float pi = 3.14;
char grade = 'A';
double salary = 50000.50;

Opérateurs et expressions

Les opérateurs en C sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques opérateurs couramment utilisés en C :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
  • % : modulo (reste de la division)
  • ++ : incrémentation
  • -- : décrémentation

Les expressions en C sont des combinaisons d'opérateurs, de variables et de valeurs qui produisent un résultat. Voici un exemple d'expression en C :

int sum = 5 + 10 * 2;

Dans cet exemple, l'expression 5 + 10 * 2 est évaluée en effectuant d'abord la multiplication, puis l'addition, ce qui donne une valeur de 25 pour la variable sum.

Instructions de contrôle (boucles et conditions)

Les instructions de contrôle en C permettent de contrôler le flux d'exécution du programme. Les boucles permettent de répéter un bloc d'instructions plusieurs fois, tandis que les conditions permettent d'exécuter des blocs d'instructions en fonction de certaines conditions. Voici quelques instructions de contrôle couramment utilisées en C :

  • if : exécute un bloc d'instructions si une condition est vraie.
  • else : exécute un bloc d'instructions si la condition de l'instruction if est fausse.
  • while : répète un bloc d'instructions tant qu'une condition est vraie.
  • for : répète un bloc d'instructions un certain nombre de fois.
  • switch : exécute un bloc d'instructions en fonction de la valeur d'une expression.

Voici un exemple d'utilisation de l'instruction if en C :

int age = 18;

if (age >= 18) {
    printf("Vous êtes majeur.");
} else {
    printf("Vous êtes mineur.");
}

Dans cet exemple, si la variable age est supérieure ou égale à 18, le message "Vous êtes majeur." sera affiché. Sinon, le message "Vous êtes mineur." sera affiché.

Fonctions et procédures

Les fonctions en C sont des blocs de code qui effectuent une tâche spécifique. Elles peuvent prendre des paramètres en entrée et renvoyer une valeur en sortie. Voici un exemple de déclaration et d'appel de fonction en C :

int add(int a, int b) {
    return a + b;
}

int result = add(5, 10);

Dans cet exemple, la fonction add prend deux paramètres a et b et renvoie la somme de ces deux paramètres. L'appel de la fonction add(5, 10) renvoie la valeur 15 qui est ensuite stockée dans la variable result.

Tableaux et pointeurs

Les tableaux en C permettent de stocker plusieurs valeurs de même type dans une seule variable. Chaque valeur dans un tableau est identifiée par un index. Voici un exemple de déclaration et d'utilisation d'un tableau en C :

int numbers[5] = {1, 2, 3, 4, 5};

for (int i = 0; i < 5; i++) {
    printf("%d ", numbers[i]);
}

Dans cet exemple, le tableau numbers est déclaré avec une taille de 5 et initialisé avec les valeurs 1, 2, 3, 4, 5. La boucle for itère sur chaque élément du tableau et les affiche à l'aide de la fonction printf.

Les pointeurs en C sont des variables qui contiennent l'adresse mémoire d'une autre variable. Ils sont utilisés pour manipuler directement la mémoire et accéder aux valeurs des variables. Voici un exemple d'utilisation de pointeurs en C :

int number = 10;
int *ptr = &number;

printf("%d", *ptr);

Dans cet exemple, la variable ptr est un pointeur qui contient l'adresse mémoire de la variable number. L'opérateur * est utilisé pour accéder à la valeur de la variable pointée par le pointeur. L'instruction printf("%d", *ptr) affiche la valeur 10 de la variable number.
