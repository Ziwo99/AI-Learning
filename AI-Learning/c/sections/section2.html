Structure d'un programme en C

Un programme en langage C est composé de plusieurs éléments clés. Tout d'abord, il doit contenir une fonction principale appelée main(). C'est à partir de cette fonction que le programme démarre son exécution. À l'intérieur de la fonction main(), on peut écrire le code qui sera exécuté.

Voici un exemple de structure de base d'un programme en C :

#include <stdio.h>

int main() {
    // Code à exécuter
    return 0;
}

Variables et types de données

En langage C, les variables sont utilisées pour stocker des données. Avant d'utiliser une variable, il est nécessaire de la déclarer en spécifiant son type. Voici quelques types de données couramment utilisés en C :

  • int : pour stocker des nombres entiers
  • float : pour stocker des nombres décimaux
  • char : pour stocker des caractères
  • double : pour stocker des nombres décimaux avec une précision accrue

Voici un exemple de déclaration et d'utilisation de variables en C :

int age = 25;
float poids = 65.5;
char lettre = 'A';
double pi = 3.14159;

Opérateurs et expressions

Les opérateurs en langage C permettent d'effectuer des opérations sur les variables et les valeurs. Voici quelques exemples d'opérateurs couramment utilisés :

  • Opérateurs arithmétiques : +, -, *, /, %
  • Opérateurs de comparaison : ==, !=, >, <, >=, <=
  • Opérateurs logiques : && (et), || (ou), ! (non)

Les expressions en langage C sont des combinaisons d'opérateurs et de variables qui produisent une valeur. Voici un exemple d'expression en C :

int a = 5;
int b = 10;
int c = a + b;

Les instructions de contrôle (if, else, switch)

Les instructions de contrôle permettent de prendre des décisions dans un programme en fonction de certaines conditions. Voici quelques exemples d'instructions de contrôle en langage C :

  • L'instruction if permet d'exécuter un bloc de code si une condition est vraie : 
int age = 18;
if (age >= 18) {
    printf("Vous êtes majeur.");
}
  • L'instruction else permet d'exécuter un bloc de code si la condition de l'instruction if est fausse :
int age = 16;
if (age >= 18) {
    printf("Vous êtes majeur.");
} else {
    printf("Vous êtes mineur.");
}
  • L'instruction switch permet d'exécuter différents blocs de code en fonction de la valeur d'une variable :
int choix = 2;
switch (choix) {
    case 1:
        printf("Vous avez choisi l'option 1.");
        break;
    case 2:
        printf("Vous avez choisi l'option 2.");
        break;
    default:
        printf("Option invalide.");
        break;
}

Les boucles (for, while, do-while)

Les boucles permettent de répéter l'exécution d'un bloc de code tant qu'une condition est vraie. Voici quelques exemples de boucles en langage C :

  • La boucle for permet de répéter un bloc de code un certain nombre de fois : 
for (int i = 0; i < 5; i++) {
    printf("Iteration %d\n", i);
}
  • La boucle while permet de répéter un bloc de code tant qu'une condition est vraie :
int i = 0;
while (i < 5) {
    printf("Iteration %d\n", i);
    i++;
}
  • La boucle do-while permet de répéter un bloc de code au moins une fois, puis tant qu'une condition est vraie :
int i = 0;
do {
    printf("Iteration %d\n", i);
    i++;
} while (i < 5);
