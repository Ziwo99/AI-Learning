Création d'un jeu en ligne de commande

Dans cette sous-section, nous allons apprendre à créer un jeu en ligne de commande en utilisant Ruby. Un jeu en ligne de commande est un jeu qui s'exécute dans la console et permet à l'utilisateur d'interagir avec le jeu en entrant des commandes textuelles.

Voici un exemple de code pour créer un jeu de devinette en ligne de commande :

def devinette
  nombre_a_deviner = rand(1..100)
  devine = false

  while !devine
    print "Devine le nombre : "
    proposition = gets.chomp.to_i

    if proposition == nombre_a_deviner
      devine = true
      puts "Bravo ! Tu as deviné le nombre."
    elsif proposition < nombre_a_deviner
      puts "Le nombre à deviner est plus grand."
    else
      puts "Le nombre à deviner est plus petit."
    end
  end
end

devinette

Dans cet exemple, nous utilisons la méthode rand() pour générer un nombre aléatoire entre 1 et 100. Ensuite, nous demandons à l'utilisateur de deviner le nombre en utilisant la méthode gets.chomp.to_i pour obtenir l'entrée de l'utilisateur et la convertir en entier. En fonction de la proposition de l'utilisateur, nous affichons un message approprié. Le jeu continue jusqu'à ce que l'utilisateur devine le bon nombre.

Développement d'une application web

Dans cette sous-section, nous allons apprendre à développer une application web en utilisant Ruby. Une application web est un logiciel qui s'exécute sur un serveur web et est accessible via un navigateur web.

Voici un exemple de code pour créer une application web simple en utilisant le framework Ruby on Rails :

# app/controllers/welcome_controller.rb
class WelcomeController < ApplicationController
  def index
    render plain: "Bienvenue sur notre application web !"
  end
end

# config/routes.rb
Rails.application.routes.draw do
  root 'welcome#index'
end

Dans cet exemple, nous avons un contrôleur WelcomeController avec une action index qui renvoie un message de bienvenue. Nous avons également configuré la route racine pour afficher la page d'accueil.

Création d'une API RESTful

Dans cette sous-section, nous allons apprendre à créer une API RESTful en utilisant Ruby. Une API RESTful est une interface de programmation d'application qui suit les principes de l'architecture REST (Representational State Transfer) et permet aux clients d'interagir avec les ressources via des requêtes HTTP.

Voici un exemple de code pour créer une API RESTful simple en utilisant le framework Sinatra :

# app.rb
require 'sinatra'
require 'json'

get '/api/users' do
  content_type :json
  users = [
    { id: 1, name: 'John Doe' },
    { id: 2, name: 'Jane Smith' }
  ]
  users.to_json
end

Dans cet exemple, nous avons une route GET '/api/users' qui renvoie une liste d'utilisateurs au format JSON. Nous utilisons la méthode to_json pour convertir l'objet Ruby en JSON.

Ce sont des exemples simples pour vous montrer comment créer un jeu en ligne de commande, une application web et une API RESTful en utilisant Ruby. Vous pouvez les personnaliser et les étendre selon vos besoins.
