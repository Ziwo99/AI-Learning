Création d'une application de gestion de tâches

Dans cette sous-section, nous allons créer une application de gestion de tâches en utilisant Ruby. Cette application permettra aux utilisateurs de créer, modifier et supprimer des tâches, ainsi que de les marquer comme terminées.

Voici un exemple de code pour créer une classe Task :

class Task
  attr_accessor :title, :description, :completed

  def initialize(title, description)
    @title = title
    @description = description
    @completed = false
  end

  def mark_as_completed
    @completed = true
  end
end

En utilisant cette classe, nous pouvons créer une liste de tâches :

tasks = []
tasks << Task.new("Faire les courses", "Acheter du lait et du pain")
tasks << Task.new("Apprendre Ruby", "Suivre un tutoriel en ligne")

Nous pouvons également modifier une tâche existante :

tasks[0].title = "Faire le ménage"
tasks[0].description = "Nettoyer la cuisine et passer l'aspirateur"

Et enfin, nous pouvons marquer une tâche comme terminée :

tasks[1].mark_as_completed

Développement d'un jeu de devinette

Dans cette sous-section, nous allons développer un jeu de devinette en utilisant Ruby. Le jeu consiste à deviner un nombre choisi aléatoirement par l'ordinateur.

Voici un exemple de code pour le jeu de devinette :

number_to_guess = rand(1..100)
guesses = 0
guessed_number = false

while !guessed_number
  print "Devinez le nombre : "
  user_guess = gets.chomp.to_i
  guesses += 1

  if user_guess == number_to_guess
    guessed_number = true
    puts "Bravo, vous avez deviné le nombre en #{guesses} essais !"
  elsif user_guess < number_to_guess
    puts "Le nombre à deviner est plus grand."
  else
    puts "Le nombre à deviner est plus petit."
  end
end

Ce code génère un nombre aléatoire entre 1 et 100, puis demande à l'utilisateur de deviner le nombre. Le programme indique si le nombre à deviner est plus grand ou plus petit que la proposition de l'utilisateur, jusqu'à ce que le nombre soit deviné correctement.

Construction d'un site web simple avec Ruby on Rails

Dans cette sous-section, nous allons construire un site web simple en utilisant le framework Ruby on Rails. Ce site web permettra aux utilisateurs de créer, modifier et supprimer des articles.

Voici les étapes pour créer ce site web :

  1. Créez une nouvelle application Rails en utilisant la commande rails new blog.
  2. Générez un modèle Article avec les attributs title et content en utilisant la commande rails generate model Article title:string content:text.
  3. Effectuez une migration de la base de données en utilisant la commande rails db:migrate.
  4. Créez un contrôleur Articles avec les actions index, show, new, create, edit, update et destroy en utilisant la commande rails generate controller Articles.
  5. Définissez les routes pour les actions du contrôleur Articles dans le fichier config/routes.rb.
  6. Implémentez les vues correspondantes pour chaque action du contrôleur Articles dans le dossier app/views/articles.

Avec ces étapes, vous aurez créé un site web fonctionnel permettant aux utilisateurs de gérer des articles.

Ce projet est plus complexe que les précédents, mais il vous permettra de mettre en pratique les concepts que vous avez appris jusqu'à présent et de vous familiariser avec le développement web en utilisant Ruby on Rails.
