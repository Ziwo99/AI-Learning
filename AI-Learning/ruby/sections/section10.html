Section 10 : Projets pratiques en Ruby

Création d'une application de gestion de tâches

Dans cette sous-section, nous allons apprendre à créer une application de gestion de tâches en utilisant Ruby. Cette application permettra aux utilisateurs de créer, afficher, mettre à jour et supprimer des tâches.

Voici un exemple de code pour créer une classe Task :

class Task
  attr_accessor :title, :description, :completed

  def initialize(title, description)
    @title = title
    @description = description
    @completed = false
  end

  def complete
    @completed = true
  end

  def to_s
    "#{@title} - #{@description} (#{@completed ? 'Completed' : 'Incomplete'})"
  end
end

En utilisant cette classe Task, vous pouvez créer des instances de tâches, les afficher, les marquer comme terminées, etc.

Développement d'un jeu simple en Ruby

Dans cette sous-section, nous allons développer un jeu simple en utilisant Ruby. Le jeu consistera à deviner un nombre aléatoire généré par l'ordinateur.

Voici un exemple de code pour le jeu :

def guess_number
  number = rand(1..100)
  guessed = false

  while !guessed
    print "Guess a number between 1 and 100: "
    guess = gets.chomp.to_i

    if guess == number
      puts "Congratulations! You guessed the number."
      guessed = true
    elsif guess < number
      puts "Too low. Try again."
    else
      puts "Too high. Try again."
    end
  end
end

guess_number

En exécutant ce code, les joueurs pourront deviner le nombre généré par l'ordinateur jusqu'à ce qu'ils le trouvent.

Création d'un site web basique avec Ruby on Rails

Dans cette sous-section, nous allons apprendre à créer un site web basique en utilisant Ruby on Rails. Ce site web aura des fonctionnalités de base telles que l'inscription des utilisateurs, la connexion, la création de pages, etc.

Voici un exemple de code pour créer une page d'accueil simple :

# app/controllers/pages_controller.rb
class PagesController < ApplicationController
  def home
  end
end

# app/views/pages/home.html.erb
<h1>Welcome to my website!</h1>
<p>This is the home page of my website.</p>

# config/routes.rb
Rails.application.routes.draw do
  root 'pages#home'
end

En utilisant ce code, vous pouvez créer une page d'accueil simple pour votre site web Ruby on Rails.
