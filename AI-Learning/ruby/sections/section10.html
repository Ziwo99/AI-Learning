Ruby on Rails

Ruby on Rails, également connu sous le nom de Rails, est un framework de développement d'applications web écrit en Ruby. Il suit le principe de convention plutôt que de configuration, ce qui signifie qu'il fournit des conventions prédéfinies pour le développement d'applications web, ce qui permet aux développeurs de se concentrer sur la logique métier plutôt que sur les détails techniques.

Rails utilise le modèle MVC (Modèle-Vue-Contrôleur) pour organiser le code de l'application. Le modèle représente les données et la logique métier, la vue est responsable de l'affichage des données à l'utilisateur et le contrôleur gère les interactions entre le modèle et la vue.

Voici un exemple de code Rails qui montre comment définir un modèle de base de données simple :

class User < ApplicationRecord
  validates :name, presence: true
  validates :email, presence: true, uniqueness: true
end

Sinatra

Sinatra est un framework web léger pour Ruby. Il est souvent utilisé pour créer des applications web simples et rapides, en particulier des API RESTful. Sinatra suit le principe de simplicité et de minimalisme, ce qui signifie qu'il fournit uniquement les fonctionnalités essentielles pour le développement d'applications web.

Voici un exemple de code Sinatra qui montre comment définir une route simple :

require 'sinatra'

get '/hello' do
  "Hello, World!"
end

RSpec

RSpec est un framework de test pour Ruby. Il permet aux développeurs d'écrire des tests unitaires et des tests d'intégration pour vérifier le bon fonctionnement de leur code. RSpec suit le principe de lisibilité et de compréhension du code de test, ce qui facilite la création et la maintenance des tests.

Voici un exemple de code RSpec qui montre comment écrire un test unitaire simple :

require 'rspec'

class Calculator
  def add(a, b)
    a + b
  end
end

RSpec.describe Calculator do
  describe "#add" do
    it "returns the sum of two numbers" do
      calculator = Calculator.new
      result = calculator.add(2, 3)
      expect(result).to eq(5)
    end
  end
end

ActiveRecord

ActiveRecord est une bibliothèque d'accès aux bases de données pour Ruby. Elle fournit une interface simple et intuitive pour interagir avec une base de données relationnelle. ActiveRecord suit le principe de convention plutôt que de configuration, ce qui signifie qu'il utilise des conventions prédéfinies pour mapper les tables de la base de données aux modèles Ruby.

Voici un exemple de code ActiveRecord qui montre comment définir un modèle de base de données et effectuer une requête simple :

require 'active_record'

class User < ActiveRecord::Base
end

# Création d'un nouvel utilisateur
user = User.new(name: "John Doe", email: "john@example.com")
user.save

# Requête pour récupérer tous les utilisateurs
users = User.all

Ces frameworks et bibliothèques populaires sont largement utilisés dans l'industrie du développement web Ruby. Ils offrent des fonctionnalités puissantes et facilitent le développement d'applications web robustes et évolutives.
