Histoire et origine de Ruby

Ruby a été créé en 1995 par Yukihiro Matsumoto, également connu sous le nom de Matz. Il a conçu Ruby pour être un langage de programmation facile à lire et à écrire, avec une syntaxe élégante et expressive. Matz s'est inspiré de plusieurs langages de programmation, notamment Perl, Smalltalk, Eiffel et Lisp, pour créer Ruby.

Le nom "Ruby" a été choisi par Matz en référence au langage de programmation Perl, dont il était fan. Il voulait un nom court, facile à retenir et agréable à prononcer.

Installation de Ruby

Pour commencer à programmer en Ruby, vous devez d'abord installer Ruby sur votre ordinateur. Voici les étapes pour installer Ruby :

  1. Rendez-vous sur le site officiel de Ruby à l'adresse https://www.ruby-lang.org.
  2. Téléchargez la dernière version stable de Ruby pour votre système d'exploitation (Windows, macOS, Linux, etc.).
  3. Exécutez le programme d'installation et suivez les instructions à l'écran.
  4. Vérifiez que Ruby est correctement installé en ouvrant une fenêtre de terminal et en tapant la commande ruby -v. Vous devriez voir s'afficher la version de Ruby que vous avez installée.

Les bases de la syntaxe Ruby

La syntaxe de Ruby est conçue pour être simple et lisible. Voici quelques éléments de base de la syntaxe Ruby :

# Ceci est un commentaire en Ruby

# Déclaration d'une variable
nom = "Alice"

# Affichage d'une chaîne de caractères
puts "Bonjour, " + nom + " !"

# Déclaration d'une fonction
def dire_bonjour(nom)
  puts "Bonjour, " + nom + " !"
end

# Appel de la fonction
dire_bonjour("Bob")

Les variables et les types de données en Ruby

En Ruby, vous pouvez utiliser des variables pour stocker des valeurs. Voici quelques exemples de déclaration de variables en Ruby :

# Déclaration d'une variable numérique
age = 25

# Déclaration d'une variable de chaîne de caractères
nom = "Alice"

# Déclaration d'une variable booléenne
est_majeur = true

# Déclaration d'une variable tableau
nombres = [1, 2, 3, 4, 5]

# Déclaration d'une variable hash
personne = { "nom" => "Alice", "age" => 25 }

En Ruby, les variables sont dynamiquement typées, ce qui signifie que vous n'avez pas besoin de spécifier le type de données lors de la déclaration de la variable. Ruby détermine automatiquement le type de données en fonction de la valeur assignée à la variable.

Il existe plusieurs types de données primitifs en Ruby, tels que les entiers, les chaînes de caractères, les booléens, les tableaux et les hashes. Vous pouvez également créer vos propres classes et objets personnalisés en Ruby.
