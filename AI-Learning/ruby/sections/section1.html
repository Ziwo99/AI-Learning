Section 1 : Introduction à Ruby

Histoire et caractéristiques de Ruby

Ruby est un langage de programmation dynamique et orienté objet créé par Yukihiro Matsumoto en 1995. Il a été conçu pour être simple et agréable à utiliser, en mettant l'accent sur la productivité du développeur. Voici quelques caractéristiques clés de Ruby :

  • Langage interprété : Ruby est un langage interprété, ce qui signifie qu'il n'est pas nécessaire de le compiler avant de l'exécuter.
  • Typage dynamique : Les variables en Ruby n'ont pas de type fixe et peuvent changer de type pendant l'exécution du programme.
  • Orienté objet : Ruby est entièrement orienté objet, ce qui signifie que tout en Ruby est un objet, y compris les entiers, les chaînes de caractères et même les fonctions.
  • Syntaxe expressive : Ruby a une syntaxe concise et expressive qui permet aux développeurs d'écrire du code clair et lisible.

Installation de Ruby

Pour commencer à utiliser Ruby, vous devez d'abord l'installer sur votre machine. Voici les étapes pour installer Ruby :

  1. Rendez-vous sur le site officiel de Ruby (https://www.ruby-lang.org) et téléchargez la dernière version de Ruby pour votre système d'exploitation.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran pour installer Ruby.
  3. Vérifiez que Ruby est correctement installé en ouvrant une fenêtre de terminal et en tapant la commande suivante :
ruby --version

Si vous voyez la version de Ruby s'afficher, cela signifie que Ruby est installé avec succès sur votre machine.

Premier programme Ruby

Maintenant que Ruby est installé, vous pouvez écrire votre premier programme Ruby. Voici un exemple simple :

puts "Hello, Ruby!"

Le code ci-dessus utilise la fonction puts pour afficher le texte "Hello, Ruby!" à l'écran. Pour exécuter ce programme, enregistrez-le dans un fichier avec l'extension .rb (par exemple, hello.rb) et exécutez la commande suivante dans une fenêtre de terminal :

ruby hello.rb

Vous devriez voir le texte "Hello, Ruby!" s'afficher à l'écran.
