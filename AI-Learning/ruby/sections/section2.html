Variables et types de données

Les variables sont utilisées pour stocker des valeurs dans un programme Ruby. Elles peuvent contenir différents types de données tels que des nombres, des chaînes de caractères, des booléens, etc. Pour déclarer une variable, utilisez le symbole "=" suivi de la valeur que vous souhaitez lui attribuer.

Exemple :

name = "John"
age = 25
is_student = true

Les types de données les plus couramment utilisés en Ruby sont :

  • Les nombres entiers (integers) : représentent des nombres entiers, par exemple 1, 2, 3.
  • Les nombres décimaux (floats) : représentent des nombres décimaux, par exemple 3.14, 2.5.
  • Les chaînes de caractères (strings) : représentent du texte, par exemple "Hello, world!".
  • Les booléens (booleans) : représentent des valeurs de vérité, soit true (vrai) soit false (faux).
  • Les tableaux (arrays) : représentent une collection ordonnée d'éléments, par exemple [1, 2, 3].
  • Les hash (hashes) : représentent une collection de paires clé-valeur, par exemple {"name" => "John", "age" => 25}.

Opérations arithmétiques et opérateurs logiques

En Ruby, vous pouvez effectuer des opérations arithmétiques telles que l'addition, la soustraction, la multiplication et la division sur des nombres.

Exemple :

a = 10
b = 5

addition = a + b
soustraction = a - b
multiplication = a * b
division = a / b

Les opérateurs logiques sont utilisés pour évaluer des expressions booléennes. Les opérateurs logiques les plus couramment utilisés sont :

  • AND (&&) : renvoie true si les deux expressions sont vraies.
  • OR (||) : renvoie true si au moins l'une des expressions est vraie.
  • NOT (!) : renvoie l'inverse de l'expression.

Exemple :

a = true
b = false

result1 = a && b
result2 = a || b
result3 = !a

Structures de contrôle (conditions et boucles)

Les structures de contrôle permettent d'exécuter certaines parties du code en fonction de conditions spécifiques.

La structure de contrôle la plus couramment utilisée est la condition "if". Elle permet d'exécuter un bloc de code si une condition est vraie.

Exemple :

age = 18

if age >= 18
  puts "Vous êtes majeur."
else
  puts "Vous êtes mineur."
end

Les boucles permettent de répéter l'exécution d'un bloc de code plusieurs fois. La boucle "while" est utilisée lorsque vous souhaitez répéter une action tant qu'une condition est vraie.

Exemple :

i = 0

while i < 10
  puts "Iteration #{i}"
  i += 1
end

Fonctions et méthodes

Les fonctions et les méthodes sont des blocs de code réutilisables qui effectuent une tâche spécifique. Les fonctions sont définies en dehors des classes, tandis que les méthodes sont définies à l'intérieur des classes.

Exemple :

def say_hello(name)
  puts "Hello, #{name}!"
end

say_hello("John")

Les fonctions et les méthodes peuvent prendre des paramètres et renvoyer des valeurs.

Tableaux et itérations

Les tableaux sont utilisés pour stocker une collection ordonnée d'éléments. Vous pouvez accéder aux éléments d'un tableau en utilisant leur index.

Exemple :

numbers = [1, 2, 3, 4, 5]

puts numbers[0] # Affiche 1
puts numbers[2] # Affiche 3

Les itérations permettent de parcourir les éléments d'un tableau et d'effectuer une action pour chaque élément.

Exemple :

numbers = [1, 2, 3, 4, 5]

numbers.each do |number|
  puts number
end

Manipulation de chaînes de caractères

Les chaînes de caractères peuvent être manipulées de différentes manières en Ruby. Vous pouvez les concaténer, les découper, les transformer en majuscules ou en minuscules, etc.

Exemple :

name = "John"
age = 25

greeting = "Hello, my name is #{name} and I am #{age} years old."
uppercase_name = name.upcase
lowercase_name = name.downcase

Il existe de nombreuses autres fonctionnalités et concepts à apprendre en Ruby, mais cette section vous a donné une base solide pour commencer à programmer en Ruby.
