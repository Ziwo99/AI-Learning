Les conditions et les opérateurs de comparaison

Les conditions et les opérateurs de comparaison sont utilisés pour prendre des décisions dans un programme Ruby. Une condition est une expression qui est évaluée comme vraie ou fausse. Les opérateurs de comparaison permettent de comparer des valeurs et de déterminer si elles sont égales, supérieures, inférieures, etc.

Voici quelques exemples d'opérateurs de comparaison :

==   # égal à
!=   # différent de
>    # supérieur à
<    # inférieur à
>=   # supérieur ou égal à
<=   # inférieur ou égal à

Voici un exemple d'utilisation des conditions et des opérateurs de comparaison :

age = 25;

if age >= 18
  puts "Vous êtes majeur."
else
  puts "Vous êtes mineur."
end

Dans cet exemple, si l'âge est supérieur ou égal à 18, le programme affiche "Vous êtes majeur.". Sinon, il affiche "Vous êtes mineur."

Les boucles while et for

Les boucles while et for sont utilisées pour répéter une série d'instructions tant qu'une condition est vraie ou pour itérer sur une collection d'éléments.

Voici un exemple d'utilisation de la boucle while :

i = 0;

while i < 10
  puts "Iteration #{i}"
  i += 1
end

Dans cet exemple, la boucle while répète les instructions à l'intérieur tant que la condition "i < 10" est vraie. À chaque itération, la valeur de "i" est augmentée de 1.

Voici un exemple d'utilisation de la boucle for :

numbers = [1, 2, 3, 4, 5]

for number in numbers
  puts number
end

Dans cet exemple, la boucle for itère sur chaque élément de la collection "numbers" et affiche sa valeur.

Les instructions de contrôle de flux (if, else, elsif)

Les instructions de contrôle de flux permettent de prendre des décisions conditionnelles dans un programme Ruby. L'instruction "if" est utilisée pour exécuter un bloc de code si une condition est vraie. L'instruction "else" est utilisée pour exécuter un bloc de code si la condition est fausse. L'instruction "elsif" est utilisée pour tester une autre condition si la première condition est fausse.

Voici un exemple d'utilisation des instructions de contrôle de flux :

age = 25;

if age < 18
  puts "Vous êtes mineur."
elsif age < 21
  puts "Vous êtes majeur, mais vous ne pouvez pas boire d'alcool."
else
  puts "Vous êtes majeur."
end

Dans cet exemple, si l'âge est inférieur à 18, le programme affiche "Vous êtes mineur.". Si l'âge est compris entre 18 et 21, le programme affiche "Vous êtes majeur, mais vous ne pouvez pas boire d'alcool.". Sinon, il affiche "Vous êtes majeur."

Les boucles do-while et until

Les boucles do-while et until sont utilisées pour répéter une série d'instructions tant qu'une condition est vraie ou fausse.

Voici un exemple d'utilisation de la boucle do-while :

i = 0;

begin
  puts "Iteration #{i}"
  i += 1
end while i < 10

Dans cet exemple, la boucle do-while répète les instructions à l'intérieur tant que la condition "i < 10" est vraie. À chaque itération, la valeur de "i" est augmentée de 1. La différence avec la boucle while est que la condition est vérifiée après l'exécution du bloc de code.

Voici un exemple d'utilisation de la boucle until :

i = 0;

until i >= 10
  puts "Iteration #{i}"
  i += 1
end

Dans cet exemple, la boucle until répète les instructions à l'intérieur tant que la condition "i >= 10" est fausse. À chaque itération, la valeur de "i" est augmentée de 1. La différence avec la boucle while est que la condition est vérifiée avant l'exécution du bloc de code.
