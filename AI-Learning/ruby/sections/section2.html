Section 2 : Les bases de Ruby

Variables et types de données

Les variables sont utilisées pour stocker des valeurs dans un programme Ruby. Elles peuvent contenir différents types de données tels que des entiers, des chaînes de caractères, des booléens, etc.

Voici un exemple de déclaration de variable en Ruby :

name = "John";
age = 25;
isMale = true;

Les variables peuvent être utilisées pour stocker des valeurs temporaires ou pour effectuer des calculs. Elles peuvent également être modifiées tout au long de l'exécution du programme.

Opérateurs

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs en Ruby. Il existe différents types d'opérateurs tels que les opérateurs arithmétiques, les opérateurs de comparaison et les opérateurs logiques.

Voici quelques exemples d'opérateurs en Ruby :

addition = 5 + 3;
soustraction = 10 - 2;
multiplication = 4 * 6;
division = 12 / 3;
estEgal = 5 == 5;

Les opérateurs peuvent être utilisés pour effectuer des calculs mathématiques, comparer des valeurs ou combiner des conditions logiques.

Structures de contrôle (conditions et boucles)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme Ruby. Elles comprennent les conditions (if/else) et les boucles (while, for).

Voici un exemple d'utilisation d'une structure de contrôle en Ruby :

if age >= 18
  puts "Vous êtes majeur."
else
  puts "Vous êtes mineur."
end

Ce code vérifie si la variable "age" est supérieure ou égale à 18 et affiche un message en conséquence.

Fonctions et méthodes

Les fonctions et les méthodes sont utilisées pour regrouper des instructions et les réutiliser dans un programme Ruby. Elles permettent d'organiser le code et d'améliorer sa lisibilité.

Voici un exemple de déclaration d'une fonction en Ruby :

def say_hello(name)
  puts "Bonjour, " + name + "!"
end

say_hello("Alice");

Ce code définit une fonction "say_hello" qui prend un paramètre "name" et affiche un message de salutation personnalisé.

Tableaux et itérations

Les tableaux sont utilisés pour stocker plusieurs valeurs dans une seule variable en Ruby. Ils peuvent contenir des éléments de différents types de données.

Voici un exemple de déclaration d'un tableau en Ruby :

numbers = [1, 2, 3, 4, 5];

numbers.each do |number|
  puts number;
end

Ce code crée un tableau "numbers" contenant les nombres de 1 à 5, puis itère sur chaque élément du tableau et l'affiche.

Manipulation de chaînes de caractères

La manipulation de chaînes de caractères est courante en programmation Ruby. Elle permet de concaténer des chaînes, d'extraire des sous-chaînes, de remplacer des caractères, etc.

Voici un exemple de manipulation de chaînes de caractères en Ruby :

message = "Hello, World!";
puts message.length;
puts message.upcase;
puts message.gsub("Hello", "Hi");

Ce code affiche la longueur de la chaîne de caractères, la version en majuscules et une version modifiée de la chaîne de caractères.
