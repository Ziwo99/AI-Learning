L'accès aux bases de données avec Ruby

En utilisant Ruby, il est possible d'accéder et de manipuler des bases de données de manière efficace. Ruby offre plusieurs bibliothèques et frameworks pour interagir avec les bases de données, tels que ActiveRecord, Sequel, et DataMapper.

Pour accéder à une base de données en Ruby, il est nécessaire d'installer le pilote approprié pour la base de données que vous souhaitez utiliser. Par exemple, pour MySQL, vous pouvez installer le pilote mysql2 en utilisant la commande suivante :

gem install mysql2

Une fois le pilote installé, vous pouvez établir une connexion à la base de données en utilisant la classe correspondante du pilote. Par exemple, pour MySQL, vous pouvez utiliser la classe Mysql2::Client :

require 'mysql2'

client = Mysql2::Client.new(
  host: 'localhost',
  username: 'root',
  password: 'password',
  database: 'my_database'
)

Une fois la connexion établie, vous pouvez exécuter des requêtes SQL sur la base de données en utilisant les méthodes fournies par le pilote. Par exemple, pour exécuter une requête SELECT :

results = client.query("SELECT * FROM users")

results.each do |row|
  puts row['name']
end

Cela exécutera la requête SELECT et affichera le nom de chaque utilisateur dans la table "users".

Les requêtes SQL en Ruby

Les requêtes SQL sont utilisées pour interagir avec les bases de données et récupérer, insérer, mettre à jour ou supprimer des données. En Ruby, vous pouvez exécuter des requêtes SQL en utilisant les méthodes fournies par le pilote de base de données que vous utilisez.

Voici quelques exemples de requêtes SQL couramment utilisées en Ruby :

  • SELECT : utilisée pour récupérer des données à partir d'une table.
  • INSERT : utilisée pour insérer de nouvelles données dans une table.
  • UPDATE : utilisée pour mettre à jour des données existantes dans une table.
  • DELETE : utilisée pour supprimer des données d'une table.

Voici un exemple d'utilisation d'une requête SELECT en Ruby :

results = client.query("SELECT * FROM users WHERE age > 18")

results.each do |row|
  puts row['name']
end

Cela exécutera la requête SELECT et affichera le nom de chaque utilisateur dont l'âge est supérieur à 18.

La manipulation des données avec Ruby

En utilisant Ruby, vous pouvez facilement manipuler les données dans une base de données. Vous pouvez insérer de nouvelles données, mettre à jour des données existantes et supprimer des données.

Voici quelques exemples de manipulation de données couramment utilisés en Ruby :

  • INSERT : utilisée pour insérer de nouvelles données dans une table.
  • UPDATE : utilisée pour mettre à jour des données existantes dans une table.
  • DELETE : utilisée pour supprimer des données d'une table.

Voici un exemple d'utilisation d'une requête INSERT en Ruby :

client.query("INSERT INTO users (name, age) VALUES ('John Doe', 25)")

Cela insérera un nouvel utilisateur avec le nom "John Doe" et l'âge de 25 dans la table "users".

Voici un exemple d'utilisation d'une requête UPDATE en Ruby :

client.query("UPDATE users SET age = 30 WHERE name = 'John Doe'")

Cela mettra à jour l'âge de l'utilisateur dont le nom est "John Doe" à 30 dans la table "users".

Voici un exemple d'utilisation d'une requête DELETE en Ruby :

client.query("DELETE FROM users WHERE age > 30")

Cela supprimera tous les utilisateurs dont l'âge est supérieur à 30 de la table "users".

Il est important de noter que l'utilisation de requêtes SQL directement dans votre code peut présenter des risques de sécurité, tels que les attaques par injection SQL. Il est recommandé d'utiliser des requêtes préparées ou des ORM (Object-Relational Mapping) pour éviter ces problèmes.
