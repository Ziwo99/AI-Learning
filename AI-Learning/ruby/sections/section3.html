Les fonctions prédéfinies en Ruby

Les fonctions prédéfinies en Ruby sont des morceaux de code réutilisables qui effectuent des tâches spécifiques. Elles sont déjà incluses dans le langage Ruby et peuvent être utilisées directement sans avoir à les définir.

Voici quelques exemples de fonctions prédéfinies en Ruby :

puts "Hello, World!"

Cette fonction affiche le texte "Hello, World!" dans la console.

rand(1..10)

Cette fonction génère un nombre aléatoire entre 1 et 10.

Math.sqrt(25)

Cette fonction calcule la racine carrée de 25.

Les méthodes de classe et les méthodes d'instance

En Ruby, les méthodes sont des morceaux de code réutilisables qui effectuent des actions spécifiques. Elles peuvent être définies à l'intérieur d'une classe et être appelées sur des objets de cette classe.

Les méthodes de classe sont des méthodes qui peuvent être appelées directement sur la classe elle-même, sans avoir besoin d'instancier un objet. Elles sont définies en utilisant le mot-clé "self" suivi d'un point et du nom de la méthode.

class Person
  def self.greet
    puts "Hello, I am a person!"
  end
end

Person.greet

Cette méthode de classe "greet" affiche le message "Hello, I am a person!" lorsque la méthode est appelée sur la classe Person.

Les méthodes d'instance sont des méthodes qui peuvent être appelées sur des objets d'une classe spécifique. Elles sont définies à l'intérieur de la classe sans utiliser le mot-clé "self".

class Person
  def greet
    puts "Hello, I am a person!"
  end
end

person = Person.new
person.greet

Cette méthode d'instance "greet" affiche le message "Hello, I am a person!" lorsque la méthode est appelée sur un objet de la classe Person.

Les paramètres et les arguments des fonctions

Les paramètres sont des variables utilisées dans une fonction pour recevoir des valeurs lors de l'appel de la fonction. Les arguments sont les valeurs réelles passées à la fonction lors de son appel.

Voici un exemple de fonction avec des paramètres :

def greet(name)
  puts "Hello, #{name}!"
end

greet("John")

Cette fonction "greet" prend un paramètre "name" et affiche le message "Hello, John!" lorsque la fonction est appelée avec l'argument "John".

Les blocs et les lambdas

Les blocs et les lambdas sont des morceaux de code qui peuvent être passés en tant qu'arguments à une méthode. Ils permettent de créer des fonctions anonymes ou des portions de code réutilisables.

Voici un exemple d'utilisation d'un bloc :

3.times do |i|
  puts "Iteration #{i}"
end

Ce bloc est passé à la méthode "times" et affiche le message "Iteration 0", "Iteration 1" et "Iteration 2" lors de son exécution.

Voici un exemple d'utilisation d'une lambda :

add = lambda { |a, b| a + b }
puts add.call(2, 3)

Cette lambda "add" prend deux arguments "a" et "b" et renvoie leur somme lorsqu'elle est appelée avec les arguments 2 et 3.
