Les tableaux et les listes en Ruby

Les tableaux et les listes sont des structures de données couramment utilisées en Ruby pour stocker et manipuler des collections d'éléments. Un tableau est une séquence ordonnée d'éléments, tandis qu'une liste est une collection d'éléments qui peut être modifiée en ajoutant, en supprimant ou en modifiant des éléments.

Voici comment déclarer et initialiser un tableau en Ruby :

array = [1, 2, 3, 4, 5]

On peut également créer une liste vide et ajouter des éléments par la suite :

list = []
list.push(1)
list.push(2)
list.push(3)

Pour accéder aux éléments d'un tableau ou d'une liste, on utilise des indices. L'indice du premier élément est 0, celui du deuxième est 1, et ainsi de suite. Voici comment accéder à un élément spécifique :

puts array[0] # Affiche 1
puts list[1] # Affiche 2

Il est également possible de modifier un élément en utilisant son indice :

array[2] = 10
list[0] = 5

Pour parcourir tous les éléments d'un tableau ou d'une liste, on peut utiliser une boucle for :

for i in 0..array.length-1
  puts array[i]
end

On peut également utiliser la méthode each pour itérer sur les éléments :

array.each do |element|
  puts element
end

Les hashages et les dictionnaires

Un hashage, également appelé dictionnaire, est une structure de données qui associe des clés à des valeurs. Chaque clé doit être unique et permet d'accéder à sa valeur correspondante.

Voici comment déclarer et initialiser un hashage en Ruby :

hash = { "nom" => "John", "age" => 30, "ville" => "Paris" }

Pour accéder à une valeur dans un hashage, on utilise la clé correspondante :

puts hash["nom"] # Affiche "John"
puts hash["age"] # Affiche 30

On peut également ajouter, modifier ou supprimer des paires clé-valeur dans un hashage :

hash["profession"] = "Développeur"
hash["age"] = 31
hash.delete("ville")

Pour parcourir toutes les paires clé-valeur d'un hashage, on peut utiliser la méthode each :

hash.each do |key, value|
  puts "#{key}: #{value}"
end

Les ensembles et les piles

En Ruby, les ensembles et les piles peuvent être implémentés à l'aide de la classe Set et de la classe Array, respectivement.

Un ensemble est une collection d'éléments uniques, sans ordre particulier. Voici comment créer un ensemble :

require 'set'

set = Set.new
set.add(1)
set.add(2)
set.add(3)

Pour vérifier si un élément est présent dans un ensemble, on peut utiliser la méthode include? :

puts set.include?(2) # Affiche true
puts set.include?(4) # Affiche false

Une pile, également appelée stack, est une structure de données dans laquelle les éléments sont ajoutés et supprimés selon le principe du dernier entré, premier sorti (LIFO). Voici comment créer une pile :

stack = []
stack.push(1)
stack.push(2)
stack.push(3)

Pour retirer le dernier élément ajouté à la pile, on utilise la méthode pop :

puts stack.pop # Affiche 3
puts stack.pop # Affiche 2

Les itérations sur les collections

En Ruby, il existe plusieurs méthodes pour itérer sur les éléments d'une collection, telles que each, map, select, etc.

La méthode each permet de parcourir tous les éléments d'une collection :

array.each do |element|
  puts element
end

hash.each do |key, value|
  puts "#{key}: #{value}"
end

set.each do |element|
  puts element
end

stack.each do |element|
  puts element
end

La méthode map permet de transformer chaque élément d'une collection en un autre élément :

new_array = array.map do |element|
  element * 2
end

new_hash = hash.map do |key, value|
  [key.to_sym, value]
end

La méthode select permet de filtrer les éléments d'une collection en fonction d'une condition :

even_numbers = array.select do |element|
  element % 2 == 0
end

adults = hash.select do |key, value|
  key == "age" && value >= 18
end

Ces méthodes d'itération sont très utiles pour manipuler et transformer les collections de données en Ruby.
