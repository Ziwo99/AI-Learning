La lecture et l'écriture de fichiers en Ruby

La lecture et l'écriture de fichiers sont des opérations courantes lors du développement d'applications en Ruby. Ruby offre des fonctionnalités puissantes pour manipuler des fichiers texte et binaire.

Pour lire le contenu d'un fichier en Ruby, nous utilisons la méthode File.open en spécifiant le nom du fichier et le mode d'ouverture. Voici un exemple :

file = File.open("mon_fichier.txt", "r")
content = file.read
file.close

puts content

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode lecture ("r"), nous lisons son contenu avec la méthode read et nous fermons le fichier avec la méthode close. Ensuite, nous affichons le contenu à l'aide de la méthode puts.

Pour écrire dans un fichier en Ruby, nous utilisons également la méthode File.open, mais cette fois-ci en spécifiant le mode d'écriture. Voici un exemple :

file = File.open("mon_fichier.txt", "w")
file.write("Contenu à écrire dans le fichier")
file.close

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode écriture ("w"), nous écrivons le contenu avec la méthode write et nous fermons le fichier avec la méthode close.

Les flux d'entrée et de sortie standard

En plus de la lecture et de l'écriture de fichiers, Ruby offre également des fonctionnalités pour interagir avec les flux d'entrée et de sortie standard. Cela permet de lire les entrées de l'utilisateur depuis le clavier et d'afficher des messages à l'utilisateur.

Pour lire une entrée de l'utilisateur, nous utilisons la méthode gets. Voici un exemple :

puts "Quel est votre nom ?"
nom = gets.chomp

puts "Bonjour, #{nom} !"

Dans cet exemple, nous affichons le message "Quel est votre nom ?" à l'utilisateur, puis nous utilisons la méthode gets pour lire l'entrée de l'utilisateur. La méthode chomp est utilisée pour supprimer le caractère de nouvelle ligne à la fin de l'entrée. Enfin, nous affichons un message de salutation personnalisé en utilisant l'entrée de l'utilisateur.

Pour afficher un message à l'utilisateur, nous utilisons la méthode puts. Voici un exemple :

puts "Bonjour, monde !"

Dans cet exemple, nous affichons le message "Bonjour, monde !" à l'utilisateur.

La sérialisation et la désérialisation d'objets

La sérialisation et la désérialisation d'objets sont des opérations permettant de convertir des objets en une représentation pouvant être stockée ou transmise, puis de les reconvertir en objets. Ruby offre des fonctionnalités intégrées pour effectuer ces opérations.

Pour sérialiser un objet en Ruby, nous utilisons la méthode Marshal.dump. Voici un exemple :

objet = { "nom" => "John", "age" => 30 }
serialized_objet = Marshal.dump(objet)

puts serialized_objet

Dans cet exemple, nous sérialisons l'objet de type hash en utilisant la méthode Marshal.dump. Le résultat sérialisé est stocké dans la variable serialized_objet. Ensuite, nous affichons le résultat sérialisé à l'aide de la méthode puts.

Pour désérialiser un objet en Ruby, nous utilisons la méthode Marshal.load. Voici un exemple :

serialized_objet = "\x04\b{\x06\"\bnom\"\b\"\vJohn\"\a\"\bage\"\x06\x00\x00\x00\x00\x00\x00\x1E"
objet = Marshal.load(serialized_objet)

puts objet["nom"]
puts objet["age"]

Dans cet exemple, nous désérialisons l'objet sérialisé en utilisant la méthode Marshal.load. Le résultat désérialisé est stocké dans la variable objet. Ensuite, nous affichons les valeurs des clés "nom" et "age" de l'objet désérialisé à l'aide de la méthode puts.

Il est important de noter que la sérialisation et la désérialisation d'objets avec Marshal ne sont pas compatibles entre différentes versions de Ruby. Si vous avez besoin de compatibilité entre différentes versions, vous devriez envisager d'utiliser d'autres formats de sérialisation, tels que JSON ou YAML.
