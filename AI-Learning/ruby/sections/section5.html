Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en programmation Ruby. Ruby offre plusieurs méthodes pour effectuer ces opérations de manière efficace.

Pour lire le contenu d'un fichier, nous utilisons la méthode File.open avec le mode de lecture approprié. Par exemple, pour lire un fichier en mode lecture seule :

file = File.open("mon_fichier.txt", "r")
content = file.read
file.close

puts content

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode lecture seule avec le paramètre "r". Ensuite, nous utilisons la méthode read pour lire le contenu du fichier et le stocker dans la variable content. Enfin, nous fermons le fichier avec la méthode close et affichons le contenu à l'aide de la méthode puts.

Pour écrire dans un fichier, nous utilisons la méthode File.open avec le mode d'écriture approprié. Par exemple, pour écrire du texte dans un fichier :

file = File.open("mon_fichier.txt", "w")
file.write("Contenu à écrire dans le fichier")
file.close

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode écriture avec le paramètre "w". Ensuite, nous utilisons la méthode write pour écrire le contenu dans le fichier. Enfin, nous fermons le fichier avec la méthode close.

Manipulation de dossiers et de chemins de fichiers

En plus de la lecture et de l'écriture de fichiers, Ruby offre également des fonctionnalités pour manipuler les dossiers et les chemins de fichiers.

Pour créer un nouveau dossier, nous utilisons la méthode Dir.mkdir. Par exemple, pour créer un dossier nommé "nouveau_dossier" :

Dir.mkdir("nouveau_dossier")

Pour supprimer un dossier, nous utilisons la méthode Dir.delete. Par exemple, pour supprimer le dossier "ancien_dossier" :

Dir.delete("ancien_dossier")

Pour vérifier si un dossier existe, nous utilisons la méthode Dir.exist?. Par exemple, pour vérifier si le dossier "mon_dossier" existe :

if Dir.exist?("mon_dossier")
  puts "Le dossier existe"
else
  puts "Le dossier n'existe pas"
end

Pour obtenir le chemin absolu d'un fichier ou d'un dossier, nous utilisons la méthode File.absolute_path. Par exemple, pour obtenir le chemin absolu du fichier "mon_fichier.txt" :

absolute_path = File.absolute_path("mon_fichier.txt")
puts absolute_path

Gestion des erreurs liées aux fichiers

Lors de la manipulation de fichiers, il est important de gérer les erreurs qui peuvent survenir. Ruby offre des mécanismes pour capturer et gérer ces erreurs.

Une façon de gérer les erreurs est d'utiliser des blocs begin-rescue-ensure. Par exemple, pour gérer une erreur lors de la lecture d'un fichier :

begin
  file = File.open("mon_fichier.txt", "r")
  content = file.read
rescue
  puts "Une erreur s'est produite lors de la lecture du fichier"
ensure
  file.close if file
end

Dans cet exemple, nous utilisons le bloc begin-rescue-ensure pour capturer toute erreur qui se produit lors de la lecture du fichier. Si une erreur se produit, le code dans le bloc rescue est exécuté. Le bloc ensure est utilisé pour s'assurer que le fichier est fermé, même en cas d'erreur.

Il est également possible de créer ses propres exceptions personnalisées en utilisant la classe StandardError. Par exemple, pour créer une exception personnalisée pour une erreur de fichier :

class FileError < StandardError
end

begin
  file = File.open("mon_fichier.txt", "r")
  raise FileError, "Une erreur s'est produite lors de la lecture du fichier" if file.nil?
  content = file.read
rescue FileError => e
  puts e.message
ensure
  file.close if file
end

Dans cet exemple, nous créons une classe FileError qui hérite de la classe StandardError. Ensuite, nous utilisons la méthode raise pour lever une exception de type FileError si le fichier est nul. Dans le bloc rescue, nous affichons le message d'erreur de l'exception.
