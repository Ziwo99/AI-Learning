Section 2 : Mise en page avec CSS

La boîte CSS

La boîte CSS est un concept fondamental en CSS qui permet de définir la mise en page et le positionnement des éléments HTML. Chaque élément HTML est considéré comme une boîte avec des propriétés telles que la largeur, la hauteur, les marges, les bordures et le remplissage.

Voici un exemple de code CSS pour définir les propriétés de la boîte :

.ma-boite {
    width: 200px;
    height: 100px;
    margin: 10px;
    border: 1px solid black;
    padding: 20px;
}

Dans cet exemple, la classe "ma-boite" est appliquée à un élément HTML. La boîte aura une largeur de 200 pixels, une hauteur de 100 pixels, une marge de 10 pixels, une bordure de 1 pixel solide et un remplissage de 20 pixels.

Le positionnement CSS

Le positionnement CSS permet de contrôler la façon dont les éléments HTML sont positionnés les uns par rapport aux autres sur la page. Il existe plusieurs méthodes de positionnement, telles que le positionnement statique, le positionnement relatif, le positionnement absolu et le positionnement fixe.

Voici un exemple de code CSS pour définir le positionnement d'un élément :

.mon-element {
    position: relative;
    top: 50px;
    left: 100px;
}

Dans cet exemple, la classe "mon-element" est appliquée à un élément HTML. L'élément sera positionné de manière relative par rapport à sa position normale, avec un décalage de 50 pixels vers le bas et de 100 pixels vers la gauche.

Les modèles de boîtes CSS

Les modèles de boîtes CSS permettent de contrôler la façon dont les éléments HTML sont affichés à l'intérieur de leur boîte. Il existe plusieurs propriétés de modèle de boîte, telles que la largeur, la hauteur, les marges, les bordures et le remplissage.

Voici un exemple de code CSS pour définir les propriétés de modèle de boîte :

.ma-boite {
    width: 200px;
    height: 100px;
    margin: 10px;
    border: 1px solid black;
    padding: 20px;
    box-sizing: border-box;
}

Dans cet exemple, la classe "ma-boite" est appliquée à un élément HTML. La boîte aura une largeur de 200 pixels, une hauteur de 100 pixels, une marge de 10 pixels, une bordure de 1 pixel solide et un remplissage de 20 pixels. La propriété "box-sizing: border-box;" permet de spécifier que la largeur et la hauteur incluent la bordure et le remplissage.

Les grilles CSS

Les grilles CSS permettent de créer des mises en page complexes et réactives en utilisant une grille de lignes et de colonnes. Les grilles CSS offrent un moyen flexible de positionner les éléments HTML sur la page.

Voici un exemple de code CSS pour créer une grille :

.ma-grille {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: auto;
    gap: 10px;
}

Dans cet exemple, la classe "ma-grille" est appliquée à un conteneur HTML. La grille aura trois colonnes de largeur égale, des lignes de hauteur automatique et un espacement de 10 pixels entre les éléments.

Ces concepts de mise en page avec CSS sont essentiels pour créer des designs attrayants et fonctionnels pour les sites web. En comprenant et en maîtrisant ces concepts, vous serez en mesure de créer des mises en page personnalisées et adaptées à vos besoins.
