Section 11 : Approfondissement du CSS

Les pseudo-classes et pseudo-éléments

Les pseudo-classes et pseudo-éléments sont des sélecteurs CSS spéciaux qui permettent de cibler des éléments dans des états spécifiques ou d'ajouter du contenu supplémentaire à un élément. Ils sont très utiles pour appliquer des styles spécifiques à des éléments lorsqu'ils se trouvent dans certaines conditions.

Les pseudo-classes sont utilisées pour cibler des éléments dans des états spécifiques, tels que :hover pour cibler un élément lorsque le curseur est dessus, :active pour cibler un élément lorsqu'il est activé, ou encore :first-child pour cibler le premier enfant d'un élément parent.

Les pseudo-éléments, quant à eux, permettent d'ajouter du contenu supplémentaire à un élément. Par exemple, ::before et ::after permettent d'ajouter du contenu avant et après un élément respectivement. Ils sont souvent utilisés pour créer des effets visuels tels que des flèches ou des icônes.

p::first-child {
    color: red;
}

a:hover {
    text-decoration: underline;
}

button:active {
    background-color: gray;
}

p::before {
    content: "Avant";
}

p::after {
    content: "Après";
}

Les variables CSS

Les variables CSS, également appelées propriétés personnalisées, permettent de stocker des valeurs réutilisables dans une feuille de style. Elles sont déclarées à l'aide de la syntaxe --nom-variable: valeur; et peuvent être utilisées dans n'importe quelle propriété CSS en utilisant la syntaxe var(--nom-variable).

Les variables CSS sont très pratiques pour définir des valeurs communes à plusieurs éléments, ce qui facilite la maintenance du code. Elles peuvent également être modifiées dynamiquement à l'aide de JavaScript, ce qui permet de créer des thèmes personnalisables.

:root {
    --couleur-principale: #ff0000;
}

h1 {
    color: var(--couleur-principale);
}

p {
    background-color: var(--couleur-principale);
}

Les fonctions CSS

Les fonctions CSS sont des outils puissants qui permettent de manipuler les valeurs des propriétés CSS. Elles peuvent être utilisées pour effectuer des calculs, des transformations, des ajustements de couleurs, etc.

Il existe de nombreuses fonctions CSS disponibles, telles que calc() pour effectuer des calculs mathématiques, transform() pour appliquer des transformations 2D ou 3D à un élément, et rgba() pour spécifier une couleur avec une transparence.

p {
    width: calc(100% - 20px);
}

img {
    transform: rotate(45deg);
}

div {
    background-color: rgba(255, 0, 0, 0.5);
}

Les modules CSS avancés (Grid, Flexbox, etc.)

Les modules CSS avancés, tels que Grid et Flexbox, sont des outils puissants pour créer des mises en page complexes et réactives.

Flexbox est un module CSS qui permet de créer des mises en page flexibles et réactives. Il utilise un système de conteneurs et d'éléments flexibles pour aligner et distribuer l'espace disponible de manière optimale.

Grid est un autre module CSS qui permet de créer des mises en page en grille. Il utilise un système de lignes et de colonnes pour organiser les éléments de manière précise et flexible.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

.item {
    flex: 1;
    margin: 10px;
}

.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 10px;
}

.grid-item {
    grid-column: span 2;
}

Avec ces modules CSS avancés, il est possible de créer des mises en page complexes et réactives avec une grande flexibilité.
