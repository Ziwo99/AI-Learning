Section 1: Introduction à CSS

Les bases de CSS

CSS (Cascading Style Sheets) est un langage utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler l'apparence des éléments HTML tels que le texte, les images, les tableaux, etc. CSS utilise des règles pour appliquer des styles aux éléments HTML.

Les règles CSS sont composées de deux parties : un sélecteur et une déclaration. Le sélecteur indique quel élément HTML sera stylé, tandis que la déclaration spécifie les styles à appliquer. Voici un exemple de règle CSS : 

p {
    color: blue;
    font-size: 16px;
}

Dans cet exemple, le sélecteur est "p" qui cible tous les éléments de paragraphe. La déclaration spécifie que le texte doit être de couleur bleue et avoir une taille de police de 16 pixels.

Les sélecteurs CSS

Les sélecteurs CSS permettent de cibler spécifiquement certains éléments HTML pour leur appliquer des styles. Il existe différents types de sélecteurs CSS, tels que les sélecteurs d'éléments, les sélecteurs de classe, les sélecteurs d'ID, etc.

Voici quelques exemples de sélecteurs CSS couramment utilisés : 

/* Sélecteur d'élément */
p {
    color: blue;
}

/* Sélecteur de classe */
.my-class {
    font-weight: bold;
}

/* Sélecteur d'ID */
#my-id {
    text-decoration: underline;
}

Dans cet exemple, le sélecteur d'élément cible tous les éléments de paragraphe, le sélecteur de classe cible tous les éléments ayant la classe "my-class" et le sélecteur d'ID cible l'élément ayant l'ID "my-id".

Les propriétés CSS

Les propriétés CSS définissent les styles à appliquer aux éléments HTML. Il existe de nombreuses propriétés CSS disponibles, telles que la couleur, la taille de police, la marge, le padding, etc.

Voici quelques exemples de propriétés CSS couramment utilisées : 

/* Propriété de couleur */
color: blue;

/* Propriété de taille de police */
font-size: 16px;

/* Propriété de marge */
margin: 10px;

/* Propriété de padding */
padding: 5px;

Dans cet exemple, la propriété de couleur définit la couleur du texte en bleu, la propriété de taille de police définit la taille de police à 16 pixels, la propriété de marge définit une marge de 10 pixels et la propriété de padding définit un padding de 5 pixels.

Les valeurs CSS

Les valeurs CSS sont utilisées pour définir les différentes options disponibles pour les propriétés CSS. Par exemple, pour la propriété de couleur, les valeurs peuvent être des noms de couleurs prédéfinis tels que "blue", des codes hexadécimaux tels que "#0000FF" ou des valeurs RGB telles que "rgb(0, 0, 255)".

Voici quelques exemples de valeurs CSS couramment utilisées : 

/* Valeur de couleur */
color: blue;

/* Valeur de taille de police */
font-size: 16px;

/* Valeur de marge */
margin: 10px;

/* Valeur de padding */
padding: 5px;

Dans cet exemple, la valeur de couleur est "blue", la valeur de taille de police est "16px", la valeur de marge est "10px" et la valeur de padding est "5px".
