Section 7 : Les transitions et les animations CSS

Les transitions CSS

Les transitions CSS permettent d'animer les changements de propriétés d'un élément HTML de manière fluide. Elles offrent une manière élégante de créer des effets de transition entre les états d'un élément.

Pour définir une transition, il faut utiliser la propriété transition suivie des propriétés que l'on souhaite animer, ainsi que la durée et le type de transition.

Voici un exemple de code CSS pour créer une transition sur la couleur de fond d'un élément :

.element {
  background-color: blue;
  transition: background-color 1s ease;
}

.element:hover {
  background-color: red;
}

Dans cet exemple, lorsque l'utilisateur survole l'élément avec la classe "element", la couleur de fond passe de bleu à rouge en une seconde avec une transition de type "ease".

Les animations CSS

Les animations CSS permettent de créer des mouvements et des effets plus complexes que les transitions. Elles permettent de définir des étapes clés (keyframes) pour contrôler précisément le comportement de l'animation.

Pour créer une animation, il faut utiliser la règle @keyframes pour définir les différentes étapes de l'animation, puis utiliser la propriété animation pour spécifier la durée, le type de timing et les étapes clés de l'animation.

Voici un exemple de code CSS pour créer une animation de rotation sur un élément :

@keyframes rotation {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}

.element {
  animation: rotation 2s linear infinite;
}

Dans cet exemple, l'élément avec la classe "element" effectue une rotation complète sur lui-même en deux secondes, de manière linéaire et en boucle infinie.

Les transformations CSS

Les transformations CSS permettent de modifier la position, la taille, la rotation et l'échelle d'un élément. Elles offrent un moyen puissant de manipuler les éléments HTML sans avoir à modifier leur structure.

Pour appliquer une transformation, il faut utiliser la propriété transform suivie de la transformation souhaitée.

Voici quelques exemples de transformations couramment utilisées :

.element {
  transform: translate(50px, 50px); /* Déplace l'élément de 50 pixels vers la droite et le bas */
  transform: rotate(45deg); /* Fait tourner l'élément de 45 degrés */
  transform: scale(2); /* Double la taille de l'élément */
}

Dans ces exemples, l'élément avec la classe "element" subit une translation, une rotation ou un agrandissement selon la transformation spécifiée.

Il est également possible de combiner plusieurs transformations en les séparant par des espaces :

.element {
  transform: translate(50px, 50px) rotate(45deg) scale(2);
}

Dans cet exemple, l'élément subit une translation, une rotation et un agrandissement simultanément.
