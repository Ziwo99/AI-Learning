Section 4 : Les mises en page CSS

Les mises en page en bloc

Les mises en page en bloc sont utilisées pour structurer le contenu en utilisant des éléments de bloc tels que les divs. Ces éléments occupent toute la largeur disponible et commencent sur une nouvelle ligne. Ils peuvent être stylisés à l'aide de propriétés CSS telles que la largeur, la hauteur, les marges, les rembourrages, etc. Voici un exemple de code CSS pour une mise en page en bloc :

.bloc {
  width: 300px;
  height: 200px;
  margin: 20px;
  padding: 10px;
  background-color: #f2f2f2;
}

Les mises en page en ligne

Les mises en page en ligne sont utilisées pour aligner les éléments horizontalement sur la même ligne. Les éléments en ligne ne peuvent pas avoir de largeur ou de hauteur définies, mais ils peuvent être stylisés avec des marges, des rembourrages, des bordures, etc. Voici un exemple de code CSS pour une mise en page en ligne :

.element {
  display: inline;
  margin: 10px;
  padding: 5px;
  border: 1px solid #ccc;
}

Les mises en page en tableau

Les mises en page en tableau sont utilisées pour organiser le contenu dans une structure de tableau. Elles sont utiles lorsque vous avez besoin d'afficher des données tabulaires. Les éléments de tableau tels que les tr, td et th peuvent être stylisés avec des propriétés CSS telles que la largeur, la hauteur, les marges, les rembourrages, etc. Voici un exemple de code CSS pour une mise en page en tableau :

table {
  width: 100%;
  border-collapse: collapse;
}

th, td {
  padding: 10px;
  border: 1px solid #ccc;
}

Les mises en page flexbox

Les mises en page flexbox sont utilisées pour créer des mises en page flexibles et réactives. Elles permettent de distribuer l'espace disponible entre les éléments de manière équitable et de les aligner horizontalement ou verticalement. Les éléments flexbox peuvent être stylisés avec des propriétés CSS telles que flex-direction, justify-content, align-items, etc. Voici un exemple de code CSS pour une mise en page flexbox :

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

Les mises en page en grille

Les mises en page en grille sont utilisées pour créer des mises en page complexes et structurées en utilisant une grille de colonnes et de lignes. Elles permettent de positionner les éléments de manière précise en utilisant des propriétés CSS telles que grid-template-columns, grid-template-rows, grid-column, grid-row, etc. Voici un exemple de code CSS pour une mise en page en grille :

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: auto;
  gap: 10px;
}

.grid-item {
  grid-column: span 2;
  grid-row: 1;
}
