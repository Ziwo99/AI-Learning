Section 2 : Les bases du CSS

Syntaxe du CSS

Le CSS utilise une syntaxe spécifique pour définir les styles des éléments HTML. Chaque règle CSS est composée d'un sélecteur et d'un bloc de déclarations.

Le sélecteur indique les éléments HTML auxquels les styles doivent être appliqués. Il peut s'agir d'un nom de balise, d'une classe, d'un ID ou d'un sélecteur de type.

Le bloc de déclarations est entouré par des accolades et contient les propriétés CSS et leurs valeurs. Chaque déclaration est composée d'une propriété suivie de deux-points et d'une valeur.

Voici un exemple de règle CSS :

p {
  color: blue;
  font-size: 16px;
}

Les sélecteurs CSS

Les sélecteurs CSS permettent de cibler spécifiquement les éléments HTML auxquels appliquer les styles. Il existe plusieurs types de sélecteurs :

  • Les sélecteurs de balise : ciblent tous les éléments d'un type spécifique, par exemple p pour les paragraphes.
  • Les sélecteurs de classe : ciblent les éléments qui ont une classe spécifique, par exemple .ma-classe.
  • Les sélecteurs d'ID : ciblent un élément avec un ID spécifique, par exemple #mon-id.
  • Les sélecteurs de descendant : ciblent les éléments qui sont descendants d'un autre élément, par exemple div p pour les paragraphes à l'intérieur d'une balise div.
  • Les sélecteurs d'attribut : ciblent les éléments qui ont un attribut spécifique, par exemple a[href] pour les liens avec un attribut href.

Les propriétés CSS

Les propriétés CSS définissent les styles des éléments HTML. Il existe de nombreuses propriétés CSS, voici quelques exemples :

  • color : définit la couleur du texte.
  • font-size : définit la taille de la police.
  • background-color : définit la couleur de l'arrière-plan.
  • border : définit les bordures d'un élément.
  • margin : définit les marges autour d'un élément.

Les valeurs CSS

Les valeurs CSS sont utilisées pour définir les propriétés CSS. Elles peuvent être des couleurs, des tailles, des positions, etc. Voici quelques exemples de valeurs CSS :

  • blue : une couleur prédéfinie.
  • 16px : une taille de police en pixels.
  • center : une position centrée.
  • solid : un style de bordure solide.
  • 10px 20px : des marges de 10 pixels en haut et en bas, et de 20 pixels à gauche et à droite.

Ces exemples ne sont qu'un aperçu des bases du CSS. Dans la suite de cette formation, nous explorerons en détail chaque aspect du CSS pour vous permettre de créer des styles avancés et professionnels pour vos sites web.
