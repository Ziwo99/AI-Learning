Section 8 : Les médias CSS

Les médias queries

Les médias queries sont des outils puissants en CSS qui permettent de modifier le style d'une page en fonction des caractéristiques du périphérique utilisé pour l'affichage. Ils permettent de créer des styles spécifiques pour différents types d'appareils, tels que les smartphones, les tablettes et les ordinateurs de bureau.

Les médias queries sont utilisés en ajoutant une condition dans la déclaration CSS à l'aide de la syntaxe suivante :

@media condition {
  /* styles spécifiques */
}

La condition peut être basée sur différentes caractéristiques, telles que la largeur de l'écran, la résolution, l'orientation, etc. Par exemple, pour appliquer un style spécifique aux écrans d'une largeur maximale de 768 pixels, on peut utiliser la condition suivante :

@media (max-width: 768px) {
  /* styles spécifiques aux écrans de 768 pixels ou moins */
}

Il est également possible de combiner plusieurs conditions en utilisant les opérateurs logiques "and" et "or". Par exemple, pour appliquer un style spécifique aux écrans d'une largeur maximale de 768 pixels en mode portrait, on peut utiliser la condition suivante :

@media (max-width: 768px) and (orientation: portrait) {
  /* styles spécifiques aux écrans de 768 pixels ou moins en mode portrait */
}

Les propriétés d'adaptation aux différents supports

Les propriétés d'adaptation aux différents supports permettent de modifier le style d'une page en fonction des caractéristiques du support utilisé pour l'affichage, tel que l'impression ou l'écran.

Voici quelques-unes des propriétés d'adaptation les plus couramment utilisées :

  • display : permet de spécifier comment un élément doit être affiché sur différents supports.
  • visibility : permet de spécifier si un élément doit être visible ou caché sur différents supports.
  • page-break-before, page-break-after, page-break-inside : permettent de contrôler les sauts de page lors de l'impression.
  • overflow : permet de spécifier comment un élément doit gérer le dépassement de contenu sur différents supports.

Par exemple, pour cacher un élément lors de l'impression, on peut utiliser la propriété display avec la valeur none :

@media print {
  .element {
    display: none;
  }
}

Cela permettra de masquer l'élément lors de l'impression, mais il sera toujours visible sur l'écran.

Il est important de noter que les médias queries et les propriétés d'adaptation aux différents supports sont des fonctionnalités avancées de CSS et nécessitent une bonne compréhension des concepts de base du CSS pour être utilisées efficacement.
