Section 3 : Les styles avancés avec CSS

Section 3 : Les styles avancés avec CSS

Les pseudo-classes CSS

Les pseudo-classes CSS permettent de sélectionner et de styler des éléments en fonction de leur état ou de leur position dans le document. Elles sont précédées par un double point (:).

Par exemple, la pseudo-classe :hover est utilisée pour appliquer un style à un élément lorsque le curseur de la souris est positionné dessus :

p:hover {
    color: blue;
}

De même, la pseudo-classe :nth-child(n) permet de sélectionner le n-ième élément enfant d'un élément parent :

ul li:nth-child(2) {
    color: red;
}

Les pseudo-éléments CSS

Les pseudo-éléments CSS permettent de styliser des parties spécifiques d'un élément. Ils sont également précédés par un double point (:).

Par exemple, le pseudo-élément ::before est utilisé pour insérer du contenu avant le contenu d'un élément :

p::before {
    content: "Avant";
}

De même, le pseudo-élément ::after est utilisé pour insérer du contenu après le contenu d'un élément :

p::after {
    content: "Après";
}

Les transitions et animations CSS

Les transitions CSS permettent de créer des effets de transition en douceur entre les différents états d'un élément. Elles sont définies à l'aide de la propriété transition.

Par exemple, la transition suivante applique un effet de fondu au survol d'un lien :

a {
    transition: opacity 0.5s ease;
}

a:hover {
    opacity: 0.5;
}

Les animations CSS permettent de créer des animations plus complexes en définissant des étapes clés. Elles sont définies à l'aide de la propriété animation.

Par exemple, l'animation suivante fait tourner un carré de 360 degrés :

@keyframes spin {
    0% {
        transform: rotate(0deg);
    }
    100% {
        transform: rotate(360deg);
    }
}

.square {
    animation: spin 2s linear infinite;
}

Les transformations CSS

Les transformations CSS permettent de modifier la position, la taille et l'apparence des éléments. Elles sont définies à l'aide de la propriété transform.

Par exemple, la transformation suivante fait pivoter un élément de 45 degrés :

.rotate {
    transform: rotate(45deg);
}

De même, la transformation suivante agrandit un élément de 50% de sa taille d'origine :

.scale {
    transform: scale(1.5);
}

Les transformations CSS offrent de nombreuses possibilités pour créer des effets visuels intéressants et dynamiques.
