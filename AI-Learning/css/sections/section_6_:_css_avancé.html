Section 6 : CSS avancé

Les variables CSS

Les variables CSS permettent de stocker des valeurs réutilisables dans une feuille de style. Elles sont déclarées avec le préfixe "--" suivi d'un nom et d'une valeur. Pour utiliser une variable, il suffit de la référencer en utilisant la fonction "var()". Voici un exemple :

:root {
  --primary-color: #ff0000;
}

h1 {
  color: var(--primary-color);
}

Dans cet exemple, nous avons déclaré une variable "--primary-color" avec la valeur "#ff0000". En utilisant la fonction "var(--primary-color)", nous appliquons cette couleur au texte des balises h1.

Les fonctions CSS

Les fonctions CSS permettent de manipuler les valeurs des propriétés CSS. Elles sont utilisées en tant que valeurs de propriétés ou dans des sélecteurs. Voici quelques exemples de fonctions couramment utilisées :

p {
  color: rgb(255, 0, 0); /* Utilisation de la fonction rgb() pour définir une couleur */
  font-size: calc(12px + 2vw); /* Utilisation de la fonction calc() pour effectuer des calculs */
  text-transform: uppercase; /* Utilisation de la fonction uppercase() pour mettre en majuscules */
}

Dans cet exemple, nous utilisons la fonction rgb() pour définir une couleur en utilisant les valeurs des composantes rouge, vert et bleu. Nous utilisons également la fonction calc() pour effectuer un calcul de taille de police basé sur la largeur de la fenêtre. Enfin, nous utilisons la fonction uppercase() pour mettre en majuscules le texte des balises p.

Les modules CSS (ex : Flexbox, Grid)

Les modules CSS sont des ensembles de fonctionnalités qui étendent les capacités de mise en page et de style du CSS. Deux modules populaires sont Flexbox et Grid :

.container {
  display: flex; /* Utilisation du module Flexbox pour créer un conteneur flexible */
  justify-content: center; /* Centrer les éléments horizontalement */
  align-items: center; /* Centrer les éléments verticalement */
}

.grid-container {
  display: grid; /* Utilisation du module Grid pour créer une grille */
  grid-template-columns: repeat(3, 1fr); /* Créer 3 colonnes de largeur égale */
  grid-gap: 10px; /* Espacement entre les éléments de la grille */
}

Dans cet exemple, nous utilisons le module Flexbox pour créer un conteneur flexible qui centre ses éléments horizontalement et verticalement. Nous utilisons également le module Grid pour créer une grille avec 3 colonnes de largeur égale et un espacement de 10 pixels entre les éléments.

Les nouveautés CSS (ex : CSS Custom Properties)

Les nouveautés CSS sont des fonctionnalités récentes qui ont été ajoutées au CSS pour améliorer sa flexibilité et sa puissance. Une de ces nouveautés est CSS Custom Properties :

:root {
  --primary-color: #ff0000; /* Déclaration d'une propriété personnalisée */
}

h1 {
  color: var(--primary-color); /* Utilisation de la propriété personnalisée */
}

Dans cet exemple, nous utilisons CSS Custom Properties pour déclarer une propriété personnalisée "--primary-color" avec la valeur "#ff0000". Nous pouvons ensuite utiliser cette propriété personnalisée en utilisant la fonction "var(--primary-color)" pour appliquer cette couleur au texte des balises h1.
