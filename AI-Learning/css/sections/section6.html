Section 6 : Les couleurs et les arrière-plans CSS

Les propriétés de couleur

Les propriétés de couleur en CSS permettent de définir la couleur du texte, des bordures et des arrière-plans des éléments HTML. Voici quelques-unes des propriétés de couleur les plus couramment utilisées :

  • color : Cette propriété permet de définir la couleur du texte. Elle accepte différentes valeurs, telles que les noms de couleurs prédéfinis (par exemple, "red" pour rouge), les codes hexadécimaux (par exemple, "#FF0000" pour rouge) et les valeurs RGB (par exemple, "rgb(255, 0, 0)" pour rouge).
  • background-color : Cette propriété permet de définir la couleur de l'arrière-plan d'un élément. Elle fonctionne de la même manière que la propriété color.
  • border-color : Cette propriété permet de définir la couleur des bordures d'un élément. Elle fonctionne de la même manière que la propriété color.

Voici un exemple d'utilisation de ces propriétés :

p {
  color: red;
  background-color: yellow;
  border-color: blue;
}

Ce code applique une couleur rouge au texte des éléments <p>, une couleur jaune à leur arrière-plan et une couleur bleue à leurs bordures.

Les propriétés d'arrière-plan

Les propriétés d'arrière-plan en CSS permettent de personnaliser l'arrière-plan des éléments HTML. Voici quelques-unes des propriétés d'arrière-plan les plus couramment utilisées :

  • background-image : Cette propriété permet de définir une image comme arrière-plan d'un élément. Elle accepte l'URL de l'image à utiliser.
  • background-repeat : Cette propriété permet de spécifier si l'image d'arrière-plan doit être répétée ou non. Les valeurs possibles sont repeat (répétition horizontale et verticale), repeat-x (répétition horizontale uniquement), repeat-y (répétition verticale uniquement) et no-repeat (pas de répétition).
  • background-position : Cette propriété permet de définir la position de l'image d'arrière-plan. Elle accepte des valeurs telles que top left, center center et bottom right.

Voici un exemple d'utilisation de ces propriétés :

div {
  background-image: url("background.jpg");
  background-repeat: no-repeat;
  background-position: center center;
}

Ce code applique une image d'arrière-plan à l'élément <div>, sans répétition, et la positionne au centre de l'élément.

Les propriétés de dégradé

Les propriétés de dégradé en CSS permettent de créer des effets de dégradé sur les arrière-plans des éléments HTML. Voici quelques-unes des propriétés de dégradé les plus couramment utilisées :

  • background-image : Cette propriété permet de définir un dégradé comme arrière-plan d'un élément. Elle accepte différentes valeurs, telles que les dégradés linéaires (par exemple, linear-gradient(red, yellow)) et les dégradés radiaux (par exemple, radial-gradient(red, yellow)).
  • background-repeat : Cette propriété permet de spécifier si le dégradé d'arrière-plan doit être répété ou non. Les valeurs possibles sont les mêmes que pour la propriété background-repeat des propriétés d'arrière-plan.
  • background-position : Cette propriété permet de définir la position du dégradé d'arrière-plan. Les valeurs possibles sont les mêmes que pour la propriété background-position des propriétés d'arrière-plan.

Voici un exemple d'utilisation de ces propriétés :

div {
  background-image: linear-gradient(red, yellow);
  background-repeat: no-repeat;
  background-position: center center;
}

Ce code applique un dégradé linéaire allant du rouge au jaune à l'arrière-plan de l'élément <div>, sans répétition, et le positionne au centre de l'élément.
