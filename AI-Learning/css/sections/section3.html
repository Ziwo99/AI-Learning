Section 3 : Les boîtes CSS

La modélisation en boîte

En CSS, chaque élément HTML est considéré comme une boîte rectangulaire appelée "boîte CSS". Cette boîte est composée de quatre parties principales : le contenu, le rembourrage (padding), la bordure (border) et la marge (margin). La modélisation en boîte est un concept fondamental en CSS car elle permet de contrôler la mise en page et le positionnement des éléments sur une page web.

Les propriétés de dimensionnement

Les propriétés de dimensionnement en CSS permettent de contrôler la taille des boîtes CSS. Voici quelques-unes des propriétés les plus couramment utilisées :

.ma-boite {
  width: 200px; /* définit la largeur de la boîte */
  height: 100px; /* définit la hauteur de la boîte */
  max-width: 500px; /* définit la largeur maximale de la boîte */
  max-height: 300px; /* définit la hauteur maximale de la boîte */
  min-width: 100px; /* définit la largeur minimale de la boîte */
  min-height: 50px; /* définit la hauteur minimale de la boîte */
}

Les propriétés de positionnement

Les propriétés de positionnement en CSS permettent de contrôler la position des boîtes CSS sur une page web. Voici quelques-unes des propriétés les plus couramment utilisées :

.ma-boite {
  position: relative; /* définit la position relative de la boîte par rapport à sa position normale */
  top: 50px; /* définit la distance entre le haut de la boîte et le haut de son conteneur */
  left: 20px; /* définit la distance entre la gauche de la boîte et la gauche de son conteneur */
  right: 10px; /* définit la distance entre la droite de la boîte et la droite de son conteneur */
  bottom: 30px; /* définit la distance entre le bas de la boîte et le bas de son conteneur */
  z-index: 1; /* définit la position de la boîte dans la pile d'éléments */
}

Les propriétés de décoration

Les propriétés de décoration en CSS permettent de modifier l'apparence visuelle des boîtes CSS. Voici quelques-unes des propriétés les plus couramment utilisées :

.ma-boite {
  background-color: #f1f1f1; /* définit la couleur de fond de la boîte */
  border: 1px solid #ccc; /* définit la bordure de la boîte */
  border-radius: 5px; /* définit le rayon des coins de la boîte */
  box-shadow: 2px 2px 5px #888; /* définit l'ombre de la boîte */
  opacity: 0.8; /* définit l'opacité de la boîte */
}

Ces propriétés permettent de personnaliser l'apparence des boîtes CSS et de créer des effets visuels attrayants.
