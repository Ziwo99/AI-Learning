Css

CSS (Cascading Style Sheets) est un langage de feuille de style utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler l'apparence des éléments d'une page web, tels que la couleur, la taille, la police, la disposition et les effets visuels, en séparant la structure du contenu de sa présentation.
