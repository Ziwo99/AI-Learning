La programmation système et les applications bas niveau

La programmation système en C++ permet de créer des applications qui interagissent directement avec le système d'exploitation. Cela inclut la gestion des fichiers, des processus, des threads et des ressources système.

Voici un exemple de code qui utilise la bibliothèque standard du C++ pour créer un fichier et écrire du texte :

#include <iostream>
#include <fstream>

int main() {
    std::ofstream file("example.txt");
    if (file.is_open()) {
        file << "Hello, world!";
        file.close();
    } else {
        std::cout << "Unable to open file.";
    }
    return 0;
}

La programmation réseau et les applications distribuées

La programmation réseau en C++ permet de créer des applications qui communiquent avec d'autres machines via des protocoles réseau tels que TCP/IP. Cela inclut la création de serveurs et de clients.

Voici un exemple de code qui utilise la bibliothèque Boost.Asio pour créer un serveur TCP simple :

#include <iostream>
#include <boost/asio.hpp>

int main() {
    boost::asio::io_context io_context;
    boost::asio::ip::tcp::acceptor acceptor(io_context, boost::asio::ip::tcp::endpoint(boost::asio::ip::tcp::v4(), 1234));

    boost::asio::ip::tcp::socket socket(io_context);
    acceptor.accept(socket);

    std::string message = "Hello, client!";
    boost::asio::write(socket, boost::asio::buffer(message));

    return 0;
}

La programmation graphique et les jeux vidéo

La programmation graphique en C++ permet de créer des applications qui affichent des graphiques, des images et des animations. Cela inclut le développement de jeux vidéo.

Voici un exemple de code qui utilise la bibliothèque SFML pour afficher une fenêtre et dessiner un cercle :

#include <SFML/Graphics.hpp>

int main() {
    sf::RenderWindow window(sf::VideoMode(800, 600), "SFML Window");

    sf::CircleShape circle(50);
    circle.setFillColor(sf::Color::Red);
    circle.setPosition(400, 300);

    while (window.isOpen()) {
        sf::Event event;
        while (window.pollEvent(event)) {
            if (event.type == sf::Event::Closed) {
                window.close();
            }
        }

        window.clear();
        window.draw(circle);
        window.display();
    }

    return 0;
}

La programmation embarquée et les systèmes embarqués

La programmation embarquée en C++ permet de créer des applications qui s'exécutent sur des systèmes embarqués tels que des microcontrôleurs. Cela inclut la gestion des entrées/sorties, des capteurs et des actionneurs.

Voici un exemple de code qui utilise la bibliothèque Arduino pour faire clignoter une LED :

#include <Arduino.h>

const int ledPin = 13;

void setup() {
    pinMode(ledPin, OUTPUT);
}

void loop() {
    digitalWrite(ledPin, HIGH);
    delay(1000);
    digitalWrite(ledPin, LOW);
    delay(1000);
}
