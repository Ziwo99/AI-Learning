La création d'une application console

Dans cette sous-section, vous apprendrez à créer une application console en utilisant le langage C++. Une application console est un programme qui s'exécute dans une fenêtre de terminal et communique avec l'utilisateur via des entrées et des sorties textuelles.

Voici un exemple de code pour créer une application console qui demande à l'utilisateur son nom et affiche un message de bienvenue :

#include <iostream>

int main() {
    std::string name;
    
    std::cout << "Entrez votre nom : ";
    std::cin >> name;
    
    std::cout << "Bonjour, " << name << "!" << std::endl;
    
    return 0;
}

Dans cet exemple, nous utilisons la bibliothèque iostream pour gérer les entrées et les sorties. La fonction main est le point d'entrée du programme et exécute les instructions à l'intérieur de son bloc de code. Nous déclarons une variable name de type string pour stocker le nom de l'utilisateur. Ensuite, nous utilisons std::cout pour afficher le message "Entrez votre nom : " à l'utilisateur, et std::cin pour lire la réponse de l'utilisateur et la stocker dans la variable name. Enfin, nous utilisons à nouveau std::cout pour afficher le message de bienvenue.

La création d'une application graphique

Dans cette sous-section, vous apprendrez à créer une application graphique en utilisant le langage C++. Une application graphique est un programme qui utilise une interface utilisateur graphique (GUI) pour interagir avec l'utilisateur.

Voici un exemple de code pour créer une application graphique simple qui affiche une fenêtre avec un bouton :

#include <iostream>
#include <SFML/Graphics.hpp>

int main() {
    sf::RenderWindow window(sf::VideoMode(800, 600), "Ma première application graphique");
    
    sf::RectangleShape button(sf::Vector2f(200, 50));
    button.setPosition(300, 275);
    button.setFillColor(sf::Color::Green);
    
    while (window.isOpen()) {
        sf::Event event;
        while (window.pollEvent(event)) {
            if (event.type == sf::Event::Closed) {
                window.close();
            }
        }
        
        window.clear();
        window.draw(button);
        window.display();
    }
    
    return 0;
}

Dans cet exemple, nous utilisons la bibliothèque SFML (Simple and Fast Multimedia Library) pour créer une fenêtre graphique. La fonction main est le point d'entrée du programme et exécute les instructions à l'intérieur de son bloc de code. Nous créons une fenêtre graphique de taille 800x600 pixels avec le titre "Ma première application graphique". Ensuite, nous créons un rectangle représentant un bouton et le positionnons à l'écran. Nous utilisons une boucle while pour maintenir la fenêtre ouverte et gérer les événements. Enfin, nous effaçons le contenu de la fenêtre, dessinons le bouton et mettons à jour l'affichage.

La création d'une application réseau

Dans cette sous-section, vous apprendrez à créer une application réseau en utilisant le langage C++. Une application réseau est un programme qui communique avec d'autres ordinateurs via un réseau, en utilisant des protocoles tels que TCP/IP.

Voici un exemple de code pour créer une application réseau simple qui se connecte à un serveur distant et envoie un message :

#include <iostream>
#include <SFML/Network.hpp>

int main() {
    sf::TcpSocket socket;
    
    if (socket.connect("127.0.0.1", 1234) != sf::Socket::Done) {
        std::cout << "Erreur lors de la connexion au serveur." << std::endl;
        return 1;
    }
    
    std::string message = "Bonjour, serveur !";
    if (socket.send(message.c_str(), message.size() + 1) != sf::Socket::Done) {
        std::cout << "Erreur lors de l'envoi du message." << std::endl;
        return 1;
    }
    
    return 0;
}

Dans cet exemple, nous utilisons la bibliothèque SFML (Simple and Fast Multimedia Library) pour gérer les communications réseau. La fonction main est le point d'entrée du programme et exécute les instructions à l'intérieur de son bloc de code. Nous créons un objet sf::TcpSocket pour gérer la connexion réseau. Ensuite, nous utilisons la méthode connect pour nous connecter à un serveur distant à l'adresse IP "127.0.0.1" et au port 1234. Si la connexion échoue, nous affichons un message d'erreur. Enfin, nous envoyons un message au serveur en utilisant la méthode send.

La création d'une bibliothèque externe

Dans cette sous-section, vous apprendrez à créer une bibliothèque externe en utilisant le langage C++. Une bibliothèque externe est un ensemble de fonctions et de classes qui peuvent être utilisées dans d'autres programmes.

Voici un exemple de code pour créer une bibliothèque externe qui fournit une fonction pour calculer la factorielle d'un nombre :

// File: factorial.h

#ifndef FACTORIAL_H
#define FACTORIAL_H

int factorial(int n);

#endif

// File: factorial.cpp

#include "factorial.h"

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

Dans cet exemple, nous créons deux fichiers : factorial.h et factorial.cpp. Le fichier factorial.h contient la déclaration de la fonction factorial, qui prend un entier en paramètre et renvoie un entier. Le fichier factorial.cpp contient la définition de la fonction factorial, qui utilise la récursivité pour calculer la factorielle d'un nombre. Ensuite, vous pouvez utiliser cette bibliothèque externe dans d'autres programmes en incluant le fichier factorial.h et en liant le fichier factorial.cpp lors de la compilation.
