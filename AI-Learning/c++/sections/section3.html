Section 3 : Les concepts avancés du C++

Les pointeurs et les références

Les pointeurs et les références sont des concepts avancés du langage C++ qui permettent de manipuler directement les adresses mémoire des variables. Ils offrent une grande flexibilité et sont largement utilisés dans la programmation C++.

Un pointeur est une variable qui contient l'adresse mémoire d'une autre variable. Il est déclaré en utilisant l'opérateur "*" suivi du type de la variable pointée. Par exemple :

int* ptr; // Déclaration d'un pointeur vers un entier

Une référence, quant à elle, est un alias d'une variable existante. Elle est déclarée en utilisant l'opérateur "&" suivi du type de la variable référencée. Par exemple :

int a = 5;
int& ref = a; // Déclaration d'une référence à la variable "a"

Les pointeurs et les références permettent d'accéder et de modifier la valeur d'une variable en utilisant son adresse mémoire. Ils sont notamment utilisés pour passer des paramètres par référence à une fonction, pour allouer et libérer dynamiquement de la mémoire, et pour implémenter des structures de données avancées.

La gestion dynamique de la mémoire

La gestion dynamique de la mémoire est une fonctionnalité avancée du langage C++ qui permet d'allouer et de libérer de la mémoire pendant l'exécution d'un programme. Elle est réalisée à l'aide des opérateurs "new" et "delete".

L'opérateur "new" est utilisé pour allouer de la mémoire pour un objet ou un tableau d'objets. Par exemple :

int* ptr = new int; // Allocation dynamique d'un entier
int* arr = new int[10]; // Allocation dynamique d'un tableau de 10 entiers

L'opérateur "delete" est utilisé pour libérer la mémoire allouée précédemment. Il est important de libérer la mémoire allouée dynamiquement pour éviter les fuites de mémoire. Par exemple :

delete ptr; // Libération de la mémoire allouée pour l'entier
delete[] arr; // Libération de la mémoire allouée pour le tableau

La gestion dynamique de la mémoire est souvent utilisée lorsque la taille d'un objet ou d'un tableau est inconnue à la compilation, ou lorsque la durée de vie d'un objet doit être contrôlée manuellement.

Les classes et les objets

Les classes et les objets sont les fondements de la programmation orientée objet en C++. Une classe est une structure de données qui regroupe des variables (appelées membres) et des fonctions (appelées méthodes) qui agissent sur ces variables. Un objet est une instance d'une classe.

Pour déclarer une classe, on utilise le mot-clé "class" suivi du nom de la classe. Par exemple :

class Person {
    private:
        string name;
        int age;
    
    public:
        void setName(string n) {
            name = n;
        }
        
        void setAge(int a) {
            age = a;
        }
        
        void printInfo() {
            cout << "Name: " << name << ", Age: " << age << endl;
        }
};

Pour créer un objet à partir d'une classe, on utilise le nom de la classe suivi du nom de l'objet. Par exemple :

Person p;
p.setName("John");
p.setAge(25);
p.printInfo(); // Affiche "Name: John, Age: 25"

Les classes permettent d'encapsuler des données et des fonctionnalités connexes, ce qui facilite la gestion et la réutilisation du code. Elles sont largement utilisées dans le développement de logiciels en C++.

L'héritage et le polymorphisme

L'héritage et le polymorphisme sont des concepts clés de la programmation orientée objet en C++. L'héritage permet de créer de nouvelles classes (appelées classes dérivées) à partir d'une classe existante (appelée classe de base), en héritant de ses membres et en ajoutant de nouveaux membres.

Le polymorphisme permet à un objet d'une classe dérivée d'être traité comme un objet de sa classe de base. Cela permet de créer des hiérarchies de classes et d'utiliser des objets de différentes classes de manière transparente.

Par exemple, considérons une classe de base "Shape" et deux classes dérivées "Circle" et "Rectangle". La classe "Shape" peut avoir une méthode virtuelle "area()" qui est redéfinie dans les classes dérivées :

class Shape {
    public:
        virtual double area() {
            return 0;
        }
};

class Circle : public Shape {
    private:
        double radius;
    
    public:
        Circle(double r) {
            radius = r;
        }
        
        double area() {
            return 3.14 * radius * radius;
        }
};

class Rectangle : public Shape {
    private:
        double width;
        double height;
    
    public:
        Rectangle(double w, double h) {
            width = w;
            height = h;
        }
        
        double area() {
            return width * height;
        }
};

En utilisant le polymorphisme, on peut créer un tableau d'objets de type "Shape" et y stocker des objets de type "Circle" et "Rectangle" :

Shape* shapes[2];
shapes[0] = new Circle(5);
shapes[1] = new Rectangle(3, 4);

for (int i = 0; i < 2; i++) {
    cout << "Area: " << shapes[i]->area() << endl;
}

Cela permet d'appeler la méthode "area()" spécifique à chaque classe dérivée, même si les objets sont stockés dans un tableau de type "Shape".

Les templates et la programmation générique

Les templates et la programmation générique sont des fonctionnalités avancées du langage C++ qui permettent de créer des fonctions et des classes génériques, c'est-à-dire capables de travailler avec différents types de données.

Un template est une sorte de modèle qui permet de définir une fonction ou une classe avec des paramètres de type génériques. Par exemple, voici une fonction template qui retourne la valeur maximale entre deux valeurs :

template <typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}

Cette fonction peut être utilisée avec différents types de données :

int maxInt = max(5, 10); // Utilisation avec des entiers
double maxDouble = max(3.14, 2.71); // Utilisation avec des nombres à virgule

Les templates permettent d'écrire du code générique et réutilisable, ce qui facilite le développement de bibliothèques et de frameworks.
