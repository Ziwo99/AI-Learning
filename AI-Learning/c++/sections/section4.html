Section 4 : Les bibliothèques standard du C++

La bibliothèque d'entrées/sorties

La bibliothèque d'entrées/sorties du C++ offre des fonctionnalités pour gérer les opérations d'entrée et de sortie de données. Elle permet de lire des données à partir d'un flux d'entrée (comme le clavier ou un fichier) et d'écrire des données vers un flux de sortie (comme l'écran ou un fichier).

Voici un exemple d'utilisation de la bibliothèque d'entrées/sorties pour lire une valeur entière à partir du clavier :

#include <iostream>

int main() {
    int num;
    std::cout << "Entrez un nombre : ";
    std::cin >> num;
    std::cout << "Vous avez saisi : " << num << std::endl;
    return 0;
}

La bibliothèque de conteneurs

La bibliothèque de conteneurs du C++ fournit des structures de données prédéfinies pour stocker et manipuler des collections d'objets. Elle comprend des conteneurs tels que les vecteurs, les listes, les ensembles et les cartes, qui offrent différentes fonctionnalités et performances en fonction des besoins spécifiques.

Voici un exemple d'utilisation de la bibliothèque de conteneurs pour stocker une liste de nombres entiers dans un vecteur :

#include <vector>

int main() {
    std::vector<int> numbers;
    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    for (int i = 0; i < numbers.size(); i++) {
        std::cout << numbers[i] << " ";
    }

    return 0;
}

La bibliothèque d'algorithmes

La bibliothèque d'algorithmes du C++ fournit un ensemble de fonctions prédéfinies pour effectuer des opérations courantes sur les conteneurs. Elle inclut des algorithmes de tri, de recherche, de manipulation et de transformation de données.

Voici un exemple d'utilisation de la bibliothèque d'algorithmes pour trier un vecteur de nombres entiers dans l'ordre croissant :

#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers = {3, 1, 2};

    std::sort(numbers.begin(), numbers.end());

    for (int num : numbers) {
        std::cout << num << " ";
    }

    return 0;
}

La bibliothèque de gestion de la mémoire

La bibliothèque de gestion de la mémoire du C++ fournit des fonctionnalités pour allouer et libérer dynamiquement de la mémoire. Elle permet de créer des objets sur le tas et de les gérer de manière efficace.

Voici un exemple d'utilisation de la bibliothèque de gestion de la mémoire pour allouer et libérer dynamiquement un tableau d'entiers :

#include <iostream>

int main() {
    int* numbers = new int[3];
    numbers[0] = 1;
    numbers[1] = 2;
    numbers[2] = 3;

    for (int i = 0; i < 3; i++) {
        std::cout << numbers[i] << " ";
    }

    delete[] numbers;

    return 0;
}
