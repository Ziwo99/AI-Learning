La bibliothèque standard du C++

La bibliothèque standard du C++ est une collection de classes et de fonctions prédéfinies qui fournissent des fonctionnalités de base pour le développement en C++. Elle est incluse dans tous les compilateurs C++ et offre un large éventail de fonctionnalités, telles que la gestion de la mémoire, les opérations sur les chaînes de caractères, les entrées/sorties, les conteneurs de données et les algorithmes.

Voici un exemple d'utilisation de la bibliothèque standard du C++ pour afficher un message à l'écran :

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

Dans cet exemple, nous utilisons la classe std::cout pour afficher le message "Hello, World!" à l'écran, suivi de std::endl pour passer à la ligne. La bibliothèque standard du C++ offre de nombreuses autres classes et fonctions utiles pour le développement en C++, et nous les explorerons plus en détail tout au long de cette formation.

Les bibliothèques tierces populaires (Boost, Qt, etc.)

En plus de la bibliothèque standard du C++, il existe de nombreuses bibliothèques tierces populaires qui étendent les fonctionnalités du langage et facilitent le développement de logiciels en C++. Parmi les bibliothèques tierces les plus utilisées, on trouve Boost, Qt, OpenCV, PCL, et bien d'autres.

Par exemple, Boost est une bibliothèque C++ très populaire qui fournit des fonctionnalités avancées telles que les expressions régulières, la programmation concurrente, la manipulation de fichiers, les structures de données, etc. Voici un exemple d'utilisation de Boost pour effectuer une recherche dans une chaîne de caractères :

#include <boost/algorithm/string.hpp>
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    boost::to_upper(str);
    std::cout << str << std::endl;
    return 0;
}

Dans cet exemple, nous utilisons la fonction boost::to_upper pour convertir la chaîne de caractères en majuscules. Boost offre de nombreuses autres fonctionnalités utiles pour le développement en C++, et nous les explorerons plus en détail tout au long de cette formation.

Les frameworks de développement (MFC, wxWidgets, etc.)

En plus des bibliothèques tierces, il existe également des frameworks de développement qui facilitent la création d'applications en C++. Ces frameworks fournissent une structure et des fonctionnalités prédéfinies pour le développement d'applications graphiques, d'applications web, etc.

Par exemple, MFC (Microsoft Foundation Classes) est un framework de développement pour les applications Windows. Il fournit des classes et des fonctionnalités pour la création d'interfaces utilisateur, la gestion des événements, la gestion des fenêtres, etc. Voici un exemple d'utilisation de MFC pour créer une fenêtre :

#include <afxwin.h>

class CMainWindow : public CFrameWnd {
public:
    CMainWindow() {
        Create(NULL, _T("Hello, World!"));
    }
};

class CMyApp : public CWinApp {
public:
    virtual BOOL InitInstance() {
        m_pMainWnd = new CMainWindow();
        m_pMainWnd->ShowWindow(SW_SHOW);
        m_pMainWnd->UpdateWindow();
        return TRUE;
    }
};

CMyApp theApp;

Dans cet exemple, nous utilisons les classes fournies par MFC pour créer une fenêtre principale avec le titre "Hello, World!". MFC offre de nombreuses autres fonctionnalités pour le développement d'applications Windows, et nous les explorerons plus en détail tout au long de cette formation.
