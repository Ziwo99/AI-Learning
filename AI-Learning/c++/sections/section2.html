Les types de données et les variables

En C++, les types de données sont utilisés pour définir le type de valeur qu'une variable peut stocker. Voici les types de données de base en C++ :

  • int : pour stocker des nombres entiers
  • float : pour stocker des nombres décimaux
  • double : pour stocker des nombres décimaux avec une précision accrue
  • char : pour stocker des caractères
  • bool : pour stocker des valeurs booléennes (vrai ou faux)

Les variables sont utilisées pour stocker des valeurs dans la mémoire de l'ordinateur. Voici comment déclarer et initialiser une variable en C++ :

int age = 25;
float pi = 3.14;
char grade = 'A';
bool isStudent = true;

Les opérateurs et les expressions

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés en C++ :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
  • % : modulo (reste de la division)
  • ++ : incrémentation
  • -- : décrémentation

Les expressions sont des combinaisons d'opérateurs, de variables et de valeurs qui produisent un résultat. Voici quelques exemples d'expressions en C++ :

int sum = 5 + 3;
float average = (10.5 + 8.2) / 2;
int remainder = 10 % 3;
int x = 5;
x++; // x vaut maintenant 6

Les structures de contrôle (boucles et conditions)

Les structures de contrôle sont utilisées pour contrôler le flux d'exécution d'un programme. Voici les principales structures de contrôle en C++ :

  • if : exécute un bloc de code si une condition est vraie
  • else : exécute un bloc de code si la condition de l'if est fausse
  • else if : permet de tester plusieurs conditions en cascade
  • switch : exécute un bloc de code en fonction de la valeur d'une variable
  • for : exécute un bloc de code un certain nombre de fois
  • while : exécute un bloc de code tant qu'une condition est vraie
  • do while : exécute un bloc de code au moins une fois, puis tant qu'une condition est vraie

Voici un exemple d'utilisation de la boucle while en C++ :

int i = 0;

while (i < 10) {
  cout << "Iteration " << i << endl;
  i++;
}

Les tableaux et les chaînes de caractères

Les tableaux sont utilisés pour stocker plusieurs valeurs du même type dans une seule variable. Voici comment déclarer et initialiser un tableau en C++ :

int numbers[5] = {1, 2, 3, 4, 5};

Les chaînes de caractères sont utilisées pour stocker des séquences de caractères. Voici comment déclarer et initialiser une chaîne de caractères en C++ :

string name = "John Doe";

Les fonctions et les procédures

Les fonctions sont utilisées pour regrouper un ensemble d'instructions qui peuvent être réutilisées à plusieurs endroits dans un programme. Voici comment déclarer et appeler une fonction en C++ :

int add(int a, int b) {
  return a + b;
}

int result = add(5, 3);

Les procédures sont similaires aux fonctions, mais elles ne renvoient pas de valeur. Voici comment déclarer et appeler une procédure en C++ :

void greet() {
  cout << "Hello!" << endl;
}

greet();
