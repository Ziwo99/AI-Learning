Les types de données et les variables

Les types de données sont utilisés pour définir le type de valeur qu'une variable peut stocker. Le langage C++ offre plusieurs types de données primitifs, tels que :

  • int : pour stocker des nombres entiers
  • float : pour stocker des nombres décimaux
  • char : pour stocker des caractères
  • bool : pour stocker des valeurs booléennes (vrai ou faux)

Il est également possible de définir des types de données personnalisés à l'aide de structures ou de classes.

Pour déclarer une variable en C++, on utilise la syntaxe suivante :

type_de_données nom_de_variable;

Par exemple, pour déclarer une variable entière nommée "age", on peut écrire :

int age;

Une fois une variable déclarée, on peut lui attribuer une valeur en utilisant l'opérateur d'assignation (=) :

age = 25;

Il est également possible de déclarer et d'initialiser une variable en même temps :

int age = 25;

Les opérateurs et les expressions

Les opérateurs sont utilisés pour effectuer des opérations sur les variables. Le langage C++ offre différents types d'opérateurs, tels que :

  • Opérateurs arithmétiques : +, -, *, /, %
  • Opérateurs de comparaison : ==, !=, >, <, >=, <=
  • Opérateurs logiques : &&, ||, !
  • Opérateurs d'incrémentation et de décrémentation : ++, --

Les expressions sont des combinaisons de variables, de valeurs et d'opérateurs. Par exemple :

int a = 5;
int b = 10;
int c = a + b;

Dans cet exemple, la variable "c" contiendra la valeur 15, qui est le résultat de l'addition de "a" et "b".

Les structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. Les boucles permettent de répéter une série d'instructions plusieurs fois, tandis que les conditions permettent d'exécuter certaines instructions uniquement si une condition est remplie.

La boucle la plus couramment utilisée en C++ est la boucle "for". Elle a la syntaxe suivante :

for (initialisation; condition; incrémentation) {
    // instructions à exécuter
}

Par exemple, pour afficher les nombres de 1 à 5, on peut écrire :

for (int i = 1; i <= 5; i++) {
    cout << i << endl;
}

Les conditions sont utilisées pour exécuter des instructions uniquement si une condition est remplie. La structure de contrôle "if" est utilisée pour cela :

if (condition) {
    // instructions à exécuter si la condition est vraie
} else {
    // instructions à exécuter si la condition est fausse
}

Par exemple, pour vérifier si un nombre est pair ou impair, on peut écrire :

int nombre = 7;

if (nombre % 2 == 0) {
    cout << "Le nombre est pair" << endl;
} else {
    cout << "Le nombre est impair" << endl;
}

Les fonctions et les procédures

Les fonctions et les procédures sont utilisées pour regrouper des instructions et les réutiliser à plusieurs endroits dans un programme. Une fonction retourne une valeur, tandis qu'une procédure n'en retourne pas.

La syntaxe pour déclarer une fonction est la suivante :

type_de_retour nom_de_fonction(paramètres) {
    // instructions à exécuter
    return valeur;
}

Par exemple, pour déclarer une fonction qui calcule la somme de deux nombres :

int somme(int a, int b) {
    return a + b;
}

La syntaxe pour déclarer une procédure est similaire, mais sans la valeur de retour :

void afficherMessage() {
    cout << "Bonjour !" << endl;
}

Les tableaux et les pointeurs

Les tableaux permettent de stocker plusieurs valeurs du même type dans une seule variable. Les pointeurs sont utilisés pour manipuler des adresses mémoire.

Pour déclarer un tableau en C++, on utilise la syntaxe suivante :

type_de_données nom_de_tableau[taille];

Par exemple, pour déclarer un tableau d'entiers de taille 5 :

int tableau[5];

Les éléments d'un tableau sont accessibles à l'aide de leur indice. L'indice du premier élément est 0. Par exemple, pour accéder au troisième élément d'un tableau :

int valeur = tableau[2];

Les pointeurs sont déclarés en utilisant le symbole *. Par exemple :

int* pointeur;

Un pointeur peut être utilisé pour stocker l'adresse mémoire d'une variable. Par exemple :

int nombre = 10;
int* pointeur = &nombre;

Pour accéder à la valeur pointée par un pointeur, on utilise l'opérateur *. Par exemple :

int valeur = *pointeur;

Il est également possible d'allouer dynamiquement de la mémoire à l'aide de l'opérateur "new". Par exemple :

int* pointeur = new int;

Il est important de libérer la mémoire allouée dynamiquement à l'aide de l'opérateur "delete" :

delete pointeur;

Ceci conclut la section sur les fondamentaux du C++. Nous avons couvert les types de données et les variables, les opérateurs et les expressions, les structures de contrôle, les fonctions et les procédures, ainsi que les tableaux et les pointeurs.
