Les types de données et les variables

En C++, les types de données sont utilisés pour définir le type de valeur qu'une variable peut contenir. Voici les types de données les plus couramment utilisés :

  • int : pour représenter les nombres entiers
  • float : pour représenter les nombres à virgule flottante
  • double : pour représenter les nombres à virgule flottante avec une précision supérieure
  • char : pour représenter les caractères
  • bool : pour représenter les valeurs booléennes (vrai ou faux)

Pour déclarer une variable, utilisez la syntaxe suivante :

type_de_données nom_de_variable;

Par exemple, pour déclarer une variable entière nommée age, vous pouvez écrire :

int age;

Vous pouvez également initialiser une variable lors de sa déclaration :

int nombre = 42;

Les opérateurs et les expressions

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés :

  • + : pour l'addition
  • - : pour la soustraction
  • * : pour la multiplication
  • / : pour la division
  • % : pour le modulo (reste de la division)
  • ++ : pour l'incrémentation
  • -- : pour la décrémentation
  • = : pour l'assignation

Les expressions sont des combinaisons d'opérateurs et de variables qui produisent une valeur. Par exemple :

int a = 5;
int b = 3;
int c = a + b; // c contient maintenant la valeur 8

Les structures de contrôle (boucles et conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. Voici les structures de contrôle les plus couramment utilisées :

  • La boucle while : exécute un bloc de code tant qu'une condition est vraie
  • La boucle for : exécute un bloc de code un certain nombre de fois
  • La structure if : exécute un bloc de code si une condition est vraie
  • La structure else : exécute un bloc de code si une condition est fausse
  • La structure switch : exécute un bloc de code en fonction de la valeur d'une variable

Voici un exemple d'utilisation de la boucle while :

int i = 0;

while (i < 10) {
  std::cout << "Iteration " << i << std::endl;
  i++;
}

Les fonctions et les paramètres

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Les paramètres sont des valeurs que vous pouvez passer à une fonction pour qu'elle les utilise. Voici comment déclarer une fonction :

type_de_retour nom_de_fonction(type_paramètre1 nom_paramètre1, type_paramètre2 nom_paramètre2, ...) {
  // bloc de code
}

Par exemple, voici une fonction qui calcule la somme de deux nombres :

int somme(int a, int b) {
  return a + b;
}

Vous pouvez ensuite appeler cette fonction en lui passant des arguments :

int resultat = somme(3, 5); // resultat contient maintenant la valeur 8

Les tableaux et les pointeurs

Les tableaux sont des structures de données qui permettent de stocker plusieurs valeurs du même type. Les pointeurs sont des variables qui contiennent l'adresse mémoire d'une autre variable. Voici comment déclarer un tableau :

type_de_données nom_tableau[taille];

Par exemple, voici comment déclarer un tableau d'entiers de taille 5 :

int tableau[5];

Vous pouvez accéder aux éléments d'un tableau en utilisant des indices :

tableau[0] = 42; // assigne la valeur 42 au premier élément du tableau

Les pointeurs peuvent être utilisés pour manipuler des tableaux. Voici comment déclarer un pointeur :

type_de_données *nom_pointeur;

Par exemple, voici comment déclarer un pointeur vers un tableau d'entiers :

int *pointeur;

Vous pouvez ensuite utiliser le pointeur pour accéder aux éléments du tableau :

pointeur = tableau; // pointe vers le premier élément du tableau
int premier_element = *pointeur; // premier_element contient maintenant la valeur 42
