Les structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme en fonction de certaines conditions. En C++, les structures de contrôle les plus couramment utilisées sont les boucles et les conditions.

Boucles

Les boucles permettent d'exécuter une série d'instructions de manière répétée tant qu'une condition est vraie. Il existe trois types de boucles en C++ : la boucle while, la boucle do-while et la boucle for.

La boucle while :

int i = 0;
while (i < 10) {
    // Instructions à exécuter
    i++;
}

La boucle do-while :

int i = 0;
do {
    // Instructions à exécuter
    i++;
} while (i < 10);

La boucle for :

for (int i = 0; i < 10; i++) {
    // Instructions à exécuter
}

Conditions

Les conditions permettent d'exécuter certaines instructions uniquement si une condition est vraie. En C++, les conditions sont généralement utilisées avec les structures de contrôle pour contrôler le flux d'exécution du programme.

Les opérateurs de comparaison les plus couramment utilisés en C++ sont :

  • == : égal à
  • != : différent de
  • < : inférieur à
  • > : supérieur à
  • <= : inférieur ou égal à
  • >= : supérieur ou égal à

Exemple d'utilisation d'une condition :

int age = 25;
if (age >= 18) {
    // Instructions à exécuter si la condition est vraie
}

Les fonctions et les procédures

Les fonctions et les procédures permettent de regrouper des instructions et de les réutiliser à plusieurs endroits dans un programme. En C++, une fonction est une séquence d'instructions qui renvoie une valeur, tandis qu'une procédure est une séquence d'instructions qui ne renvoie pas de valeur.

Exemple de déclaration et d'appel d'une fonction :

// Déclaration de la fonction
int addition(int a, int b) {
    return a + b;
}

// Appel de la fonction
int resultat = addition(3, 5);

Exemple de déclaration et d'appel d'une procédure :

// Déclaration de la procédure
void afficherMessage() {
    std::cout << "Bonjour !" << std::endl;
}

// Appel de la procédure
afficherMessage();

Les tableaux et les chaînes de caractères

Les tableaux et les chaînes de caractères permettent de stocker et de manipuler plusieurs valeurs de même type. En C++, les tableaux sont des structures de données statiques, tandis que les chaînes de caractères sont des tableaux de caractères.

Exemple de déclaration et d'utilisation d'un tableau :

int tableau[5] = {1, 2, 3, 4, 5};

// Accès aux éléments du tableau
int premierElement = tableau[0];
int dernierElement = tableau[4];

Exemple de déclaration et d'utilisation d'une chaîne de caractères :

char chaine[] = "Bonjour";

// Affichage de la chaîne de caractères
std::cout << chaine << std::endl;

Il est important de noter que les indices des tableaux commencent à 0. Ainsi, le premier élément d'un tableau est à l'indice 0, le deuxième à l'indice 1, et ainsi de suite.
