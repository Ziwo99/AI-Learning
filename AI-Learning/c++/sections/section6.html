Les templates et la programmation générique

Les templates sont un mécanisme puissant en C++ qui permet de créer des fonctions et des classes génériques. Ils permettent de définir des algorithmes et des structures de données qui peuvent être utilisés avec différents types de données, sans avoir à réécrire le code pour chaque type spécifique.

Un template est défini en utilisant le mot-clé template, suivi d'une liste de paramètres de type entre chevrons (<>). Par exemple, voici comment définir une fonction template qui calcule la somme de deux valeurs :

template <typename T>
T sum(T a, T b) {
    return a + b;
}

Dans cet exemple, T est un paramètre de type générique. Lorsque la fonction est utilisée, le compilateur déduit automatiquement le type approprié en fonction des arguments passés. Par exemple :

int result = sum(5, 10); // result vaut 15
double result2 = sum(3.14, 2.71); // result2 vaut 5.85

Les templates peuvent également être utilisés pour définir des classes génériques. Par exemple, voici comment définir une classe template Pair qui représente une paire de valeurs de types différents :

template <typename T, typename U>
class Pair {
private:
    T first;
    U second;
public:
    Pair(T a, U b) : first(a), second(b) {}
    T getFirst() { return first; }
    U getSecond() { return second; }
};

La classe Pair peut être instanciée avec différents types de données :

Pair<int, double> p(5, 3.14);
int a = p.getFirst(); // a vaut 5
double b = p.getSecond(); // b vaut 3.14

Les exceptions et la gestion des erreurs

Les exceptions sont utilisées en C++ pour gérer les erreurs et les situations exceptionnelles qui peuvent se produire pendant l'exécution d'un programme. Une exception est une valeur ou un objet qui est lancé (throw) par une partie du code et qui peut être attrapé (catch) et géré par une autre partie du code.

Pour lancer une exception, on utilise le mot-clé throw suivi de la valeur ou de l'objet à lancer. Par exemple :

void divide(int a, int b) {
    if (b == 0) {
        throw "Division by zero is not allowed!";
    }
    int result = a / b;
}

Dans cet exemple, si la variable b est égale à zéro, une exception de type const char* est lancée avec le message "Division by zero is not allowed!".

Pour attraper une exception, on utilise le bloc try-catch. Le bloc try contient le code qui peut lancer une exception, et le bloc catch contient le code qui gère l'exception. Par exemple :

try {
    divide(10, 0);
} catch (const char* message) {
    std::cout << "An exception occurred: " << message << std::endl;
}

Dans cet exemple, si l'exception est lancée dans la fonction divide, le bloc catch est exécuté et affiche le message d'erreur.

Les opérateurs surchargés et les conversions de types

En C++, il est possible de surcharger les opérateurs existants pour les types de données personnalisés. Cela permet d'utiliser des opérateurs tels que +, -, *, /, etc. avec des objets de ces types personnalisés.

La surcharge des opérateurs se fait en définissant des fonctions spéciales appelées "opérateurs surchargés". Par exemple, voici comment surcharger l'opérateur + pour une classe Vector qui représente un vecteur mathématique :

class Vector {
private:
    double x, y, z;
public:
    Vector(double a, double b, double c) : x(a), y(b), z(c) {}
    Vector operator+(const Vector& other) const {
        return Vector(x + other.x, y + other.y, z + other.z);
    }
};

Dans cet exemple, la fonction operator+ est définie comme une méthode de la classe Vector. Elle prend un objet Vector en argument et retourne un nouvel objet Vector qui est la somme des deux vecteurs.

Une fois l'opérateur surchargé, il peut être utilisé comme n'importe quel autre opérateur. Par exemple :

Vector v1(1, 2, 3);
Vector v2(4, 5, 6);
Vector v3 = v1 + v2; // v3 contient le vecteur (5, 7, 9)

En plus de la surcharge des opérateurs, il est également possible de définir des conversions de types personnalisées en utilisant des fonctions spéciales appelées "constructeurs de conversion" et "opérateurs de conversion". Ces fonctions permettent de convertir un objet d'un type à un autre de manière explicite ou implicite.

Par exemple, voici comment définir un constructeur de conversion pour une classe Point qui représente un point dans un plan cartésien :

class Point {
private:
    double x, y;
public:
    Point(double a, double b) : x(a), y(b) {}
    operator Vector() const {
        return Vector(x, y, 0);
    }
};

Dans cet exemple, la fonction operator Vector() est définie comme une méthode de la classe Point. Elle retourne un nouvel objet Vector qui est créé à partir des coordonnées x et y du point, avec une valeur de z égale à zéro.

Une fois le constructeur de conversion défini, un objet de type Point peut être converti en un objet de type Vector de manière implicite :

Point p(1, 2);
Vector v = p; // p est converti en un vecteur (1, 2, 0)
