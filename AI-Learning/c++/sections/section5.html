Les conteneurs (vecteurs, listes, piles, files)

Les conteneurs sont des structures de données qui permettent de stocker et de manipuler des collections d'objets en C++. La bibliothèque standard du C++ propose plusieurs types de conteneurs, dont les vecteurs, les listes, les piles et les files.

Les vecteurs

Un vecteur est un conteneur séquentiel qui permet de stocker un ensemble d'objets de même type. Il offre des opérations efficaces pour l'insertion, la suppression et l'accès aux éléments. Voici un exemple d'utilisation d'un vecteur :

#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers;

    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    for (int i = 0; i < numbers.size(); i++) {
        std::cout << numbers[i] << " ";
    }

    return 0;
}

Les listes

Une liste est un conteneur séquentiel doublement chaîné qui permet de stocker un ensemble d'objets de même type. Contrairement aux vecteurs, les listes offrent des opérations efficaces pour l'insertion et la suppression en position quelconque. Voici un exemple d'utilisation d'une liste :

#include <list>
#include <iostream>

int main() {
    std::list<int> numbers;

    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    for (std::list<int>::iterator it = numbers.begin(); it != numbers.end(); it++) {
        std::cout << *it << " ";
    }

    return 0;
}

Les piles

Une pile est un conteneur séquentiel qui permet de stocker un ensemble d'objets de même type selon le principe LIFO (Last In, First Out). Les opérations principales sur une pile sont l'empilement (push) et le dépilement (pop). Voici un exemple d'utilisation d'une pile :

#include <stack>
#include <iostream>

int main() {
    std::stack<int> numbers;

    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }

    return 0;
}

Les files

Une file est un conteneur séquentiel qui permet de stocker un ensemble d'objets de même type selon le principe FIFO (First In, First Out). Les opérations principales sur une file sont l'ajout en queue (push) et la suppression en tête (pop). Voici un exemple d'utilisation d'une file :

#include <queue>
#include <iostream>

int main() {
    std::queue<int> numbers;

    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.front() << " ";
        numbers.pop();
    }

    return 0;
}

Les algorithmes (tri, recherche, manipulation de conteneurs)

La bibliothèque standard du C++ propose également un ensemble d'algorithmes prédéfinis qui peuvent être utilisés avec les conteneurs. Ces algorithmes permettent d'effectuer des opérations courantes telles que le tri, la recherche et la manipulation des conteneurs.

Le tri

Le tri est une opération courante qui permet de réorganiser les éléments d'un conteneur dans un ordre croissant ou décroissant. La bibliothèque standard du C++ propose l'algorithme de tri std::sort qui peut être utilisé avec les vecteurs et les listes. Voici un exemple d'utilisation du tri :

#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers = {3, 1, 2};

    std::sort(numbers.begin(), numbers.end());

    for (int number : numbers) {
        std::cout << number << " ";
    }

    return 0;
}

La recherche

La recherche est une opération qui permet de trouver un élément spécifique dans un conteneur. La bibliothèque standard du C++ propose l'algorithme de recherche std::find qui peut être utilisé avec les vecteurs et les listes. Voici un exemple d'utilisation de la recherche :

#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers = {1, 2, 3};

    std::vector<int>::iterator it = std::find(numbers.begin(), numbers.end(), 2);

    if (it != numbers.end()) {
        std::cout << "Element found";
    } else {
        std::cout << "Element not found";
    }

    return 0;
}

La manipulation de conteneurs

La bibliothèque standard du C++ propose également d'autres algorithmes de manipulation des conteneurs, tels que l'insertion, la suppression et la transformation des éléments. Voici quelques exemples :

#include <algorithm>
#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers = {1, 2, 3};

    // Insertion
    numbers.insert(numbers.begin() + 1, 4);

    // Suppression
    numbers.erase(numbers.begin() + 2);

    // Transformation
    std::transform(numbers.begin(), numbers.end(), numbers.begin(), [](int n) { return n * 2; });

    for (int number : numbers) {
        std::cout << number << " ";
    }

    return 0;
}

Les entrées/sorties (fichiers, flux)

Les entrées/sorties (E/S) sont des opérations qui permettent de lire des données à partir d'un flux d'entrée (par exemple, un fichier) et d'écrire des données dans un flux de sortie (par exemple, la console). La bibliothèque standard du C++ propose des classes et des fonctions pour effectuer des opérations d'E/S.

Les fichiers

La classe std::fstream permet de manipuler des fichiers en C++. Voici un exemple d'ouverture d'un fichier en lecture :

#include <fstream>
#include <iostream>

int main() {
    std::ifstream file("example.txt");

    if (file.is_open()) {
        std::string line;
        while (std::getline(file, line)) {
            std::cout << line << std::endl;
        }
        file.close();
    } else {
        std::cout << "Unable to open file";
    }

    return 0;
}

Les flux

Les flux d'entrée (std::istream) et de sortie (std::ostream) permettent de lire et d'écrire des données à partir de la console ou d'autres flux. Voici un exemple d'utilisation des flux :

#include <iostream>

int main() {
    int number;
    std::cout << "Enter a number: ";
    std::cin >> number;
    std::cout << "You entered: " << number;

    return 0;
}

Résultat :

Enter a number: 5
You entered: 5

Conclusion et perspectives

Les ressources supplémentaires pour approfondir ses connaissances en C++

Pour approfondir vos connaissances en C++, voici quelques ressources supplémentaires recommandées :

  • Les livres spécialisés sur le langage C++ tels que "The C++ Programming Language" de Bjarne Stroustrup.
  • Les tutoriels en ligne et les forums de discussion dédiés au C++.
  • Les documentations officielles du langage C++ et de la bibliothèque standard.

Les opportunités professionnelles liées à la maîtrise du langage C++

La maîtrise du langage C++ ouvre de nombreuses opportunités professionnelles dans le domaine du développement logiciel. Voici quelques exemples de carrières possibles :

  • Développeur C++
  • Ingénieur logiciel
  • Architecte logiciel
  • Analyste de données

Les prochaines étapes pour continuer à progresser en C++

Pour continuer à progresser en C++, voici quelques suggestions :

  • Participer à des projets open source en C++ pour acquérir de l'expérience pratique.
  • Se tenir au courant des nouvelles versions du langage C++ et des nouvelles fonctionnalités.
  • Participer à des formations et des conférences sur le C++ pour rester à jour sur les bonnes pratiques et les tendances du secteur.
