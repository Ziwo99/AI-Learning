Les concepts de probabilité

La probabilité est une mesure de la possibilité qu'un événement se produise. Elle est souvent exprimée comme un nombre entre 0 et 1, où 0 signifie que l'événement est impossible et 1 signifie qu'il est certain. La probabilité peut également être exprimée en pourcentage.

Il existe deux types de probabilités : la probabilité théorique et la probabilité empirique. La probabilité théorique est basée sur des calculs mathématiques et des modèles probabilistes, tandis que la probabilité empirique est basée sur des observations et des expériences réelles.

Les concepts clés de la probabilité comprennent l'événement, l'espace des échantillons, l'événement complémentaire, l'événement indépendant et l'événement conditionnel. Un événement est un résultat possible d'une expérience aléatoire. L'espace des échantillons est l'ensemble de tous les résultats possibles de l'expérience aléatoire. L'événement complémentaire est l'événement qui ne se produit pas. Deux événements sont indépendants si la probabilité de l'un n'est pas affectée par la réalisation de l'autre. Un événement conditionnel est la probabilité qu'un événement se produise, sachant que l'autre événement s'est déjà produit.

Les distributions de probabilité discrètes

Une distribution de probabilité discrète est une fonction qui associe à chaque valeur possible d'une variable aléatoire discrète une probabilité. Une variable aléatoire discrète est une variable qui peut prendre un nombre fini ou dénombrable de valeurs distinctes.

Les distributions de probabilité discrètes les plus couramment utilisées sont la distribution uniforme, la distribution de Bernoulli, la distribution binomiale, la distribution de Poisson et la distribution géométrique.

La distribution uniforme attribue la même probabilité à chaque valeur possible de la variable aléatoire. La distribution de Bernoulli est utilisée pour modéliser une expérience aléatoire avec deux résultats possibles, généralement notés 0 et 1. La distribution binomiale est utilisée pour modéliser le nombre de succès dans un certain nombre d'essais indépendants et identiquement distribués. La distribution de Poisson est utilisée pour modéliser le nombre d'événements rares qui se produisent dans un intervalle de temps ou d'espace donné. La distribution géométrique est utilisée pour modéliser le nombre d'essais nécessaires pour obtenir le premier succès dans une série d'essais indépendants et identiquement distribués.

Les distributions de probabilité continues

Une distribution de probabilité continue est une fonction qui associe à chaque valeur possible d'une variable aléatoire continue une probabilité. Une variable aléatoire continue est une variable qui peut prendre n'importe quelle valeur dans un intervalle donné.

Les distributions de probabilité continues les plus couramment utilisées sont la distribution normale, la distribution exponentielle et la distribution uniforme continue.

La distribution normale, également connue sous le nom de courbe en cloche, est la distribution de probabilité la plus importante en statistique. Elle est caractérisée par sa moyenne et son écart-type. La distribution exponentielle est utilisée pour modéliser le temps entre les événements rares et indépendants qui se produisent de manière continue. La distribution uniforme continue attribue la même probabilité à chaque valeur possible de la variable aléatoire dans un intervalle donné.

Voici un exemple de code pour générer des valeurs aléatoires à partir d'une distribution normale :

import numpy as np

mean = 0
std = 1
size = 1000

samples = np.random.normal(mean, std, size)
print(samples)
