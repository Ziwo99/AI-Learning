Section 13: Développement web avec Django

Introduction à Django

Django est un framework web puissant et flexible écrit en Python. Il est conçu pour faciliter le développement rapide d'applications web robustes et sécurisées. Dans cette section, nous allons explorer les bases de Django et apprendre comment créer des applications web en utilisant ce framework.

Pour commencer, vous devez installer Django sur votre machine. Vous pouvez le faire en utilisant la commande suivante :

pip install django

Une fois que Django est installé, vous pouvez créer un nouveau projet Django en utilisant la commande suivante :

django-admin startproject nom_du_projet

Cela créera un nouveau répertoire contenant les fichiers de base d'un projet Django.

Les modèles et les vues

Dans Django, les modèles sont utilisés pour définir la structure et le comportement des données de votre application. Les vues sont responsables de la logique métier et de l'interaction avec les modèles et les templates.

Pour créer un modèle dans Django, vous devez définir une classe qui hérite de la classe Model. Voici un exemple :

from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
    pub_date = models.DateTimeField(auto_now_add=True)

Dans cet exemple, nous avons créé un modèle Article avec trois champs : title, content et pub_date. Le champ pub_date est automatiquement défini avec la date et l'heure actuelles lorsqu'un nouvel article est créé.

Pour créer une vue dans Django, vous devez définir une fonction qui prend une requête HTTP en paramètre et renvoie une réponse HTTP. Voici un exemple :

from django.http import HttpResponse

def hello(request):
    return HttpResponse("Hello, world!")

Dans cet exemple, nous avons créé une vue simple qui renvoie une réponse "Hello, world!" lorsque la fonction est appelée.

Les formulaires et les templates

Dans Django, les formulaires sont utilisés pour collecter les données des utilisateurs. Les templates sont utilisés pour afficher les données et générer des pages web dynamiques.

Pour créer un formulaire dans Django, vous devez définir une classe qui hérite de la classe Form. Voici un exemple :

from django import forms

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

Dans cet exemple, nous avons créé un formulaire de contact avec trois champs : name, email et message.

Pour utiliser un template dans Django, vous devez créer un fichier HTML contenant le code HTML de votre page. Vous pouvez utiliser des balises spéciales pour insérer des variables et des boucles dans votre template. Voici un exemple :

<html>
<head>
    <title>My Blog</title>
</head>
<body>
    <h1>{{ title }}</h1>
    
    <ul>
    {% for article in articles %}
        <li>{{ article.title }} - {{ article.pub_date }}</li>
    {% endfor %}
    </ul>
</body>
</html>

Dans cet exemple, nous avons utilisé des balises spéciales comme {{ title }} et {% for article in articles %} pour insérer des variables et des boucles dans notre template.

C'est tout pour cette section sur le développement web avec Django. Vous avez maintenant une bonne compréhension des bases de Django et vous êtes prêt à créer des applications web puissantes en utilisant ce framework.
