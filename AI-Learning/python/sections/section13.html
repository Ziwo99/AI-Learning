Section 13 : Ressources supplémentaires

Retour au cours

Livres recommandés

Voici quelques livres recommandés pour approfondir vos connaissances en Python :

  • "Apprenez à programmer en Python" par Gérard Swinnen
  • "Python Crash Course" par Eric Matthes
  • "Automate the Boring Stuff with Python" par Al Sweigart
  • "Fluent Python" par Luciano Ramalho

Sites web et forums utiles

Voici quelques sites web et forums où vous pouvez trouver des ressources supplémentaires et poser vos questions :

  • Site officiel de Python
  • Documentation officielle de Python
  • Stack Overflow - Une communauté de développeurs où vous pouvez poser des questions et trouver des réponses
  • r/learnpython - Un subreddit dédié à l'apprentissage de Python

Communautés Python

Voici quelques communautés en ligne où vous pouvez interagir avec d'autres développeurs Python :

  • Python Community - La communauté officielle de Python
  • PythonAnywhere - Une plateforme d'hébergement Python avec une communauté active
  • Meetup.com - Recherchez des groupes locaux Python près de chez vous
  • Python Software Foundation (PSF) - L'organisation à but non lucratif qui soutient le développement de Python
