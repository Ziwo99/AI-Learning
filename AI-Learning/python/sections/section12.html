Section 12: Les projets pratiques en Python

Projet 1: Création d'un jeu de devinettes

Dans ce projet, nous allons créer un jeu de devinettes en utilisant Python. Le jeu fonctionnera de la manière suivante :

  • Le programme choisira un nombre aléatoire entre 1 et 100.
  • L'utilisateur devra deviner le nombre choisi par le programme.
  • Le programme donnera des indications à l'utilisateur pour l'aider à trouver le nombre.
  • Le jeu se terminera lorsque l'utilisateur aura deviné le nombre correctement.

Voici un exemple de code pour implémenter ce jeu de devinettes :

import random

number_to_guess = random.randint(1, 100)
guess = 0
attempts = 0

while guess != number_to_guess:
    guess = int(input("Devinez le nombre : "))
    attempts += 1
    
    if guess < number_to_guess:
        print("Le nombre à deviner est plus grand.")
    elif guess > number_to_guess:
        print("Le nombre à deviner est plus petit.")

print("Bravo ! Vous avez deviné le nombre en", attempts, "tentatives.")

Projet 2: Création d'un gestionnaire de tâches

Dans ce projet, nous allons créer un gestionnaire de tâches en utilisant Python. Le gestionnaire de tâches permettra à l'utilisateur de :

  • Ajouter une tâche avec une description et une date d'échéance.
  • Afficher toutes les tâches enregistrées.
  • Marquer une tâche comme terminée.
  • Supprimer une tâche de la liste.

Voici un exemple de code pour implémenter ce gestionnaire de tâches :

tasks = []

def add_task(description, deadline):
    task = {"description": description, "deadline": deadline, "done": False}
    tasks.append(task)

def display_tasks():
    for task in tasks:
        print("Description:", task["description"])
        print("Deadline:", task["deadline"])
        print("Done:", task["done"])
        print()

def mark_task_as_done(index):
    if index >= 0 and index < len(tasks):
        tasks[index]["done"] = True

def remove_task(index):
    if index >= 0 and index < len(tasks):
        del tasks[index]

add_task("Faire les courses", "2022-01-01")
add_task("Apprendre Python", "2022-02-01")
display_tasks()

mark_task_as_done(0)
remove_task(1)
display_tasks()

Projet 3: Création d'un convertisseur de devises

Dans ce projet, nous allons créer un convertisseur de devises en utilisant Python. Le convertisseur de devises permettra à l'utilisateur de :

  • Entrer un montant dans une devise de départ.
  • Choisir une devise cible dans laquelle convertir le montant.
  • Afficher le montant converti.

Voici un exemple de code pour implémenter ce convertisseur de devises :

exchange_rates = {
    "USD": 1.18,
    "EUR": 1,
    "GBP": 0.85,
    "JPY": 130.68
}

def convert_currency(amount, from_currency, to_currency):
    if from_currency in exchange_rates and to_currency in exchange_rates:
        converted_amount = amount * exchange_rates[to_currency] / exchange_rates[from_currency]
        return converted_amount
    else:
        return None

amount = float(input("Entrez le montant à convertir : "))
from_currency = input("Entrez la devise de départ : ")
to_currency = input("Entrez la devise cible : ")

converted_amount = convert_currency(amount, from_currency, to_currency)

if converted_amount is not None:
    print("Le montant converti est :", converted_amount)
else:
    print("Les devises spécifiées ne sont pas prises en charge.")
