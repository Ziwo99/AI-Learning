Section 12: Développement web avec Flask

Introduction à Flask

Flask est un framework web léger et flexible pour Python. Il est utilisé pour créer des applications web rapidement et facilement. Flask suit le principe de "micro-framework", ce qui signifie qu'il fournit uniquement les fonctionnalités de base nécessaires pour développer des applications web. Cela permet aux développeurs d'avoir un contrôle total sur leur application et de choisir les extensions et les bibliothèques qu'ils souhaitent utiliser.

Pour commencer à utiliser Flask, vous devez d'abord l'installer. Voici comment procéder :

pip install flask

Une fois Flask installé, vous pouvez créer une application Flask en quelques étapes simples :

  1. Importez la classe Flask :
    2. from flask import Flask
  2. Créez une instance de l'application Flask :
    3. app = Flask(__name__)
  3. Définissez les routes et les vues de votre application :
    4. @app.route('/')
def index():
    return 'Hello, World!'
  4. Exécutez l'application Flask :
    5. if __name__ == '__main__':
    app.run()

Les routes et les vues

Les routes et les vues sont des éléments essentiels dans le développement web avec Flask. Les routes définissent les URL de votre application et les vues sont les fonctions qui sont exécutées lorsque ces URL sont visitées.

Voici comment définir une route et une vue dans Flask :

@app.route('/hello')
def hello():
    return 'Hello, Flask!'

Dans cet exemple, la route "/hello" est définie et la fonction "hello" est associée à cette route. Lorsque l'URL "/hello" est visitée, la fonction "hello" est exécutée et renvoie la chaîne de caractères "Hello, Flask!".

Vous pouvez également utiliser des paramètres dans les routes pour rendre votre application plus dynamique :

@app.route('/user/<username>')
def show_user(username):
    return 'Hello, ' + username + '!'

Dans cet exemple, la route "/user/<username>" est définie avec un paramètre "username". Lorsque l'URL "/user/john" est visitée, la fonction "show_user" est exécutée et renvoie la chaîne de caractères "Hello, john!".

Les templates et les formulaires

Les templates et les formulaires sont des éléments importants pour rendre vos applications Flask plus interactives et personnalisées.

Les templates sont des fichiers HTML qui permettent de séparer la logique de présentation de votre application Flask. Ils peuvent contenir des variables et des boucles pour afficher dynamiquement les données.

Voici comment utiliser un template dans Flask :

from flask import render_template

@app.route('/hello/<name>')
def hello(name):
    return render_template('hello.html', name=name)

Dans cet exemple, la fonction "hello" renvoie le contenu du template "hello.html" en utilisant la fonction "render_template". La variable "name" est passée au template et peut être utilisée pour afficher le nom dans le fichier HTML.

Les formulaires permettent aux utilisateurs d'envoyer des données à votre application Flask. Vous pouvez utiliser la bibliothèque Flask-WTF pour gérer les formulaires de manière simple et sécurisée.

Voici comment créer un formulaire dans Flask :

from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField

class MyForm(FlaskForm):
    name = StringField('Name')
    submit = SubmitField('Submit')

Dans cet exemple, nous avons créé un formulaire "MyForm" avec un champ de texte "name" et un bouton de soumission "submit". Vous pouvez personnaliser les champs et les boutons selon vos besoins.

Pour afficher le formulaire dans votre application Flask, vous pouvez utiliser la fonction "render_template" :

@app.route('/form', methods=['GET', 'POST'])
def form():
    form = MyForm()
    if form.validate_on_submit():
        return 'Form submitted successfully!'
    return render_template('form.html', form=form)

Dans cet exemple, la fonction "form" renvoie le contenu du template "form.html" avec le formulaire "MyForm". Lorsque le formulaire est soumis avec succès, la chaîne de caractères "Form submitted successfully!" est renvoyée.

Ce sont les bases du développement web avec Flask. Avec ces connaissances, vous pouvez créer des applications web puissantes et personnalisées en utilisant Python et Flask.
