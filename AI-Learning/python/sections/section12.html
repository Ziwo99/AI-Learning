Section 12 : Projets pratiques en Python

Projet 1 : Création d'un jeu simple

Dans ce projet, nous allons créer un jeu simple en utilisant Python. Le jeu consistera à deviner un nombre choisi aléatoirement par l'ordinateur. Voici comment nous allons procéder :

Tout d'abord, nous devons importer le module random pour générer un nombre aléatoire :

import random

Ensuite, nous allons définir une fonction qui génère un nombre aléatoire entre 1 et 100 :

def generate_random_number():
    return random.randint(1, 100)

Ensuite, nous allons définir une fonction principale qui permettra à l'utilisateur de deviner le nombre :

def guess_number():
    random_number = generate_random_number()
    guessed = False

    while not guessed:
        user_guess = int(input("Devinez le nombre : "))

        if user_guess == random_number:
            print("Bravo, vous avez deviné le nombre !")
            guessed = True
        elif user_guess < random_number:
            print("Le nombre est plus grand.")
        else:
            print("Le nombre est plus petit.")

Enfin, nous allons appeler la fonction principale pour lancer le jeu :

guess_number()

Projet 2 : Analyse de données avec Pandas

Dans ce projet, nous allons utiliser la bibliothèque Pandas pour effectuer une analyse de données. Nous allons travailler avec un ensemble de données contenant des informations sur des étudiants :

Tout d'abord, nous devons importer la bibliothèque Pandas :

import pandas as pd

Ensuite, nous allons charger les données à partir d'un fichier CSV :

data = pd.read_csv('students.csv')

Nous pouvons maintenant effectuer différentes opérations sur les données, par exemple :

Pour afficher les premières lignes du tableau :

print(data.head())

Pour calculer la moyenne des notes :

average_grade = data['Grade'].mean()
print("La moyenne des notes est de :", average_grade)

Pour filtrer les étudiants ayant obtenu une note supérieure à 90 :

top_students = data[data['Grade'] > 90]
print(top_students)

Projet 3 : Création d'un site web avec Flask

Dans ce projet, nous allons utiliser le framework Flask pour créer un site web simple en Python. Voici les étapes à suivre :

Tout d'abord, nous devons installer Flask :

pip install flask

Ensuite, nous allons importer les modules nécessaires :

from flask import Flask, render_template

Ensuite, nous allons créer une instance de l'application Flask :

app = Flask(__name__)

Nous allons ensuite définir une route pour la page d'accueil :

@app.route('/')
def home():
    return render_template('index.html')

Enfin, nous allons lancer l'application :

if __name__ == '__main__':
    app.run()

Assurez-vous d'avoir un fichier HTML nommé "index.html" dans le même répertoire que votre script Python, contenant le contenu de votre page d'accueil.
