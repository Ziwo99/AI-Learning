Section 12 : Projets pratiques en Python

Création d'un jeu en Python

Dans cette sous-section, nous allons apprendre à créer un jeu simple en utilisant Python. Nous allons utiliser les concepts que nous avons appris tout au long de cette formation pour construire notre jeu.

Voici un exemple de code pour créer un jeu de devinette en Python :

import random

number = random.randint(1, 100)
guess = int(input("Devinez le nombre entre 1 et 100 : "))

while guess != number:
    if guess < number:
        print("Trop bas")
    else:
        print("Trop haut")
    guess = int(input("Devinez le nombre entre 1 et 100 : "))

print("Bravo, vous avez deviné le nombre !")

Dans ce jeu, l'ordinateur choisit un nombre aléatoire entre 1 et 100, puis le joueur doit deviner ce nombre en entrant des valeurs. L'ordinateur donne des indications "Trop bas" ou "Trop haut" en fonction de la valeur entrée par le joueur. Le jeu continue jusqu'à ce que le joueur devine le nombre correctement.

Développement d'une application web simple

Dans cette sous-section, nous allons apprendre à développer une application web simple en utilisant le framework Flask en Python. Flask est un framework léger et facile à utiliser pour le développement web en Python.

Voici un exemple de code pour créer une application web simple avec Flask :

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return "Bonjour, bienvenue sur mon site web !"

if __name__ == '__main__':
    app.run()

Dans cet exemple, nous importons la classe Flask du module flask. Ensuite, nous créons une instance de l'application Flask et définissons une route pour la page d'accueil ("/"). Lorsque l'utilisateur accède à cette route, la fonction hello() est appelée et renvoie le message "Bonjour, bienvenue sur mon site web !". Enfin, nous exécutons l'application en utilisant la méthode run().

Analyse de données en utilisant Pandas

Dans cette sous-section, nous allons apprendre à utiliser la bibliothèque Pandas en Python pour l'analyse de données. Pandas est une bibliothèque puissante et populaire pour la manipulation et l'analyse de données en Python.

Voici un exemple de code pour effectuer une analyse de données simple en utilisant Pandas :

import pandas as pd

data = {'Nom': ['Alice', 'Bob', 'Charlie', 'David'],
        'Âge': [25, 30, 35, 40],
        'Ville': ['Paris', 'Londres', 'New York', 'Tokyo']}

df = pd.DataFrame(data)

print(df)

Dans cet exemple, nous créons un DataFrame à partir d'un dictionnaire de données. Le DataFrame est une structure de données tabulaire bidimensionnelle avec des colonnes étiquetées. Nous affichons ensuite le DataFrame à l'aide de la fonction print().

Ce sont là quelques exemples de projets pratiques que vous pouvez réaliser en utilisant Python. N'hésitez pas à explorer davantage ces sujets et à créer vos propres projets pour approfondir vos connaissances en Python.
