Projet pratique 1 : Création d'un jeu en Python

Dans ce projet pratique, vous allez créer un jeu en utilisant Python. Vous apprendrez à concevoir et à implémenter les mécanismes de jeu, ainsi qu'à gérer les entrées utilisateur et à afficher les résultats.

Voici un exemple de code pour un jeu simple de devinette de nombre :

import random

def guess_number():
    secret_number = random.randint(1, 100)
    attempts = 0

    while True:
        user_guess = int(input("Devinez le nombre : "))
        attempts += 1

        if user_guess == secret_number:
            print("Bravo, vous avez deviné le nombre en", attempts, "tentatives !")
            break
        elif user_guess < secret_number:
            print("Le nombre est plus grand.")
        else:
            print("Le nombre est plus petit.")

guess_number()

Dans cet exemple, le programme choisit un nombre aléatoire entre 1 et 100, puis demande à l'utilisateur de deviner ce nombre. Le programme compare ensuite la réponse de l'utilisateur avec le nombre secret et fournit des indications pour aider l'utilisateur à se rapprocher du nombre secret. Le jeu continue jusqu'à ce que l'utilisateur devine le nombre correctement.

Projet pratique 3 : Analyse de données en Python

Dans ce projet pratique, vous allez apprendre à utiliser Python pour analyser des données. Vous découvrirez comment importer des données à partir de fichiers, les nettoyer, les manipuler et les visualiser.

Voici un exemple de code pour analyser un fichier CSV contenant des données sur les ventes :

import pandas as pd

# Importation des données à partir du fichier CSV
data = pd.read_csv('sales_data.csv')

# Affichage des premières lignes des données
print(data.head())

# Calcul de la somme des ventes
total_sales = data['Sales'].sum()
print("Total des ventes :", total_sales)

# Calcul de la moyenne des ventes par mois
monthly_sales = data.groupby('Month')['Sales'].mean()
print("Ventes mensuelles moyennes :", monthly_sales)

Dans cet exemple, le programme utilise la bibliothèque pandas pour importer les données à partir d'un fichier CSV. Ensuite, il affiche les premières lignes des données, calcule la somme totale des ventes et la moyenne des ventes par mois. Ces informations peuvent ensuite être utilisées pour prendre des décisions commerciales éclairées.

Ce projet pratique vous permettra de développer vos compétences en analyse de données et de découvrir les nombreuses possibilités offertes par Python dans ce domaine.
