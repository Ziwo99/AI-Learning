Projet 1 : Création d'un jeu en Python

Dans ce projet, nous allons créer un jeu simple en utilisant Python. Le jeu consistera à deviner un nombre aléatoire généré par l'ordinateur. Voici les étapes pour créer ce jeu :

1. Générer un nombre aléatoire entre 1 et 100.

import random

nombre_secret = random.randint(1, 100)

2. Demander à l'utilisateur de deviner le nombre.

nombre_devine = int(input("Devinez le nombre : "))

3. Vérifier si le nombre deviné est correct.

if nombre_devine == nombre_secret:
    print("Félicitations ! Vous avez deviné le nombre.")
else:
    print("Désolé, ce n'est pas le bon nombre. Le nombre était", nombre_secret)

4. Répéter les étapes 2 et 3 jusqu'à ce que le nombre deviné soit correct.

while nombre_devine != nombre_secret:
    nombre_devine = int(input("Devinez le nombre : "))
    
    if nombre_devine == nombre_secret:
        print("Félicitations ! Vous avez deviné le nombre.")
        break
    else:
        print("Désolé, ce n'est pas le bon nombre. Veuillez réessayer.")

Projet 2 : Création d'une application web en Python

Dans ce projet, nous allons créer une application web simple en utilisant Python et le framework Flask. L'application consistera en une page d'accueil et une page de contact. Voici les étapes pour créer cette application :

1. Installer Flask en utilisant la commande suivante :

pip install flask

2. Importer Flask et créer une instance de l'application.

from flask import Flask

app = Flask(__name__)

3. Créer une route pour la page d'accueil.

@app.route('/')
def home():
    return "Bienvenue sur la page d'accueil !"

4. Créer une route pour la page de contact.

@app.route('/contact')
def contact():
    return "Voici nos coordonnées de contact : ..."

5. Exécuter l'application.

if __name__ == '__main__':
    app.run()

Projet 3 : Création d'un programme de gestion de base de données en Python

Dans ce projet, nous allons créer un programme de gestion de base de données en utilisant Python et le module sqlite3. Le programme permettra de créer une base de données, d'ajouter des données, de les modifier et de les supprimer. Voici les étapes pour créer ce programme :

1. Importer le module sqlite3.

import sqlite3

2. Créer une connexion à la base de données.

conn = sqlite3.connect('ma_base_de_donnees.db')

3. Créer une table dans la base de données.

conn.execute('''CREATE TABLE utilisateurs
                (id INT PRIMARY KEY NOT NULL,
                nom TEXT NOT NULL,
                age INT NOT NULL);''')

4. Insérer des données dans la table.

conn.execute("INSERT INTO utilisateurs (id, nom, age) VALUES (1, 'John Doe', 25)")

5. Modifier des données dans la table.

conn.execute("UPDATE utilisateurs SET age = 30 WHERE id = 1")

6. Supprimer des données de la table.

conn.execute("DELETE FROM utilisateurs WHERE id = 1")

7. Fermer la connexion à la base de données.

conn.close()

Ce projet vous permettra de comprendre les bases de la création et de la gestion d'une base de données en utilisant Python.
