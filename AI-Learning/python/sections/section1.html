Section 1 : Introduction à Python

Historique et évolution de Python

Python a été créé par Guido van Rossum et a été initialement publié en 1991. Le langage tire son nom de la série télévisée britannique "Monty Python's Flying Circus", dont Guido était fan. Depuis sa création, Python a connu une évolution constante et est devenu l'un des langages de programmation les plus populaires au monde.

Installation de Python

Pour commencer à utiliser Python, vous devez d'abord installer l'interpréteur Python sur votre ordinateur. Voici les étapes à suivre pour installer Python :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org) et téléchargez la dernière version de Python.
  2. Exécutez le programme d'installation téléchargé et suivez les instructions à l'écran.
  3. Assurez-vous de cocher la case "Add Python to PATH" lors de l'installation pour pouvoir exécuter Python depuis n'importe quel répertoire de votre système.
  4. Une fois l'installation terminée, vous pouvez ouvrir l'invite de commande (ou le terminal) et exécuter la commande "python" pour vérifier que Python est correctement installé.

Premier programme en Python

Maintenant que Python est installé, vous pouvez écrire votre premier programme en Python. Voici un exemple simple de programme Python qui affiche "Hello, World!" à l'écran : 

print("Hello, World!")

Pour exécuter ce programme, enregistrez-le dans un fichier avec l'extension ".py" (par exemple, "hello.py") et exécutez-le en utilisant la commande "python" suivie du nom du fichier : 

python hello.py

Vous devriez voir la phrase "Hello, World!" s'afficher à l'écran.
