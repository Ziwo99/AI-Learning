Section 1: Introduction à Python

Introduction à Python

Python est un langage de programmation interprété, polyvalent et facile à apprendre. Il a été créé par Guido van Rossum et publié pour la première fois en 1991. Python se distingue par sa syntaxe claire et lisible, ce qui en fait un excellent choix pour les débutants en programmation.

Installation de Python

Pour commencer à programmer en Python, vous devez d'abord installer Python sur votre ordinateur. Voici les étapes pour installer Python :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org) et téléchargez la dernière version de Python.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Vérifiez que Python est correctement installé en ouvrant une fenêtre de terminal et en tapant "python --version". Vous devriez voir s'afficher la version de Python installée.

Les bases de la programmation en Python

Python offre de nombreuses fonctionnalités pour la programmation, y compris les types de données, les opérations arithmétiques, les structures de contrôle et les fonctions. Voici quelques exemples :

Types de données :

nombre_entier = 10
nombre_decimal = 3.14
chaine_de_caracteres = "Bonjour"
liste = [1, 2, 3, 4, 5]
dictionnaire = {"clé1": "valeur1", "clé2": "valeur2"}

Opérations arithmétiques :

a = 10
b = 5

addition = a + b
soustraction = a - b
multiplication = a * b
division = a / b
modulo = a % b
exposant = a ** b

Structures de contrôle :

if condition:
    # instructions à exécuter si la condition est vraie
elif autre_condition:
    # instructions à exécuter si une autre condition est vraie
else:
    # instructions à exécuter si aucune des conditions précédentes n'est vraie

Fonctions :

def ma_fonction(parametre1, parametre2):
    # instructions de la fonction
    return resultat

Ces exemples ne sont qu'un aperçu des fonctionnalités de base de Python. Au fur et à mesure de votre apprentissage, vous découvrirez de nombreuses autres fonctionnalités puissantes offertes par Python.
