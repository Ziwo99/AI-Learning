Section 1: Introduction à Python

Introduction à Python

Python est un langage de programmation interprété, polyvalent et facile à apprendre. Il a été créé par Guido van Rossum et publié pour la première fois en 1991. Python est connu pour sa syntaxe claire et lisible, ce qui en fait un excellent choix pour les débutants en programmation.

Installation de Python

Pour commencer à programmer en Python, vous devez d'abord installer Python sur votre ordinateur. Voici les étapes à suivre :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org) et téléchargez la dernière version de Python.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Vérifiez que Python est correctement installé en ouvrant une fenêtre de terminal et en tapant "python --version". Vous devriez voir s'afficher la version de Python installée.

Les bases de la programmation en Python

Python est un langage de programmation polyvalent qui peut être utilisé pour diverses tâches, telles que le développement web, l'analyse de données, l'intelligence artificielle, etc. Voici quelques concepts de base que vous devez connaître :

Variables

En Python, vous pouvez stocker des valeurs dans des variables. Une variable est un espace de mémoire réservé pour stocker une valeur. Voici un exemple :

message = "Bonjour, monde!"
print(message)

Types de données

Python prend en charge plusieurs types de données, tels que les entiers, les flottants, les chaînes de caractères, les listes, les tuples, les dictionnaires, etc. Voici quelques exemples :

age = 25
pi = 3.14
name = "John Doe"
fruits = ["pomme", "banane", "orange"]
person = {"nom": "John", "âge": 30}

Structures de contrôle

Python offre différentes structures de contrôle pour exécuter des instructions conditionnelles et répétitives. Voici quelques exemples :

Instructions conditionnelles (if-else)

age = 18

if age >= 18:
    print("Vous êtes majeur.")
else:
    print("Vous êtes mineur.")

Boucles (for)

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

Boucles (while)

i = 0

while i < 5:
    print(i)
    i += 1

Fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Voici un exemple :

def say_hello(name):
    print("Bonjour, " + name + "!")

say_hello("Alice")

Modules

Python dispose d'une vaste bibliothèque de modules qui étendent les fonctionnalités de base du langage. Vous pouvez importer ces modules dans votre programme pour utiliser leurs fonctionnalités. Voici un exemple :

import math

print(math.sqrt(16))

Ceci conclut la section d'introduction à Python. Vous avez maintenant une compréhension de base du langage et de ses fonctionnalités principales. Dans les sections suivantes, nous approfondirons chaque concept et explorerons des sujets plus avancés.
