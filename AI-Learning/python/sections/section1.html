Introduction à Python

Python est un langage de programmation interprété, orienté objet et de haut niveau. Il a été créé par Guido van Rossum et publié pour la première fois en 1991. Python est connu pour sa syntaxe simple et lisible, ce qui en fait un excellent choix pour les débutants en programmation.

Python est utilisé dans de nombreux domaines, tels que le développement web, l'analyse de données, l'intelligence artificielle, l'automatisation de tâches et bien plus encore. Il est également populaire en raison de sa grande communauté de développeurs, qui contribuent à son écosystème en créant des bibliothèques et des frameworks.

Installation de Python

Pour commencer à programmer en Python, vous devez d'abord installer Python sur votre ordinateur. Voici les étapes pour installer Python :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org) et téléchargez la dernière version de Python pour votre système d'exploitation.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Assurez-vous de cocher la case "Add Python to PATH" lors de l'installation. Cela permettra d'accéder facilement à Python depuis n'importe quel répertoire de votre ordinateur.
  4. Une fois l'installation terminée, ouvrez une fenêtre de terminal (ou invite de commande) et tapez "python". Si Python est correctement installé, vous devriez voir la version de Python s'afficher.

Les bases de la syntaxe Python

Avant de commencer à écrire du code Python, il est important de comprendre les bases de la syntaxe Python. Voici quelques éléments clés de la syntaxe Python :

  • Python utilise l'indentation pour délimiter les blocs de code. Les blocs de code sont généralement indentés avec 4 espaces.
  • Les commentaires commencent par le symbole "#" et sont utilisés pour ajouter des explications ou des notes dans le code.
  • Python est sensible à la casse, ce qui signifie que les noms de variables "maVariable" et "mavariable" sont considérés comme différents.
  • Les lignes de code se terminent généralement par un point-virgule, mais ce n'est pas obligatoire.

Les variables et les types de données

En Python, les variables sont utilisées pour stocker des valeurs. Voici quelques types de données couramment utilisés en Python :

  • Les entiers (int) : représentent les nombres entiers, tels que 1, 2, -5, etc.
  • Les nombres à virgule flottante (float) : représentent les nombres décimaux, tels que 3.14, -0.5, etc.
  • Les chaînes de caractères (str) : représentent du texte, tel que "Bonjour", 'Python', etc.
  • Les booléens (bool) : représentent les valeurs de vérité, True (vrai) ou False (faux).

Voici comment déclarer et utiliser des variables en Python :

x = 5;
y = 3.14;
message = "Bonjour";
isTrue = True;

Les opérations arithmétiques et les opérateurs logiques

Python prend en charge les opérations arithmétiques de base, telles que l'addition, la soustraction, la multiplication et la division. Voici quelques exemples d'opérations arithmétiques en Python :

a = 5 + 3;  # addition
b = 10 - 2;  # soustraction
c = 4 * 6;  # multiplication
d = 20 / 5;  # division

Python prend également en charge les opérateurs logiques, tels que "et" (and), "ou" (or) et "non" (not). Ces opérateurs sont utilisés pour évaluer des expressions booléennes. Voici quelques exemples d'opérations logiques en Python :

a = True and False;  # False
b = True or False;  # True
c = not True;  # False

Les structures de contrôle (boucles et conditions)

Les structures de contrôle sont utilisées pour contrôler le flux d'exécution d'un programme. En Python, nous avons les structures de contrôle suivantes :

  • Les boucles "while" : exécutent un bloc de code tant qu'une condition est vraie.
  • Les boucles "for" : itèrent sur une séquence d'éléments.
  • Les instructions "if", "else" et "elif" : permettent d'exécuter un bloc de code en fonction d'une condition.

Voici quelques exemples d'utilisation des structures de contrôle en Python :

i = 0;

while i < 10:
    print("Iteration", i);
    i += 1;

for i in range(5):
    print("Iteration", i);

x = 5;

if x > 0:
    print("x est positif");
elif x < 0:
    print("x est négatif");
else:
    print("x est nul");

Les fonctions et les modules

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. En Python, nous pouvons définir nos propres fonctions en utilisant le mot-clé "def". Voici un exemple de fonction en Python :

def say_hello():
    print("Bonjour");

say_hello();

Les modules sont des fichiers contenant du code Python supplémentaire. Ils sont utilisés pour organiser et réutiliser du code. Pour utiliser un module dans votre programme, vous devez l'importer. Voici un exemple d'importation de module en Python :

import math;

x = math.sqrt(25);
print(x);

Vous pouvez également importer des fonctions spécifiques d'un module :

from math import sqrt;

x = sqrt(25);
print(x);
