Section 1: Introduction à Python

Introduction à Python

Python est un langage de programmation interprété, polyvalent et facile à apprendre. Il a été créé par Guido van Rossum et est largement utilisé dans le développement web, l'analyse de données, l'intelligence artificielle et bien d'autres domaines. Python se distingue par sa syntaxe claire et lisible, ce qui en fait un excellent choix pour les débutants en programmation.

Installation de Python

Pour commencer à utiliser Python, vous devez d'abord l'installer sur votre ordinateur. Voici les étapes à suivre :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org/) et téléchargez la dernière version stable de Python.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Vérifiez que Python est correctement installé en ouvrant une fenêtre de terminal (ou une invite de commandes) et en tapant la commande python --version. Vous devriez voir s'afficher la version de Python installée.

Les bases de Python : variables, types de données et opérations

Python est un langage fortement typé, ce qui signifie que chaque variable doit avoir un type spécifique. Voici les principaux types de données que vous pouvez utiliser en Python :

  • Entiers : représentent les nombres entiers, par exemple 1, 2, -3.
  • Flottants : représentent les nombres décimaux, par exemple 3.14, -0.5.
  • Chaînes de caractères : représentent du texte, par exemple "Bonjour", 'Python'.
  • Booléens : représentent les valeurs de vérité, True (vrai) ou False (faux).

Vous pouvez effectuer des opérations mathématiques simples en Python, telles que l'addition, la soustraction, la multiplication et la division. Voici quelques exemples :

a = 5;
b = 2;

addition = a + b;
soustraction = a - b;
multiplication = a * b;
division = a / b;

print("Résultats :", addition, soustraction, multiplication, division);

La sortie de ce code sera :

Résultats : 7 3 10 2.5

En plus des opérations mathématiques, Python offre également des opérations sur les chaînes de caractères, telles que la concaténation et la longueur :

nom = "John";
age = 25;

message = "Je m'appelle " + nom + " et j'ai " + str(age) + " ans.";
longueur_message = len(message);

print(message);
print("Longueur du message :", longueur_message);

La sortie de ce code sera :

Je m'appelle John et j'ai 25 ans.
Longueur du message : 30
