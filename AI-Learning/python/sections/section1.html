Section 1: Introduction à Python

Introduction à Python

Python est un langage de programmation interprété, polyvalent et facile à apprendre. Il a été créé par Guido van Rossum et publié pour la première fois en 1991. Python est connu pour sa syntaxe claire et lisible, ce qui en fait un excellent choix pour les débutants en programmation.

Installation de Python

Pour commencer à utiliser Python, vous devez d'abord l'installer sur votre ordinateur. Voici les étapes à suivre pour installer Python :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org/) et téléchargez la dernière version de Python.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Vérifiez que Python est correctement installé en ouvrant une fenêtre de terminal et en tapant "python --version". Vous devriez voir s'afficher la version de Python que vous avez installée.

Les bases de la programmation en Python

Python offre de nombreuses fonctionnalités puissantes pour la programmation. Voici quelques concepts de base que vous devez connaître :

Variables

En Python, vous pouvez créer des variables pour stocker des valeurs. Voici un exemple :

message = "Bonjour, monde!"
print(message)

Types de données

Python prend en charge différents types de données, tels que les entiers, les flottants, les chaînes de caractères et les listes. Voici quelques exemples :

age = 25
pi = 3.14
nom = "Alice"
liste_nombres = [1, 2, 3, 4, 5]

Structures de contrôle

Python propose des structures de contrôle telles que les boucles et les conditions pour contrôler le flux d'exécution du programme. Voici un exemple d'une boucle while :

i = 0
while i < 10:
    print(i)
    i += 1

Fonctions

Les fonctions vous permettent de regrouper du code réutilisable. Voici un exemple de fonction qui calcule la somme de deux nombres :

def addition(a, b):
    return a + b

resultat = addition(3, 5)
print(resultat)

Ce sont là quelques-uns des concepts de base de la programmation en Python. Dans cette formation e-learning, nous approfondirons ces sujets et explorerons d'autres fonctionnalités avancées de Python.
