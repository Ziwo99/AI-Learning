Section 1 : Introduction à Python

Introduction à Python

Python est un langage de programmation interprété, orienté objet et de haut niveau. Il a été créé par Guido van Rossum et publié pour la première fois en 1991. Python est connu pour sa syntaxe claire et lisible, ce qui en fait un excellent choix pour les débutants en programmation.

Python est utilisé dans de nombreux domaines, tels que le développement web, l'analyse de données, l'intelligence artificielle et l'automatisation des tâches. Il est également populaire pour son vaste écosystème de bibliothèques et de frameworks.

Installation de Python

Pour commencer à programmer en Python, vous devez d'abord installer Python sur votre ordinateur. Voici les étapes pour installer Python :

  1. Rendez-vous sur le site officiel de Python (https://www.python.org) et téléchargez la dernière version de Python.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Vérifiez que Python est correctement installé en ouvrant une fenêtre de terminal et en tapant la commande python --version. Vous devriez voir la version de Python installée.

Premier programme en Python

Maintenant que Python est installé, vous pouvez écrire votre premier programme en Python. Voici un exemple de programme Python qui affiche "Hello, World!" :

print("Hello, World!")

Explication du code :

La fonction print() est utilisée pour afficher du texte à l'écran. Dans cet exemple, nous utilisons la fonction print() pour afficher la chaîne de caractères "Hello, World!".

Pour exécuter ce programme, enregistrez-le dans un fichier avec l'extension ".py" (par exemple, "hello.py") et exécutez-le en utilisant la commande python hello.py dans une fenêtre de terminal.

Congratulations! Vous avez écrit et exécuté votre premier programme en Python.
