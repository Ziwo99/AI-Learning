Section 3: Les structures de contrôle en Python

Les conditions

Les conditions permettent d'exécuter certaines parties du code en fonction de l'évaluation d'une expression booléenne. En Python, les conditions sont généralement utilisées avec les instructions if, elif (pour "else if") et else.

Voici un exemple d'utilisation des conditions en Python :

x = 10

if x > 0:
    print("x est positif")
elif x < 0:
    print("x est négatif")
else:
    print("x est nul")

Dans cet exemple, si la valeur de x est supérieure à 0, le programme affiche "x est positif". Si la valeur de x est inférieure à 0, le programme affiche "x est négatif". Sinon, c'est-à-dire si la valeur de x est égale à 0, le programme affiche "x est nul".

Les boucles

Les boucles permettent de répéter l'exécution d'un bloc de code plusieurs fois. En Python, les boucles les plus couramment utilisées sont la boucle for et la boucle while.

Voici un exemple d'utilisation de la boucle for en Python :

for i in range(5):
    print(i)

Dans cet exemple, la boucle for répète l'exécution du bloc de code qui suit pour chaque valeur de i dans la plage de 0 à 4 (inclus). Le programme affiche donc les nombres de 0 à 4.

Voici un exemple d'utilisation de la boucle while en Python :

i = 0

while i < 5:
    print(i)
    i += 1

Dans cet exemple, la boucle while répète l'exécution du bloc de code qui suit tant que la condition i < 5 est vraie. Le programme affiche donc les nombres de 0 à 4, tout comme dans l'exemple précédent avec la boucle for.

Les instructions de contrôle de flux

Les instructions de contrôle de flux permettent de modifier le flux d'exécution d'un programme en sautant certaines parties du code ou en répétant l'exécution d'un bloc de code.

Voici quelques exemples d'instructions de contrôle de flux en Python :

L'instruction break permet de sortir d'une boucle prématurément :

for i in range(10):
    if i == 5:
        break
    print(i)

Dans cet exemple, la boucle for s'arrête lorsque la valeur de i est égale à 5. Le programme affiche donc les nombres de 0 à 4.

L'instruction continue permet de passer à l'itération suivante d'une boucle sans exécuter le reste du bloc de code :

for i in range(10):
    if i == 5:
        continue
    print(i)

Dans cet exemple, la boucle for continue avec l'itération suivante lorsque la valeur de i est égale à 5. Le programme affiche donc tous les nombres de 0 à 9, sauf 5.

L'instruction pass est utilisée comme espace réservé pour une instruction vide. Elle permet de ne rien faire dans un bloc de code sans provoquer d'erreur de syntaxe :

if x > 0:
    pass
else:
    print("x est négatif")

Dans cet exemple, si la valeur de x est supérieure à 0, le programme ne fait rien. Sinon, le programme affiche "x est négatif".
