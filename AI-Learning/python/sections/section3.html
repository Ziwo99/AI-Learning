Section 3 : Les structures de données en Python

Listes

Les listes sont des structures de données couramment utilisées en Python. Elles permettent de stocker plusieurs éléments dans un ordre spécifique. Pour créer une liste, vous pouvez utiliser des crochets [] et séparer les éléments par des virgules. Voici un exemple :

ma_liste = [1, 2, 3, 4, 5]

Les listes peuvent contenir des éléments de différents types, tels que des entiers, des chaînes de caractères, des booléens, voire même d'autres listes. Vous pouvez également accéder aux éléments d'une liste en utilisant leur indice. Par exemple, pour accéder au premier élément de la liste ma_liste, vous pouvez utiliser ma_liste[0]. Voici un exemple :

ma_liste = [1, 2, 3, 4, 5]
premier_element = ma_liste[0]
print(premier_element)  # Affiche 1

Les listes sont également modifiables, ce qui signifie que vous pouvez ajouter, supprimer ou modifier des éléments. Par exemple, pour ajouter un élément à une liste, vous pouvez utiliser la méthode append(). Voici un exemple :

ma_liste = [1, 2, 3, 4, 5]
ma_liste.append(6)
print(ma_liste)  # Affiche [1, 2, 3, 4, 5, 6]

Tuples

Les tuples sont similaires aux listes, mais avec une différence importante : ils sont immuables, ce qui signifie qu'une fois créés, leurs éléments ne peuvent pas être modifiés. Pour créer un tuple, vous pouvez utiliser des parenthèses () et séparer les éléments par des virgules. Voici un exemple :

mon_tuple = (1, 2, 3, 4, 5)

Comme les listes, les tuples peuvent contenir des éléments de différents types. Vous pouvez également accéder aux éléments d'un tuple en utilisant leur indice. Par exemple, pour accéder au premier élément du tuple mon_tuple, vous pouvez utiliser mon_tuple[0]. Voici un exemple :

mon_tuple = (1, 2, 3, 4, 5)
premier_element = mon_tuple[0]
print(premier_element)  # Affiche 1

Étant donné que les tuples sont immuables, vous ne pouvez pas ajouter, supprimer ou modifier des éléments une fois qu'ils ont été créés. Cependant, vous pouvez effectuer des opérations sur les tuples, telles que la concaténation ou la recherche d'éléments. Voici un exemple :

mon_tuple1 = (1, 2, 3)
mon_tuple2 = (4, 5, 6)
concatenation = mon_tuple1 + mon_tuple2
print(concatenation)  # Affiche (1, 2, 3, 4, 5, 6)

Dictionnaires

Les dictionnaires sont des structures de données qui associent des clés à des valeurs. Ils sont très utiles lorsque vous avez besoin de stocker des informations sous forme de paires clé-valeur. Pour créer un dictionnaire, vous pouvez utiliser des accolades {} et séparer les paires clé-valeur par des virgules. Voici un exemple :

mon_dictionnaire = {"clé1": valeur1, "clé2": valeur2, "clé3": valeur3}

Les clés d'un dictionnaire doivent être uniques, mais les valeurs peuvent être de différents types. Vous pouvez accéder aux valeurs d'un dictionnaire en utilisant leurs clés. Par exemple, pour accéder à la valeur associée à la clé "clé1" dans le dictionnaire mon_dictionnaire, vous pouvez utiliser mon_dictionnaire["clé1"]. Voici un exemple :

mon_dictionnaire = {"nom": "John", "âge": 25, "ville": "Paris"}
age = mon_dictionnaire["âge"]
print(age)  # Affiche 25

Vous pouvez également ajouter, supprimer ou modifier des paires clé-valeur dans un dictionnaire. Par exemple, pour ajouter une nouvelle paire clé-valeur, vous pouvez utiliser la syntaxe mon_dictionnaire[nouvelle_clé] = nouvelle_valeur. Voici un exemple :

mon_dictionnaire = {"nom": "John", "âge": 25, "ville": "Paris"}
mon_dictionnaire["profession"] = "Développeur"
print(mon_dictionnaire)  # Affiche {"nom": "John", "âge": 25, "ville": "Paris", "profession": "Développeur"}

Ensembles

Les ensembles sont des structures de données qui permettent de stocker des éléments uniques, sans ordre spécifique. Pour créer un ensemble, vous pouvez utiliser la fonction set() ou des accolades {} et séparer les éléments par des virgules. Voici un exemple :

mon_ensemble = set([1, 2, 3, 4, 5])

Les ensembles ne peuvent pas contenir de doublons, ce qui signifie que chaque élément est unique. Vous pouvez effectuer des opérations sur les ensembles, telles que l'union, l'intersection ou la différence. Par exemple, pour effectuer l'union de deux ensembles, vous pouvez utiliser l'opérateur |. Voici un exemple :

ensemble1 = set([1, 2, 3])
ensemble2 = set([3, 4, 5])
union = ensemble1 | ensemble2
print(union)  # Affiche {1, 2, 3, 4, 5}

Les ensembles sont également modifiables, ce qui signifie que vous pouvez ajouter ou supprimer des éléments. Par exemple, pour ajouter un élément à un ensemble, vous pouvez utiliser la méthode add(). Voici un exemple :

mon_ensemble = set([1, 2, 3, 4, 5])
mon_ensemble.add(6)
print(mon_ensemble)  # Affiche {1, 2, 3, 4, 5, 6}
