Section 3: Les structures de données en Python

Listes

Les listes sont des structures de données qui permettent de stocker plusieurs éléments dans un seul objet. Chaque élément est séparé par une virgule et peut être de n'importe quel type de données. Les listes sont définies en utilisant des crochets [ ] et les éléments peuvent être accédés en utilisant des indices.

Exemple de création d'une liste :

ma_liste = [1, 2, 3, "quatre", 5.0]

Pour accéder à un élément spécifique de la liste, on utilise son indice :

print(ma_liste[0])  # Affiche le premier élément de la liste

Tuples

Les tuples sont similaires aux listes, mais ils sont immuables, ce qui signifie qu'ils ne peuvent pas être modifiés une fois créés. Les tuples sont définis en utilisant des parenthèses ( ) et les éléments sont séparés par des virgules.

Exemple de création d'un tuple :

mon_tuple = (1, 2, 3, "quatre", 5.0)

Comme les tuples sont immuables, on ne peut pas modifier leurs éléments :

mon_tuple[0] = 10  # Cela générera une erreur car les tuples sont immuables

Dictionnaires

Les dictionnaires sont des structures de données qui permettent de stocker des paires clé-valeur. Chaque élément est constitué d'une clé et d'une valeur associée. Les dictionnaires sont définis en utilisant des accolades { } et les paires clé-valeur sont séparées par des virgules.

Exemple de création d'un dictionnaire :

mon_dictionnaire = {"clé1": "valeur1", "clé2": "valeur2", "clé3": "valeur3"}

Pour accéder à une valeur spécifique dans le dictionnaire, on utilise sa clé :

print(mon_dictionnaire["clé1"])  # Affiche la valeur associée à la clé "clé1"

Ensembles

Les ensembles sont des structures de données qui permettent de stocker des éléments uniques, sans ordre spécifique. Les ensembles sont définis en utilisant des accolades { } et les éléments sont séparés par des virgules.

Exemple de création d'un ensemble :

mon_ensemble = {1, 2, 3, 4, 5}

Les ensembles ne peuvent pas contenir de doublons :

mon_ensemble = {1, 2, 2, 3, 3, 3}
print(mon_ensemble)  # Affiche {1, 2, 3}

Les ensembles peuvent être utilisés pour effectuer des opérations mathématiques telles que l'union, l'intersection et la différence :

ensemble1 = {1, 2, 3}
ensemble2 = {3, 4, 5}

union = ensemble1.union(ensemble2)
intersection = ensemble1.intersection(ensemble2)
difference = ensemble1.difference(ensemble2)

print(union)  # Affiche {1, 2, 3, 4, 5}
print(intersection)  # Affiche {3}
print(difference)  # Affiche {1, 2}
