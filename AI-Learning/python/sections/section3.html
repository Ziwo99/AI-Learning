Les opérations arithmétiques en Python

En Python, les opérations arithmétiques permettent de réaliser des calculs mathématiques. Voici les opérations arithmétiques de base :

  • L'addition : utilisez le symbole "+" pour ajouter deux nombres. Par exemple, 2 + 3 donne 5.
  • La soustraction : utilisez le symbole "-" pour soustraire un nombre d'un autre. Par exemple, 5 - 3 donne 2.
  • La multiplication : utilisez le symbole "*" pour multiplier deux nombres. Par exemple, 2 * 3 donne 6.
  • La division : utilisez le symbole "/" pour diviser un nombre par un autre. Par exemple, 6 / 3 donne 2.
  • La division entière : utilisez le symbole "//" pour obtenir le quotient entier de la division. Par exemple, 7 // 3 donne 2.
  • Le modulo : utilisez le symbole "%" pour obtenir le reste de la division. Par exemple, 7 % 3 donne 1.
  • L'exponentiation : utilisez le symbole "**" pour élever un nombre à une puissance. Par exemple, 2 ** 3 donne 8.

Voici un exemple d'utilisation des opérations arithmétiques en Python :

a = 5
b = 2

addition = a + b
soustraction = a - b
multiplication = a * b
division = a / b
division_entiere = a // b
modulo = a % b
exponentiation = a ** b

print("Résultats des opérations arithmétiques :")
print("Addition :", addition)
print("Soustraction :", soustraction)
print("Multiplication :", multiplication)
print("Division :", division)
print("Division entière :", division_entiere)
print("Modulo :", modulo)
print("Exponentiation :", exponentiation)

Résultat :

Résultats des opérations arithmétiques :
Addition : 7
Soustraction : 3
Multiplication : 10
Division : 2.5
Division entière : 2
Modulo : 1
Exponentiation : 25

Les opérations de comparaison en Python

En Python, les opérations de comparaison permettent de comparer des valeurs et de retourner un résultat booléen (True ou False) en fonction de la condition. Voici les opérations de comparaison disponibles :

  • L'égalité : utilisez le symbole "==" pour vérifier si deux valeurs sont égales. Par exemple, 2 == 3 retourne False.
  • L'inégalité : utilisez le symbole "!=" pour vérifier si deux valeurs sont différentes. Par exemple, 2 != 3 retourne True.
  • L'infériorité : utilisez le symbole "<" pour vérifier si une valeur est inférieure à une autre. Par exemple, 2 < 3 retourne True.
  • La supériorité : utilisez le symbole ">" pour vérifier si une valeur est supérieure à une autre. Par exemple, 2 > 3 retourne False.
  • L'infériorité ou égalité : utilisez le symbole "<=" pour vérifier si une valeur est inférieure ou égale à une autre. Par exemple, 2 <= 3 retourne True.
  • La supériorité ou égalité : utilisez le symbole ">=" pour vérifier si une valeur est supérieure ou égale à une autre. Par exemple, 2 >= 3 retourne False.

Voici un exemple d'utilisation des opérations de comparaison en Python :

a = 5
b = 2

egalite = a == b
inegalite = a != b
inferiorite = a < b
superiorite = a > b
inferiorite_ou_egalite = a <= b
superiorite_ou_egalite = a >= b

print("Résultats des opérations de comparaison :")
print("Égalité :", egalite)
print("Inégalité :", inegalite)
print("Infériorité :", inferiorite)
print("Supériorité :", superiorite)
print("Infériorité ou égalité :", inferiorite_ou_egalite)
print("Supériorité ou égalité :", superiorite_ou_egalite)

Résultat :

Résultats des opérations de comparaison :
Égalité : False
Inégalité : True
Infériorité : False
Supériorité : True
Infériorité ou égalité : False
Supériorité ou égalité : True

Les opérations logiques en Python

En Python, les opérations logiques permettent de combiner des expressions booléennes et de retourner un résultat booléen. Voici les opérations logiques disponibles :

  • L'opérateur "and" : retourne True si les deux expressions sont True, sinon retourne False.
  • L'opérateur "or" : retourne True si au moins l'une des expressions est True, sinon retourne False.
  • L'opérateur "not" : retourne l'inverse de l'expression booléenne. Si l'expression est True, retourne False, et vice versa.

Voici un exemple d'utilisation des opérations logiques en Python :

a = True
b = False

et_logique = a and b
ou_logique = a or b
non_logique = not a

print("Résultats des opérations logiques :")
print("ET logique :", et_logique)
print("OU logique :", ou_logique)
print("NON logique :", non_logique)

Résultat :

Résultats des opérations logiques :
ET logique : False
OU logique : True
NON logique : False

Les opérations sur les chaînes de caractères en Python

En Python, les opérations sur les chaînes de caractères permettent de manipuler et de combiner des chaînes de caractères. Voici quelques opérations couramment utilisées :

  • La concaténation : utilisez le symbole "+" pour concaténer deux chaînes de caractères. Par exemple, "Hello" + "World" donne "HelloWorld".
  • La répétition : utilisez le symbole "*" pour répéter une chaîne de caractères plusieurs fois. Par exemple, "Hello" * 3 donne "HelloHelloHello".
  • L'indexation : utilisez les crochets "[]" pour accéder à un caractère spécifique dans une chaîne de caractères. Par exemple, "Hello"[0] donne "H".
  • La découpe (slicing) : utilisez les crochets "[]" avec des indices de début et de fin pour extraire une sous-chaîne d'une chaîne de caractères. Par exemple, "Hello"[1:4] donne "ell".
  • La longueur : utilisez la fonction len() pour obtenir la longueur d'une chaîne de caractères. Par exemple, len("Hello") donne 5.
