Section 3: Structures de données

Les listes

Une liste est une structure de données qui permet de stocker plusieurs éléments de manière ordonnée. Chaque élément dans une liste est identifié par un index, qui commence à 0 pour le premier élément. Les listes sont définies en utilisant des crochets [] et les éléments sont séparés par des virgules.

Exemple de création d'une liste :

ma_liste = [1, 2, 3, 4, 5]

Les listes peuvent contenir des éléments de différents types de données, tels que des entiers, des chaînes de caractères, des booléens, voire même d'autres listes.

Exemple d'accès aux éléments d'une liste :

print(ma_liste[0])  # Affiche le premier élément de la liste (1)
print(ma_liste[2])  # Affiche le troisième élément de la liste (3)

Les tuples

Un tuple est une structure de données similaire à une liste, mais avec une différence importante : les tuples sont immuables, ce qui signifie qu'une fois créés, leurs éléments ne peuvent pas être modifiés. Les tuples sont définis en utilisant des parenthèses () et les éléments sont séparés par des virgules.

Exemple de création d'un tuple :

mon_tuple = (1, 2, 3, 4, 5)

Les tuples peuvent également contenir des éléments de différents types de données, tout comme les listes.

Exemple d'accès aux éléments d'un tuple :

print(mon_tuple[0])  # Affiche le premier élément du tuple (1)
print(mon_tuple[2])  # Affiche le troisième élément du tuple (3)

Les dictionnaires

Un dictionnaire est une structure de données qui permet de stocker des paires clé-valeur. Chaque élément dans un dictionnaire est identifié par une clé unique. Les dictionnaires sont définis en utilisant des accolades {} et les paires clé-valeur sont séparées par des virgules.

Exemple de création d'un dictionnaire :

mon_dictionnaire = {'nom': 'John', 'age': 25, 'ville': 'Paris'}

Les clés d'un dictionnaire peuvent être de différents types de données, tels que des chaînes de caractères, des entiers ou même des tuples. Les valeurs associées aux clés peuvent également être de différents types de données.

Exemple d'accès aux éléments d'un dictionnaire :

print(mon_dictionnaire['nom'])   # Affiche la valeur associée à la clé 'nom' (John)
print(mon_dictionnaire['age'])   # Affiche la valeur associée à la clé 'age' (25)

Les ensembles

Un ensemble est une structure de données qui permet de stocker des éléments uniques, sans ordre spécifique. Les ensembles sont définis en utilisant des accolades {}.

Exemple de création d'un ensemble :

mon_ensemble = {1, 2, 3, 4, 5}

Les ensembles ne peuvent pas contenir de doublons, ce qui signifie que chaque élément dans un ensemble est unique.

Exemple d'opérations sur les ensembles :

mon_ensemble.add(6)     # Ajoute l'élément 6 à l'ensemble
mon_ensemble.remove(3)  # Supprime l'élément 3 de l'ensemble

Les ensembles peuvent être utilisés pour effectuer des opérations ensemblistes telles que l'union, l'intersection et la différence entre ensembles.

Exemple d'opérations ensemblistes :

ensemble1 = {1, 2, 3}
ensemble2 = {3, 4, 5}

union = ensemble1.union(ensemble2)           # Union de deux ensembles
intersection = ensemble1.intersection(ensemble2) # Intersection de deux ensembles
difference = ensemble1.difference(ensemble2)   # Différence entre deux ensembles
