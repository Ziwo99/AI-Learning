Section 3: Les structures de contrôle en Python

Les conditions

Les conditions sont utilisées pour prendre des décisions dans un programme Python. Elles permettent d'exécuter certaines parties du code uniquement si une condition spécifique est remplie.

La syntaxe de base pour une condition en Python est la suivante :

if condition:
    # code à exécuter si la condition est vraie
else:
    # code à exécuter si la condition est fausse

Voici un exemple qui illustre l'utilisation des conditions en Python :

a = 10

if a > 5:
    print("a est supérieur à 5")
else:
    print("a est inférieur ou égal à 5")

Les boucles

Les boucles permettent de répéter l'exécution d'un bloc de code plusieurs fois. En Python, il existe deux types de boucles : la boucle "for" et la boucle "while".

La boucle "for" est utilisée lorsque le nombre d'itérations est connu à l'avance :

for element in sequence:
    # code à exécuter pour chaque élément de la séquence

Voici un exemple qui montre comment utiliser une boucle "for" en Python :

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

La boucle "while" est utilisée lorsque le nombre d'itérations n'est pas connu à l'avance :

i = 0

while i < 5:
    print(i)
    i += 1

Les instructions de contrôle de flux

Les instructions de contrôle de flux permettent de modifier le flux d'exécution d'un programme Python. Elles incluent les instructions "break", "continue" et "pass".

L'instruction "break" est utilisée pour arrêter l'exécution d'une boucle de manière anticipée :

for i in range(10):
    if i == 5:
        break
    print(i)

L'instruction "continue" est utilisée pour passer à l'itération suivante d'une boucle :

for i in range(10):
    if i == 5:
        continue
    print(i)

L'instruction "pass" est utilisée comme espace réservé pour du code qui sera implémenté ultérieurement :

if condition:
    pass  # code à implémenter plus tard
else:
    print("Condition remplie")
