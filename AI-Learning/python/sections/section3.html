Listes

Une liste est une collection ordonnée et modifiable d'éléments. Les éléments d'une liste peuvent être de différents types tels que des nombres, des chaînes de caractères, des booléens, etc. Une liste est définie en utilisant des crochets [] et les éléments sont séparés par des virgules.

Exemple :

ma_liste = [1, 2, 3, "quatre", True]

Pour accéder à un élément spécifique d'une liste, on utilise l'index de l'élément. L'index d'une liste commence à 0. On peut également utiliser des index négatifs pour accéder aux éléments à partir de la fin de la liste.

Exemple :

print(ma_liste[0])  # Affiche le premier élément de la liste
print(ma_liste[-1])  # Affiche le dernier élément de la liste

Les listes peuvent être modifiées en ajoutant, supprimant ou modifiant des éléments.

Exemple :

ma_liste.append(5)  # Ajoute l'élément 5 à la fin de la liste
ma_liste.remove("quatre")  # Supprime l'élément "quatre" de la liste
ma_liste[0] = 10  # Modifie le premier élément de la liste

Tuples

Un tuple est une collection ordonnée et immuable d'éléments. Contrairement aux listes, les tuples ne peuvent pas être modifiés une fois créés. Les tuples sont définis en utilisant des parenthèses () et les éléments sont séparés par des virgules.

Exemple :

mon_tuple = (1, 2, 3, "quatre", True)

Comme pour les listes, on peut accéder aux éléments d'un tuple en utilisant l'index de l'élément.

Exemple :

print(mon_tuple[0])  # Affiche le premier élément du tuple
print(mon_tuple[-1])  # Affiche le dernier élément du tuple

Les tuples ne peuvent pas être modifiés, mais on peut effectuer des opérations sur les tuples comme la concaténation ou la duplication.

Exemple :

nouveau_tuple = mon_tuple + (5, 6)  # Concatène mon_tuple avec un nouveau tuple contenant les éléments 5 et 6
duplicated_tuple = mon_tuple * 2  # Duplique mon_tuple

Dictionnaires

Un dictionnaire est une collection non ordonnée et modifiable d'éléments, où chaque élément est associé à une clé unique. Les dictionnaires sont définis en utilisant des accolades {} et les éléments sont spécifiés sous la forme "clé: valeur", séparés par des virgules.

Exemple :

mon_dictionnaire = {"nom": "John", "âge": 25, "ville": "Paris"}

Pour accéder à une valeur spécifique dans un dictionnaire, on utilise la clé correspondante.

Exemple :

print(mon_dictionnaire["nom"])  # Affiche la valeur associée à la clé "nom"

Les dictionnaires peuvent être modifiés en ajoutant, supprimant ou modifiant des paires clé-valeur.

Exemple :

mon_dictionnaire["âge"] = 26  # Modifie la valeur associée à la clé "âge"
mon_dictionnaire["profession"] = "Développeur"  # Ajoute une nouvelle paire clé-valeur

Ensembles

Un ensemble est une collection non ordonnée et modifiable d'éléments uniques. Les ensembles sont définis en utilisant des accolades {} ou la fonction set(). Les éléments d'un ensemble sont séparés par des virgules.

Exemple :

mon_ensemble = {1, 2, 3, 4, 5}

Les ensembles peuvent être modifiés en ajoutant ou en supprimant des éléments.

Exemple :

mon_ensemble.add(6)  # Ajoute l'élément 6 à l'ensemble
mon_ensemble.remove(3)  # Supprime l'élément 3 de l'ensemble

Manipulation de données

La manipulation de données en Python implique l'utilisation des structures de données mentionnées ci-dessus pour stocker, organiser et traiter des informations. Par exemple, on peut utiliser des listes pour stocker une série de valeurs, des dictionnaires pour stocker des données associées à des clés, etc.

Exemple :

ma_liste = [1, 2, 3, 4, 5]
mon_dictionnaire = {"nom": "John", "âge": 25, "ville": "Paris"}

# Parcourir une liste
for element in ma_liste:
    print(element)

# Parcourir un dictionnaire
for cle, valeur in mon_dictionnaire.items():
    print(cle, valeur)

La manipulation de données est essentielle dans de nombreux domaines tels que l'analyse de données, le traitement des données, etc. Python offre de nombreuses fonctionnalités et bibliothèques pour faciliter la manipulation de données de manière efficace et puissante.
