Les fonctions et leurs paramètres

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser et de structurer le code en le divisant en tâches plus petites et plus gérables. En Python, une fonction est définie à l'aide du mot-clé "def" suivi du nom de la fonction et de ses paramètres entre parenthèses.

Voici un exemple de fonction simple qui affiche un message :

def say_hello():
    print("Bonjour !")

say_hello()  # Appel de la fonction

Les paramètres sont des valeurs que l'on peut passer à une fonction pour qu'elle les utilise dans son exécution. Les paramètres sont définis entre parenthèses lors de la définition de la fonction.

Voici un exemple de fonction qui prend un paramètre :

def say_hello(name):
    print("Bonjour", name)

say_hello("Alice")  # Appel de la fonction avec un argument

Il est également possible de définir des valeurs par défaut pour les paramètres d'une fonction. Cela permet de rendre ces paramètres optionnels lors de l'appel de la fonction.

Voici un exemple de fonction avec un paramètre ayant une valeur par défaut :

def say_hello(name=""):
    if name:
        print("Bonjour", name)
    else:
        print("Bonjour à tous")

say_hello()  # Appel de la fonction sans argument
say_hello("Alice")  # Appel de la fonction avec un argument

Les fonctions récursives

Une fonction récursive est une fonction qui s'appelle elle-même dans son propre corps. Cela permet de résoudre des problèmes en les divisant en sous-problèmes plus petits.

Voici un exemple de fonction récursive qui calcule la factorielle d'un nombre :

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

result = factorial(5)
print("Le résultat est", result)

La fonction "factorial" s'appelle elle-même avec un argument plus petit à chaque itération, jusqu'à atteindre le cas de base où n est égal à zéro.

Les modules et l'importation de modules externes

Un module est un fichier contenant du code Python qui peut être importé et utilisé dans d'autres programmes. Les modules permettent d'organiser le code en regroupant des fonctionnalités similaires.

Pour importer un module, on utilise le mot-clé "import" suivi du nom du module :

import math

result = math.sqrt(16)
print("La racine carrée de 16 est", result)

Il est également possible d'importer uniquement certaines fonctions ou classes d'un module :

from math import sqrt

result = sqrt(16)
print("La racine carrée de 16 est", result)

Il est également possible de renommer un module lors de l'importation :

import math as m

result = m.sqrt(16)
print("La racine carrée de 16 est", result)

Il existe de nombreux modules disponibles dans la bibliothèque standard de Python, qui offrent des fonctionnalités supplémentaires telles que la manipulation de dates, la génération de nombres aléatoires, etc.

Il est également possible d'installer des modules externes à l'aide de l'outil "pip". Par exemple, pour installer le module "requests", on peut exécuter la commande suivante dans le terminal :

pip install requests

Une fois le module installé, on peut l'importer et l'utiliser dans notre programme.
