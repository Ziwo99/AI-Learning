Listes et tuples

Les listes et les tuples sont des structures de données couramment utilisées en Python pour stocker des collections d'éléments. Une liste est une séquence modifiable d'éléments, tandis qu'un tuple est une séquence immuable d'éléments.

Voici comment créer une liste en Python :

ma_liste = [1, 2, 3, 4, 5]

Et voici comment créer un tuple :

mon_tuple = (1, 2, 3, 4, 5)

Vous pouvez accéder aux éléments d'une liste ou d'un tuple en utilisant des indices :

print(ma_liste[0])  # Affiche le premier élément de la liste
print(mon_tuple[2])  # Affiche le troisième élément du tuple

Les listes et les tuples peuvent contenir des éléments de différents types, y compris d'autres listes ou tuples :

ma_liste = [1, "deux", [3, 4, 5]]
mon_tuple = (1, "deux", (3, 4, 5))

Vous pouvez également modifier les éléments d'une liste, mais pas ceux d'un tuple :

ma_liste[0] = 10  # Modifie le premier élément de la liste
mon_tuple[0] = 10  # Lève une erreur car les tuples sont immuables

Dictionnaires et ensembles

Les dictionnaires et les ensembles sont d'autres structures de données couramment utilisées en Python. Un dictionnaire est une collection modifiable d'éléments associés à des clés uniques, tandis qu'un ensemble est une collection non ordonnée d'éléments uniques.

Voici comment créer un dictionnaire en Python :

mon_dictionnaire = {"clé1": "valeur1", "clé2": "valeur2", "clé3": "valeur3"}

Et voici comment créer un ensemble :

mon_ensemble = {1, 2, 3, 4, 5}

Vous pouvez accéder aux valeurs d'un dictionnaire en utilisant les clés correspondantes :

print(mon_dictionnaire["clé1"])  # Affiche la valeur associée à la clé "clé1"

Les ensembles ne permettent pas d'accéder à des éléments spécifiques, mais vous pouvez vérifier si un élément est présent dans un ensemble :

print(1 in mon_ensemble)  # Vérifie si l'élément 1 est présent dans l'ensemble

Vous pouvez ajouter, supprimer ou modifier des éléments dans un dictionnaire :

mon_dictionnaire["clé4"] = "valeur4"  # Ajoute une nouvelle paire clé-valeur
del mon_dictionnaire["clé2"]  # Supprime la paire clé-valeur correspondante
mon_dictionnaire["clé3"] = "nouvelle_valeur"  # Modifie la valeur associée à la clé "clé3"

Les ensembles permettent également d'ajouter et de supprimer des éléments :

mon_ensemble.add(6)  # Ajoute l'élément 6 à l'ensemble
mon_ensemble.remove(3)  # Supprime l'élément 3 de l'ensemble

Manipulation de chaînes de caractères

Les chaînes de caractères sont des séquences de caractères utilisées pour représenter du texte en Python. Vous pouvez manipuler les chaînes de caractères de différentes manières, y compris en les concaténant, en les découpant et en les formatant.

Voici comment créer une chaîne de caractères en Python :

ma_chaine = "Bonjour, monde!"

Vous pouvez concaténer des chaînes de caractères en utilisant l'opérateur de concaténation (+) :

nom = "Alice"
message = "Bonjour, " + nom + "!"

Vous pouvez découper une chaîne de caractères en utilisant la méthode split() :

ma_chaine = "Bonjour, monde!"
mots = ma_chaine.split(", ")  # Divise la chaîne en une liste de mots

Vous pouvez également formater une chaîne de caractères en utilisant la méthode format() :

nom = "Alice"
age = 25
message = "Je m'appelle {} et j'ai {} ans.".format(nom, age)

Les fichiers et l'entrée/sortie

Python offre des fonctionnalités puissantes pour travailler avec des fichiers et gérer l'entrée/sortie. Vous pouvez ouvrir des fichiers, les lire, les écrire et les fermer à l'aide de fonctions intégrées.

Voici comment ouvrir un fichier en Python :

fichier = open("mon_fichier.txt", "r")  # Ouvre le fichier en mode lecture

Vous pouvez lire le contenu d'un fichier en utilisant la méthode read() :

contenu = fichier.read()  # Lit tout le contenu du fichier

Vous pouvez également écrire dans un fichier en utilisant la méthode write() :

fichier.write("Bonjour, monde!")  # Écrit une chaîne de caractères dans le fichier

N'oubliez pas de fermer le fichier après avoir terminé :

fichier.close()  # Ferme le fichier

En plus de travailler avec des fichiers, vous pouvez également lire l'entrée de l'utilisateur et afficher des résultats à l'écran :

nom = input("Quel est votre nom ? ")
print("Bonjour, " + nom + "!")

Ces fonctionnalités d'entrée/sortie sont essentielles pour interagir avec les utilisateurs et traiter des données externes.
