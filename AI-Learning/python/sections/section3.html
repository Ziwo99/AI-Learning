Section 3 : Structures de données

Retour au cours

Listes

Les listes sont des structures de données qui permettent de stocker une collection ordonnée d'éléments. Chaque élément peut être de n'importe quel type de données, y compris d'autres listes. Les listes sont définies en utilisant des crochets [] et les éléments sont séparés par des virgules.

Exemple :

ma_liste = [1, 2, 3, "quatre", 5.0]

Les listes sont mutables, ce qui signifie que vous pouvez modifier leurs éléments en utilisant leur indice. L'indice d'un élément dans une liste commence à 0.

Exemple :

ma_liste[0] = "un"

Tuples

Les tuples sont similaires aux listes, mais ils sont immuables, ce qui signifie que vous ne pouvez pas modifier leurs éléments une fois qu'ils ont été définis. Les tuples sont définis en utilisant des parenthèses () et les éléments sont séparés par des virgules.

Exemple :

mon_tuple = (1, 2, 3, "quatre", 5.0)

Comme les listes, les tuples peuvent contenir des éléments de différents types de données.

Dictionnaires

Les dictionnaires sont des structures de données qui permettent de stocker des paires clé-valeur. Chaque élément du dictionnaire est composé d'une clé et d'une valeur associée. Les dictionnaires sont définis en utilisant des accolades {} et les paires clé-valeur sont séparées par des virgules.

Exemple :

mon_dictionnaire = {"clé1": "valeur1", "clé2": "valeur2", "clé3": "valeur3"}

Les clés dans un dictionnaire doivent être uniques. Vous pouvez accéder à la valeur d'un élément en utilisant sa clé.

Exemple :

valeur = mon_dictionnaire["clé1"]

Ensembles

Les ensembles sont des structures de données qui permettent de stocker une collection non ordonnée d'éléments uniques. Les ensembles sont définis en utilisant des accolades {}.

Exemple :

mon_ensemble = {1, 2, 3, 4, 5}

Les ensembles ne permettent pas d'avoir des éléments en double. Vous pouvez effectuer des opérations ensemblistes telles que l'union, l'intersection et la différence sur les ensembles.

Exemple :

ensemble1 = {1, 2, 3}
ensemble2 = {3, 4, 5}

union = ensemble1.union(ensemble2)
intersection = ensemble1.intersection(ensemble2)
différence = ensemble1.difference(ensemble2)

Les ensembles sont également utiles pour éliminer les doublons d'une liste.

Exemple :

ma_liste = [1, 2, 3, 3, 4, 5, 5]
mon_ensemble = set(ma_liste)
