Les conditions et les boucles

Les conditions et les boucles sont des éléments essentiels de la programmation en Python. Ils permettent de contrôler l'exécution du code en fonction de certaines conditions ou de répéter des instructions plusieurs fois.

Les conditions

Les conditions sont utilisées pour effectuer des actions différentes en fonction de la véracité d'une expression booléenne. En Python, les conditions sont généralement utilisées avec les instructions if, elif et else.

x = 10

if x > 0:
    print("x est positif")
elif x < 0:
    print("x est négatif")
else:
    print("x est nul")

Dans cet exemple, si la valeur de x est supérieure à 0, le message "x est positif" sera affiché. Si la valeur de x est inférieure à 0, le message "x est négatif" sera affiché. Sinon, c'est-à-dire si la valeur de x est égale à 0, le message "x est nul" sera affiché.

Les boucles

Les boucles permettent de répéter des instructions plusieurs fois. En Python, il existe deux types de boucles : la boucle while et la boucle for.

La boucle while

La boucle while répète un bloc d'instructions tant qu'une condition est vraie.

i = 0

while i < 5:
    print("Valeur de i :", i)
    i += 1

Dans cet exemple, le bloc d'instructions sera répété tant que la valeur de i est inférieure à 5. À chaque itération, la valeur de i est affichée, puis incrémentée de 1.

La boucle for

La boucle for permet de parcourir une séquence d'éléments (comme une liste, un tuple ou une chaîne de caractères) et d'exécuter un bloc d'instructions pour chaque élément de la séquence.

liste = [1, 2, 3, 4, 5]

for element in liste:
    print("Valeur de l'élément :", element)

Dans cet exemple, le bloc d'instructions sera exécuté pour chaque élément de la liste. La valeur de chaque élément sera affichée.

Les instructions de contrôle de flux

Les instructions de contrôle de flux permettent de modifier le flux d'exécution du code en sautant des instructions, en sortant d'une boucle ou en passant à l'itération suivante d'une boucle.

L'instruction break

L'instruction break permet de sortir d'une boucle prématurément, sans exécuter les instructions restantes.

for i in range(10):
    if i == 5:
        break
    print("Valeur de i :", i)

Dans cet exemple, la boucle for s'arrêtera lorsque la valeur de i sera égale à 5. Les instructions restantes ne seront pas exécutées.

L'instruction continue

L'instruction continue permet de passer à l'itération suivante d'une boucle, sans exécuter les instructions restantes pour l'itération actuelle.

for i in range(10):
    if i == 5:
        continue
    print("Valeur de i :", i)

Dans cet exemple, lorsque la valeur de i est égale à 5, l'instruction continue est exécutée, ce qui passe à l'itération suivante de la boucle sans exécuter les instructions restantes.

Les fonctions et les modules

Les fonctions et les modules sont des éléments importants de la programmation en Python. Ils permettent de regrouper des instructions et de les réutiliser, ce qui favorise la modularité et la réutilisabilité du code.

Les fonctions

Une fonction est un bloc d'instructions qui peut être appelé et exécuté à partir d'autres parties du code. Les fonctions permettent de découper le code en morceaux plus petits et plus faciles à gérer.

def addition(a, b):
    return a + b

resultat = addition(3, 5)
print("Résultat de l'addition :", resultat)

Dans cet exemple, la fonction addition prend deux paramètres a et b et retourne leur somme. La fonction est appelée avec les arguments 3 et 5, et le résultat est stocké dans la variable resultat.

Les modules

Un module est un fichier contenant des définitions de fonctions, de variables et d'autres éléments, qui peuvent être importés et utilisés dans d'autres parties du code.

import math

racine_carree = math.sqrt(16)
print("Racine carrée de 16 :", racine_carree)

Dans cet exemple, le module math est importé et la fonction sqrt est utilisée pour calculer la racine carrée de 16. Le résultat est stocké dans la variable racine_carree.
