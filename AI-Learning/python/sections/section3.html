Section 3: Les structures de contrôle en Python

Section 3: Les structures de contrôle en Python

Les conditions en Python

Les conditions en Python permettent d'exécuter certaines parties du code en fonction de la véracité d'une expression booléenne. Une expression booléenne est une expression qui peut être soit vraie (True) soit fausse (False).

Pour écrire une condition en Python, on utilise le mot-clé if suivi de l'expression à évaluer, puis du bloc de code à exécuter si l'expression est vraie. On peut également ajouter des blocs de code supplémentaires avec les mots-clés elif (pour "else if") et else.

Voici un exemple :

a = 5

if a > 10:
    print("a est supérieur à 10")
elif a > 5:
    print("a est supérieur à 5 mais inférieur ou égal à 10")
else:
    print("a est inférieur ou égal à 5")

# Résultat : "a est inférieur ou égal à 5"

Les boucles en Python

Les boucles en Python permettent d'exécuter une partie du code de manière répétée tant qu'une condition est vraie, ou pour chaque élément d'une séquence.

Il existe deux types de boucles en Python : la boucle while et la boucle for.

La boucle while exécute le bloc de code tant que la condition spécifiée est vraie :

i = 0

while i < 5:
    print(i)
    i += 1

# Résultat : 0 1 2 3 4

La boucle for permet d'itérer sur les éléments d'une séquence (comme une liste, un tuple ou une chaîne de caractères) :

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

# Résultat : "pomme" "banane" "orange"

Les instructions de contrôle en Python

Les instructions de contrôle en Python permettent de modifier le flux d'exécution du code en utilisant des mots-clés tels que break, continue et pass.

Le mot-clé break permet de sortir d'une boucle prématurément :

for i in range(10):
    if i == 5:
        break
    print(i)

# Résultat : 0 1 2 3 4

Le mot-clé continue permet de passer à l'itération suivante d'une boucle :

for i in range(5):
    if i == 2:
        continue
    print(i)

# Résultat : 0 1 3 4

Le mot-clé pass est utilisé comme espace réservé pour une instruction vide. Il est souvent utilisé lors de la création d'une structure de contrôle qui sera complétée ultérieurement :

for i in range(5):
    if i == 2:
        pass
    print(i)

# Résultat : 0 1 2 3 4
