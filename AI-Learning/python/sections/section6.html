Gestion des dates et heures

La gestion des dates et heures est une fonctionnalité essentielle dans de nombreuses applications. Python offre le module datetime pour manipuler les dates et heures de manière efficace.

Voici un exemple d'utilisation du module datetime :

import datetime

# Obtenir la date et l'heure actuelles
maintenant = datetime.datetime.now()
print("Date et heure actuelles :", maintenant)

# Obtenir uniquement la date
aujourd'hui = maintenant.date()
print("Date actuelle :", aujourd'hui)

# Obtenir uniquement l'heure
heure = maintenant.time()
print("Heure actuelle :", heure)

# Formater une date selon un format spécifique
date_formatee = maintenant.strftime("%d/%m/%Y")
print("Date formatée :", date_formatee)

# Effectuer des opérations sur les dates
demain = maintenant + datetime.timedelta(days=1)
print("Demain :", demain.date())

Expressions régulières

Les expressions régulières sont des motifs de recherche utilisés pour trouver des correspondances dans les chaînes de caractères. Python propose le module re pour travailler avec les expressions régulières.

Voici un exemple d'utilisation du module re :

import re

# Vérifier si une chaîne correspond à un motif
pattern = r"^[A-Za-z]+$"
texte = "Bonjour"
correspondance = re.match(pattern, texte)
if correspondance:
    print("La chaîne correspond au motif.")
else:
    print("La chaîne ne correspond pas au motif.")

# Rechercher toutes les correspondances dans une chaîne
texte = "Le chat dort sur le tapis."
correspondances = re.findall(r"\b\w{4}\b", texte)
print("Correspondances :", correspondances)

Génération de nombres aléatoires

La génération de nombres aléatoires est souvent nécessaire dans les applications. Python propose le module random pour générer des nombres aléatoires.

Voici un exemple de génération de nombres aléatoires en Python :

import random

# Générer un nombre entier aléatoire entre 1 et 10
nombre_entier = random.randint(1, 10)
print("Nombre entier aléatoire :", nombre_entier)

# Générer un nombre décimal aléatoire entre 0 et 1
nombre_decimal = random.random()
print("Nombre décimal aléatoire :", nombre_decimal)

# Mélanger une liste aléatoirement
liste = [1, 2, 3, 4, 5]
random.shuffle(liste)
print("Liste mélangée :", liste)

Manipulation de fichiers XML et JSON

La manipulation de fichiers XML et JSON est courante dans le développement d'applications. Python propose les modules xml.etree.ElementTree et json pour travailler avec ces formats de données.

Voici un exemple de manipulation de fichiers XML et JSON en Python :

import xml.etree.ElementTree as ET
import json

# Lire un fichier XML
arbre_xml = ET.parse("fichier.xml")
racine = arbre_xml.getroot()

# Parcourir les éléments de l'arbre XML
for element in racine:
    print(element.tag, ":", element.text)

# Écrire un fichier JSON
donnees = {
    "nom": "John Doe",
    "age": 30,
    "ville": "Paris"
}
with open("fichier.json", "w") as fichier:
    json.dump(donnees, fichier)
