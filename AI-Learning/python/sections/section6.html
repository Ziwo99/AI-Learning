Section 6: La manipulation de fichiers en Python

Section 6: La manipulation de fichiers en Python

L'ouverture et la fermeture de fichiers en Python

En Python, pour manipuler des fichiers, nous devons d'abord ouvrir le fichier que nous souhaitons utiliser. Pour ouvrir un fichier, nous utilisons la fonction open(). Cette fonction prend deux arguments : le nom du fichier et le mode d'ouverture.

Le mode d'ouverture peut être :

  • 'r' : lecture (par défaut)
  • 'w' : écriture
  • 'a' : ajout (ajouter du contenu à la fin du fichier)
  • 'x' : création (créer un nouveau fichier)

Voici un exemple d'ouverture d'un fichier en mode lecture :

fichier = open('mon_fichier.txt', 'r')

Une fois que nous avons terminé de travailler avec un fichier, il est important de le fermer en utilisant la méthode close(). Cela permet de libérer les ressources système utilisées par le fichier.

Voici un exemple de fermeture d'un fichier :

fichier.close()

La lecture et l'écriture de fichiers en Python

Une fois que nous avons ouvert un fichier en mode lecture ou écriture, nous pouvons effectuer des opérations de lecture ou d'écriture sur le fichier.

Pour lire le contenu d'un fichier, nous utilisons la méthode read(). Cette méthode renvoie le contenu du fichier sous forme de chaîne de caractères.

Voici un exemple de lecture d'un fichier :

fichier = open('mon_fichier.txt', 'r')
contenu = fichier.read()
print(contenu)
fichier.close()

Pour écrire dans un fichier, nous utilisons la méthode write(). Cette méthode prend une chaîne de caractères en argument et écrit cette chaîne dans le fichier.

Voici un exemple d'écriture dans un fichier :

fichier = open('mon_fichier.txt', 'w')
fichier.write('Bonjour, monde!')
fichier.close()

La gestion des erreurs lors de la manipulation de fichiers en Python

Lorsque nous manipulons des fichiers en Python, il est important de gérer les erreurs qui peuvent survenir. Cela permet de s'assurer que notre programme ne plante pas si quelque chose ne se passe pas comme prévu.

Une façon de gérer les erreurs lors de la manipulation de fichiers est d'utiliser des blocs try et except. Dans le bloc try, nous plaçons le code qui peut générer une erreur, et dans le bloc except, nous plaçons le code qui sera exécuté si une erreur se produit.

Voici un exemple de gestion des erreurs lors de l'ouverture d'un fichier :

try:
    fichier = open('mon_fichier.txt', 'r')
    contenu = fichier.read()
    print(contenu)
    fichier.close()
except FileNotFoundError:
    print("Le fichier n'a pas été trouvé.")

Dans cet exemple, si le fichier 'mon_fichier.txt' n'existe pas, une erreur de type FileNotFoundError sera générée. Le bloc except sera alors exécuté et affichera le message "Le fichier n'a pas été trouvé."
