Section 6 : Les modules avancés

Le module math

Le module math est une bibliothèque intégrée de Python qui fournit des fonctions mathématiques avancées. Il permet d'effectuer des opérations mathématiques complexes telles que le calcul de racines carrées, de logarithmes, de fonctions trigonométriques, etc.

Voici un exemple d'utilisation du module math pour calculer la racine carrée d'un nombre :

import math

x = 16
sqrt_x = math.sqrt(x)

print("La racine carrée de", x, "est", sqrt_x)

Le module random

Le module random est une bibliothèque intégrée de Python qui permet de générer des nombres aléatoires. Il est souvent utilisé dans les jeux, les simulations et les algorithmes de machine learning.

Voici un exemple d'utilisation du module random pour générer un nombre aléatoire entre 1 et 10 :

import random

random_number = random.randint(1, 10)

print("Le nombre aléatoire est", random_number)

Le module datetime

Le module datetime est une bibliothèque intégrée de Python qui permet de manipuler des dates et des heures. Il offre des fonctionnalités pour créer, manipuler et formater des objets de date et d'heure.

Voici un exemple d'utilisation du module datetime pour afficher la date et l'heure actuelles :

import datetime

current_datetime = datetime.datetime.now()

print("La date et l'heure actuelles sont", current_datetime)

Le module os

Le module os est une bibliothèque intégrée de Python qui permet d'interagir avec le système d'exploitation. Il offre des fonctionnalités pour manipuler des fichiers, des répertoires, exécuter des commandes système, etc.

Voici un exemple d'utilisation du module os pour lister tous les fichiers d'un répertoire :

import os

directory = "/chemin/vers/le/repertoire"

files = os.listdir(directory)

print("Les fichiers du répertoire sont :", files)
