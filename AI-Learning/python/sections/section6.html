La lecture et l'écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en programmation Python. Pour lire le contenu d'un fichier, nous utilisons la fonction open() avec le mode de lecture approprié. Voici un exemple :

file = open("mon_fichier.txt", "r")
content = file.read()
file.close()

print(content)

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode lecture ("r"). Ensuite, nous utilisons la méthode read() pour lire tout le contenu du fichier et le stocker dans la variable content. Enfin, nous fermons le fichier à l'aide de la méthode close(). Ensuite, nous affichons le contenu du fichier à l'aide de la fonction print().

Pour écrire dans un fichier, nous utilisons la fonction open() avec le mode d'écriture approprié. Voici un exemple :

file = open("mon_fichier.txt", "w")
file.write("Bonjour, monde !")
file.close()

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode écriture ("w"). Ensuite, nous utilisons la méthode write() pour écrire le texte "Bonjour, monde !" dans le fichier. Enfin, nous fermons le fichier à l'aide de la méthode close().

La manipulation de dossiers

La manipulation de dossiers est également une tâche courante en programmation Python. Pour créer un nouveau dossier, nous utilisons la fonction mkdir() du module os. Voici un exemple :

import os

os.mkdir("nouveau_dossier")

Dans cet exemple, nous importons le module os et utilisons la fonction mkdir() pour créer un nouveau dossier appelé "nouveau_dossier".

Pour supprimer un dossier, nous utilisons la fonction rmdir() du module os. Voici un exemple :

import os

os.rmdir("ancien_dossier")

Dans cet exemple, nous importons le module os et utilisons la fonction rmdir() pour supprimer le dossier "ancien_dossier".

Les opérations sur les fichiers et les dossiers

En plus de la lecture, de l'écriture, de la création et de la suppression de fichiers et de dossiers, Python offre de nombreuses autres opérations pour manipuler ces éléments. Voici quelques exemples :

Pour vérifier si un fichier ou un dossier existe, nous utilisons la fonction exists() du module os.path. Voici un exemple :

import os.path

if os.path.exists("mon_fichier.txt"):
    print("Le fichier existe.")
else:
    print("Le fichier n'existe pas.")

Dans cet exemple, nous importons le module os.path et utilisons la fonction exists() pour vérifier si le fichier "mon_fichier.txt" existe. Ensuite, nous affichons un message approprié en fonction du résultat.

Pour renommer un fichier ou un dossier, nous utilisons la fonction rename() du module os. Voici un exemple :

import os

os.rename("ancien_nom.txt", "nouveau_nom.txt")

Dans cet exemple, nous importons le module os et utilisons la fonction rename() pour renommer le fichier "ancien_nom.txt" en "nouveau_nom.txt".

Pour copier un fichier, nous utilisons la fonction copy() du module shutil. Voici un exemple :

import shutil

shutil.copy("source.txt", "destination.txt")

Dans cet exemple, nous importons le module shutil et utilisons la fonction copy() pour copier le fichier "source.txt" vers "destination.txt".

Exemple complet :

import os
import shutil

# Lecture de fichier
file = open("mon_fichier.txt", "r")
content = file.read()
file.close()

print(content)

# Écriture dans un fichier
file = open("mon_fichier.txt", "w")
file.write("Bonjour, monde !")
file.close()

# Création d'un dossier
os.mkdir("nouveau_dossier")

# Suppression d'un dossier
os.rmdir("ancien_dossier")

# Vérification de l'existence d'un fichier
if os.path.exists("mon_fichier.txt"):
    print("Le fichier existe.")
else:
    print("Le fichier n'existe pas.")

# Renommage d'un fichier
os.rename("ancien_nom.txt", "nouveau_nom.txt")

# Copie d'un fichier
shutil.copy("source.txt", "destination.txt")

Cet exemple illustre les différentes opérations de manipulation de fichiers et de dossiers en Python. Assurez-vous de comprendre chaque ligne de code et de l'adapter à vos besoins spécifiques.
