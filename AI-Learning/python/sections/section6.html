Section 6: Les modules avancés de Python

Gestion des exceptions

La gestion des exceptions est une fonctionnalité essentielle de Python qui permet de gérer les erreurs et les exceptions qui se produisent lors de l'exécution d'un programme. Les exceptions sont des événements imprévus qui peuvent interrompre le flux normal d'un programme. Python fournit des mécanismes pour détecter, signaler et gérer ces exceptions de manière efficace.

Pour gérer les exceptions, vous pouvez utiliser la structure try-except. La structure try permet d'exécuter un bloc de code susceptible de générer une exception, tandis que la structure except permet de spécifier comment gérer cette exception. 


try:
    # Code susceptible de générer une exception
    result = 10 / 0
except ZeroDivisionError:
    # Gestion de l'exception ZeroDivisionError
    print("Erreur: division par zéro")

Dans cet exemple, le code à l'intérieur du bloc try tente de diviser 10 par zéro, ce qui génère une exception ZeroDivisionError. Le bloc except spécifie comment gérer cette exception en affichant un message d'erreur approprié.

Gestion des erreurs et des exceptions

En plus des exceptions intégrées telles que ZeroDivisionError, Python permet également de définir et de gérer des erreurs personnalisées. Vous pouvez créer vos propres exceptions en définissant des classes qui héritent de la classe de base Exception. 


class MonErreur(Exception):
    pass

try:
    raise MonErreur("Ceci est une erreur personnalisée")
except MonErreur as e:
    print(e)

Dans cet exemple, nous définissons une classe MonErreur qui hérite de la classe Exception. Ensuite, nous utilisons le mot-clé raise pour générer une instance de cette exception personnalisée. Le bloc except spécifie comment gérer cette exception en affichant le message d'erreur associé.

Gestion des fichiers CSV et JSON

Python offre des modules intégrés pour faciliter la manipulation des fichiers CSV et JSON. Le module csv permet de lire et d'écrire des fichiers CSV, tandis que le module json permet de lire et d'écrire des fichiers JSON.

Voici un exemple de lecture d'un fichier CSV avec le module csv : 


import csv

with open('fichier.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Dans cet exemple, nous ouvrons le fichier 'fichier.csv' en mode lecture et utilisons le module csv pour lire son contenu. Nous utilisons ensuite une boucle for pour parcourir chaque ligne du fichier et afficher son contenu.

Voici un exemple d'écriture d'un fichier JSON avec le module json : 


import json

data = {
    "nom": "John",
    "age": 30,
    "ville": "Paris"
}

with open('fichier.json', 'w') as file:
    json.dump(data, file)

Dans cet exemple, nous créons un dictionnaire contenant des données, puis utilisons le module json pour écrire ces données dans un fichier JSON appelé 'fichier.json'.

Gestion des bases de données avec Python

Python offre également des modules pour interagir avec des bases de données. Le module le plus couramment utilisé est sqlite3, qui permet de travailler avec des bases de données SQLite.

Voici un exemple de création d'une base de données SQLite et d'une table : 


import sqlite3

conn = sqlite3.connect('ma_base_de_donnees.db')
cursor = conn.cursor()

cursor.execute("CREATE TABLE utilisateurs (id INT, nom TEXT, age INT)")

conn.commit()
conn.close()

Dans cet exemple, nous utilisons le module sqlite3 pour se connecter à une base de données SQLite appelée 'ma_base_de_donnees.db'. Ensuite, nous utilisons un curseur pour exécuter une requête SQL qui crée une table appelée 'utilisateurs' avec trois colonnes. Enfin, nous validons les modifications avec la méthode commit et fermons la connexion à la base de données.

Ceci conclut la section sur les modules avancés de Python. Vous avez appris à gérer les exceptions, à manipuler les fichiers CSV et JSON, ainsi qu'à interagir avec des bases de données SQLite. Ces connaissances vous seront utiles pour développer des applications Python plus robustes et fonctionnelles.
