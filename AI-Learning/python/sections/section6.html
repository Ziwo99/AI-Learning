Section 6: La manipulation de fichiers en Python

L'ouverture et la fermeture de fichiers

En Python, pour manipuler des fichiers, nous devons d'abord les ouvrir et les fermer. Pour ouvrir un fichier, nous utilisons la fonction open(). Cette fonction prend deux arguments : le nom du fichier et le mode d'ouverture.

Le mode d'ouverture peut être :

  • 'r' : lecture (par défaut)
  • 'w' : écriture (crée un nouveau fichier s'il n'existe pas, sinon écrase le contenu existant)
  • 'a' : ajout (ajoute du contenu à la fin du fichier)
  • 'x' : création (crée un nouveau fichier, génère une erreur si le fichier existe déjà)

Voici un exemple d'ouverture et de fermeture d'un fichier :

fichier = open('mon_fichier.txt', 'r')
# Effectuez des opérations sur le fichier
fichier.close()

La lecture et l'écriture de fichiers

Une fois que nous avons ouvert un fichier en mode lecture ou écriture, nous pouvons effectuer des opérations de lecture ou d'écriture sur ce fichier.

Pour lire le contenu d'un fichier, nous utilisons la méthode read(). Cette méthode renvoie une chaîne de caractères contenant tout le contenu du fichier.

Voici un exemple de lecture d'un fichier :

fichier = open('mon_fichier.txt', 'r')
contenu = fichier.read()
print(contenu)
fichier.close()

Pour écrire dans un fichier, nous utilisons la méthode write(). Cette méthode prend une chaîne de caractères en argument et écrit cette chaîne dans le fichier.

Voici un exemple d'écriture dans un fichier :

fichier = open('mon_fichier.txt', 'w')
fichier.write('Bonjour, monde!')
fichier.close()

La gestion des erreurs de fichiers

Lorsque nous manipulons des fichiers, il est important de gérer les erreurs qui peuvent survenir. Par exemple, si le fichier que nous essayons d'ouvrir n'existe pas, une erreur sera générée.

Pour gérer les erreurs de fichiers, nous utilisons des blocs try et except. Dans le bloc try, nous plaçons le code qui peut générer une erreur, et dans le bloc except, nous gérons cette erreur.

Voici un exemple de gestion d'erreur de fichier :

try:
    fichier = open('mon_fichier.txt', 'r')
    contenu = fichier.read()
    print(contenu)
    fichier.close()
except FileNotFoundError:
    print("Le fichier n'existe pas.")

En utilisant cette structure, nous pouvons gérer différentes erreurs de fichiers et prendre des mesures appropriées en conséquence.
