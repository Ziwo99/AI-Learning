Le traitement de fichiers et de données

Le traitement de fichiers et de données est une application courante de Python. Python offre de nombreuses fonctionnalités pour lire, écrire et manipuler des fichiers et des données.

Pour lire un fichier en Python, vous pouvez utiliser la fonction open() avec le mode de lecture approprié. Par exemple, pour lire un fichier texte, vous pouvez utiliser le mode 'r' :

file = open('fichier.txt', 'r')
content = file.read()
file.close()
print(content)

Vous pouvez également lire le contenu d'un fichier ligne par ligne en utilisant la méthode readline() :

file = open('fichier.txt', 'r')
line = file.readline()
while line:
    print(line)
    line = file.readline()
file.close()

Pour écrire dans un fichier, vous pouvez utiliser le mode d'écriture 'w' :

file = open('fichier.txt', 'w')
file.write('Contenu à écrire dans le fichier')
file.close()

Python offre également des fonctionnalités pour manipuler des données, telles que la lecture et l'écriture de fichiers CSV, JSON, XML, etc. Vous pouvez utiliser des bibliothèques externes telles que csv, json et xml.etree.ElementTree pour cela.

Les bases de données et l'accès aux données

Python est souvent utilisé pour interagir avec des bases de données. Il existe plusieurs bibliothèques Python populaires pour cela, telles que sqlite3, MySQLdb, psycopg2, etc.

Pour se connecter à une base de données SQLite en utilisant la bibliothèque sqlite3, vous pouvez utiliser le code suivant :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_données.db')

# Création d'un curseur
cursor = conn.cursor()

# Exécution d'une requête SQL
cursor.execute("SELECT * FROM ma_table")

# Récupération des résultats
results = cursor.fetchall()

# Fermeture de la connexion
conn.close()

# Affichage des résultats
for row in results:
    print(row)

Vous pouvez également utiliser des bibliothèques spécifiques à chaque base de données, telles que MySQLdb pour MySQL ou psycopg2 pour PostgreSQL, pour vous connecter à d'autres types de bases de données.

Les interfaces graphiques

Python offre plusieurs bibliothèques pour créer des interfaces graphiques. L'une des bibliothèques les plus populaires est tkinter, qui est incluse dans la distribution standard de Python.

Voici un exemple simple de création d'une fenêtre avec tkinter :

import tkinter as tk

# Création de la fenêtre
window = tk.Tk()

# Ajout d'un label
label = tk.Label(window, text="Hello, World!")
label.pack()

# Affichage de la fenêtre
window.mainloop()

Vous pouvez également utiliser d'autres bibliothèques pour créer des interfaces graphiques plus avancées, telles que PyQt, wxPython ou Kivy.

Le développement web avec Python

Python est largement utilisé dans le développement web. Il existe plusieurs frameworks web populaires en Python, tels que Django, Flask et Pyramid.

Voici un exemple simple d'une application web avec le framework Flask :

from flask import Flask

# Création de l'application Flask
app = Flask(__name__)

# Définition d'une route
@app.route('/')
def hello():
    return 'Hello, World!'

# Lancement de l'application
if __name__ == '__main__':
    app.run()

Avec Django, vous pouvez créer des applications web plus complexes en utilisant le modèle MVC (Modèle-Vue-Contrôleur).

L'intelligence artificielle et le machine learning avec Python

Python est également largement utilisé dans le domaine de l'intelligence artificielle et du machine learning. Il existe plusieurs bibliothèques Python populaires pour cela, telles que NumPy, SciPy, Pandas, scikit-learn, TensorFlow et Keras.

Voici un exemple simple d'utilisation de la bibliothèque scikit-learn pour créer un modèle de régression linéaire :

from sklearn.linear_model import LinearRegression

# Création du modèle
model = LinearRegression()

# Entraînement du modèle
X_train = [[1], [2], [3], [4]]
y_train = [2, 4, 6, 8]
model.fit(X_train, y_train)

# Prédiction avec le modèle entraîné
X_test = [[5], [6]]
y_pred = model.predict(X_test)

# Affichage des prédictions
print(y_pred)

Avec TensorFlow et Keras, vous pouvez créer des réseaux de neurones et des modèles d'apprentissage profond plus avancés.
