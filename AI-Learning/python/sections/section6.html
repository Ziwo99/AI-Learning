Les modules en Python

En Python, un module est un fichier contenant du code Python qui peut être utilisé dans d'autres programmes. Les modules sont utilisés pour organiser et réutiliser du code, ce qui permet de rendre les programmes plus modulaires et plus faciles à maintenir.

Pour créer un module en Python, il suffit de créer un fichier avec l'extension ".py" et d'y écrire le code que vous souhaitez inclure dans le module. Par exemple, vous pouvez créer un fichier appelé "mon_module.py" et y écrire le code suivant :

def dire_bonjour():
    print("Bonjour !")

def dire_aurevoir():
    print("Au revoir !")

Pour utiliser ce module dans un autre programme Python, vous devez l'importer en utilisant le mot-clé "import". Par exemple :

import mon_module

mon_module.dire_bonjour()
mon_module.dire_aurevoir()

Cela affichera "Bonjour !" suivi de "Au revoir !" lorsque vous exécutez le programme.

Les packages en Python

Un package en Python est un dossier contenant un ensemble de modules. Les packages sont utilisés pour organiser et structurer le code Python en hiérarchies logiques.

Pour créer un package en Python, vous devez créer un dossier et y placer les fichiers de module. Par exemple, vous pouvez créer un dossier appelé "mon_package" et y placer le fichier "mon_module.py" que nous avons créé précédemment.

La structure du dossier "mon_package" ressemblerait à ceci :

mon_package/
    __init__.py
    mon_module.py

Lorsque vous importez un package, Python exécute automatiquement le fichier "__init__.py" situé dans le dossier du package. Cela vous permet d'initialiser des variables, de définir des fonctions ou des classes, ou d'effectuer d'autres opérations d'initialisation.

Pour utiliser un module à l'intérieur d'un package, vous devez spécifier le nom du package suivi du nom du module. Par exemple :

import mon_package.mon_module

mon_package.mon_module.dire_bonjour()
mon_package.mon_module.dire_aurevoir()

Cela affichera également "Bonjour !" suivi de "Au revoir !" lorsque vous exécutez le programme.

L'importation de modules et de packages en Python

En plus d'importer des modules et des packages entiers, vous pouvez également importer des fonctions, des classes ou des variables spécifiques à partir de modules ou de packages.

Pour importer une fonction spécifique à partir d'un module, vous pouvez utiliser la syntaxe suivante :

from mon_module import dire_bonjour

dire_bonjour()

Cela importe uniquement la fonction "dire_bonjour" du module "mon_module" et vous permet de l'utiliser directement sans avoir à spécifier le nom du module.

De même, pour importer une fonction spécifique à partir d'un package, vous pouvez utiliser la syntaxe suivante :

from mon_package.mon_module import dire_bonjour

dire_bonjour()

Cela importe uniquement la fonction "dire_bonjour" du module "mon_module" du package "mon_package" et vous permet de l'utiliser directement.

Vous pouvez également utiliser l'aliasing pour renommer les modules, les packages, les fonctions, les classes ou les variables lors de l'importation. Cela peut être utile pour éviter les conflits de noms ou pour rendre le code plus lisible. Par exemple :

import mon_module as mm

mm.dire_bonjour()
mm.dire_aurevoir()

Cela importe le module "mon_module" en utilisant l'alias "mm", ce qui vous permet d'utiliser les fonctions du module en utilisant "mm.dire_bonjour()" et "mm.dire_aurevoir()".

De même, vous pouvez utiliser l'aliasing lors de l'importation de fonctions spécifiques :

from mon_module import dire_bonjour as bonjour

bonjour()

Cela importe la fonction "dire_bonjour" du module "mon_module" en utilisant l'alias "bonjour", ce qui vous permet d'utiliser la fonction en utilisant simplement "bonjour()".
