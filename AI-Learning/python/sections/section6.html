Section 6: Programmation orientée objet

Introduction à la programmation orientée objet

La programmation orientée objet (POO) est un paradigme de programmation qui permet de structurer et d'organiser le code de manière plus modulaire et réutilisable. En POO, les programmes sont conçus autour d'objets qui représentent des entités du monde réel ou des concepts abstraits.

Un objet est une instance d'une classe, qui est un modèle ou un plan pour créer des objets. Les objets ont des attributs (variables) et des méthodes (fonctions) qui leur sont propres.

Voici un exemple simple de classe en Python :

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is", self.name)

# Création d'un objet de la classe Person
person1 = Person("Alice", 25)

# Appel de la méthode greet() de l'objet
person1.greet()

Les classes et les objets

Une classe est un modèle ou un plan pour créer des objets. Elle définit les attributs et les méthodes communes à tous les objets de cette classe.

Les attributs sont des variables qui stockent des données spécifiques à chaque objet. Les méthodes sont des fonctions qui définissent le comportement des objets de la classe.

Voici un exemple de classe en Python :

class Circle:
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius ** 2

# Création d'un objet de la classe Circle
circle1 = Circle(5)

# Appel de la méthode area() de l'objet
print("Area:", circle1.area())

L'héritage et le polymorphisme

L'héritage est un mécanisme qui permet à une classe d'hériter des attributs et des méthodes d'une autre classe. La classe qui hérite est appelée classe dérivée ou sous-classe, et la classe dont elle hérite est appelée classe de base ou superclasse.

Le polymorphisme est la capacité d'un objet à prendre différentes formes ou à se comporter de différentes manières en fonction du contexte. Il permet d'utiliser une classe dérivée comme si elle était une classe de base.

Voici un exemple d'héritage et de polymorphisme en Python :

class Animal:
    def __init__(self, name):
        self.name = name

    def sound(self):
        pass

class Dog(Animal):
    def sound(self):
        return "Woof!"

class Cat(Animal):
    def sound(self):
        return "Meow!"

# Création d'objets de différentes classes dérivées
dog1 = Dog("Buddy")
cat1 = Cat("Whiskers")

# Appel de la méthode sound() des objets
print(dog1.name, "makes", dog1.sound())
print(cat1.name, "makes", cat1.sound())
