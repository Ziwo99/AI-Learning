Section 6 : Modules et bibliothèques populaires

Utilisation de modules intégrés (math, random, etc.)

Les modules intégrés sont des bibliothèques de fonctions et de classes qui sont incluses dans l'installation de base de Python. Ils fournissent des fonctionnalités supplémentaires prêtes à l'emploi pour faciliter le développement. Voici quelques-uns des modules intégrés les plus couramment utilisés :

  • math : Ce module fournit des fonctions mathématiques avancées telles que les fonctions trigonométriques, exponentielles et logarithmiques.
  • random : Ce module permet de générer des nombres aléatoires. Il offre des fonctions pour générer des nombres entiers aléatoires, des nombres flottants aléatoires et des séquences aléatoires.
  • datetime : Ce module permet de manipuler des objets de date et d'heure. Il offre des fonctions pour créer, formater et manipuler des dates et des heures.
  • os : Ce module fournit des fonctions pour interagir avec le système d'exploitation. Il permet d'accéder aux fonctionnalités du système d'exploitation telles que la création de répertoires, la suppression de fichiers, etc.

Voici un exemple d'utilisation du module math :

import math

# Utilisation de la fonction trigonométrique sin()
angle = math.pi / 2
sin_value = math.sin(angle)
print(sin_value)  # Affiche 1.0

# Utilisation de la fonction exponentielle exp()
x = 2
exp_value = math.exp(x)
print(exp_value)  # Affiche 7.3890560989306495

Introduction à NumPy

NumPy est une bibliothèque Python populaire utilisée pour effectuer des calculs numériques et des opérations sur des tableaux multidimensionnels. Il fournit des fonctions et des méthodes efficaces pour manipuler les tableaux, effectuer des opérations mathématiques sur les tableaux et résoudre des problèmes liés aux données numériques.

Voici un exemple d'utilisation de NumPy :

import numpy as np

# Création d'un tableau unidimensionnel
arr = np.array([1, 2, 3, 4, 5])
print(arr)  # Affiche [1 2 3 4 5]

# Création d'un tableau multidimensionnel
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d)
# Affiche [[1 2 3]
#          [4 5 6]]

# Opérations mathématiques sur les tableaux
arr_squared = np.square(arr)
print(arr_squared)  # Affiche [ 1  4  9 16 25]

# Fonctions statistiques sur les tableaux
arr_mean = np.mean(arr)
print(arr_mean)  # Affiche 3.0

Introduction à Pandas

Pandas est une bibliothèque Python utilisée pour la manipulation et l'analyse de données. Elle fournit des structures de données flexibles et performantes, telles que les DataFrames et les Series, qui permettent de manipuler facilement les données et d'effectuer des opérations de traitement et d'analyse de données.

Voici un exemple d'utilisation de Pandas :

import pandas as pd

# Création d'un DataFrame à partir d'un dictionnaire
data = {'Nom': ['Alice', 'Bob', 'Charlie'],
        'Âge': [25, 30, 35],
        'Ville': ['Paris', 'Londres', 'New York']}
df = pd.DataFrame(data)
print(df)
# Affiche :
#        Nom  Âge     Ville
# 0    Alice   25     Paris
# 1      Bob   30   Londres
# 2  Charlie   35  New York

# Accès aux colonnes du DataFrame
ages = df['Âge']
print(ages)  # Affiche [25, 30, 35]

# Filtrage des lignes du DataFrame
filtered_df = df[df['Âge'] > 30]
print(filtered_df)
# Affiche :
#        Nom  Âge     Ville
# 2  Charlie   35  New York

Introduction à Matplotlib

Matplotlib est une bibliothèque Python utilisée pour créer des visualisations de données. Elle permet de créer des graphiques, des diagrammes, des histogrammes, des diagrammes en boîte et bien plus encore. Matplotlib offre une grande flexibilité pour personnaliser les graphiques et les rendre attrayants.

Voici un exemple d'utilisation de Matplotlib pour tracer un graphique linéaire :

import matplotlib.pyplot as plt

# Données pour l'axe des abscisses
x = [1, 2, 3, 4, 5]
# Données pour l'axe des ordonnées
y = [2, 4, 6, 8, 10]

# Tracé du graphique linéaire
plt.plot(x, y)

# Ajout de labels et de titre
plt.xlabel('Axe des abscisses')
plt.ylabel('Axe des ordonnées')
plt.title('Graphique linéaire')

# Affichage du graphique
plt.show()

Ce code produira un graphique linéaire avec les valeurs de x sur l'axe des abscisses et les valeurs de y sur l'axe des ordonnées.
