Section 6 : Modules et bibliothèques

Utilisation des modules standards

Les modules standards sont des bibliothèques de code Python pré-installées qui offrent des fonctionnalités supplémentaires prêtes à l'emploi. Ils peuvent être importés dans votre programme à l'aide de l'instruction import. Voici quelques exemples de modules standards couramment utilisés :

import math

# Utilisation du module math pour calculer la racine carrée d'un nombre
x = math.sqrt(25)
print(x)

import random

# Utilisation du module random pour générer un nombre aléatoire entre 1 et 10
y = random.randint(1, 10)
print(y)

import datetime

# Utilisation du module datetime pour obtenir la date et l'heure actuelles
current_time = datetime.datetime.now()
print(current_time)

Installation et utilisation de bibliothèques externes

En plus des modules standards, vous pouvez également utiliser des bibliothèques externes pour étendre les fonctionnalités de Python. Ces bibliothèques doivent être installées avant de pouvoir être utilisées. Voici comment installer une bibliothèque externe à l'aide de l'outil de gestion de paquets pip :

pip install nom_de_la_bibliothèque

Une fois la bibliothèque installée, vous pouvez l'importer dans votre programme et l'utiliser. Voici un exemple d'utilisation de la bibliothèque requests pour effectuer une requête HTTP :

import requests

response = requests.get("https://www.example.com")
print(response.text)

Création de modules personnalisés

En plus d'utiliser des modules existants, vous pouvez également créer vos propres modules personnalisés pour organiser votre code en fonctionnalités réutilisables. Voici comment créer un module personnalisé :

1. Créez un fichier Python avec l'extension .py, par exemple mon_module.py.

2. Définissez les fonctions, classes et variables que vous souhaitez inclure dans votre module.

3. Importez votre module dans un autre programme en utilisant l'instruction import.

Voici un exemple de création d'un module personnalisé :

# mon_module.py

def dire_bonjour():
    print("Bonjour !")

def addition(a, b):
    return a + b

Vous pouvez ensuite utiliser ce module personnalisé dans un autre programme :

import mon_module

mon_module.dire_bonjour()
resultat = mon_module.addition(3, 5)
print(resultat)

En utilisant des modules personnalisés, vous pouvez organiser votre code de manière plus modulaire et réutilisable.
