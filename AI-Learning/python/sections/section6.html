Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en Python. Pour lire un fichier, vous pouvez utiliser la fonction intégrée open() avec le mode de lecture approprié. Par exemple, pour lire un fichier texte en mode lecture, vous pouvez utiliser :

file = open("nom_du_fichier.txt", "r")
content = file.read()
file.close()
print(content)

Ici, nous ouvrons le fichier "nom_du_fichier.txt" en mode lecture ("r"), nous lisons son contenu avec la méthode read(), puis nous fermons le fichier avec la méthode close(). Ensuite, nous imprimons le contenu du fichier.

Pour écrire dans un fichier, vous pouvez utiliser la fonction open() avec le mode d'écriture approprié. Par exemple, pour écrire dans un fichier texte en mode écriture, vous pouvez utiliser :

file = open("nom_du_fichier.txt", "w")
file.write("Contenu à écrire dans le fichier")
file.close()

Ici, nous ouvrons le fichier "nom_du_fichier.txt" en mode écriture ("w"), nous écrivons le contenu spécifié avec la méthode write(), puis nous fermons le fichier avec la méthode close().

Manipulation de fichiers CSV et JSON

Python offre des bibliothèques intégrées pour manipuler des fichiers CSV et JSON.

Pour manipuler des fichiers CSV, vous pouvez utiliser le module csv. Voici un exemple de lecture d'un fichier CSV :

import csv

with open('fichier.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Ici, nous utilisons la fonction open() pour ouvrir le fichier CSV en mode lecture. Ensuite, nous utilisons la fonction csv.reader() pour créer un objet lecteur qui nous permet de parcourir les lignes du fichier CSV. Enfin, nous utilisons une boucle for pour imprimer chaque ligne du fichier.

Pour écrire dans un fichier CSV, vous pouvez utiliser le module csv avec la fonction writer(). Voici un exemple :

import csv

data = [
    ['Nom', 'Âge', 'Ville'],
    ['Alice', 25, 'Paris'],
    ['Bob', 30, 'Londres'],
    ['Charlie', 35, 'New York']
]

with open('fichier.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerows(data)

Ici, nous utilisons la fonction open() pour ouvrir le fichier CSV en mode écriture. Ensuite, nous utilisons la fonction csv.writer() pour créer un objet écrivain qui nous permet d'écrire des lignes dans le fichier CSV. Enfin, nous utilisons la méthode writerows() pour écrire toutes les lignes de données dans le fichier.

Pour manipuler des fichiers JSON, vous pouvez utiliser le module json. Voici un exemple de lecture d'un fichier JSON :

import json

with open('fichier.json', 'r') as file:
    data = json.load(file)
    print(data)

Ici, nous utilisons la fonction open() pour ouvrir le fichier JSON en mode lecture. Ensuite, nous utilisons la fonction json.load() pour charger les données JSON du fichier dans un objet Python. Enfin, nous imprimons les données.

Pour écrire dans un fichier JSON, vous pouvez utiliser le module json avec la fonction dump(). Voici un exemple :

import json

data = {
    'nom': 'Alice',
    'âge': 25,
    'ville': 'Paris'
}

with open('fichier.json', 'w') as file:
    json.dump(data, file)

Ici, nous utilisons la fonction open() pour ouvrir le fichier JSON en mode écriture. Ensuite, nous utilisons la fonction json.dump() pour écrire les données JSON dans le fichier.

Connexion à une base de données

Python offre de nombreuses bibliothèques pour se connecter à différentes bases de données, telles que MySQL, PostgreSQL, SQLite, etc. Dans cet exemple, nous utiliserons la bibliothèque sqlite3 pour se connecter à une base de données SQLite.

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_données.db')

# Création d'un curseur
cursor = conn.cursor()

# Exécution d'une requête SQL
cursor.execute("SELECT * FROM ma_table")

# Récupération des résultats
results = cursor.fetchall()

# Affichage des résultats
for row in results:
    print(row)

# Fermeture de la connexion
conn.close()

Ici, nous utilisons la fonction connect() du module sqlite3 pour se connecter à la base de données SQLite spécifiée. Ensuite, nous créons un curseur à l'aide de la méthode cursor(). Nous exécutons une requête SQL à l'aide de la méthode execute() et récupérons les résultats avec la méthode fetchall(). Enfin, nous parcourons les résultats et les affichons.

Requêtes SQL avec Python

Pour exécuter des requêtes SQL avec Python, vous pouvez utiliser les méthodes execute() et fetchall() du curseur. Voici un exemple :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_données.db')
cursor = conn.cursor()

# Exécution d'une requête SELECT
cursor.execute("SELECT * FROM ma_table")
results = cursor.fetchall()
for row in results:
    print(row)

# Exécution d'une requête INSERT
cursor.execute("INSERT INTO ma_table (colonne1, colonne2) VALUES (?, ?)", ('valeur1', 'valeur2'))
conn.commit()

# Exécution d'une requête UPDATE
cursor.execute("UPDATE ma_table SET colonne1 = ? WHERE colonne2 = ?", ('nouvelle_valeur', 'valeur2'))
conn.commit()

# Exécution d'une requête DELETE
cursor.execute("DELETE FROM ma_table WHERE colonne1 = ?", ('valeur1',))
conn.commit()

# Fermeture de la connexion
conn.close()

Ici, nous utilisons les méthodes execute() et fetchall() pour exécuter des requêtes SELECT et récupérer les résultats. Pour les requêtes INSERT, UPDATE et DELETE, nous utilisons également la méthode execute(), puis nous appelons la méthode commit() pour valider les modifications dans la base de données.
