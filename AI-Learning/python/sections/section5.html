Introduction aux fonctions en Python

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent de structurer et d'organiser le code en le divisant en tâches plus petites et plus gérables. En Python, une fonction est définie à l'aide du mot-clé "def", suivi du nom de la fonction et de parenthèses contenant éventuellement des paramètres. Voici un exemple de définition de fonction :

def dire_bonjour():
    print("Bonjour !")

Pour appeler une fonction, il suffit d'utiliser son nom suivi de parenthèses :

dire_bonjour()

Cela affichera "Bonjour !" à l'écran. Les fonctions peuvent également prendre des paramètres, qui sont des valeurs que vous pouvez passer à la fonction pour qu'elle les utilise dans son code. Par exemple :

def dire_bonjour(nom):
    print("Bonjour", nom)

Vous pouvez ensuite appeler cette fonction en passant un argument :

dire_bonjour("Alice")

Cela affichera "Bonjour Alice" à l'écran. Les fonctions peuvent également retourner une valeur à l'aide du mot-clé "return". Par exemple :

def ajouter(a, b):
    return a + b

Vous pouvez ensuite utiliser cette fonction pour effectuer une opération et stocker le résultat dans une variable :

resultat = ajouter(3, 5)
print(resultat)  # affiche 8

Les paramètres et les arguments en Python

Les paramètres sont des variables déclarées dans la définition d'une fonction, qui reçoivent les valeurs passées en tant qu'arguments lors de l'appel de la fonction. Les arguments sont les valeurs réelles passées à une fonction lors de son appel. Voici un exemple :

def multiplier(a, b):
    return a * b

resultat = multiplier(4, 6)
print(resultat)  # affiche 24

Dans cet exemple, "a" et "b" sont les paramètres de la fonction "multiplier", et "4" et "6" sont les arguments passés lors de l'appel de la fonction.

Les fonctions récursives en Python

Une fonction récursive est une fonction qui s'appelle elle-même. Cela peut être utile pour résoudre des problèmes qui peuvent être décomposés en sous-problèmes plus petits. Voici un exemple de fonction récursive qui calcule la factorielle d'un nombre :

def factorielle(n):
    if n == 0:
        return 1
    else:
        return n * factorielle(n - 1)

resultat = factorielle(5)
print(resultat)  # affiche 120

Dans cet exemple, la fonction "factorielle" s'appelle elle-même avec un argument plus petit à chaque itération jusqu'à ce que la condition de base soit atteinte (n == 0).

Les fonctions lambda en Python

Les fonctions lambda, également appelées fonctions anonymes, sont des fonctions sans nom qui peuvent être utilisées pour effectuer des opérations simples. Elles sont définies à l'aide du mot-clé "lambda" suivi des paramètres et du corps de la fonction. Voici un exemple :

carre = lambda x: x ** 2

resultat = carre(5)
print(resultat)  # affiche 25

Dans cet exemple, la fonction lambda "carre" prend un paramètre "x" et renvoie "x ** 2". Elle est ensuite appelée avec l'argument "5" pour obtenir le carré de 5.
