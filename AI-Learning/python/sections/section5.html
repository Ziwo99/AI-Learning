Section 5: Les modules en Python

Introduction aux modules en Python

Les modules en Python sont des fichiers contenant des définitions et des instructions qui peuvent être utilisées dans d'autres programmes Python. Ils permettent de regrouper du code lié à une fonctionnalité spécifique, ce qui facilite la réutilisation et la gestion du code.

Pour utiliser un module en Python, il faut l'importer dans le programme. Cela se fait en utilisant le mot-clé import, suivi du nom du module. Par exemple :

import math

Une fois le module importé, on peut utiliser ses fonctions, classes et variables en utilisant la syntaxe nom_du_module.nom_de_la_fonction() ou nom_du_module.nom_de_la_variable. Par exemple :

import math

print(math.sqrt(16))  # Affiche la racine carrée de 16

Les modules intégrés en Python

Python dispose de nombreux modules intégrés qui offrent des fonctionnalités prêtes à l'emploi. Voici quelques exemples :

  • math : fournit des fonctions mathématiques avancées telles que les racines carrées, les logarithmes, les fonctions trigonométriques, etc.
  • random : permet de générer des nombres aléatoires.
  • datetime : offre des fonctionnalités pour manipuler les dates et les heures.
  • os : permet d'interagir avec le système d'exploitation, par exemple pour manipuler des fichiers et des répertoires.

Pour utiliser ces modules, il suffit de les importer comme expliqué précédemment. Par exemple :

import random

print(random.randint(1, 10))  # Affiche un nombre aléatoire entre 1 et 10

L'importation de modules en Python

En plus des modules intégrés, il est possible d'importer des modules créés par d'autres développeurs ou par vous-même. Ces modules peuvent être téléchargés à partir de sources externes ou être présents dans le même répertoire que votre programme.

Pour importer un module externe, vous devez d'abord l'installer à l'aide d'un gestionnaire de paquets tel que pip. Une fois installé, vous pouvez l'importer de la même manière que les modules intégrés.

Pour importer un module présent dans le même répertoire que votre programme, vous pouvez utiliser la syntaxe from nom_du_module import nom_de_la_fonction ou from nom_du_module import * pour importer toutes les fonctions du module. Par exemple :

from mon_module import ma_fonction

ma_fonction()  # Appelle la fonction ma_fonction du module mon_module

Il est également possible de renommer un module lors de l'importation en utilisant le mot-clé as. Cela peut être utile pour éviter les conflits de noms. Par exemple :

import mon_module as mm

mm.ma_fonction()  # Appelle la fonction ma_fonction du module mon_module en utilisant le nom mm

Il est recommandé de consulter la documentation du module que vous souhaitez utiliser pour connaître les fonctions et les fonctionnalités disponibles.
