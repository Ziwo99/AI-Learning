Les classes et les objets

En Python, la programmation orientée objet (POO) repose sur le concept de classes et d'objets. Une classe est un modèle ou un plan pour créer des objets, tandis qu'un objet est une instance d'une classe. Les objets sont utilisés pour représenter des entités du monde réel ou des concepts abstraits.

Pour définir une classe en Python, utilisez le mot-clé class suivi du nom de la classe. Par exemple, voici comment définir une classe simple appelée "Personne" :

class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def dire_bonjour(self):
        print("Bonjour, je m'appelle", self.nom, "et j'ai", self.age, "ans.")

Dans cet exemple, la méthode spéciale __init__ est utilisée pour initialiser les attributs de l'objet lors de sa création. Les méthodes d'une classe sont des fonctions qui peuvent être appelées sur les objets de cette classe.

Pour créer un objet à partir d'une classe, utilisez le nom de la classe suivi de parenthèses. Par exemple :

personne1 = Personne("Alice", 25)

Une fois que vous avez créé un objet, vous pouvez accéder à ses attributs et appeler ses méthodes. Par exemple :

print(personne1.nom)  # Affiche "Alice"
print(personne1.age)  # Affiche 25
personne1.dire_bonjour()  # Affiche "Bonjour, je m'appelle Alice et j'ai 25 ans."

Les attributs et les méthodes

Les attributs sont des variables associées à un objet. Ils représentent les caractéristiques de l'objet. Les méthodes, quant à elles, sont des fonctions associées à un objet. Elles permettent de définir le comportement de l'objet.

Les attributs peuvent être définis à l'intérieur de la méthode __init__ ou en dehors de celle-ci. Par exemple :

class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def dire_bonjour(self):
        print("Bonjour, je m'appelle", self.nom, "et j'ai", self.age, "ans.")

    def changer_nom(self, nouveau_nom):
        self.nom = nouveau_nom

Dans cet exemple, la méthode changer_nom permet de modifier l'attribut nom d'un objet.

L'héritage et le polymorphisme

L'héritage est un concept clé de la POO qui permet de créer de nouvelles classes en se basant sur des classes existantes. Une classe qui hérite d'une autre classe est appelée une classe dérivée ou une sous-classe, tandis que la classe dont elle hérite est appelée une classe de base ou une superclasse.

En Python, l'héritage s'effectue en spécifiant la classe de base entre parenthèses après le nom de la classe dérivée. Par exemple :

class Employe(Personne):
    def __init__(self, nom, age, salaire):
        super().__init__(nom, age)
        self.salaire = salaire

    def dire_bonjour(self):
        super().dire_bonjour()
        print("Je suis un employé et mon salaire est de", self.salaire, "euros.")

Dans cet exemple, la classe Employe hérite de la classe Personne. Elle ajoute un nouvel attribut salaire et redéfinit la méthode dire_bonjour pour afficher des informations spécifiques aux employés.

Le polymorphisme est un autre concept de la POO qui permet à des objets de différentes classes d'être traités de manière similaire. Cela signifie qu'un objet peut être utilisé comme s'il était de la classe de base, même s'il est en réalité de la classe dérivée.

Les exceptions personnalisées

En Python, vous pouvez créer vos propres exceptions personnalisées en définissant une classe qui hérite de la classe de base Exception. Cela vous permet de gérer des erreurs spécifiques à votre programme.

Voici un exemple de définition d'une exception personnalisée :

class MonException(Exception):
    pass

Dans cet exemple, la classe MonException hérite de la classe Exception. Vous pouvez ensuite utiliser cette exception personnalisée dans votre code pour signaler des erreurs spécifiques.

Les modules et les packages orientés objet

En Python, vous pouvez organiser votre code orienté objet en utilisant des modules et des packages. Un module est un fichier contenant du code Python, tandis qu'un package est un dossier contenant plusieurs modules.

Pour créer un module, créez simplement un fichier Python avec l'extension .py et placez-y votre code. Vous pouvez ensuite importer ce module dans d'autres fichiers Python pour réutiliser le code.

Pour créer un package, créez un dossier et placez-y plusieurs fichiers Python. Assurez-vous d'inclure un fichier spécial appelé __init__.py dans le dossier pour indiquer à Python qu'il s'agit d'un package.

Voici un exemple de structure de package orienté objet :

mon_package/
    __init__.py
    module1.py
    module2.py

Dans cet exemple, le package mon_package contient deux modules, module1 et module2. Vous pouvez ensuite importer ces modules dans d'autres fichiers Python pour utiliser les classes et les objets qu'ils contiennent.
