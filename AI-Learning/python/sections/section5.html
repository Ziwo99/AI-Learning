Section 5: Les modules en Python

Introduction aux modules

Les modules en Python sont des fichiers contenant du code Python qui peuvent être importés et utilisés dans d'autres programmes. Ils permettent d'organiser et de réutiliser du code de manière efficace. Les modules peuvent contenir des fonctions, des classes, des variables et d'autres éléments de code.

Pour créer un module, il suffit de créer un fichier avec l'extension .py et d'y écrire le code Python. Par exemple, pour créer un module nommé "mon_module.py", vous pouvez utiliser un éditeur de texte pour écrire le code suivant :

def ma_fonction():
    print("Bonjour depuis mon module")
    
ma_variable = 42

Pour utiliser ce module dans un autre programme Python, vous devez l'importer en utilisant le mot-clé import. Par exemple :

import mon_module

mon_module.ma_fonction()
print(mon_module.ma_variable)

Cela affichera :

Bonjour depuis mon module
42

Les modules intégrés

Python dispose de nombreux modules intégrés qui fournissent des fonctionnalités supplémentaires prêtes à l'emploi. Ces modules sont inclus dans l'installation standard de Python et peuvent être utilisés sans avoir besoin d'installer des bibliothèques supplémentaires.

Par exemple, le module math fournit des fonctions mathématiques telles que sqrt (racine carrée), sin (sinus), cos (cosinus), etc. Pour utiliser ce module, vous devez l'importer :

import math

print(math.sqrt(16))
print(math.sin(math.pi/2))

Cela affichera :

4.0
1.0

L'importation de modules externes

En plus des modules intégrés, vous pouvez également utiliser des modules externes dans vos programmes Python. Ces modules sont développés par la communauté Python et fournissent des fonctionnalités supplémentaires pour des tâches spécifiques.

Pour utiliser un module externe, vous devez d'abord l'installer. Cela peut être fait en utilisant l'outil de gestion de paquets de Python appelé pip. Par exemple, pour installer le module requests qui est utilisé pour effectuer des requêtes HTTP, vous pouvez exécuter la commande suivante dans votre terminal :

pip install requests

Une fois le module installé, vous pouvez l'importer dans votre programme Python :

import requests

response = requests.get("https://www.example.com")
print(response.status_code)

Cela affichera le code d'état de la requête HTTP effectuée.
