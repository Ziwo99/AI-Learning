Section 5: Programmation orientée objet en Python

Introduction à la programmation orientée objet

La programmation orientée objet (POO) est un paradigme de programmation qui permet de structurer le code en utilisant des objets. Un objet est une instance d'une classe, qui est un modèle ou un plan pour créer des objets. La POO offre une approche modulaire et réutilisable pour développer des applications.

En Python, tout est un objet. Les types de données primitifs tels que les entiers, les flottants et les chaînes de caractères sont des objets. Les objets ont des attributs (variables) et des méthodes (fonctions) qui leur sont propres.

Classes et objets

Une classe est un modèle pour créer des objets. Elle définit les attributs et les méthodes que les objets de cette classe auront. Pour créer un objet à partir d'une classe, on utilise le mot-clé class suivi du nom de la classe.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is " + self.name + " and I am " + str(self.age) + " years old.")

person1 = Person("Alice", 25)
person1.greet()

Dans cet exemple, nous avons créé une classe Person avec deux attributs : name et age. La méthode spéciale __init__ est appelée lors de la création d'un nouvel objet et permet d'initialiser ses attributs. La méthode greet affiche un message de salutation en utilisant les attributs de l'objet.

Héritage et polymorphisme

L'héritage est un concept clé de la POO qui permet de créer de nouvelles classes en se basant sur des classes existantes. Une classe qui hérite d'une autre classe est appelée une classe dérivée ou sous-classe, tandis que la classe dont elle hérite est appelée une classe de base ou superclasse.

Le polymorphisme est la capacité d'un objet à prendre différentes formes. En Python, le polymorphisme est géré automatiquement grâce à la dynamique du langage. Cela signifie que vous pouvez utiliser une méthode d'une classe de base sur un objet d'une classe dérivée.

class Animal:
    def __init__(self, name):
        self.name = name

    def make_sound(self):
        pass

class Dog(Animal):
    def make_sound(self):
        print("Woof!")

class Cat(Animal):
    def make_sound(self):
        print("Meow!")

dog = Dog("Buddy")
cat = Cat("Whiskers")

dog.make_sound()
cat.make_sound()

Dans cet exemple, nous avons une classe de base Animal avec une méthode make_sound qui est définie comme une méthode abstraite (elle ne fait rien). Les classes dérivées Dog et Cat héritent de la classe Animal et redéfinissent la méthode make_sound pour produire des sons spécifiques.

En utilisant le polymorphisme, nous pouvons appeler la méthode make_sound sur des objets de différentes classes dérivées, et le comportement sera spécifique à chaque classe.
