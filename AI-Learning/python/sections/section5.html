La lecture et l'écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en Python. Pour lire le contenu d'un fichier, nous utilisons la fonction open() avec le mode de lecture approprié. Voici un exemple :

file = open('mon_fichier.txt', 'r')
content = file.read()
file.close()
print(content)

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode lecture ('r'), nous lisons son contenu avec la méthode read(), puis nous fermons le fichier avec la méthode close(). Ensuite, nous affichons le contenu du fichier.

Pour écrire dans un fichier, nous utilisons la fonction open() avec le mode d'écriture approprié. Voici un exemple :

file = open('mon_fichier.txt', 'w')
file.write('Contenu à écrire dans le fichier')
file.close()

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode écriture ('w'), nous écrivons le contenu spécifié avec la méthode write(), puis nous fermons le fichier avec la méthode close().

La gestion des exceptions et des erreurs

La gestion des exceptions et des erreurs est essentielle pour garantir la robustesse de nos programmes. En Python, nous utilisons des blocs try et except pour gérer les exceptions. Voici un exemple :

try:
    # Code susceptible de générer une exception
    file = open('mon_fichier.txt', 'r')
    content = file.read()
    file.close()
    print(content)
except FileNotFoundError:
    # Code à exécuter si une exception de type FileNotFoundError est levée
    print("Le fichier n'a pas été trouvé")
except:
    # Code à exécuter si une exception de tout autre type est levée
    print("Une erreur s'est produite")

Dans cet exemple, nous essayons d'ouvrir le fichier "mon_fichier.txt" en mode lecture. Si le fichier n'est pas trouvé, une exception de type FileNotFoundError est levée et le code dans le bloc except FileNotFoundError est exécuté. Si une exception de tout autre type est levée, le code dans le bloc except est exécuté.

Il est également possible d'utiliser un bloc finally pour exécuter du code quel que soit le résultat du bloc try. Voici un exemple :

try:
    # Code susceptible de générer une exception
    file = open('mon_fichier.txt', 'r')
    content = file.read()
    file.close()
    print(content)
except FileNotFoundError:
    # Code à exécuter si une exception de type FileNotFoundError est levée
    print("Le fichier n'a pas été trouvé")
except:
    # Code à exécuter si une exception de tout autre type est levée
    print("Une erreur s'est produite")
finally:
    # Code à exécuter quel que soit le résultat du bloc try
    print("Fin du programme")

Dans cet exemple, le code dans le bloc finally est exécuté après l'exécution du bloc try ou du bloc except.
