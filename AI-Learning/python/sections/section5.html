Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en Python. Pour lire un fichier, vous pouvez utiliser la fonction intégrée open() avec le mode de lecture approprié. Voici un exemple :

file = open('mon_fichier.txt', 'r')
content = file.read()
file.close()

print(content)

Dans cet exemple, nous ouvrons le fichier mon_fichier.txt en mode lecture ('r'), nous lisons son contenu avec la méthode read() et nous fermons ensuite le fichier avec la méthode close(). Enfin, nous affichons le contenu du fichier.

Pour écrire dans un fichier, vous pouvez utiliser la fonction open() avec le mode d'écriture approprié. Voici un exemple :

file = open('mon_fichier.txt', 'w')
file.write('Contenu à écrire dans le fichier')
file.close()

Dans cet exemple, nous ouvrons le fichier mon_fichier.txt en mode écriture ('w'), nous écrivons le contenu spécifié avec la méthode write() et nous fermons ensuite le fichier avec la méthode close().

Manipulation de fichiers CSV et JSON

Python offre des bibliothèques intégrées pour manipuler des fichiers CSV et JSON.

Pour lire un fichier CSV, vous pouvez utiliser le module csv. Voici un exemple :

import csv

with open('mon_fichier.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Dans cet exemple, nous ouvrons le fichier mon_fichier.csv en mode lecture ('r') avec la clause with, ce qui garantit que le fichier sera correctement fermé même en cas d'erreur. Nous utilisons ensuite la fonction reader() du module csv pour lire le contenu du fichier ligne par ligne, et nous affichons chaque ligne.

Pour écrire dans un fichier CSV, vous pouvez utiliser le module csv en combinaison avec la fonction writer(). Voici un exemple :

import csv

data = [
    ['Nom', 'Âge', 'Ville'],
    ['Alice', 25, 'Paris'],
    ['Bob', 30, 'Londres'],
    ['Charlie', 35, 'New York']
]

with open('mon_fichier.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerows(data)

Dans cet exemple, nous créons une liste de listes data contenant les données que nous voulons écrire dans le fichier CSV. Nous ouvrons ensuite le fichier mon_fichier.csv en mode écriture ('w') avec la clause with, et nous utilisons la fonction writer() du module csv pour écrire les données dans le fichier.

Pour manipuler des fichiers JSON, vous pouvez utiliser le module json. Voici un exemple de lecture :

import json

with open('mon_fichier.json', 'r') as file:
    data = json.load(file)

print(data)

Dans cet exemple, nous ouvrons le fichier mon_fichier.json en mode lecture ('r') avec la clause with. Nous utilisons ensuite la fonction load() du module json pour charger les données JSON du fichier dans une variable data, et nous l'affichons.

Pour écrire dans un fichier JSON, vous pouvez utiliser la fonction dump() du module json. Voici un exemple :

import json

data = {
    'nom': 'Alice',
    'age': 25,
    'ville': 'Paris'
}

with open('mon_fichier.json', 'w') as file:
    json.dump(data, file)

Dans cet exemple, nous créons un dictionnaire data contenant les données que nous voulons écrire dans le fichier JSON. Nous ouvrons ensuite le fichier mon_fichier.json en mode écriture ('w') avec la clause with, et nous utilisons la fonction dump() du module json pour écrire les données dans le fichier.

Connexion à une base de données

Pour se connecter à une base de données en Python, vous pouvez utiliser un module adapté au type de base de données que vous utilisez. Par exemple, pour se connecter à une base de données MySQL, vous pouvez utiliser le module mysql-connector-python. Voici un exemple :

import mysql.connector

# Paramètres de connexion
config = {
    'user': 'utilisateur',
    'password': 'mot_de_passe',
    'host': 'localhost',
    'database': 'ma_base_de_donnees'
}

# Connexion à la base de données
conn = mysql.connector.connect(**config)

# Utilisation de la connexion
cursor = conn.cursor()
cursor.execute('SELECT * FROM ma_table')
result = cursor.fetchall()

for row in result:
    print(row)

# Fermeture de la connexion
conn.close()

Dans cet exemple, nous définissons les paramètres de connexion dans un dictionnaire config. Nous utilisons ensuite la fonction connect() du module mysql.connector pour établir la connexion à la base de données. Nous créons un curseur avec la méthode cursor() de la connexion, nous exécutons une requête SQL avec la méthode execute() du curseur, nous récupérons les résultats avec la méthode fetchall() du curseur, et nous affichons chaque ligne.

Enfin, nous fermons la connexion avec la méthode close().

Requêtes SQL avec Python

Python offre plusieurs bibliothèques pour exécuter des requêtes SQL sur une base de données. L'une des plus populaires est sqlite3, qui permet de travailler avec des bases de données SQLite intégrées. Voici un exemple :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_donnees.db')

# Utilisation de la connexion
cursor = conn.cursor()
cursor.execute('SELECT * FROM ma_table')
result = cursor.fetchall()

for row in result:
    print(row)

# Fermeture de la connexion
conn.close()

Dans cet exemple, nous utilisons la fonction connect() du module sqlite3 pour établir la connexion à la base de données SQLite. Nous créons un curseur avec la méthode cursor() de la connexion, nous exécutons une requête SQL avec la méthode execute() du curseur, nous récupérons les résultats avec la méthode fetchall() du curseur, et nous affichons chaque ligne.

Enfin, nous fermons la connexion avec la méthode close().
