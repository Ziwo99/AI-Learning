NumPy pour le calcul scientifique

NumPy est une bibliothèque Python très populaire utilisée pour effectuer des calculs scientifiques et mathématiques. Elle fournit des structures de données de tableau multidimensionnel, ainsi que des fonctions pour effectuer des opérations mathématiques avancées sur ces tableaux.

Voici un exemple d'utilisation de NumPy pour effectuer des opérations mathématiques de base :

import numpy as np

# Création d'un tableau NumPy
arr = np.array([1, 2, 3, 4, 5])

# Calcul de la moyenne des éléments du tableau
mean = np.mean(arr)

# Calcul de l'écart-type des éléments du tableau
std = np.std(arr)

# Affichage des résultats
print("Moyenne :", mean)
print("Écart-type :", std)

Dans cet exemple, nous importons la bibliothèque NumPy en utilisant l'alias "np". Nous créons ensuite un tableau NumPy à partir d'une liste d'entiers. En utilisant les fonctions "mean" et "std" de NumPy, nous calculons respectivement la moyenne et l'écart-type des éléments du tableau. Enfin, nous affichons les résultats.

Pandas pour la manipulation de données

Pandas est une bibliothèque Python très utilisée pour la manipulation et l'analyse de données. Elle fournit des structures de données de haut niveau, telles que les DataFrames, qui permettent de stocker et de manipuler facilement des données tabulaires.

Voici un exemple d'utilisation de Pandas pour lire et manipuler un fichier CSV :

import pandas as pd

# Lecture du fichier CSV
data = pd.read_csv("data.csv")

# Affichage des premières lignes du DataFrame
print(data.head())

# Calcul de la moyenne d'une colonne
mean = data["column_name"].mean()

# Filtrage des données
filtered_data = data[data["column_name"] > 10]

# Affichage des résultats
print("Moyenne :", mean)
print("Données filtrées :", filtered_data)

Dans cet exemple, nous importons la bibliothèque Pandas en utilisant l'alias "pd". Nous utilisons la fonction "read_csv" de Pandas pour lire un fichier CSV et stocker les données dans un DataFrame. En utilisant les fonctionnalités de Pandas, nous pouvons facilement manipuler les données en effectuant des opérations telles que le calcul de la moyenne d'une colonne ou le filtrage des données en fonction d'une condition. Enfin, nous affichons les résultats.

Matplotlib pour la visualisation de données

Matplotlib est une bibliothèque Python très utilisée pour la visualisation de données. Elle permet de créer des graphiques et des visualisations de haute qualité, adaptés à différents types de données et de besoins.

Voici un exemple d'utilisation de Matplotlib pour créer un graphique à barres :

import matplotlib.pyplot as plt

# Données du graphique
x = [1, 2, 3, 4, 5]
y = [10, 15, 7, 12, 9]

# Création du graphique à barres
plt.bar(x, y)

# Ajout de labels et de titres
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.title("Bar Chart")

# Affichage du graphique
plt.show()

Dans cet exemple, nous importons la bibliothèque Matplotlib en utilisant l'alias "plt". Nous créons ensuite deux listes, "x" et "y", qui représentent les données du graphique à barres. En utilisant la fonction "bar" de Matplotlib, nous créons le graphique à barres en utilisant les données fournies. Enfin, nous ajoutons des labels et un titre au graphique, puis nous l'affichons.

Flask pour le développement web

Flask est un framework web léger et flexible écrit en Python. Il est utilisé pour développer des applications web de toutes tailles et complexités. Flask facilite la création de routes, la gestion des requêtes HTTP et la génération de réponses dynamiques.

Voici un exemple d'utilisation de Flask pour créer une application web simple :

from flask import Flask, render_template

# Création de l'application Flask
app = Flask(__name__)

# Définition d'une route
@app.route("/")
def home():
    return "Hello, World!"

# Exécution de l'application
if __name__ == "__main__":
    app.run()

Dans cet exemple, nous importons la classe "Flask" de la bibliothèque Flask. Nous créons ensuite une instance de l'application Flask en utilisant le constructeur de classe. En utilisant le décorateur "@app.route", nous définissons une route pour l'URL racine ("/") de notre application. Lorsque cette route est accédée, la fonction "home" est exécutée et renvoie la chaîne de caractères "Hello, World!". Enfin, nous exécutons l'application en utilisant la méthode "run".

Remarque : Pour exécuter cette application Flask, vous devez installer Flask en utilisant la commande "pip install flask" dans votre environnement Python.
