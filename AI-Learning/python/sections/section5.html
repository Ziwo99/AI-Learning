Section 5: Les modules et les packages en Python

Introduction aux modules

Les modules en Python sont des fichiers contenant du code Python qui peuvent être importés et utilisés dans d'autres programmes Python. Ils permettent d'organiser le code en le regroupant par fonctionnalités ou par thèmes. L'utilisation de modules favorise la réutilisabilité du code et facilite la maintenance.

Un module peut contenir des variables, des fonctions, des classes et d'autres objets. Pour utiliser un module, il faut l'importer dans le programme principal à l'aide de l'instruction import. Par exemple :

import math

print(math.pi)

Dans cet exemple, le module math est importé et la valeur de la constante pi est affichée.

L'importation de modules

Pour importer un module, on utilise l'instruction import. Il existe plusieurs façons d'importer un module :

1. Importer tout le module :

import module_name

module_name.function_name()

2. Importer une fonction spécifique du module :

from module_name import function_name

function_name()

3. Importer tout le module avec un alias :

import module_name as alias

alias.function_name()

4. Importer toutes les fonctions du module :

from module_name import *

La création de modules

Pour créer un module, il suffit de créer un fichier Python avec l'extension .py et d'y écrire le code correspondant. Par exemple, pour créer un module nommé my_module :

# my_module.py

def my_function():
    print("Hello from my_module")

Pour utiliser ce module dans un autre programme, il faut l'importer :

import my_module

my_module.my_function()

Dans cet exemple, la fonction my_function du module my_module est appelée.

Les packages

Un package en Python est un dossier contenant des modules et d'autres packages. Les packages permettent d'organiser les modules en une structure hiérarchique. Ils sont utilisés pour créer des bibliothèques et des frameworks.

Pour créer un package, il suffit de créer un dossier et d'y placer les modules correspondants. Par exemple, pour créer un package nommé my_package :

my_package/
    __init__.py
    module1.py
    module2.py

Dans cet exemple, le fichier __init__.py est requis pour indiquer que le dossier est un package. Les modules module1.py et module2.py sont placés dans le package my_package.

Pour utiliser un module d'un package, il faut importer le package et le module correspondant :

import my_package.module1

my_package.module1.my_function()

Dans cet exemple, la fonction my_function du module module1 du package my_package est appelée.
