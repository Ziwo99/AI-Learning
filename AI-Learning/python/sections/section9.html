Section 9: Les modules avancés en Python

Le module math en Python

Le module math en Python fournit des fonctions mathématiques avancées pour effectuer des calculs complexes. Il contient des fonctions pour les opérations mathématiques de base, telles que les fonctions trigonométriques, exponentielles et logarithmiques.

Voici un exemple d'utilisation de quelques fonctions du module math :

import math

# Utilisation de la fonction trigonométrique sin()
angle = math.pi / 4
sin_value = math.sin(angle)
print("Le sinus de", angle, "est", sin_value)

# Utilisation de la fonction exponentielle exp()
x = 2
exp_value = math.exp(x)
print("L'exponentielle de", x, "est", exp_value)

# Utilisation de la fonction logarithmique log()
y = 10
log_value = math.log(y)
print("Le logarithme naturel de", y, "est", log_value)

Le module random en Python

Le module random en Python permet de générer des nombres aléatoires. Il offre des fonctions pour générer des nombres aléatoires entiers, des nombres aléatoires à virgule flottante et des choix aléatoires à partir d'une séquence donnée.

Voici un exemple d'utilisation de quelques fonctions du module random :

import random

# Génération d'un nombre aléatoire entier entre 1 et 10
random_int = random.randint(1, 10)
print("Nombre aléatoire entier :", random_int)

# Génération d'un nombre aléatoire à virgule flottante entre 0 et 1
random_float = random.random()
print("Nombre aléatoire à virgule flottante :", random_float)

# Choix aléatoire d'un élément dans une liste
my_list = [1, 2, 3, 4, 5]
random_choice = random.choice(my_list)
print("Choix aléatoire :", random_choice)

Le module datetime en Python

Le module datetime en Python permet de manipuler des dates et des heures. Il offre des fonctions pour créer, manipuler et formater des objets de date et d'heure.

Voici un exemple d'utilisation de quelques fonctions du module datetime :

import datetime

# Création d'un objet de date
date = datetime.date(2022, 1, 1)
print("Date :", date)

# Obtention de la date actuelle
current_date = datetime.date.today()
print("Date actuelle :", current_date)

# Création d'un objet d'heure
time = datetime.time(12, 30, 0)
print("Heure :", time)

# Obtention de l'heure actuelle
current_time = datetime.datetime.now().time()
print("Heure actuelle :", current_time)
