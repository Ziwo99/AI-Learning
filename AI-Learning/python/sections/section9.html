Section 9: Les modules avancés en Python

Les expressions régulières

Les expressions régulières sont des séquences de caractères utilisées pour rechercher et manipuler des motifs de texte. Elles sont extrêmement puissantes et flexibles pour effectuer des opérations de recherche et de remplacement dans des chaînes de caractères.

Voici un exemple simple d'utilisation des expressions régulières en Python :

import re

# Recherche d'un motif spécifique dans une chaîne de caractères
pattern = r"python"
text = "J'aime programmer en Python."
match = re.search(pattern, text)
if match:
    print("Motif trouvé !")
else:
    print("Motif non trouvé.")

Dans cet exemple, nous importons le module re qui fournit les fonctionnalités pour travailler avec les expressions régulières. Nous définissons ensuite un motif à rechercher dans la chaîne de caractères text. La fonction re.search() recherche le motif dans la chaîne de caractères et renvoie un objet de correspondance si le motif est trouvé.

Le traitement des fichiers CSV

Le format CSV (Comma-Separated Values) est couramment utilisé pour stocker des données tabulaires. Python offre des modules intégrés pour lire et écrire des fichiers CSV de manière efficace.

Voici un exemple d'utilisation du module CSV en Python :

import csv

# Lecture d'un fichier CSV
with open('donnees.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

# Écriture dans un fichier CSV
with open('resultats.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerow(['Nom', 'Âge'])
    writer.writerow(['Alice', 25])
    writer.writerow(['Bob', 30])

Dans cet exemple, nous utilisons le module csv pour lire un fichier CSV existant et afficher chaque ligne. Ensuite, nous utilisons le même module pour créer un nouveau fichier CSV et y écrire des données.

La manipulation de dates et d'heures

Python offre des modules intégrés pour manipuler facilement les dates et les heures. Ces modules fournissent des fonctionnalités pour effectuer des opérations telles que la conversion de formats de date, le calcul de différences de temps et la manipulation de composants de date individuels.

Voici un exemple d'utilisation du module datetime en Python :

from datetime import datetime

# Obtention de la date et de l'heure actuelles
now = datetime.now()
print("Date et heure actuelles :", now)

# Conversion d'une chaîne de caractères en objet de date
date_string = "2022-01-01"
date_object = datetime.strptime(date_string, "%Y-%m-%d")
print("Objet de date :", date_object)

# Calcul de la différence entre deux dates
date1 = datetime(2022, 1, 1)
date2 = datetime(2023, 1, 1)
difference = date2 - date1
print("Différence de temps :", difference)

Dans cet exemple, nous importons le module datetime et utilisons ses fonctionnalités pour obtenir la date et l'heure actuelles, convertir une chaîne de caractères en objet de date et calculer la différence entre deux dates.
