Section 9 : Les bases de données avec Python

Introduction aux bases de données

Les bases de données sont des outils essentiels pour stocker et organiser des données de manière structurée. Elles permettent de gérer efficacement de grandes quantités d'informations et de les interroger rapidement. Python offre de nombreuses bibliothèques pour interagir avec les bases de données, telles que SQLite, MySQL, PostgreSQL, etc.

Connexion à une base de données

Pour se connecter à une base de données, il est nécessaire d'installer la bibliothèque appropriée. Par exemple, pour se connecter à une base de données SQLite, vous pouvez utiliser la bibliothèque sqlite3 intégrée à Python.

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_donnees.db')

Manipulation des données

Une fois connecté à une base de données, vous pouvez effectuer diverses opérations telles que l'insertion, la mise à jour, la suppression et la récupération de données.

# Création d'une table
conn.execute('''CREATE TABLE employes
                (id INT PRIMARY KEY     NOT NULL,
                nom           TEXT    NOT NULL,
                age            INT     NOT NULL,
                salaire        REAL);''')

# Insertion de données
conn.execute("INSERT INTO employes (id, nom, age, salaire) VALUES (1, 'John Doe', 30, 5000.0)")

# Récupération de données
cursor = conn.execute("SELECT id, nom, age, salaire from employes")
for row in cursor:
    print("ID = ", row[0])
    print("Nom = ", row[1])
    print("Age = ", row[2])
    print("Salaire = ", row[3])

Gestion des erreurs lors de l'utilisation de bases de données

Lors de l'utilisation de bases de données, il est important de gérer les erreurs qui peuvent survenir. Cela permet de garantir la fiabilité et la robustesse de votre application.

try:
    # Tentative d'exécution d'une requête
    conn.execute("INSERT INTO employes (id, nom, age, salaire) VALUES (2, 'Jane Smith', 25, 4500.0)")
    conn.commit()
except sqlite3.Error as e:
    # Gestion de l'erreur
    print("Erreur lors de l'insertion des données :", e)

Assurez-vous de bien comprendre les concepts et les exemples présentés dans cette formation. Les bases de données sont un sujet vaste et complexe, mais une fois maîtrisées, elles peuvent être extrêmement utiles dans le développement de vos applications Python.
