Développement d'une application complète en Python

Dans cette section, nous allons mettre en pratique toutes les connaissances que vous avez acquises jusqu'à présent en développant une application complète en Python. Cette application sera un gestionnaire de tâches simple, qui vous permettra de créer, modifier et supprimer des tâches.

Pour commencer, nous allons définir les fonctionnalités de base de notre application :

  • Afficher la liste des tâches existantes
  • Ajouter une nouvelle tâche
  • Modifier une tâche existante
  • Supprimer une tâche

Pour implémenter ces fonctionnalités, nous allons utiliser les concepts que nous avons vus dans les sections précédentes, tels que les variables, les structures de contrôle, les fonctions et les classes.

Afficher la liste des tâches existantes

Pour afficher la liste des tâches existantes, nous allons utiliser une liste pour stocker les tâches. Chaque tâche sera représentée par un dictionnaire, avec les clés "titre" et "description". Voici un exemple de code :

tasks = [
    {"titre": "Faire les courses", "description": "Acheter du lait et du pain"},
    {"titre": "Appeler le médecin", "description": "Prendre rendez-vous pour le check-up annuel"}
]

def afficher_taches():
    for index, tache in enumerate(tasks):
        print(f"Tâche {index+1}: {tache['titre']}")
        print(f"Description: {tache['description']}")
        print()

Ce code utilise une boucle for pour parcourir la liste des tâches et affiche le titre et la description de chaque tâche. La fonction enumerate est utilisée pour obtenir à la fois l'index et la valeur de chaque tâche dans la boucle.

Ajouter une nouvelle tâche

Pour ajouter une nouvelle tâche, nous allons demander à l'utilisateur d'entrer le titre et la description de la tâche. Ensuite, nous allons créer un nouveau dictionnaire avec ces informations et l'ajouter à la liste des tâches. Voici un exemple de code :

def ajouter_tache():
    titre = input("Entrez le titre de la tâche : ")
    description = input("Entrez la description de la tâche : ")
    
    nouvelle_tache = {"titre": titre, "description": description}
    tasks.append(nouvelle_tache)
    
    print("La tâche a été ajoutée avec succès.")

Ce code utilise la fonction input pour demander à l'utilisateur d'entrer le titre et la description de la tâche. Ensuite, il crée un nouveau dictionnaire avec ces informations et l'ajoute à la liste des tâches à l'aide de la méthode append.

Modifier une tâche existante

Pour modifier une tâche existante, nous allons demander à l'utilisateur de sélectionner la tâche qu'il souhaite modifier en entrant son numéro. Ensuite, nous allons lui demander d'entrer le nouveau titre et la nouvelle description de la tâche. Voici un exemple de code :

def modifier_tache():
    afficher_taches()
    
    numero_tache = int(input("Entrez le numéro de la tâche à modifier : "))
    
    if numero_tache < 1 or numero_tache > len(tasks):
        print("Numéro de tâche invalide.")
        return
    
    nouvelle_tache = tasks[numero_tache-1]
    
    nouveau_titre = input("Entrez le nouveau titre de la tâche : ")
    nouvelle_description = input("Entrez la nouvelle description de la tâche : ")
    
    nouvelle_tache["titre"] = nouveau_titre
    nouvelle_tache["description"] = nouvelle_description
    
    print("La tâche a été modifiée avec succès.")

Ce code utilise la fonction afficher_taches pour afficher la liste des tâches existantes. Ensuite, il demande à l'utilisateur d'entrer le numéro de la tâche qu'il souhaite modifier. Si le numéro est invalide, un message d'erreur est affiché. Sinon, le code récupère la tâche correspondante dans la liste des tâches. Ensuite, il demande à l'utilisateur d'entrer le nouveau titre et la nouvelle description de la tâche, et met à jour les valeurs correspondantes dans le dictionnaire de la tâche.

Supprimer une tâche

Pour supprimer une tâche, nous allons demander à l'utilisateur de sélectionner la tâche qu'il souhaite supprimer en entrant son numéro. Ensuite, nous allons supprimer la tâche correspondante de la liste des tâches. Voici un exemple de code :

def supprimer_tache():
    afficher_taches()
    
    numero_tache = int(input("Entrez le numéro de la tâche à supprimer : "))
    
    if numero_tache < 1 or numero_tache > len(tasks):
        print("Numéro de tâche invalide.")
        return
    
    tache_supprimee = tasks.pop(numero_tache-1)
    
    print("La tâche suivante a été supprimée :")
    print(f"Titre: {tache_supprimee['titre']}")
    print(f"Description: {tache_supprimee['description']}")

Ce code utilise la fonction afficher_taches pour afficher la liste des tâches existantes. Ensuite, il demande à l'utilisateur d'entrer le numéro de la tâche qu'il souhaite supprimer. Si le numéro est invalide, un message d'erreur est affiché. Sinon, le code supprime la tâche correspondante de la liste des tâches à l'aide de la méthode pop. Enfin, il affiche les informations de la tâche supprimée.

Avec ces fonctionnalités de base, vous avez maintenant les connaissances nécessaires pour développer une application de gestion de tâches en Python. Vous pouvez bien sûr ajouter des fonctionnalités supplémentaires selon vos besoins et votre créativité.

Continuez à pratiquer et à explorer les différentes possibilités offertes par Python pour devenir un développeur Python compétent et créatif !
