Section 9: Les modules avancés en Python

Les expressions régulières

Les expressions régulières sont des séquences de caractères utilisées pour rechercher et manipuler des motifs de texte. En Python, nous utilisons le module re pour travailler avec les expressions régulières.

Voici un exemple simple qui montre comment rechercher un motif de texte spécifique :

import re

texte = "Bonjour, je m'appelle Alice. Je suis ravie de vous rencontrer."

motif = r"Alice"

resultat = re.search(motif, texte)

if resultat:
    print("Motif trouvé !")
else:
    print("Motif non trouvé.")

Ce code recherche le motif "Alice" dans la variable texte. Si le motif est trouvé, il affiche "Motif trouvé !", sinon il affiche "Motif non trouvé."

Le traitement des fichiers CSV

Le format CSV (Comma-Separated Values) est couramment utilisé pour stocker des données tabulaires. En Python, nous utilisons le module csv pour lire et écrire des fichiers CSV.

Voici un exemple qui montre comment lire un fichier CSV :

import csv

fichier_csv = open('donnees.csv', 'r')
lecteur_csv = csv.reader(fichier_csv)

for ligne in lecteur_csv:
    print(ligne)

fichier_csv.close()

Ce code ouvre le fichier CSV "donnees.csv" en mode lecture, puis utilise le lecteur CSV pour parcourir chaque ligne du fichier et l'afficher.

L'accès aux bases de données

Python offre de nombreuses bibliothèques pour interagir avec des bases de données. L'une des plus populaires est le module sqlite3, qui permet d'accéder à des bases de données SQLite.

Voici un exemple qui montre comment se connecter à une base de données SQLite et exécuter une requête :

import sqlite3

connexion = sqlite3.connect('ma_base_de_donnees.db')
curseur = connexion.cursor()

requete = "SELECT * FROM utilisateurs"
curseur.execute(requete)

resultat = curseur.fetchall()

for ligne in resultat:
    print(ligne)

connexion.close()

Ce code se connecte à la base de données "ma_base_de_donnees.db", exécute une requête pour sélectionner toutes les lignes de la table "utilisateurs", puis affiche les résultats.

La manipulation des dates et heures

Python offre le module datetime pour travailler avec les dates et heures. Ce module permet de créer, manipuler et formater des objets de date et d'heure.

Voici un exemple qui montre comment afficher la date et l'heure actuelles :

import datetime

maintenant = datetime.datetime.now()

print(maintenant)

Ce code utilise la fonction now() du module datetime pour obtenir la date et l'heure actuelles, puis les affiche.
