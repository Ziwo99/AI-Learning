Section 9 : Développement web avec Python

Introduction à Flask

Flask est un framework web léger et flexible pour Python. Il est conçu pour être simple à utiliser et permet de créer rapidement des applications web. Flask suit le principe "Keep It Simple, Stupid" (KISS) et ne nécessite pas de configuration complexe. Il est idéal pour les petites et moyennes applications web.

Pour commencer à utiliser Flask, vous devez d'abord l'installer. Vous pouvez le faire en utilisant la commande suivante dans votre terminal : 

pip install flask

Création de routes et de vues

Dans Flask, une route est une URL spécifique à laquelle votre application web répondra. Une vue est une fonction Python qui est exécutée lorsque la route correspondante est appelée. Les vues renvoient généralement une réponse HTML au client.

Voici un exemple de création d'une route et d'une vue dans Flask : 

from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run()

Dans cet exemple, nous avons créé une application Flask et défini une route '/' qui renvoie la chaîne de caractères 'Hello, World!' lorsque la route est appelée. Lorsque vous exécutez l'application, Flask démarre un serveur web local et écoute les requêtes sur le port par défaut (généralement le port 5000). Vous pouvez accéder à votre application en ouvrant votre navigateur et en visitant l'URL http://localhost:5000/.

Utilisation de templates

Les templates sont des fichiers HTML avec des balises spéciales qui permettent d'insérer des données dynamiques dans vos pages web. Flask utilise le moteur de templates Jinja2 par défaut, qui offre une syntaxe simple et puissante pour générer des pages web.

Voici un exemple d'utilisation de templates dans Flask : 

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    name = 'John Doe'
    return render_template('index.html', name=name)

if __name__ == '__main__':
    app.run()

Dans cet exemple, nous avons créé une route '/' qui appelle la vue 'index'. La vue utilise la fonction 'render_template' pour renvoyer la page 'index.html' en passant la variable 'name' à utiliser dans le template. Le template peut ensuite utiliser la variable 'name' pour afficher le nom dynamiquement.

Interaction avec une base de données

Flask facilite l'interaction avec une base de données dans vos applications web. Vous pouvez utiliser différents outils et bibliothèques pour cela, tels que SQLAlchemy, Flask-SQLAlchemy, ou encore Flask-MongoEngine pour les bases de données MongoDB.

Voici un exemple d'utilisation de Flask-SQLAlchemy pour interagir avec une base de données SQLite : 

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///mydatabase.db'
db = SQLAlchemy(app)

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(50))

@app.route('/')
def index():
    users = User.query.all()
    return render_template('index.html', users=users)

if __name__ == '__main__':
    app.run()

Dans cet exemple, nous avons créé une classe 'User' qui représente une table dans notre base de données. Nous avons également défini une route '/' qui récupère tous les utilisateurs de la base de données et les passe au template 'index.html'. Le template peut ensuite afficher les utilisateurs de manière dynamique.

Ceci conclut la section sur le développement web avec Python. Vous avez appris les bases de Flask, y compris la création de routes et de vues, l'utilisation de templates et l'interaction avec une base de données. Vous êtes maintenant prêt à créer vos propres applications web en utilisant Python et Flask.
