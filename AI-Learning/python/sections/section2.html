Les nombres et les opérations mathématiques

Les nombres sont des types de données couramment utilisés en programmation. En Python, nous avons différents types de nombres, tels que les entiers (int) et les nombres à virgule flottante (float).

Les opérations mathématiques de base peuvent être effectuées sur les nombres en utilisant les opérateurs arithmétiques suivants :

  • L'addition (+) : ajoute deux nombres
  • La soustraction (-) : soustrait un nombre d'un autre
  • La multiplication (*) : multiplie deux nombres
  • La division (/) : divise un nombre par un autre
  • Le modulo (%) : renvoie le reste de la division entre deux nombres
  • L'exponentiation (**) : élève un nombre à la puissance d'un autre

Voici un exemple d'utilisation des opérations mathématiques en Python : 

a = 5
b = 2

addition = a + b
soustraction = a - b
multiplication = a * b
division = a / b
modulo = a % b
exponentiation = a ** b

print("Résultats des opérations mathématiques :")
print("Addition :", addition)
print("Soustraction :", soustraction)
print("Multiplication :", multiplication)
print("Division :", division)
print("Modulo :", modulo)
print("Exponentiation :", exponentiation)

Ce code affichera les résultats des opérations mathématiques effectuées sur les nombres a et b.

Les chaînes de caractères

Les chaînes de caractères sont utilisées pour représenter du texte en Python. Elles sont définies en utilisant des guillemets simples ('') ou doubles ("").

Les opérations courantes sur les chaînes de caractères incluent la concaténation (combinaison de deux chaînes), la longueur d'une chaîne et l'accès aux caractères individuels d'une chaîne.

Voici un exemple d'utilisation des opérations sur les chaînes de caractères en Python : 

nom = "Alice"
age = 25

# Concaténation de chaînes de caractères
message = "Bonjour, je m'appelle " + nom + " et j'ai " + str(age) + " ans."

# Longueur d'une chaîne de caractères
longueur = len(message)

# Accès aux caractères individuels d'une chaîne
premier_caractere = message[0]
dernier_caractere = message[-1]

print("Message :", message)
print("Longueur :", longueur)
print("Premier caractère :", premier_caractere)
print("Dernier caractère :", dernier_caractere)

Ce code affichera le message, la longueur de la chaîne de caractères, le premier caractère et le dernier caractère de la chaîne.

Les listes, les tuples et les ensembles

Les listes, les tuples et les ensembles sont des structures de données utilisées pour stocker des collections d'éléments en Python.

Une liste est une collection ordonnée et modifiable d'éléments, définie entre crochets ([]). Les éléments d'une liste peuvent être de différents types de données.

Un tuple est une collection ordonnée et immuable d'éléments, définie entre parenthèses (()). Les éléments d'un tuple peuvent également être de différents types de données.

Un ensemble est une collection non ordonnée et non indexée d'éléments uniques, définie entre accolades ({}). Les éléments d'un ensemble doivent être de type immuable (par exemple, des nombres, des chaînes de caractères ou des tuples).

Voici un exemple d'utilisation des listes, des tuples et des ensembles en Python : 

# Liste
fruits = ["pomme", "banane", "orange"]
fruits.append("kiwi")  # Ajoute un élément à la liste
fruits.remove("banane")  # Supprime un élément de la liste

# Tuple
coordonnees = (3, 4)
x, y = coordonnees  # Décomposition d'un tuple

# Ensemble
nombres = {1, 2, 3, 4, 5}
nombres.add(6)  # Ajoute un élément à l'ensemble
nombres.remove(3)  # Supprime un élément de l'ensemble

print("Fruits :", fruits)
print("Coordonnées :", coordonnees)
print("Valeur de x :", x)
print("Valeur de y :", y)
print("Nombres :", nombres)

Ce code affichera les éléments de la liste des fruits, les coordonnées, la valeur de x et de y, ainsi que les éléments de l'ensemble des nombres.

Les dictionnaires

Un dictionnaire est une structure de données utilisée pour stocker des paires clé-valeur en Python. Chaque élément d'un dictionnaire est composé d'une clé et d'une valeur associée.

Les clés d'un dictionnaire doivent être uniques et immuables (par exemple, des nombres, des chaînes de caractères ou des tuples). Les valeurs peuvent être de différents types de données.

Voici un exemple d'utilisation des dictionnaires en Python : 

# Dictionnaire
personne = {
  "nom": "Alice",
  "age": 25,
  "ville": "Paris"
}

# Accès aux valeurs d'un dictionnaire
nom = personne["nom"]
age = personne.get("age")

# Modification des valeurs d'un dictionnaire
personne["age"] = 26
personne["profession"] = "Ingénieur"

# Suppression d'une paire clé-valeur d'un dictionnaire
del personne["ville"]

print("Nom :", nom)
print("Age :", age)
print("Personne :", personne)

Ce code affichera le nom, l'âge et les informations de la personne stockées dans le dictionnaire.

Les opérations sur les types de données

En Python, il existe des opérations spécifiques pour chaque type de données. Par exemple, pour les chaînes de caractères, nous avons des opérations telles que la recherche de sous-chaînes, le remplacement de caractères, la conversion en majuscules ou en minuscules, etc.

Il est important de connaître les opérations disponibles pour chaque type de données afin de pouvoir les manipuler efficacement.

Voici un exemple d'utilisation des opérations sur les types de données en Python : 

# Opérations sur les chaînes de caractères
message = "Bonjour, je m'appelle Alice."

# Recherche de sous-chaînes
if "Alice" in message:
  print("Alice est présent dans le message.")

# Remplacement de caractères
nouveau_message = message.replace("Alice", "Bob")

# Conversion en majuscules ou en minuscules
message_majuscules = message.upper()
message_minuscules = message.lower()

print("Nouveau message :", nouveau_message)
print("Message en majuscules :", message_majuscules)
print("Message en minuscules :", message_minuscules)

Ce code affichera le nouveau message après le remplacement de la sous-chaîne, le message en majuscules et le message en minuscules.
