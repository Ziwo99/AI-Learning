Les conditions et les instructions conditionnelles

Les conditions et les instructions conditionnelles permettent d'exécuter certaines parties du code uniquement si une condition spécifique est remplie. En Python, nous utilisons principalement l'instruction if pour cela.

Voici un exemple de code utilisant une instruction conditionnelle :

x = 10

if x > 0:
    print("x est positif")
elif x == 0:
    print("x est nul")
else:
    print("x est négatif")

Dans cet exemple, nous utilisons l'instruction if pour vérifier si la valeur de x est supérieure à 0. Si c'est le cas, nous affichons "x est positif". Si ce n'est pas le cas, nous utilisons l'instruction elif pour vérifier si x est égal à 0. Si c'est le cas, nous affichons "x est nul". Finalement, si aucune des conditions précédentes n'est remplie, nous utilisons l'instruction else pour afficher "x est négatif".

Les boucles for et while

Les boucles permettent d'exécuter une partie du code plusieurs fois. En Python, nous avons deux types de boucles couramment utilisées : la boucle for et la boucle while.

Voici un exemple de code utilisant une boucle for :

for i in range(5):
    print(i)

Dans cet exemple, nous utilisons la fonction range() pour générer une séquence de nombres de 0 à 4. La boucle for itère ensuite sur cette séquence et affiche chaque nombre.

Voici un exemple de code utilisant une boucle while :

i = 0

while i < 5:
    print(i)
    i += 1

Dans cet exemple, nous initialisons une variable i à 0. La boucle while s'exécute tant que la condition i < 5 est vraie. À chaque itération, nous affichons la valeur de i et incrémentons sa valeur de 1.

Les instructions de contrôle de boucle (break, continue)

En plus des boucles for et while, Python propose également des instructions de contrôle de boucle pour modifier le comportement de la boucle.

L'instruction break permet de sortir de la boucle prématurément :

for i in range(10):
    if i == 5:
        break
    print(i)

Dans cet exemple, la boucle for s'exécute jusqu'à ce que la valeur de i soit égale à 5. À ce moment-là, l'instruction break est exécutée et la boucle est interrompue.

L'instruction continue permet de passer à l'itération suivante de la boucle sans exécuter le reste du code :

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Dans cet exemple, la boucle for itère sur les nombres de 0 à 9. Si un nombre est pair (c'est-à-dire que le reste de la division par 2 est égal à 0), l'instruction continue est exécutée et le reste du code de la boucle n'est pas exécuté. Ainsi, seuls les nombres impairs sont affichés.
