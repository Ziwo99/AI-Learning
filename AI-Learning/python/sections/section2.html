Variables et types de données

En Python, les variables sont utilisées pour stocker des valeurs. Elles peuvent être de différents types de données, tels que les entiers, les flottants, les chaînes de caractères, les listes, les tuples, les dictionnaires, etc. Voici quelques exemples :

# Exemples de variables et de types de données en Python

# Entier
age = 25

# Flottant
prix = 9.99

# Chaîne de caractères
nom = "John Doe"

# Liste
fruits = ["pomme", "banane", "orange"]

# Tuple
coordonnees = (10, 20)

# Dictionnaire
personne = {"nom": "John", "age": 25}

Il est important de noter que Python est un langage à typage dynamique, ce qui signifie que vous n'avez pas besoin de déclarer explicitement le type d'une variable. Le type est déterminé automatiquement en fonction de la valeur assignée à la variable.

Opérations arithmétiques et opérateurs logiques

Python prend en charge les opérations arithmétiques de base telles que l'addition, la soustraction, la multiplication et la division. Voici quelques exemples :

# Exemples d'opérations arithmétiques en Python

a = 10
b = 5

# Addition
resultat = a + b
print(resultat)  # Affiche 15

# Soustraction
resultat = a - b
print(resultat)  # Affiche 5

# Multiplication
resultat = a * b
print(resultat)  # Affiche 50

# Division
resultat = a / b
print(resultat)  # Affiche 2.0

En plus des opérations arithmétiques, Python propose également des opérateurs logiques tels que "et" (and), "ou" (or) et "non" (not) pour effectuer des opérations de comparaison et de logique booléenne. Voici quelques exemples :

# Exemples d'opérateurs logiques en Python

a = 10
b = 5

# Opérateur "et" (and)
resultat = (a > 0) and (b < 10)
print(resultat)  # Affiche True

# Opérateur "ou" (or)
resultat = (a > 0) or (b > 10)
print(resultat)  # Affiche True

# Opérateur "non" (not)
resultat = not (a > 0)
print(resultat)  # Affiche False

Structures de contrôle (boucles et conditions)

Les structures de contrôle permettent d'exécuter des instructions conditionnellement ou de manière répétée. En Python, nous avons les structures de contrôle suivantes :

Conditions (if-elif-else)

La structure de contrôle conditionnelle "if-elif-else" permet d'exécuter des blocs de code en fonction de conditions spécifiques. Voici un exemple :

# Exemple de structure de contrôle conditionnelle en Python

age = 18

if age < 18:
    print("Vous êtes mineur.")
elif age == 18:
    print("Vous venez d'atteindre la majorité.")
else:
    print("Vous êtes majeur.")

Boucles (for et while)

Les boucles permettent d'exécuter un bloc de code de manière répétée. En Python, nous avons les boucles "for" et "while". Voici des exemples :

# Exemple de boucle "for" en Python

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)  # Affiche chaque fruit de la liste

# Exemple de boucle "while" en Python

i = 0

while i < 5:
    print(i)  # Affiche la valeur de i
    i += 1

Fonctions et modules

Les fonctions permettent de regrouper des instructions dans un bloc réutilisable. En Python, nous pouvons définir nos propres fonctions à l'aide du mot-clé "def". Voici un exemple :

# Exemple de définition de fonction en Python

def dire_bonjour(nom):
    print("Bonjour, " + nom + "!")

# Appel de la fonction
dire_bonjour("John")  # Affiche "Bonjour, John!"

Les modules sont des fichiers contenant des définitions de fonctions, de variables et d'autres objets, qui peuvent être importés et utilisés dans d'autres programmes Python. Voici un exemple :

# Exemple d'importation de module en Python

import math

# Utilisation de la fonction sqrt() du module math
resultat = math.sqrt(16)
print(resultat)  # Affiche 4.0

Il existe de nombreux modules Python disponibles qui offrent des fonctionnalités supplémentaires pour diverses tâches, tels que le calcul scientifique, la manipulation de données, la visualisation de données, etc.
