Les conditions et les boucles

Les conditions et les boucles sont des structures de contrôle essentielles en Python. Elles permettent d'exécuter certaines parties du code en fonction de conditions spécifiques ou de répéter des blocs de code plusieurs fois.

Les instructions conditionnelles (if, else, elif)

Les instructions conditionnelles permettent d'exécuter certaines parties du code en fonction de conditions spécifiques. En Python, les instructions conditionnelles sont réalisées avec les mots-clés if, else et elif.

x = 10

if x > 0:
    print("x est positif")
elif x < 0:
    print("x est négatif")
else:
    print("x est nul")

Les boucles (for, while)

Les boucles permettent de répéter des blocs de code plusieurs fois. En Python, il existe deux types de boucles : la boucle for et la boucle while.

# Boucle for
for i in range(5):
    print(i)

# Boucle while
i = 0
while i < 5:
    print(i)
    i += 1

Les instructions de contrôle (break, continue)

Les instructions de contrôle permettent de modifier le comportement des boucles. L'instruction break permet de sortir de la boucle prématurément, tandis que l'instruction continue permet de passer à l'itération suivante.

for i in range(10):
    if i == 5:
        break
    print(i)

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Les exceptions et la gestion des erreurs

Les exceptions sont des événements qui se produisent lorsqu'une erreur survient pendant l'exécution du code. En Python, il est possible de gérer ces exceptions à l'aide des blocs try et except.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Erreur : division par zéro")

Il est également possible de capturer des exceptions spécifiques et d'exécuter du code supplémentaire en utilisant les blocs except multiples.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Erreur : division par zéro")
except ValueError:
    print("Erreur : valeur incorrecte")

Ces structures de contrôle sont essentielles pour maîtriser la programmation en Python. Elles permettent de contrôler le flux d'exécution du code et de gérer les erreurs de manière efficace.
