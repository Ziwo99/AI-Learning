Section 2: Structures de contrôle

Les conditions : if, else, elif

Les conditions permettent d'exécuter certaines parties du code en fonction de l'évaluation d'une expression booléenne. En Python, on utilise les mots-clés if, else et elif pour gérer les conditions.

La structure générale d'une condition est la suivante :

if condition:
    # Code à exécuter si la condition est vraie
elif autre_condition:
    # Code à exécuter si la condition précédente est fausse et cette condition est vraie
else:
    # Code à exécuter si toutes les conditions précédentes sont fausses

Voici un exemple concret :

age = 18

if age < 18:
    print("Vous êtes mineur.")
elif age == 18:
    print("Vous venez d'atteindre la majorité.")
else:
    print("Vous êtes majeur.")

Dans cet exemple, le code affiche "Vous venez d'atteindre la majorité." car la variable age est égale à 18.

Les boucles : for, while

Les boucles permettent de répéter l'exécution d'un bloc de code plusieurs fois. En Python, on utilise les boucles for et while.

La boucle for permet de parcourir une séquence (comme une liste, un tuple ou une chaîne de caractères) et d'exécuter le code pour chaque élément de la séquence :

for element in sequence:
    # Code à exécuter pour chaque élément

Voici un exemple :

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

Ce code affiche chaque élément de la liste fruits sur une nouvelle ligne.

La boucle while permet d'exécuter le code tant qu'une condition est vraie :

while condition:
    # Code à exécuter tant que la condition est vraie

Voici un exemple :

i = 0

while i < 5:
    print(i)
    i += 1

Ce code affiche les nombres de 0 à 4, car la condition i < 5 est vérifiée à chaque itération.

Les instructions de contrôle : break, continue, pass

Les instructions de contrôle permettent de modifier le comportement des boucles.

L'instruction break permet de sortir immédiatement de la boucle :

for element in sequence:
    if condition:
        break
    # Code à exécuter pour chaque élément tant que la condition est fausse

L'instruction continue permet de passer à l'itération suivante de la boucle :

for element in sequence:
    if condition:
        continue
    # Code à exécuter pour chaque élément sauf ceux qui satisfont la condition

L'instruction pass est utilisée comme espace réservé pour un code qui n'a pas encore été écrit :

if condition:
    pass
else:
    # Code à exécuter si la condition est fausse

Ces instructions permettent de contrôler le flux d'exécution du code à l'intérieur des boucles.
