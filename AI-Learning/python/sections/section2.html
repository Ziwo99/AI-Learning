Section 2: Les types de données en Python

Les nombres

En Python, les nombres sont utilisés pour représenter des valeurs numériques. Il existe deux types de nombres en Python : les entiers (int) et les nombres à virgule flottante (float).

Les entiers sont des nombres entiers sans décimales. Par exemple :

x = 5
print(x)  # Affiche 5

Les nombres à virgule flottante sont des nombres avec une partie décimale. Par exemple :

y = 3.14
print(y)  # Affiche 3.14

Les chaînes de caractères

En Python, les chaînes de caractères sont utilisées pour représenter du texte. Elles sont entourées de guillemets simples ('') ou doubles (""). Par exemple :

message = "Bonjour, monde!"
print(message)  # Affiche Bonjour, monde!

Les chaînes de caractères peuvent être concaténées en utilisant l'opérateur +. Par exemple :

nom = "Alice"
age = 25
message = "Je m'appelle " + nom + " et j'ai " + str(age) + " ans."
print(message)  # Affiche Je m'appelle Alice et j'ai 25 ans.

Les listes

En Python, les listes sont utilisées pour stocker une collection ordonnée d'éléments. Les éléments d'une liste peuvent être de différents types. Par exemple :

fruits = ["pomme", "banane", "orange"]
print(fruits)  # Affiche ['pomme', 'banane', 'orange']

Les éléments d'une liste peuvent être accédés en utilisant leur indice. L'indice commence à 0 pour le premier élément. Par exemple :

fruit = fruits[0]
print(fruit)  # Affiche pomme

Les tuples

En Python, les tuples sont similaires aux listes, mais ils sont immuables, ce qui signifie qu'ils ne peuvent pas être modifiés une fois créés. Les tuples sont souvent utilisés pour stocker des collections d'éléments hétérogènes. Par exemple :

coordonnees = (3, 4)
print(coordonnees)  # Affiche (3, 4)

Les éléments d'un tuple peuvent être accédés de la même manière que ceux d'une liste, en utilisant leur indice. Par exemple :

x = coordonnees[0]
print(x)  # Affiche 3

Les dictionnaires

En Python, les dictionnaires sont utilisés pour stocker des paires clé-valeur. Chaque clé est associée à une valeur. Les dictionnaires sont utiles lorsque vous avez besoin d'accéder à une valeur en utilisant une clé plutôt qu'un indice. Par exemple :

personne = {"nom": "Alice", "age": 25, "ville": "Paris"}
print(personne)  # Affiche {'nom': 'Alice', 'age': 25, 'ville': 'Paris'}

Les valeurs d'un dictionnaire peuvent être accédées en utilisant leur clé. Par exemple :

nom = personne["nom"]
print(nom)  # Affiche Alice

Les ensembles

En Python, les ensembles sont utilisés pour stocker une collection non ordonnée d'éléments uniques. Les ensembles sont utiles lorsque vous avez besoin de vérifier rapidement si un élément est présent ou non. Par exemple :

fruits = {"pomme", "banane", "orange"}
print(fruits)  # Affiche {'pomme', 'banane', 'orange'}

Les ensembles ne permettent pas d'accéder à leurs éléments par un indice, car ils ne sont pas ordonnés. Cependant, vous pouvez vérifier si un élément est présent dans un ensemble en utilisant l'opérateur in. Par exemple :

if "pomme" in fruits:
    print("La pomme est dans l'ensemble.")
else:
    print("La pomme n'est pas dans l'ensemble.")
