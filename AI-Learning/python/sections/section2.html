Variables et types de données

En Python, les variables sont utilisées pour stocker des valeurs. Une variable peut contenir différents types de données tels que des nombres, des chaînes de caractères, des listes, etc. Pour déclarer une variable, utilisez le signe égal (=) suivi de la valeur que vous souhaitez lui attribuer.

Exemple :

age = 25
name = "John"
is_student = True

Python prend en charge différents types de données, tels que :

  • Entiers : utilisés pour stocker des nombres entiers, par exemple 10, -5, 0.
  • Flottants : utilisés pour stocker des nombres décimaux, par exemple 3.14, -2.5.
  • Chaînes de caractères : utilisées pour stocker du texte, par exemple "Bonjour", 'Python'.
  • Listes : utilisées pour stocker une collection ordonnée d'éléments, par exemple [1, 2, 3], ['a', 'b', 'c'].
  • Tuples : similaires aux listes, mais immuables (c'est-à-dire qu'ils ne peuvent pas être modifiés une fois créés), par exemple (1, 2, 3), ('a', 'b', 'c').
  • Dictionnaires : utilisés pour stocker des paires clé-valeur, par exemple {'nom': 'John', 'âge': 25}.
  • Ensembles : utilisés pour stocker une collection non ordonnée d'éléments uniques, par exemple {1, 2, 3}.
  • Booléens : utilisés pour représenter les valeurs de vérité (Vrai ou Faux).

Opérations arithmétiques et opérateurs logiques

Python prend en charge les opérations arithmétiques de base telles que l'addition (+), la soustraction (-), la multiplication (*), la division (/) et le modulo (%).

Exemple :

a = 10
b = 5

addition = a + b
soustraction = a - b
multiplication = a * b
division = a / b
modulo = a % b

print(addition)        # Résultat : 15
print(soustraction)    # Résultat : 5
print(multiplication)  # Résultat : 50
print(division)        # Résultat : 2.0
print(modulo)          # Résultat : 0

Python prend également en charge les opérateurs logiques tels que :

  • et : retourne Vrai si les deux conditions sont Vraies.
  • ou : retourne Vrai si au moins l'une des conditions est Vraie.
  • non : inverse la valeur de vérité d'une condition.

Exemple :

a = 10
b = 5
c = 7

resultat_et = (a > b) and (b < c)
resultat_ou = (a > b) or (b > c)
resultat_non = not (a > b)

print(resultat_et)  # Résultat : Vrai
print(resultat_ou)  # Résultat : Vrai
print(resultat_non) # Résultat : Faux

Structures de contrôle : conditions et boucles

Les structures de contrôle permettent d'exécuter certaines parties du code en fonction de conditions spécifiques ou de répéter des blocs de code plusieurs fois.

La structure de contrôle if permet d'exécuter un bloc de code si une condition est Vraie.

Exemple :

a = 10
b = 5

if a > b:
    print("a est supérieur à b")
else:
    print("a est inférieur ou égal à b")

La structure de contrôle while permet de répéter un bloc de code tant qu'une condition est Vraie.

Exemple :

i = 0

while i < 5:
    print(i)
    i += 1

La structure de contrôle for permet de répéter un bloc de code pour chaque élément d'une séquence (par exemple une liste).

Exemple :

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

Fonctions et modules

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser le code en le divisant en tâches plus petites et plus gérables.

Exemple :

def dire_bonjour(nom):
    print("Bonjour, " + nom + "!")

dire_bonjour("John")  # Résultat : Bonjour, John!

Les modules sont des fichiers contenant du code Python qui peuvent être importés dans d'autres programmes. Ils permettent d'organiser et de réutiliser du code.

Exemple :

import math

racine_carree = math.sqrt(16)
print(racine_carree)  # Résultat : 4.0

Vous pouvez également importer des fonctions spécifiques d'un module :

from math import sqrt

racine_carree = sqrt(16)
print(racine_carree)  # Résultat : 4.0

Il existe de nombreux modules Python prêts à l'emploi qui offrent des fonctionnalités supplémentaires, tels que le module random pour la génération de nombres aléatoires ou le module datetime pour la manipulation des dates et heures.
