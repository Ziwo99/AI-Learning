Les nombres en Python

En Python, les nombres sont utilisés pour représenter des valeurs numériques. Il existe différents types de nombres en Python :

  • Les entiers (int) : représentent les nombres entiers, par exemple 1, 2, -3.
  • Les nombres à virgule flottante (float) : représentent les nombres décimaux, par exemple 3.14, -0.5.
  • Les nombres complexes (complex) : représentent les nombres avec une partie réelle et une partie imaginaire, par exemple 2+3j, -1+2j.

Voici quelques exemples d'utilisation des nombres en Python :

a = 5
b = 2.5
c = 1 + 2j

print(a)  # Affiche 5
print(b)  # Affiche 2.5
print(c)  # Affiche (1+2j)

# Opérations arithmétiques
addition = a + b  # Addition : 5 + 2.5 = 7.5
soustraction = a - b  # Soustraction : 5 - 2.5 = 2.5
multiplication = a * b  # Multiplication : 5 * 2.5 = 12.5
division = a / b  # Division : 5 / 2.5 = 2.0
puissance = a ** 2  # Puissance : 5^2 = 25
modulo = a % 2  # Modulo : 5 % 2 = 1

print(addition)
print(soustraction)
print(multiplication)
print(division)
print(puissance)
print(modulo)

Les chaînes de caractères en Python

En Python, les chaînes de caractères sont utilisées pour représenter du texte. Une chaîne de caractères est définie en utilisant des guillemets simples ('') ou des guillemets doubles ("").

Voici quelques exemples d'utilisation des chaînes de caractères en Python :

message1 = 'Bonjour'
message2 = "monde"

# Concaténation de chaînes de caractères
message_complet = message1 + ' ' + message2  # 'Bonjour monde'

# Accès aux caractères individuels
premier_caractere = message1[0]  # 'B'
dernier_caractere = message2[-1]  # 'e'

# Longueur d'une chaîne de caractères
longueur = len(message_complet)  # 13

print(message_complet)
print(premier_caractere)
print(dernier_caractere)
print(longueur)

Les listes en Python

En Python, les listes sont utilisées pour stocker une collection ordonnée d'éléments. Les éléments d'une liste peuvent être de différents types (nombres, chaînes de caractères, etc.) et peuvent être modifiés une fois la liste créée.

Voici quelques exemples d'utilisation des listes en Python :

liste_vide = []
liste_nombres = [1, 2, 3, 4, 5]
liste_mixte = [1, 'deux', 3.0, [4, 5]]

# Accès aux éléments d'une liste
premier_element = liste_nombres[0]  # 1
dernier_element = liste_mixte[-1]  # [4, 5]

# Modification d'une liste
liste_nombres[0] = 10
liste_mixte.append(6)

# Longueur d'une liste
longueur = len(liste_nombres)  # 5

print(liste_nombres)
print(liste_mixte)
print(premier_element)
print(dernier_element)
print(longueur)

Les tuples en Python

En Python, les tuples sont similaires aux listes, mais ils sont immuables, ce qui signifie qu'ils ne peuvent pas être modifiés une fois créés. Les tuples sont souvent utilisés pour représenter des collections d'éléments qui ne doivent pas être modifiés.

Voici quelques exemples d'utilisation des tuples en Python :

tuple_vide = ()
tuple_nombres = (1, 2, 3, 4, 5)
tuple_mixte = (1, 'deux', 3.0, (4, 5))

# Accès aux éléments d'un tuple
premier_element = tuple_nombres[0]  # 1
dernier_element = tuple_mixte[-1]  # (4, 5)

# Longueur d'un tuple
longueur = len(tuple_nombres)  # 5

print(tuple_nombres)
print(tuple_mixte)
print(premier_element)
print(dernier_element)
print(longueur)

Les dictionnaires en Python

En Python, les dictionnaires sont utilisés pour stocker des paires clé-valeur. Chaque élément d'un dictionnaire est constitué d'une clé et d'une valeur associée. Les clés doivent être uniques et les valeurs peuvent être de différents types.

Voici quelques exemples d'utilisation des dictionnaires en Python :

dictionnaire_vide = {}
dictionnaire_fr_en = {'bonjour': 'hello', 'au revoir': 'goodbye', 'merci': 'thank you'}

# Accès aux valeurs d'un dictionnaire
traduction_bonjour = dictionnaire_fr_en['bonjour']  # 'hello'

# Modification d'un dictionnaire
dictionnaire_fr_en['merci'] = 'thanks'
dictionnaire_fr_en['s'il vous plaît'] = 'please'

# Suppression d'une paire clé-valeur
del dictionnaire_fr_en['au revoir']

# Longueur d'un dictionnaire
longueur = len(dictionnaire_fr_en)  # 3

print(dictionnaire_fr_en)
print(traduction_bonjour)
print(longueur)

Les ensembles en Python

En Python, les ensembles sont utilisés pour stocker une collection non ordonnée d'éléments uniques. Les ensembles sont utiles lorsque l'ordre des éléments n'a pas d'importance et lorsque chaque élément doit être unique.

Voici quelques exemples d'utilisation des ensembles en Python :

ensemble_vide = set()
ensemble_nombres = {1, 2, 3, 4, 5}
ensemble_mixte = {1, 'deux', 3.0}

# Ajout d'éléments à un ensemble
ensemble_nombres.add(6)

# Suppression d'un élément d'un ensemble
ensemble_nombres.remove(3)

# Opérations ensemblistes
ensemble_pair = {2, 4, 6, 8, 10}
ensemble_impair = {1, 3, 5, 7, 9}

ensemble_union = ensemble_pair.union(ensemble_impair)
ensemble_intersection = ensemble_pair.intersection(ensemble_impair)
ensemble_difference = ensemble_pair.difference(ensemble_impair)

print(ensemble_nombres)
print(ensemble_mixte)
print(ensemble_union)
print(ensemble_intersection)
print(ensemble_difference)
