Section 2: Les types de données en Python

Les nombres

En Python, les nombres sont utilisés pour représenter des valeurs numériques. Il existe deux types de nombres en Python : les entiers (int) et les nombres à virgule flottante (float).

Les entiers sont des nombres entiers sans partie décimale. Par exemple :

x = 5
y = -10
z = 0

Les nombres à virgule flottante sont des nombres avec une partie décimale. Par exemple :

a = 3.14
b = -2.5
c = 0.0

Python prend également en charge les opérations mathématiques de base sur les nombres, telles que l'addition, la soustraction, la multiplication et la division.

x = 5 + 3  # x vaut 8
y = 10 - 2  # y vaut 8
z = 3 * 4  # z vaut 12
w = 10 / 2  # w vaut 5.0 (division avec un nombre à virgule flottante)

Les chaînes de caractères

En Python, les chaînes de caractères sont utilisées pour représenter du texte. Elles sont entourées de guillemets simples ('') ou doubles (""). Par exemple :

message = 'Bonjour'
name = "Alice"

Les chaînes de caractères peuvent être concaténées en utilisant l'opérateur +. Par exemple :

greeting = message + ' ' + name  # greeting vaut 'Bonjour Alice'

Python fournit également de nombreuses méthodes pour manipuler les chaînes de caractères, telles que la conversion en majuscules ou en minuscules, la recherche de sous-chaînes, le remplacement de texte, etc.

text = 'Bonjour tout le monde'
uppercase_text = text.upper()  # uppercase_text vaut 'BONJOUR TOUT LE MONDE'
lowercase_text = text.lower()  # lowercase_text vaut 'bonjour tout le monde'
substring_index = text.find('tout')  # substring_index vaut 8
replaced_text = text.replace('monde', 'ami')  # replaced_text vaut 'Bonjour tout le ami'

Les listes

En Python, les listes sont utilisées pour stocker une collection ordonnée d'éléments. Les éléments d'une liste peuvent être de différents types de données. Par exemple :

numbers = [1, 2, 3, 4, 5]
names = ['Alice', 'Bob', 'Charlie']
mixed_list = [1, 'Alice', 3.14, True]

Les éléments d'une liste peuvent être accédés en utilisant leur indice (commençant à 0). Par exemple :

first_number = numbers[0]  # first_number vaut 1
second_name = names[1]  # second_name vaut 'Bob'

Python fournit également de nombreuses méthodes pour manipuler les listes, telles que l'ajout d'éléments, la suppression d'éléments, le tri, etc.

numbers.append(6)  # Ajoute 6 à la fin de la liste numbers
names.remove('Charlie')  # Supprime 'Charlie' de la liste names
sorted_numbers = sorted(numbers)  # Tri les nombres de la liste numbers dans l'ordre croissant

Les tuples

En Python, les tuples sont similaires aux listes, mais ils sont immuables, ce qui signifie qu'ils ne peuvent pas être modifiés une fois créés. Les tuples sont souvent utilisés pour stocker des collections d'éléments hétérogènes.

person = ('Alice', 25, 'Femme')
coordinates = (3.14, -2.5)

Les éléments d'un tuple peuvent être accédés de la même manière que ceux d'une liste, en utilisant leur indice.

name = person[0]  # name vaut 'Alice'
y_coordinate = coordinates[1]  # y_coordinate vaut -2.5

Comme les tuples sont immuables, ils ne peuvent pas être modifiés. Cependant, vous pouvez créer de nouveaux tuples en combinant des tuples existants.

new_person = person + ('Paris',)  # new_person vaut ('Alice', 25, 'Femme', 'Paris')

Les dictionnaires

En Python, les dictionnaires sont utilisés pour stocker des paires clé-valeur. Chaque élément d'un dictionnaire est composé d'une clé et d'une valeur associée. Les clés doivent être uniques.

person = {'name': 'Alice', 'age': 25, 'gender': 'Femme'}
coordinates = {'x': 3.14, 'y': -2.5}

Les valeurs d'un dictionnaire peuvent être accédées en utilisant leurs clés.

name = person['name']  # name vaut 'Alice'
y_coordinate = coordinates['y']  # y_coordinate vaut -2.5

Vous pouvez également ajouter de nouvelles paires clé-valeur à un dictionnaire ou modifier les valeurs existantes.

person['city'] = 'Paris'  # Ajoute la paire clé-valeur {'city': 'Paris'} au dictionnaire person
person['age'] = 26  # Modifie la valeur associée à la clé 'age' dans le dictionnaire person

Les ensembles

En Python, les ensembles sont utilisés pour stocker une collection non ordonnée d'éléments uniques. Les ensembles ne peuvent pas contenir de doublons.

fruits = {'pomme', 'banane', 'orange'}

Les ensembles sont souvent utilisés pour effectuer des opérations ensemblistes telles que l'union, l'intersection et la différence.

vegetables = {'carotte', 'brocoli', 'pomme'}
all_food = fruits.union(vegetables)  # all_food contient tous les éléments de fruits et de vegetables, sans doublons
common_food = fruits.intersection(vegetables)  # common_food contient les éléments communs à fruits et vegetables
unique_fruits = fruits.difference(vegetables)  # unique_fruits contient les éléments de fruits qui ne sont pas dans vegetables
