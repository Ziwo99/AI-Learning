Section 2: Les bases de Python

Variables et types de données

Les variables sont utilisées pour stocker des valeurs dans la mémoire de l'ordinateur. En Python, vous pouvez créer une variable en lui attribuant une valeur à l'aide du signe égal (=). Python est un langage à typage dynamique, ce qui signifie que vous n'avez pas besoin de déclarer le type de variable avant de l'utiliser.

Voici un exemple de création de variables en Python :

age = 25
name = "John"
is_student = True

Python prend en charge différents types de données, tels que les entiers (int), les nombres à virgule flottante (float), les chaînes de caractères (str) et les booléens (bool).

Voici quelques exemples de types de données en Python :

age = 25  # entier
weight = 65.5  # nombre à virgule flottante
name = "John"  # chaîne de caractères
is_student = True  # booléen

Opérations arithmétiques

Python prend en charge les opérations arithmétiques de base, telles que l'addition (+), la soustraction (-), la multiplication (*), la division (/) et le modulo (%).

Voici quelques exemples d'opérations arithmétiques en Python :

a = 10
b = 5

addition = a + b  # 15
subtraction = a - b  # 5
multiplication = a * b  # 50
division = a / b  # 2.0
modulo = a % b  # 0

Structures de contrôle (boucles et conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. En Python, vous pouvez utiliser des boucles (comme la boucle while et la boucle for) et des conditions (comme l'instruction if) pour contrôler le flux d'exécution.

Voici un exemple d'utilisation de la boucle while en Python :

i = 0

while i < 10:
    print("Iteration " + str(i))
    i += 1

Voici un exemple d'utilisation de l'instruction if en Python :

age = 25

if age >= 18:
    print("Vous êtes majeur.")
else:
    print("Vous êtes mineur.")

Fonctions et modules

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. En Python, vous pouvez créer vos propres fonctions en utilisant le mot-clé def.

Voici un exemple de création d'une fonction en Python :

def say_hello():
    print("Hello!")

say_hello()  # Appel de la fonction

Les modules sont des fichiers contenant des fonctions, des classes et d'autres éléments de code réutilisables. En Python, vous pouvez importer des modules pour utiliser leurs fonctionnalités.

Voici un exemple d'importation d'un module en Python :

import math

print(math.sqrt(16))  # Utilisation de la fonction sqrt du module math
