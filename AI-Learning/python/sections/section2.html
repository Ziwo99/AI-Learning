Section 2 : Les bases de Python

Variables et types de données

En Python, les variables sont utilisées pour stocker des valeurs. Elles peuvent être de différents types de données, tels que les entiers, les flottants, les chaînes de caractères, les booléens, etc. Voici quelques exemples :

a = 5  # entier
b = 3.14  # flottant
c = "Bonjour"  # chaîne de caractères
d = True  # booléen

Il est important de noter que Python est un langage à typage dynamique, ce qui signifie que le type d'une variable peut changer au cours de l'exécution du programme.

Opérations arithmétiques et opérateurs logiques

Python prend en charge les opérations arithmétiques de base, telles que l'addition, la soustraction, la multiplication et la division. Voici quelques exemples :

a = 5
b = 3

addition = a + b  # 8
soustraction = a - b  # 2
multiplication = a * b  # 15
division = a / b  # 1.6666666666666667

Python prend également en charge les opérateurs logiques, tels que "et", "ou" et "non". Voici quelques exemples :

a = True
b = False

et_logique = a and b  # False
ou_logique = a or b  # True
non_logique = not a  # False

Structures de contrôle (boucles et conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. En Python, nous avons les boucles et les conditions.

Les boucles permettent de répéter une série d'instructions plusieurs fois. Voici un exemple de boucle "while" :

i = 0

while i < 10:
    print(i)
    i += 1

Les conditions permettent d'exécuter certaines instructions uniquement si une condition est remplie. Voici un exemple de condition "if" :

a = 5

if a > 0:
    print("a est positif")
elif a == 0:
    print("a est nul")
else:
    print("a est négatif")

Fonctions et modules

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. En Python, nous pouvons définir nos propres fonctions. Voici un exemple :

def addition(a, b):
    return a + b

resultat = addition(5, 3)  # 8

Les modules sont des fichiers contenant des fonctions et des variables qui peuvent être importés dans d'autres fichiers Python. Voici un exemple :

import math

racine_carree = math.sqrt(16)  # 4.0

Il existe de nombreux modules intégrés à Python, tels que "math", "random", etc., qui offrent des fonctionnalités supplémentaires.
