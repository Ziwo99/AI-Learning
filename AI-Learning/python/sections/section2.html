Variables et types de données

En Python, les variables sont utilisées pour stocker des valeurs. Une variable peut contenir différents types de données, tels que des nombres, des chaînes de caractères, des listes, etc. Pour déclarer une variable, utilisez le signe égal (=) et attribuez-lui une valeur.

Exemple :

age = 25
name = "John"
is_student = True

Python prend en charge différents types de données, notamment :

  • Nombres : entiers (int), nombres à virgule flottante (float), nombres complexes (complex)
  • Chaînes de caractères : séquences de caractères entourées de guillemets simples ('') ou doubles ("")
  • Listes : collections ordonnées et modifiables d'éléments
  • Tuples : collections ordonnées et immuables d'éléments
  • Dictionnaires : collections non ordonnées de paires clé-valeur
  • Ensembles : collections non ordonnées d'éléments uniques

Opérations arithmétiques et opérateurs logiques

Python prend en charge les opérations arithmétiques de base, telles que l'addition (+), la soustraction (-), la multiplication (*), la division (/) et le modulo (%). Vous pouvez également utiliser des parenthèses pour grouper les opérations.

Exemple :

a = 10
b = 5

addition = a + b
soustraction = a - b
multiplication = a * b
division = a / b
modulo = a % b

print(addition)        # Affiche 15
print(soustraction)    # Affiche 5
print(multiplication)  # Affiche 50
print(division)        # Affiche 2.0
print(modulo)          # Affiche 0

Les opérateurs logiques tels que et (and), ou (or) et non (not) sont utilisés pour évaluer des conditions.

Exemple :

a = 10
b = 5
c = 7

# Utilisation de l'opérateur et
if a > b and a > c:
    print("a est le plus grand nombre")

# Utilisation de l'opérateur ou
if b > a or b > c:
    print("b est plus grand que soit a, soit c")

# Utilisation de l'opérateur non
if not a == b:
    print("a n'est pas égal à b")

Structures de contrôle : conditions et boucles

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. En Python, les structures de contrôle les plus couramment utilisées sont les conditions (if-else) et les boucles (for et while).

Exemple de condition :

age = 18

if age >= 18:
    print("Vous êtes majeur")
else:
    print("Vous êtes mineur")

Exemple de boucle for :

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

Exemple de boucle while :

i = 0

while i < 5:
    print(i)
    i += 1

Fonctions et modules

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. En Python, vous pouvez définir vos propres fonctions en utilisant le mot-clé def.

Exemple :

def say_hello():
    print("Bonjour")

say_hello()  # Appel de la fonction say_hello

Les modules sont des fichiers contenant du code Python qui peuvent être importés dans d'autres programmes. Ils permettent d'organiser et de réutiliser du code.

Exemple :

import math

print(math.sqrt(16))  # Utilisation de la fonction sqrt du module math

Vous pouvez également créer vos propres modules personnalisés en créant un fichier Python avec du code réutilisable.
