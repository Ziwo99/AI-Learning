Syntaxe de base

La syntaxe de base de Python est simple et facile à comprendre. Voici quelques éléments clés de la syntaxe :

print("Hello, World!")  # Affiche "Hello, World!" à l'écran

x = 5  # Déclaration d'une variable x avec la valeur 5

if x > 0:  # Structure de contrôle if-else
    print("x est positif")
else:
    print("x est négatif ou nul")

Variables et types de données

Python est un langage à typage dynamique, ce qui signifie que les variables n'ont pas besoin d'être déclarées avec un type spécifique. Voici quelques types de données couramment utilisés en Python :

x = 5  # Entier (int)
y = 3.14  # Flottant (float)
z = "Hello"  # Chaîne de caractères (str)
is_true = True  # Booléen (bool)

Opérations arithmétiques

Python prend en charge les opérations arithmétiques de base :

x = 5
y = 3

addition = x + y  # Addition
soustraction = x - y  # Soustraction
multiplication = x * y  # Multiplication
division = x / y  # Division
modulo = x % y  # Modulo (reste de la division)
exposant = x ** y  # Exposant (élévation à la puissance)

Structures de contrôle (boucles et conditions)

Python propose plusieurs structures de contrôle pour exécuter des instructions conditionnelles et répétitives :

# Structure if-else
x = 5

if x > 0:
    print("x est positif")
else:
    print("x est négatif ou nul")

# Boucle while
i = 0
while i < 5:
    print(i)
    i += 1

# Boucle for
for i in range(5):
    print(i)

Fonctions et modules

Les fonctions permettent d'organiser le code en blocs réutilisables. Les modules sont des fichiers contenant des fonctions et des variables qui peuvent être importés dans d'autres programmes :

# Définition d'une fonction
def say_hello():
    print("Hello, World!")

# Appel d'une fonction
say_hello()

# Importation d'un module
import math

# Utilisation d'une fonction d'un module
x = math.sqrt(25)
