Section 2: Les types de données en Python

Section 2: Les types de données en Python

Les nombres en Python

En Python, les nombres sont utilisés pour représenter des valeurs numériques. Il existe deux types de nombres en Python : les entiers (int) et les nombres à virgule flottante (float).

Les entiers sont des nombres entiers sans décimales. Par exemple :

x = 5
y = -10
z = 0

Les nombres à virgule flottante sont des nombres avec une partie décimale. Par exemple :

a = 3.14
b = -2.5
c = 0.0

Python prend en charge les opérations mathématiques de base sur les nombres, telles que l'addition, la soustraction, la multiplication et la division. Par exemple :

addition = 5 + 3
soustraction = 10 - 2
multiplication = 4 * 2
division = 10 / 3

Les chaînes de caractères en Python

En Python, les chaînes de caractères sont utilisées pour représenter du texte. Elles sont entourées de guillemets simples ('') ou doubles (""). Par exemple :

message = 'Bonjour'
nom = "Alice"

Les chaînes de caractères peuvent être concaténées en utilisant l'opérateur +. Par exemple :

message_complet = message + ' ' + nom

Python prend en charge de nombreuses opérations sur les chaînes de caractères, telles que la longueur d'une chaîne, l'accès à des caractères spécifiques et le découpage de chaînes. Par exemple :

longueur = len(message)
premier_caractere = message[0]
derniers_caracteres = message[1:]

Les listes en Python

En Python, les listes sont utilisées pour stocker une collection ordonnée d'éléments. Les éléments d'une liste peuvent être de différents types de données. Par exemple :

ma_liste = [1, 2, 3, 'quatre', 'cinq']

Les listes peuvent être modifiées en ajoutant, supprimant ou modifiant des éléments. Par exemple :

ma_liste.append(6)
ma_liste.remove('quatre')
ma_liste[0] = 'un'

Python fournit de nombreuses méthodes pour manipuler les listes, telles que le tri, la recherche d'éléments et le découpage de listes. Par exemple :

ma_liste.sort()
index = ma_liste.index('cinq')
sous_liste = ma_liste[2:4]

Les tuples en Python

En Python, les tuples sont similaires aux listes, mais ils sont immuables, ce qui signifie qu'ils ne peuvent pas être modifiés une fois créés. Les tuples sont souvent utilisés pour stocker des collections d'éléments hétérogènes. Par exemple :

mon_tuple = (1, 'deux', 3.0)

Les tuples peuvent être utilisés de la même manière que les listes pour accéder à des éléments spécifiques. Par exemple :

premier_element = mon_tuple[0]
sous_tuple = mon_tuple[1:]

Les dictionnaires en Python

En Python, les dictionnaires sont utilisés pour stocker des paires clé-valeur. Chaque élément d'un dictionnaire est composé d'une clé et d'une valeur associée. Par exemple :

mon_dictionnaire = {'nom': 'Alice', 'age': 25, 'ville': 'Paris'}

Les dictionnaires permettent d'accéder rapidement à une valeur en utilisant sa clé. Par exemple :

age = mon_dictionnaire['age']

Les dictionnaires peuvent être modifiés en ajoutant, supprimant ou modifiant des paires clé-valeur. Par exemple :

mon_dictionnaire['profession'] = 'ingénieur'
del mon_dictionnaire['ville']

Les ensembles en Python

En Python, les ensembles sont utilisés pour stocker une collection non ordonnée d'éléments uniques. Les ensembles ne peuvent pas contenir de doublons. Par exemple :

mon_ensemble = {1, 2, 3, 4, 5}

Les ensembles peuvent être modifiés en ajoutant ou en supprimant des éléments. Par exemple :

mon_ensemble.add(6)
mon_ensemble.remove(3)

Les ensembles peuvent être utilisés pour effectuer des opérations mathématiques telles que l'union, l'intersection et la différence. Par exemple :

ensemble1 = {1, 2, 3}
ensemble2 = {3, 4, 5}
union = ensemble1.union(ensemble2)
intersection = ensemble1.intersection(ensemble2)
difference = ensemble1.difference(ensemble2)
