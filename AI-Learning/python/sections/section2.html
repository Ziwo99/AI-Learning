Les listes

Une liste est une structure de données qui permet de stocker une collection ordonnée d'éléments. Les éléments d'une liste peuvent être de différents types tels que des entiers, des chaînes de caractères, des booléens, etc. Une liste est définie en utilisant des crochets [] et les éléments sont séparés par des virgules.

Exemple :

ma_liste = [1, 2, 3, 4, 5]
print(ma_liste)

Résultat :

[1, 2, 3, 4, 5]

Les listes sont mutables, ce qui signifie que vous pouvez modifier leurs éléments en utilisant leur indice. L'indice d'un élément dans une liste commence à 0.

Exemple :

ma_liste = [1, 2, 3, 4, 5]
ma_liste[0] = 10
print(ma_liste)

Résultat :

[10, 2, 3, 4, 5]

Les tuples

Un tuple est une structure de données similaire à une liste, mais contrairement à une liste, un tuple est immuable, ce qui signifie que vous ne pouvez pas modifier ses éléments une fois qu'ils sont définis. Un tuple est défini en utilisant des parenthèses () et les éléments sont séparés par des virgules.

Exemple :

mon_tuple = (1, 2, 3, 4, 5)
print(mon_tuple)

Résultat :

(1, 2, 3, 4, 5)

Comme les listes, les tuples peuvent contenir des éléments de différents types.

Les dictionnaires

Un dictionnaire est une structure de données qui permet de stocker des paires clé-valeur. Chaque élément d'un dictionnaire est composé d'une clé et d'une valeur associée. Les clés d'un dictionnaire doivent être uniques et immuables, tandis que les valeurs peuvent être de n'importe quel type.

Exemple :

mon_dictionnaire = {"nom": "John", "age": 25, "ville": "Paris"}
print(mon_dictionnaire)

Résultat :

{"nom": "John", "age": 25, "ville": "Paris"}

Vous pouvez accéder aux valeurs d'un dictionnaire en utilisant leurs clés.

Exemple :

mon_dictionnaire = {"nom": "John", "age": 25, "ville": "Paris"}
print(mon_dictionnaire["nom"])

Résultat :

John

Les ensembles

Un ensemble est une structure de données qui permet de stocker une collection non ordonnée d'éléments uniques. Les ensembles sont utiles lorsque vous avez besoin de vérifier rapidement si un élément est présent ou non, car ils offrent une recherche très efficace. Les ensembles sont définis en utilisant des accolades {}.

Exemple :

mon_ensemble = {1, 2, 3, 4, 5}
print(mon_ensemble)

Résultat :

{1, 2, 3, 4, 5}

Les ensembles ne permettent pas d'avoir des éléments en double.

Les chaînes de caractères

Une chaîne de caractères est une séquence de caractères. Les chaînes de caractères sont immuables, ce qui signifie que vous ne pouvez pas modifier leurs caractères une fois qu'ils sont définis. Les chaînes de caractères sont définies en utilisant des guillemets simples '' ou des guillemets doubles "".

Exemple :

ma_chaine = "Bonjour"
print(ma_chaine)

Résultat :

Bonjour

Vous pouvez accéder aux caractères d'une chaîne de caractères en utilisant leurs indices.

Exemple :

ma_chaine = "Bonjour"
print(ma_chaine[0])

Résultat :

B

Les fichiers et les entrées/sorties

Python offre des fonctionnalités intégrées pour travailler avec des fichiers. Vous pouvez ouvrir un fichier en utilisant la fonction open(), spécifier le mode d'ouverture (lecture, écriture, ajout) et effectuer des opérations sur le fichier telles que la lecture, l'écriture et la fermeture.

Exemple :

fichier = open("mon_fichier.txt", "w")
fichier.write("Contenu du fichier")
fichier.close()

Résultat :

Un fichier "mon_fichier.txt" est créé avec le contenu "Contenu du fichier".

Pour lire le contenu d'un fichier, vous pouvez utiliser la méthode read().

Exemple :

fichier = open("mon_fichier.txt", "r")
contenu = fichier.read()
print(contenu)
fichier.close()

Résultat :

Contenu du fichier
