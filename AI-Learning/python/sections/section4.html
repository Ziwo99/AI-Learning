Section 4 : Manipulation de fichiers et gestion des erreurs

Lecture et écriture de fichiers

La manipulation de fichiers est une tâche courante en programmation, et Python offre des fonctionnalités puissantes pour lire et écrire des fichiers. Pour ouvrir un fichier en lecture, vous pouvez utiliser la fonction open() en spécifiant le nom du fichier et le mode d'ouverture. Par exemple, pour ouvrir un fichier texte en lecture :

file = open("mon_fichier.txt", "r")

Une fois le fichier ouvert, vous pouvez utiliser différentes méthodes pour lire son contenu. La méthode read() permet de lire tout le contenu du fichier en une seule fois :

content = file.read()
print(content)

Si vous souhaitez lire le contenu ligne par ligne, vous pouvez utiliser la méthode readline() :

line = file.readline()
while line:
    print(line)
    line = file.readline()

Pour écrire dans un fichier, vous pouvez ouvrir le fichier en mode écriture ("w") ou en mode ajout ("a"). Par exemple, pour écrire dans un fichier texte :

file = open("mon_fichier.txt", "w")
file.write("Contenu à écrire dans le fichier")
file.close()

Il est important de fermer le fichier après avoir terminé de le lire ou de l'écrire en utilisant la méthode close().

Gestion des exceptions

La gestion des exceptions est une pratique essentielle en programmation pour gérer les erreurs et les situations imprévues. En Python, vous pouvez utiliser des blocs try et except pour capturer et gérer les exceptions.

Voici un exemple de bloc try et except :

try:
    # Code susceptible de générer une exception
    file = open("mon_fichier.txt", "r")
    content = file.read()
    print(content)
    file.close()
except FileNotFoundError:
    # Code à exécuter si une exception FileNotFoundError est levée
    print("Le fichier n'a pas été trouvé.")
except Exception as e:
    # Code à exécuter pour toutes les autres exceptions
    print("Une erreur s'est produite :", str(e))

Dans cet exemple, si le fichier "mon_fichier.txt" n'est pas trouvé, une exception de type FileNotFoundError est levée et le code dans le bloc except FileNotFoundError est exécuté. Si une autre exception est levée, le code dans le bloc except Exception as e est exécuté, avec la description de l'exception stockée dans la variable e.

Il est également possible d'utiliser un bloc finally pour exécuter du code qui doit toujours être exécuté, qu'une exception soit levée ou non :

try:
    # Code susceptible de générer une exception
    file = open("mon_fichier.txt", "r")
    content = file.read()
    print(content)
except FileNotFoundError:
    # Code à exécuter si une exception FileNotFoundError est levée
    print("Le fichier n'a pas été trouvé.")
except Exception as e:
    # Code à exécuter pour toutes les autres exceptions
    print("Une erreur s'est produite :", str(e))
finally:
    # Code à exécuter toujours
    file.close()

Cela garantit que le fichier est toujours fermé, même en cas d'exception.
