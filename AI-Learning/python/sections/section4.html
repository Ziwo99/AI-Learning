Les conditions en Python

Les conditions en Python permettent d'exécuter certaines parties du code uniquement si une condition spécifique est remplie. Les conditions sont généralement basées sur des expressions booléennes, qui peuvent être soit vraies (True) soit fausses (False).

La structure de base pour une condition en Python est la suivante :

if condition:
    # code à exécuter si la condition est vraie
else:
    # code à exécuter si la condition est fausse

Il est également possible d'ajouter des conditions supplémentaires en utilisant les mots-clés "elif" (contraction de "else if") :

if condition1:
    # code à exécuter si la condition1 est vraie
elif condition2:
    # code à exécuter si la condition2 est vraie
else:
    # code à exécuter si aucune des conditions précédentes n'est vraie

Les conditions peuvent être basées sur des opérateurs de comparaison tels que "==", "!=", "<", ">", "<=", ">=", ainsi que sur des opérateurs logiques tels que "and", "or" et "not". Par exemple :

x = 5

if x > 0 and x < 10:
    print("x est un nombre positif à un seul chiffre")
elif x >= 10:
    print("x est un nombre positif à deux chiffres ou plus")
else:
    print("x est un nombre négatif")

Les boucles en Python

Les boucles en Python permettent d'exécuter une partie du code plusieurs fois. Il existe deux types de boucles couramment utilisées en Python : la boucle "for" et la boucle "while".

La boucle "for" est utilisée pour itérer sur une séquence (comme une liste, une chaîne de caractères, un tuple, etc.) ou sur tout objet itérable. La structure de base d'une boucle "for" est la suivante :

for element in sequence:
    # code à exécuter pour chaque élément de la séquence

Par exemple, pour afficher tous les éléments d'une liste :

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

La boucle "while" est utilisée pour exécuter une partie du code tant qu'une condition spécifique est vraie. La structure de base d'une boucle "while" est la suivante :

while condition:
    # code à exécuter tant que la condition est vraie

Par exemple, pour afficher les nombres de 1 à 5 :

i = 1

while i <= 5:
    print(i)
    i += 1

Les instructions de contrôle en Python

En plus des conditions et des boucles, Python propose également d'autres instructions de contrôle pour gérer le flux d'exécution du code.

L'instruction "break" permet de sortir d'une boucle prématurément si une condition spécifique est remplie :

for i in range(1, 10):
    if i == 5:
        break
    print(i)

L'instruction "continue" permet de passer à l'itération suivante d'une boucle sans exécuter le reste du code pour cette itération :

for i in range(1, 6):
    if i == 3:
        continue
    print(i)

L'instruction "pass" est utilisée comme espace réservé pour une fonctionnalité à implémenter ultérieurement. Elle ne fait rien et permet d'éviter une erreur de syntaxe :

if condition:
    pass
else:
    # code à exécuter si la condition est fausse

Ces instructions de contrôle peuvent être utilisées en combinaison avec les conditions et les boucles pour créer des structures de contrôle plus complexes.
