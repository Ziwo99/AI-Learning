Section 4 : La manipulation de fichiers

Lecture de fichiers

La lecture de fichiers est une opération courante en Python. Elle permet de récupérer le contenu d'un fichier existant pour le traiter dans votre programme. Pour lire un fichier, vous devez suivre les étapes suivantes :

  1. Ouvrir le fichier en utilisant la fonction open() avec le nom du fichier et le mode d'ouverture (par exemple, "r" pour la lecture).
  2. Lire le contenu du fichier en utilisant la méthode read() sur l'objet fichier.
  3. Fermer le fichier en utilisant la méthode close() sur l'objet fichier.

Voici un exemple de code qui lit le contenu d'un fichier texte :

file = open("mon_fichier.txt", "r")
content = file.read()
file.close()

print(content)

Ecriture dans des fichiers

L'écriture dans des fichiers est une autre opération courante en Python. Elle permet d'écrire du contenu dans un fichier existant ou de créer un nouveau fichier. Pour écrire dans un fichier, vous devez suivre les étapes suivantes :

  1. Ouvrir le fichier en utilisant la fonction open() avec le nom du fichier et le mode d'ouverture (par exemple, "w" pour l'écriture).
  2. Écrire le contenu dans le fichier en utilisant la méthode write() sur l'objet fichier.
  3. Fermer le fichier en utilisant la méthode close() sur l'objet fichier.

Voici un exemple de code qui écrit du contenu dans un fichier texte :

file = open("mon_fichier.txt", "w")
file.write("Bonjour, monde!")
file.close()

Gestion des erreurs lors de la manipulation de fichiers

Lorsque vous manipulez des fichiers, il est important de gérer les erreurs qui peuvent survenir. Par exemple, si le fichier que vous essayez de lire n'existe pas, une erreur sera levée. Pour gérer ces erreurs, vous pouvez utiliser des blocs try et except.

Voici un exemple de code qui gère une erreur de fichier inexistant :

try:
    file = open("mon_fichier.txt", "r")
    content = file.read()
    file.close()
    print(content)
except FileNotFoundError:
    print("Le fichier n'existe pas.")

En utilisant les blocs try et except, vous pouvez capturer les erreurs et exécuter un code spécifique en cas d'erreur.
