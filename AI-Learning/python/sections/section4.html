Les listes et les opérations sur les listes

Les listes sont des structures de données couramment utilisées en Python. Elles permettent de stocker une collection ordonnée d'éléments. Une liste peut contenir des éléments de différents types tels que des entiers, des chaînes de caractères, des booléens, voire même d'autres listes.

Pour créer une liste en Python, on utilise des crochets [] et on sépare les éléments par des virgules. Par exemple :

ma_liste = [1, 2, 3, 4, 5]

On peut accéder aux éléments d'une liste en utilisant leur indice. L'indice du premier élément est 0, celui du deuxième élément est 1, et ainsi de suite. Par exemple :

premier_element = ma_liste[0]
deuxieme_element = ma_liste[1]

On peut également modifier les éléments d'une liste en utilisant leur indice. Par exemple :

ma_liste[0] = 10

Pour ajouter un élément à une liste, on utilise la méthode append(). Par exemple :

ma_liste.append(6)

Pour supprimer un élément d'une liste, on utilise la méthode remove(). Par exemple :

ma_liste.remove(3)

Il est également possible de concaténer deux listes en utilisant l'opérateur +. Par exemple :

liste1 = [1, 2, 3]
liste2 = [4, 5, 6]
liste_concatenee = liste1 + liste2

Les tuples et les opérations sur les tuples

Les tuples sont similaires aux listes, mais ils sont immuables, c'est-à-dire qu'ils ne peuvent pas être modifiés une fois créés. Les tuples sont souvent utilisés pour stocker des collections d'éléments hétérogènes.

Pour créer un tuple en Python, on utilise des parenthèses () et on sépare les éléments par des virgules. Par exemple :

mon_tuple = (1, "deux", True)

On peut accéder aux éléments d'un tuple de la même manière qu'une liste, en utilisant leur indice. Par exemple :

premier_element = mon_tuple[0]
deuxieme_element = mon_tuple[1]

Comme les tuples sont immuables, on ne peut pas modifier leurs éléments. Par exemple, la ligne de code suivante générera une erreur :

mon_tuple[0] = 10

Il est cependant possible de concaténer deux tuples en utilisant l'opérateur +. Par exemple :

tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
tuple_concatene = tuple1 + tuple2

Les dictionnaires et les opérations sur les dictionnaires

Les dictionnaires sont des structures de données qui associent des clés à des valeurs. Chaque élément d'un dictionnaire est composé d'une clé et d'une valeur correspondante. Les dictionnaires sont très utiles pour stocker des données structurées.

Pour créer un dictionnaire en Python, on utilise des accolades {} et on sépare les clés et les valeurs par des deux-points (:). Par exemple :

mon_dictionnaire = {"nom": "John", "age": 30, "ville": "Paris"}

On peut accéder aux valeurs d'un dictionnaire en utilisant leurs clés. Par exemple :

nom = mon_dictionnaire["nom"]
age = mon_dictionnaire["age"]

On peut également modifier les valeurs d'un dictionnaire en utilisant leurs clés. Par exemple :

mon_dictionnaire["age"] = 31

Pour ajouter une nouvelle paire clé-valeur à un dictionnaire, on utilise simplement une nouvelle clé et on lui associe une valeur. Par exemple :

mon_dictionnaire["profession"] = "ingénieur"

Pour supprimer une paire clé-valeur d'un dictionnaire, on utilise l'instruction del suivie de la clé à supprimer. Par exemple :

del mon_dictionnaire["ville"]

Les ensembles et les opérations sur les ensembles

Les ensembles sont des structures de données qui permettent de stocker une collection non ordonnée d'éléments uniques. Les ensembles sont très utiles pour effectuer des opérations ensemblistes telles que l'union, l'intersection et la différence.

Pour créer un ensemble en Python, on utilise la fonction set() et on passe les éléments en tant qu'arguments. Par exemple :

mon_ensemble = set([1, 2, 3, 4, 5])

On peut également créer un ensemble en utilisant des accolades {} et en séparant les éléments par des virgules. Par exemple :

mon_ensemble = {1, 2, 3, 4, 5}

Pour ajouter un élément à un ensemble, on utilise la méthode add(). Par exemple :

mon_ensemble.add(6)

Pour supprimer un élément d'un ensemble, on utilise la méthode remove(). Par exemple :

mon_ensemble.remove(3)

Il est possible d'effectuer des opérations ensemblistes sur les ensembles tels que l'union, l'intersection et la différence. Par exemple :

ensemble1 = {1, 2, 3}
ensemble2 = {3, 4, 5}
union = ensemble1.union(ensemble2)
intersection = ensemble1.intersection(ensemble2)
difference = ensemble1.difference(ensemble2)

Les ensembles peuvent également être utilisés pour éliminer les doublons d'une liste. Par exemple :

ma_liste = [1, 2, 3, 3, 4, 5, 5]
mon_ensemble = set(ma_liste)
