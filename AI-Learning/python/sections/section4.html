Section 4: Fonctions et modules

Les fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser le code en le divisant en tâches plus petites et plus gérables. En Python, vous pouvez définir une fonction en utilisant le mot-clé def suivi du nom de la fonction et des paramètres entre parenthèses. Voici un exemple de fonction qui calcule la somme de deux nombres :

def addition(a, b):
    return a + b

resultat = addition(3, 5)
print(resultat)  # Output: 8

Dans cet exemple, la fonction addition prend deux paramètres a et b et renvoie leur somme. La fonction est ensuite appelée avec les arguments 3 et 5, et le résultat est stocké dans la variable resultat et affiché.

Les modules

Les modules sont des fichiers contenant du code Python qui peuvent être importés dans d'autres programmes. Ils permettent d'organiser le code en le divisant en fichiers plus petits et plus spécialisés. Python dispose d'un grand nombre de modules prédéfinis, tels que math pour les opérations mathématiques, random pour la génération de nombres aléatoires, etc. Pour utiliser un module, vous devez l'importer en utilisant le mot-clé import. Voici un exemple d'utilisation du module math :

import math

racine_carree = math.sqrt(16)
print(racine_carree)  # Output: 4.0

Dans cet exemple, le module math est importé et la fonction sqrt est utilisée pour calculer la racine carrée de 16. Le résultat est stocké dans la variable racine_carree et affiché.

Les packages

Les packages sont des dossiers contenant des modules Python. Ils permettent d'organiser le code en le divisant en dossiers et sous-dossiers. Un package peut contenir plusieurs modules et sous-packages. Pour utiliser un module d'un package, vous devez spécifier le nom du package suivi du nom du module. Voici un exemple d'utilisation d'un module d'un package :

import numpy as np

liste = [1, 2, 3, 4, 5]
moyenne = np.mean(liste)
print(moyenne)  # Output: 3.0

Dans cet exemple, le module mean du package numpy est importé en utilisant l'alias np. La fonction mean est utilisée pour calculer la moyenne de la liste [1, 2, 3, 4, 5]. Le résultat est stocké dans la variable moyenne et affiché.
