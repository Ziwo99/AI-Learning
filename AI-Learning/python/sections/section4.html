Lecture et écriture de fichiers texte

La manipulation de fichiers texte est une tâche courante en programmation Python. Pour lire le contenu d'un fichier texte, nous utilisons la fonction intégrée open() avec le mode de lecture approprié. Voici un exemple :

file = open("mon_fichier.txt", "r")
content = file.read()
file.close()

print(content)

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode lecture ("r") et stockons son contenu dans la variable content. Ensuite, nous fermons le fichier à l'aide de la méthode close() et affichons le contenu.

Pour écrire dans un fichier texte, nous utilisons également la fonction open(), mais cette fois avec le mode d'écriture approprié. Voici un exemple :

file = open("mon_fichier.txt", "w")
file.write("Bonjour, monde!")
file.close()

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode écriture ("w") et utilisons la méthode write() pour écrire la chaîne de caractères "Bonjour, monde!" dans le fichier. Ensuite, nous fermons le fichier.

Gestion des exceptions

La gestion des exceptions est une pratique essentielle en programmation Python pour gérer les erreurs qui peuvent survenir lors de la manipulation de fichiers. Nous utilisons les blocs try et except pour capturer et gérer les exceptions. Voici un exemple :

try:
    file = open("mon_fichier.txt", "r")
    content = file.read()
    file.close()
except FileNotFoundError:
    print("Le fichier n'a pas été trouvé.")
except:
    print("Une erreur s'est produite lors de la manipulation du fichier.")

Dans cet exemple, nous essayons d'ouvrir le fichier "mon_fichier.txt" en mode lecture. Si le fichier n'est pas trouvé, une exception de type FileNotFoundError est levée et capturée par le bloc except FileNotFoundError. Si une autre exception se produit, elle est capturée par le bloc except générique.

Il est important de gérer les exceptions de manière appropriée pour éviter les plantages inattendus de notre programme et pour fournir des messages d'erreur clairs aux utilisateurs.
