Section 4: Manipulation de fichiers et dossiers

Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en Python. Pour lire le contenu d'un fichier, vous pouvez utiliser la fonction open() avec le mode de lecture approprié. Voici un exemple :

fichier = open("mon_fichier.txt", "r")
contenu = fichier.read()
print(contenu)
fichier.close()

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode de lecture ("r") et utilisons la méthode read() pour lire tout le contenu du fichier. Ensuite, nous affichons le contenu à l'aide de la fonction print(). Enfin, nous fermons le fichier à l'aide de la méthode close().

Pour écrire dans un fichier, vous pouvez utiliser la fonction open() avec le mode d'écriture approprié. Voici un exemple :

fichier = open("mon_fichier.txt", "w")
fichier.write("Bonjour, monde!")
fichier.close()

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode d'écriture ("w") et utilisons la méthode write() pour écrire la chaîne de caractères "Bonjour, monde!" dans le fichier. Ensuite, nous fermons le fichier à l'aide de la méthode close().

Manipulation de dossiers

La manipulation de dossiers en Python peut être réalisée à l'aide du module os. Voici quelques opérations courantes :

Pour créer un nouveau dossier, vous pouvez utiliser la fonction mkdir() :

import os
os.mkdir("nouveau_dossier")

Dans cet exemple, nous importons le module os et utilisons la fonction mkdir() pour créer un nouveau dossier nommé "nouveau_dossier".

Pour supprimer un dossier, vous pouvez utiliser la fonction rmdir() :

import os
os.rmdir("nouveau_dossier")

Dans cet exemple, nous importons le module os et utilisons la fonction rmdir() pour supprimer le dossier "nouveau_dossier".

Pour vérifier si un dossier existe, vous pouvez utiliser la fonction exists() :

import os
if os.path.exists("nouveau_dossier"):
    print("Le dossier existe.")
else:
    print("Le dossier n'existe pas.")

Dans cet exemple, nous importons le module os et utilisons la fonction exists() pour vérifier si le dossier "nouveau_dossier" existe. Ensuite, nous affichons un message approprié en fonction du résultat.

Ce sont là quelques-unes des opérations de base pour la manipulation de fichiers et de dossiers en Python. Vous pouvez explorer davantage le module os pour découvrir d'autres fonctionnalités utiles.
