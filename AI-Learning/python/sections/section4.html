Les listes

Une liste est une structure de données qui permet de stocker une collection ordonnée d'éléments. Les éléments d'une liste peuvent être de différents types tels que des entiers, des chaînes de caractères, des booléens, voire même d'autres listes. Une liste est définie en utilisant des crochets [] et les éléments sont séparés par des virgules.

Exemple :

ma_liste = [1, 2, 3, "quatre", True]

Pour accéder à un élément spécifique d'une liste, on utilise l'index de l'élément. L'index d'une liste commence à 0 pour le premier élément, 1 pour le deuxième élément, et ainsi de suite.

Exemple :

print(ma_liste[0])  # Affiche 1
print(ma_liste[3])  # Affiche "quatre"

Les listes sont mutables, ce qui signifie que vous pouvez modifier les éléments d'une liste après sa création. Vous pouvez ajouter des éléments à une liste, supprimer des éléments, ou même modifier des éléments existants.

Exemple :

ma_liste.append(5)  # Ajoute l'élément 5 à la fin de la liste
print(ma_liste)  # Affiche [1, 2, 3, "quatre", True, 5]

ma_liste.remove("quatre")  # Supprime l'élément "quatre" de la liste
print(ma_liste)  # Affiche [1, 2, 3, True, 5]

ma_liste[2] = "trois"  # Modifie l'élément à l'index 2
print(ma_liste)  # Affiche [1, 2, "trois", True, 5]

Il existe de nombreuses opérations que vous pouvez effectuer sur les listes, telles que la concaténation de listes, la recherche d'éléments, le tri de la liste, etc. Vous pouvez également utiliser des boucles pour parcourir les éléments d'une liste.

Les tuples

Un tuple est une structure de données similaire à une liste, mais contrairement aux listes, les tuples sont immuables, ce qui signifie que vous ne pouvez pas modifier les éléments d'un tuple après sa création. Les tuples sont définis en utilisant des parenthèses () et les éléments sont séparés par des virgules.

Exemple :

mon_tuple = (1, 2, 3, "quatre", True)

Vous pouvez accéder aux éléments d'un tuple de la même manière que pour une liste, en utilisant l'index de l'élément.

Exemple :

print(mon_tuple[0])  # Affiche 1
print(mon_tuple[3])  # Affiche "quatre"

Comme les tuples sont immuables, vous ne pouvez pas ajouter, supprimer ou modifier des éléments d'un tuple après sa création.

Les dictionnaires

Un dictionnaire est une structure de données qui permet de stocker des paires clé-valeur. Chaque élément d'un dictionnaire est composé d'une clé et d'une valeur associée. Les clés d'un dictionnaire doivent être uniques et les valeurs peuvent être de différents types.

Un dictionnaire est défini en utilisant des accolades {} et les paires clé-valeur sont séparées par des virgules. La clé et la valeur sont séparées par deux points :

Exemple :

mon_dictionnaire = {"nom": "John", "âge": 25, "ville": "Paris"}

Pour accéder à une valeur spécifique dans un dictionnaire, on utilise la clé correspondante.

Exemple :

print(mon_dictionnaire["nom"])  # Affiche "John"
print(mon_dictionnaire["âge"])  # Affiche 25

Vous pouvez également modifier les valeurs d'un dictionnaire en utilisant la clé correspondante.

Exemple :

mon_dictionnaire["ville"] = "Lyon"
print(mon_dictionnaire)  # Affiche {"nom": "John", "âge": 25, "ville": "Lyon"}

Les dictionnaires sont très utiles lorsque vous avez besoin de stocker des informations associées à des clés spécifiques, comme par exemple une base de données.

Les ensembles

Un ensemble est une structure de données qui permet de stocker une collection non ordonnée d'éléments uniques. Les ensembles sont définis en utilisant des accolades {} et les éléments sont séparés par des virgules.

Exemple :

mon_ensemble = {1, 2, 3, 4, 5}

Les ensembles ne permettent pas d'avoir des éléments en double. Si vous ajoutez un élément déjà présent dans l'ensemble, il ne sera pas ajouté une deuxième fois.

Exemple :

mon_ensemble.add(3)
print(mon_ensemble)  # Affiche {1, 2, 3, 4, 5}

Les ensembles sont très utiles lorsque vous avez besoin de vérifier rapidement si un élément est présent ou non dans une collection.

Les opérations sur les structures de données

Les structures de données en Python offrent de nombreuses opérations et méthodes pour effectuer des manipulations et des traitements spécifiques.

Voici quelques exemples d'opérations couramment utilisées sur les structures de données :

  • Concaténation : vous pouvez concaténer deux listes, deux tuples ou deux ensembles en utilisant l'opérateur +.
  • Recherche : vous pouvez rechercher un élément spécifique dans une liste, un tuple ou un ensemble en utilisant l'opérateur in.
  • Tri : vous pouvez trier les éléments d'une liste en utilisant la méthode sort().
  • Longueur : vous pouvez obtenir la longueur d'une liste, d'un tuple ou d'un ensemble en utilisant la fonction len().

Exemple :

liste1 = [1, 2, 3]
liste2 = [4, 5, 6]
concatenation = liste1 + liste2
print(concatenation)  # Affiche [1, 2, 3, 4, 5, 6]

if 3 in liste1:
    print("L'élément 3 est présent dans la liste1")

liste1.sort()
print(liste1)  # Affiche [1, 2, 3]

longueur_liste1 = len(liste1)
print(longueur_liste1)  # Affiche 3

Ces opérations et méthodes sont très utiles pour manipuler et traiter les structures de données en Python.
