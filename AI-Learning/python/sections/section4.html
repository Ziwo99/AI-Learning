Section 4: Les fonctions en Python

Introduction aux fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser et de structurer votre code, ce qui le rend plus lisible et plus facile à maintenir. En Python, vous pouvez définir vos propres fonctions à l'aide du mot-clé def.

Voici un exemple de fonction simple qui affiche un message :

def say_hello():
    print("Bonjour !")

say_hello() # Appel de la fonction

La fonction say_hello() affiche le message "Bonjour !" lorsque vous l'appelez. Vous pouvez appeler une fonction autant de fois que vous le souhaitez dans votre code.

Les paramètres et les arguments

Les fonctions peuvent accepter des paramètres, qui sont des valeurs que vous pouvez passer lors de l'appel de la fonction. Ces paramètres permettent à la fonction de traiter différentes valeurs sans avoir à réécrire le code à chaque fois.

Voici un exemple de fonction qui prend un paramètre :

def say_hello(name):
    print("Bonjour " + name + "!")

say_hello("Alice") # Appel de la fonction avec l'argument "Alice"
say_hello("Bob") # Appel de la fonction avec l'argument "Bob"

La fonction say_hello() accepte un paramètre name et affiche un message personnalisé en utilisant cet argument. Vous pouvez passer différents arguments lors de l'appel de la fonction.

Les fonctions récursives

Une fonction récursive est une fonction qui s'appelle elle-même. Cela peut être utile pour résoudre des problèmes qui peuvent être décomposés en sous-problèmes plus petits.

Voici un exemple de fonction récursive qui calcule la factorielle d'un nombre :

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

result = factorial(5)
print(result) # Affiche 120

La fonction factorial() calcule la factorielle d'un nombre en multipliant ce nombre par la factorielle du nombre précédent. La condition d'arrêt est lorsque n est égal à 0, auquel cas la fonction retourne 1.

Les fonctions lambda

Les fonctions lambda, également appelées fonctions anonymes, sont des fonctions sans nom. Elles sont souvent utilisées pour des opérations simples et ponctuelles.

Voici un exemple d'utilisation d'une fonction lambda :

addition = lambda x, y: x + y
result = addition(3, 5)
print(result) # Affiche 8

La fonction lambda addition prend deux arguments x et y et retourne leur somme. Vous pouvez appeler cette fonction lambda comme n'importe quelle autre fonction.
