Section 4: Les fonctions en Python

Introduction aux fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser le code en le divisant en tâches plus petites et plus gérables. En Python, vous pouvez définir vos propres fonctions à l'aide du mot-clé def.

Voici un exemple de fonction simple qui affiche un message :

def say_hello():
    print("Bonjour !")

say_hello()  # Appel de la fonction

Ce code définit une fonction say_hello() qui affiche le message "Bonjour !". L'appel de la fonction à la fin du code exécute cette fonction et affiche le message.

Les paramètres et les arguments

Les fonctions peuvent prendre des paramètres, qui sont des valeurs que vous pouvez passer à la fonction pour qu'elle les utilise. Les paramètres sont spécifiés entre parenthèses après le nom de la fonction.

Voici un exemple de fonction qui prend un paramètre :

def say_hello(name):
    print("Bonjour " + name + "!")

say_hello("Alice")  # Appel de la fonction avec l'argument "Alice"

Ce code définit une fonction say_hello() qui prend un paramètre name. Lorsque la fonction est appelée avec l'argument "Alice", elle affiche le message "Bonjour Alice !".

Les fonctions récursives

Une fonction récursive est une fonction qui s'appelle elle-même à l'intérieur de son propre corps. Cela permet de résoudre des problèmes qui peuvent être décomposés en sous-problèmes plus petits.

Voici un exemple de fonction récursive qui calcule la factorielle d'un nombre :

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

result = factorial(5)
print(result)  # Affiche 120

Ce code définit une fonction factorial() qui calcule la factorielle d'un nombre en utilisant la récursion. L'appel de la fonction factorial(5) renvoie le résultat 120.

Les fonctions lambda

Les fonctions lambda, également appelées fonctions anonymes, sont des fonctions sans nom. Elles sont utilisées pour créer des fonctions simples et courtes en une seule ligne.

Voici un exemple de fonction lambda qui calcule le carré d'un nombre :

square = lambda x: x**2

result = square(5)
print(result)  # Affiche 25

Ce code définit une fonction lambda square qui calcule le carré d'un nombre. L'appel de la fonction square(5) renvoie le résultat 25.
