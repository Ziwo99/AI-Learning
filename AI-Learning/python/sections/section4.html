Section 4: Les fonctions en Python

Introduction aux fonctions en Python

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser et de structurer le code, ce qui facilite la lecture et la maintenance. En Python, une fonction est définie à l'aide du mot-clé "def", suivi du nom de la fonction et de ses paramètres entre parenthèses. Voici un exemple de définition de fonction :

def say_hello(name):
    print("Hello, " + name + "!")

Dans cet exemple, la fonction "say_hello" prend un paramètre "name" et affiche un message de salutation avec ce nom. Pour appeler cette fonction, on utilise simplement son nom suivi des arguments entre parenthèses :

say_hello("Alice")

Cela affichera "Hello, Alice!".

Les paramètres et les arguments en Python

Les paramètres sont des variables utilisées dans la définition d'une fonction, tandis que les arguments sont les valeurs réelles passées à une fonction lors de son appel. En Python, les paramètres peuvent avoir une valeur par défaut, ce qui signifie qu'ils peuvent être omis lors de l'appel de la fonction. Voici un exemple :

def multiply(a, b=2):
    return a * b

Dans cet exemple, la fonction "multiply" prend deux paramètres, "a" et "b". Le paramètre "b" a une valeur par défaut de 2. Ainsi, si on appelle la fonction avec un seul argument :

result = multiply(3)

La variable "result" sera égale à 6, car "a" est égal à 3 et "b" est égal à sa valeur par défaut de 2.

Les fonctions récursives en Python

Une fonction récursive est une fonction qui s'appelle elle-même à l'intérieur de son propre corps. Cela permet de résoudre des problèmes qui peuvent être décomposés en sous-problèmes plus simples. Voici un exemple de fonction récursive qui calcule la factorielle d'un nombre :

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

Dans cet exemple, la fonction "factorial" vérifie si le nombre "n" est égal à zéro. Si c'est le cas, elle renvoie 1. Sinon, elle renvoie le produit de "n" par l'appel récursif de la fonction "factorial" avec l'argument "n-1". Ainsi, si on appelle la fonction avec "n" égal à 5 :

result = factorial(5)

La variable "result" sera égale à 120, car 5! (factorielle de 5) est égal à 5 * 4 * 3 * 2 * 1 = 120.

Les fonctions lambda en Python

Les fonctions lambda, également appelées fonctions anonymes, sont des fonctions sans nom qui peuvent être utilisées pour créer des fonctions simples et courtes. Elles sont définies à l'aide du mot-clé "lambda" suivi des paramètres entre parenthèses, puis d'une expression qui est évaluée et renvoyée. Voici un exemple :

add = lambda x, y: x + y

Dans cet exemple, la fonction lambda "add" prend deux paramètres, "x" et "y", et renvoie leur somme. On peut appeler cette fonction de la même manière qu'une fonction normale :

result = add(3, 5)

La variable "result" sera égale à 8, car la fonction lambda renvoie la somme de 3 et 5.
