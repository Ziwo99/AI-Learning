Section 10: Les bases de données en Python

Introduction aux bases de données

Une base de données est un outil essentiel pour stocker et organiser de grandes quantités de données de manière structurée. En Python, nous avons plusieurs bibliothèques qui nous permettent de travailler avec des bases de données, telles que SQLite, MySQL et PostgreSQL. Dans cette section, nous allons explorer les bases de données en Python et apprendre comment les utiliser efficacement.

La connexion à une base de données

Avant de pouvoir interagir avec une base de données, nous devons établir une connexion. La bibliothèque SQLite3 est intégrée à Python, ce qui en fait un excellent choix pour commencer. Voici comment établir une connexion à une base de données SQLite en Python : 

import sqlite3

# Établir une connexion à la base de données
conn = sqlite3.connect('ma_base_de_donnees.db')

# Créer un curseur pour exécuter des requêtes
cur = conn.cursor()

# Fermer la connexion à la base de données
conn.close()

Les requêtes SQL en Python

Maintenant que nous avons établi une connexion à notre base de données, nous pouvons exécuter des requêtes SQL pour récupérer, insérer, mettre à jour ou supprimer des données. Voici quelques exemples de requêtes SQL couramment utilisées en Python : 

# Récupérer toutes les lignes d'une table
cur.execute("SELECT * FROM ma_table")
rows = cur.fetchall()

# Insérer une nouvelle ligne dans une table
cur.execute("INSERT INTO ma_table (colonne1, colonne2) VALUES (?, ?)", (valeur1, valeur2))
conn.commit()

# Mettre à jour une ligne dans une table
cur.execute("UPDATE ma_table SET colonne1 = ? WHERE condition", (nouvelle_valeur,))

# Supprimer une ligne d'une table
cur.execute("DELETE FROM ma_table WHERE condition")
conn.commit()

Ces exemples vous donnent une idée de la façon dont vous pouvez interagir avec une base de données en utilisant Python. Assurez-vous de consulter la documentation spécifique à la base de données que vous utilisez pour en savoir plus sur les requêtes SQL disponibles.

En conclusion, les bases de données sont un élément essentiel de la plupart des applications modernes. En utilisant Python, vous pouvez facilement vous connecter à une base de données, exécuter des requêtes SQL et manipuler les données de manière efficace. Continuez à pratiquer et à explorer les différentes bibliothèques de bases de données disponibles pour devenir un expert en la matière.
