Section 10: Les bases de données en Python

Introduction aux bases de données en Python

Les bases de données sont des outils essentiels pour stocker et organiser de grandes quantités de données de manière structurée. En Python, nous pouvons utiliser différents modules pour interagir avec les bases de données, tels que SQLite, MySQL, PostgreSQL, etc.

Pour commencer à travailler avec les bases de données en Python, nous devons d'abord installer le module approprié. Par exemple, pour utiliser SQLite, nous pouvons installer le module sqlite3 en utilisant la commande suivante :

pip install pysqlite3

Une fois le module installé, nous pouvons importer la bibliothèque dans notre programme Python :

import sqlite3

La connexion à une base de données en Python

Pour interagir avec une base de données, nous devons d'abord établir une connexion. Dans le cas de SQLite, nous pouvons créer une connexion en utilisant la fonction connect() du module sqlite3 :

conn = sqlite3.connect('ma_base_de_donnees.db')

Cette ligne de code crée une connexion à une base de données SQLite appelée "ma_base_de_donnees.db". Si la base de données n'existe pas, elle sera créée automatiquement.

Les requêtes SQL en Python

Une fois que nous avons établi une connexion à la base de données, nous pouvons exécuter des requêtes SQL pour récupérer, insérer, mettre à jour ou supprimer des données.

Pour exécuter une requête SQL en Python, nous devons d'abord créer un curseur à l'aide de la méthode cursor() :

cursor = conn.cursor()

Ensuite, nous pouvons exécuter une requête en utilisant la méthode execute() du curseur :

cursor.execute("SELECT * FROM ma_table")

Cette ligne de code exécute une requête SELECT pour récupérer toutes les lignes de la table "ma_table". Les résultats de la requête peuvent être récupérés en utilisant la méthode fetchall() :

rows = cursor.fetchall()

Nous pouvons également exécuter des requêtes INSERT, UPDATE ou DELETE pour insérer, mettre à jour ou supprimer des données dans la base de données.

Voici un exemple d'une requête INSERT pour insérer une nouvelle ligne dans une table :

cursor.execute("INSERT INTO ma_table (colonne1, colonne2) VALUES (?, ?)", (valeur1, valeur2))

Cette ligne de code insère une nouvelle ligne dans la table "ma_table" avec les valeurs spécifiées pour les colonnes "colonne1" et "colonne2". Les valeurs sont passées en tant que paramètres pour éviter les injections SQL.

Une fois que nous avons terminé d'exécuter nos requêtes, nous devons valider les modifications et fermer la connexion à la base de données :

conn.commit()
conn.close()

Cela garantit que toutes les modifications sont enregistrées dans la base de données et que la connexion est correctement fermée.

En résumé, les bases de données en Python nous permettent de stocker, organiser et manipuler des données de manière efficace. En utilisant les modules appropriés et en exécutant des requêtes SQL, nous pouvons interagir avec les bases de données de manière transparente.
