Les expressions régulières en Python

Les expressions régulières sont des séquences de caractères utilisées pour décrire un motif de recherche. En utilisant des expressions régulières, vous pouvez rechercher, extraire et manipuler des chaînes de caractères selon des critères spécifiques.

En Python, les expressions régulières sont prises en charge par le module re. Voici un exemple d'utilisation des expressions régulières pour rechercher un motif dans une chaîne de caractères :

import re

text = "Bonjour, je m'appelle Alice et j'ai 25 ans."
pattern = r"j'ai \d+ ans"

result = re.search(pattern, text)
if result:
    print("Motif trouvé :", result.group())
else:
    print("Motif non trouvé.")

Dans cet exemple, nous utilisons la fonction search() du module re pour rechercher le motif "j'ai \d+ ans" dans la chaîne de caractères. Le motif \d+ correspond à un ou plusieurs chiffres. Si le motif est trouvé, nous affichons le résultat à l'aide de la méthode group().

La manipulation de dates et d'heures en Python

En Python, le module datetime permet de manipuler les dates et les heures. Il fournit des classes et des fonctions pour créer, manipuler et formater les dates et les heures.

Voici un exemple d'utilisation du module datetime pour afficher la date et l'heure actuelles :

from datetime import datetime

now = datetime.now()
print("Date et heure actuelles :", now)

Le module datetime offre également des méthodes pour formater les dates et les heures selon différents formats. Voici un exemple :

from datetime import datetime

now = datetime.now()
formatted_date = now.strftime("%d/%m/%Y %H:%M:%S")
print("Date et heure formatées :", formatted_date)

Dans cet exemple, nous utilisons la méthode strftime() pour formater la date et l'heure actuelles selon le format "%d/%m/%Y %H:%M:%S".

La gestion des fichiers CSV en Python

Le format CSV (Comma-Separated Values) est couramment utilisé pour stocker des données tabulaires. En Python, le module csv permet de lire et d'écrire des fichiers CSV.

Voici un exemple d'utilisation du module csv pour lire un fichier CSV :

import csv

with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Dans cet exemple, nous utilisons la fonction reader() du module csv pour lire le contenu du fichier CSV. Nous itérons ensuite sur les lignes du fichier et affichons chaque ligne.

Le module csv offre également des fonctions pour écrire des données dans un fichier CSV. Voici un exemple :

import csv

data = [
    ['Nom', 'Âge', 'Ville'],
    ['Alice', '25', 'Paris'],
    ['Bob', '30', 'Londres'],
    ['Charlie', '35', 'New York']
]

with open('data.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerows(data)

Dans cet exemple, nous utilisons la fonction writer() du module csv pour écrire les données dans le fichier CSV. Nous utilisons ensuite la méthode writerows() pour écrire toutes les lignes en une seule fois.

La manipulation de fichiers JSON en Python

Le format JSON (JavaScript Object Notation) est couramment utilisé pour échanger des données structurées entre différents systèmes. En Python, le module json permet de lire et d'écrire des fichiers JSON.

Voici un exemple d'utilisation du module json pour lire un fichier JSON :

import json

with open('data.json', 'r') as file:
    data = json.load(file)
    print(data)

Dans cet exemple, nous utilisons la fonction load() du module json pour charger le contenu du fichier JSON dans une variable. Nous affichons ensuite le contenu de la variable.

Le module json offre également des fonctions pour écrire des données dans un fichier JSON. Voici un exemple :

import json

data = {
    'nom': 'Alice',
    'age': 25,
    'ville': 'Paris'
}

with open('data.json', 'w') as file:
    json.dump(data, file)

Dans cet exemple, nous utilisons la fonction dump() du module json pour écrire les données dans le fichier JSON.

Il est important de noter que les fichiers JSON doivent être valides selon la syntaxe JSON. Assurez-vous que les données que vous écrivez dans un fichier JSON sont correctement formatées.
