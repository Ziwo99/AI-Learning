Section 8: Les classes et les objets en Python

Introduction aux classes et aux objets

Les classes et les objets sont des concepts fondamentaux en programmation orientée objet (POO). En Python, une classe est une structure qui permet de regrouper des variables (appelées attributs) et des fonctions (appelées méthodes) liées à ces variables. Un objet est une instance d'une classe, c'est-à-dire une entité concrète qui possède ses propres valeurs d'attributs et peut exécuter les méthodes définies dans la classe.

Pour définir une classe en Python, utilisez le mot-clé class suivi du nom de la classe. Par convention, le nom de classe commence par une lettre majuscule. Voici un exemple de définition de classe :

class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def se_presenter(self):
        print("Je m'appelle", self.nom, "et j'ai", self.age, "ans.")

Dans cet exemple, nous avons défini une classe Personne avec deux attributs (nom et age) et une méthode (se_presenter). La méthode __init__ est un constructeur spécial qui est appelé lors de la création d'un nouvel objet de la classe. Elle permet d'initialiser les attributs de l'objet avec les valeurs passées en paramètres.

Les attributs et les méthodes

Les attributs sont des variables associées à une classe ou à un objet. Ils représentent les caractéristiques de la classe ou de l'objet. Les méthodes sont des fonctions associées à une classe ou à un objet. Elles permettent de définir le comportement de la classe ou de l'objet.

Les attributs peuvent être définis à l'intérieur de la classe ou dans le constructeur. Voici un exemple :

class Voiture:
    marque = "Ford"  # attribut de classe

    def __init__(self, couleur):
        self.couleur = couleur  # attribut d'instance

    def demarrer(self):
        print("La voiture démarre.")  # méthode

Dans cet exemple, nous avons un attribut de classe marque qui est partagé par toutes les instances de la classe Voiture. Nous avons également un attribut d'instance couleur qui est spécifique à chaque objet de la classe.

Les méthodes peuvent être appelées sur une instance de la classe. Par exemple :

ma_voiture = Voiture("rouge")
ma_voiture.demarrer()

Cela affichera "La voiture démarre." car la méthode demarrer est appelée sur l'objet ma_voiture.

L'héritage et le polymorphisme

L'héritage est un mécanisme qui permet de créer une nouvelle classe (appelée classe dérivée ou sous-classe) à partir d'une classe existante (appelée classe de base ou superclasse). La classe dérivée hérite des attributs et des méthodes de la classe de base, et peut ajouter de nouveaux attributs et méthodes ou modifier ceux existants.

Le polymorphisme est la capacité d'un objet à prendre plusieurs formes. En Python, le polymorphisme est souvent utilisé en conjonction avec l'héritage. Par exemple, une classe dérivée peut redéfinir une méthode héritée de la classe de base pour lui donner un comportement spécifique à la classe dérivée.

Voici un exemple d'utilisation de l'héritage et du polymorphisme :

class Animal:
    def __init__(self, nom):
        self.nom = nom

    def faire_son_cri(self):
        pass

class Chien(Animal):
    def faire_son_cri(self):
        print("Le chien aboie.")

class Chat(Animal):
    def faire_son_cri(self):
        print("Le chat miaule.")

Dans cet exemple, nous avons une classe de base Animal avec une méthode faire_son_cri. Les classes dérivées Chien et Chat redéfinissent cette méthode pour donner un comportement spécifique à chaque classe. Par exemple :

mon_chien = Chien("Rex")
mon_chien.faire_son_cri()  # affiche "Le chien aboie."

Cela affichera "Le chien aboie." car la méthode faire_son_cri de la classe Chien est appelée sur l'objet mon_chien.
