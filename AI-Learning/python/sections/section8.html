Introduction à la bibliothèque NumPy

La bibliothèque NumPy est l'une des bibliothèques les plus populaires en Python pour le traitement des données numériques. Elle fournit des structures de données de tableau multidimensionnel, ainsi que des fonctions mathématiques pour manipuler ces tableaux. NumPy est largement utilisé dans le domaine de l'analyse de données, de l'apprentissage automatique et de la science des données.

Pour utiliser NumPy, vous devez d'abord l'installer. Vous pouvez le faire en exécutant la commande suivante dans votre terminal : 

pip install numpy

Une fois installé, vous pouvez importer NumPy dans votre code Python en utilisant l'instruction suivante : 

import numpy as np

Manipulation de tableaux multidimensionnels

NumPy offre une structure de données appelée "ndarray" (n-dimensional array) pour représenter des tableaux multidimensionnels. Ces tableaux peuvent avoir une ou plusieurs dimensions et peuvent contenir des éléments de même type de données.

Pour créer un tableau NumPy, vous pouvez utiliser la fonction "array" en passant une liste ou un tuple contenant les éléments du tableau. Par exemple, pour créer un tableau à une dimension, vous pouvez utiliser le code suivant : 

arr = np.array([1, 2, 3, 4, 5])

Vous pouvez également créer des tableaux multidimensionnels en passant une liste de listes ou un tuple de tuples. Par exemple, pour créer un tableau à deux dimensions, vous pouvez utiliser le code suivant : 

arr = np.array([[1, 2, 3], [4, 5, 6]])

Une fois que vous avez créé un tableau NumPy, vous pouvez accéder à ses éléments en utilisant des indices. Par exemple, pour accéder au premier élément d'un tableau à une dimension, vous pouvez utiliser le code suivant : 

print(arr[0])

Pour accéder à un élément spécifique d'un tableau à deux dimensions, vous pouvez utiliser des indices séparés par des virgules. Par exemple, pour accéder à l'élément à la deuxième ligne et à la troisième colonne d'un tableau à deux dimensions, vous pouvez utiliser le code suivant : 

print(arr[1, 2])

Calculs mathématiques et statistiques

NumPy offre de nombreuses fonctions mathématiques pour effectuer des calculs sur les tableaux. Vous pouvez effectuer des opérations arithmétiques de base, telles que l'addition, la soustraction, la multiplication et la division, sur les tableaux NumPy. Par exemple, pour ajouter deux tableaux ensemble, vous pouvez utiliser le code suivant : 

arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
result = arr1 + arr2
print(result)

NumPy offre également des fonctions mathématiques avancées, telles que la racine carrée, le logarithme, le sinus, le cosinus, etc. Vous pouvez utiliser ces fonctions en passant le tableau sur lequel vous souhaitez effectuer le calcul. Par exemple, pour calculer la racine carrée de chaque élément d'un tableau, vous pouvez utiliser le code suivant : 

arr = np.array([1, 4, 9])
result = np.sqrt(arr)
print(result)

Visualisation de données avec Matplotlib

Matplotlib est une bibliothèque de visualisation de données en Python. Elle permet de créer des graphiques, des diagrammes et des visualisations à partir des données. Matplotlib est souvent utilisé en conjonction avec NumPy pour visualiser les résultats d'analyses de données.

Pour utiliser Matplotlib, vous devez d'abord l'installer. Vous pouvez le faire en exécutant la commande suivante dans votre terminal : 

pip install matplotlib

Une fois installé, vous pouvez importer Matplotlib dans votre code Python en utilisant l'instruction suivante : 

import matplotlib.pyplot as plt

Pour créer un graphique avec Matplotlib, vous devez d'abord créer des données à visualiser. Vous pouvez ensuite utiliser les fonctions de Matplotlib pour créer différents types de graphiques, tels que des graphiques linéaires, des graphiques à barres, des graphiques circulaires, etc. Par exemple, pour créer un graphique linéaire à partir de deux tableaux NumPy, vous pouvez utiliser le code suivant : 

x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])
plt.plot(x, y)
plt.show()

Ce code créera un graphique linéaire avec les valeurs de "x" sur l'axe des abscisses et les valeurs de "y" sur l'axe des ordonnées.

Il existe de nombreuses autres fonctionnalités et options de personnalisation disponibles dans Matplotlib pour créer des visualisations de données plus avancées. Vous pouvez consulter la documentation officielle de Matplotlib pour en savoir plus sur ces fonctionnalités.
