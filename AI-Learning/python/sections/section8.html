Section 8: Manipulation de données avec NumPy

Introduction à NumPy

NumPy est une bibliothèque Python populaire utilisée pour effectuer des calculs numériques et manipuler des tableaux multidimensionnels. Il fournit des structures de données efficaces pour stocker et manipuler de grandes quantités de données, ainsi que des fonctions mathématiques pour effectuer des opérations sur ces données.

Pour commencer à utiliser NumPy, vous devez d'abord l'installer. Vous pouvez le faire en utilisant la commande suivante dans votre terminal :

pip install numpy

Une fois que NumPy est installé, vous pouvez l'importer dans votre programme Python en utilisant la commande suivante :

import numpy as np

Les tableaux NumPy

Les tableaux NumPy sont la structure de données principale utilisée par NumPy. Ils sont similaires aux listes Python, mais offrent des fonctionnalités supplémentaires pour effectuer des opérations mathématiques sur les données.

Vous pouvez créer un tableau NumPy en utilisant la fonction numpy.array(). Par exemple, pour créer un tableau contenant les nombres 1, 2 et 3, vous pouvez utiliser le code suivant :

import numpy as np

my_array = np.array([1, 2, 3])
print(my_array)

Cela affichera :

[1 2 3]

Les opérations sur les tableaux NumPy

NumPy offre de nombreuses fonctions pour effectuer des opérations sur les tableaux. Vous pouvez effectuer des opérations mathématiques de base, telles que l'addition, la soustraction, la multiplication et la division, sur les tableaux NumPy.

Voici un exemple montrant comment effectuer une addition sur deux tableaux NumPy :

import numpy as np

array1 = np.array([1, 2, 3])
array2 = np.array([4, 5, 6])

result = array1 + array2
print(result)

Cela affichera :

[5 7 9]

Vous pouvez également effectuer des opérations mathématiques sur des tableaux multidimensionnels. Par exemple, vous pouvez calculer la somme des éléments d'un tableau multidimensionnel en utilisant la fonction numpy.sum().

import numpy as np

my_array = np.array([[1, 2, 3], [4, 5, 6]])

result = np.sum(my_array)
print(result)

Cela affichera :

21

NumPy offre de nombreuses autres fonctionnalités pour manipuler et effectuer des opérations sur les tableaux. Il est recommandé de consulter la documentation officielle de NumPy pour en savoir plus sur ces fonctionnalités.
