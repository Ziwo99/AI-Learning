Section 8: Développement web avec Python

Introduction au développement web

Le développement web est un domaine passionnant qui permet de créer des sites web interactifs et dynamiques. Python est un langage de programmation polyvalent qui peut être utilisé pour développer des applications web. Dans cette section, nous explorerons les bases du développement web avec Python.

Flask

Flask est un framework web léger et flexible qui permet de créer rapidement des applications web en utilisant Python. Il est idéal pour les petits projets et les prototypes. Voici un exemple de code Flask qui affiche "Hello, World!" sur une page web : 

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run()

Django

Django est un framework web complet et puissant qui facilite le développement d'applications web complexes. Il suit le principe du "batteries included", ce qui signifie qu'il offre de nombreuses fonctionnalités prêtes à l'emploi. Voici un exemple de code Django qui affiche une liste de tâches à faire : 

from django.shortcuts import render

def todo_list(request):
    tasks = ['Task 1', 'Task 2', 'Task 3']
    return render(request, 'todo_list.html', {'tasks': tasks})

Dans cet exemple, nous utilisons le moteur de templates de Django pour afficher la liste des tâches sur une page web. Le fichier "todo_list.html" contient le code HTML correspondant à la présentation des tâches.

En conclusion, le développement web avec Python offre de nombreuses possibilités pour créer des applications web puissantes et interactives. Que vous choisissiez Flask pour des projets plus simples ou Django pour des projets plus complexes, Python est un excellent choix pour le développement web.
