Section 11: Les projets pratiques en Python

Projet 1 : Création d'un jeu en Python

Dans ce projet, vous allez apprendre à créer un jeu simple en utilisant Python. Vous allez découvrir les concepts de base de la programmation orientée objet et comment les appliquer pour créer un jeu interactif.

Voici un exemple de code pour créer un jeu de devinette en Python :

import random

class GuessingGame:
    def __init__(self):
        self.number = random.randint(1, 100)
        self.guesses_taken = 0

    def play(self):
        print("Welcome to the Guessing Game!")
        while True:
            guess = int(input("Enter your guess: "))
            self.guesses_taken += 1

            if guess < self.number:
                print("Too low!")
            elif guess > self.number:
                print("Too high!")
            else:
                print(f"Congratulations! You guessed the number in {self.guesses_taken} tries.")
                break

game = GuessingGame()
game.play()

Projet 3 : Développement d'une application web avec Flask

Dans ce projet, vous allez apprendre à développer une application web en utilisant le framework Flask en Python. Vous allez découvrir comment créer des routes, gérer les requêtes HTTP et afficher des pages web dynamiques.

Voici un exemple de code pour créer une application web simple avec Flask :

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def index():
    return render_template("index.html")

@app.route("/about")
def about():
    return render_template("about.html")

if __name__ == "__main__":
    app.run()
