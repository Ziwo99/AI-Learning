Section 11: Les interfaces graphiques en Python

Introduction aux interfaces graphiques

Les interfaces graphiques permettent aux utilisateurs d'interagir avec des programmes en utilisant des éléments visuels tels que des boutons, des fenêtres et des boîtes de dialogue. En Python, il existe plusieurs bibliothèques populaires pour créer des interfaces graphiques, dont Tkinter, PyQt et wxPython. Dans cette sous-section, nous nous concentrerons sur Tkinter, qui est inclus par défaut dans la distribution standard de Python.

La création d'une interface graphique avec Tkinter

Tkinter est une bibliothèque Python simple et facile à utiliser pour créer des interfaces graphiques. Elle offre une grande variété de widgets (éléments d'interface utilisateur) tels que des boutons, des étiquettes, des zones de texte, des listes déroulantes, etc. Dans cette sous-section, nous apprendrons comment créer une interface graphique de base en utilisant Tkinter.

Voici un exemple de code pour créer une fenêtre avec un bouton en utilisant Tkinter :

import tkinter as tk

def on_button_click():
    print("Le bouton a été cliqué!")

# Création de la fenêtre principale
window = tk.Tk()

# Création d'un bouton
button = tk.Button(window, text="Cliquez-moi!", command=on_button_click)

# Affichage du bouton
button.pack()

# Lancement de la boucle principale de l'interface graphique
window.mainloop()

Dans cet exemple, nous importons le module Tkinter sous le nom "tk" pour faciliter son utilisation. Nous définissons ensuite une fonction "on_button_click" qui sera appelée lorsque le bouton est cliqué. Ensuite, nous créons une fenêtre principale en utilisant la classe "Tk" de Tkinter. Nous créons également un bouton en utilisant la classe "Button" de Tkinter, en spécifiant le texte du bouton et la fonction à appeler lorsqu'il est cliqué. Enfin, nous affichons le bouton en utilisant la méthode "pack" et lançons la boucle principale de l'interface graphique avec la méthode "mainloop".

Ceci est juste un exemple simple pour vous montrer comment créer une interface graphique de base avec Tkinter. Vous pouvez explorer davantage les fonctionnalités de Tkinter pour créer des interfaces graphiques plus complexes et interactives.
