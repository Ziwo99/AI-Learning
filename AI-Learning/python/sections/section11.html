L'accès aux bases de données en Python

Python offre de nombreuses bibliothèques pour accéder et manipuler des bases de données. L'une des bibliothèques les plus populaires est SQLite, qui est intégrée à Python et ne nécessite aucune installation supplémentaire.

Pour utiliser SQLite en Python, vous devez importer le module sqlite3. Voici un exemple de code qui montre comment se connecter à une base de données SQLite :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_données.db')

# Création d'un curseur pour exécuter des requêtes
cur = conn.cursor()

# Exécution d'une requête
cur.execute("SELECT * FROM ma_table")

# Récupération des résultats
results = cur.fetchall()

# Fermeture de la connexion
conn.close()

Les requêtes SQL en Python

Une fois connecté à une base de données, vous pouvez exécuter des requêtes SQL pour récupérer, insérer, mettre à jour ou supprimer des données. Voici quelques exemples de requêtes SQL couramment utilisées en Python :

  • SELECT : permet de récupérer des données à partir d'une table
  • INSERT : permet d'insérer de nouvelles données dans une table
  • UPDATE : permet de mettre à jour des données existantes dans une table
  • DELETE : permet de supprimer des données d'une table

Voici un exemple de code qui montre comment exécuter une requête SELECT en utilisant SQLite :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_données.db')
cur = conn.cursor()

# Exécution d'une requête SELECT
cur.execute("SELECT * FROM ma_table")

# Récupération des résultats
results = cur.fetchall()

# Affichage des résultats
for row in results:
    print(row)

# Fermeture de la connexion
conn.close()

La manipulation des données en Python

En plus d'exécuter des requêtes SQL, Python offre également des fonctionnalités pour manipuler les données récupérées à partir d'une base de données. Vous pouvez effectuer des opérations telles que le tri, le filtrage, l'agrégation et la transformation des données.

Voici un exemple de code qui montre comment trier les résultats d'une requête SELECT en utilisant SQLite :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('ma_base_de_données.db')
cur = conn.cursor()

# Exécution d'une requête SELECT
cur.execute("SELECT * FROM ma_table")

# Récupération des résultats
results = cur.fetchall()

# Tri des résultats par ordre croissant
sorted_results = sorted(results)

# Affichage des résultats triés
for row in sorted_results:
    print(row)

# Fermeture de la connexion
conn.close()

Ce sont là quelques-unes des fonctionnalités de base pour accéder et manipuler des bases de données en Python. Il existe également d'autres bibliothèques populaires telles que MySQLdb pour MySQL, psycopg2 pour PostgreSQL, et pyodbc pour Microsoft SQL Server, qui offrent des fonctionnalités plus avancées et spécifiques à chaque base de données.
