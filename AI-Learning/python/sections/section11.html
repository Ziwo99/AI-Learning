Section 11: Les frameworks en Python

Introduction aux frameworks en Python

Un framework est un ensemble d'outils et de bibliothèques préétablis qui facilitent le développement d'applications. En utilisant un framework, vous pouvez accélérer le processus de développement en réutilisant du code existant et en suivant des conventions établies.

En Python, il existe plusieurs frameworks populaires qui vous permettent de développer rapidement des applications web. Deux des frameworks les plus couramment utilisés sont Flask et Django.

Flask : un framework web en Python

Flask est un framework web léger et flexible qui vous permet de créer des applications web rapidement et facilement. Il est connu pour sa simplicité et sa facilité d'apprentissage, ce qui en fait un excellent choix pour les débutants en développement web.

Voici un exemple de code Flask qui crée une application web simple affichant "Hello, World!" :

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run()

En exécutant ce code, vous pouvez accéder à l'application web en ouvrant votre navigateur et en visitant l'URL "http://localhost:5000/". Vous verrez alors le message "Hello, World!" affiché dans votre navigateur.

Django : un framework web en Python

Django est un framework web complet et puissant qui suit le principe du "batteries included". Il offre de nombreuses fonctionnalités intégrées, telles que l'authentification utilisateur, l'administration de base de données et la gestion des formulaires, ce qui en fait un choix populaire pour les applications web complexes.

Voici un exemple de code Django qui crée une application web simple affichant "Hello, World!" :

from django.http import HttpResponse
from django.urls import path

def hello_world(request):
    return HttpResponse("Hello, World!")

urlpatterns = [
    path('', hello_world),
]

En exécutant ce code, vous pouvez accéder à l'application web en ouvrant votre navigateur et en visitant l'URL "http://localhost:8000/". Vous verrez alors le message "Hello, World!" affiché dans votre navigateur.

En conclusion, Flask et Django sont deux frameworks web populaires en Python. Flask est léger et facile à apprendre, idéal pour les petites applications web, tandis que Django est complet et puissant, adapté aux projets plus complexes. Choisissez le framework qui correspond le mieux à vos besoins et à votre niveau d'expérience en développement web.
