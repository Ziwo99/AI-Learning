Création d'un jeu simple

Dans cette sous-section, nous allons apprendre à créer un jeu simple en utilisant Python. Nous allons utiliser les concepts que nous avons appris jusqu'à présent pour créer un jeu interactif.

Voici un exemple de code pour créer un jeu simple de devinettes :

import random

number = random.randint(1, 10)
guess = 0
tries = 0

print("Bienvenue dans le jeu de devinettes !")
print("Je pense à un nombre entre 1 et 10. Pouvez-vous le deviner ?")

while guess != number:
    guess = int(input("Entrez votre estimation : "))
    tries += 1

    if guess < number:
        print("Trop bas ! Essayez encore.")
    elif guess > number:
        print("Trop haut ! Essayez encore.")

print("Félicitations ! Vous avez deviné le nombre en", tries, "essais.")

Dans cet exemple, nous utilisons le module random pour générer un nombre aléatoire entre 1 et 10. Ensuite, nous demandons à l'utilisateur de deviner le nombre en utilisant la fonction input(). Nous comparons ensuite la supposition de l'utilisateur avec le nombre généré et fournissons des indications appropriées. Le jeu continue jusqu'à ce que l'utilisateur devine correctement le nombre.

Développement d'une application de gestion de tâches

Dans cette sous-section, nous allons apprendre à développer une application de gestion de tâches en utilisant Python. Cette application nous permettra de créer, afficher, mettre à jour et supprimer des tâches.

Voici un exemple de code pour développer une application de gestion de tâches :

tasks = []

def add_task():
    task = input("Entrez la tâche : ")
    tasks.append(task)
    print("Tâche ajoutée avec succès !")

def view_tasks():
    if len(tasks) == 0:
        print("Aucune tâche trouvée.")
    else:
        print("Liste des tâches :")
        for task in tasks:
            print(task)

def update_task():
    task_index = int(input("Entrez l'index de la tâche à mettre à jour : "))
    if task_index < len(tasks):
        new_task = input("Entrez la nouvelle tâche : ")
        tasks[task_index] = new_task
        print("Tâche mise à jour avec succès !")
    else:
        print("Index de tâche invalide.")

def delete_task():
    task_index = int(input("Entrez l'index de la tâche à supprimer : "))
    if task_index < len(tasks):
        tasks.pop(task_index)
        print("Tâche supprimée avec succès !")
    else:
        print("Index de tâche invalide.")

while True:
    print("Que souhaitez-vous faire ?")
    print("1. Ajouter une tâche")
    print("2. Afficher les tâches")
    print("3. Mettre à jour une tâche")
    print("4. Supprimer une tâche")
    print("5. Quitter")

    choice = int(input("Entrez votre choix : "))

    if choice == 1:
        add_task()
    elif choice == 2:
        view_tasks()
    elif choice == 3:
        update_task()
    elif choice == 4:
        delete_task()
    elif choice == 5:
        break
    else:
        print("Choix invalide. Veuillez réessayer.")

Dans cet exemple, nous utilisons une liste pour stocker les tâches. L'utilisateur peut ajouter une tâche, afficher toutes les tâches, mettre à jour une tâche existante ou supprimer une tâche. Le programme continue à afficher un menu jusqu'à ce que l'utilisateur choisisse de quitter.

Création d'un site web dynamique

Dans cette sous-section, nous allons apprendre à créer un site web dynamique en utilisant Python. Nous utiliserons un framework web tel que Django ou Flask pour faciliter le processus de développement.

Voici un exemple de code pour créer un site web dynamique en utilisant Flask :

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/about')
def about():
    return render_template('about.html')

@app.route('/contact')
def contact():
    return render_template('contact.html')

if __name__ == '__main__':
    app.run()

Dans cet exemple, nous utilisons le framework Flask pour créer un site web avec trois pages : index, about et contact. Chaque page est associée à une fonction qui renvoie un modèle HTML correspondant. Les modèles HTML peuvent être créés en utilisant des fichiers séparés et rendus à l'aide de la fonction render_template(). Le serveur web est démarré en exécutant le fichier Python.

Ce sont quelques exemples de projets pratiques que vous pouvez réaliser en utilisant Python. N'hésitez pas à explorer davantage et à développer vos propres projets en utilisant les connaissances que vous avez acquises.
