Projet 1 : Création d'un jeu en Python

Dans ce projet, vous allez créer un jeu en utilisant Python. Vous apprendrez à concevoir et à développer un jeu interactif en utilisant les concepts que vous avez appris jusqu'à présent.

Voici un exemple de code pour créer un jeu simple de devinette de nombre :

import random

number = random.randint(1, 100)
guess = 0
tries = 0

print("Bienvenue dans le jeu de devinette de nombre !")
print("Je pense à un nombre entre 1 et 100. Pouvez-vous le deviner ?")

while guess != number:
    guess = int(input("Entrez votre estimation : "))
    tries += 1

    if guess < number:
        print("Trop bas ! Essayez encore.")
    elif guess > number:
        print("Trop haut ! Essayez encore.")

print("Félicitations ! Vous avez deviné le nombre en", tries, "essais.")

Ce code génère un nombre aléatoire entre 1 et 100, puis demande à l'utilisateur de deviner ce nombre. Le jeu fournit des indications pour aider l'utilisateur à trouver le nombre.

Projet 2 : Développement d'une application web

Dans ce projet, vous allez apprendre à développer une application web en utilisant Python. Vous découvrirez les bases du développement web et comment créer des pages web dynamiques.

Voici un exemple de code pour créer une application web simple en utilisant le framework Flask :

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/about')
def about():
    return render_template('about.html')

if __name__ == '__main__':
    app.run()

Ce code utilise le framework Flask pour créer une application web avec deux pages : la page d'accueil ("/") et la page "about" ("/about"). Les fonctions home() et about() renvoient les templates HTML correspondants.

Projet 3 : Analyse de données avec Python

Dans ce projet, vous allez apprendre à utiliser Python pour analyser des données. Vous découvrirez comment manipuler des fichiers de données, effectuer des opérations statistiques et créer des visualisations.

Voici un exemple de code pour lire un fichier CSV et afficher les données sous forme de graphique à barres :

import pandas as pd
import matplotlib.pyplot as plt

data = pd.read_csv('data.csv')

plt.bar(data['Country'], data['Population'])
plt.xlabel('Pays')
plt.ylabel('Population')
plt.title('Population par pays')

plt.show()

Ce code utilise la bibliothèque pandas pour lire un fichier CSV contenant des données sur la population par pays. Il utilise ensuite la bibliothèque matplotlib pour créer un graphique à barres affichant la population de chaque pays.

Projet 4 : Automatisation de tâches avec Python

Dans ce projet, vous allez apprendre à automatiser des tâches répétitives en utilisant Python. Vous découvrirez comment écrire des scripts qui effectuent des actions spécifiques, comme le traitement de fichiers ou l'envoi de courriels.

Voici un exemple de code pour renommer tous les fichiers d'un dossier en ajoutant un préfixe :

import os

folder = 'chemin/vers/le/dossier'

for filename in os.listdir(folder):
    if not filename.startswith('prefixe_'):
        new_filename = 'prefixe_' + filename
        os.rename(os.path.join(folder, filename), os.path.join(folder, new_filename))

Ce code utilise le module os pour parcourir tous les fichiers d'un dossier et renommer ceux qui ne commencent pas par "prefixe_". Il ajoute simplement le préfixe "prefixe_" au nom de chaque fichier.

Projet 5 : Création d'un chatbot en Python

Dans ce projet, vous allez apprendre à créer un chatbot en utilisant Python. Vous découvrirez comment utiliser des bibliothèques de traitement du langage naturel pour créer un chatbot capable de comprendre et de répondre aux questions des utilisateurs.

Voici un exemple de code pour créer un chatbot simple en utilisant la bibliothèque ChatterBot :

from chatterbot import ChatBot
from chatterbot.trainers import ChatterBotCorpusTrainer

bot = ChatBot('Mon Chatbot')

trainer = ChatterBotCorpusTrainer(bot)
trainer.train('chatterbot.corpus.french')

while True:
    user_input = input("Vous : ")
    response = bot.get_response(user_input)
    print("Chatbot : ", response)

Ce code utilise la bibliothèque ChatterBot pour créer un chatbot qui utilise des réponses pré-enregistrées. Le chatbot apprend à partir du corpus de formation "chatterbot.corpus.french" et peut répondre aux questions de l'utilisateur en fonction de ses connaissances.
