Les opérations sur les chaînes de caractères

Les chaînes de caractères sont des séquences de caractères, et Python offre de nombreuses fonctionnalités pour les manipuler.

Voici quelques opérations courantes sur les chaînes de caractères :

# Création d'une chaîne de caractères
texte = "Bonjour, comment ça va ?"

# Longueur d'une chaîne de caractères
longueur = len(texte)
print("La longueur de la chaîne de caractères est :", longueur)

# Concaténation de chaînes de caractères
nom = "John"
prenom = "Doe"
nom_complet = prenom + " " + nom
print("Le nom complet est :", nom_complet)

# Accès à un caractère spécifique
premier_caractere = texte[0]
print("Le premier caractère est :", premier_caractere)

# Recherche d'un mot dans une chaîne de caractères
mot = "comment"
if mot in texte:
    print("Le mot", mot, "a été trouvé dans la chaîne de caractères.")
else:
    print("Le mot", mot, "n'a pas été trouvé dans la chaîne de caractères.")

# Remplacement d'un mot dans une chaîne de caractères
nouveau_texte = texte.replace("comment", "quel")
print("Le nouveau texte est :", nouveau_texte)

Les expressions régulières

Les expressions régulières sont des motifs utilisés pour rechercher et manipuler des chaînes de caractères selon des règles spécifiques.

Voici comment utiliser les expressions régulières en Python :

import re

# Recherche d'un motif dans une chaîne de caractères
texte = "Bonjour, je m'appelle John Doe."
motif = r"John Doe"
resultat = re.search(motif, texte)
if resultat:
    print("Le motif a été trouvé dans la chaîne de caractères.")
else:
    print("Le motif n'a pas été trouvé dans la chaîne de caractères.")

# Remplacement d'un motif dans une chaîne de caractères
nouveau_texte = re.sub(motif, "Jane Smith", texte)
print("Le nouveau texte est :", nouveau_texte)

Les opérations sur les fichiers CSV et JSON

Python offre des fonctionnalités intégrées pour lire et écrire des fichiers CSV et JSON.

Voici comment manipuler des fichiers CSV et JSON en Python :

# Lecture d'un fichier CSV
import csv

with open('fichier.csv', 'r') as fichier:
    lecteur_csv = csv.reader(fichier)
    for ligne in lecteur_csv:
        print(ligne)

# Écriture dans un fichier CSV
donnees = [['Nom', 'Âge'], ['John Doe', '30'], ['Jane Smith', '25']]
with open('nouveau_fichier.csv', 'w') as fichier:
    ecrivain_csv = csv.writer(fichier)
    ecrivain_csv.writerows(donnees)

# Lecture d'un fichier JSON
import json

with open('fichier.json', 'r') as fichier:
    donnees_json = json.load(fichier)
    print(donnees_json)

# Écriture dans un fichier JSON
donnees = {'nom': 'John Doe', 'age': 30}
with open('nouveau_fichier.json', 'w') as fichier:
    json.dump(donnees, fichier)

Les bases de données et l'accès aux bases de données avec Python

Python offre une variété de bibliothèques pour interagir avec des bases de données, telles que SQLite, MySQL, PostgreSQL, etc.

Voici comment accéder à une base de données et effectuer des opérations courantes en Python :

# Connexion à une base de données SQLite
import sqlite3

connexion = sqlite3.connect('ma_base_de_donnees.db')
curseur = connexion.cursor()

# Création d'une table
curseur.execute("CREATE TABLE IF NOT EXISTS utilisateurs (id INTEGER PRIMARY KEY, nom TEXT, age INTEGER)")

# Insertion de données dans la table
curseur.execute("INSERT INTO utilisateurs (nom, age) VALUES (?, ?)", ('John Doe', 30))
curseur.execute("INSERT INTO utilisateurs (nom, age) VALUES (?, ?)", ('Jane Smith', 25))
connexion.commit()

# Sélection de données dans la table
curseur.execute("SELECT * FROM utilisateurs")
resultat = curseur.fetchall()
for ligne in resultat:
    print(ligne)

# Fermeture de la connexion à la base de données
connexion.close()

Conclusion

Dans cette section, nous avons exploré différentes techniques de manipulation de données avec Python. Nous avons appris à effectuer des opérations sur les chaînes de caractères, à utiliser des expressions régulières, à manipuler des fichiers CSV et JSON, ainsi qu'à accéder et interagir avec des bases de données.

Ces compétences sont essentielles pour travailler avec des données dans de nombreux domaines, tels que l'analyse de données, le développement web et bien d'autres.
