Section 7: Manipulation de fichiers

Lecture et écriture de fichiers texte

La manipulation de fichiers texte est une tâche courante en Python. Pour lire le contenu d'un fichier texte, vous pouvez utiliser la fonction open() avec le mode de lecture approprié. Voici un exemple :

file = open('mon_fichier.txt', 'r')
content = file.read()
file.close()

print(content)

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode lecture ('r'), nous lisons son contenu avec la méthode read() et nous fermons le fichier avec la méthode close(). Ensuite, nous affichons le contenu du fichier.

Pour écrire dans un fichier texte, vous pouvez utiliser la fonction open() avec le mode d'écriture approprié. Voici un exemple :

file = open('mon_fichier.txt', 'w')
file.write('Contenu à écrire dans le fichier')
file.close()

Dans cet exemple, nous ouvrons le fichier "mon_fichier.txt" en mode écriture ('w'), nous écrivons le contenu spécifié avec la méthode write() et nous fermons le fichier avec la méthode close().

Lecture et écriture de fichiers CSV

La manipulation de fichiers CSV (Comma-Separated Values) est courante lors du traitement de données tabulaires. Python fournit le module csv pour faciliter cette tâche. Voici un exemple de lecture d'un fichier CSV :

import csv

with open('mon_fichier.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Dans cet exemple, nous importons le module csv, nous ouvrons le fichier "mon_fichier.csv" en mode lecture ('r') avec l'instruction with open(), nous créons un objet reader à partir du fichier et nous parcourons chaque ligne du fichier avec une boucle for. Nous affichons ensuite chaque ligne.

Pour écrire dans un fichier CSV, vous pouvez utiliser le module csv avec la méthode writer(). Voici un exemple :

import csv

data = [
    ['Nom', 'Âge', 'Ville'],
    ['John Doe', 25, 'Paris'],
    ['Jane Smith', 30, 'Londres']
]

with open('mon_fichier.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerows(data)

Dans cet exemple, nous importons le module csv, nous définissons les données à écrire dans une liste data, nous ouvrons le fichier "mon_fichier.csv" en mode écriture ('w') avec l'instruction with open(), nous créons un objet writer à partir du fichier et nous utilisons la méthode writerows() pour écrire les données dans le fichier.

Lecture et écriture de fichiers JSON

JSON (JavaScript Object Notation) est un format de données couramment utilisé pour l'échange de données. Python fournit le module json pour faciliter la manipulation de fichiers JSON. Voici un exemple de lecture d'un fichier JSON :

import json

with open('mon_fichier.json', 'r') as file:
    data = json.load(file)

print(data)

Dans cet exemple, nous importons le module json, nous ouvrons le fichier "mon_fichier.json" en mode lecture ('r') avec l'instruction with open() et nous utilisons la méthode load() pour charger les données JSON dans la variable data. Ensuite, nous affichons les données.

Pour écrire dans un fichier JSON, vous pouvez utiliser le module json avec la méthode dump(). Voici un exemple :

import json

data = {
    'Nom': 'John Doe',
    'Âge': 25,
    'Ville': 'Paris'
}

with open('mon_fichier.json', 'w') as file:
    json.dump(data, file)

Dans cet exemple, nous importons le module json, nous définissons les données à écrire dans un dictionnaire data, nous ouvrons le fichier "mon_fichier.json" en mode écriture ('w') avec l'instruction with open() et nous utilisons la méthode dump() pour écrire les données dans le fichier.
