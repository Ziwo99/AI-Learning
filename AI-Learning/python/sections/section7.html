Les expressions régulières

Les expressions régulières sont des séquences de caractères utilisées pour rechercher et manipuler des motifs de texte. Elles sont extrêmement puissantes et flexibles, et peuvent être utilisées dans de nombreux cas de traitement de données.

En Python, les expressions régulières sont prises en charge par le module re. Voici un exemple simple d'utilisation des expressions régulières pour rechercher un motif dans une chaîne de caractères :

import re

text = "Bonjour, je m'appelle Alice et j'ai 25 ans."
pattern = r"Alice"

match = re.search(pattern, text)
if match:
    print("Motif trouvé !")
else:
    print("Motif non trouvé.")

Dans cet exemple, nous utilisons la fonction search() du module re pour rechercher le motif "Alice" dans la chaîne de caractères text. Si le motif est trouvé, nous affichons "Motif trouvé !", sinon nous affichons "Motif non trouvé."

Les expressions régulières offrent de nombreuses fonctionnalités avancées, telles que la recherche de motifs complexes, la substitution de motifs, la recherche et le remplacement avec des groupes de capture, etc. Pour en savoir plus sur les expressions régulières en Python, consultez la documentation officielle du module re.

Le traitement de fichiers CSV

Les fichiers CSV (Comma-Separated Values) sont des fichiers texte qui contiennent des données tabulaires, où chaque ligne représente une entrée et chaque colonne représente un champ de données. Python offre des fonctionnalités intégrées pour lire et écrire des fichiers CSV.

Voici un exemple de lecture d'un fichier CSV :

import csv

with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Dans cet exemple, nous utilisons la fonction reader() du module csv pour créer un objet lecteur qui nous permet de parcourir les lignes du fichier CSV. Nous utilisons ensuite une boucle pour afficher chaque ligne.

Python offre également des fonctionnalités pour écrire des données dans un fichier CSV :

import csv

data = [
    ['Nom', 'Âge', 'Ville'],
    ['Alice', 25, 'Paris'],
    ['Bob', 30, 'Londres'],
    ['Charlie', 35, 'New York']
]

with open('data.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerows(data)

Dans cet exemple, nous utilisons la fonction writer() du module csv pour créer un objet écrivain qui nous permet d'écrire les données dans le fichier CSV. Nous utilisons ensuite la méthode writerows() pour écrire toutes les lignes en une seule fois.

Le module csv offre également des fonctionnalités avancées pour spécifier des délimiteurs personnalisés, gérer les en-têtes de colonne, etc. Pour en savoir plus sur le traitement des fichiers CSV en Python, consultez la documentation officielle du module csv.

Le traitement de fichiers JSON

JSON (JavaScript Object Notation) est un format de données largement utilisé pour échanger des données entre un serveur et un client web. Python offre des fonctionnalités intégrées pour lire et écrire des fichiers JSON.

Voici un exemple de lecture d'un fichier JSON :

import json

with open('data.json', 'r') as file:
    data = json.load(file)
    print(data)

Dans cet exemple, nous utilisons la fonction load() du module json pour charger les données du fichier JSON dans un objet Python. Nous utilisons ensuite la fonction print() pour afficher les données.

Python offre également des fonctionnalités pour écrire des données dans un fichier JSON :

import json

data = {
    'nom': 'Alice',
    'age': 25,
    'ville': 'Paris'
}

with open('data.json', 'w') as file:
    json.dump(data, file)

Dans cet exemple, nous utilisons la fonction dump() du module json pour écrire les données dans le fichier JSON.

Le module json offre également des fonctionnalités avancées pour formater les données JSON, gérer les types de données complexes, etc. Pour en savoir plus sur le traitement des fichiers JSON en Python, consultez la documentation officielle du module json.

Le traitement de fichiers XML

XML (eXtensible Markup Language) est un format de données utilisé pour stocker et transporter des informations structurées. Python offre des fonctionnalités intégrées pour lire et écrire des fichiers XML.

Voici un exemple de lecture d'un fichier XML :

import xml.etree.ElementTree as ET

tree = ET.parse('data.xml')
root = tree.getroot()

for child in root:
    print(child.tag, child.attrib)

Dans cet exemple, nous utilisons le module xml.etree.ElementTree pour analyser le fichier XML et obtenir l'élément racine. Nous utilisons ensuite une boucle pour parcourir les enfants de l'élément racine et afficher leurs balises et attributs.

Python offre également des fonctionnalités pour écrire des données dans un fichier XML :

import xml.etree.ElementTree as ET

root = ET.Element('data')
child = ET.SubElement(root, 'item')
child.text = 'Contenu de l\'élément'

tree = ET.ElementTree(root)
tree.write('data.xml')

Dans cet exemple, nous utilisons le module xml.etree.ElementTree pour créer un nouvel élément racine et ajouter un enfant avec du contenu. Nous utilisons ensuite la méthode write() pour écrire l'arbre XML dans le fichier.

Le module xml.etree.ElementTree offre également des fonctionnalités avancées pour manipuler les éléments XML, rechercher des éléments par balise ou attribut, etc. Pour en savoir plus sur le traitement des fichiers XML en Python, consultez la documentation officielle du module xml.etree.ElementTree.

Les bases de données et Python

Python offre une prise en charge intégrée pour interagir avec des bases de données, ce qui permet de stocker et de récupérer des données de manière persistante. Les bases de données relationnelles sont les plus couramment utilisées avec Python, et le module sqlite3 est inclus dans la bibliothèque standard de Python pour interagir avec la base de données SQLite.

Voici un exemple de création d'une base de données SQLite et d'une table :

import sqlite3

conn = sqlite3.connect('example.db')
c = conn.cursor()

c.execute('''CREATE TABLE users
             (id INT PRIMARY KEY NOT NULL,
              name TEXT NOT NULL,
              age INT NOT NULL)''')

conn.commit()
conn.close()

Dans cet exemple, nous utilisons la fonction connect() du module sqlite3 pour établir une connexion à la base de données SQLite. Nous utilisons ensuite la méthode execute() pour exécuter une requête SQL pour créer une table "users" avec des colonnes "id", "name" et "age". Enfin, nous utilisons les méthodes commit() et close() pour valider les modifications et fermer la connexion à la base de données.

Voici un exemple d'insertion de données dans la table :

import sqlite3

conn = sqlite3.connect('example.db')
c = conn.cursor()

c.execute("INSERT INTO users (id, name, age) VALUES (1, 'Alice', 25)")
c.execute("INSERT INTO users (id, name, age) VALUES (2, 'Bob', 30)")
c.execute("INSERT INTO users (id, name, age) VALUES (3, 'Charlie', 35)")

conn.commit()
conn.close()

Dans cet exemple, nous utilisons la méthode execute() pour exécuter des requêtes SQL pour insérer des données dans la table "users".

Python offre également des fonctionnalités pour récupérer des données à partir de la base de données :

import sqlite3

conn = sqlite3.connect('example.db')
c = conn.cursor()

c.execute("SELECT * FROM users")
rows = c.fetchall()

for row in rows:
    print(row)

conn.close()

Dans cet exemple, nous utilisons la méthode execute() pour exécuter une requête SQL pour sélectionner toutes les lignes de la table "users". Nous utilisons ensuite la méthode fetchall() pour récupérer toutes les lignes résultantes et les afficher.

Le module sqlite3 offre également des fonctionnalités avancées pour exécuter des requêtes paramétrées, gérer les transactions, etc. Pour en savoir plus sur l'utilisation des bases de données avec Python, consultez la documentation officielle du module sqlite3.
