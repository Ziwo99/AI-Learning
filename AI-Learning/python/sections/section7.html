Générateurs et itérateurs

Les générateurs et les itérateurs sont des concepts avancés en Python qui permettent de créer des objets qui peuvent être itérés ou générés de manière efficace. Les générateurs sont des fonctions spéciales qui utilisent le mot-clé "yield" pour retourner une valeur à chaque itération, sans avoir à stocker toutes les valeurs en mémoire. Les itérateurs sont des objets qui implémentent les méthodes spéciales "__iter__" et "__next__", permettant de parcourir une séquence d'éléments.

Voici un exemple de générateur qui génère les nombres de la suite de Fibonacci :

def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib = fibonacci()
for i in range(10):
    print(next(fib))

Ce code utilise un générateur pour générer les 10 premiers nombres de la suite de Fibonacci. À chaque itération de la boucle "for", la fonction "next()" est appelée pour obtenir la prochaine valeur du générateur.

Décorateurs

Les décorateurs sont des fonctions qui permettent de modifier le comportement d'une autre fonction sans la modifier directement. Ils sont souvent utilisés pour ajouter des fonctionnalités supplémentaires à une fonction existante, comme la journalisation, la gestion des erreurs, la mise en cache, etc.

Voici un exemple de décorateur qui mesure le temps d'exécution d'une fonction :

import time

def timer(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print("Temps d'exécution :", end_time - start_time, "secondes")
        return result
    return wrapper

@timer
def my_function():
    # Code de la fonction

my_function()

Dans cet exemple, le décorateur "timer" mesure le temps d'exécution de la fonction "my_function". Le décorateur enveloppe la fonction dans une fonction intermédiaire appelée "wrapper", qui mesure le temps avant et après l'exécution de la fonction et affiche le résultat.

Métaclasses

Les métaclasses sont des classes qui définissent le comportement des classes. Elles permettent de contrôler la création et le comportement des instances d'une classe. Les métaclasses sont utilisées pour créer des classes personnalisées avec des fonctionnalités spécifiques, comme la validation des attributs, la gestion des méthodes, etc.

Voici un exemple de métaclass qui ajoute une méthode spéciale "__repr__" à toutes les classes :

class ReprMeta(type):
    def __repr__(cls):
        return f"Classe {cls.__name__}"

class MyClass(metaclass=ReprMeta):
    pass

print(MyClass)

Dans cet exemple, la métaclass "ReprMeta" est définie avec une méthode spéciale "__repr__" qui retourne une représentation en chaîne de caractères de la classe. La classe "MyClass" utilise cette métaclass en spécifiant "metaclass=ReprMeta". Lorsque la classe est imprimée, la méthode "__repr__" de la métaclass est appelée et affiche "Classe MyClass".

Programmation fonctionnelle avec Python

La programmation fonctionnelle est un paradigme de programmation qui se concentre sur l'utilisation de fonctions pures et l'absence de mutations d'état et d'effets de bord. Python prend en charge la programmation fonctionnelle avec des fonctionnalités telles que les fonctions lambda, les fonctions d'ordre supérieur, les fonctions de rappel, etc.

Voici un exemple d'utilisation de la fonction "map()" pour appliquer une fonction à une liste :

def square(x):
    return x ** 2

numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(square, numbers))
print(squared_numbers)

Dans cet exemple, la fonction "square" est définie pour calculer le carré d'un nombre. La fonction "map()" est utilisée pour appliquer la fonction "square" à chaque élément de la liste "numbers". Le résultat est une nouvelle liste "squared_numbers" contenant les carrés des nombres d'origine.
