Section 7: La programmation orientée objet en Python

Introduction à la programmation orientée objet en Python

La programmation orientée objet (POO) est un paradigme de programmation qui permet de structurer et d'organiser le code en utilisant des objets. En Python, tout est un objet, ce qui signifie que chaque élément du programme est une instance d'une classe.

Les principaux concepts de la POO en Python sont les classes, les objets, les méthodes et les attributs. Une classe est un modèle ou un plan pour créer des objets. Un objet est une instance d'une classe. Les méthodes sont des fonctions qui sont définies à l'intérieur d'une classe et qui peuvent être appelées sur les objets de cette classe. Les attributs sont des variables qui sont associées à un objet.

Les classes et les objets en Python

Pour créer une classe en Python, utilisez le mot-clé class suivi du nom de la classe. Les méthodes de la classe sont définies à l'intérieur de cette classe en utilisant le mot-clé def. Pour créer un objet à partir d'une classe, utilisez le nom de la classe suivi de parenthèses.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is " + self.name + " and I am " + str(self.age) + " years old.")

person1 = Person("John", 25)
person1.greet()

Dans cet exemple, nous avons créé une classe Person avec une méthode __init__ qui est appelée lors de la création d'un nouvel objet. La méthode greet affiche un message de salutation en utilisant les attributs name et age de l'objet.

L'héritage en Python

L'héritage est un concept clé de la POO qui permet de créer de nouvelles classes en utilisant des classes existantes. Une classe qui hérite d'une autre classe est appelée une classe dérivée ou une sous-classe, et la classe dont elle hérite est appelée une classe de base ou une superclasse.

En Python, pour créer une classe dérivée, utilisez le nom de la classe de base entre parenthèses après le nom de la classe dérivée. Les méthodes de la classe de base peuvent être appelées sur les objets de la classe dérivée.

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print("The animal speaks.")

class Dog(Animal):
    def speak(self):
        print("The dog barks.")

animal1 = Animal("Generic Animal")
animal1.speak()

dog1 = Dog("Buddy")
dog1.speak()

Dans cet exemple, nous avons une classe de base Animal avec une méthode speak qui affiche un message générique. La classe dérivée Dog a également une méthode speak, mais elle remplace la méthode de la classe de base pour afficher un message spécifique aux chiens.

Les méthodes spéciales en Python

Les méthodes spéciales en Python sont des méthodes prédéfinies qui ont un comportement spécial lorsqu'elles sont appelées. Elles sont souvent utilisées pour effectuer des opérations spécifiques sur les objets, telles que la comparaison, l'ajout ou la soustraction.

Voici quelques-unes des méthodes spéciales les plus couramment utilisées en Python :

  • __init__: Cette méthode est appelée lors de la création d'un nouvel objet.
  • __str__: Cette méthode retourne une représentation sous forme de chaîne de l'objet.
  • __eq__: Cette méthode est utilisée pour comparer deux objets pour l'égalité.
  • __add__: Cette méthode est utilisée pour ajouter deux objets.
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __str__(self):
        return "(" + str(self.x) + ", " + str(self.y) + ")"

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y

    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)

point1 = Point(1, 2)
point2 = Point(3, 4)

print(point1)
print(point1 == point2)
print(point1 + point2)

Dans cet exemple, nous avons une classe Point avec les méthodes spéciales __init__, __str__, __eq__ et __add__. La méthode __str__ retourne une représentation sous forme de chaîne de l'objet, la méthode __eq__ compare deux objets pour l'égalité, et la méthode __add__ ajoute deux objets.
