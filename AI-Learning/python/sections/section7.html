Section 7 : Traitement de données et manipulation de fichiers

Lecture et écriture de fichiers CSV

La manipulation de fichiers CSV est une tâche courante dans le traitement de données en Python. Le format CSV (Comma-Separated Values) est utilisé pour stocker des données tabulaires, où chaque ligne représente un enregistrement et les valeurs sont séparées par des virgules.

Pour lire un fichier CSV, nous pouvons utiliser le module intégré csv. Voici un exemple de code qui lit un fichier CSV et affiche son contenu :

import csv

with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Dans cet exemple, nous ouvrons le fichier CSV en mode lecture ('r') à l'aide de la fonction open(). Ensuite, nous utilisons la fonction csv.reader() pour créer un objet lecteur qui nous permet de parcourir les lignes du fichier. Enfin, nous utilisons une boucle for pour afficher chaque ligne.

Pour écrire dans un fichier CSV, nous pouvons utiliser le module csv de la même manière. Voici un exemple de code qui crée un fichier CSV et y écrit des données :

import csv

data = [
    ['Nom', 'Âge', 'Ville'],
    ['Alice', 25, 'Paris'],
    ['Bob', 30, 'Londres'],
    ['Charlie', 35, 'New York']
]

with open('data.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerows(data)

Dans cet exemple, nous créons une liste de listes appelée data, où chaque liste représente une ligne du fichier CSV. Ensuite, nous ouvrons le fichier CSV en mode écriture ('w') et utilisons la fonction csv.writer() pour créer un objet écrivain. Enfin, nous utilisons la méthode writerows() pour écrire toutes les lignes dans le fichier.

Manipulation de données avec Pandas

Pandas est une bibliothèque populaire pour la manipulation et l'analyse de données en Python. Elle fournit des structures de données puissantes, telles que les DataFrame, qui permettent de manipuler facilement des données tabulaires.

Voici un exemple de code qui utilise Pandas pour lire un fichier CSV et effectuer des opérations de manipulation de données :

import pandas as pd

data = pd.read_csv('data.csv')
print(data.head())  # Affiche les premières lignes du DataFrame

# Filtrer les données
filtered_data = data[data['Âge'] > 30]
print(filtered_data)

# Calculer la moyenne de l'âge
mean_age = data['Âge'].mean()
print(mean_age)

Dans cet exemple, nous utilisons la fonction read_csv() de Pandas pour lire le fichier CSV et créer un objet DataFrame. Ensuite, nous utilisons la méthode head() pour afficher les premières lignes du DataFrame.

Nous pouvons également filtrer les données en utilisant des conditions, comme dans l'exemple ci-dessus où nous filtrons les personnes dont l'âge est supérieur à 30. Enfin, nous utilisons la méthode mean() pour calculer la moyenne de l'âge.

Analyse de données avec Pandas

Pandas offre également des fonctionnalités avancées pour l'analyse de données. Voici un exemple de code qui utilise Pandas pour effectuer une analyse de données sur un ensemble de données :

import pandas as pd

data = pd.read_csv('data.csv')

# Calculer la somme des âges par ville
sum_by_city = data.groupby('Ville')['Âge'].sum()
print(sum_by_city)

# Calculer le nombre de personnes par ville
count_by_city = data['Ville'].value_counts()
print(count_by_city)

# Tracer un graphique à barres des âges par ville
data.groupby('Ville')['Âge'].mean().plot(kind='bar')

Dans cet exemple, nous utilisons la méthode groupby() pour regrouper les données par ville, puis nous utilisons la méthode sum() pour calculer la somme des âges par ville.

Nous pouvons également utiliser la méthode value_counts() pour compter le nombre de personnes par ville. Enfin, nous utilisons la méthode plot() pour tracer un graphique à barres des âges moyens par ville.

Ces exemples ne représentent qu'une petite partie des fonctionnalités offertes par Pandas pour la manipulation et l'analyse de données. Pour en savoir plus, vous pouvez consulter la documentation officielle de Pandas et explorer les nombreux tutoriels et exemples disponibles en ligne.
