Section 7: Les bibliothèques populaires de Python

Numpy

Numpy est une bibliothèque populaire de Python utilisée pour effectuer des opérations mathématiques et statistiques sur des tableaux multidimensionnels. Elle fournit des fonctions efficaces pour la manipulation de données numériques, telles que l'addition, la soustraction, la multiplication, la division, etc.

Voici un exemple d'utilisation de Numpy pour effectuer une opération mathématique simple :

import numpy as np

# Création d'un tableau numpy
arr = np.array([1, 2, 3, 4, 5])

# Addition de 1 à chaque élément du tableau
arr = arr + 1

print(arr)

Ce code crée un tableau numpy contenant les nombres de 1 à 5, puis ajoute 1 à chaque élément du tableau. Le résultat affiché sera [2, 3, 4, 5, 6].

Pandas

Pandas est une bibliothèque Python utilisée pour la manipulation et l'analyse de données. Elle offre des structures de données puissantes, telles que les DataFrames, qui permettent de stocker et de manipuler facilement des données tabulaires.

Voici un exemple d'utilisation de Pandas pour lire un fichier CSV et effectuer des opérations sur les données :

import pandas as pd

# Lecture du fichier CSV
df = pd.read_csv('data.csv')

# Affichage des premières lignes du DataFrame
print(df.head())

Ce code lit un fichier CSV appelé "data.csv" et stocke les données dans un DataFrame. Ensuite, il affiche les premières lignes du DataFrame à l'aide de la fonction head().

Matplotlib

Matplotlib est une bibliothèque de visualisation de données en Python. Elle permet de créer des graphiques et des visualisations de haute qualité pour représenter les données de manière claire et compréhensible.

Voici un exemple d'utilisation de Matplotlib pour tracer un graphique à partir de données :

import matplotlib.pyplot as plt

# Données à tracer
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# Tracé du graphique
plt.plot(x, y)

# Affichage du graphique
plt.show()

Ce code crée un graphique à partir des données fournies dans les listes x et y. Le graphique affiche une ligne reliant les points (1, 2), (2, 4), (3, 6), (4, 8) et (5, 10).

Scikit-learn

Scikit-learn est une bibliothèque Python utilisée pour l'apprentissage automatique (machine learning). Elle fournit des outils et des algorithmes pour effectuer des tâches courantes d'apprentissage automatique, telles que la classification, la régression, le regroupement, etc.

Voici un exemple d'utilisation de Scikit-learn pour effectuer une classification avec l'algorithme des k plus proches voisins :

from sklearn.neighbors import KNeighborsClassifier
from sklearn.datasets import load_iris

# Chargement du jeu de données Iris
iris = load_iris()

# Création du modèle de classification
knn = KNeighborsClassifier(n_neighbors=3)

# Entraînement du modèle
knn.fit(iris.data, iris.target)

# Prédiction d'une nouvelle observation
new_observation = [[5.2, 3.6, 1.4, 0.2]]
prediction = knn.predict(new_observation)

print(prediction)

Ce code utilise l'algorithme des k plus proches voisins pour effectuer une classification sur le jeu de données Iris. Il crée un modèle de classification en utilisant les trois voisins les plus proches, puis entraîne le modèle avec les données d'entrée et les étiquettes correspondantes. Ensuite, il effectue une prédiction sur une nouvelle observation représentée par les valeurs [5.2, 3.6, 1.4, 0.2] et affiche la prédiction.
