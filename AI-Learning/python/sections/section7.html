L'ouverture et la fermeture de fichiers en Python

En Python, pour manipuler des fichiers, nous devons d'abord ouvrir le fichier, effectuer les opérations nécessaires, puis le fermer. Pour ouvrir un fichier, nous utilisons la fonction open(). Cette fonction prend deux paramètres : le nom du fichier et le mode d'ouverture.

Le mode d'ouverture spécifie comment le fichier doit être ouvert. Voici les modes d'ouverture les plus couramment utilisés :

  • 'r' : ouvre le fichier en mode lecture (par défaut)
  • 'w' : ouvre le fichier en mode écriture. Si le fichier existe déjà, son contenu sera écrasé. Sinon, un nouveau fichier sera créé.
  • 'a' : ouvre le fichier en mode ajout. Si le fichier existe déjà, les nouvelles données seront ajoutées à la fin du fichier. Sinon, un nouveau fichier sera créé.
  • 'x' : ouvre le fichier en mode création. Si le fichier existe déjà, une erreur sera levée.

Voici un exemple d'ouverture d'un fichier en mode lecture :

file = open('exemple.txt', 'r')

Une fois que nous avons terminé de travailler avec un fichier, il est important de le fermer en utilisant la méthode close(). Cela permet de libérer les ressources système utilisées par le fichier.

Voici un exemple de fermeture d'un fichier :

file.close()

La lecture et l'écriture de fichiers en Python

Une fois que nous avons ouvert un fichier, nous pouvons effectuer des opérations de lecture ou d'écriture sur celui-ci.

Pour lire le contenu d'un fichier, nous utilisons la méthode read(). Cette méthode renvoie une chaîne de caractères contenant tout le contenu du fichier.

Voici un exemple de lecture d'un fichier :

file = open('exemple.txt', 'r')
content = file.read()
print(content)
file.close()

Pour écrire dans un fichier, nous utilisons la méthode write(). Cette méthode prend une chaîne de caractères en paramètre et écrit cette chaîne dans le fichier.

Voici un exemple d'écriture dans un fichier :

file = open('exemple.txt', 'w')
file.write('Bonjour, monde!')
file.close()

La gestion des erreurs lors de la manipulation de fichiers en Python

Lorsque nous manipulons des fichiers, il est important de prendre en compte les erreurs qui peuvent se produire. Pour cela, nous utilisons des blocs try et except pour capturer et gérer les exceptions.

Voici un exemple de gestion des erreurs lors de l'ouverture d'un fichier :

try:
    file = open('exemple.txt', 'r')
    content = file.read()
    print(content)
    file.close()
except FileNotFoundError:
    print("Le fichier n'a pas été trouvé.")

Dans cet exemple, si le fichier "exemple.txt" n'est pas trouvé, une exception de type FileNotFoundError sera levée. Nous capturons cette exception avec le bloc except et affichons un message d'erreur approprié.

Il est également possible d'utiliser le bloc finally pour exécuter du code qui doit être exécuté, que des exceptions se produisent ou non. Par exemple, nous pouvons utiliser le bloc finally pour nous assurer que le fichier est toujours fermé, même en cas d'erreur :

try:
    file = open('exemple.txt', 'r')
    content = file.read()
    print(content)
except FileNotFoundError:
    print("Le fichier n'a pas été trouvé.")
finally:
    file.close()

De cette façon, même si une exception se produit, le fichier sera toujours fermé correctement.
