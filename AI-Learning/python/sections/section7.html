Manipulation de chaînes de caractères

La manipulation de chaînes de caractères est une tâche courante en Python. Python offre de nombreuses fonctionnalités pour travailler avec des chaînes de caractères, telles que la concaténation, la recherche de sous-chaînes, le remplacement de texte, etc.

Voici quelques exemples de manipulation de chaînes de caractères en Python :

# Concaténation de chaînes de caractères
str1 = "Hello"
str2 = "World"
result = str1 + " " + str2
print(result)  # Affiche "Hello World"

# Recherche de sous-chaînes
str3 = "Bonjour tout le monde"
substring = "tout"
if substring in str3:
    print("La sous-chaîne est présente")
else:
    print("La sous-chaîne est absente")

# Remplacement de texte
str4 = "J'aime les pommes"
new_str = str4.replace("pommes", "bananes")
print(new_str)  # Affiche "J'aime les bananes"

Expressions régulières

Les expressions régulières sont des motifs de recherche utilisés pour trouver des correspondances dans les chaînes de caractères. Elles sont très puissantes et flexibles pour effectuer des opérations de recherche et de remplacement avancées.

Voici un exemple d'utilisation des expressions régulières en Python :

import re

# Recherche de motifs spécifiques
pattern = r"\b\w+@\w+\.\w+\b"  # Motif pour trouver des adresses e-mail
text = "Contactez-moi à l'adresse email@example.com"
matches = re.findall(pattern, text)
for match in matches:
    print(match)  # Affiche "email@example.com"

# Remplacement de motifs
new_text = re.sub(pattern, "EMAIL", text)
print(new_text)  # Affiche "Contactez-moi à l'adresse EMAIL"

Manipulation de fichiers CSV et JSON

Python offre des fonctionnalités intégrées pour lire et écrire des fichiers CSV (Comma-Separated Values) et JSON (JavaScript Object Notation). Ces formats de fichiers sont couramment utilisés pour stocker des données structurées.

Voici un exemple de manipulation de fichiers CSV et JSON en Python :

# Lecture d'un fichier CSV
import csv

with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

# Écriture dans un fichier CSV
data = [['Nom', 'Âge'], ['Alice', '25'], ['Bob', '30']]
with open('data.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerows(data)

# Lecture d'un fichier JSON
import json

with open('data.json', 'r') as file:
    data = json.load(file)
    print(data)

# Écriture dans un fichier JSON
data = {'nom': 'Alice', 'âge': 25}
with open('data.json', 'w') as file:
    json.dump(data, file)

Accès aux bases de données

Python offre une variété de bibliothèques pour interagir avec des bases de données, telles que SQLite, MySQL, PostgreSQL, etc. Ces bibliothèques permettent d'exécuter des requêtes SQL et de manipuler les données de manière efficace.

Voici un exemple d'accès à une base de données SQLite en Python :

import sqlite3

# Connexion à la base de données
conn = sqlite3.connect('example.db')

# Création d'une table
conn.execute('''CREATE TABLE IF NOT EXISTS employees
                (id INT PRIMARY KEY NOT NULL,
                name TEXT NOT NULL,
                age INT NOT NULL);''')

# Insertion de données
conn.execute("INSERT INTO employees (id, name, age) VALUES (1, 'Alice', 25)")
conn.execute("INSERT INTO employees (id, name, age) VALUES (2, 'Bob', 30)")

# Exécution d'une requête
cursor = conn.execute("SELECT * FROM employees")
for row in cursor:
    print(row)

# Fermeture de la connexion
conn.close()

Ce sont là quelques-unes des fonctionnalités de traitement de données disponibles en Python. Avec ces outils, vous pouvez manipuler des chaînes de caractères, utiliser des expressions régulières, travailler avec des fichiers CSV et JSON, et accéder à des bases de données.

La maîtrise de ces concepts vous permettra de traiter efficacement les données dans vos projets Python.
