Syntaxe de base

La syntaxe de base de Swift est similaire à celle d'autres langages de programmation tels que C, C++ et Objective-C. Voici un exemple de code Swift :

let message = "Hello, World!"
print(message)

Dans cet exemple, nous déclarons une constante appelée "message" qui contient la chaîne de caractères "Hello, World!". Ensuite, nous utilisons la fonction "print" pour afficher le contenu de la constante.

Variables et constantes

En Swift, vous pouvez utiliser à la fois des variables et des constantes pour stocker des valeurs. Les variables sont déclarées avec le mot-clé "var" et peuvent être modifiées ultérieurement, tandis que les constantes sont déclarées avec le mot-clé "let" et ne peuvent pas être modifiées une fois qu'elles ont été définies. Voici un exemple :

var age = 25
let name = "John"

age = 26
// name = "Mike" - Cette ligne générera une erreur car "name" est une constante

Dans cet exemple, nous déclarons une variable "age" et une constante "name". Ensuite, nous modifions la valeur de la variable "age" de 25 à 26. Cependant, si nous essayons de modifier la valeur de la constante "name", une erreur sera générée.

Types de données

Swift est un langage fortement typé, ce qui signifie que chaque variable et constante doit avoir un type de données spécifié. Voici quelques types de données couramment utilisés en Swift :

  • Int : représente les nombres entiers
  • Double : représente les nombres à virgule flottante
  • String : représente les chaînes de caractères
  • Bool : représente les valeurs booléennes (true ou false)
  • Array : représente une collection ordonnée d'éléments du même type
  • Dictionary : représente une collection de paires clé-valeur

Voici un exemple montrant comment déclarer des variables avec différents types de données :

var age: Int = 25
var height: Double = 1.75
var name: String = "John"
var isStudent: Bool = true
var numbers: [Int] = [1, 2, 3, 4, 5]
var person: [String: Any] = ["name": "John", "age": 25, "isStudent": true]

Dans cet exemple, nous déclarons des variables avec différents types de données. Notez que nous pouvons également spécifier le type de données lors de la déclaration de la variable.

Opérateurs

Swift prend en charge une variété d'opérateurs pour effectuer des opérations sur les variables et les constantes. Voici quelques-uns des opérateurs les plus couramment utilisés en Swift :

  • Opérateurs arithmétiques : +, -, *, /, %
  • Opérateurs de comparaison : ==, !=, >, <, >=, <=
  • Opérateurs logiques : &&, ||, !
  • Opérateurs d'affectation : =, +=, -=, *=, /=

Voici un exemple montrant comment utiliser ces opérateurs :

let a = 10
let b = 5

let sum = a + b
let difference = a - b
let product = a * b
let quotient = a / b
let remainder = a % b

let isEqual = a == b
let isGreater = a > b
let isLess = a < b

let logicalAnd = isEqual && isGreater
let logicalOr = isEqual || isGreater
let logicalNot = !isEqual

var x = 10
x += 5 // équivaut à x = x + 5

Dans cet exemple, nous effectuons différentes opérations arithmétiques et logiques à l'aide des opérateurs appropriés.

Structures de contrôle (boucles, conditions)

Les structures de contrôle sont utilisées pour contrôler le flux d'exécution d'un programme. En Swift, nous avons plusieurs structures de contrôle, notamment les boucles et les conditions.

Boucles

Les boucles sont utilisées pour répéter une série d'instructions plusieurs fois. Voici les boucles les plus couramment utilisées en Swift :

  • Boucle "for" : utilisée pour répéter un bloc de code un certain nombre de fois
  • Boucle "while" : utilisée pour répéter un bloc de code tant qu'une condition est vraie
  • Boucle "repeat-while" : utilisée pour répéter un bloc de code au moins une fois, puis tant qu'une condition est vraie

Voici un exemple montrant comment utiliser ces boucles :

// Boucle "for"
for i in 1...5 {
    print(i)
}

// Boucle "while"
var j = 1
while j <= 5 {
    print(j)
    j += 1
}

// Boucle "repeat-while"
var k = 1
repeat {
    print(k)
    k += 1
} while k <= 5

Dans cet exemple, nous utilisons les différentes boucles pour afficher les nombres de 1 à 5.

Conditions

Les conditions sont utilisées pour exécuter un bloc de code uniquement si une condition est vraie. Voici les conditions les plus couramment utilisées en Swift :

  • Condition "if" : utilisée pour exécuter un bloc de code si une condition est vraie
  • Condition "if-else" : utilisée pour exécuter un bloc de code si une condition est vraie, sinon exécuter un autre bloc de code
  • Condition "if-else if-else" : utilisée pour exécuter un bloc de code si une condition est vraie, sinon vérifier une autre condition et exécuter un autre bloc de code, et ainsi de suite
  • Opérateur ternaire : utilisé pour exécuter une expression conditionnelle en une seule ligne

Voici un exemple montrant comment utiliser ces conditions :

let number = 10

if number > 0 {
    print("Positive")
} else if number < 0 {
    print("Negative")
} else {
    print("Zero")
}

let result = number > 0 ? "Positive" : "Negative"
print(result)

Dans cet exemple, nous utilisons les différentes conditions pour déterminer si un nombre est positif, négatif ou nul. Nous utilisons également l'opérateur ternaire pour affecter une valeur à la variable "result" en fonction de la condition.

Fonctions et procédures

Les fonctions et les procédures sont utilisées pour regrouper un ensemble d'instructions et les exécuter à partir d'un endroit spécifique dans le code. La différence entre les fonctions et les procédures est que les fonctions renvoient une valeur, tandis que les procédures n'en renvoient pas. Voici un exemple de fonction en Swift :

func greet(name: String) -> String {
    return "Hello, " + name + "!"
}

let message = greet(name: "John")
print(message)

Dans cet exemple, nous définissons une fonction appelée "greet" qui prend un paramètre de type String appelé "name" et renvoie une chaîne de caractères contenant un message de salutation. Ensuite, nous appelons cette fonction avec le nom "John" et stockons le résultat dans la constante "message". Enfin, nous affichons le contenu de la constante.

Les tableaux et les dictionnaires

Les tableaux et les dictionnaires sont utilisés pour stocker des collections d'éléments en Swift.

Tableaux

Un tableau est une collection ordonnée d'éléments du même type. Voici un exemple de tableau en Swift :

var numbers: [Int] = [1, 2, 3, 4, 5]
print(numbers[0]) // Affiche 1

numbers.append(6)
print(numbers) // Affiche [1, 2, 3, 4, 5, 6]

numbers.remove(at: 3)
print(numbers) // Affiche [1, 2, 3, 5, 6]

Dans cet exemple, nous déclarons un tableau appelé "numbers" qui contient les nombres de 1 à 5. Nous pouvons accéder aux éléments du tableau en utilisant leur indice, commençant par 0. Nous pouvons également ajouter de nouveaux éléments à la fin du tableau à l'aide de la méthode "append" et supprimer des éléments à l'aide de la méthode "remove".

Dictionnaires

Un dictionnaire est une collection de paires clé-valeur. Voici un exemple de dictionnaire en Swift :

var person: [String: Any] = ["name": "John", "age": 25, "isStudent": true]
print(person["name"]) // Affiche "John"

person["age"] = 26
print(person) // Affiche ["name": "John", "age": 26, "isStudent": true]

person.removeValue(forKey: "isStudent")
print(person) // Affiche ["name": "John", "age": 26]

Dans cet exemple, nous déclarons un dictionnaire appelé "person" qui contient des informations sur une personne. Nous pouvons accéder aux valeurs du dictionnaire en utilisant leur clé. Nous pouvons également modifier les valeurs existantes en utilisant leur clé ou supprimer des paires clé-valeur à l'aide de la méthode "removeValue".
