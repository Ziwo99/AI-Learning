Création d'une application de gestion de tâches

Dans cette sous-section, nous allons apprendre à créer une application de gestion de tâches en utilisant Swift. Cette application permettra aux utilisateurs de créer, modifier et supprimer des tâches, ainsi que de les marquer comme terminées.

Tout d'abord, nous allons créer une classe Task qui représentera une tâche. Cette classe aura les propriétés suivantes :

  • id : un identifiant unique pour chaque tâche
  • title : le titre de la tâche
  • description : une description détaillée de la tâche
  • isCompleted : un booléen indiquant si la tâche est terminée ou non

Ensuite, nous allons créer une classe TaskManager qui sera responsable de la gestion des tâches. Cette classe aura les méthodes suivantes :

  • addTask(task: Task) : ajoute une tâche à la liste des tâches
  • removeTask(task: Task) : supprime une tâche de la liste des tâches
  • updateTask(task: Task) : met à jour les informations d'une tâche existante
  • completeTask(task: Task) : marque une tâche comme terminée
  • getTasks() : renvoie la liste des tâches

Une fois que nous avons défini nos classes, nous pouvons commencer à créer l'interface utilisateur de notre application. Nous allons utiliser les Storyboards pour cela.

Tout d'abord, nous allons ajouter une table view à notre view controller principal. Cette table view affichera la liste des tâches. Chaque cellule de la table view affichera le titre de la tâche et une icône indiquant si la tâche est terminée ou non.

Ensuite, nous allons ajouter un bouton "Ajouter" en haut à droite de la navigation bar. Lorsque l'utilisateur appuie sur ce bouton, une nouvelle vue s'affiche où il peut saisir les informations de la nouvelle tâche.

Une fois que l'utilisateur a saisi les informations de la tâche et appuyé sur le bouton "Enregistrer", la tâche est ajoutée à la liste des tâches et la table view est mise à jour pour afficher la nouvelle tâche.

Enfin, nous allons ajouter la possibilité de modifier et supprimer une tâche existante. Lorsque l'utilisateur appuie sur une cellule de la table view, une nouvelle vue s'affiche où il peut modifier les informations de la tâche. Il y aura également un bouton "Supprimer" pour supprimer la tâche.

Voici un exemple de code pour la classe Task :

class Task {
    let id: Int
    var title: String
    var description: String
    var isCompleted: Bool
    
    init(id: Int, title: String, description: String, isCompleted: Bool) {
        self.id = id
        self.title = title
        self.description = description
        self.isCompleted = isCompleted
    }
}

Et voici un exemple de code pour la classe TaskManager :

class TaskManager {
    var tasks: [Task] = []
    
    func addTask(task: Task) {
        tasks.append(task)
    }
    
    func removeTask(task: Task) {
        if let index = tasks.firstIndex(where: { $0.id == task.id }) {
            tasks.remove(at: index)
        }
    }
    
    func updateTask(task: Task) {
        if let index = tasks.firstIndex(where: { $0.id == task.id }) {
            tasks[index] = task
        }
    }
    
    func completeTask(task: Task) {
        if let index = tasks.firstIndex(where: { $0.id == task.id }) {
            tasks[index].isCompleted = true
        }
    }
    
    func getTasks() -> [Task] {
        return tasks
    }
}

Création d'un jeu simple

Dans cette sous-section, nous allons créer un jeu simple en utilisant Swift. Le jeu consistera à deviner un nombre aléatoire généré par l'ordinateur.

Tout d'abord, nous allons créer une fonction qui génère un nombre aléatoire entre 1 et 100 :

func generateRandomNumber() -> Int {
    return Int.random(in: 1...100)
}

Ensuite, nous allons créer une boucle qui demande à l'utilisateur de deviner le nombre. À chaque itération de la boucle, nous allons comparer la réponse de l'utilisateur avec le nombre généré aléatoirement.

Si la réponse de l'utilisateur est égale au nombre généré, nous affichons un message de félicitations et sortons de la boucle. Sinon, nous affichons un message indiquant si la réponse de l'utilisateur est trop grande ou trop petite par rapport au nombre généré.

Voici un exemple de code pour le jeu :

let randomNumber = generateRandomNumber()
var guessedNumber: Int?

while guessedNumber != randomNumber {
    print("Devinez le nombre:")
    let input = readLine()
    
    if let number = Int(input ?? "") {
        guessedNumber = number
        
        if guessedNumber == randomNumber {
            print("Félicitations, vous avez deviné le nombre!")
        } else if guessedNumber < randomNumber {
            print("Le nombre est plus grand.")
        } else {
            print("Le nombre est plus petit.")
        }
    } else {
        print("Veuillez entrer un nombre valide.")
    }
}

Ce jeu simple peut être étendu en ajoutant des fonctionnalités telles que le nombre maximum d'essais, un système de points, etc. N'hésitez pas à expérimenter et à ajouter vos propres fonctionnalités au jeu.
