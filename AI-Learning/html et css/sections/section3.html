Section 3 : HTML avancé

Formulaires HTML

Les formulaires HTML permettent aux utilisateurs d'interagir avec un site web en saisissant des informations et en les soumettant. Ils sont utilisés pour collecter des données telles que des noms, des adresses e-mail, des commentaires, etc. Voici un exemple de code HTML pour créer un formulaire simple :

<form>
    <label for="name">Nom :</label>
    <input type="text" id="name" name="name"><br>

    <label for="email">E-mail :</label>
    <input type="email" id="email" name="email"><br>

    <input type="submit" value="Envoyer">
</form>

Dans cet exemple, nous utilisons la balise <form> pour envelopper les éléments du formulaire. Chaque champ de saisie est créé à l'aide de la balise <input>, qui a différents types tels que "text" pour les champs de texte, "email" pour les adresses e-mail, etc. Les balises <label> sont utilisées pour étiqueter chaque champ de saisie. Enfin, la balise <input> de type "submit" est utilisée pour créer un bouton d'envoi.

Médias dans HTML (images, vidéos, audio)

HTML permet d'inclure différents types de médias tels que des images, des vidéos et des fichiers audio dans une page web. Voici un exemple de code HTML pour insérer une image :

<img src="chemin/vers/image.jpg" alt="Description de l'image">

Dans cet exemple, nous utilisons la balise <img> pour insérer une image. L'attribut "src" spécifie le chemin vers l'image, tandis que l'attribut "alt" fournit une description alternative de l'image qui sera affichée si l'image ne peut pas être chargée.

De manière similaire, vous pouvez utiliser les balises <video> et <audio> pour insérer des vidéos et des fichiers audio respectivement. Vous devrez spécifier le chemin vers la vidéo ou le fichier audio à l'aide de l'attribut "src".

Liens et ancres

Les liens HTML permettent aux utilisateurs de naviguer entre différentes pages web. Voici un exemple de code HTML pour créer un lien :

<a href="chemin/vers/page.html">Cliquez ici</a>

Dans cet exemple, nous utilisons la balise <a> pour créer un lien. L'attribut "href" spécifie le chemin vers la page vers laquelle le lien doit pointer. Le texte "Cliquez ici" est le texte qui sera affiché pour le lien.

Vous pouvez également utiliser les ancres pour créer des liens vers des sections spécifiques d'une page. Voici un exemple de code HTML pour créer une ancre :

<a href="#section1">Aller à la section 1</a>

...

<h2 id="section1">Section 1</h2>

Dans cet exemple, nous utilisons l'attribut "href" avec le symbole "#" suivi de l'identifiant de la section vers laquelle nous voulons créer un lien. L'identifiant est spécifié à l'aide de l'attribut "id" de la balise <h2>.

Tableaux HTML

Les tableaux HTML sont utilisés pour organiser les données sous forme de lignes et de colonnes. Voici un exemple de code HTML pour créer un tableau simple :

<table>
    <tr>
        <th>Nom</th>
        <th>Âge</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>30</td>
    </tr>
</table>

Dans cet exemple, nous utilisons la balise <table> pour créer un tableau. Les balises <tr> sont utilisées pour créer des lignes, tandis que les balises <th> et <td> sont utilisées pour créer des cellules d'en-tête et de données respectivement.

Ce sont là quelques-uns des concepts avancés d'HTML que vous pouvez utiliser pour créer des sites web interactifs et attrayants. Continuez à explorer et à expérimenter pour améliorer vos compétences en HTML et CSS !
