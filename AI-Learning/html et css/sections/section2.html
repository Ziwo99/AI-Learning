Section 2 : Mise en forme avec CSS

Introduction à CSS

Le CSS (Cascading Style Sheets) est un langage de feuilles de style utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler l'apparence et la mise en forme des éléments HTML.

Pour utiliser le CSS, vous pouvez soit inclure des styles directement dans votre document HTML en utilisant la balise <style>, soit lier un fichier CSS externe à votre document HTML en utilisant la balise <link>.

Voici un exemple de code CSS :

p {
    color: blue;
    font-size: 18px;
}

Sélecteurs CSS

Les sélecteurs CSS permettent de cibler les éléments HTML auxquels vous souhaitez appliquer des styles. Il existe différents types de sélecteurs :

  • Sélecteur d'élément : cible tous les éléments du type spécifié. Par exemple, p cible tous les paragraphes.
  • Sélecteur de classe : cible les éléments qui ont une classe spécifique. Par exemple, .highlight cible tous les éléments avec la classe "highlight".
  • Sélecteur d'ID : cible un élément avec un ID spécifique. Par exemple, #header cible l'élément avec l'ID "header".
  • Sélecteur d'attribut : cible les éléments qui ont un attribut spécifique. Par exemple, [type="text"] cible tous les éléments avec l'attribut "type" ayant la valeur "text".

Voici un exemple de code CSS utilisant des sélecteurs :

p {
    color: blue;
}

.highlight {
    background-color: yellow;
}

#header {
    font-size: 24px;
}

input[type="text"] {
    border: 1px solid black;
}

Propriétés CSS courantes

Il existe de nombreuses propriétés CSS que vous pouvez utiliser pour modifier l'apparence des éléments HTML. Voici quelques-unes des propriétés les plus couramment utilisées :

  • color : définit la couleur du texte.
  • font-size : définit la taille de la police.
  • background-color : définit la couleur de fond.
  • border : définit le style, la largeur et la couleur de la bordure.
  • margin : définit les marges extérieures de l'élément.
  • padding : définit les marges intérieures de l'élément.

Voici un exemple de code CSS utilisant ces propriétés :

p {
    color: blue;
    font-size: 18px;
    background-color: yellow;
    border: 1px solid black;
    margin: 10px;
    padding: 5px;
}

Box model

Le modèle de boîte (box model) est un concept important en CSS qui définit comment les éléments HTML sont rendus et occupent de l'espace sur la page. Il se compose de quatre parties :

  • Le contenu (content) : c'est le contenu de l'élément, comme du texte ou des images.
  • La marge (margin) : c'est l'espace autour de l'élément, à l'extérieur de la bordure.
  • La bordure (border) : c'est la ligne qui entoure le contenu et la marge.
  • Le rembourrage (padding) : c'est l'espace entre le contenu et la bordure.

Voici un exemple de code CSS montrant comment utiliser le modèle de boîte :

p {
    width: 200px;
    height: 100px;
    margin: 10px;
    padding: 5px;
    border: 1px solid black;
}

Avec ce code, un paragraphe sera affiché avec une largeur de 200 pixels, une hauteur de 100 pixels, une marge de 10 pixels, un rembourrage de 5 pixels et une bordure de 1 pixel solide.
