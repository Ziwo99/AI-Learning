Section 9 : Projets pratiques

Création d'une page web statique

La création d'une page web statique est un excellent moyen de commencer à apprendre HTML et CSS. Une page web statique est une page qui ne change pas en fonction des actions de l'utilisateur. Elle est composée de balises HTML qui structurent le contenu et de règles CSS qui définissent l'apparence de la page.

Voici un exemple de code HTML pour une page web statique simple :

<!DOCTYPE html>
<html>
<head>
    <title>Ma première page web statique</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Bienvenue sur ma page web statique</h1>
    <p>Ceci est un paragraphe de texte.</p>
    <img src="image.jpg" alt="Une image">
</body>
</html>

Et voici un exemple de code CSS pour le fichier "style.css" :

h1 {
    color: blue;
}

p {
    font-size: 16px;
}

img {
    width: 200px;
}

Création d'un formulaire interactif

Un formulaire interactif permet aux utilisateurs de saisir des informations et de les envoyer à un serveur. Les formulaires sont couramment utilisés pour collecter des données, tels que les noms, les adresses e-mail, les commentaires, etc. HTML fournit des balises spécifiques pour créer des formulaires, tandis que CSS peut être utilisé pour styliser les éléments du formulaire.

Voici un exemple de code HTML pour un formulaire simple :

<form action="traitement.php" method="post">
    <label for="nom">Nom :</label>
    <input type="text" id="nom" name="nom"><br>

    <label for="email">Email :</label>
    <input type="email" id="email" name="email"><br>

    <label for="message">Message :</label>
    <textarea id="message" name="message"></textarea><br>

    <input type="submit" value="Envoyer">
</form>

Création d'un site web responsive

Un site web responsive est un site qui s'adapte automatiquement à différents appareils et tailles d'écran, tels que les ordinateurs de bureau, les tablettes et les smartphones. Pour créer un site web responsive, vous pouvez utiliser des techniques CSS telles que les media queries et les grilles CSS.

Voici un exemple de code CSS pour rendre un site web responsive :

@media screen and (max-width: 600px) {
    body {
        font-size: 14px;
    }

    h1 {
        font-size: 24px;
    }
}

@media screen and (min-width: 601px) and (max-width: 1200px) {
    body {
        font-size: 16px;
    }

    h1 {
        font-size: 32px;
    }
}

@media screen and (min-width: 1201px) {
    body {
        font-size: 18px;
    }

    h1 {
        font-size: 48px;
    }
}

En utilisant ces techniques, vous pouvez créer des sites web qui offrent une expérience utilisateur optimale sur tous les appareils.
