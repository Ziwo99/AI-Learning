Section 1 : Introduction à HTML

Les bases de HTML

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises.

Voici un exemple de code HTML de base :

<!DOCTYPE html>
<html>
<head>
    <title>Titre de la page</title>
</head>
<body>
    <h1>Titre principal</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>

Structure d'une page HTML

Une page HTML est composée de plusieurs éléments principaux :

  • <!DOCTYPE html> : Cette déclaration indique au navigateur que le document est écrit en HTML5.
  • <html> : La balise racine de tout document HTML.
  • <head> : La section d'en-tête de la page, qui contient des informations sur le document.
  • <title> : La balise qui définit le titre de la page, qui s'affiche dans l'onglet du navigateur.
  • <body> : La section du corps de la page, qui contient le contenu visible.

Balises HTML courantes

Il existe de nombreuses balises HTML couramment utilisées pour structurer le contenu d'une page. En voici quelques-unes :

  • <h1> à <h6> : Les balises de titre, utilisées pour définir les niveaux de titres.
  • <p> : La balise de paragraphe, utilisée pour afficher du texte.
  • <a> : La balise de lien, utilisée pour créer des liens vers d'autres pages ou ressources.
  • <img> : La balise d'image, utilisée pour afficher des images sur la page.
  • <ul> et <li> : Les balises de liste non ordonnée, utilisées pour créer des listes à puces.
  • <ol> et <li> : Les balises de liste ordonnée, utilisées pour créer des listes numérotées.

Attributs HTML

Les balises HTML peuvent également avoir des attributs, qui fournissent des informations supplémentaires sur les éléments. Voici quelques exemples d'attributs couramment utilisés :

  • class : Permet de spécifier une classe CSS pour un élément.
  • id : Permet de spécifier un identifiant unique pour un élément.
  • src : Utilisé avec la balise <img> pour spécifier l'emplacement de l'image.
  • href : Utilisé avec la balise <a> pour spécifier l'URL de destination du lien.

Ce sont là quelques-uns des concepts de base de HTML et des balises couramment utilisées. En apprenant ces fondamentaux, vous serez en mesure de créer et de structurer du contenu web de manière efficace.
