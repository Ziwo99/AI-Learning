html et css

HTML est un langage de balisage utilisé pour structurer le contenu d'une page web, tandis que CSS est un langage de style utilisé pour définir l'apparence et la mise en page de ces pages. Ensemble, ils permettent de créer des sites web interactifs et esthétiquement attrayants.
