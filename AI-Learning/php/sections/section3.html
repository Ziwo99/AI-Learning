Les tableaux

Les tableaux sont des structures de données qui permettent de stocker et d'organiser plusieurs valeurs sous un même nom. En PHP, les tableaux peuvent contenir des valeurs de différents types tels que des entiers, des chaînes de caractères, des booléens, etc.

Il existe plusieurs façons de créer un tableau en PHP :

// Méthode 1: Utilisation de la fonction array()
$tableau1 = array(1, 2, 3, 4, 5);

// Méthode 2: Utilisation de la syntaxe raccourcie []
$tableau2 = [1, 2, 3, 4, 5];

// Méthode 3: Création d'un tableau vide puis ajout des éléments
$tableau3 = [];
$tableau3[] = 1;
$tableau3[] = 2;
$tableau3[] = 3;
$tableau3[] = 4;
$tableau3[] = 5;

Pour accéder aux éléments d'un tableau, on utilise l'indice correspondant à la position de l'élément dans le tableau. Les indices commencent à 0 :

$tableau = [10, 20, 30, 40, 50];

echo $tableau[0]; // Affiche 10
echo $tableau[2]; // Affiche 30

Il est également possible d'ajouter, de modifier ou de supprimer des éléments d'un tableau :

$tableau = [10, 20, 30, 40, 50];

// Ajouter un élément à la fin du tableau
$tableau[] = 60;

// Modifier un élément du tableau
$tableau[1] = 25;

// Supprimer un élément du tableau
unset($tableau[3]);

Les chaînes de caractères

Les chaînes de caractères sont des séquences de caractères, tels que des lettres, des chiffres ou des symboles. En PHP, les chaînes de caractères peuvent être définies en utilisant des guillemets simples ('') ou des guillemets doubles ("").

Voici quelques exemples de déclaration de chaînes de caractères :

$chaine1 = 'Bonjour';
$chaine2 = "Monde";
$chaine3 = 'Il fait beau aujourd\'hui';
$chaine4 = "J'ai 25 ans";

Il est possible de concaténer des chaînes de caractères en utilisant l'opérateur de concaténation (.) :

$nom = 'John';
$age = 30;

$message = 'Je m\'appelle ' . $nom . ' et j\'ai ' . $age . ' ans.';
echo $message; // Affiche "Je m'appelle John et j'ai 30 ans."

Il existe de nombreuses fonctions prédéfinies en PHP pour manipuler les chaînes de caractères, telles que strlen() pour obtenir la longueur d'une chaîne, strtoupper() pour convertir une chaîne en majuscules, etc.

Les fichiers

La manipulation de fichiers est une tâche courante en programmation. En PHP, il existe plusieurs fonctions pour créer, lire, écrire et supprimer des fichiers.

Pour ouvrir un fichier en lecture :

$fichier = fopen('monfichier.txt', 'r');

Pour lire le contenu d'un fichier ligne par ligne :

while (!feof($fichier)) {
    $ligne = fgets($fichier);
    echo $ligne;
}

Pour écrire dans un fichier :

$fichier = fopen('monfichier.txt', 'w');
fwrite($fichier, 'Contenu à écrire dans le fichier');
fclose($fichier);

Pour supprimer un fichier :

unlink('monfichier.txt');

Les bases de données (MySQL, SQLite)

Les bases de données sont utilisées pour stocker et gérer des données de manière structurée. En PHP, il est possible de se connecter à différentes bases de données, telles que MySQL et SQLite, en utilisant des extensions spécifiques.

Pour se connecter à une base de données MySQL :

$serveur = 'localhost';
$utilisateur = 'root';
$motdepasse = 'monmotdepasse';
$basededonnees = 'ma_base_de_donnees';

$connexion = mysqli_connect($serveur, $utilisateur, $motdepasse, $basededonnees);

if (!$connexion) {
    die('Erreur de connexion : ' . mysqli_connect_error());
}

Pour exécuter une requête SQL :

$requete = 'SELECT * FROM utilisateurs';
$resultat = mysqli_query($connexion, $requete);

while ($ligne = mysqli_fetch_assoc($resultat)) {
    echo $ligne['nom'] . ' ' . $ligne['prenom'];
}

mysqli_close($connexion);

Pour se connecter à une base de données SQLite :

$fichier = 'ma_base_de_donnees.sqlite';

$connexion = new SQLite3($fichier);

if (!$connexion) {
    die('Erreur de connexion : ' . $connexion->lastErrorMsg());
}

Pour exécuter une requête SQL :

$requete = 'SELECT * FROM utilisateurs';
$resultat = $connexion->query($requete);

while ($ligne = $resultat->fetchArray()) {
    echo $ligne['nom'] . ' ' . $ligne['prenom'];
}

$connexion->close();
