Syntaxe de base

La syntaxe de base de PHP est similaire à celle d'autres langages de programmation tels que C, C++ et Java. Voici un exemple de code PHP simple :

<?php
    // Ceci est un commentaire en PHP

    // Déclaration d'une variable
    $nom = "John Doe";

    // Affichage du contenu de la variable
    echo "Bonjour, " . $nom . "!";

    // Fin du script PHP
?>

Dans cet exemple, nous avons utilisé la balise <?php pour indiquer le début du code PHP. Les commentaires en PHP commencent par // et sont ignorés lors de l'exécution du code. La variable $nom est déclarée et contient la valeur "John Doe". Ensuite, nous utilisons la fonction echo pour afficher le message "Bonjour, John Doe!". Enfin, nous utilisons la balise ?> pour indiquer la fin du code PHP.

Variables et types de données

En PHP, les variables sont utilisées pour stocker des valeurs. Voici quelques exemples de déclaration de variables en PHP :

$nom = "John Doe"; // Chaîne de caractères
$age = 25; // Nombre entier
$salaire = 2500.50; // Nombre à virgule flottante
$estEtudiant = true; // Booléen (vrai ou faux)
$listeAmis = array("Alice", "Bob", "Charlie"); // Tableau
$utilisateur = null; // Valeur nulle

En PHP, les variables commencent toujours par le symbole $ suivi du nom de la variable. Les noms de variables sont sensibles à la casse, ce qui signifie que $nom et $Nom sont deux variables différentes. Les types de données en PHP comprennent les chaînes de caractères, les nombres entiers, les nombres à virgule flottante, les booléens, les tableaux et les valeurs nulles.

Opérateurs

Les opérateurs en PHP sont utilisés pour effectuer des opérations sur des variables et des valeurs. Voici quelques exemples d'opérateurs en PHP :

$a = 10;
$b = 5;

// Opérateurs arithmétiques
$addition = $a + $b; // Addition
$soustraction = $a - $b; // Soustraction
$multiplication = $a * $b; // Multiplication
$division = $a / $b; // Division
$modulo = $a % $b; // Modulo

// Opérateurs de comparaison
$estEgal = ($a == $b); // Est égal
$estDifferent = ($a != $b); // Est différent
$estSuperieur = ($a > $b); // Est supérieur
$estInferieur = ($a < $b); // Est inférieur
$estSuperieurOuEgal = ($a >= $b); // Est supérieur ou égal
$estInferieurOuEgal = ($a <= $b); // Est inférieur ou égal

// Opérateurs logiques
$etLogique = ($a > 0 && $b > 0); // ET logique
$ouLogique = ($a > 0 || $b > 0); // OU logique
$nonLogique = !($a > 0); // NON logique

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques telles que l'addition, la soustraction, la multiplication, la division et le modulo. Les opérateurs de comparaison sont utilisés pour comparer des valeurs et renvoyer un résultat booléen (vrai ou faux). Les opérateurs logiques sont utilisés pour combiner des expressions booléennes.

Structures de contrôle (if, else, switch)

Les structures de contrôle en PHP sont utilisées pour exécuter des blocs de code conditionnellement. Voici quelques exemples de structures de contrôle en PHP :

$age = 18;

// Structure if...else
if ($age >= 18) {
    echo "Vous êtes majeur.";
} else {
    echo "Vous êtes mineur.";
}

// Structure switch
$jour = "lundi";

switch ($jour) {
    case "lundi":
        echo "C'est le début de la semaine.";
        break;
    case "vendredi":
        echo "C'est bientôt le week-end.";
        break;
    default:
        echo "C'est un autre jour de la semaine.";
        break;
}

La structure if...else est utilisée pour exécuter un bloc de code si une condition est vraie et un autre bloc de code si la condition est fausse. La structure switch est utilisée pour exécuter différents blocs de code en fonction de la valeur d'une variable.

Boucles (for, while, do-while)

Les boucles en PHP sont utilisées pour répéter l'exécution d'un bloc de code plusieurs fois. Voici quelques exemples de boucles en PHP :

// Boucle for
for ($i = 0; $i < 5; $i++) {
    echo $i;
}

// Boucle while
$i = 0;
while ($i < 5) {
    echo $i;
    $i++;
}

// Boucle do...while
$i = 0;
do {
    echo $i;
    $i++;
} while ($i < 5);

La boucle for est utilisée pour répéter l'exécution d'un bloc de code un nombre spécifié de fois. La boucle while est utilisée pour répéter l'exécution d'un bloc de code tant qu'une condition est vraie. La boucle do...while est similaire à la boucle while, mais elle exécute le bloc de code au moins une fois, même si la condition est fausse.

Fonctions

Les fonctions en PHP sont utilisées pour regrouper un ensemble d'instructions et les exécuter à partir d'un endroit spécifié dans le code. Voici un exemple de déclaration et d'appel de fonction en PHP :

// Déclaration d'une fonction
function direBonjour($nom) {
    echo "Bonjour, " . $nom . "!";
}

// Appel de la fonction
direBonjour("John Doe");

Dans cet exemple, nous avons déclaré une fonction nommée direBonjour qui prend un paramètre $nom. La fonction affiche le message "Bonjour, " suivi du nom passé en paramètre. Ensuite, nous appelons la fonction en passant le nom "John Doe". Le résultat affiché sera "Bonjour, John Doe!".
