Section 5: Les tableaux en JavaScript

Introduction aux tableaux en JavaScript

Un tableau est une structure de données qui permet de stocker et d'organiser plusieurs valeurs dans une seule variable. En JavaScript, un tableau peut contenir des éléments de différents types tels que des nombres, des chaînes de caractères, des objets, etc. Pour créer un tableau, on utilise les crochets [] et on sépare les éléments par des virgules.

var fruits = ['pomme', 'banane', 'orange'];

Dans cet exemple, nous avons créé un tableau appelé "fruits" qui contient trois éléments: 'pomme', 'banane' et 'orange'.

La manipulation des tableaux en JavaScript

Une fois que nous avons créé un tableau, nous pouvons effectuer différentes opérations pour manipuler et accéder à ses éléments. Voici quelques-unes des opérations couramment utilisées :

  • Accéder à un élément spécifique : on utilise l'indice de l'élément dans le tableau pour y accéder. Les indices commencent à zéro.
  • Ajouter un élément à la fin du tableau : on utilise la méthode push().
  • Supprimer le dernier élément du tableau : on utilise la méthode pop().
  • Modifier un élément existant : on utilise l'indice pour accéder à l'élément et on le remplace par une nouvelle valeur.
// Accéder à un élément spécifique
var premierFruit = fruits[0];

// Ajouter un élément à la fin du tableau
fruits.push('raisin');

// Supprimer le dernier élément du tableau
fruits.pop();

// Modifier un élément existant
fruits[1] = 'kiwi';

Les méthodes des tableaux en JavaScript

JavaScript fournit également plusieurs méthodes intégrées pour effectuer des opérations plus avancées sur les tableaux. Voici quelques-unes des méthodes les plus couramment utilisées :

  • La méthode length : elle permet de connaître la taille d'un tableau, c'est-à-dire le nombre d'éléments qu'il contient.
  • La méthode forEach : elle permet d'itérer sur tous les éléments d'un tableau et d'effectuer une action spécifique pour chaque élément.
  • La méthode sort : elle permet de trier les éléments d'un tableau dans l'ordre croissant.
  • La méthode reverse : elle permet d'inverser l'ordre des éléments d'un tableau.
// Connaître la taille d'un tableau
var taille = fruits.length;

// Itérer sur tous les éléments d'un tableau
fruits.forEach(function(fruit) {
  console.log(fruit);
});

// Trier les éléments d'un tableau
fruits.sort();

// Inverser l'ordre des éléments d'un tableau
fruits.reverse();

Les tableaux multidimensionnels en JavaScript

En JavaScript, il est également possible de créer des tableaux multidimensionnels, c'est-à-dire des tableaux qui contiennent d'autres tableaux. Cela peut être utile pour représenter des structures de données plus complexes.

var matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

// Accéder à un élément spécifique dans un tableau multidimensionnel
var element = matrix[1][2];

Dans cet exemple, nous avons créé un tableau multidimensionnel appelé "matrix" qui contient trois tableaux. Chaque tableau représente une ligne de la matrice. Pour accéder à un élément spécifique, nous utilisons deux indices : le premier pour sélectionner la ligne et le deuxième pour sélectionner la colonne.

Les tableaux associatifs en JavaScript

En JavaScript, les tableaux peuvent également être utilisés comme des tableaux associatifs, où les indices ne sont pas nécessairement des nombres, mais peuvent être des chaînes de caractères.

var person = {
  name: 'John',
  age: 30,
  city: 'Paris'
};

// Accéder à une propriété spécifique dans un tableau associatif
var age = person['age'];

Dans cet exemple, nous avons créé un tableau associatif appelé "person" qui contient des propriétés telles que "name", "age" et "city". Pour accéder à une propriété spécifique, nous utilisons la notation entre crochets [] et nous spécifions le nom de la propriété.
