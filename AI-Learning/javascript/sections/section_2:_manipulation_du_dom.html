Section 2: Manipulation du DOM

Section 2: Manipulation du DOM

Introduction au DOM

Le DOM (Document Object Model) est une interface de programmation qui permet de manipuler les éléments d'une page HTML en utilisant JavaScript. Il représente la structure de la page sous forme d'un arbre, où chaque élément est un nœud.

Pour accéder au DOM, on utilise l'objet document. Par exemple, pour sélectionner un élément avec l'ID "myElement", on peut utiliser la méthode getElementById:

var element = document.getElementById("myElement");

Sélection d'éléments du DOM

Pour sélectionner un élément ou plusieurs éléments du DOM, on peut utiliser différentes méthodes :

  • getElementById: sélectionne un élément par son ID.
  • getElementsByClassName: sélectionne les éléments par leur classe.
  • getElementsByTagName: sélectionne les éléments par leur balise.
  • querySelector: sélectionne le premier élément correspondant à un sélecteur CSS.
  • querySelectorAll: sélectionne tous les éléments correspondant à un sélecteur CSS.

Par exemple, pour sélectionner tous les éléments de classe "myClass", on peut utiliser la méthode getElementsByClassName:

var elements = document.getElementsByClassName("myClass");

Manipulation des éléments du DOM

Une fois qu'on a sélectionné un élément du DOM, on peut le manipuler en utilisant différentes propriétés et méthodes :

  • innerHTML: permet de modifier le contenu HTML d'un élément.
  • textContent: permet de modifier le contenu texte d'un élément.
  • setAttribute: permet de définir un attribut sur un élément.
  • style: permet de modifier les styles CSS d'un élément.
  • appendChild: permet d'ajouter un élément enfant à un autre élément.
  • removeChild: permet de supprimer un élément enfant d'un autre élément.

Par exemple, pour modifier le contenu HTML d'un élément avec l'ID "myElement", on peut utiliser la propriété innerHTML:

var element = document.getElementById("myElement");
element.innerHTML = "Nouveau contenu HTML";

Gestion des événements en Javascript

Les événements permettent de réagir aux actions de l'utilisateur, comme un clic sur un bouton ou une soumission de formulaire. En JavaScript, on peut ajouter des gestionnaires d'événements à des éléments du DOM.

Pour ajouter un gestionnaire d'événements à un élément, on utilise la méthode addEventListener. Par exemple, pour ajouter un gestionnaire de clic à un bouton avec l'ID "myButton", on peut faire :

var button = document.getElementById("myButton");
button.addEventListener("click", function() {
  // Code à exécuter lorsque le bouton est cliqué
});

Il existe de nombreux types d'événements, tels que "click", "mouseover", "keydown", etc. On peut également utiliser des fonctions nommées à la place des fonctions anonymes pour les gestionnaires d'événements.
