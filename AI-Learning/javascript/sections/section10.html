Section 10 : Les tests et le débogage en Javascript

Introduction aux tests en Javascript

Les tests en Javascript sont essentiels pour s'assurer du bon fonctionnement de notre code. Ils permettent de vérifier si les différentes parties de notre programme répondent correctement aux spécifications. Les tests peuvent être effectués à différents niveaux, tels que les tests unitaires et les tests d'intégration.

Les tests unitaires en Javascript

Les tests unitaires en Javascript consistent à tester chaque unité de code individuellement, c'est-à-dire chaque fonction ou méthode isolément. Cela permet de s'assurer que chaque unité fonctionne correctement avant de les intégrer ensemble.

Voici un exemple de test unitaire en Javascript :

function add(a, b) {
    return a + b;
}

// Test unitaire pour la fonction add
function testAdd() {
    var result = add(2, 3);
    if (result === 5) {
        console.log("Test passed!");
    } else {
        console.log("Test failed!");
    }
}

// Appel du test unitaire
testAdd();

Les tests d'intégration en Javascript

Les tests d'intégration en Javascript permettent de vérifier si les différentes parties de notre programme fonctionnent correctement lorsqu'elles sont intégrées ensemble. Cela permet de détecter d'éventuels problèmes de compatibilité ou d'interactions entre les différentes unités de code.

Voici un exemple de test d'intégration en Javascript :

function multiply(a, b) {
    return a * b;
}

function subtract(a, b) {
    return a - b;
}

// Test d'intégration pour les fonctions multiply et subtract
function testIntegration() {
    var result1 = multiply(2, 3);
    var result2 = subtract(result1, 1);
    if (result2 === 5) {
        console.log("Test passed!");
    } else {
        console.log("Test failed!");
    }
}

// Appel du test d'intégration
testIntegration();

Le débogage en Javascript

Le débogage en Javascript consiste à identifier et corriger les erreurs dans notre code. Cela peut inclure la recherche de bugs, la compréhension du comportement inattendu du programme et l'optimisation des performances.

Voici un exemple de débogage en Javascript :

function divide(a, b) {
    if (b === 0) {
        console.log("Error: Division by zero!");
        return null;
    }
    return a / b;
}

// Appel de la fonction divide avec des valeurs incorrectes
var result = divide(10, 0);
console.log(result);

Les outils de débogage en Javascript

Il existe plusieurs outils de débogage en Javascript qui facilitent l'identification et la résolution des erreurs. Certains navigateurs web, tels que Google Chrome, proposent des outils de débogage intégrés qui permettent de mettre des points d'arrêt, d'inspecter les variables et d'exécuter le code pas à pas.

Voici un exemple d'utilisation de l'outil de débogage de Google Chrome :

function factorial(n) {
    if (n === 0) {
        return 1;
    }
    return n * factorial(n - 1);
}

// Appel de la fonction factorial avec une valeur incorrecte
var result = factorial(-1);
console.log(result);

Pour utiliser l'outil de débogage de Google Chrome, il suffit d'ouvrir la console de développement (F12), de sélectionner l'onglet "Sources" et de mettre des points d'arrêt à l'endroit souhaité. Ensuite, il est possible d'exécuter le code pas à pas et d'inspecter les variables pour comprendre le comportement du programme.
