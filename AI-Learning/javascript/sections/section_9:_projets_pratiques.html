Projet pratique 1: Création d'un jeu en Javascript

Dans ce projet pratique, nous allons créer un jeu simple en utilisant Javascript. Le jeu consistera à deviner un nombre aléatoire généré par l'ordinateur. Voici les étapes pour créer ce jeu :

1. Générer un nombre aléatoire :

var randomNumber = Math.floor(Math.random() * 100) + 1;

2. Demander à l'utilisateur de deviner le nombre :

var userGuess = prompt("Devinez le nombre entre 1 et 100");

3. Vérifier si la devinette est correcte :

if (userGuess == randomNumber) {
    alert("Bravo, vous avez deviné le nombre !");
} else {
    alert("Désolé, ce n'est pas le bon nombre. Le nombre était " + randomNumber);
}

4. Répéter les étapes 2 et 3 jusqu'à ce que la devinette soit correcte :

while (userGuess != randomNumber) {
    userGuess = prompt("Devinez le nombre entre 1 et 100");
    
    if (userGuess == randomNumber) {
        alert("Bravo, vous avez deviné le nombre !");
    } else {
        alert("Désolé, ce n'est pas le bon nombre. Le nombre était " + randomNumber);
    }
}

5. Féliciter l'utilisateur une fois qu'il a deviné le nombre :

alert("Félicitations, vous avez deviné le nombre !");

Voilà, vous avez créé un jeu de devinette en Javascript ! Amusez-vous bien à jouer avec votre création.

Projet pratique 2: Développement d'une application web en Javascript

Dans ce projet pratique, nous allons développer une application web en utilisant Javascript. L'application consistera en un convertisseur de devises qui permettra à l'utilisateur de convertir une somme d'argent d'une devise à une autre. Voici les étapes pour développer cette application :

1. Créer une interface utilisateur :

<html>
<head>
    <title>Convertisseur de devises</title>
</head>
<body>
    <h1>Convertisseur de devises</h1>
    <label>Montant en devise de départ :</label>
    <input type="number" id="amount" />
    <br />
    <label>Devise de départ :</label>
    <select id="fromCurrency">
        <option value="USD">USD</option>
        <option value="EUR">EUR</option>
        <option value="GBP">GBP</option>
    </select>
    <br />
    <label>Devise de destination :</label>
    <select id="toCurrency">
        <option value="USD">USD</option>
        <option value="EUR">EUR</option>
        <option value="GBP">GBP</option>
    </select>
    <br />
    <button onclick="convertCurrency()">Convertir</button>
    <p id="result"></p>
</body>
</html>

2. Écrire une fonction pour convertir la devise :

function convertCurrency() {
    var amount = document.getElementById("amount").value;
    var fromCurrency = document.getElementById("fromCurrency").value;
    var toCurrency = document.getElementById("toCurrency").value;
    
    // Effectuer la conversion ici
    
    document.getElementById("result").innerHTML = "Le montant converti est : " + convertedAmount + " " + toCurrency;
}

3. Effectuer la conversion en utilisant un taux de change fixe :

var exchangeRates = {
    "USD": {
        "EUR": 0.85,
        "GBP": 0.72
    },
    "EUR": {
        "USD": 1.18,
        "GBP": 0.85
    },
    "GBP": {
        "USD": 1.39,
        "EUR": 1.18
    }
};

var convertedAmount;

if (fromCurrency == toCurrency) {
    convertedAmount = amount;
} else {
    convertedAmount = amount * exchangeRates[fromCurrency][toCurrency];
}

4. Afficher le résultat de la conversion :

document.getElementById("result").innerHTML = "Le montant converti est : " + convertedAmount + " " + toCurrency;

Voilà, vous avez développé une application web de convertisseur de devises en Javascript ! Testez-la en entrant différentes valeurs et devises pour voir les résultats de conversion.
