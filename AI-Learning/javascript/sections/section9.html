Section 9: Projets pratiques en Javascript

Création d'une calculatrice

La création d'une calculatrice en Javascript est un excellent moyen de mettre en pratique vos connaissances en programmation. Une calculatrice basique peut effectuer des opérations simples telles que l'addition, la soustraction, la multiplication et la division.

Voici un exemple de code pour une calculatrice simple en Javascript :

function addition(a, b) {
    return a + b;
}

function soustraction(a, b) {
    return a - b;
}

function multiplication(a, b) {
    return a * b;
}

function division(a, b) {
    return a / b;
}

var resultat = addition(5, 3);
console.log(resultat); // Affiche 8

resultat = soustraction(10, 4);
console.log(resultat); // Affiche 6

resultat = multiplication(2, 6);
console.log(resultat); // Affiche 12

resultat = division(10, 2);
console.log(resultat); // Affiche 5

Développement d'un jeu en ligne

Le développement d'un jeu en ligne est un projet passionnant qui vous permettra d'explorer les fonctionnalités avancées de Javascript. Vous pouvez créer des jeux tels que des jeux de plateforme, des jeux de tir ou des jeux de puzzle.

Voici un exemple de code pour un jeu de plateforme simple en Javascript :

// Définition du personnage
var personnage = {
    x: 0,
    y: 0,
    vitesse: 5,
    sauter: function() {
        console.log("Le personnage saute !");
    },
    deplacer: function(direction) {
        if (direction === "gauche") {
            this.x -= this.vitesse;
        } else if (direction === "droite") {
            this.x += this.vitesse;
        }
    }
};

// Déplacement du personnage
personnage.deplacer("droite");
console.log(personnage.x); // Affiche 5

personnage.deplacer("gauche");
console.log(personnage.x); // Affiche 0

personnage.sauter(); // Affiche "Le personnage saute !"

Construction d'une application de gestion de tâches

La construction d'une application de gestion de tâches est un excellent moyen de mettre en pratique vos compétences en Javascript et de créer une application utile. Une application de gestion de tâches permet aux utilisateurs de créer, modifier et supprimer des tâches.

Voici un exemple de code pour une application de gestion de tâches simple en Javascript :

// Définition de la classe Tache
class Tache {
    constructor(nom, description) {
        this.nom = nom;
        this.description = description;
    }

    afficher() {
        console.log("Nom de la tâche : " + this.nom);
        console.log("Description de la tâche : " + this.description);
    }
}

// Création d'une nouvelle tâche
var tache1 = new Tache("Faire les courses", "Acheter du lait et des œufs");

// Affichage de la tâche
tache1.afficher();

Réalisation d'un site web interactif

La réalisation d'un site web interactif est un projet passionnant qui vous permettra de mettre en pratique vos compétences en Javascript et d'ajouter des fonctionnalités dynamiques à votre site web. Vous pouvez créer des formulaires interactifs, des diaporamas ou des jeux en ligne.

Voici un exemple de code pour un formulaire interactif en Javascript :

// Récupération de l'élément du formulaire
var formulaire = document.getElementById("monFormulaire");

// Ajout d'un écouteur d'événement pour le soumission du formulaire
formulaire.addEventListener("submit", function(event) {
    event.preventDefault(); // Empêche le rechargement de la page

    // Récupération des valeurs des champs du formulaire
    var nom = document.getElementById("nom").value;
    var email = document.getElementById("email").value;

    // Affichage des valeurs dans la console
    console.log("Nom : " + nom);
    console.log("Email : " + email);
});

Ces exemples vous donnent un aperçu des projets pratiques que vous pouvez réaliser en Javascript. N'hésitez pas à explorer davantage et à créer vos propres projets pour approfondir vos connaissances et compétences en programmation Javascript.
