Section 17: JavaScript for Server-Side Development

Introduction to Node.js and server-side JavaScript

Node.js is a runtime environment that allows you to run JavaScript on the server-side. It uses an event-driven, non-blocking I/O model, making it lightweight and efficient for building scalable network applications. With Node.js, you can create web servers, handle HTTP requests, and perform server-side operations using JavaScript.

Building RESTful APIs with Express.js

Express.js is a popular web application framework for Node.js. It provides a set of features and tools for building RESTful APIs quickly and easily. With Express.js, you can define routes, handle HTTP requests and responses, and implement middleware for authentication, error handling, and more. Here's an example of a basic Express.js server: 

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello, World!');
});

app.listen(3000, () => {
  console.log('Server is running on port 3000');
});

Database integration with MongoDB or MySQL

When building server-side applications, you often need to store and retrieve data from a database. Two popular choices for database integration with Node.js are MongoDB and MySQL.

MongoDB is a NoSQL database that stores data in a flexible, JSON-like format called BSON. It is known for its scalability and flexibility, making it a good choice for handling large amounts of data. Here's an example of connecting to a MongoDB database using the official MongoDB driver for Node.js: 

const MongoClient = require('mongodb').MongoClient;

const url = 'mongodb://localhost:27017';
const dbName = 'mydatabase';

MongoClient.connect(url, (err, client) => {
  if (err) throw err;

  const db = client.db(dbName);
  console.log('Connected to MongoDB');

  // Perform database operations here
});

MySQL is a popular relational database management system that uses structured query language (SQL) for managing and manipulating data. It provides a robust and scalable solution for handling structured data. Here's an example of connecting to a MySQL database using the mysql2 package: 

const mysql = require('mysql2');

const connection = mysql.createConnection({
  host: 'localhost',
  user: 'root',
  password: 'password',
  database: 'mydatabase'
});

connection.connect((err) => {
  if (err) throw err;

  console.log('Connected to MySQL');
  
  // Perform database operations here
});

These examples demonstrate how to establish a connection to the respective databases. Once connected, you can perform various operations such as inserting, updating, and querying data.

Note: It's important to handle database connections securely and follow best practices to prevent security vulnerabilities such as SQL injection attacks.
