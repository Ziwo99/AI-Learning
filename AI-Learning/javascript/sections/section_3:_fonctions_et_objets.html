Section 3: Fonctions et Objets

Les fonctions en Javascript

Les fonctions en Javascript sont des blocs de code réutilisables qui peuvent être appelés à partir d'autres parties du programme. Elles permettent d'organiser le code en le divisant en tâches plus petites et plus gérables.

Pour déclarer une fonction en Javascript, utilisez le mot-clé function suivi du nom de la fonction et des parenthèses. Par exemple :

function maFonction() {
    // Code de la fonction
}

Les fonctions peuvent prendre des paramètres, qui sont des valeurs que vous pouvez passer à la fonction lors de son appel. Les paramètres sont spécifiés entre les parenthèses de la déclaration de la fonction. Par exemple :

function addition(a, b) {
    return a + b;
}

Pour appeler une fonction, utilisez simplement son nom suivi de parenthèses. Par exemple :

maFonction();

Les fonctions peuvent également renvoyer une valeur à l'endroit où elles ont été appelées à l'aide du mot-clé return. Par exemple :

var resultat = addition(3, 5);
console.log(resultat); // Affiche 8

Les objets en Javascript

Les objets en Javascript sont des structures de données qui regroupent des propriétés et des méthodes. Les propriétés sont des variables qui contiennent des valeurs, tandis que les méthodes sont des fonctions associées à l'objet.

Pour créer un objet en Javascript, utilisez la notation littérale d'objet. Par exemple :

var personne = {
    nom: "John",
    age: 30,
    direBonjour: function() {
        console.log("Bonjour !");
    }
};

Vous pouvez accéder aux propriétés d'un objet en utilisant la notation pointée. Par exemple :

console.log(personne.nom); // Affiche "John"

Vous pouvez également appeler les méthodes d'un objet en utilisant la notation pointée suivie de parenthèses. Par exemple :

personne.direBonjour(); // Affiche "Bonjour !"

Les prototypes en Javascript

Les prototypes en Javascript sont utilisés pour ajouter de nouvelles propriétés et méthodes à un objet existant. Ils permettent d'étendre les fonctionnalités d'un objet sans avoir à le modifier directement.

Pour créer un prototype en Javascript, utilisez la fonction constructeur et le mot-clé prototype. Par exemple :

function Personne(nom, age) {
    this.nom = nom;
    this.age = age;
}

Personne.prototype.direBonjour = function() {
    console.log("Bonjour !");
};

var personne = new Personne("John", 30);
personne.direBonjour(); // Affiche "Bonjour !"

Les prototypes permettent de créer des objets avec des propriétés et des méthodes communes, tout en économisant de la mémoire en ne dupliquant pas ces propriétés et méthodes pour chaque instance d'objet.

Les classes en Javascript

Les classes en Javascript sont une nouvelle façon d'écrire des objets et des prototypes. Elles fournissent une syntaxe plus claire et plus orientée objet pour la création d'objets.

Pour créer une classe en Javascript, utilisez le mot-clé class. Par exemple :

class Personne {
    constructor(nom, age) {
        this.nom = nom;
        this.age = age;
    }

    direBonjour() {
        console.log("Bonjour !");
    }
}

var personne = new Personne("John", 30);
personne.direBonjour(); // Affiche "Bonjour !"

Les classes en Javascript sont basées sur les prototypes et offrent une syntaxe plus concise pour créer des objets avec des propriétés et des méthodes.
