Section 4 : Les tableaux et les objets en Javascript

Introduction aux tableaux en Javascript

Les tableaux sont des structures de données utilisées pour stocker et organiser des collections d'éléments de données de manière ordonnée. En Javascript, un tableau peut contenir des valeurs de différents types, tels que des nombres, des chaînes de caractères, des objets, etc.

Voici comment déclarer et initialiser un tableau en Javascript :

var fruits = ['pomme', 'banane', 'orange'];

Les éléments d'un tableau sont indexés à partir de zéro. Par exemple, pour accéder au premier élément du tableau "fruits", nous utilisons la syntaxe suivante :

var premierFruit = fruits[0]; // pomme

Manipulation des tableaux en Javascript

Il existe de nombreuses opérations que nous pouvons effectuer sur les tableaux en Javascript. Voici quelques-unes des opérations les plus courantes :

Ajouter un élément à la fin du tableau :

fruits.push('raisin');

Supprimer le dernier élément du tableau :

fruits.pop();

Ajouter un élément au début du tableau :

fruits.unshift('kiwi');

Supprimer le premier élément du tableau :

fruits.shift();

Obtenir la longueur du tableau :

var longueur = fruits.length;

Introduction aux objets en Javascript

Les objets sont des structures de données plus complexes qui permettent de stocker des collections d'informations liées. En Javascript, un objet est composé de paires clé-valeur, où chaque clé est une propriété et chaque valeur est la valeur associée à cette propriété.

Voici comment déclarer et initialiser un objet en Javascript :

var personne = {
    nom: 'Dupont',
    age: 30,
    profession: 'ingénieur'
};

Pour accéder aux propriétés d'un objet, nous utilisons la syntaxe suivante :

var nomPersonne = personne.nom; // Dupont

Manipulation des objets en Javascript

Il existe plusieurs façons de manipuler les objets en Javascript. Voici quelques-unes des opérations les plus courantes :

Ajouter une nouvelle propriété à un objet :

personne.ville = 'Paris';

Modifier la valeur d'une propriété existante :

personne.age = 35;

Supprimer une propriété d'un objet :

delete personne.profession;

Les tableaux d'objets en Javascript

En Javascript, il est également possible de stocker des objets dans un tableau. Cela peut être utile lorsque nous avons besoin de manipuler une collection d'objets similaires.

Voici un exemple de tableau d'objets en Javascript :

var personnes = [
    { nom: 'Dupont', age: 30 },
    { nom: 'Martin', age: 25 },
    { nom: 'Durand', age: 40 }
];

Pour accéder aux propriétés d'un objet dans un tableau, nous utilisons la syntaxe suivante :

var nomPremierePersonne = personnes[0].nom; // Dupont

Il est également possible de parcourir un tableau d'objets à l'aide d'une boucle :

for (var i = 0; i < personnes.length; i++) {
    var nom = personnes[i].nom;
    var age = personnes[i].age;
    console.log(nom + ' a ' + age + ' ans.');
}

Ceci conclut la section sur les tableaux et les objets en Javascript. Vous avez maintenant une compréhension solide de ces concepts et vous êtes prêt à les utiliser dans vos projets Javascript.
