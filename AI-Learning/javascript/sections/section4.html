Section 4: Les fonctions en JavaScript

Introduction aux fonctions en JavaScript

Les fonctions en JavaScript sont des blocs de code réutilisables qui peuvent être appelés pour effectuer une tâche spécifique. Elles permettent d'organiser et de structurer le code, ce qui facilite la maintenance et la réutilisation du code.

Pour déclarer une fonction en JavaScript, utilisez le mot-clé function, suivi du nom de la fonction et des parenthèses. Par exemple:

function maFonction() {
    // Code à exécuter
}

Pour appeler une fonction, utilisez simplement son nom suivi des parenthèses. Par exemple:

maFonction();

Les fonctions anonymes en JavaScript

Les fonctions anonymes en JavaScript sont des fonctions qui n'ont pas de nom. Elles sont souvent utilisées comme arguments pour d'autres fonctions ou pour créer des fonctions auto-exécutables.

Voici un exemple de fonction anonyme utilisée comme argument pour la méthode setTimeout():

setTimeout(function() {
    console.log('Cette fonction sera exécutée après 2 secondes.');
}, 2000);

Les fonctions fléchées en JavaScript

Les fonctions fléchées en JavaScript sont une syntaxe plus concise pour déclarer des fonctions. Elles sont souvent utilisées pour les fonctions anonymes.

Voici un exemple de fonction fléchée:

const maFonction = () => {
    // Code à exécuter
};

Les fonctions fléchées ont une syntaxe plus courte et ne modifient pas la valeur de this lorsqu'elles sont utilisées à l'intérieur d'autres fonctions.

Les fonctions récursives en JavaScript

Les fonctions récursives en JavaScript sont des fonctions qui s'appellent elles-mêmes. Elles sont souvent utilisées pour résoudre des problèmes qui peuvent être divisés en sous-problèmes plus petits.

Voici un exemple de fonction récursive pour calculer le factoriel d'un nombre:

function factoriel(n) {
    if (n === 0) {
        return 1;
    } else {
        return n * factoriel(n - 1);
    }
}

console.log(factoriel(5)); // Affiche 120

Les fonctions de rappel en JavaScript

Les fonctions de rappel en JavaScript sont des fonctions qui sont passées en tant qu'arguments à d'autres fonctions. Elles sont souvent utilisées pour exécuter du code après l'exécution d'une opération asynchrone.

Voici un exemple de fonction de rappel utilisée avec la méthode forEach() d'un tableau:

const nombres = [1, 2, 3, 4, 5];

nombres.forEach(function(nombre) {
    console.log(nombre * 2);
});

Cette fonction de rappel sera exécutée pour chaque élément du tableau nombres et affichera le double de chaque nombre.
