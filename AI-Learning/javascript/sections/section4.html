Les tableaux (création, manipulation, parcours)

Un tableau est une structure de données qui permet de stocker et d'organiser plusieurs valeurs dans une seule variable. En JavaScript, un tableau peut contenir des éléments de différents types de données tels que des nombres, des chaînes de caractères, des objets, etc.

Pour créer un tableau, on utilise la syntaxe suivante :

let tableau = [valeur1, valeur2, valeur3, ...];

Par exemple :

let fruits = ['pomme', 'banane', 'orange'];

Pour accéder à un élément spécifique dans un tableau, on utilise l'indice de l'élément. Les indices commencent à 0 pour le premier élément, 1 pour le deuxième élément, et ainsi de suite.

let premierFruit = fruits[0]; // Accès au premier élément (pomme)
let deuxiemeFruit = fruits[1]; // Accès au deuxième élément (banane)

Pour modifier la valeur d'un élément dans un tableau, on utilise l'indice de l'élément suivi de l'opérateur d'affectation (=).

fruits[2] = 'citron'; // Modification de la valeur du troisième élément

Pour ajouter un nouvel élément à la fin d'un tableau, on utilise la méthode push().

fruits.push('kiwi'); // Ajout du fruit 'kiwi' à la fin du tableau

Pour supprimer le dernier élément d'un tableau, on utilise la méthode pop().

fruits.pop(); // Suppression du dernier élément du tableau

Pour parcourir les éléments d'un tableau, on peut utiliser une boucle for ou une boucle forEach().

// Boucle for
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// Boucle forEach
fruits.forEach(function(fruit) {
    console.log(fruit);
});

Les objets (création, propriétés, méthodes)

Un objet est une collection de propriétés, où chaque propriété est composée d'une clé et d'une valeur. En JavaScript, les objets sont utilisés pour représenter des entités du monde réel, avec leurs caractéristiques et leurs comportements.

Pour créer un objet, on utilise la syntaxe suivante :

let objet = {
    propriete1: valeur1,
    propriete2: valeur2,
    propriete3: valeur3,
    ...
};

Par exemple :

let personne = {
    nom: 'Dupont',
    prenom: 'Jean',
    age: 30,
    adresse: {
        rue: 'Rue du Paradis',
        ville: 'Paris'
    }
};

Pour accéder à une propriété spécifique d'un objet, on utilise la notation pointée (.) ou la notation entre crochets ([]).

let nomPersonne = personne.nom; // Accès à la propriété 'nom' (Dupont)
let prenomPersonne = personne['prenom']; // Accès à la propriété 'prenom' (Jean)

Pour modifier la valeur d'une propriété d'un objet, on utilise la notation pointée ou la notation entre crochets.

personne.age = 35; // Modification de la valeur de la propriété 'age'
personne['adresse']['ville'] = 'Marseille'; // Modification de la valeur de la propriété 'ville' de l'objet 'adresse'

Pour ajouter une nouvelle propriété à un objet, on utilise la notation pointée ou la notation entre crochets.

personne.profession = 'Ingénieur'; // Ajout de la propriété 'profession' à l'objet 'personne'
personne['adresse']['codePostal'] = '75000'; // Ajout de la propriété 'codePostal' à l'objet 'adresse'

Les prototypes et l'héritage

En JavaScript, l'héritage est réalisé à l'aide des prototypes. Un prototype est un objet à partir duquel d'autres objets peuvent hériter des propriétés et des méthodes.

Pour créer un prototype, on utilise la fonction constructeur :

function Personne(nom, age) {
    this.nom = nom;
    this.age = age;
}

Personne.prototype.saluer = function() {
    console.log('Bonjour, je m\'appelle ' + this.nom + ' et j\'ai ' + this.age + ' ans.');
};

let personne1 = new Personne('Dupont', 30);
let personne2 = new Personne('Martin', 25);

personne1.saluer(); // Bonjour, je m'appelle Dupont et j'ai 30 ans.
personne2.saluer(); // Bonjour, je m'appelle Martin et j'ai 25 ans.

Les objets créés à partir d'un prototype héritent de toutes les propriétés et méthodes du prototype. Ils peuvent également ajouter ou modifier leurs propres propriétés et méthodes.

Il est également possible de créer une chaîne d'héritage en utilisant la méthode Object.create().

let animal = {
    manger: function() {
        console.log('Je mange.');
    }
};

let chien = Object.create(animal);
chien.aboyer = function() {
    console.log('Wouaf !');
};

chien.manger(); // Je mange.
chien.aboyer(); // Wouaf !
