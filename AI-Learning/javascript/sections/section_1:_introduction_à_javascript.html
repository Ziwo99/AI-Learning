Section 1: Introduction à Javascript

Introduction à Javascript

JavaScript est un langage de programmation utilisé pour ajouter de l'interactivité aux pages web. Il est principalement utilisé pour créer des fonctionnalités dynamiques, telles que des formulaires interactifs, des animations et des effets visuels. JavaScript est un langage de script interprété, ce qui signifie qu'il est exécuté directement dans le navigateur web du client.

Les bases de la programmation en Javascript

La programmation en JavaScript repose sur les concepts de base suivants :

  • Les variables : utilisées pour stocker des valeurs.
  • Les opérateurs : utilisés pour effectuer des opérations sur les valeurs.
  • Les structures de contrôle : utilisées pour contrôler le flux d'exécution du programme.
  • Les fonctions : utilisées pour regrouper des instructions et les réutiliser.

Les variables et les types de données en Javascript

En JavaScript, les variables sont utilisées pour stocker des données. Il existe plusieurs types de données couramment utilisés :

  • Les nombres : utilisés pour représenter des valeurs numériques.
  • Les chaînes de caractères : utilisées pour représenter du texte.
  • Les booléens : utilisés pour représenter des valeurs vraies ou fausses.
  • Les tableaux : utilisés pour stocker des collections de valeurs.
  • Les objets : utilisés pour représenter des entités complexes avec des propriétés et des méthodes.

Voici un exemple de déclaration de variable en JavaScript :

var age = 25;

Les opérateurs en Javascript

Les opérateurs en JavaScript sont utilisés pour effectuer des opérations sur les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés :

  • L'opérateur d'assignation (=) : utilisé pour attribuer une valeur à une variable.
  • L'opérateur arithmétique (+, -, *, /) : utilisé pour effectuer des opérations mathématiques.
  • L'opérateur de comparaison (==, !=, >, <) : utilisé pour comparer des valeurs.
  • L'opérateur logique (&&, ||, !) : utilisé pour effectuer des opérations logiques.

Voici un exemple d'utilisation de l'opérateur d'assignation :

var x = 5;

Les structures de contrôle en Javascript

Les structures de contrôle en JavaScript permettent de contrôler le flux d'exécution du programme. Les structures de contrôle les plus couramment utilisées sont :

  • Les instructions conditionnelles (if, else if, else) : utilisées pour exécuter des blocs de code en fonction d'une condition.
  • Les boucles (for, while, do while) : utilisées pour répéter des blocs de code plusieurs fois.
  • Les instructions de saut (break, continue) : utilisées pour modifier le flux d'exécution du programme.

Voici un exemple d'utilisation de l'instruction conditionnelle if :

var age = 18;

if (age >= 18) {
  console.log("Vous êtes majeur.");
} else {
  console.log("Vous êtes mineur.");
}
