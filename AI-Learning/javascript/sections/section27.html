Section 27: JavaScript for Web Speech Recognition

Introduction to web speech recognition with JavaScript

Web speech recognition is a technology that allows users to interact with web applications using their voice. With JavaScript, we can leverage the Web Speech API to implement speech recognition capabilities in our web applications.

Using the Web Speech API for speech recognition

The Web Speech API provides a set of interfaces and methods that enable speech recognition in the browser. To use the Web Speech API, we need to create an instance of the SpeechRecognition object and configure it with the desired options. We can then start and stop the recognition process, and handle the recognized speech data. 


// Creating a SpeechRecognition instance
const recognition = new SpeechRecognition();

// Configuring recognition options
recognition.lang = 'en-US'; // Set the language for speech recognition
recognition.continuous = true; // Enable continuous recognition

// Starting the recognition process
recognition.start();

// Handling recognized speech
recognition.onresult = function(event) {
  const transcript = event.results[0][0].transcript;
  console.log('Recognized speech:', transcript);
};

Building voice-controlled applications

With web speech recognition, we can build voice-controlled applications that respond to specific voice commands. By defining a set of commands and associating them with corresponding actions, we can create interactive and hands-free user experiences. 


// Defining voice commands and actions
const commands = {
  'hello': function() {
    console.log('Hello!');
  },
  'open website': function() {
    window.open('https://www.example.com', '_blank');
  },
  'search for *query': function(query) {
    console.log('Searching for:', query);
  }
};

// Handling recognized speech and executing actions
recognition.onresult = function(event) {
  const transcript = event.results[0][0].transcript;
  console.log('Recognized speech:', transcript);

  for (const command in commands) {
    if (transcript.includes(command)) {
      commands[command]();
      break;
    }
  }
};

In the example above, we define three voice commands: "hello", "open website", and "search for *query". When the user speaks any of these commands, the corresponding action is executed. For example, if the user says "hello", the console will log "Hello!". If the user says "search for JavaScript", the console will log "Searching for: JavaScript".

Web speech recognition opens up possibilities for creating innovative and intuitive user interfaces. By leveraging the power of voice, we can enhance the accessibility and usability of our web applications.
