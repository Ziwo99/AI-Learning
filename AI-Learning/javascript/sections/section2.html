Les variables et les types de données

En JavaScript, les variables sont utilisées pour stocker des valeurs. Elles peuvent être déclarées en utilisant le mot-clé var, let ou const. Voici quelques exemples :

var name = 'John';
let age = 25;
const PI = 3.14;

JavaScript prend en charge plusieurs types de données, tels que les chaînes de caractères, les nombres, les booléens, les tableaux, les objets, etc. Voici quelques exemples :

var name = 'John'; // chaîne de caractères
var age = 25; // nombre
var isStudent = true; // booléen
var fruits = ['apple', 'banana', 'orange']; // tableau
var person = { name: 'John', age: 25 }; // objet

Les opérateurs et les expressions

Les opérateurs sont utilisés pour effectuer des opérations sur les valeurs. JavaScript prend en charge les opérateurs arithmétiques, les opérateurs de comparaison, les opérateurs logiques, etc. Voici quelques exemples :

var x = 10;
var y = 5;

var sum = x + y; // addition
var difference = x - y; // soustraction
var product = x * y; // multiplication
var quotient = x / y; // division

var isGreater = x > y; // supérieur à
var isEqual = x === y; // égal à

var logicalAnd = x > 0 && y > 0; // ET logique
var logicalOr = x > 0 || y > 0; // OU logique
var logicalNot = !(x > 0); // NON logique

Les structures de contrôle (boucles, conditions)

Les structures de contrôle permettent d'exécuter des blocs de code conditionnellement ou de manière répétée. JavaScript prend en charge les instructions if, else, else if, les boucles for, while, do while, etc. Voici quelques exemples :

var age = 18;

if (age >= 18) {
    console.log('Vous êtes majeur.');
} else {
    console.log('Vous êtes mineur.');
}

for (var i = 0; i < 5; i++) {
    console.log('Itération ' + i);
}

var j = 0;
while (j < 5) {
    console.log('Itération ' + j);
    j++;
}

var k = 0;
do {
    console.log('Itération ' + k);
    k++;
} while (k < 5);

Les fonctions et les objets

Les fonctions sont des blocs de code réutilisables qui peuvent être appelés avec des arguments. Les objets sont des entités qui regroupent des propriétés et des méthodes. Voici quelques exemples :

function sayHello(name) {
    console.log('Bonjour ' + name + ' !');
}

sayHello('John');

var person = {
    name: 'John',
    age: 25,
    sayHello: function() {
        console.log('Bonjour, je m\'appelle ' + this.name + '.');
    }
};

person.sayHello();

Les tableaux et les objets littéraux

Les tableaux sont utilisés pour stocker plusieurs valeurs dans une seule variable. Les objets littéraux sont des objets définis directement dans le code. Voici quelques exemples :

var fruits = ['apple', 'banana', 'orange'];

console.log(fruits[0]); // affiche 'apple'
console.log(fruits.length); // affiche 3

var person = {
    name: 'John',
    age: 25,
    hobbies: ['reading', 'playing guitar', 'swimming']
};

console.log(person.name); // affiche 'John'
console.log(person.hobbies[0]); // affiche 'reading'
