Syntaxe de base

La syntaxe de base de Javascript est assez simple et ressemble à celle des autres langages de programmation. Voici un exemple de code Javascript :

let x = 5;
let y = 10;
let z = x + y;

console.log(z); // Affiche 15 dans la console

Dans cet exemple, nous déclarons trois variables (x, y et z) et effectuons une opération d'addition. Le résultat est ensuite affiché dans la console à l'aide de la fonction console.log().

Variables et types de données

En Javascript, les variables sont utilisées pour stocker des valeurs. Il existe plusieurs types de données que l'on peut assigner à une variable :

  • Nombre (number) : utilisé pour stocker des valeurs numériques, par exemple 5 ou 3.14.
  • Chaîne de caractères (string) : utilisée pour stocker du texte, par exemple "Bonjour" ou 'Javascript'.
  • Booléen (boolean) : utilisé pour stocker des valeurs de vérité, soit true (vrai) soit false (faux).
  • Tableau (array) : utilisé pour stocker plusieurs valeurs dans une seule variable.
  • Objet (object) : utilisé pour stocker des données structurées, avec des propriétés et des méthodes.
  • Null : utilisé pour représenter l'absence de valeur.
  • Undefined : utilisé pour représenter une variable qui n'a pas été assignée.

Voici un exemple montrant comment déclarer et utiliser différentes variables :

let age = 25; // Nombre
let nom = "Jean"; // Chaîne de caractères
let estMajeur = true; // Booléen

let fruits = ["pomme", "banane", "orange"]; // Tableau
console.log(fruits[0]); // Affiche "pomme" dans la console

let personne = { nom: "Jean", age: 25 }; // Objet
console.log(personne.nom); // Affiche "Jean" dans la console

let valeurNull = null; // Null
let valeurNonAssignee; // Undefined

Opérateurs

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés en Javascript :

  • Opérateurs arithmétiques : + (addition), - (soustraction), * (multiplication), / (division), % (modulo).
  • Opérateurs d'affectation : = (assignation), += (addition et assignation), -= (soustraction et assignation), *= (multiplication et assignation), /= (division et assignation), %= (modulo et assignation).
  • Opérateurs de comparaison : == (égal à), != (différent de), === (égal à en termes de valeur et de type), !== (différent de en termes de valeur et de type), > (supérieur à), < (inférieur à), >= (supérieur ou égal à), <= (inférieur ou égal à).
  • Opérateurs logiques : && (et logique), || (ou logique), ! (non logique).

Voici un exemple montrant comment utiliser certains de ces opérateurs :

let x = 5;
let y = 10;

let addition = x + y; // Addition
let soustraction = x - y; // Soustraction
let multiplication = x * y; // Multiplication
let division = x / y; // Division
let modulo = x % y; // Modulo

console.log(addition); // Affiche 15 dans la console
console.log(soustraction); // Affiche -5 dans la console
console.log(multiplication); // Affiche 50 dans la console
console.log(division); // Affiche 0.5 dans la console
console.log(modulo); // Affiche 5 dans la console

let a = 5;
let b = 10;
let c = 15;

let estSuperieur = a > b; // Est-ce que a est supérieur à b ?
let estInferieurOuEgal = b <= c; // Est-ce que b est inférieur ou égal à c ?

console.log(estSuperieur); // Affiche false dans la console
console.log(estInferieurOuEgal); // Affiche true dans la console

let estVrai = true;
let estFaux = false;

let etLogique = estVrai && estFaux; // Est-ce que estVrai ET estFaux ?
let ouLogique = estVrai || estFaux; // Est-ce que estVrai OU estFaux ?
let nonLogique = !estVrai; // Est-ce que NON estVrai ?

console.log(etLogique); // Affiche false dans la console
console.log(ouLogique); // Affiche true dans la console
console.log(nonLogique); // Affiche false dans la console

Structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme en fonction de certaines conditions. Les boucles et les conditions sont les structures de contrôle les plus couramment utilisées en Javascript.

Boucles

Les boucles permettent d'exécuter une série d'instructions plusieurs fois. Voici les boucles les plus couramment utilisées en Javascript :

  • Boucle while : exécute un bloc d'instructions tant qu'une condition est vraie.
  • Boucle for : exécute un bloc d'instructions un certain nombre de fois.
  • Boucle do...while : exécute un bloc d'instructions au moins une fois, puis tant qu'une condition est vraie.

Voici un exemple montrant comment utiliser une boucle while :

let i = 0;

while (i < 5) {
  console.log(i);
  i++;
}

Cet exemple affiche les nombres de 0 à 4 dans la console.

Conditions

Les conditions permettent d'exécuter certaines instructions uniquement si une condition est vraie. Voici les conditions les plus couramment utilisées en Javascript :

  • if...else : exécute un bloc d'instructions si une condition est vraie, sinon exécute un autre bloc d'instructions.
  • switch : exécute un bloc d'instructions en fonction de la valeur d'une variable.

Voici un exemple montrant comment utiliser une condition if...else :

let age = 18;

if (age >= 18) {
  console.log("Vous êtes majeur.");
} else {
  console.log("Vous êtes mineur.");
}

Cet exemple affiche "Vous êtes majeur." si l'âge est supérieur ou égal à 18, sinon il affiche "Vous êtes mineur.".

Fonctions

Les fonctions sont des blocs de code réutilisables qui peuvent être appelés à partir d'autres parties du programme. Elles permettent d'organiser le code en le divisant en tâches plus petites et plus gérables.

Voici un exemple montrant comment déclarer et utiliser une fonction :

function direBonjour() {
  console.log("Bonjour !");
}

direBonjour(); // Appelle la fonction et affiche "Bonjour !" dans la console

Dans cet exemple, nous déclarons une fonction appelée direBonjour() qui affiche "Bonjour !" dans la console. Nous appelons ensuite cette fonction pour l'exécuter.

Les fonctions peuvent également prendre des paramètres, qui sont des valeurs que l'on peut passer à la fonction lors de son appel. Voici un exemple montrant comment déclarer une fonction avec des paramètres :

function direBonjour(nom) {
  console.log("Bonjour " + nom + " !");
}

direBonjour("Jean"); // Appelle la fonction avec le paramètre "Jean" et affiche "Bonjour Jean !" dans la console

Dans cet exemple, nous déclarons une fonction appelée direBonjour() qui prend un paramètre nom. Lorsque nous appelons cette fonction avec le paramètre "Jean", elle affiche "Bonjour Jean !" dans la console.

Les fonctions peuvent également renvoyer une valeur à partir du bloc de code. Voici un exemple montrant comment déclarer une fonction avec une valeur de retour :

function additionner(a, b) {
  return a + b;
}

let resultat = additionner(5, 10); // Appelle la fonction avec les paramètres 5 et 10, et assigne le résultat à la variable resultat
console.log(resultat); // Affiche 15 dans la console

Dans cet exemple, nous déclarons une fonction appelée additionner() qui prend deux paramètres a et b. La fonction renvoie la somme de ces deux paramètres. Lorsque nous appelons cette fonction avec les paramètres 5 et 10, elle renvoie 15, que nous assignons à la variable resultat. Nous affichons ensuite cette variable dans la console.
