Section 2: Manipulation du DOM avec JavaScript

Introduction à la manipulation du DOM

La manipulation du DOM (Document Object Model) est une technique essentielle en JavaScript pour interagir avec les éléments HTML d'une page web. Elle permet de modifier dynamiquement le contenu, la structure et le style des éléments.

Pour commencer à manipuler le DOM, il est important de comprendre que chaque élément HTML est représenté par un nœud dans une structure arborescente. Le nœud racine est l'élément document, qui représente l'ensemble de la page.

Voici un exemple simple pour accéder à un élément du DOM :

let element = document.getElementById('monElement');

Dans cet exemple, la méthode getElementById permet de sélectionner un élément en utilisant son identifiant unique. L'élément sélectionné est ensuite stocké dans la variable element pour une utilisation ultérieure.

Sélection d'éléments du DOM

Il existe plusieurs méthodes pour sélectionner des éléments du DOM en JavaScript. Voici quelques-unes des méthodes les plus couramment utilisées :

  • getElementById(id) : sélectionne un élément en utilisant son identifiant unique.
  • getElementsByClassName(className) : sélectionne tous les éléments ayant une classe spécifique.
  • getElementsByTagName(tagName) : sélectionne tous les éléments ayant une balise HTML spécifique.
  • querySelector(selector) : sélectionne le premier élément correspondant à un sélecteur CSS spécifique.
  • querySelectorAll(selector) : sélectionne tous les éléments correspondant à un sélecteur CSS spécifique.

Voici un exemple d'utilisation de la méthode querySelector :

let element = document.querySelector('.maClasse');

Dans cet exemple, la méthode querySelector sélectionne le premier élément correspondant à la classe CSS spécifiée et le stocke dans la variable element.

Modification du contenu du DOM

Une fois que vous avez sélectionné un élément du DOM, vous pouvez modifier son contenu en utilisant différentes propriétés et méthodes. Voici quelques exemples courants :

  • innerHTML : permet de modifier le contenu HTML d'un élément.
  • textContent : permet de modifier le contenu textuel d'un élément.
  • setAttribute(name, value) : permet de définir la valeur d'un attribut spécifique.

Voici un exemple d'utilisation de la propriété innerHTML :

let element = document.getElementById('monElement');
element.innerHTML = 'Nouveau contenu';

Dans cet exemple, la propriété innerHTML est utilisée pour modifier le contenu HTML de l'élément sélectionné.

Ajout et suppression d'éléments du DOM

En plus de modifier le contenu des éléments existants, vous pouvez également ajouter de nouveaux éléments ou supprimer des éléments existants du DOM. Voici quelques méthodes couramment utilisées :

  • createElement(tagName) : crée un nouvel élément avec la balise HTML spécifiée.
  • appendChild(node) : ajoute un nœud en tant que dernier enfant d'un élément.
  • removeChild(node) : supprime un nœud enfant d'un élément.

Voici un exemple d'ajout d'un nouvel élément :

let newElement = document.createElement('div');
newElement.textContent = 'Nouvel élément';
document.body.appendChild(newElement);

Dans cet exemple, un nouvel élément <div> est créé, son contenu textuel est défini, puis il est ajouté en tant que dernier enfant de l'élément <body> de la page.

Gestion des événements du DOM

Les événements du DOM permettent de détecter et de réagir aux actions de l'utilisateur, telles que le clic sur un bouton ou la soumission d'un formulaire. Voici quelques exemples courants d'événements du DOM :

  • click : se déclenche lorsqu'un élément est cliqué.
  • submit : se déclenche lorsqu'un formulaire est soumis.
  • keydown : se déclenche lorsqu'une touche du clavier est enfoncée.

Pour gérer un événement, vous pouvez utiliser la méthode addEventListener. Voici un exemple :

let button = document.getElementById('monBouton');
button.addEventListener('click', function() {
  alert('Le bouton a été cliqué !');
});

Dans cet exemple, la méthode addEventListener est utilisée pour détecter le clic sur le bouton et exécuter une fonction qui affiche une boîte de dialogue d'alerte.
