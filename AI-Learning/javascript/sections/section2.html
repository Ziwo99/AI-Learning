Section 2: JavaScript Basics

Variables and data types

In JavaScript, variables are used to store data. They can hold different types of data, such as numbers, strings, booleans, and more. To declare a variable, you can use the var, let, or const keyword.

Here's an example:

var name = 'John';
let age = 25;
const isStudent = true;

In this example, we declare three variables: name, age, and isStudent. The var keyword is used for variable declaration in older versions of JavaScript, while let and const are introduced in newer versions.

Variables declared with var and let can be reassigned, while variables declared with const are read-only and cannot be reassigned.

Operators and expressions

Operators are used to perform operations on variables and values. JavaScript supports various types of operators, such as arithmetic, assignment, comparison, logical, and more.

Here are some examples:

var x = 5;
var y = 10;
var sum = x + y;

var isGreater = x > y;

var result = (x * y) / 2;

In this example, we use the addition operator (+) to calculate the sum of x and y. We also use the greater than operator (>) to compare x and y, and the multiplication and division operators to calculate the result.

Control flow and loops

Control flow allows you to control the execution of your code based on certain conditions. JavaScript provides various control flow statements, such as if, else, switch, and loops like for and while.

Here's an example of using an if statement:

var age = 18;

if (age >= 18) {
  console.log('You are an adult.');
} else {
  console.log('You are a minor.');
}

In this example, we check if the age variable is greater than or equal to 18. If it is, we log 'You are an adult.' to the console. Otherwise, we log 'You are a minor.'

Loops are used to repeat a block of code multiple times. Here's an example of using a for loop:

for (var i = 0; i < 5; i++) {
  console.log('Iteration ' + i);
}

In this example, the loop will iterate five times. In each iteration, it logs 'Iteration' followed by the value of i to the console.

Functions and scope

Functions are reusable blocks of code that perform a specific task. They can take input, called parameters, and return a value. JavaScript functions can be declared using the function keyword.

Here's an example of a function:

function greet(name) {
  console.log('Hello, ' + name + '!');
}

greet('John');

In this example, we declare a function called greet that takes a parameter name. Inside the function, we log 'Hello, ' followed by the value of name to the console. We then call the function and pass 'John' as the argument.

Scope refers to the visibility and accessibility of variables. JavaScript has function scope, which means that variables declared inside a function are only accessible within that function.

Arrays and objects

Arrays and objects are used to store multiple values in a single variable. Arrays are ordered lists of values, while objects are collections of key-value pairs.

Here's an example of an array:

var fruits = ['apple', 'banana', 'orange'];

console.log(fruits[0]); // Output: 'apple'

In this example, we declare an array called fruits that contains three elements. We can access individual elements using their index, starting from 0.

Here's an example of an object:

var person = {
  name: 'John',
  age: 25,
  isStudent: true
};

console.log(person.name); // Output: 'John'

In this example, we declare an object called person with three properties: name, age, and isStudent. We can access the values of these properties using dot notation.
