Section 2. Les bases de Javascript

Retour au cours

Syntaxe de base

La syntaxe de base de Javascript est assez similaire à celle des autres langages de programmation. Voici un exemple de code Javascript :

let message = 'Bonjour, monde!';
console.log(message);

Dans cet exemple, nous utilisons la déclaration de variable 'let' pour créer une variable appelée 'message' et lui attribuer la valeur 'Bonjour, monde!'. Ensuite, nous utilisons la fonction 'console.log()' pour afficher le contenu de la variable dans la console du navigateur.

Variables et types de données

En Javascript, il existe plusieurs types de données que vous pouvez utiliser pour stocker des informations. Voici quelques-uns des types de données les plus couramment utilisés :

  • Nombre : utilisé pour stocker des valeurs numériques, par exemple 42 ou 3.14.
  • Chaîne de caractères : utilisée pour stocker du texte, par exemple 'Bonjour' ou "Javascript".
  • Booléen : utilisé pour stocker des valeurs de vérité, soit true (vrai) soit false (faux).
  • Tableau : utilisé pour stocker une collection ordonnée de valeurs, par exemple [1, 2, 3] ou ['rouge', 'vert', 'bleu'].
  • Objet : utilisé pour stocker des collections de propriétés et de méthodes, par exemple {nom: 'John', age: 30}.
  • Null : utilisé pour représenter l'absence de valeur.
  • Undefined : utilisé pour représenter une variable qui n'a pas été définie.

Pour déclarer une variable en Javascript, vous pouvez utiliser les mots-clés 'var', 'let' ou 'const'. Par exemple :

let age = 25;
const pi = 3.14;
var nom = 'John';

Dans cet exemple, nous déclarons une variable 'age' avec la valeur 25, une constante 'pi' avec la valeur 3.14, et une variable 'nom' avec la valeur 'John'.

Opérateurs

Les opérateurs en Javascript sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés :

  • Opérateurs arithmétiques : utilisés pour effectuer des opérations mathématiques, par exemple + (addition), - (soustraction), * (multiplication), / (division).
  • Opérateurs d'affectation : utilisés pour attribuer une valeur à une variable, par exemple = (affectation simple), += (addition et affectation), -= (soustraction et affectation).
  • Opérateurs de comparaison : utilisés pour comparer des valeurs, par exemple == (égal à), != (différent de), > (supérieur à), < (inférieur à).
  • Opérateurs logiques : utilisés pour combiner des expressions logiques, par exemple && (et logique), || (ou logique), ! (non logique).

Voici un exemple d'utilisation des opérateurs en Javascript :

let a = 10;
let b = 5;

let addition = a + b;
let soustraction = a - b;
let multiplication = a * b;
let division = a / b;

console.log(addition); // Affiche 15
console.log(soustraction); // Affiche 5
console.log(multiplication); // Affiche 50
console.log(division); // Affiche 2

Structures de contrôle (boucles, conditions)

Les structures de contrôle en Javascript sont utilisées pour contrôler le flux d'exécution du programme. Les boucles et les conditions sont les structures de contrôle les plus couramment utilisées.

Voici un exemple d'utilisation de la boucle 'for' en Javascript :

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Ce code affiche les nombres de 0 à 4 dans la console.

Voici un exemple d'utilisation de la condition 'if' en Javascript :

let age = 18;

if (age >= 18) {
  console.log('Vous êtes majeur.');
} else {
  console.log('Vous êtes mineur.');
}

Ce code affiche 'Vous êtes majeur.' si l'âge est supérieur ou égal à 18, sinon il affiche 'Vous êtes mineur.'.

Fonctions

Les fonctions en Javascript sont utilisées pour regrouper des instructions et les exécuter lorsque cela est nécessaire. Voici un exemple de déclaration de fonction en Javascript :

function direBonjour() {
  console.log('Bonjour!');
}

direBonjour(); // Appelle la fonction pour afficher 'Bonjour!' dans la console.

Les fonctions peuvent également prendre des paramètres et renvoyer des valeurs. Voici un exemple :

function additionner(a, b) {
  return a + b;
}

let resultat = additionner(3, 5);
console.log(resultat); // Affiche 8

Dans cet exemple, la fonction 'additionner' prend deux paramètres 'a' et 'b', et renvoie la somme de ces deux paramètres. Nous stockons ensuite le résultat de l'appel de la fonction dans la variable 'resultat' et l'affichons dans la console.

Objets et classes

Les objets et les classes en Javascript sont utilisés pour créer des structures de données complexes. Voici un exemple de création d'un objet en Javascript :

let personne = {
  nom: 'John',
  age: 30,
  direBonjour: function() {
    console.log('Bonjour, je m\'appelle ' + this.nom + ' et j\'ai ' + this.age + ' ans.');
  }
};

personne.direBonjour(); // Appelle la méthode 'direBonjour' de l'objet pour afficher 'Bonjour, je m'appelle John et j'ai 30 ans.' dans la console.

Dans cet exemple, nous créons un objet 'personne' avec deux propriétés 'nom' et 'age', ainsi qu'une méthode 'direBonjour' qui affiche une phrase contenant les valeurs des propriétés.

Les classes en Javascript sont utilisées pour créer des objets similaires. Voici un exemple de création d'une classe en Javascript :

class Personne {
  constructor(nom, age) {
    this.nom = nom;
    this.age = age;
  }

  direBonjour() {
    console.log('Bonjour, je m\'appelle ' + this.nom + ' et j\'ai ' + this.age + ' ans.');
  }
}

let john = new Personne('John', 30);
john.direBonjour(); // Appelle la méthode 'direBonjour' de l'objet 'john' pour afficher 'Bonjour, je m'appelle John et j'ai 30 ans.' dans la console.

Dans cet exemple, nous créons une classe 'Personne' avec un constructeur qui prend deux paramètres 'nom' et 'age', ainsi qu'une méthode 'direBonjour' similaire à celle de l'exemple précédent. Nous créons ensuite un nouvel objet 'john' à partir de la classe 'Personne' en utilisant le mot-clé 'new', et appelons la méthode 'direBonjour' de cet objet.
