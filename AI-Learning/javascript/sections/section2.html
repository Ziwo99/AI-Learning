Les variables et les types de données

En Javascript, les variables sont utilisées pour stocker des valeurs. Elles peuvent contenir différents types de données tels que des nombres, des chaînes de caractères, des booléens, des tableaux, des objets, etc.

Pour déclarer une variable, on utilise le mot-clé var, suivi du nom de la variable et éventuellement de sa valeur initiale :

var age = 25;
var name = "John";
var isStudent = true;

Il existe également deux autres mots-clés pour déclarer des variables : let et const. La différence entre ces trois mots-clés réside dans la portée et la mutabilité des variables, mais cela sera expliqué plus en détail dans les sous-sections suivantes.

En Javascript, il existe plusieurs types de données :

  • Number : représente les nombres, qu'ils soient entiers ou à virgule flottante.
  • String : représente une séquence de caractères.
  • Boolean : représente une valeur de vérité, soit true (vrai) soit false (faux).
  • Array : représente une collection ordonnée d'éléments. Les éléments d'un tableau peuvent être de différents types.
  • Object : représente une collection de propriétés et de valeurs. Les propriétés d'un objet peuvent être de différents types.
  • Null : représente une valeur nulle ou inexistante.
  • Undefined : représente une variable qui n'a pas été définie ou à laquelle aucune valeur n'a été attribuée.

Exemple :

var age = 25;
var name = "John";
var isStudent = true;
var fruits = ["apple", "banana", "orange"];
var person = { firstName: "John", lastName: "Doe", age: 25 };
var nullValue = null;
var undefinedValue;

Les opérateurs et les expressions

En Javascript, les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Il existe différents types d'opérateurs :

  • Opérateurs arithmétiques : utilisés pour effectuer des opérations mathématiques telles que l'addition, la soustraction, la multiplication, la division, etc.
  • Opérateurs d'assignation : utilisés pour attribuer une valeur à une variable.
  • Opérateurs de comparaison : utilisés pour comparer des valeurs et renvoyer un résultat booléen.
  • Opérateurs logiques : utilisés pour combiner des expressions booléennes et renvoyer un résultat booléen.
  • Opérateurs de concaténation : utilisés pour concaténer des chaînes de caractères.

Exemple :

var x = 5;
var y = 10;

var addition = x + y; // 15
var soustraction = x - y; // -5
var multiplication = x * y; // 50
var division = x / y; // 0.5

var a = 5;
a += 10; // équivalent à a = a + 10, a vaut maintenant 15

var b = 5;
var c = 10;
var estSuperieur = b > c; // false

var d = true;
var e = false;
var estVrai = d && e; // false

var firstName = "John";
var lastName = "Doe";
var fullName = firstName + " " + lastName; // "John Doe"

Les structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution du code en fonction de certaines conditions. Les structures de contrôle les plus couramment utilisées en Javascript sont les boucles et les conditions.

Les boucles permettent d'exécuter une série d'instructions plusieurs fois. Il existe plusieurs types de boucles en Javascript :

  • La boucle for : utilisée pour exécuter une série d'instructions un nombre spécifié de fois.
  • La boucle while : utilisée pour exécuter une série d'instructions tant qu'une condition spécifiée est vraie.
  • La boucle do...while : utilisée pour exécuter une série d'instructions une fois, puis tant qu'une condition spécifiée est vraie.

Exemple :

// Boucle for
for (var i = 0; i < 5; i++) {
  console.log(i);
}

// Boucle while
var j = 0;
while (j < 5) {
  console.log(j);
  j++;
}

// Boucle do...while
var k = 0;
do {
  console.log(k);
  k++;
} while (k < 5);

Les conditions permettent d'exécuter une série d'instructions uniquement si une condition spécifiée est vraie. Les conditions en Javascript sont généralement exprimées à l'aide des mots-clés if, else if et else.

var age = 18;

if (age < 18) {
  console.log("Vous êtes mineur.");
} else if (age >= 18 && age < 65) {
  console.log("Vous êtes majeur.");
} else {
  console.log("Vous êtes senior.");
}

Les fonctions et les paramètres

Les fonctions sont des blocs de code réutilisables qui peuvent être appelés à partir d'autres parties du code. Elles permettent d'organiser et de structurer le code en le divisant en tâches plus petites et plus gérables.

En Javascript, les fonctions sont déclarées à l'aide du mot-clé function, suivi du nom de la fonction et de ses paramètres entre parenthèses. Les paramètres sont des variables utilisées pour recevoir des valeurs lors de l'appel de la fonction.

function sayHello(name) {
  console.log("Hello, " + name + "!");
}

sayHello("John"); // "Hello, John!"

Les fonctions peuvent également renvoyer une valeur à l'aide du mot-clé return. La valeur renvoyée peut être utilisée dans d'autres parties du code.

function add(a, b) {
  return a + b;
}

var result = add(5, 10); // 15

La gestion des erreurs (try-catch)

En Javascript, la gestion des erreurs est réalisée à l'aide du bloc try...catch. Le bloc try est utilisé pour entourer le code susceptible de générer une erreur, tandis que le bloc catch est utilisé pour capturer et gérer l'erreur.

try {
  // Code susceptible de générer une erreur
  var result = 10 / 0;
} catch (error) {
  // Gestion de l'erreur
  console.log("Une erreur s'est produite : " + error.message);
}

Il est également possible d'utiliser les blocs finally et throw pour effectuer des actions spécifiques après l'exécution du bloc try ou pour générer une erreur personnalisée.

try {
  // Code susceptible de générer une erreur
  var result = 10 / 0;
} catch (error) {
  // Gestion de l'erreur
  console.log("Une erreur s'est produite : " + error.message);
} finally {
  // Actions à effectuer après l'exécution du bloc try
  console.log("Fin de l'exécution");
}

Exemple de génération d'une erreur personnalisée :

try {
  // Code susceptible de générer une erreur
  throw new Error("Une erreur s'est produite");
} catch (error) {
  // Gestion de l'erreur
  console.log("Une erreur s'est produite : " + error.message);
}
