Section 2: Manipulation du DOM avec Javascript

Introduction au DOM

Le Document Object Model (DOM) est une interface de programmation qui permet d'accéder et de manipuler les éléments d'une page HTML. Avec Javascript, vous pouvez utiliser le DOM pour modifier dynamiquement le contenu, la structure et le style d'une page web.

Le DOM représente la structure hiérarchique d'une page web sous forme d'un arbre. Chaque élément HTML est représenté par un nœud dans cet arbre, et vous pouvez accéder à ces nœuds pour effectuer des opérations telles que la modification du contenu, l'ajout ou la suppression d'éléments, etc.

Sélection des éléments du DOM

Pour sélectionner un élément du DOM, vous pouvez utiliser différentes méthodes en fonction de vos besoins. Voici quelques-unes des méthodes les plus couramment utilisées :

// Sélectionner un élément par son ID
var element = document.getElementById('monElement');

// Sélectionner un élément par sa classe
var elements = document.getElementsByClassName('maClasse');

// Sélectionner tous les éléments d'un certain type
var elements = document.getElementsByTagName('div');

// Sélectionner le premier élément correspondant à un sélecteur CSS
var element = document.querySelector('.maClasse');

// Sélectionner tous les éléments correspondant à un sélecteur CSS
var elements = document.querySelectorAll('div');

Manipulation des éléments du DOM

Une fois que vous avez sélectionné un élément du DOM, vous pouvez le manipuler de différentes manières. Voici quelques-unes des opérations les plus couramment utilisées :

// Modifier le contenu d'un élément
element.innerHTML = 'Nouveau contenu';

// Modifier l'attribut d'un élément
element.setAttribute('src', 'image.jpg');

// Ajouter une classe à un élément
element.classList.add('maClasse');

// Supprimer une classe d'un élément
element.classList.remove('maClasse');

// Ajouter un nouvel élément à la page
var newElement = document.createElement('div');
newElement.innerHTML = 'Nouvel élément';
document.body.appendChild(newElement);

// Supprimer un élément de la page
element.parentNode.removeChild(element);

Gestion des événements

Les événements sont des actions ou des occurrences qui se produisent dans une page web, comme un clic de souris, une pression de touche, etc. Vous pouvez utiliser Javascript pour gérer ces événements et exécuter du code en réponse à ceux-ci.

Voici comment vous pouvez ajouter un gestionnaire d'événements à un élément du DOM :

// Ajouter un gestionnaire d'événements à un élément
element.addEventListener('click', function() {
    // Code à exécuter lorsque l'événement se produit
});

Vous pouvez également utiliser des fonctions nommées comme gestionnaires d'événements :

// Définir une fonction comme gestionnaire d'événements
function handleClick() {
    // Code à exécuter lorsque l'événement se produit
}

// Ajouter le gestionnaire d'événements à l'élément
element.addEventListener('click', handleClick);

En utilisant la manipulation du DOM avec Javascript, vous pouvez créer des pages web interactives et dynamiques. Continuez à explorer et à pratiquer pour maîtriser cette compétence essentielle du développement web.
