Section 2 : Manipulation du DOM

Introduction à la manipulation du DOM

La manipulation du DOM (Document Object Model) est une étape essentielle dans le développement JavaScript. Le DOM représente la structure d'une page HTML et permet d'accéder et de modifier les éléments de cette page.

Grâce à la manipulation du DOM, vous pouvez :

  • Sélectionner des éléments spécifiques
  • Modifier le contenu des éléments
  • Ajouter ou supprimer des éléments
  • Gérer les événements

Sélection d'éléments du DOM

Pour sélectionner des éléments du DOM, vous pouvez utiliser différentes méthodes :

// Sélection par ID
var element = document.getElementById('myElement');

// Sélection par classe
var elements = document.getElementsByClassName('myClass');

// Sélection par balise
var elements = document.getElementsByTagName('div');

// Sélection par sélecteur CSS
var elements = document.querySelectorAll('.myClass');

Modification du contenu des éléments du DOM

Une fois que vous avez sélectionné un élément du DOM, vous pouvez modifier son contenu en utilisant la propriété innerHTML. Voici un exemple :

var element = document.getElementById('myElement');
element.innerHTML = 'Nouveau contenu';

Ajout et suppression d'éléments du DOM

Pour ajouter un nouvel élément au DOM, vous pouvez utiliser la méthode createElement pour créer un nouvel élément, puis la méthode appendChild pour l'ajouter à un élément parent. Voici un exemple :

var parentElement = document.getElementById('parentElement');
var newElement = document.createElement('div');
newElement.innerHTML = 'Nouvel élément';
parentElement.appendChild(newElement);

Pour supprimer un élément du DOM, vous pouvez utiliser la méthode removeChild. Voici un exemple :

var parentElement = document.getElementById('parentElement');
var elementToRemove = document.getElementById('elementToRemove');
parentElement.removeChild(elementToRemove);

Gestion des événements du DOM

Les événements du DOM permettent de réagir aux actions de l'utilisateur, comme un clic sur un bouton ou la soumission d'un formulaire. Vous pouvez utiliser la méthode addEventListener pour attacher un gestionnaire d'événements à un élément. Voici un exemple :

var button = document.getElementById('myButton');
button.addEventListener('click', function() {
  alert('Le bouton a été cliqué !');
});

Vous pouvez également utiliser des gestionnaires d'événements prédéfinis, tels que onclick ou onsubmit, directement dans les balises HTML. Voici un exemple :

<button onclick="alert('Le bouton a été cliqué !')">Cliquez-moi</button>

Il est important de noter que la manipulation du DOM peut avoir un impact sur les performances de votre application, surtout si vous effectuez des modifications fréquentes ou complexes. Il est donc recommandé d'optimiser votre code pour minimiser ces impacts.
