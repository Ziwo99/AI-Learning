Tableaux et méthodes associées

Les tableaux sont des structures de données qui permettent de stocker et d'organiser plusieurs valeurs dans une seule variable. En JavaScript, les tableaux peuvent contenir des éléments de différents types de données tels que des nombres, des chaînes de caractères, des objets, etc.

Pour créer un tableau, on utilise les crochets [] et on sépare les éléments par des virgules. Par exemple :

let fruits = ['pomme', 'banane', 'orange'];

Pour accéder à un élément spécifique dans un tableau, on utilise l'indice de cet élément. Les indices commencent à 0. Par exemple, pour accéder à la première valeur du tableau "fruits", on utilise :

let premierFruit = fruits[0]; // 'pomme'

Il existe plusieurs méthodes intégrées pour manipuler les tableaux en JavaScript :

  • push() : ajoute un élément à la fin du tableau
  • pop() : supprime le dernier élément du tableau
  • shift() : supprime le premier élément du tableau
  • unshift() : ajoute un élément au début du tableau
  • splice() : ajoute, supprime ou remplace des éléments dans le tableau
  • slice() : extrait une partie du tableau et renvoie un nouveau tableau
  • concat() : fusionne deux tableaux en un seul
  • sort() : trie les éléments du tableau
  • reverse() : inverse l'ordre des éléments du tableau

Voici un exemple d'utilisation de certaines de ces méthodes :

let fruits = ['pomme', 'banane', 'orange'];

fruits.push('kiwi'); // ['pomme', 'banane', 'orange', 'kiwi']

fruits.pop(); // ['pomme', 'banane', 'orange']

fruits.shift(); // ['banane', 'orange']

fruits.unshift('fraise'); // ['fraise', 'banane', 'orange']

fruits.splice(1, 0, 'mangue'); // ['fraise', 'mangue', 'banane', 'orange']

let citrus = fruits.slice(1, 3); // ['mangue', 'banane']

let allFruits = fruits.concat(citrus); // ['fraise', 'mangue', 'banane', 'orange', 'mangue', 'banane']

fruits.sort(); // ['banane', 'fraise', 'mangue', 'orange']

fruits.reverse(); // ['orange', 'mangue', 'fraise', 'banane']

Objets et méthodes associées

Les objets sont des structures de données qui permettent de stocker des collections de propriétés et de méthodes. En JavaScript, les objets sont des entités autonomes avec leurs propres attributs et comportements.

Pour créer un objet, on utilise les accolades {}. Par exemple :

let personne = {
  nom: 'John',
  age: 30,
  adresse: 'Paris'
};

Pour accéder à une propriété spécifique d'un objet, on utilise la notation pointée. Par exemple, pour accéder à la propriété "nom" de l'objet "personne", on utilise :

let nomPersonne = personne.nom; // 'John'

Il est également possible d'accéder aux propriétés d'un objet en utilisant la notation entre crochets. Par exemple :

let nomPersonne = personne['nom']; // 'John'

Les objets peuvent également avoir des méthodes, qui sont des fonctions associées à l'objet. Par exemple :

let personne = {
  nom: 'John',
  age: 30,
  adresse: 'Paris',
  sePresenter: function() {
    console.log('Je m\'appelle ' + this.nom + ', j\'ai ' + this.age + ' ans.');
  }
};

personne.sePresenter(); // Affiche "Je m'appelle John, j'ai 30 ans."

Il est possible d'ajouter, de modifier ou de supprimer des propriétés et des méthodes d'un objet à tout moment. Par exemple :

personne.profession = 'Développeur'; // Ajoute une nouvelle propriété

personne.age = 31; // Modifie la valeur d'une propriété existante

delete personne.adresse; // Supprime une propriété

Utilisation de JSON

JSON (JavaScript Object Notation) est un format de données couramment utilisé pour échanger des données entre un serveur et un client, ou entre différentes parties d'une application. JSON est basé sur la syntaxe des objets JavaScript.

Pour convertir un objet JavaScript en JSON, on utilise la méthode JSON.stringify(). Par exemple :

let personne = {
  nom: 'John',
  age: 30,
  adresse: 'Paris'
};

let jsonPersonne = JSON.stringify(personne);
console.log(jsonPersonne); // {"nom":"John","age":30,"adresse":"Paris"}

Pour convertir une chaîne JSON en objet JavaScript, on utilise la méthode JSON.parse(). Par exemple :

let jsonPersonne = '{"nom":"John","age":30,"adresse":"Paris"}';

let personne = JSON.parse(jsonPersonne);
console.log(personne.nom); // 'John'

JSON est largement utilisé dans les applications Web pour échanger des données avec des serveurs, stocker des configurations, etc.
