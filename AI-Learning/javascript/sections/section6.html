Les chaînes de caractères

Les chaînes de caractères sont utilisées pour représenter du texte en JavaScript. Elles peuvent être déclarées en utilisant des guillemets simples ('') ou des guillemets doubles (""). Voici un exemple :

let message = 'Bonjour';
let name = "Alice";

Les chaînes de caractères peuvent être concaténées en utilisant l'opérateur de concaténation (+) :

let greeting = 'Bonjour ' + name;
console.log(greeting); // Affiche "Bonjour Alice"

Il est possible d'accéder à des caractères spécifiques dans une chaîne de caractères en utilisant l'index :

let str = 'Hello';
console.log(str[0]); // Affiche "H"
console.log(str[1]); // Affiche "e"

Les chaînes de caractères ont également de nombreuses méthodes intégrées pour effectuer des opérations telles que la recherche de sous-chaînes, le remplacement de texte, la conversion en majuscules ou en minuscules, etc. :

let str = 'Hello, world!';
console.log(str.length); // Affiche la longueur de la chaîne (13)
console.log(str.indexOf('world')); // Affiche l'index de la première occurrence de 'world' (7)
console.log(str.replace('world', 'JavaScript')); // Remplace 'world' par 'JavaScript' (Hello, JavaScript!)
console.log(str.toUpperCase()); // Convertit la chaîne en majuscules (HELLO, WORLD!)

Les nombres

Les nombres sont utilisés pour représenter des valeurs numériques en JavaScript. Ils peuvent être des nombres entiers ou des nombres à virgule flottante. Voici quelques exemples :

let age = 25;
let pi = 3.14159;

Les nombres peuvent être utilisés dans des opérations mathématiques telles que l'addition, la soustraction, la multiplication et la division :

let x = 10;
let y = 5;
let sum = x + y;
let difference = x - y;
let product = x * y;
let quotient = x / y;

Il est également possible d'effectuer des opérations plus avancées telles que le calcul de la racine carrée, l'arrondi, etc. en utilisant les méthodes intégrées de l'objet Math :

console.log(Math.sqrt(16)); // Calcule la racine carrée de 16 (4)
console.log(Math.round(3.7)); // Arrondit 3.7 à l'entier le plus proche (4)
console.log(Math.floor(3.7)); // Arrondit 3.7 à l'entier inférieur (3)
console.log(Math.ceil(3.2)); // Arrondit 3.2 à l'entier supérieur (4)

Les dates et heures

Les dates et heures sont représentées en JavaScript à l'aide de l'objet Date. Cet objet permet de créer, manipuler et formater des dates et heures. Voici un exemple de création d'une date :

let currentDate = new Date();
console.log(currentDate); // Affiche la date et l'heure actuelles

Il est possible d'accéder à des éléments spécifiques d'une date tels que l'année, le mois, le jour, l'heure, etc. en utilisant les méthodes intégrées de l'objet Date :

console.log(currentDate.getFullYear()); // Affiche l'année actuelle
console.log(currentDate.getMonth()); // Affiche le mois actuel (0-11)
console.log(currentDate.getDate()); // Affiche le jour du mois actuel (1-31)
console.log(currentDate.getHours()); // Affiche l'heure actuelle (0-23)
console.log(currentDate.getMinutes()); // Affiche les minutes actuelles (0-59)
console.log(currentDate.getSeconds()); // Affiche les secondes actuelles (0-59)

Il est également possible de formater une date en utilisant des méthodes intégrées ou des bibliothèques externes telles que Moment.js :

console.log(currentDate.toLocaleDateString()); // Affiche la date au format local
console.log(currentDate.toISOString()); // Affiche la date au format ISO (YYYY-MM-DDTHH:mm:ss.sssZ)

Les objets JSON

JSON (JavaScript Object Notation) est un format de données largement utilisé pour échanger des données entre un serveur et un client. En JavaScript, les objets JSON peuvent être représentés en utilisant une syntaxe similaire à celle des objets JavaScript :

let person = {
  "name": "Alice",
  "age": 25,
  "city": "Paris"
};

console.log(person.name); // Affiche le nom de la personne (Alice)
console.log(person.age); // Affiche l'âge de la personne (25)
console.log(person.city); // Affiche la ville de la personne (Paris)

Il est possible de convertir un objet JavaScript en JSON en utilisant la méthode JSON.stringify() :

let person = {
  "name": "Alice",
  "age": 25,
  "city": "Paris"
};

let json = JSON.stringify(person);
console.log(json); // Affiche '{"name":"Alice","age":25,"city":"Paris"}'

De même, il est possible de convertir une chaîne JSON en objet JavaScript en utilisant la méthode JSON.parse() :

let json = '{"name":"Alice","age":25,"city":"Paris"}';
let person = JSON.parse(json);
console.log(person.name); // Affiche le nom de la personne (Alice)
console.log(person.age); // Affiche l'âge de la personne (25)
console.log(person.city); // Affiche la ville de la personne (Paris)
