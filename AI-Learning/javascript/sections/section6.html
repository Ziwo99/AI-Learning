Section 6: Les requêtes HTTP avec JavaScript

Introduction aux requêtes HTTP

Les requêtes HTTP (Hypertext Transfer Protocol) sont utilisées pour communiquer avec des serveurs et récupérer des données à partir d'Internet. En JavaScript, nous pouvons effectuer des requêtes HTTP en utilisant l'objet XMLHttpRequest ou la méthode fetch. Ces outils nous permettent d'envoyer des requêtes vers des serveurs distants et de recevoir des réponses contenant des données.

Les requêtes GET en JavaScript

Les requêtes GET sont utilisées pour récupérer des données à partir d'un serveur distant. En JavaScript, nous pouvons effectuer une requête GET en utilisant l'objet XMLHttpRequest ou la méthode fetch. Voici un exemple de requête GET en utilisant l'objet XMLHttpRequest : 


var xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.example.com/data', true);
xhr.onreadystatechange = function() {
    if (xhr.readyState === 4 && xhr.status === 200) {
        var response = JSON.parse(xhr.responseText);
        console.log(response);
    }
};
xhr.send();

Les requêtes POST en JavaScript

Les requêtes POST sont utilisées pour envoyer des données à un serveur distant. En JavaScript, nous pouvons effectuer une requête POST en utilisant l'objet XMLHttpRequest ou la méthode fetch. Voici un exemple de requête POST en utilisant l'objet XMLHttpRequest : 


var xhr = new XMLHttpRequest();
xhr.open('POST', 'https://api.example.com/data', true);
xhr.setRequestHeader('Content-Type', 'application/json');
xhr.onreadystatechange = function() {
    if (xhr.readyState === 4 && xhr.status === 200) {
        var response = JSON.parse(xhr.responseText);
        console.log(response);
    }
};
var data = {
    name: 'John Doe',
    age: 30
};
xhr.send(JSON.stringify(data));

La gestion des réponses en JavaScript

Lorsque nous effectuons une requête HTTP en JavaScript, nous recevons une réponse du serveur distant. Cette réponse peut être au format JSON, XML ou texte brut. En JavaScript, nous pouvons gérer les réponses en utilisant les méthodes appropriées de l'objet XMLHttpRequest ou en utilisant les fonctions de traitement de la réponse de la méthode fetch. Voici un exemple de gestion de réponse en utilisant l'objet XMLHttpRequest : 


var xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.example.com/data', true);
xhr.onreadystatechange = function() {
    if (xhr.readyState === 4) {
        if (xhr.status === 200) {
            var response = JSON.parse(xhr.responseText);
            console.log(response);
        } else {
            console.log('Une erreur s\'est produite.');
        }
    }
};
xhr.send();

Les requêtes asynchrones en JavaScript

Les requêtes HTTP en JavaScript peuvent être effectuées de manière synchrone ou asynchrone. Les requêtes synchrones bloquent l'exécution du code jusqu'à ce que la réponse soit reçue, tandis que les requêtes asynchrones permettent à l'exécution du code de se poursuivre pendant que la requête est en cours. En JavaScript, nous utilisons l'objet XMLHttpRequest ou la méthode fetch avec le paramètre asynchrone défini sur true pour effectuer des requêtes asynchrones. Voici un exemple de requête asynchrone en utilisant l'objet XMLHttpRequest : 


var xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.example.com/data', true);
xhr.onreadystatechange = function() {
    if (xhr.readyState === 4 && xhr.status === 200) {
        var response = JSON.parse(xhr.responseText);
        console.log(response);
    }
};
xhr.send();
console.log('La requête est en cours...');
