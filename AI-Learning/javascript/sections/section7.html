Les requêtes HTTP (AJAX)

Les requêtes HTTP (Hypertext Transfer Protocol) sont utilisées pour communiquer avec des serveurs web et récupérer des données. En utilisant AJAX (Asynchronous JavaScript and XML), nous pouvons effectuer des requêtes HTTP de manière asynchrone, ce qui signifie que la page web ne sera pas bloquée pendant le chargement des données.

Pour effectuer une requête HTTP en utilisant AJAX, nous utilisons l'objet XMLHttpRequest. Voici un exemple de code :

var xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.example.com/data', true);
xhr.onreadystatechange = function() {
  if (xhr.readyState === 4 && xhr.status === 200) {
    var response = JSON.parse(xhr.responseText);
    console.log(response);
  }
};
xhr.send();

Dans cet exemple, nous créons un nouvel objet XMLHttpRequest et nous ouvrons une connexion en spécifiant la méthode (GET dans cet exemple) et l'URL de l'API. Nous utilisons également une fonction de rappel (onreadystatechange) pour traiter la réponse lorsque la requête est terminée. Si le statut de la requête est 200 (OK) et que l'état de la requête est 4 (DONE), nous pouvons accéder aux données de la réponse.

Les API RESTful

Les API RESTful (Representational State Transfer) sont des interfaces de programmation qui utilisent les principes de l'architecture REST pour communiquer avec des serveurs web. Elles permettent d'effectuer des opérations CRUD (Create, Read, Update, Delete) sur des ressources.

Pour utiliser une API RESTful, nous devons envoyer des requêtes HTTP avec des méthodes spécifiques pour effectuer différentes actions. Voici les méthodes les plus couramment utilisées :

  • GET: récupère des données à partir de l'API
  • POST: crée une nouvelle ressource sur l'API
  • PUT: met à jour une ressource existante sur l'API
  • DELETE: supprime une ressource de l'API

Voici un exemple de code qui utilise l'API RESTful pour récupérer des utilisateurs :

fetch('https://api.example.com/users')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error('Error:', error);
  });

Dans cet exemple, nous utilisons la fonction fetch pour envoyer une requête GET à l'URL de l'API. Nous utilisons ensuite les fonctions de rappel then pour traiter la réponse et catch pour gérer les erreurs éventuelles.

Les services web (création et consommation)

Les services web sont des applications qui permettent à différents systèmes de communiquer entre eux via des protocoles standardisés tels que HTTP. Ils peuvent être utilisés pour créer des services web (serveurs) ou pour consommer des services web (clients).

Pour créer un service web en utilisant JavaScript, nous pouvons utiliser des frameworks tels que Node.js ou Express.js. Voici un exemple de code qui crée un service web simple qui renvoie "Hello, World!" lorsque nous accédons à l'URL "/hello" :

const express = require('express');
const app = express();

app.get('/hello', (req, res) => {
  res.send('Hello, World!');
});

app.listen(3000, () => {
  console.log('Server is running on port 3000');
});

Dans cet exemple, nous utilisons le framework Express.js pour créer une application web. Nous définissons une route ("/hello") qui renvoie la réponse "Hello, World!" lorsque nous accédons à cette URL. Ensuite, nous démarrons le serveur en écoutant le port 3000.

Pour consommer un service web en utilisant JavaScript, nous pouvons utiliser les techniques que nous avons vues précédemment, telles que les requêtes HTTP (AJAX) ou les API RESTful. Nous pouvons également utiliser des bibliothèques telles que Axios ou Fetch pour simplifier le processus de consommation des services web.
