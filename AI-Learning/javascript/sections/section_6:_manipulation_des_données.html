Section 6: Manipulation des données

Les tableaux en Javascript

Les tableaux sont des structures de données utilisées pour stocker et manipuler des collections d'éléments de manière ordonnée. En Javascript, un tableau peut contenir des valeurs de différents types, y compris des nombres, des chaînes de caractères, des objets, etc.

Voici comment créer un tableau en Javascript :

var fruits = ['pomme', 'banane', 'orange'];

Pour accéder à un élément spécifique dans un tableau, vous pouvez utiliser l'index de cet élément. L'index commence à 0 pour le premier élément :

var premierFruit = fruits[0]; // pomme

Vous pouvez également modifier la valeur d'un élément en utilisant son index :

fruits[1] = 'kiwi'; // ['pomme', 'kiwi', 'orange']

Il existe de nombreuses méthodes intégrées pour manipuler les tableaux en Javascript, telles que push() pour ajouter un élément à la fin du tableau, pop() pour supprimer le dernier élément, splice() pour ajouter ou supprimer des éléments à des positions spécifiques, etc.

Les objets JSON en Javascript

JSON (JavaScript Object Notation) est un format de données largement utilisé pour échanger des informations structurées entre un serveur et un client. En Javascript, un objet JSON est une collection de paires clé-valeur, où les clés sont des chaînes de caractères et les valeurs peuvent être de n'importe quel type de données valide.

Voici un exemple d'objet JSON en Javascript :

var personne = {
    "nom": "John Doe",
    "âge": 30,
    "ville": "Paris"
};

Pour accéder à une valeur spécifique dans un objet JSON, vous pouvez utiliser la notation pointée :

var nomPersonne = personne.nom; // John Doe

Vous pouvez également modifier la valeur d'une clé en utilisant la notation pointée :

personne.ville = "Lyon"; // {"nom": "John Doe", "âge": 30, "ville": "Lyon"}

Les objets JSON peuvent également contenir des tableaux, des objets imbriqués et d'autres structures de données complexes.

Les requêtes HTTP en Javascript

Les requêtes HTTP sont utilisées pour communiquer avec des serveurs et récupérer des données à partir de ressources externes. En Javascript, vous pouvez effectuer des requêtes HTTP en utilisant l'API Fetch ou l'objet XMLHttpRequest.

Voici un exemple d'utilisation de l'API Fetch pour effectuer une requête GET :

fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error(error);
    });

Cet exemple envoie une requête GET à l'URL spécifiée et récupère les données JSON renvoyées par le serveur. Les données sont ensuite affichées dans la console.

Vous pouvez également effectuer des requêtes POST, PUT, DELETE, etc., en spécifiant la méthode appropriée dans la configuration de la requête.

Il est important de noter que les requêtes HTTP peuvent être soumises à des restrictions de sécurité, telles que la politique de même origine (Same Origin Policy) ou les en-têtes CORS (Cross-Origin Resource Sharing). Assurez-vous de comprendre ces concepts et de les prendre en compte lors de l'implémentation de requêtes HTTP en Javascript.
