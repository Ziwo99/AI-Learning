Section 1 : Introduction à Javascript

Introduction à Javascript

JavaScript est un langage de programmation utilisé pour ajouter de l'interactivité aux sites web. Il est principalement utilisé pour créer des fonctionnalités dynamiques, telles que des formulaires interactifs, des animations et des effets visuels. JavaScript est un langage de script côté client, ce qui signifie qu'il est exécuté par le navigateur web du client.

Les bases de la programmation en Javascript

La programmation en JavaScript repose sur certains concepts fondamentaux. Voici quelques-uns des concepts de base :

  • Les variables : utilisées pour stocker des valeurs et les manipuler.
  • Les fonctions : des blocs de code réutilisables qui effectuent une tâche spécifique.
  • Les boucles : utilisées pour répéter des instructions plusieurs fois.
  • Les conditions : permettent d'exécuter des instructions en fonction d'une condition donnée.

Les variables et les types de données en Javascript

Les variables en JavaScript sont utilisées pour stocker des données. Il existe différents types de données que vous pouvez stocker dans une variable :

  • Les nombres : utilisés pour stocker des valeurs numériques.
  • Les chaînes de caractères : utilisées pour stocker du texte.
  • Les booléens : utilisés pour stocker des valeurs de vérité (true ou false).
  • Les tableaux : utilisés pour stocker une collection de valeurs.
  • Les objets : utilisés pour stocker des données structurées.

Voici un exemple de déclaration de variable en JavaScript :

var age = 25;

Les opérateurs en Javascript

Les opérateurs en JavaScript sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés :

  • L'opérateur d'assignation (=) : utilisé pour assigner une valeur à une variable.
  • L'opérateur arithmétique (+, -, *, /) : utilisé pour effectuer des opérations mathématiques.
  • L'opérateur de comparaison (==, !=, >, <) : utilisé pour comparer des valeurs.
  • L'opérateur logique (&&, ||, !) : utilisé pour combiner des conditions.

Voici un exemple d'utilisation d'opérateurs en JavaScript :

var x = 5;
var y = 10;
var z = x + y;

Les structures de contrôle en Javascript

Les structures de contrôle en JavaScript permettent de contrôler le flux d'exécution du code. Voici quelques-unes des structures de contrôle les plus couramment utilisées :

  • Les instructions conditionnelles (if, else if, else) : utilisées pour exécuter des instructions en fonction d'une condition donnée.
  • La boucle while : utilisée pour répéter des instructions tant qu'une condition donnée est vraie.
  • La boucle for : utilisée pour répéter des instructions un certain nombre de fois.
  • La boucle do...while : utilisée pour répéter des instructions au moins une fois, puis tant qu'une condition donnée est vraie.

Voici un exemple d'utilisation d'une structure de contrôle en JavaScript :

var i = 0;

while (i < 10) {
  console.log('Iteration ' + i);
  i++;
}
