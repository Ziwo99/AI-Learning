Section 1: Introduction à Javascript

Les bases de Javascript

Les bases de Javascript sont essentielles pour comprendre le langage et pouvoir créer des programmes fonctionnels. Voici quelques concepts clés :

JavaScript est un langage de programmation utilisé pour créer des fonctionnalités interactives sur les sites web. Il est principalement utilisé pour ajouter des comportements dynamiques aux pages web.

Les variables sont utilisées pour stocker des valeurs dans la mémoire de l'ordinateur. Elles peuvent être déclarées en utilisant le mot-clé "var", suivi du nom de la variable. Par exemple :

var x;

Les types de données en Javascript incluent les nombres, les chaînes de caractères, les booléens, les tableaux et les objets. Par exemple :

var age = 25;
var name = "John";
var isStudent = true;
var fruits = ["apple", "banana", "orange"];
var person = {firstName:"John", lastName:"Doe"};

Les variables et les types de données

Les variables et les types de données sont essentiels pour manipuler et stocker des informations dans un programme Javascript. Voici quelques concepts clés :

Les variables peuvent être utilisées pour stocker des valeurs de différents types de données, tels que des nombres, des chaînes de caractères, des booléens, des tableaux et des objets.

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Par exemple, l'opérateur "+" est utilisé pour concaténer des chaînes de caractères :

var firstName = "John";
var lastName = "Doe";
var fullName = firstName + " " + lastName;

Les expressions sont des combinaisons d'opérateurs, de variables et de valeurs qui produisent un résultat. Par exemple :

var x = 5;
var y = 10;
var sum = x + y;

Les opérateurs et les expressions

Les opérateurs et les expressions sont utilisés pour effectuer des calculs et des manipulations sur les variables et les valeurs en Javascript. Voici quelques concepts clés :

Les opérateurs arithmétiques, tels que l'addition (+), la soustraction (-), la multiplication (*) et la division (/), sont utilisés pour effectuer des calculs mathématiques :

var x = 5;
var y = 10;
var sum = x + y;
var difference = x - y;
var product = x * y;
var quotient = x / y;

Les opérateurs de comparaison, tels que l'égalité (==), l'inégalité (!=), l'infériorité (<), la supériorité (>), l'infériorité ou égalité (<=) et la supériorité ou égalité (>=), sont utilisés pour comparer des valeurs :

var x = 5;
var y = 10;
var isEqual = x == y;
var isNotEqual = x != y;
var isLessThan = x < y;
var isGreaterThan = x > y;
var isLessThanOrEqual = x <= y;
var isGreaterThanOrEqual = x >= y;

Les structures de contrôle

Les structures de contrôle sont utilisées pour contrôler le flux d'exécution d'un programme Javascript. Voici quelques concepts clés :

La structure de contrôle "if" permet d'exécuter un bloc de code si une condition est vraie :

var x = 5;
if (x > 0) {
    console.log("x is positive");
}

La structure de contrôle "else" permet d'exécuter un bloc de code alternatif si la condition de l'instruction "if" n'est pas remplie :

var x = 5;
if (x > 0) {
    console.log("x is positive");
} else {
    console.log("x is negative or zero");
}

La structure de contrôle "for" permet d'exécuter un bloc de code plusieurs fois, en utilisant une variable de compteur :

for (var i = 0; i < 5; i++) {
    console.log("Iteration " + i);
}

La structure de contrôle "while" permet d'exécuter un bloc de code tant qu'une condition est vraie :

var i = 0;
while (i < 5) {
    console.log("Iteration " + i);
    i++;
}

La structure de contrôle "switch" permet d'exécuter différents blocs de code en fonction de la valeur d'une variable :

var day = "Monday";
switch (day) {
    case "Monday":
        console.log("It's Monday");
        break;
    case "Tuesday":
        console.log("It's Tuesday");
        break;
    default:
        console.log("It's another day");
        break;
}

Ces structures de contrôle sont essentielles pour créer des programmes Javascript fonctionnels et interactifs.
