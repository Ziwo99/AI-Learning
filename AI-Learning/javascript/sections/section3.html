Section 3 : Les fonctions en Javascript

Introduction aux fonctions en Javascript

Les fonctions en Javascript sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser et de structurer le code, et sont essentielles pour développer des applications Javascript.

Pour déclarer une fonction, utilisez le mot-clé function, suivi du nom de la fonction et de parenthèses vides ou contenant les paramètres. Par exemple :

function maFonction() {
    // Code à exécuter
}

Déclaration et appel de fonctions

Pour appeler une fonction, utilisez simplement son nom suivi de parenthèses. Par exemple :

maFonction();

Les fonctions peuvent également retourner une valeur en utilisant le mot-clé return. Par exemple :

function addition(a, b) {
    return a + b;
}

var resultat = addition(3, 5);
console.log(resultat); // Affiche 8

Les paramètres et les arguments des fonctions

Les paramètres sont des variables déclarées dans la déclaration de la fonction, qui reçoivent les valeurs des arguments lors de l'appel de la fonction. Par exemple :

function multiplier(a, b) {
    return a * b;
}

var resultat = multiplier(4, 6);
console.log(resultat); // Affiche 24

Il est possible de définir des valeurs par défaut pour les paramètres en utilisant l'opérateur d'affectation. Par exemple :

function puissance(a, b = 2) {
    return Math.pow(a, b);
}

var resultat1 = puissance(3);
console.log(resultat1); // Affiche 9

var resultat2 = puissance(3, 4);
console.log(resultat2); // Affiche 81

Les fonctions anonymes et les fonctions fléchées

Les fonctions anonymes sont des fonctions sans nom, souvent utilisées comme arguments d'autres fonctions ou pour créer des fonctions auto-exécutables. Par exemple :

var maFonction = function() {
    // Code à exécuter
};

maFonction();

Les fonctions fléchées sont une syntaxe plus concise pour déclarer des fonctions anonymes. Elles utilisent la flèche (=>) pour définir la fonction. Par exemple :

var maFonction = () => {
    // Code à exécuter
};

maFonction();

Les fonctions récursives

Les fonctions récursives sont des fonctions qui s'appellent elles-mêmes. Elles sont souvent utilisées pour résoudre des problèmes qui peuvent être décomposés en sous-problèmes similaires. Par exemple :

function factorielle(n) {
    if (n === 0) {
        return 1;
    } else {
        return n * factorielle(n - 1);
    }
}

var resultat = factorielle(5);
console.log(resultat); // Affiche 120
