Introduction au DOM

Le DOM (Document Object Model) est une interface de programmation qui permet de manipuler les éléments d'une page HTML. Il représente la structure du document sous forme d'un arbre, où chaque élément est un nœud.

Grâce au DOM, il est possible de sélectionner des éléments, de modifier leur contenu, d'ajouter ou de supprimer des éléments, et de gérer les événements.

Sélection d'éléments

Pour sélectionner des éléments dans le DOM, on utilise la méthode querySelector() ou querySelectorAll(). La méthode querySelector() renvoie le premier élément correspondant au sélecteur spécifié, tandis que la méthode querySelectorAll() renvoie une liste de tous les éléments correspondants.

Voici quelques exemples :

// Sélection d'un élément par son ID
const element = document.querySelector('#monElement');

// Sélection de tous les éléments de classe "maClasse"
const elements = document.querySelectorAll('.maClasse');

// Sélection de tous les éléments de balise <p>
const paragraphs = document.querySelectorAll('p');

Modification du contenu

Pour modifier le contenu d'un élément dans le DOM, on utilise les propriétés innerHTML ou textContent. La propriété innerHTML permet de modifier le contenu HTML de l'élément, tandis que la propriété textContent permet de modifier le contenu texte de l'élément.

Voici quelques exemples :

// Modification du contenu HTML d'un élément
element.innerHTML = '<strong>Nouveau contenu</strong>';

// Modification du contenu texte d'un élément
element.textContent = 'Nouveau contenu';

Gestion des événements

Pour gérer les événements dans le DOM, on utilise la méthode addEventListener(). Cette méthode permet d'ajouter un écouteur d'événement à un élément spécifié, en spécifiant le type d'événement à écouter et la fonction à exécuter lorsque l'événement se produit.

Voici un exemple :

// Ajout d'un écouteur d'événement au clic sur un bouton
const button = document.querySelector('#monBouton');

button.addEventListener('click', function() {
  // Code à exécuter lorsque le bouton est cliqué
});

Il est également possible de supprimer un écouteur d'événement en utilisant la méthode removeEventListener().

Voici un exemple :

// Suppression de l'écouteur d'événement au clic sur un bouton
button.removeEventListener('click', maFonction);

Il est important de noter que les événements peuvent être de différents types, tels que "click", "mouseover", "keydown", etc. De plus, il est possible de capturer des événements au niveau du document entier ou d'éléments spécifiques.

Ceci conclut la section sur la manipulation du DOM. Nous avons appris comment sélectionner des éléments, modifier leur contenu et gérer les événements. Dans la prochaine section, nous aborderons la programmation orientée objet en Javascript.
