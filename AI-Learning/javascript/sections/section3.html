Introduction au DOM (Document Object Model)

Le DOM (Document Object Model) est une interface de programmation qui permet de manipuler les éléments d'une page HTML. Il représente la structure du document sous forme d'un arbre, où chaque élément est un nœud.

Grâce au DOM, il est possible d'accéder aux éléments HTML, de les modifier, de les supprimer ou d'en ajouter de nouveaux. Cela permet de créer des interactions dynamiques avec la page, en répondant aux actions de l'utilisateur.

Sélection des éléments du DOM

Pour sélectionner un élément du DOM, on utilise la méthode querySelector() ou querySelectorAll(). La méthode querySelector() renvoie le premier élément correspondant à un sélecteur CSS spécifié, tandis que la méthode querySelectorAll() renvoie tous les éléments correspondants.

// Sélectionner un élément par son ID
const element = document.querySelector('#monElement');

// Sélectionner un élément par sa classe
const elements = document.querySelectorAll('.maClasse');

// Sélectionner un élément par son nom de balise
const elements = document.querySelectorAll('div');

Modification des éléments du DOM

Pour modifier un élément du DOM, on peut utiliser les propriétés et méthodes disponibles. Par exemple, pour modifier le contenu d'un élément, on utilise la propriété innerHTML.

// Modifier le contenu d'un élément
const element = document.querySelector('#monElement');
element.innerHTML = 'Nouveau contenu';

Gestion des événements

Les événements permettent de détecter les actions de l'utilisateur, comme un clic sur un bouton ou le survol d'une image. On peut ajouter des écouteurs d'événements à un élément du DOM en utilisant la méthode addEventListener().

// Ajouter un écouteur d'événement
const bouton = document.querySelector('#monBouton');
bouton.addEventListener('click', function() {
  // Code à exécuter lorsque le bouton est cliqué
});

Il existe de nombreux types d'événements, tels que click, mouseover, keydown, etc. On peut également utiliser des fonctions fléchées pour définir l'action à effectuer.

// Utilisation d'une fonction fléchée
bouton.addEventListener('click', () => {
  // Code à exécuter lorsque le bouton est cliqué
});

Il est également possible de supprimer un écouteur d'événement en utilisant la méthode removeEventListener().

// Supprimer un écouteur d'événement
bouton.removeEventListener('click', maFonction);

Il est important de noter que les événements peuvent être propagés de l'élément cible vers ses parents ou descendants. On peut arrêter la propagation de l'événement en utilisant la méthode stopPropagation().

// Arrêter la propagation de l'événement
element.addEventListener('click', function(event) {
  event.stopPropagation();
});

De plus, certains événements peuvent être annulés, par exemple pour empêcher le comportement par défaut d'un lien ou d'un formulaire. On peut annuler un événement en utilisant la méthode preventDefault().

// Annuler le comportement par défaut d'un lien
lien.addEventListener('click', function(event) {
  event.preventDefault();
});

Il est également possible de déléguer des événements, c'est-à-dire d'ajouter un écouteur d'événement à un élément parent et de détecter les événements déclenchés par ses enfants.

// Déléguer un événement
const parent = document.querySelector('#parent');
parent.addEventListener('click', function(event) {
  if (event.target.matches('.enfant')) {
    // Code à exécuter lorsque l'enfant est cliqué
  }
});

En utilisant la méthode matches(), on peut vérifier si l'élément cible correspond à un sélecteur CSS spécifié.

Il existe de nombreux autres aspects de la manipulation du DOM, tels que la création d'éléments, la suppression d'éléments, la modification des attributs, etc. Il est important de bien comprendre ces concepts pour pouvoir créer des interactions dynamiques avec une page web.
