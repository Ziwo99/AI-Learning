Section 3: DOM Manipulation

Introduction to the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a web page as a tree-like structure, where each node in the tree represents an HTML element. The DOM allows JavaScript to interact with and manipulate the elements and content of a web page.

Selecting and manipulating DOM elements

To select DOM elements, JavaScript provides several methods such as getElementById, getElementsByClassName, and getElementsByTagName. These methods allow you to retrieve specific elements based on their ID, class, or tag name.

Once you have selected an element, you can manipulate its properties and attributes using JavaScript. For example, you can change the text content of an element using the textContent property or modify its style using the style property. 


// Selecting an element by ID
const element = document.getElementById('myElement');

// Changing the text content of an element
element.textContent = 'New text content';

// Modifying the style of an element
element.style.color = 'red';

Event handling and listeners

JavaScript allows you to handle events triggered by user interactions, such as clicks, mouse movements, or keyboard inputs. You can attach event listeners to DOM elements to execute specific code when an event occurs.

To add an event listener, you can use the addEventListener method. This method takes two arguments: the event type and a callback function that will be executed when the event occurs. 


// Adding an event listener to a button element
const button = document.getElementById('myButton');
button.addEventListener('click', function() {
  // Code to be executed when the button is clicked
  console.log('Button clicked!');
});

Creating and modifying HTML elements dynamically

JavaScript allows you to create new HTML elements and add them to the DOM dynamically. You can also modify existing elements by adding or removing attributes, classes, or styles.

To create a new element, you can use the createElement method. This method takes the tag name of the element as an argument. Once the element is created, you can set its attributes, content, and style before appending it to the DOM. 


// Creating a new paragraph element
const paragraph = document.createElement('p');

// Setting attributes and content
paragraph.textContent = 'This is a new paragraph';
paragraph.setAttribute('class', 'highlight');

// Appending the element to the DOM
const container = document.getElementById('myContainer');
container.appendChild(paragraph);
