Section 3. Manipulation du DOM

Introduction au DOM

Le DOM (Document Object Model) est une interface de programmation qui permet d'accéder et de manipuler les éléments d'une page HTML. Il représente la structure du document sous forme d'un arbre, où chaque élément est un nœud du DOM.

Sélection d'éléments

Pour sélectionner un élément du DOM, on utilise la méthode querySelector en lui passant un sélecteur CSS comme argument. Cette méthode renvoie le premier élément correspondant au sélecteur spécifié. 

const element = document.querySelector('.my-element');

Pour sélectionner plusieurs éléments, on utilise la méthode querySelectorAll. Cette méthode renvoie une NodeList contenant tous les éléments correspondant au sélecteur spécifié. 

const elements = document.querySelectorAll('.my-elements');

Modification du contenu

Pour modifier le contenu d'un élément du DOM, on peut utiliser les propriétés innerHTML ou textContent. La propriété innerHTML permet de manipuler le contenu HTML de l'élément, tandis que la propriété textContent permet de manipuler le texte brut de l'élément. 

const element = document.querySelector('.my-element');
element.innerHTML = 'Nouveau contenu HTML';
element.textContent = 'Nouveau texte brut';

Gestion des événements

Pour gérer les événements dans le DOM, on utilise la méthode addEventListener. Cette méthode prend en paramètre le type d'événement à écouter (par exemple, "click" pour un clic de souris) et une fonction de rappel qui sera exécutée lorsque l'événement se produira. 

const button = document.querySelector('.my-button');
button.addEventListener('click', () => {
  console.log('Le bouton a été cliqué');
});

Création d'éléments dynamiques

Pour créer un nouvel élément dans le DOM, on utilise la méthode createElement. Cette méthode prend en paramètre le nom de la balise de l'élément à créer. Ensuite, on peut utiliser les méthodes appendChild ou insertBefore pour insérer le nouvel élément dans le DOM. 

const newElement = document.createElement('div');
newElement.textContent = 'Nouvel élément';
document.body.appendChild(newElement);
