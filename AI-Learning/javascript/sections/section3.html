Introduction au DOM (Document Object Model)

Le DOM (Document Object Model) est une interface de programmation qui représente la structure d'un document HTML ou XML. Il permet de manipuler les éléments du document de manière dynamique en utilisant JavaScript.

Le DOM est organisé sous forme d'une hiérarchie d'objets, où chaque élément du document est représenté par un nœud. Les nœuds peuvent être des éléments HTML, des attributs, du texte ou d'autres types d'éléments.

Pour accéder et manipuler les éléments du DOM, on utilise des méthodes et des propriétés fournies par l'objet global "document". Par exemple, pour sélectionner un élément du DOM, on peut utiliser la méthode "getElementById" ou "querySelector".

Sélection des éléments du DOM

La sélection des éléments du DOM est une étape importante pour pouvoir les manipuler. Voici quelques méthodes couramment utilisées pour sélectionner des éléments du DOM :

  • getElementById(id) : Sélectionne un élément par son attribut "id".
  • getElementsByClassName(className) : Sélectionne tous les éléments qui ont une classe spécifique.
  • getElementsByTagName(tagName) : Sélectionne tous les éléments qui ont une balise HTML spécifique.
  • querySelector(selector) : Sélectionne le premier élément qui correspond à un sélecteur CSS spécifique.
  • querySelectorAll(selector) : Sélectionne tous les éléments qui correspondent à un sélecteur CSS spécifique.

Ces méthodes renvoient des objets NodeList ou HTMLCollection, qui sont des listes d'éléments du DOM. On peut accéder aux éléments individuels en utilisant leur index. 

const element = document.getElementById('myElement');
const elements = document.getElementsByClassName('myClass');
const elementsByTag = document.getElementsByTagName('div');
const elementByQuery = document.querySelector('.myClass');
const elementsByQueryAll = document.querySelectorAll('.myClass');

Modification des éléments du DOM

Une fois que nous avons sélectionné un élément du DOM, nous pouvons le modifier en utilisant les propriétés et les méthodes disponibles. Voici quelques exemples courants de modifications d'éléments du DOM :

  • textContent : Permet de modifier le contenu textuel d'un élément.
  • innerHTML : Permet de modifier le contenu HTML d'un élément.
  • setAttribute(name, value) : Permet de définir la valeur d'un attribut spécifique.
  • style.property = value : Permet de modifier les styles CSS d'un élément. 
const element = document.getElementById('myElement');
element.textContent = 'Nouveau texte';
element.innerHTML = '<strong>Texte en gras</strong>';
element.setAttribute('class', 'newClass');
element.style.color = 'red';

Gestion des événements

Les événements sont des actions ou des occurrences qui se produisent dans le navigateur, comme le clic sur un bouton ou le chargement d'une page. En JavaScript, nous pouvons réagir à ces événements en utilisant des gestionnaires d'événements.

Voici quelques exemples courants de gestion des événements :

  • addEventListener(event, callback) : Permet d'ajouter un gestionnaire d'événements à un élément spécifique.
  • removeEventListener(event, callback) : Permet de supprimer un gestionnaire d'événements d'un élément spécifique. 
const button = document.getElementById('myButton');

function handleClick() {
  console.log('Le bouton a été cliqué !');
}

button.addEventListener('click', handleClick);

// Pour supprimer le gestionnaire d'événements
button.removeEventListener('click', handleClick);

Ceci conclut la section sur la manipulation du DOM. Nous avons appris comment sélectionner des éléments du DOM, les modifier et gérer les événements. Ces connaissances sont essentielles pour créer des applications web interactives en utilisant JavaScript.
