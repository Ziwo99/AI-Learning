Section 3: Programmation orientée objet en JavaScript

Introduction à la programmation orientée objet

La programmation orientée objet (POO) est un paradigme de programmation qui permet de structurer et d'organiser le code de manière plus modulaire et réutilisable. En JavaScript, la POO est basée sur la création d'objets et l'utilisation de classes.

Les objets en JavaScript

En JavaScript, un objet est une entité qui regroupe des propriétés et des méthodes. Les propriétés représentent les caractéristiques de l'objet, tandis que les méthodes sont les actions qu'il peut effectuer. Voici un exemple d'objet JavaScript :

let voiture = {
    marque: "Toyota",
    modèle: "Corolla",
    année: 2020,
    démarrer: function() {
        console.log("La voiture démarre.");
    },
    arrêter: function() {
        console.log("La voiture s'arrête.");
    }
};

Les classes en JavaScript

En JavaScript, une classe est un modèle pour créer des objets. Elle définit les propriétés et les méthodes communes à tous les objets de cette classe. Voici un exemple de classe JavaScript :

class Animal {
    constructor(nom, espèce) {
        this.nom = nom;
        this.espèce = espèce;
    }

    faireSonCri() {
        console.log("Le " + this.nom + " fait son cri.");
    }
}

let chien = new Animal("Chien", "Canin");
chien.faireSonCri();

L'héritage en JavaScript

L'héritage en JavaScript permet à une classe d'hériter des propriétés et des méthodes d'une autre classe. Cela permet de créer une hiérarchie de classes et de réutiliser du code. Voici un exemple d'héritage en JavaScript :

class Véhicule {
    constructor(marque, modèle) {
        this.marque = marque;
        this.modèle = modèle;
    }

    démarrer() {
        console.log("Le véhicule démarre.");
    }
}

class Voiture extends Véhicule {
    constructor(marque, modèle, année) {
        super(marque, modèle);
        this.année = année;
    }

    klaxonner() {
        console.log("La voiture klaxonne.");
    }
}

let maVoiture = new Voiture("Toyota", "Corolla", 2020);
maVoiture.démarrer();
maVoiture.klaxonner();

Les modules en JavaScript

Les modules en JavaScript permettent d'organiser le code en le divisant en plusieurs fichiers. Chaque fichier représente un module qui peut être importé et utilisé dans d'autres fichiers. Voici un exemple de module en JavaScript :

// module.js
export function direBonjour() {
    console.log("Bonjour !");
}

// main.js
import { direBonjour } from './module.js';
direBonjour();

Ce sont là les bases de la programmation orientée objet en JavaScript. En comprenant ces concepts, vous serez en mesure de créer des applications plus modulaires et réutilisables. Continuez à pratiquer et à explorer d'autres fonctionnalités de la POO en JavaScript pour approfondir vos connaissances.
