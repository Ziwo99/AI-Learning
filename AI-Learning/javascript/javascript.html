Javascript

JavaScript est un langage de programmation de haut niveau, interprété et orienté objet. Il est principalement utilisé pour créer des sites web interactifs en ajoutant des fonctionnalités dynamiques et des effets visuels. Il peut également être utilisé côté serveur grâce à Node.js.
