Composants Vue

Les composants sont les blocs de construction de Vue.js. Ils permettent de créer des éléments réutilisables et modulaires dans une application Vue.js. Un composant Vue est essentiellement une instance de Vue avec des options prédéfinies.

Voici un exemple de définition d'un composant Vue :

Vue.component('mon-composant', {
  template: '<div>Ceci est mon composant</div>'
})

Dans cet exemple, nous avons défini un composant appelé "mon-composant" avec un template qui affiche simplement un div contenant le texte "Ceci est mon composant". Ce composant peut maintenant être utilisé dans d'autres parties de l'application.

Directives Vue

Les directives sont des instructions spéciales que l'on ajoute aux éléments HTML pour leur donner des comportements dynamiques. Les directives Vue sont préfixées par le préfixe "v-".

Voici un exemple d'utilisation de la directive v-if :

<div v-if="condition">
  Ce texte s'affiche si la condition est vraie.
</div>

Dans cet exemple, le div ne sera affiché que si la condition est vraie.

Interpolation et liaisons de données

L'interpolation et les liaisons de données sont des fonctionnalités clés de Vue.js qui permettent d'afficher et de mettre à jour dynamiquement les données dans le template.

Voici un exemple d'interpolation de données :

<p>{{ message }}</p>

Dans cet exemple, la variable "message" sera affichée à l'intérieur du paragraphe.

Voici un exemple de liaison de données bidirectionnelle :

<input v-model="message">

Dans cet exemple, la valeur de l'input sera liée à la variable "message". Toute modification de l'input mettra à jour la valeur de "message" et vice versa.

Gestion des événements

La gestion des événements permet de réagir aux actions de l'utilisateur, comme les clics de souris ou les saisies au clavier. Dans Vue.js, les événements sont gérés à l'aide de la directive v-on.

Voici un exemple de gestion d'un événement de clic :

<button v-on:click="handleClick">Cliquez ici</button>

Dans cet exemple, lorsque le bouton est cliqué, la méthode "handleClick" sera appelée.

Les cycles de vie des composants

Les composants Vue passent par différents cycles de vie, de leur création à leur destruction. Chaque cycle de vie correspond à un moment spécifique dans lequel des actions peuvent être effectuées.

Voici les principales étapes du cycle de vie d'un composant Vue :

  • beforeCreate: appelé avant la création du composant
  • created: appelé après la création du composant
  • beforeMount: appelé avant que le composant soit inséré dans le DOM
  • mounted: appelé après que le composant ait été inséré dans le DOM
  • beforeUpdate: appelé avant que le composant soit mis à jour
  • updated: appelé après que le composant ait été mis à jour
  • beforeDestroy: appelé avant la destruction du composant
  • destroyed: appelé après la destruction du composant

Chaque étape du cycle de vie peut être utilisée pour effectuer des actions spécifiques, comme l'initialisation des données ou la nettoyage des ressources.
