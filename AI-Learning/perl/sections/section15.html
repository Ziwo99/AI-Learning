Réalisation d'un script de traitement de fichiers

Dans cette sous-section, nous allons apprendre à créer un script Perl qui permet de traiter des fichiers. Nous allons voir comment ouvrir, lire et écrire dans des fichiers en utilisant Perl.

Voici un exemple de code qui ouvre un fichier, lit son contenu et l'affiche à l'écran :

open(my $file, '<', 'mon_fichier.txt') or die "Impossible d'ouvrir le fichier : $!";
while (my $ligne = <$file>) {
    print $ligne;
}
close($file);

Dans cet exemple, nous utilisons la fonction open pour ouvrir le fichier "mon_fichier.txt" en mode lecture ("<"). Si le fichier ne peut pas être ouvert, le programme affiche un message d'erreur et s'arrête grâce à la fonction die.

Ensuite, nous utilisons une boucle while pour lire chaque ligne du fichier et l'afficher à l'écran à l'aide de la fonction print.

Enfin, nous utilisons la fonction close pour fermer le fichier.

Création d'une application web simple

Dans cette sous-section, nous allons apprendre à créer une application web simple en utilisant Perl. Nous allons voir comment utiliser le module CGI (Common Gateway Interface) pour traiter les requêtes HTTP et générer des réponses HTML.

Voici un exemple de code qui crée un formulaire HTML et traite les données soumises par l'utilisateur :

use CGI;

my $cgi = CGI->new;

print $cgi->header;
print $cgi->start_html('Mon formulaire');
print $cgi->h1('Mon formulaire');
print $cgi->start_form;
print $cgi->textfield('nom');
print $cgi->submit('submit', 'Envoyer');
print $cgi->end_form;

if ($cgi->param('submit')) {
    my $nom = $cgi->param('nom');
    print $cgi->p("Bonjour, $nom !");
}

print $cgi->end_html;

Dans cet exemple, nous commençons par créer une instance de la classe CGI en utilisant la méthode new. Ensuite, nous utilisons les méthodes de cette instance pour générer le code HTML de notre formulaire.

La méthode header génère l'en-tête HTTP, la méthode start_html génère la balise <html> avec le titre spécifié, la méthode h1 génère un titre de niveau 1, la méthode start_form génère la balise <form>, la méthode textfield génère un champ de texte, la méthode submit génère un bouton d'envoi et la méthode end_form génère la balise de fin </form>.

Ensuite, nous utilisons la méthode param pour récupérer la valeur du champ de texte lorsque le formulaire est soumis. Si le bouton d'envoi est cliqué, nous affichons un message de salutation personnalisé à l'aide de la méthode p.

Enfin, nous utilisons les méthodes end_html pour générer la balise de fin </html> et nous imprimons le code HTML généré à l'aide de la fonction print.

Développement d'un programme orienté objet

Dans cette sous-section, nous allons apprendre à développer un programme orienté objet en Perl. Nous allons voir comment définir des classes, créer des objets et utiliser l'héritage et le polymorphisme.

Voici un exemple de code qui illustre la création d'une classe "Personne" avec des attributs et des méthodes :

package Personne;

sub new {
    my ($classe, $nom, $age) = @_;
    my $self = {
        nom => $nom,
        age => $age
    };
    bless $self, $classe;
    return $self;
}

sub parler {
    my ($self, $message) = @_;
    print "$self->{nom} dit : $message\n";
}

sub sePresenter {
    my ($self) = @_;
    print "Je m'appelle $self->{nom} et j'ai $self->{age} ans.\n";
}

1;

Dans cet exemple, nous commençons par définir un package Perl appelé "Personne". À l'intérieur de ce package, nous définissons une méthode new qui sert de constructeur pour la classe "Personne". Cette méthode prend en paramètres le nom et l'âge de la personne, crée un nouvel objet et le bénit avec la classe "Personne" à l'aide de la fonction bless.

Nous définissons également deux autres méthodes : parler et sePresenter. La méthode parler prend en paramètres un message et l'affiche à l'écran avec le nom de la personne. La méthode sePresenter affiche simplement le nom et l'âge de la personne.

Pour utiliser cette classe, nous pouvons créer un nouvel objet "Personne" en utilisant la méthode new :

my $personne = Personne->new('John Doe', 30);

Ensuite, nous pouvons appeler les méthodes de cet objet :

$personne->parler('Bonjour !');
$personne->sePresenter();

Cela affichera :

John Doe dit : Bonjour !
Je m'appelle John Doe et j'ai 30 ans.

Ce sont là quelques exemples de projets pratiques que vous pouvez réaliser en Perl. N'hésitez pas à explorer davantage et à créer vos propres projets pour approfondir vos connaissances en Perl.
