Structures conditionnelles (if, else, elsif)

Les structures conditionnelles en Perl permettent d'exécuter des blocs de code en fonction de conditions spécifiques. La structure if permet d'exécuter un bloc de code si une condition est vraie. Voici un exemple :

my $age = 25;

if ($age >= 18) {
    print "Vous êtes majeur.";
}

Dans cet exemple, si la variable $age est supérieure ou égale à 18, le message "Vous êtes majeur." sera affiché.

La structure else permet d'exécuter un bloc de code si la condition de l'instruction if est fausse. Voici un exemple :

my $age = 15;

if ($age >= 18) {
    print "Vous êtes majeur.";
} else {
    print "Vous êtes mineur.";
}

Dans cet exemple, si la variable $age est inférieure à 18, le message "Vous êtes mineur." sera affiché.

La structure elsif permet d'ajouter des conditions supplémentaires à une structure if. Voici un exemple :

my $age = 25;

if ($age < 18) {
    print "Vous êtes mineur.";
} elsif ($age < 25) {
    print "Vous êtes jeune adulte.";
} else {
    print "Vous êtes adulte.";
}

Dans cet exemple, si la variable $age est inférieure à 18, le message "Vous êtes mineur." sera affiché. Si la variable $age est supérieure ou égale à 18 mais inférieure à 25, le message "Vous êtes jeune adulte." sera affiché. Sinon, le message "Vous êtes adulte." sera affiché.

Boucles (for, while, do-while)

Les boucles en Perl permettent d'exécuter un bloc de code plusieurs fois. Il existe trois types de boucles couramment utilisées en Perl : la boucle for, la boucle while et la boucle do-while.

La boucle for permet d'exécuter un bloc de code un nombre spécifié de fois. Voici un exemple :

for (my $i = 0; $i < 5; $i++) {
    print "Iteration $i\n";
}

Dans cet exemple, le bloc de code à l'intérieur de la boucle for sera exécuté cinq fois, avec la variable $i prenant les valeurs de 0 à 4.

La boucle while permet d'exécuter un bloc de code tant qu'une condition spécifiée est vraie. Voici un exemple :

my $i = 0;

while ($i < 5) {
    print "Iteration $i\n";
    $i++;
}

Dans cet exemple, le bloc de code à l'intérieur de la boucle while sera exécuté tant que la variable $i est inférieure à 5. À chaque itération, la variable $i est incrémentée de 1.

La boucle do-while est similaire à la boucle while, mais elle exécute le bloc de code au moins une fois, puis vérifie la condition. Voici un exemple :

my $i = 0;

do {
    print "Iteration $i\n";
    $i++;
} while ($i < 5);

Dans cet exemple, le bloc de code à l'intérieur de la boucle do-while sera exécuté une première fois, puis la condition sera vérifiée. Si la condition est vraie, le bloc de code sera exécuté à nouveau, et ainsi de suite, tant que la condition est vraie.

Instructions de contrôle (next, last, redo)

Les instructions de contrôle en Perl permettent de contrôler le flux d'exécution d'une boucle. Il existe trois instructions de contrôle couramment utilisées en Perl : next, last et redo.

L'instruction next permet de passer à l'itération suivante d'une boucle. Voici un exemple :

for (my $i = 0; $i < 5; $i++) {
    if ($i == 2) {
        next;
    }
    print "Iteration $i\n";
}

Dans cet exemple, lorsque la variable $i est égale à 2, l'instruction next est exécutée, ce qui passe à l'itération suivante de la boucle sans exécuter le reste du bloc de code.

L'instruction last permet de sortir d'une boucle prématurément. Voici un exemple :

for (my $i = 0; $i < 5; $i++) {
    if ($i == 2) {
        last;
    }
    print "Iteration $i\n";
}

Dans cet exemple, lorsque la variable $i est égale à 2, l'instruction last est exécutée, ce qui sort de la boucle sans exécuter le reste du bloc de code.

L'instruction redo permet de recommencer l'itération actuelle d'une boucle. Voici un exemple :

for (my $i = 0; $i < 5; $i++) {
    if ($i == 2) {
        redo;
    }
    print "Iteration $i\n";
}

Dans cet exemple, lorsque la variable $i est égale à 2, l'instruction redo est exécutée, ce qui recommence l'itération actuelle de la boucle sans incrémenter la variable $i.
