Section 2 : Variables et types de données en Perl

Variables scalaires

En Perl, les variables scalaires sont utilisées pour stocker des valeurs individuelles. Elles peuvent contenir des nombres, des chaînes de caractères ou des références à d'autres types de données.

Voici comment déclarer une variable scalaire en Perl :

my $nom_variable;

Il est également possible d'initialiser une variable scalaire lors de sa déclaration :

my $nom_variable = valeur;

Les variables scalaires peuvent être utilisées dans des opérations arithmétiques, des opérations de chaînes de caractères et d'autres manipulations de données.

Variables tableaux

En Perl, les variables tableaux sont utilisées pour stocker des listes de valeurs. Elles peuvent contenir des nombres, des chaînes de caractères ou des références à d'autres types de données.

Voici comment déclarer un tableau en Perl :

my @nom_tableau;

Il est également possible d'initialiser un tableau lors de sa déclaration :

my @nom_tableau = (valeur1, valeur2, valeur3);

Les éléments d'un tableau peuvent être accédés en utilisant leur indice :

my $element = $nom_tableau[indice];

Les tableaux peuvent être utilisés dans des boucles, des opérations de recherche et de tri, et d'autres manipulations de données.

Variables hachages

En Perl, les variables hachages sont utilisées pour stocker des paires clé-valeur. Elles sont également appelées tableaux associatifs ou dictionnaires dans d'autres langages de programmation.

Voici comment déclarer une hachage en Perl :

my %nom_hachage;

Il est également possible d'initialiser une hachage lors de sa déclaration :

my %nom_hachage = (clé1 => valeur1, clé2 => valeur2, clé3 => valeur3);

Les valeurs d'une hachage peuvent être accédées en utilisant leur clé :

my $valeur = $nom_hachage{clé};

Les hachages peuvent être utilisées dans des opérations de recherche, de tri et de manipulation de données basées sur des clés.

Variables de type référence

En Perl, les variables de type référence sont utilisées pour stocker des références à d'autres types de données, tels que des tableaux, des hachages ou des sous-routines.

Voici comment déclarer une variable de type référence en Perl :

my $nom_reference = \valeur;

Les références peuvent être utilisées pour accéder aux données contenues dans une structure de données complexe, ou pour passer des structures de données en tant qu'arguments à des sous-routines.

Il est important de noter que les références doivent être déréférencées avant d'accéder à leurs valeurs réelles. Cela peut être fait en utilisant l'opérateur de déréférencement approprié, tel que $ pour les scalaires, @ pour les tableaux et % pour les hachages.

my $valeur = $$nom_reference;  # déréférencement d'une référence scalaire
my @tableau = @$nom_reference;  # déréférencement d'une référence tableau
my %hachage = %$nom_reference;  # déréférencement d'une référence hachage
