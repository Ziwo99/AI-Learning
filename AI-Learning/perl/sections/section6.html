Section 6 : Manipulation de fichiers en Perl

Ouverture, lecture et écriture de fichiers

En Perl, la manipulation de fichiers est une tâche courante. Pour ouvrir un fichier en lecture ou en écriture, vous pouvez utiliser la fonction open(). Voici un exemple :

open(my $file, '<', 'nom_du_fichier.txt') or die "Impossible d'ouvrir le fichier : $!";

Dans cet exemple, nous ouvrons le fichier nom_du_fichier.txt en mode lecture (<). Si le fichier ne peut pas être ouvert, la fonction die() affiche un message d'erreur.

Une fois le fichier ouvert, vous pouvez lire son contenu ligne par ligne à l'aide de la fonction readline(). Voici un exemple :

while (my $ligne = <$file>) {
    print $ligne;
}

Dans cet exemple, nous utilisons une boucle while pour lire chaque ligne du fichier. La variable $ligne contient le contenu de chaque ligne.

Pour écrire dans un fichier, vous pouvez utiliser la fonction print(). Voici un exemple :

print $file "Contenu à écrire dans le fichier";

Dans cet exemple, nous utilisons la fonction print() pour écrire le texte "Contenu à écrire dans le fichier" dans le fichier ouvert.

Une fois que vous avez terminé de lire ou d'écrire dans un fichier, vous devez le fermer à l'aide de la fonction close(). Voici un exemple :

close($file);

Dans cet exemple, nous utilisons la fonction close() pour fermer le fichier ouvert.

Manipulation de fichiers CSV

En Perl, la manipulation de fichiers CSV (Comma-Separated Values) est courante lors du traitement de données tabulaires. Pour cela, vous pouvez utiliser le module Text::CSV. Voici un exemple :

use Text::CSV;

my $csv = Text::CSV->new({ sep_char => ',' });

open(my $file, '<', 'nom_du_fichier.csv') or die "Impossible d'ouvrir le fichier : $!";

while (my $ligne = <$file>) {
    if ($csv->parse($ligne)) {
        my @colonnes = $csv->fields();
        # Faites quelque chose avec les colonnes
    } else {
        my $erreur = $csv->error_input;
        print "Erreur de parsing : $erreur";
    }
}

close($file);

Dans cet exemple, nous utilisons le module Text::CSV pour manipuler le fichier CSV. Nous ouvrons le fichier en lecture, puis nous utilisons la méthode parse() pour analyser chaque ligne du fichier. Si la ligne est correctement analysée, nous pouvons accéder aux colonnes à l'aide de la méthode fields().

Si une erreur se produit lors du parsing, nous pouvons utiliser la méthode error_input() pour obtenir le message d'erreur.

Manipulation de fichiers texte

En Perl, la manipulation de fichiers texte est similaire à la manipulation de fichiers normaux. Vous pouvez utiliser les mêmes fonctions open(), print() et close() pour ouvrir, écrire et fermer un fichier texte.

Voici un exemple d'ouverture, de lecture et d'écriture d'un fichier texte :

open(my $file, '<', 'nom_du_fichier.txt') or die "Impossible d'ouvrir le fichier : $!";

while (my $ligne = <$file>) {
    print $ligne;
}

print $file "Contenu à écrire dans le fichier";

close($file);

Dans cet exemple, nous ouvrons le fichier texte nom_du_fichier.txt en lecture, nous lisons chaque ligne du fichier à l'aide d'une boucle while, puis nous écrivons du contenu dans le fichier à l'aide de la fonction print().

N'oubliez pas de fermer le fichier une fois que vous avez terminé de le manipuler.
