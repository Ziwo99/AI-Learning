Opérateurs arithmétiques

Les opérateurs arithmétiques permettent d'effectuer des opérations mathématiques sur des valeurs numériques en Perl. Voici les opérateurs arithmétiques disponibles :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
  • % : modulo (reste de la division)
  • ** : exponentiation

Exemple :

my $a = 10;
my $b = 5;

my $sum = $a + $b;          # Addition
my $difference = $a - $b;   # Soustraction
my $product = $a * $b;      # Multiplication
my $quotient = $a / $b;     # Division
my $remainder = $a % $b;    # Modulo
my $power = $a ** $b;       # Exponentiation

print "Sum: $sum\n";
print "Difference: $difference\n";
print "Product: $product\n";
print "Quotient: $quotient\n";
print "Remainder: $remainder\n";
print "Power: $power\n";

Opérateurs de comparaison

Les opérateurs de comparaison permettent de comparer des valeurs en Perl. Voici les opérateurs de comparaison disponibles :

  • == : égal à
  • != : différent de
  • < : inférieur à
  • > : supérieur à
  • <= : inférieur ou égal à
  • >= : supérieur ou égal à

Exemple :

my $a = 10;
my $b = 5;

if ($a == $b) {
    print "$a is equal to $b\n";
} else {
    print "$a is not equal to $b\n";
}

if ($a != $b) {
    print "$a is not equal to $b\n";
} else {
    print "$a is equal to $b\n";
}

if ($a < $b) {
    print "$a is less than $b\n";
} else {
    print "$a is not less than $b\n";
}

if ($a > $b) {
    print "$a is greater than $b\n";
} else {
    print "$a is not greater than $b\n";
}

if ($a <= $b) {
    print "$a is less than or equal to $b\n";
} else {
    print "$a is not less than or equal to $b\n";
}

if ($a >= $b) {
    print "$a is greater than or equal to $b\n";
} else {
    print "$a is not greater than or equal to $b\n";
}

Opérateurs logiques

Les opérateurs logiques permettent de combiner des expressions booléennes en Perl. Voici les opérateurs logiques disponibles :

  • && : ET logique
  • || : OU logique
  • ! : NON logique

Exemple :

my $a = 10;
my $b = 5;
my $c = 7;

if ($a > $b && $a > $c) {
    print "$a is the largest number\n";
}

if ($a > $b || $a > $c) {
    print "$a is greater than either $b or $c\n";
}

if (!($a < $b)) {
    print "$a is not less than $b\n";
}

Opérateurs de chaînes de caractères

Les opérateurs de chaînes de caractères permettent de concaténer des chaînes en Perl. Voici les opérateurs de chaînes de caractères disponibles :

  • . : concaténation
  • x : répétition

Exemple :

my $str1 = "Hello";
my $str2 = "World";

my $concatenated = $str1 . " " . $str2;   # Concaténation
my $repeated = $str1 x 3;                 # Répétition

print "$concatenated\n";
print "$repeated\n";
