Syntaxe de base

La syntaxe de base de Lua est simple et facile à comprendre. Voici un exemple de code Lua :

print("Bonjour, Lua!")

Dans cet exemple, la fonction print est utilisée pour afficher le texte "Bonjour, Lua!" à l'écran.

Variables et types de données

En Lua, les variables sont utilisées pour stocker des valeurs. Voici comment déclarer et utiliser une variable en Lua :

local age = 25
local nom = "Jean"

Dans cet exemple, la variable age est déclarée et initialisée avec la valeur 25, et la variable nom est déclarée et initialisée avec la valeur "Jean".

Il existe plusieurs types de données en Lua, tels que les nombres, les chaînes de caractères, les booléens, les tableaux, etc. Voici quelques exemples :

local age = 25
local nom = "Jean"
local estVrai = true
local tableau = {1, 2, 3, 4, 5}

Dans cet exemple, la variable age est de type nombre, la variable nom est de type chaîne de caractères, la variable estVrai est de type booléen, et la variable tableau est de type tableau.

Opérations arithmétiques et logiques

En Lua, vous pouvez effectuer des opérations arithmétiques telles que l'addition, la soustraction, la multiplication, la division, etc. Voici quelques exemples :

local a = 10
local b = 5

local addition = a + b
local soustraction = a - b
local multiplication = a * b
local division = a / b

Dans cet exemple, la variable addition contient le résultat de l'addition de a et b, la variable soustraction contient le résultat de la soustraction de a et b, etc.

En plus des opérations arithmétiques, Lua prend également en charge les opérations logiques telles que l'ET logique (and), le OU logique (or), et la négation logique (not). Voici quelques exemples :

local a = true
local b = false

local etLogique = a and b
local ouLogique = a or b
local negationLogique = not a

Dans cet exemple, la variable etLogique contient le résultat de l'opération ET logique entre a et b, la variable ouLogique contient le résultat de l'opération OU logique entre a et b, etc.

Structures de contrôle (boucles et conditions)

En Lua, vous pouvez utiliser des structures de contrôle telles que les boucles et les conditions pour contrôler le flux d'exécution de votre programme. Voici quelques exemples :

Utilisation de la boucle while :

local i = 0

while i < 10 do
    print("Iteration " .. i)
    i = i + 1
end

Dans cet exemple, la boucle while s'exécute tant que la condition i < 10 est vraie. À chaque itération, le texte "Iteration" suivi de la valeur de i est affiché à l'écran, puis la valeur de i est incrémentée de 1.

Utilisation de la boucle for :

for i = 1, 10 do
    print("Iteration " .. i)
end

Dans cet exemple, la boucle for s'exécute pour chaque valeur de i allant de 1 à 10. À chaque itération, le texte "Iteration" suivi de la valeur de i est affiché à l'écran.

Utilisation de la structure de condition if :

local age = 25

if age >= 18 then
    print("Vous êtes majeur.")
else
    print("Vous êtes mineur.")
end

Dans cet exemple, si la valeur de age est supérieure ou égale à 18, le texte "Vous êtes majeur." est affiché à l'écran. Sinon, le texte "Vous êtes mineur." est affiché à l'écran.

Fonctions et procédures

En Lua, vous pouvez définir vos propres fonctions et procédures pour regrouper du code et le réutiliser. Voici comment définir une fonction en Lua :

function direBonjour(nom)
    print("Bonjour, " .. nom .. "!")
end

direBonjour("Jean")

Dans cet exemple, la fonction direBonjour est définie avec un paramètre nom. À l'intérieur de la fonction, le texte "Bonjour," suivi de la valeur du paramètre nom, est affiché à l'écran. Ensuite, la fonction est appelée avec l'argument "Jean".

Les procédures sont similaires aux fonctions, mais elles ne renvoient pas de valeur. Voici un exemple de procédure en Lua :

function afficherMessage()
    print("Ceci est un message.")
end

afficherMessage()

Dans cet exemple, la procédure afficherMessage est définie. À l'intérieur de la procédure, le texte "Ceci est un message." est affiché à l'écran. Ensuite, la procédure est appelée.
