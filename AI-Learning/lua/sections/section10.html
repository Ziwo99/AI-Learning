Création d'un jeu simple en Lua

Dans cette sous-section, nous allons apprendre à créer un jeu simple en Lua. Nous allons utiliser les concepts que nous avons appris dans les sections précédentes pour développer un jeu de devinette.

Voici le code pour notre jeu de devinette :

math.randomseed(os.time())

function devinette()
    local nombreAleatoire = math.random(1, 100)
    local devine = false

    print("Bienvenue dans le jeu de devinette !")
    print("Je pense à un nombre entre 1 et 100. Devine quel est ce nombre.")

    while not devine do
        local proposition = tonumber(io.read())

        if proposition == nombreAleatoire then
            print("Bravo ! Tu as deviné le bon nombre.")
            devine = true
        elseif proposition < nombreAleatoire then
            print("Non, c'est plus grand. Essaye encore.")
        else
            print("Non, c'est plus petit. Essaye encore.")
        end
    end
end

devinette()

Explication du code :

Tout d'abord, nous utilisons la fonction math.randomseed(os.time()) pour initialiser la génération de nombres aléatoires avec une graine basée sur l'heure actuelle. Cela garantit que chaque fois que nous exécutons le jeu, nous obtenons un nombre aléatoire différent.

Ensuite, nous définissons une fonction devinette() qui contient la logique du jeu. À l'intérieur de cette fonction, nous déclarons une variable nombreAleatoire qui contient le nombre aléatoire que le joueur doit deviner. Nous initialisons également une variable devine à false pour indiquer que le joueur n'a pas encore deviné le nombre.

Nous utilisons ensuite une boucle while pour permettre au joueur de faire des propositions jusqu'à ce qu'il devine le bon nombre. À chaque itération de la boucle, nous demandons au joueur de saisir une proposition à l'aide de la fonction io.read(). Nous convertissons ensuite cette proposition en un nombre en utilisant la fonction tonumber().

Ensuite, nous comparons la proposition du joueur avec le nombre aléatoire. Si la proposition est égale au nombre aléatoire, nous affichons un message de félicitations et nous mettons la variable devine à true pour sortir de la boucle. Sinon, nous affichons un message indiquant au joueur si la proposition est trop grande ou trop petite par rapport au nombre aléatoire.

Enfin, nous appelons la fonction devinette() pour démarrer le jeu.

Développement d'une application web en Lua

Dans cette sous-section, nous allons apprendre à développer une application web en Lua en utilisant le framework web OpenResty.

Voici un exemple de code pour une application web simple :

local resty_http = require "resty.http"

local httpc = resty_http.new()
local res, err = httpc:request_uri("https://api.example.com/data", {
    method = "GET",
    headers = {
        ["Content-Type"] = "application/json",
    },
})

if not res then
    ngx.say("Erreur lors de la requête : ", err)
    return
end

ngx.say("Statut de la réponse : ", res.status)
ngx.say("Corps de la réponse : ", res.body)

Explication du code :

Tout d'abord, nous importons le module resty.http en utilisant la ligne de code local resty_http = require "resty.http". Ce module nous permet d'effectuer des requêtes HTTP depuis notre application web.

Ensuite, nous créons une instance de la classe resty.http en utilisant la ligne de code local httpc = resty_http.new(). Cette instance sera utilisée pour effectuer la requête HTTP.

Nous utilisons ensuite la méthode request_uri() de l'instance httpc pour effectuer une requête GET vers l'URL "https://api.example.com/data". Nous passons également un tableau de paramètres supplémentaires à cette méthode, tels que les en-têtes de la requête.

Si la requête est réussie, nous affichons le statut de la réponse et le corps de la réponse en utilisant la fonction ngx.say(). Si la requête échoue, nous affichons un message d'erreur.

Ce code est un exemple simple pour illustrer comment effectuer une requête HTTP dans une application web en Lua. Dans une application web réelle, vous auriez probablement besoin de gérer les routes, les modèles de vue, les sessions, etc. OpenResty fournit des fonctionnalités supplémentaires pour faciliter le développement d'applications web en Lua.
