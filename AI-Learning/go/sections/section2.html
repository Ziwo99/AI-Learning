Syntaxe de base et conventions de nommage

En Go, la syntaxe de base suit une approche simple et concise. Les instructions sont terminées par un point-virgule facultatif, et les blocs de code sont délimités par des accolades. Voici un exemple de code Go :

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

En ce qui concerne les conventions de nommage, Go suit la convention camelCase pour les noms de variables et de fonctions. Les noms de variables et de fonctions doivent commencer par une lettre majuscule pour être exportés et utilisés à l'extérieur du package. Voici un exemple :

package main

import "fmt"

func main() {
    var myVariable int
    myVariable = 10

    fmt.Println(myVariable)
}

Types de données et variables

Go est un langage fortement typé, ce qui signifie que chaque variable doit avoir un type spécifié. Voici quelques types de données de base en Go :

  • int : entier signé
  • float64 : nombre à virgule flottante
  • string : chaîne de caractères
  • bool : valeur booléenne (true ou false)

Pour déclarer une variable en Go, nous utilisons le mot-clé "var" suivi du nom de la variable et du type :

package main

import "fmt"

func main() {
    var age int
    age = 25

    fmt.Println("J'ai", age, "ans.")
}

Opérations arithmétiques et opérateurs logiques

Go prend en charge les opérations arithmétiques de base telles que l'addition, la soustraction, la multiplication et la division. Voici un exemple :

package main

import "fmt"

func main() {
    var a = 10
    var b = 5

    fmt.Println("a + b =", a + b)
    fmt.Println("a - b =", a - b)
    fmt.Println("a * b =", a * b)
    fmt.Println("a / b =", a / b)
}

En plus des opérations arithmétiques, Go propose également des opérateurs logiques tels que "&&" (ET), "||" (OU) et "!" (NON). Voici un exemple :

package main

import "fmt"

func main() {
    var a = true
    var b = false

    fmt.Println("a && b =", a && b)
    fmt.Println("a || b =", a || b)
    fmt.Println("!a =", !a)
}

Structures de contrôle : boucles et conditions

En Go, nous avons plusieurs structures de contrôle pour gérer les boucles et les conditions. Voici un exemple d'utilisation de la boucle "for" :

package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}

En ce qui concerne les conditions, nous utilisons le mot-clé "if" suivi de la condition. Voici un exemple :

package main

import "fmt"

func main() {
    var age = 18

    if age >= 18 {
        fmt.Println("Vous êtes majeur.")
    } else {
        fmt.Println("Vous êtes mineur.")
    }
}

Fonctions et méthodes

En Go, les fonctions sont déclarées à l'aide du mot-clé "func" suivi du nom de la fonction, des paramètres entre parenthèses et du type de retour. Voici un exemple :

package main

import "fmt"

func add(a, b int) int {
    return a + b
}

func main() {
    result := add(5, 3)
    fmt.Println("Le résultat est", result)
}

En plus des fonctions, Go prend également en charge les méthodes, qui sont des fonctions liées à un type spécifique. Voici un exemple :

package main

import "fmt"

type Rectangle struct {
    width  int
    height int
}

func (r Rectangle) area() int {
    return r.width * r.height
}

func main() {
    rect := Rectangle{width: 5, height: 3}
    fmt.Println("L'aire du rectangle est", rect.area())
}
