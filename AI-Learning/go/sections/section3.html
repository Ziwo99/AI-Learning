Tableaux et tranches (slices)

En Go, un tableau est une séquence de valeurs de même type, tandis qu'une tranche (slice) est une vue dynamique sur un tableau. Les tranches sont plus flexibles que les tableaux car elles peuvent être redimensionnées et manipulées plus facilement.

Voici comment déclarer et initialiser un tableau en Go :

var tableau [5]int // Déclaration d'un tableau de taille fixe de 5 éléments de type int
tableau = [5]int{1, 2, 3, 4, 5} // Initialisation du tableau avec des valeurs

Et voici comment déclarer et initialiser une tranche en Go :

var tranche []int // Déclaration d'une tranche vide
tranche = []int{1, 2, 3, 4, 5} // Initialisation de la tranche avec des valeurs

Les tranches peuvent être manipulées à l'aide de fonctions intégrées telles que append pour ajouter des éléments, copy pour copier une tranche, et len pour obtenir la longueur d'une tranche.

Maps (dictionnaires)

En Go, une map est une structure de données qui associe des clés à des valeurs. Les maps sont similaires aux dictionnaires dans d'autres langages de programmation.

Voici comment déclarer et initialiser une map en Go :

var m map[string]int // Déclaration d'une map vide avec des clés de type string et des valeurs de type int
m = make(map[string]int) // Initialisation de la map
m["clé1"] = 1 // Ajout d'une paire clé-valeur à la map
m["clé2"] = 2

Les maps peuvent être utilisées pour rechercher des valeurs à l'aide de clés, ajouter ou supprimer des paires clé-valeur, et itérer sur les paires clé-valeur à l'aide d'une boucle for.

Structures (structs)

En Go, une structure (struct) est une collection de champs de données de différents types. Les structures sont utilisées pour regrouper des données connexes et les manipuler en tant qu'unité.

Voici comment déclarer et initialiser une structure en Go :

type Personne struct {
    nom string
    age int
}

var p Personne // Déclaration d'une variable de type Personne
p = Personne{"John Doe", 30} // Initialisation de la structure avec des valeurs

Les structures peuvent être utilisées pour créer des objets avec des propriétés et des comportements. Les champs d'une structure peuvent être accédés à l'aide de la notation pointée (p.nom) ou de la notation avec point ((*p).nom).

Pointeurs et références

En Go, un pointeur est une variable qui contient l'adresse mémoire d'une autre variable. Les pointeurs sont utilisés pour référencer et manipuler des variables indirectement.

Voici comment déclarer et utiliser un pointeur en Go :

var x int = 10
var p *int // Déclaration d'un pointeur vers un entier
p = &x // Attribution de l'adresse de x au pointeur p
fmt.Println(*p) // Affiche la valeur de x en utilisant le pointeur p

Les pointeurs peuvent être utilisés pour passer des variables par référence, allouer dynamiquement de la mémoire et accéder à des variables dans des structures complexes.
