Fonctions de manipulation de chaînes de caractères

Les fonctions de manipulation de chaînes de caractères permettent de manipuler et de modifier les valeurs de type chaîne de caractères dans MySQL. Voici quelques-unes des fonctions les plus couramment utilisées :

SELECT CONCAT('Hello', ' ', 'World') AS ConcatenatedString;
-- Résultat : 'Hello World'

SELECT UPPER('hello') AS UppercaseString;
-- Résultat : 'HELLO'

SELECT LOWER('WORLD') AS LowercaseString;
-- Résultat : 'world'

SELECT LENGTH('MySQL') AS StringLength;
-- Résultat : 5

SELECT SUBSTRING('Hello World', 7) AS Substring;
-- Résultat : 'World'

SELECT REPLACE('Hello World', 'World', 'MySQL') AS ReplacedString;
-- Résultat : 'Hello MySQL'

Fonctions mathématiques et statistiques

Les fonctions mathématiques et statistiques permettent d'effectuer des opérations mathématiques et statistiques sur les valeurs numériques dans MySQL. Voici quelques exemples de fonctions couramment utilisées :

SELECT ABS(-10) AS AbsoluteValue;
-- Résultat : 10

SELECT ROUND(3.14159, 2) AS RoundedValue;
-- Résultat : 3.14

SELECT CEILING(4.5) AS CeilingValue;
-- Résultat : 5

SELECT FLOOR(4.5) AS FloorValue;
-- Résultat : 4

SELECT AVG(salary) AS AverageSalary FROM employees;
-- Résultat : Moyenne des salaires de tous les employés

SELECT SUM(sales) AS TotalSales FROM orders;
-- Résultat : Somme des ventes de toutes les commandes

Fonctions de date et heure

Les fonctions de date et heure permettent de manipuler et de formater les valeurs de type date et heure dans MySQL. Voici quelques exemples de fonctions couramment utilisées :

SELECT CURDATE() AS CurrentDate;
-- Résultat : Date actuelle

SELECT CURTIME() AS CurrentTime;
-- Résultat : Heure actuelle

SELECT NOW() AS CurrentDateTime;
-- Résultat : Date et heure actuelles

SELECT DATE_FORMAT(order_date, '%Y-%m-%d') AS FormattedDate FROM orders;
-- Résultat : Date formatée selon le modèle spécifié

Opérations sur les données NULL

Les opérations sur les données NULL permettent de gérer les valeurs NULL dans MySQL. Voici quelques exemples d'opérations couramment utilisées :

SELECT column1 IS NULL AS IsNull FROM table;
-- Résultat : Vrai si la valeur de column1 est NULL, sinon faux

SELECT column1 IS NOT NULL AS IsNotNull FROM table;
-- Résultat : Vrai si la valeur de column1 n'est pas NULL, sinon faux

SELECT COALESCE(column1, column2, column3) AS CoalescedValue FROM table;
-- Résultat : Première valeur non NULL parmi column1, column2 et column3

Sous-requêtes

Les sous-requêtes permettent d'inclure une requête à l'intérieur d'une autre requête dans MySQL. Voici un exemple de sous-requête :

SELECT column1 FROM table1 WHERE column2 IN (SELECT column3 FROM table2);
-- Résultat : Sélectionne les valeurs de column1 dans table1 où column2 est présent dans les valeurs de column3 dans table2

Vues

Les vues permettent de créer des vues virtuelles basées sur les résultats d'une requête dans MySQL. Voici un exemple de création de vue :

CREATE VIEW view_name AS
SELECT column1, column2
FROM table
WHERE condition;

Une fois la vue créée, vous pouvez l'utiliser comme une table normale dans vos requêtes.

Ces fonctions et opérations avancées vous permettent de manipuler et de traiter les données de manière plus avancée dans MySQL. Elles sont essentielles pour des tâches telles que la transformation des données, les calculs complexes et l'optimisation des requêtes.
