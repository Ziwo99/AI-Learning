Vecteurisation des opérations

La vecteurisation des opérations est une technique clé pour optimiser les performances de NumPy. Elle consiste à effectuer des opérations sur des tableaux entiers plutôt que sur des éléments individuels. Cela permet d'exploiter les capacités de calcul parallèle des processeurs modernes.

Par exemple, au lieu d'utiliser une boucle pour multiplier chaque élément d'un tableau par un scalaire, vous pouvez simplement multiplier le tableau entier par ce scalaire :

a = np.array([1, 2, 3, 4, 5])
b = 2

result = a * b

print(result)

Cette opération est beaucoup plus rapide que d'itérer sur chaque élément du tableau et de le multiplier individuellement.

Utilisation de mémoire efficace avec NumPy

NumPy offre des fonctionnalités pour optimiser l'utilisation de la mémoire. Par exemple, vous pouvez spécifier le type de données des tableaux pour minimiser l'espace mémoire utilisé :

a = np.array([1, 2, 3], dtype=np.int8)

print(a.dtype)

Ici, nous avons spécifié que les éléments du tableau a sont des entiers signés sur 8 bits. Cela permet de réduire la consommation de mémoire par rapport à l'utilisation du type de données par défaut (int32).

Utilisation de fonctions universelles optimisées

NumPy propose des fonctions universelles (ufuncs) qui sont des opérations mathématiques et logiques optimisées pour les tableaux. Ces fonctions sont implémentées en C, ce qui les rend beaucoup plus rapides que les opérations équivalentes en Python pur.

Par exemple, la fonction np.sin() calcule le sinus de chaque élément d'un tableau :

a = np.array([0, np.pi/2, np.pi])

result = np.sin(a)

print(result)

Cette fonction universelle est bien plus rapide que si vous aviez utilisé une boucle pour calculer le sinus de chaque élément individuellement.

Utilisation de fonctions vectorisées

NumPy propose également des fonctions vectorisées qui permettent d'appliquer une fonction à chaque élément d'un tableau. Ces fonctions sont optimisées pour les performances et peuvent être utilisées pour effectuer des opérations complexes sur les tableaux.

Par exemple, la fonction np.vectorize() permet d'appliquer une fonction personnalisée à chaque élément d'un tableau :

def square(x):
    return x ** 2

v_square = np.vectorize(square)

a = np.array([1, 2, 3, 4, 5])

result = v_square(a)

print(result)

Cette fonction vectorisée est plus rapide que si vous aviez utilisé une boucle pour appliquer la fonction à chaque élément individuellement.

En utilisant ces techniques de performances et d'optimisation, vous pouvez améliorer considérablement les performances de vos calculs avec NumPy.
