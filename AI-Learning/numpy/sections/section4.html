Changement de forme des tableaux

La manipulation de la forme d'un tableau est une opération courante lors de l'utilisation de NumPy. NumPy offre plusieurs méthodes pour changer la forme d'un tableau selon nos besoins.

La méthode la plus courante pour changer la forme d'un tableau est la fonction reshape(). Cette fonction permet de redimensionner un tableau en spécifiant les nouvelles dimensions souhaitées. Par exemple, pour redimensionner un tableau en une matrice de 2 lignes et 3 colonnes, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([1, 2, 3, 4, 5, 6])
reshaped_arr = arr.reshape(2, 3)

print(reshaped_arr)

Résultat :

[[1 2 3]
 [4 5 6]]

Une autre méthode couramment utilisée pour changer la forme d'un tableau est la fonction resize(). Cette fonction permet de redimensionner un tableau en modifiant directement le tableau existant. Par exemple, pour redimensionner un tableau en une matrice de 2 lignes et 3 colonnes, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([1, 2, 3, 4, 5, 6])
arr.resize(2, 3)

print(arr)

Résultat :

[[1 2 3]
 [4 5 6]]

Il est également possible de changer la forme d'un tableau en utilisant la fonction transpose(). Cette fonction permet de permuter les axes d'un tableau multidimensionnel. Par exemple, pour permuter les lignes et les colonnes d'une matrice, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])
transposed_arr = arr.transpose()

print(transposed_arr)

Résultat :

[[1 4]
 [2 5]
 [3 6]]

Concaténation et division de tableaux

La concaténation et la division de tableaux sont des opérations courantes lors de la manipulation de tableaux avec NumPy. NumPy offre plusieurs méthodes pour concaténer et diviser des tableaux selon nos besoins.

La méthode la plus courante pour concaténer des tableaux est la fonction concatenate(). Cette fonction permet de concaténer plusieurs tableaux le long d'un axe spécifié. Par exemple, pour concaténer deux tableaux en une seule dimension, on peut utiliser la syntaxe suivante :

import numpy as np

arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
concatenated_arr = np.concatenate((arr1, arr2))

print(concatenated_arr)

Résultat :

[1 2 3 4 5 6]

Il est également possible de concaténer des tableaux le long d'un axe spécifié en utilisant la fonction stack(). Cette fonction permet de concaténer plusieurs tableaux en les empilant les uns sur les autres. Par exemple, pour empiler deux tableaux en une seule dimension, on peut utiliser la syntaxe suivante :

import numpy as np

arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
stacked_arr = np.stack((arr1, arr2))

print(stacked_arr)

Résultat :

[[1 2 3]
 [4 5 6]]

La division d'un tableau peut être réalisée en utilisant la fonction split(). Cette fonction permet de diviser un tableau en plusieurs sous-tableaux selon un indice spécifié. Par exemple, pour diviser un tableau en deux sous-tableaux, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([1, 2, 3, 4, 5, 6])
split_arr = np.split(arr, 2)

print(split_arr)

Résultat :

[array([1, 2, 3]), array([4, 5, 6])]

Tri des tableaux

Le tri des tableaux est une opération courante lors de la manipulation de tableaux avec NumPy. NumPy offre plusieurs méthodes pour trier les éléments d'un tableau selon nos besoins.

La méthode la plus courante pour trier les éléments d'un tableau est la fonction sort(). Cette fonction permet de trier les éléments d'un tableau dans l'ordre croissant. Par exemple, pour trier les éléments d'un tableau, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([3, 2, 1, 6, 5, 4])
arr.sort()

print(arr)

Résultat :

[1 2 3 4 5 6]

Il est également possible de trier les éléments d'un tableau dans l'ordre décroissant en utilisant la fonction sort() avec le paramètre optionnel kind='quicksort'. Par exemple, pour trier les éléments d'un tableau dans l'ordre décroissant, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([3, 2, 1, 6, 5, 4])
arr.sort(kind='quicksort')[::-1]

print(arr)

Résultat :

[6 5 4 3 2 1]

Répétition et agrégation de tableaux

La répétition et l'agrégation de tableaux sont des opérations courantes lors de la manipulation de tableaux avec NumPy. NumPy offre plusieurs méthodes pour répéter et agréger des tableaux selon nos besoins.

La méthode la plus courante pour répéter les éléments d'un tableau est la fonction repeat(). Cette fonction permet de répéter les éléments d'un tableau selon un nombre spécifié. Par exemple, pour répéter les éléments d'un tableau trois fois, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([1, 2, 3])
repeated_arr = np.repeat(arr, 3)

print(repeated_arr)

Résultat :

[1 1 1 2 2 2 3 3 3]

Il est également possible d'agréger les éléments d'un tableau en utilisant des fonctions d'agrégation telles que sum(), mean(), min(), max(), etc. Par exemple, pour calculer la somme des éléments d'un tableau, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([1, 2, 3, 4, 5, 6])
sum_arr = np.sum(arr)

print(sum_arr)

Résultat :

21

Il est également possible d'agréger les éléments d'un tableau le long d'un axe spécifié en utilisant des fonctions d'agrégation telles que sum(), mean(), min(), max(), etc. Par exemple, pour calculer la somme des éléments d'une matrice le long de l'axe des colonnes, on peut utiliser la syntaxe suivante :

import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])
sum_arr = np.sum(arr, axis=0)

print(sum_arr)

Résultat :

[5 7 9]
