Création de tableaux NumPy

NumPy est une bibliothèque Python qui permet de manipuler des tableaux multidimensionnels et d'effectuer des opérations mathématiques sur ces tableaux de manière efficace. Pour commencer à utiliser NumPy, nous devons d'abord créer des tableaux. Il existe plusieurs façons de créer des tableaux NumPy :

# Importer la bibliothèque NumPy
import numpy as np

# Créer un tableau à partir d'une liste
arr1 = np.array([1, 2, 3, 4, 5])

# Créer un tableau de zéros
arr2 = np.zeros(5)

# Créer un tableau de uns
arr3 = np.ones((3, 3))

# Créer un tableau avec des valeurs aléatoires
arr4 = np.random.rand(2, 2)

# Créer un tableau avec une séquence de valeurs
arr5 = np.arange(0, 10, 2)

# Créer un tableau avec des valeurs espacées uniformément
arr6 = np.linspace(0, 1, 5)

Dans l'exemple ci-dessus, nous avons utilisé la fonction np.array() pour créer un tableau à partir d'une liste, la fonction np.zeros() pour créer un tableau de zéros, la fonction np.ones() pour créer un tableau de uns, la fonction np.random.rand() pour créer un tableau avec des valeurs aléatoires, la fonction np.arange() pour créer un tableau avec une séquence de valeurs et la fonction np.linspace() pour créer un tableau avec des valeurs espacées uniformément.

Accès et manipulation des éléments d'un tableau

Une fois que nous avons créé un tableau NumPy, nous pouvons accéder et manipuler ses éléments. Voici quelques opérations courantes :

# Accéder à un élément spécifique
print(arr1[0])  # Affiche le premier élément du tableau

# Accéder à une tranche de tableau
print(arr1[1:4])  # Affiche les éléments du deuxième au quatrième

# Modifier un élément spécifique
arr1[0] = 10  # Modifie le premier élément du tableau

# Modifier une tranche de tableau
arr1[1:4] = [20, 30, 40]  # Modifie les éléments du deuxième au quatrième

# Copier un tableau
arr2 = arr1.copy()  # Copie le tableau arr1 dans le tableau arr2

Dans l'exemple ci-dessus, nous avons utilisé des index pour accéder à des éléments spécifiques du tableau, des tranches pour accéder à plusieurs éléments à la fois, et nous avons modifié des éléments spécifiques ou des tranches de tableau en leur attribuant de nouvelles valeurs. Nous avons également utilisé la méthode copy() pour créer une copie indépendante d'un tableau existant.

Opérations mathématiques avec NumPy

NumPy offre une grande variété d'opérations mathématiques que nous pouvons effectuer sur les tableaux. Voici quelques exemples :

# Addition de tableaux
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
arr3 = arr1 + arr2

# Soustraction de tableaux
arr4 = arr2 - arr1

# Multiplication de tableaux
arr5 = arr1 * arr2

# Division de tableaux
arr6 = arr2 / arr1

# Calcul de la moyenne d'un tableau
mean = np.mean(arr1)

# Calcul de la somme des éléments d'un tableau
sum = np.sum(arr2)

Dans l'exemple ci-dessus, nous avons utilisé les opérateurs mathématiques standard pour effectuer des opérations d'addition, de soustraction, de multiplication et de division sur les tableaux. Nous avons également utilisé les fonctions mean() et sum() pour calculer respectivement la moyenne et la somme des éléments d'un tableau.

Fonctions universelles de NumPy

NumPy propose également un ensemble de fonctions universelles (ufuncs) qui permettent d'effectuer des opérations mathématiques sur les tableaux de manière vectorisée. Voici quelques exemples :

# Calcul de la racine carrée des éléments d'un tableau
arr1 = np.array([1, 4, 9])
sqrt_arr = np.sqrt(arr1)

# Calcul de l'exponentielle des éléments d'un tableau
exp_arr = np.exp(arr1)

# Calcul du logarithme des éléments d'un tableau
log_arr = np.log(arr1)

# Calcul du sinus des éléments d'un tableau
sin_arr = np.sin(arr1)

Dans l'exemple ci-dessus, nous avons utilisé les fonctions universelles sqrt(), exp(), log() et sin() pour effectuer respectivement le calcul de la racine carrée, de l'exponentielle, du logarithme et du sinus des éléments d'un tableau.
