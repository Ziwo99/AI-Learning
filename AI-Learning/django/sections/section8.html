Utilisation de bases de données externes

Django offre la possibilité d'intégrer des bases de données externes à votre application. Cela peut être utile si vous avez déjà une base de données existante que vous souhaitez utiliser avec Django. Pour cela, vous devez configurer les paramètres de connexion à la base de données externe dans le fichier de configuration de votre projet Django.

Voici un exemple de configuration pour une base de données externe MySQL :

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'nom_de_la_base_de_donnees',
        'USER': 'utilisateur',
        'PASSWORD': 'mot_de_passe',
        'HOST': 'adresse_ip',
        'PORT': 'port',
    }
}

Une fois la configuration effectuée, vous pouvez utiliser les modèles Django pour interagir avec la base de données externe de la même manière que vous le feriez avec une base de données locale.

Intégration avec des services web

Django facilite l'intégration avec des services web externes. Vous pouvez utiliser des bibliothèques Python telles que Requests pour effectuer des requêtes HTTP vers des API externes et récupérer les données.

Voici un exemple de code qui utilise la bibliothèque Requests pour récupérer des données à partir d'une API externe :

import requests

def get_data_from_api():
    response = requests.get('https://api.example.com/data')
    data = response.json()
    return data

Vous pouvez ensuite utiliser les données récupérées dans votre application Django selon vos besoins.

Utilisation de Django REST Framework

Django REST Framework est une extension de Django qui facilite la création d'API RESTful. Il fournit des outils pour la sérialisation des données, la gestion des autorisations, la validation des données et bien plus encore.

Voici un exemple de configuration d'une API REST avec Django REST Framework :

# serializers.py
from rest_framework import serializers
from .models import Article

class ArticleSerializer(serializers.ModelSerializer):
    class Meta:
        model = Article
        fields = ['title', 'content']

# views.py
from rest_framework import viewsets
from .models import Article
from .serializers import ArticleSerializer

class ArticleViewSet(viewsets.ModelViewSet):
    queryset = Article.objects.all()
    serializer_class = ArticleSerializer

# urls.py
from django.urls import include, path
from rest_framework import routers
from .views import ArticleViewSet

router = routers.DefaultRouter()
router.register(r'articles', ArticleViewSet)

urlpatterns = [
    path('', include(router.urls)),
]

Avec Django REST Framework, vous pouvez facilement créer des API RESTful pour votre application Django, ce qui facilite l'intégration avec d'autres services et applications.
