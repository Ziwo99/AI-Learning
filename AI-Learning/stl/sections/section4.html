Les algorithmes de tri

Les algorithmes de tri sont utilisés pour organiser les éléments d'un conteneur dans un ordre spécifique. Le STL propose plusieurs algorithmes de tri, tels que :

  • sort() : trie les éléments d'un conteneur dans l'ordre croissant par défaut.
  • stable_sort() : trie les éléments d'un conteneur de manière stable, c'est-à-dire en préservant l'ordre relatif des éléments égaux.
  • partial_sort() : trie les premiers éléments d'un conteneur dans l'ordre croissant, laissant les autres éléments dans un état non trié.

Voici un exemple d'utilisation de l'algorithme sort() :

std::vector numbers = {5, 2, 8, 1, 9};
std::sort(numbers.begin(), numbers.end());

// Résultat : numbers = {1, 2, 5, 8, 9}

Les algorithmes de recherche

Les algorithmes de recherche permettent de trouver des éléments dans un conteneur en fonction de critères spécifiques. Le STL propose plusieurs algorithmes de recherche, tels que :

  • find() : recherche la première occurrence d'un élément dans un conteneur.
  • binary_search() : vérifie si un élément est présent dans un conteneur trié en utilisant une recherche binaire.
  • count() : compte le nombre d'occurrences d'un élément dans un conteneur.

Voici un exemple d'utilisation de l'algorithme find() :

std::vector numbers = {5, 2, 8, 1, 9};
auto it = std::find(numbers.begin(), numbers.end(), 8);

if (it != numbers.end()) {
    std::cout << "Element found at index: " << std::distance(numbers.begin(), it) << std::endl;
} else {
    std::cout << "Element not found" << std::endl;
}

Les algorithmes de manipulation de conteneurs

Les algorithmes de manipulation de conteneurs permettent d'effectuer des opérations telles que l'insertion, la suppression et la modification d'éléments. Le STL propose plusieurs algorithmes de manipulation de conteneurs, tels que :

  • copy() : copie les éléments d'un conteneur source vers un conteneur destination.
  • remove() : supprime tous les éléments égaux à une valeur donnée d'un conteneur.
  • replace() : remplace tous les éléments égaux à une valeur donnée par une autre valeur dans un conteneur.

Voici un exemple d'utilisation de l'algorithme copy() :

std::vector source = {1, 2, 3, 4, 5};
std::vector destination(5);

std::copy(source.begin(), source.end(), destination.begin());

// Résultat : destination = {1, 2, 3, 4, 5}

Les algorithmes de transformation

Les algorithmes de transformation permettent d'appliquer une opération à chaque élément d'un conteneur et de stocker les résultats dans un autre conteneur. Le STL propose plusieurs algorithmes de transformation, tels que :

  • transform() : applique une fonction à chaque élément d'un conteneur et stocke les résultats dans un autre conteneur.
  • accumulate() : calcule la somme des éléments d'un conteneur en utilisant une opération spécifiée.
  • replace_if() : remplace les éléments d'un conteneur qui satisfont un prédicat par une autre valeur.

Voici un exemple d'utilisation de l'algorithme transform() :

std::vector numbers = {1, 2, 3, 4, 5};
std::vector squared_numbers(numbers.size());

std::transform(numbers.begin(), numbers.end(), squared_numbers.begin(), [](int n) {
    return n * n;
});

// Résultat : squared_numbers = {1, 4, 9, 16, 25}

Les algorithmes numériques

Les algorithmes numériques permettent d'effectuer des opérations mathématiques sur les éléments d'un conteneur. Le STL propose plusieurs algorithmes numériques, tels que :

  • accumulate() : calcule la somme des éléments d'un conteneur en utilisant une opération spécifiée.
  • inner_product() : calcule le produit scalaire de deux conteneurs.
  • partial_sum() : calcule les sommes partielles des éléments d'un conteneur.

Voici un exemple d'utilisation de l'algorithme accumulate() :

std::vector numbers = {1, 2, 3, 4, 5};
int sum = std::accumulate(numbers.begin(), numbers.end(), 0);

// Résultat : sum = 15
