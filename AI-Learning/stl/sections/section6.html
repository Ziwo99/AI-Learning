Les adaptateurs de conteneurs

Les adaptateurs de conteneurs sont des classes qui fournissent une interface différente pour accéder aux éléments d'un conteneur STL existant. Ils permettent de modifier la façon dont les éléments sont organisés ou accessibles.

Un exemple d'adaptateur de conteneurs est la classe stack. Elle permet d'accéder aux éléments d'un conteneur sous forme de pile, où le dernier élément ajouté est le premier à être retiré (principe LIFO - Last In, First Out).

Voici un exemple d'utilisation de la classe stack :

#include <stack>
#include <iostream>

int main() {
    std::stack<int> myStack;

    myStack.push(1);
    myStack.push(2);
    myStack.push(3);

    while (!myStack.empty()) {
        std::cout << myStack.top() << " ";
        myStack.pop();
    }

    return 0;
}

Dans cet exemple, nous créons une pile d'entiers à l'aide de la classe stack. Nous ajoutons ensuite les entiers 1, 2 et 3 à la pile à l'aide de la fonction push. Ensuite, nous utilisons une boucle pour afficher les éléments de la pile à l'aide de la fonction top, puis nous les retirons de la pile à l'aide de la fonction pop.

Les adaptateurs d'itérateurs

Les adaptateurs d'itérateurs sont des classes qui fournissent une interface différente pour parcourir les éléments d'un conteneur STL existant. Ils permettent de modifier la façon dont les éléments sont parcourus ou manipulés.

Un exemple d'adaptateur d'itérateurs est la classe reverse_iterator. Elle permet de parcourir un conteneur STL dans l'ordre inverse.

Voici un exemple d'utilisation de la classe reverse_iterator :

#include <vector>
#include <iostream>

int main() {
    std::vector<int> myVector = {1, 2, 3, 4, 5};

    for (auto it = myVector.rbegin(); it != myVector.rend(); ++it) {
        std::cout << *it << " ";
    }

    return 0;
}

Dans cet exemple, nous créons un vecteur d'entiers à l'aide de la classe vector. Nous utilisons ensuite la classe reverse_iterator pour parcourir le vecteur dans l'ordre inverse. La fonction rbegin retourne un itérateur pointant vers le dernier élément du vecteur, et la fonction rend retourne un itérateur pointant avant le premier élément du vecteur. Nous utilisons ensuite une boucle pour afficher les éléments du vecteur à l'envers.

Il existe de nombreux autres adaptateurs de conteneurs et d'itérateurs dans le STL, chacun offrant des fonctionnalités spécifiques pour répondre à différents besoins de programmation.
