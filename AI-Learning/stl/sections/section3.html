Section 3 : Les itérateurs du STL

Les itérateurs d'entrée (input iterators)

Les itérateurs d'entrée sont utilisés pour parcourir les éléments d'un conteneur STL dans une seule direction, de gauche à droite. Ils permettent uniquement la lecture des éléments et ne peuvent pas être utilisés pour modifier les valeurs des éléments.

Voici un exemple d'utilisation d'un itérateur d'entrée pour parcourir un vecteur :

std::vector numbers = {1, 2, 3, 4, 5};

// Parcours du vecteur avec un itérateur d'entrée
for (std::vector::iterator it = numbers.begin(); it != numbers.end(); ++it) {
    std::cout << *it << " ";
}

Dans cet exemple, l'itérateur "it" est utilisé pour parcourir le vecteur "numbers" et afficher chaque élément.

Les itérateurs de sortie (output iterators)

Les itérateurs de sortie sont utilisés pour écrire des valeurs dans un conteneur STL. Ils permettent uniquement l'écriture des éléments et ne peuvent pas être utilisés pour lire les valeurs des éléments.

Voici un exemple d'utilisation d'un itérateur de sortie pour écrire des valeurs dans un vecteur :

std::vector numbers(5);

// Utilisation d'un itérateur de sortie pour écrire des valeurs dans le vecteur
for (std::vector::iterator it = numbers.begin(); it != numbers.end(); ++it) {
    *it = 10;
}

Dans cet exemple, l'itérateur "it" est utilisé pour écrire la valeur 10 dans chaque élément du vecteur "numbers".

Les itérateurs avant (forward iterators)

Les itérateurs avant sont similaires aux itérateurs d'entrée, mais ils permettent également de modifier les valeurs des éléments d'un conteneur STL.

Voici un exemple d'utilisation d'un itérateur avant pour parcourir et modifier un vecteur :

std::vector numbers = {1, 2, 3, 4, 5};

// Parcours et modification du vecteur avec un itérateur avant
for (std::vector::iterator it = numbers.begin(); it != numbers.end(); ++it) {
    *it *= 2;
}

Dans cet exemple, l'itérateur "it" est utilisé pour parcourir le vecteur "numbers" et multiplier chaque élément par 2.

Les itérateurs bidirectionnels (bidirectional iterators)

Les itérateurs bidirectionnels permettent de parcourir les éléments d'un conteneur STL dans les deux sens, de gauche à droite et de droite à gauche. Ils peuvent être utilisés pour lire et modifier les valeurs des éléments.

Voici un exemple d'utilisation d'un itérateur bidirectionnel pour parcourir et modifier une liste :

std::list numbers = {1, 2, 3, 4, 5};

// Parcours et modification de la liste avec un itérateur bidirectionnel
for (std::list::iterator it = numbers.begin(); it != numbers.end(); ++it) {
    *it += 1;
}

// Parcours inversé de la liste avec un itérateur bidirectionnel
for (std::list::reverse_iterator it = numbers.rbegin(); it != numbers.rend(); ++it) {
    std::cout << *it << " ";
}

Dans cet exemple, l'itérateur "it" est utilisé pour parcourir la liste "numbers" et ajouter 1 à chaque élément. Ensuite, un autre itérateur bidirectionnel est utilisé pour parcourir la liste dans l'ordre inverse et afficher les éléments.

Les itérateurs aléatoires (random access iterators)

Les itérateurs aléatoires permettent un accès direct aux éléments d'un conteneur STL. Ils peuvent être utilisés pour lire et modifier les valeurs des éléments, ainsi que pour effectuer des opérations arithmétiques sur les itérateurs.

Voici un exemple d'utilisation d'un itérateur aléatoire pour accéder aux éléments d'un tableau :

int numbers[] = {1, 2, 3, 4, 5};

// Accès aux éléments du tableau avec un itérateur aléatoire
for (int* it = std::begin(numbers); it != std::end(numbers); ++it) {
    std::cout << *it << " ";
}

// Utilisation d'opérations arithmétiques sur l'itérateur aléatoire
int* thirdElement = std::begin(numbers) + 2;
std::cout << *thirdElement; // Affiche 3

Dans cet exemple, l'itérateur "it" est utilisé pour accéder aux éléments du tableau "numbers" et les afficher. Ensuite, un autre itérateur aléatoire est utilisé pour accéder au troisième élément du tableau en utilisant des opérations arithmétiques sur l'itérateur.
