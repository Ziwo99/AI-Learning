Les foncteurs

Les foncteurs, également appelés objets fonctionnels, sont des objets qui peuvent être utilisés comme des fonctions. Ils sont souvent utilisés avec les algorithmes du STL pour effectuer des opérations sur les conteneurs. Les foncteurs peuvent être des objets de classe avec une surcharge de l'opérateur de parenthèses (), ou des objets de classe avec une surcharge de l'opérateur de fonction ().

Voici un exemple de foncteur qui multiplie chaque élément d'un conteneur par un facteur :

class MultiplyByFactor {
public:
    MultiplyByFactor(int factor) : m_factor(factor) {}

    int operator()(int value) const {
        return value * m_factor;
    }

private:
    int m_factor;
};

std::vector numbers = {1, 2, 3, 4, 5};
std::transform(numbers.begin(), numbers.end(), numbers.begin(), MultiplyByFactor(2));
// numbers contiendra maintenant {2, 4, 6, 8, 10}

Dans cet exemple, nous utilisons le foncteur MultiplyByFactor avec l'algorithme std::transform pour multiplier chaque élément du vecteur numbers par 2.

Les prédicats

Les prédicats sont des fonctions ou des foncteurs qui retournent une valeur booléenne. Ils sont souvent utilisés avec les algorithmes du STL pour effectuer des opérations conditionnelles sur les conteneurs. Les prédicats peuvent être utilisés pour filtrer, trier ou rechercher des éléments dans un conteneur.

Voici un exemple de prédicat qui vérifie si un nombre est pair :

class IsEven {
public:
    bool operator()(int value) const {
        return value % 2 == 0;
    }
};

std::vector numbers = {1, 2, 3, 4, 5};
int count = std::count_if(numbers.begin(), numbers.end(), IsEven());
// count contiendra maintenant 2, car il y a 2 nombres pairs dans le vecteur numbers

Dans cet exemple, nous utilisons le prédicat IsEven avec l'algorithme std::count_if pour compter le nombre d'éléments pairs dans le vecteur numbers.

Utilisation des foncteurs et des prédicats avec les algorithmes du STL

Les foncteurs et les prédicats peuvent être utilisés avec de nombreux algorithmes du STL pour effectuer des opérations spécifiques sur les conteneurs. Voici quelques exemples :

  • std::transform : utilise un foncteur pour appliquer une opération à chaque élément d'un conteneur.
  • std::find_if : utilise un prédicat pour rechercher le premier élément qui satisfait une condition dans un conteneur.
  • std::sort : utilise un prédicat pour trier les éléments d'un conteneur selon un ordre spécifique.
  • std::remove_if : utilise un prédicat pour supprimer tous les éléments qui satisfont une condition dans un conteneur.

Il est important de noter que les foncteurs et les prédicats doivent être des objets immuables, c'est-à-dire qu'ils ne doivent pas modifier l'état des objets sur lesquels ils opèrent. Cela garantit que les opérations effectuées avec les foncteurs et les prédicats sont prévisibles et sans effets secondaires.

En utilisant les foncteurs et les prédicats avec les algorithmes du STL, vous pouvez effectuer des opérations complexes sur les conteneurs de manière concise et efficace.
