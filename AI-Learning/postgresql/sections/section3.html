Création de bases de données

La création d'une base de données est une étape essentielle dans PostgreSQL. Une base de données est un conteneur logique qui regroupe des objets tels que des tables, des vues, des fonctions, etc. Pour créer une base de données, vous pouvez utiliser la commande SQL CREATE DATABASE.

CREATE DATABASE nom_de_la_base_de_données;

Vous pouvez également spécifier des options supplémentaires lors de la création de la base de données, telles que l'encodage, le modèle de données, etc.

CREATE DATABASE nom_de_la_base_de_données
    WITH ENCODING 'UTF8'
    TEMPLATE template0;

Tables, colonnes et types de données

Les tables sont les objets principaux dans une base de données PostgreSQL. Elles sont utilisées pour stocker les données de manière structurée. Pour créer une table, vous pouvez utiliser la commande SQL CREATE TABLE.

CREATE TABLE nom_de_la_table (
    colonne1 type_de_données,
    colonne2 type_de_données,
    ...
);

Vous devez spécifier le nom de la table et les colonnes avec leurs types de données respectifs. PostgreSQL prend en charge différents types de données tels que les entiers, les chaînes de caractères, les dates, etc.

Contraintes et index

Les contraintes sont utilisées pour imposer des règles sur les données stockées dans une table. Par exemple, vous pouvez définir une contrainte d'intégrité référentielle pour garantir que les valeurs d'une colonne correspondent à celles d'une autre colonne dans une autre table.

CREATE TABLE table1 (
    ...
    colonne1 type_de_données REFERENCES table2(colonne2),
    ...
);

Les index sont utilisés pour améliorer les performances des requêtes en accélérant la recherche de données. Vous pouvez créer un index sur une ou plusieurs colonnes d'une table.

CREATE INDEX nom_de_l_index ON nom_de_la_table (colonne1, colonne2);

Vues, séquences et fonctions

Les vues sont des objets virtuels qui permettent de visualiser les données d'une ou plusieurs tables. Elles sont créées à partir d'une requête SQL et peuvent être utilisées comme des tables normales.

CREATE VIEW nom_de_la_vue AS
    SELECT colonne1, colonne2
    FROM nom_de_la_table
    WHERE condition;

Les séquences sont utilisées pour générer des valeurs numériques uniques. Elles sont souvent utilisées pour les clés primaires ou les identifiants uniques.

CREATE SEQUENCE nom_de_la_sequence
    START WITH 1
    INCREMENT BY 1
    MINVALUE 1
    MAXVALUE 1000;

Les fonctions sont des blocs de code réutilisables qui effectuent des opérations spécifiques. Elles peuvent être créées en utilisant le langage de programmation PL/pgSQL.

CREATE FUNCTION nom_de_la_fonction(paramètre1 type, paramètre2 type)
    RETURNS type_de_retour AS $$
    BEGIN
        -- Code de la fonction
    END;
    $$ LANGUAGE plpgsql;

Ceci conclut la section sur la structure de base de données PostgreSQL. Vous avez appris comment créer des bases de données, des tables, des contraintes, des index, des vues, des séquences et des fonctions. Dans la prochaine section, nous explorerons les requêtes SQL avancées.
