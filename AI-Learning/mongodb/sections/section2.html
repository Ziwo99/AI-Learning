Création et suppression de bases de données

Pour créer une base de données MongoDB, vous pouvez utiliser la commande use suivie du nom de la base de données que vous souhaitez créer. Par exemple :

use ma_base_de_donnees

Cette commande créera une nouvelle base de données appelée "ma_base_de_donnees" si elle n'existe pas déjà.

Pour supprimer une base de données, vous pouvez utiliser la commande db.dropDatabase(). Par exemple :

use ma_base_de_donnees
db.dropDatabase()

Cette commande supprimera la base de données "ma_base_de_donnees" et tous ses documents.

Création et suppression de collections

Pour créer une collection dans une base de données MongoDB, vous pouvez utiliser la méthode db.createCollection(). Par exemple :

use ma_base_de_donnees
db.createCollection("ma_collection")

Cette commande créera une nouvelle collection appelée "ma_collection" dans la base de données "ma_base_de_donnees".

Pour supprimer une collection, vous pouvez utiliser la méthode db.collection.drop(). Par exemple :

use ma_base_de_donnees
db.ma_collection.drop()

Cette commande supprimera la collection "ma_collection" de la base de données "ma_base_de_donnees".

Insertion, mise à jour et suppression de documents

Pour insérer un document dans une collection, vous pouvez utiliser la méthode db.collection.insert(). Par exemple :

use ma_base_de_donnees
db.ma_collection.insert({ "nom": "John", "age": 30 })

Cette commande insérera un nouveau document avec les champs "nom" et "age" dans la collection "ma_collection" de la base de données "ma_base_de_donnees".

Pour mettre à jour un document existant, vous pouvez utiliser la méthode db.collection.update(). Par exemple :

use ma_base_de_donnees
db.ma_collection.update({ "nom": "John" }, { $set: { "age": 35 } })

Cette commande mettra à jour le champ "age" du document avec le champ "nom" égal à "John" dans la collection "ma_collection" de la base de données "ma_base_de_donnees".

Pour supprimer un document, vous pouvez utiliser la méthode db.collection.remove(). Par exemple :

use ma_base_de_donnees
db.ma_collection.remove({ "nom": "John" })

Cette commande supprimera tous les documents avec le champ "nom" égal à "John" de la collection "ma_collection" de la base de données "ma_base_de_donnees".

Requêtes de base : recherche, tri et filtrage

Pour effectuer une recherche dans une collection, vous pouvez utiliser la méthode db.collection.find(). Par exemple :

use ma_base_de_donnees
db.ma_collection.find({ "age": 30 })

Cette commande renverra tous les documents de la collection "ma_collection" de la base de données "ma_base_de_donnees" avec le champ "age" égal à 30.

Pour trier les résultats d'une requête, vous pouvez utiliser la méthode db.collection.find().sort(). Par exemple :

use ma_base_de_donnees
db.ma_collection.find().sort({ "age": 1 })

Cette commande triera les documents de la collection "ma_collection" de la base de données "ma_base_de_donnees" par ordre croissant en fonction du champ "age".

Pour filtrer les résultats d'une requête, vous pouvez utiliser la méthode db.collection.find().filter(). Par exemple :

use ma_base_de_donnees
db.ma_collection.find().filter({ "age": { $gt: 30 } })

Cette commande renverra tous les documents de la collection "ma_collection" de la base de données "ma_base_de_donnees" avec le champ "age" supérieur à 30.

Indexation et optimisation des requêtes

Pour créer un index sur une collection, vous pouvez utiliser la méthode db.collection.createIndex(). Par exemple :

use ma_base_de_donnees
db.ma_collection.createIndex({ "nom": 1 })

Cette commande créera un index sur le champ "nom" de la collection "ma_collection" de la base de données "ma_base_de_donnees".

Pour optimiser les requêtes, vous pouvez utiliser la méthode db.collection.explain(). Par exemple :

use ma_base_de_donnees
db.ma_collection.find({ "age": 30 }).explain()

Cette commande affichera des informations détaillées sur la façon dont la requête est exécutée, ce qui peut vous aider à identifier les problèmes de performance et à les résoudre.
