Composants React

Les composants sont les éléments de base de React. Ils permettent de diviser l'interface utilisateur en morceaux réutilisables et indépendants. Un composant peut être une partie d'une page ou une page entière.

Voici un exemple de création d'un composant React :

import React from 'react';

class MonComposant extends React.Component {
  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <p>Ceci est mon composant React.</p>
      </div>
    );
  }
}

export default MonComposant;

JSX (JavaScript XML)

JSX est une extension de syntaxe qui permet d'écrire du code HTML dans du code JavaScript. Il facilite la création d'éléments React en utilisant une syntaxe familière.

Voici un exemple d'utilisation de JSX :

import React from 'react';

class MonComposant extends React.Component {
  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <p>Ceci est mon composant React.</p>
      </div>
    );
  }
}

export default MonComposant;

Rendu des composants

Le rendu des composants est la manière dont les composants React sont affichés à l'écran. Le rendu est effectué en utilisant la méthode render() d'un composant.

Voici un exemple de rendu d'un composant :

import React from 'react';
import ReactDOM from 'react-dom';

class MonComposant extends React.Component {
  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <p>Ceci est mon composant React.</p>
      </div>
    );
  }
}

ReactDOM.render(<MonComposant />, document.getElementById('root'));

Gestion des événements

La gestion des événements permet de réagir aux actions de l'utilisateur, comme un clic sur un bouton ou une soumission de formulaire. Les événements sont gérés en utilisant des attributs spécifiques dans les éléments JSX.

Voici un exemple de gestion d'événements :

import React from 'react';

class MonComposant extends React.Component {
  handleClick() {
    console.log('Le bouton a été cliqué.');
  }

  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <button onClick={this.handleClick}>Cliquez ici</button>
      </div>
    );
  }
}

export default MonComposant;

Utilisation des états (state)

Les états sont des objets JavaScript qui contiennent des données qui peuvent être modifiées dans un composant React. Les états permettent de rendre les composants dynamiques et réactifs aux changements.

Voici un exemple d'utilisation des états :

import React from 'react';

class MonComposant extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      count: 0
    };
  }

  handleClick() {
    this.setState({ count: this.state.count + 1 });
  }

  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <p>Count: {this.state.count}</p>
        <button onClick={this.handleClick.bind(this)}>Incrémenter</button>
      </div>
    );
  }
}

export default MonComposant;

Utilisation des propriétés (props)

Les propriétés sont des valeurs qui peuvent être passées à un composant React depuis son parent. Les propriétés permettent de personnaliser les composants et de leur transmettre des données.

Voici un exemple d'utilisation des propriétés :

import React from 'react';

class MonComposant extends React.Component {
  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <p>Bonjour, {this.props.name} !</p>
      </div>
    );
  }
}

export default MonComposant;

Cycle de vie des composants

Les composants React ont un cycle de vie qui comprend différentes étapes, comme la création, la mise à jour et la suppression. Les méthodes du cycle de vie permettent d'exécuter du code à des moments spécifiques du cycle de vie d'un composant.

Voici un exemple des méthodes du cycle de vie :

import React from 'react';

class MonComposant extends React.Component {
  constructor(props) {
    super(props);
    console.log('Le composant est en cours de création.');
  }

  componentDidMount() {
    console.log('Le composant a été monté.');
  }

  componentDidUpdate() {
    console.log('Le composant a été mis à jour.');
  }

  componentWillUnmount() {
    console.log('Le composant va être supprimé.');
  }

  render() {
    return (
      <div>
        <h2>Mon Composant</h2>
        <p>Ceci est mon composant React.</p>
      </div>
    );
  }
}

export default MonComposant;
