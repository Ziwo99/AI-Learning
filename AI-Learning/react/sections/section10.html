Création d'une application de liste de tâches

Dans cette sous-section, nous allons créer une application de liste de tâches en utilisant React. Cette application permettra aux utilisateurs de créer, modifier et supprimer des tâches.

Voici les étapes pour créer cette application :

  1. Créez un nouveau projet React en utilisant la commande suivante dans votre terminal :
    2. npx create-react-app liste-taches
  2. Naviguez vers le répertoire du projet en utilisant la commande suivante :
    3. cd liste-taches
  3. Ouvrez le projet dans votre éditeur de code préféré.
  4. Créez un nouveau composant appelé ListeTaches dans le dossier src. Ce composant sera responsable de l'affichage de la liste des tâches.
  5. Dans le composant ListeTaches, créez un état initial vide pour stocker les tâches :
    6. const [taches, setTaches] = useState([]);
  6. Créez une fonction pour ajouter une nouvelle tâche à la liste :
    7. const ajouterTache = (nouvelleTache) => {
  setTaches([...taches, nouvelleTache]);
}
  7. Créez une fonction pour supprimer une tâche de la liste :
    8. const supprimerTache = (index) => {
  const nouvellesTaches = [...taches];
  nouvellesTaches.splice(index, 1);
  setTaches(nouvellesTaches);
}
  8. Créez une fonction pour modifier une tâche existante :
    9. const modifierTache = (index, nouvelleTache) => {
  const nouvellesTaches = [...taches];
  nouvellesTaches[index] = nouvelleTache;
  setTaches(nouvellesTaches);
}
  9. Affichez la liste des tâches dans le rendu du composant :
    10. {taches.map((tache, index) => (
  <div key={index}>
    <span>{tache}</span>
    <button onClick={() => supprimerTache(index)}>Supprimer</button>
    <button onClick={() => modifierTache(index, 'Nouvelle tâche')}>Modifier</button>
  </div>
))}
  10. Créez un autre composant appelé AjouterTache dans le dossier src. Ce composant sera responsable de l'ajout de nouvelles tâches.
  11. Dans le composant AjouterTache, créez un état pour stocker la valeur de la nouvelle tâche :
    12. const [nouvelleTache, setNouvelleTache] = useState('');
  12. Créez une fonction pour mettre à jour la valeur de la nouvelle tâche :
    13. const handleChange = (event) => {
  setNouvelleTache(event.target.value);
}
  13. Créez une fonction pour ajouter la nouvelle tâche à la liste :
    14. const handleSubmit = (event) => {
  event.preventDefault();
  ajouterTache(nouvelleTache);
  setNouvelleTache('');
}
  14. Affichez un formulaire dans le rendu du composant pour permettre aux utilisateurs d'ajouter une nouvelle tâche :
    15. <form onSubmit={handleSubmit}>
  <input type="text" value={nouvelleTache} onChange={handleChange} />
  <button type="submit">Ajouter</button>
</form>
  15. Importez les composants ListeTaches et AjouterTache dans le composant racine App :
    16. import ListeTaches from './ListeTaches';
import AjouterTache from './AjouterTache';
  16. Affichez les composants importés dans le rendu du composant App :
    17. <ListeTaches />
<AjouterTache />
  17. Testez votre application en exécutant la commande suivante dans votre terminal :
    18. npm start

Vous avez maintenant créé une application de liste de tâches fonctionnelle en utilisant React ! Vous pouvez continuer à améliorer cette application en ajoutant des fonctionnalités supplémentaires, telles que la possibilité de marquer une tâche comme terminée ou de trier les tâches par ordre de priorité.

Création d'une application de blog

Dans cette sous-section, nous allons créer une application de blog en utilisant React. Cette application permettra aux utilisateurs de créer, modifier et supprimer des articles de blog.

Voici les étapes pour créer cette application :

  1. Créez un nouveau projet React en utilisant la commande suivante dans votre terminal :
    2. npx create-react-app blog
  2. Naviguez vers le répertoire du projet en utilisant la commande suivante :
    3. cd blog
  3. Ouvrez le projet dans votre éditeur de code préféré.
  4. Créez un nouveau composant appelé ListeArticles dans le dossier src. Ce composant sera responsable de l'affichage de la liste des articles de blog.
  5. Dans le composant ListeArticles, créez un état initial vide pour stocker les articles :
    6. const [articles, setArticles] = useState([]);
  6. Créez une fonction pour ajouter un nouvel article à la liste :
    7. const ajouterArticle = (nouvelArticle) => {
  setArticles([...articles, nouvelArticle]);
}
  7. Créez une fonction pour supprimer un article de la liste :
    8. const supprimerArticle = (index) => {
  const nouveauxArticles = [...articles];
  nouveauxArticles.splice(index, 1);
  setArticles(nouveauxArticles);
}
  8. Créez une fonction pour modifier un article existant :
    9. const modifierArticle = (index, nouvelArticle) => {
  const nouveauxArticles = [...articles];
  nouveauxArticles[index] = nouvelArticle;
  setArticles(nouveauxArticles);
}
  9. Affichez la liste des articles dans le rendu du composant :
    10. {articles.map((article, index) => (
  <div key={index}>
    <h3>{article.titre}</h3>
    <p>{article.contenu}</p>
    <button onClick={() => supprimerArticle(index)}>Supprimer</button>
    <button onClick={() => modifierArticle(index, {titre: 'Nouveau titre', contenu: 'Nouveau contenu'})}>Modifier</button>
  </div>
))}
  10. Créez un autre composant appelé AjouterArticle dans le dossier src. Ce composant sera responsable de l'ajout de nouveaux articles.
  11. Dans le composant AjouterArticle, créez un état pour stocker les valeurs du titre et du contenu du nouvel article :
    12. const [nouveauTitre, setNouveauTitre] = useState('');
const [nouveauContenu, setNouveauContenu] = useState('');
  12. Créez des fonctions pour mettre à jour les valeurs du titre et du contenu du nouvel article :
    13. const handleChangeTitre = (event) => {
  setNouveauTitre(event.target.value);
}

const handleChangeContenu = (event) => {
  setNouveauContenu(event.target.value);
}
  13. Créez une fonction pour ajouter le nouvel article à la liste :
    14. const handleSubmit = (event) => {
  event.preventDefault();
  ajouterArticle({titre: nouveauTitre, contenu: nouveauContenu});
  setNouveauTitre('');
  setNouveauContenu('');
}
  14. Affichez un formulaire dans le rendu du composant pour permettre aux utilisateurs d'ajouter un nouvel article :
    15. <form onSubmit={handleSubmit}>
  <input type="text" value={nouveauTitre} onChange={handleChangeTitre} />
  <textarea value={nouveauContenu} onChange={handleChangeContenu} />
  <button type="submit">Ajouter</button>
</form>
  15. Importez les composants ListeArticles et AjouterArticle dans le composant racine App :
    16. import ListeArticles from './ListeArticles';
import AjouterArticle from './AjouterArticle';
  16. Affichez les composants importés dans le rendu du composant App :
    17. <ListeArticles />
<AjouterArticle />
  17. Testez votre application en exécutant la commande suivante dans votre terminal :
    18. npm start

Vous avez maintenant créé une application de blog fonctionnelle en utilisant React ! Vous pouvez continuer à améliorer cette application en ajoutant des fonctionnalités supplémentaires, telles que la possibilité de commenter les articles ou de les trier par date de publication.

Création d'une application de commerce électronique

Dans cette sous-section, nous allons créer une application de commerce électronique en utilisant React. Cette application permettra aux utilisateurs de parcourir et d'acheter des produits.

Voici les étapes pour créer cette application :

  1. Créez un nouveau projet React en utilisant la commande suivante dans votre terminal :
    2. npx create-react-app ecommerce
  2. Naviguez vers le répertoire du projet en utilisant la commande suivante :
    3. cd ecommerce
  3. Ouvrez le projet dans votre éditeur de code préféré.
  4. Créez un nouveau composant appelé ListeProduits dans le dossier src. Ce composant sera responsable de l'affichage de la liste des produits.
  5. Dans le composant ListeProduits, créez un état initial vide pour stocker les produits :
    6. const [produits, setProduits] = useState([]);
  6. Créez une fonction pour récupérer les produits à partir d'une API :
    7. const fetchProduits = async () => {
  const response = await fetch('https://api.example.com/produits');
  const data = await response.json();
  setProduits(data);
}
  7. Appelez la fonction fetchProduits dans le cycle de vie du composant ListeProduits :
    8. useEffect(() => {
  fetchProduits();
}, []);
  8. Affichez la liste des produits dans le rendu du composant :
    9. {produits.map((produit) => (
  <div key={produit.id}>
    <h3>{produit.nom}</h3>
    <p>{produit.prix}</p>
    <button>Acheter</button>
  </div>
))}
  9. Créez un autre composant appelé Panier dans le dossier src. Ce composant sera responsable de l'affichage du panier d'achat.
  10. Dans le composant Panier, créez un état initial vide pour stocker les produits du panier :
    11. const [panier, setPanier] = useState([]);
  11. Créez une fonction pour ajouter un produit au panier :
    12. const ajouterProduit = (nouveauProduit) => {
  setPanier([...panier, nouveauProduit]);
}
  12. Affichez la liste des produits du panier dans le rendu du composant :
    13. {panier.map((produit) => (
  <div key={produit.id}>
    <h3>{produit.nom}</h3>
    <p>{produit.prix}</p>
  </div>
))}
  13. Importez les composants ListeProduits et Panier dans le composant racine App :
    14. import ListeProduits from './ListeProduits';
import Panier from './Panier';
  14. Affichez les composants importés dans le rendu du composant App :
    15. <ListeProduits />
<Panier />
  15. Testez votre application en exécutant la commande suivante dans votre terminal :
    16. npm start

Vous avez maintenant créé une application de commerce électronique fonctionnelle en utilisant React ! Vous pouvez continuer à améliorer cette application en ajoutant des fonctionnalités supplémentaires, telles que la possibilité de filtrer les produits par catégorie ou de mettre à jour la quantité des produits dans le panier.
