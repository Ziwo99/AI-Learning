Statistiques descriptives

Les statistiques descriptives sont utilisées pour résumer et analyser les caractéristiques principales d'un ensemble de données. Avec Pandas, nous pouvons facilement calculer des statistiques descriptives telles que la moyenne, la médiane, l'écart-type, etc. pour chaque colonne d'un DataFrame.

# Importer la bibliothèque Pandas
import pandas as pd

# Charger les données dans un DataFrame
data = pd.read_csv('data.csv')

# Calculer la moyenne de chaque colonne
mean_values = data.mean()

# Calculer l'écart-type de chaque colonne
std_values = data.std()

# Calculer la médiane de chaque colonne
median_values = data.median()

# Afficher les résultats
print("Moyenne:")
print(mean_values)
print("\nÉcart-type:")
print(std_values)
print("\nMédiane:")
print(median_values)

Visualisation des données

La visualisation des données est essentielle pour comprendre et communiquer les informations contenues dans un ensemble de données. Pandas offre des fonctionnalités intégrées pour créer des graphiques tels que des histogrammes, des diagrammes en boîte, des diagrammes à barres, etc.

# Importer la bibliothèque Pandas
import pandas as pd
import matplotlib.pyplot as plt

# Charger les données dans un DataFrame
data = pd.read_csv('data.csv')

# Créer un histogramme de la colonne 'age'
data['age'].plot(kind='hist')

# Ajouter un titre et des étiquettes d'axe
plt.title('Distribution des âges')
plt.xlabel('Âge')
plt.ylabel('Fréquence')

# Afficher le graphique
plt.show()

Analyse de corrélation

L'analyse de corrélation est utilisée pour mesurer la relation entre deux variables dans un ensemble de données. Pandas fournit des fonctions pour calculer le coefficient de corrélation de Pearson, qui mesure la corrélation linéaire entre deux variables.

# Importer la bibliothèque Pandas
import pandas as pd

# Charger les données dans un DataFrame
data = pd.read_csv('data.csv')

# Calculer la corrélation entre les colonnes 'age' et 'income'
correlation = data['age'].corr(data['income'])

# Afficher le résultat
print("Corrélation entre l'âge et le revenu:", correlation)

Analyse de séries temporelles

L'analyse de séries temporelles est utilisée pour étudier les données qui évoluent dans le temps. Pandas offre des fonctionnalités pour manipuler et analyser des séries temporelles, telles que le filtrage par date, le calcul de la moyenne mobile, etc.

# Importer la bibliothèque Pandas
import pandas as pd

# Charger les données dans un DataFrame avec une colonne de dates
data = pd.read_csv('data.csv', parse_dates=['date'])

# Filtrer les données pour une période spécifique
filtered_data = data[(data['date'] >= '2021-01-01') & (data['date'] <= '2021-12-31')]

# Calculer la moyenne mobile sur 7 jours
filtered_data['moving_average'] = filtered_data['value'].rolling(window=7).mean()

# Afficher les résultats
print(filtered_data)

Analyse de données géospatiales

L'analyse de données géospatiales est utilisée pour étudier les données qui sont liées à des emplacements géographiques. Pandas offre des fonctionnalités pour manipuler et analyser des données géospatiales, telles que le regroupement par région, le calcul de la distance entre deux points, etc.

# Importer la bibliothèque Pandas
import pandas as pd

# Charger les données dans un DataFrame avec des coordonnées géographiques
data = pd.read_csv('data.csv', names=['latitude', 'longitude'])

# Regrouper les données par région
grouped_data = data.groupby('region').size()

# Calculer la distance entre deux points
distance = data.distance('point1', 'point2')

# Afficher les résultats
print("Données groupées par région:")
print(grouped_data)
print("\nDistance entre deux points:")
print(distance)
