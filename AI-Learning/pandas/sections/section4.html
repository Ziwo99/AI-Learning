Manipulation des chaînes de caractères

La manipulation des chaînes de caractères est une tâche courante lors de l'analyse de données. Pandas offre de nombreuses fonctionnalités pour manipuler les chaînes de caractères dans les DataFrames.

Voici quelques exemples de manipulations courantes :

# Création d'une colonne avec la longueur des chaînes de caractères
df['length'] = df['texte'].str.len()

# Conversion en majuscules
df['texte_upper'] = df['texte'].str.upper()

# Extraction d'une sous-chaîne de caractères
df['substring'] = df['texte'].str.slice(0, 5)

# Remplacement de certaines valeurs
df['texte'] = df['texte'].str.replace('a', 'b')

Manipulation des dates et heures

La manipulation des dates et heures est essentielle lors de l'analyse de données temporelles. Pandas offre de nombreuses fonctionnalités pour manipuler les dates et heures dans les DataFrames.

Voici quelques exemples de manipulations courantes :

# Conversion d'une colonne en type datetime
df['date'] = pd.to_datetime(df['date'])

# Extraction de l'année, du mois et du jour
df['year'] = df['date'].dt.year
df['month'] = df['date'].dt.month
df['day'] = df['date'].dt.day

# Calcul de la différence entre deux dates
df['diff'] = df['date1'] - df['date2']

# Filtrage des données par date
df_filtered = df[df['date'] > '2020-01-01']

Manipulation des catégories

La manipulation des catégories est utile lorsque vous avez des données avec des valeurs discrètes. Pandas offre des fonctionnalités pour manipuler les catégories dans les DataFrames.

Voici quelques exemples de manipulations courantes :

# Conversion d'une colonne en catégorie
df['category'] = df['category'].astype('category')

# Renommage des catégories
df['category'] = df['category'].cat.rename_categories({'A': 'Category A', 'B': 'Category B'})

# Tri des catégories par ordre personnalisé
df['category'] = df['category'].cat.reorder_categories(['Category B', 'Category A'], ordered=True)

# Groupement des données par catégorie
df_grouped = df.groupby('category').sum()

Manipulation des données textuelles

La manipulation des données textuelles est courante lors de l'analyse de données non structurées. Pandas offre des fonctionnalités pour manipuler les données textuelles dans les DataFrames.

Voici quelques exemples de manipulations courantes :

# Recherche de motifs dans une colonne
df['contains_pattern'] = df['text'].str.contains('pattern')

# Extraction de mots clés
df['keywords'] = df['text'].str.findall('keyword')

# Suppression des caractères spéciaux
df['text_clean'] = df['text'].str.replace('[^a-zA-Z0-9]', ' ')

# Tokenisation des phrases
df['tokens'] = df['text'].str.split()

Manipulation des données numériques

La manipulation des données numériques est essentielle lors de l'analyse de données quantitatives. Pandas offre des fonctionnalités pour manipuler les données numériques dans les DataFrames.

Voici quelques exemples de manipulations courantes :

# Calcul de la somme, de la moyenne et de la médiane
sum_value = df['column'].sum()
mean_value = df['column'].mean()
median_value = df['column'].median()

# Calcul de la différence entre deux colonnes
df['difference'] = df['column1'] - df['column2']

# Normalisation des données
df['normalized'] = (df['column'] - df['column'].min()) / (df['column'].max() - df['column'].min())

# Calcul de statistiques descriptives
df_stats = df.describe()

Manipulation des données booléennes

La manipulation des données booléennes est courante lors de l'analyse de données binaires. Pandas offre des fonctionnalités pour manipuler les données booléennes dans les DataFrames.

Voici quelques exemples de manipulations courantes :

# Filtrage des données en fonction d'une condition booléenne
df_filtered = df[df['column'] == True]

# Comptage des occurrences de valeurs booléennes
count_true = df['column'].sum()
count_false = len(df) - count_true

# Opérations logiques entre colonnes booléennes
df['result'] = df['column1'] & df['column2']

# Calcul de la proportion de valeurs booléennes
proportion_true = count_true / len(df)
