Importation et exportation des données

L'importation et l'exportation des données sont des étapes essentielles dans la manipulation des données avec Pandas. Pour importer des données dans Pandas, nous pouvons utiliser la fonction read_csv() pour lire des fichiers CSV, la fonction read_excel() pour lire des fichiers Excel, ou la fonction read_sql() pour lire des données à partir d'une base de données SQL.

# Importer la bibliothèque Pandas
import pandas as pd

# Importer un fichier CSV
data = pd.read_csv('data.csv')

# Importer un fichier Excel
data = pd.read_excel('data.xlsx')

# Importer des données à partir d'une base de données SQL
data = pd.read_sql('SELECT * FROM table', connection)

Pour exporter des données à partir de Pandas, nous pouvons utiliser la fonction to_csv() pour exporter des données vers un fichier CSV, la fonction to_excel() pour exporter des données vers un fichier Excel, ou la fonction to_sql() pour exporter des données vers une base de données SQL.

# Exporter des données vers un fichier CSV
data.to_csv('data.csv', index=False)

# Exporter des données vers un fichier Excel
data.to_excel('data.xlsx', index=False)

# Exporter des données vers une base de données SQL
data.to_sql('table', connection, if_exists='replace')

Exploration des données

L'exploration des données est une étape cruciale pour comprendre les caractéristiques et la structure des données. Pandas offre de nombreuses fonctionnalités pour explorer les données, telles que la fonction head() pour afficher les premières lignes du DataFrame, la fonction info() pour obtenir des informations sur les colonnes et les types de données, et la fonction describe() pour obtenir des statistiques descriptives sur les données.

# Afficher les premières lignes du DataFrame
print(data.head())

# Obtenir des informations sur les colonnes et les types de données
print(data.info())

# Obtenir des statistiques descriptives sur les données
print(data.describe())

Nettoyage des données

Le nettoyage des données est une étape cruciale pour garantir la qualité des données. Pandas offre de nombreuses fonctionnalités pour nettoyer les données, telles que la fonction dropna() pour supprimer les lignes contenant des valeurs manquantes, la fonction fillna() pour remplir les valeurs manquantes avec une valeur spécifique, et la fonction replace() pour remplacer les valeurs spécifiques.

# Supprimer les lignes contenant des valeurs manquantes
data = data.dropna()

# Remplir les valeurs manquantes avec une valeur spécifique
data = data.fillna(0)

# Remplacer les valeurs spécifiques
data = data.replace('old_value', 'new_value')

Filtrage des données

Le filtrage des données permet de sélectionner les lignes ou les colonnes qui répondent à certaines conditions. Pandas offre de nombreuses fonctionnalités pour filtrer les données, telles que l'indexation basée sur des conditions, l'utilisation de l'opérateur logique & pour combiner des conditions, et l'utilisation de l'opérateur logique | pour sélectionner les lignes qui répondent à au moins l'une des conditions.

# Filtrer les lignes qui répondent à une condition
filtered_data = data[data['column'] > 10]

# Filtrer les lignes qui répondent à plusieurs conditions
filtered_data = data[(data['column1'] > 10) & (data['column2'] < 20)]

# Filtrer les lignes qui répondent à au moins l'une des conditions
filtered_data = data[(data['column1'] > 10) | (data['column2'] < 20)]

Tri des données

Le tri des données permet de réorganiser les lignes ou les colonnes dans un ordre spécifique. Pandas offre la fonction sort_values() pour trier les données en fonction d'une ou plusieurs colonnes.

# Trier les données en fonction d'une colonne
sorted_data = data.sort_values('column')

# Trier les données en fonction de plusieurs colonnes
sorted_data = data.sort_values(['column1', 'column2'])

Agrégation des données

L'agrégation des données permet de regrouper les données en fonction de certaines caractéristiques et d'effectuer des calculs sur ces groupes. Pandas offre la fonction groupby() pour regrouper les données en fonction d'une ou plusieurs colonnes, et les fonctions d'agrégation telles que sum(), mean(), count(), etc. pour effectuer des calculs sur ces groupes.

# Regrouper les données en fonction d'une colonne et calculer la somme
grouped_data = data.groupby('column').sum()

# Regrouper les données en fonction de plusieurs colonnes et calculer la moyenne
grouped_data = data.groupby(['column1', 'column2']).mean()

Transformation des données

La transformation des données permet d'appliquer des fonctions à des colonnes spécifiques ou à l'ensemble du DataFrame. Pandas offre la fonction apply() pour appliquer une fonction à une colonne spécifique, et la fonction applymap() pour appliquer une fonction à l'ensemble du DataFrame.

# Appliquer une fonction à une colonne spécifique
data['new_column'] = data['column'].apply(function)

# Appliquer une fonction à l'ensemble du DataFrame
data = data.applymap(function)

Fusion des données

La fusion des données permet de combiner plusieurs DataFrames en un seul. Pandas offre la fonction concat() pour concaténer des DataFrames verticalement ou horizontalement, et la fonction merge() pour fusionner des DataFrames en fonction d'une ou plusieurs colonnes communes.

# Concaténer des DataFrames verticalement
concatenated_data = pd.concat([data1, data2])

# Concaténer des DataFrames horizontalement
concatenated_data = pd.concat([data1, data2], axis=1)

# Fusionner des DataFrames en fonction d'une colonne commune
merged_data = pd.merge(data1, data2, on='column')

Gestion des valeurs manquantes

La gestion des valeurs manquantes est une étape importante dans la manipulation des données. Pandas offre la fonction isnull() pour vérifier si une valeur est manquante, la fonction notnull() pour vérifier si une valeur n'est pas manquante, et la fonction dropna() pour supprimer les lignes ou les colonnes contenant des valeurs manquantes.

# Vérifier si une valeur est manquante
missing_values = data.isnull()

# Vérifier si une valeur n'est pas manquante
not_missing_values = data.notnull()

# Supprimer les lignes ou les colonnes contenant des valeurs manquantes
data = data.dropna()
