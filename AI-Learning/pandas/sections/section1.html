Qu'est-ce que Pandas ?

Pandas est une bibliothèque open-source Python qui offre des structures de données et des outils d'analyse de données performants. Elle est largement utilisée dans le domaine de la science des données et est particulièrement adaptée à la manipulation et à l'analyse de données tabulaires.

Pandas permet de traiter facilement des données provenant de différentes sources, telles que des fichiers CSV, des bases de données SQL ou même des feuilles de calcul Excel. Il offre également des fonctionnalités avancées pour nettoyer, transformer et analyser les données, ainsi que pour effectuer des opérations statistiques et de visualisation.

Installation de Pandas

Pour installer Pandas, vous pouvez utiliser l'outil de gestion de paquets pip. Assurez-vous d'avoir Python installé sur votre système avant de procéder à l'installation.

pip install pandas

Une fois l'installation terminée, vous pouvez importer la bibliothèque dans votre code Python en utilisant l'instruction suivante :

import pandas as pd

Les structures de données de base en Pandas

Pandas offre deux structures de données principales : les Séries et les DataFrames.

Une Série est une structure unidimensionnelle qui peut contenir des données de différents types (nombres, chaînes de caractères, etc.). Elle est similaire à un tableau ou à une colonne dans une feuille de calcul.

Un DataFrame est une structure bidimensionnelle qui peut contenir des données de différents types. Il est similaire à une table de base de données ou à une feuille de calcul avec des lignes et des colonnes.

Les Séries et les DataFrames sont flexibles et offrent de nombreuses fonctionnalités pour manipuler et analyser les données. Ils permettent également de réaliser des opérations vectorisées, ce qui les rend très performants pour le traitement de grandes quantités de données.

Voici comment créer une Série et un DataFrame en utilisant Pandas :

# Création d'une Série
s = pd.Series([1, 3, 5, np.nan, 6, 8])

# Création d'un DataFrame
df = pd.DataFrame({'A': [1, 2, 3, 4], 'B': ['a', 'b', 'c', 'd'], 'C': [True, False, True, False]})

La Série s contient des nombres entiers et un NaN (valeur manquante). Le DataFrame df est créé à partir d'un dictionnaire avec trois colonnes : 'A', 'B' et 'C'. La colonne 'A' contient des nombres entiers, la colonne 'B' contient des chaînes de caractères et la colonne 'C' contient des valeurs booléennes.
