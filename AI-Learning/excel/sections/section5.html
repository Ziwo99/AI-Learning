Section 5 : Analyse des données avec Excel

Utilisation des tableaux croisés dynamiques

Les tableaux croisés dynamiques sont un outil puissant dans Excel pour analyser et résumer des données. Ils permettent de regrouper et de synthétiser des informations à partir d'un ensemble de données.

Pour créer un tableau croisé dynamique, suivez les étapes suivantes :

  1. Sélectionnez les données que vous souhaitez analyser.
  2. Cliquez sur l'onglet "Insertion" dans le ruban Excel.
  3. Cliquez sur le bouton "Tableau croisé dynamique" dans le groupe "Tableaux".
  4. Sélectionnez l'emplacement où vous souhaitez placer le tableau croisé dynamique.
  5. Dans la boîte de dialogue qui s'ouvre, faites glisser les champs de données dans les zones appropriées (lignes, colonnes, valeurs).
  6. Personnalisez le tableau croisé dynamique en utilisant les options disponibles dans le ruban Excel.

Voici un exemple de code pour créer un tableau croisé dynamique à partir d'une plage de données :

Sub CreatePivotTable()
    Dim ws As Worksheet
    Dim rng As Range
    Dim pvtCache As PivotCache
    Dim pvtTable As PivotTable
    
    ' Set the worksheet and range variables
    Set ws = ThisWorkbook.Worksheets("Sheet1")
    Set rng = ws.Range("A1:D10")
    
    ' Create a new PivotCache object
    Set pvtCache = ThisWorkbook.PivotCaches.Create(SourceType:=xlDatabase, SourceData:=rng)
    
    ' Create a new PivotTable object
    Set pvtTable = pvtCache.CreatePivotTable(TableDestination:=ws.Range("F1"), TableName:="PivotTable1")
    
    ' Add fields to the PivotTable
    With pvtTable
        .PivotFields("Category").Orientation = xlRowField
        .PivotFields("Product").Orientation = xlRowField
        .PivotFields("Sales").Orientation = xlDataField
    End With
End Sub

Utilisation des graphiques croisés dynamiques

Les graphiques croisés dynamiques sont une autre fonctionnalité d'Excel qui permet de visualiser les données d'un tableau croisé dynamique sous forme de graphique. Ils offrent une représentation visuelle des informations pour une meilleure compréhension.

Pour créer un graphique croisé dynamique, procédez comme suit :

  1. Sélectionnez le tableau croisé dynamique que vous souhaitez représenter graphiquement.
  2. Cliquez sur l'onglet "Insertion" dans le ruban Excel.
  3. Cliquez sur le type de graphique souhaité dans le groupe "Graphiques recommandés" ou "Autres graphiques".
  4. Personnalisez le graphique en utilisant les options disponibles dans le ruban Excel.

Voici un exemple de code pour créer un graphique croisé dynamique à partir d'un tableau croisé dynamique existant :

Sub CreatePivotChart()
    Dim ws As Worksheet
    Dim pvtTable As PivotTable
    Dim pvtChart As Chart
    
    ' Set the worksheet and PivotTable variables
    Set ws = ThisWorkbook.Worksheets("Sheet1")
    Set pvtTable = ws.PivotTables("PivotTable1")
    
    ' Create a new PivotChart object
    Set pvtChart = ws.Shapes.AddChart2(240, xlColumnClustered).Chart
    
    ' Set the source data for the PivotChart
    pvtChart.SetSourceData Source:=pvtTable.TableRange1
    
    ' Customize the PivotChart
    With pvtChart
        .HasTitle = True
        .ChartTitle.Text = "Sales by Category and Product"
        .Axes(xlCategory).HasTitle = True
        .Axes(xlCategory).AxisTitle.Text = "Category"
        .Axes(xlValue).HasTitle = True
        .Axes(xlValue).AxisTitle.Text = "Sales"
    End With
End Sub

Utilisation des fonctions statistiques

Excel propose de nombreuses fonctions statistiques pour analyser et interpréter des données. Ces fonctions permettent de calculer des mesures telles que la moyenne, la médiane, l'écart type, etc.

Voici quelques exemples de fonctions statistiques couramment utilisées :

  • AVERAGE : calcule la moyenne des valeurs d'une plage de cellules.
  • MEDIAN : renvoie la valeur médiane d'une plage de cellules.
  • STDEV : calcule l'écart type des valeurs d'une plage de cellules.
  • COUNT : compte le nombre de cellules contenant des valeurs numériques dans une plage.

Voici un exemple de code utilisant la fonction statistique AVERAGE :

Sub CalculateAverage()
    Dim ws As Worksheet
    Dim rng As Range
    Dim averageValue As Double
    
    ' Set the worksheet and range variables
    Set ws = ThisWorkbook.Worksheets("Sheet1")
    Set rng = ws.Range("A1:A10")
    
    ' Calculate the average value
    averageValue = Application.WorksheetFunction.Average(rng)
    
    ' Display the average value
    MsgBox "The average value is: " & averageValue
End Sub
