Introduction aux requêtes avec Hibernate

Hibernate offre plusieurs méthodes pour effectuer des requêtes sur la base de données. Ces requêtes peuvent être utilisées pour récupérer des données, les modifier ou les supprimer. Les requêtes avec Hibernate peuvent être écrites en utilisant le langage HQL (Hibernate Query Language), les critères de requête ou les requêtes natives.

Utilisation du langage HQL (Hibernate Query Language)

Le langage HQL est un langage de requête orienté objet qui permet d'écrire des requêtes en utilisant les objets et les propriétés de vos entités. Il est similaire au SQL, mais utilise les noms des classes et des propriétés au lieu des noms des tables et des colonnes.

Voici un exemple de requête HQL pour récupérer tous les utilisateurs dont le nom commence par "John" : 

String hql = "FROM User u WHERE u.name LIKE 'John%'";
Query query = session.createQuery(hql);
List<User> users = query.list();

Utilisation des critères de requête

Les critères de requête sont une autre méthode pour effectuer des requêtes avec Hibernate. Ils permettent de construire des requêtes de manière programmatique en utilisant des objets Criteria. Les critères de requête offrent une syntaxe fluide et expressive pour spécifier les conditions de recherche.

Voici un exemple de requête avec des critères pour récupérer tous les utilisateurs dont l'âge est supérieur à 18 ans : 

Criteria criteria = session.createCriteria(User.class);
criteria.add(Restrictions.gt("age", 18));
List<User> users = criteria.list();

Utilisation des requêtes natives

Les requêtes natives permettent d'écrire des requêtes SQL directement dans votre code Hibernate. Elles peuvent être utiles lorsque vous avez besoin d'utiliser des fonctionnalités spécifiques de votre base de données qui ne sont pas prises en charge par HQL ou les critères de requête.

Voici un exemple de requête native pour récupérer tous les utilisateurs dont le nom commence par "John" : 

String sql = "SELECT * FROM users WHERE name LIKE 'John%'";
SQLQuery query = session.createSQLQuery(sql);
query.addEntity(User.class);
List<User> users = query.list();

Ces différentes méthodes de requête offrent une grande flexibilité pour interagir avec votre base de données en utilisant Hibernate. Choisissez la méthode qui convient le mieux à votre cas d'utilisation et assurez-vous de comprendre les avantages et les limitations de chaque méthode.
