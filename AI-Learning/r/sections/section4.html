Statistiques descriptives

Les statistiques descriptives sont utilisées pour résumer et décrire les caractéristiques principales d'un ensemble de données. Elles fournissent des mesures telles que la moyenne, la médiane, l'écart-type, etc., qui permettent de comprendre la distribution et les tendances des données.

Voici un exemple de code pour calculer la moyenne et l'écart-type d'un vecteur de données :

# Création d'un vecteur de données
data <- c(10, 15, 20, 25, 30)

# Calcul de la moyenne
mean_value <- mean(data)
print(mean_value)

# Calcul de l'écart-type
sd_value <- sd(data)
print(sd_value)

Dans cet exemple, nous avons créé un vecteur de données appelé "data" contenant les valeurs 10, 15, 20, 25 et 30. En utilisant les fonctions mean() et sd(), nous avons calculé la moyenne et l'écart-type de ce vecteur respectivement. Les résultats sont ensuite affichés à l'aide de la fonction print().

Tests statistiques

Les tests statistiques sont utilisés pour évaluer l'hypothèse nulle selon laquelle il n'y a pas de différence significative entre les groupes ou les variables. Ils permettent de prendre des décisions basées sur des preuves statistiques.

Voici un exemple de code pour effectuer un test t de Student sur deux échantillons :

# Création de deux échantillons
sample1 <- c(10, 15, 20, 25, 30)
sample2 <- c(12, 18, 22, 28, 32)

# Test t de Student
t_test <- t.test(sample1, sample2)
print(t_test)

Dans cet exemple, nous avons créé deux échantillons, "sample1" et "sample2", contenant des valeurs numériques. En utilisant la fonction t.test(), nous avons effectué un test t de Student pour comparer les moyennes des deux échantillons. Les résultats du test, y compris la statistique t, la valeur p et l'intervalle de confiance, sont affichés à l'aide de la fonction print().

Régression linéaire

La régression linéaire est utilisée pour modéliser la relation entre une variable dépendante et une ou plusieurs variables indépendantes. Elle permet de prédire la valeur de la variable dépendante en fonction des valeurs des variables indépendantes.

Voici un exemple de code pour effectuer une régression linéaire simple :

# Création des variables indépendante et dépendante
x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)

# Régression linéaire
lm_model <- lm(y ~ x)
print(lm_model)

Dans cet exemple, nous avons créé deux vecteurs, "x" et "y", représentant respectivement les variables indépendante et dépendante. En utilisant la fonction lm(), nous avons ajusté un modèle de régression linéaire simple en utilisant la formule "y ~ x". Le modèle ajusté est ensuite affiché à l'aide de la fonction print().

Modèles de régression avancés

En plus de la régression linéaire simple, R offre également des fonctionnalités pour ajuster des modèles de régression plus avancés, tels que la régression linéaire multiple, la régression logistique, la régression polynomiale, etc. Ces modèles permettent de prendre en compte des relations plus complexes entre les variables.

Voici un exemple de code pour effectuer une régression linéaire multiple :

# Création des variables indépendantes et dépendante
x1 <- c(1, 2, 3, 4, 5)
x2 <- c(2, 4, 6, 8, 10)
y <- c(3, 6, 9, 12, 15)

# Régression linéaire multiple
lm_model <- lm(y ~ x1 + x2)
print(lm_model)

Dans cet exemple, nous avons créé trois vecteurs, "x1", "x2" et "y", représentant respectivement les variables indépendantes et dépendante. En utilisant la fonction lm(), nous avons ajusté un modèle de régression linéaire multiple en utilisant la formule "y ~ x1 + x2". Le modèle ajusté est ensuite affiché à l'aide de la fonction print().

Analyse de variance (ANOVA)

L'analyse de variance (ANOVA) est utilisée pour comparer les moyennes de plusieurs groupes et déterminer s'il existe des différences significatives entre eux. Elle permet de tester l'hypothèse nulle selon laquelle les moyennes sont égales pour tous les groupes.

Voici un exemple de code pour effectuer une ANOVA à un facteur :

# Création des groupes
group1 <- c(10, 15, 20, 25, 30)
group2 <- c(12, 18, 22, 28, 32)
group3 <- c(8, 16, 24, 32, 40)

# ANOVA à un facteur
anova_result <- aov(c(group1, group2, group3) ~ c(rep("Group 1", 5), rep("Group 2", 5), rep("Group 3", 5)))
print(anova_result)

Dans cet exemple, nous avons créé trois vecteurs, "group1", "group2" et "group3", représentant respectivement les données des trois groupes. En utilisant la fonction aov(), nous avons effectué une ANOVA à un facteur en utilisant la formule "group ~ factor", où "group" est un vecteur contenant toutes les données des groupes et "factor" est un vecteur indiquant l'appartenance à chaque groupe. Les résultats de l'ANOVA sont ensuite affichés à l'aide de la fonction print().

Analyse de données multivariées

L'analyse de données multivariées est utilisée pour étudier les relations entre plusieurs variables simultanément. Elle permet de réduire la dimensionnalité des données, d'identifier des structures cachées et de visualiser les relations entre les variables.

Voici un exemple de code pour effectuer une analyse en composantes principales (PCA) :

# Création des variables
x1 <- c(1, 2, 3, 4, 5)
x2 <- c(2, 4, 6, 8, 10)
x3 <- c(3, 6, 9, 12, 15)

# PCA
pca_result <- prcomp(data.frame(x1, x2, x3))
print(pca_result)

Dans cet exemple, nous avons créé trois vecteurs, "x1", "x2" et "x3", représentant respectivement les variables. En utilisant la fonction prcomp(), nous avons effectué une analyse en composantes principales (PCA) en utilisant les données sous forme de data frame. Les résultats de la PCA, y compris les valeurs propres, les vecteurs propres et les coordonnées des individus, sont ensuite affichés à l'aide de la fonction print().
