Syntaxe de base en R

La syntaxe de base en R est relativement simple et intuitive. Voici quelques éléments clés à connaître :

1. Les objets : en R, vous pouvez stocker des valeurs dans des objets. Par exemple, vous pouvez créer un objet appelé "x" et lui attribuer la valeur 5 en utilisant l'opérateur d'affectation "<-".

x <- 5

2. Les opérations mathématiques : R prend en charge les opérations mathématiques de base telles que l'addition, la soustraction, la multiplication et la division. Par exemple, vous pouvez ajouter deux nombres en utilisant l'opérateur "+".

resultat <- 2 + 3

3. Les fonctions : R dispose d'un large éventail de fonctions intégrées qui peuvent être utilisées pour effectuer des opérations spécifiques. Par exemple, la fonction "mean()" peut être utilisée pour calculer la moyenne d'un vecteur de nombres.

vecteur <- c(1, 2, 3, 4, 5)
moyenne <- mean(vecteur)

Structures de contrôle

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme en fonction de certaines conditions. Voici quelques structures de contrôle couramment utilisées en R :

1. Les boucles : les boucles permettent de répéter une série d'instructions plusieurs fois. La boucle "for" est couramment utilisée pour itérer sur une séquence de valeurs.

for (i in 1:5) {
  print(i)
}

2. Les instructions conditionnelles : les instructions conditionnelles permettent d'exécuter certaines instructions uniquement si une condition spécifiée est vraie. L'instruction "if" est couramment utilisée pour cela.

x <- 10
if (x > 5) {
  print("x est supérieur à 5")
}

Fonctions et packages

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. R dispose d'un grand nombre de fonctions intégrées, mais vous pouvez également créer vos propres fonctions. Voici comment définir une fonction en R :

maFonction <- function(parametre1, parametre2) {
  # Instructions de la fonction
}

Les packages sont des ensembles de fonctions et de données supplémentaires qui étendent les fonctionnalités de base de R. Vous pouvez installer des packages à partir du référentiel CRAN à l'aide de la fonction "install.packages()". Une fois installé, vous pouvez charger un package à l'aide de la fonction "library()".

install.packages("nom_du_package")
library(nom_du_package)

Programmation orientée objet en R

R prend également en charge la programmation orientée objet (POO), qui permet de structurer le code en utilisant des objets et des classes. Voici comment créer une classe en R :

MaClasse <- setClass("MaClasse",
                   slots = list(
                     attribut1 = "numeric",
                     attribut2 = "character"
                   ),
                   prototype = list(
                     attribut1 = 0,
                     attribut2 = ""
                   ))

Une fois que vous avez défini une classe, vous pouvez créer des objets de cette classe en utilisant la fonction "new()".

monObjet <- new(MaClasse)

Ce sont là quelques-uns des concepts de base de la programmation en R. En comprenant ces concepts, vous serez en mesure de créer des programmes R plus avancés et de tirer pleinement parti de la puissance de ce langage.
