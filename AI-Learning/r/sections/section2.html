Importation de données

L'importation de données est une étape essentielle dans l'analyse de données avec R. Elle permet de charger des données externes dans l'environnement R afin de les manipuler et de les analyser. R propose plusieurs méthodes pour importer des données, notamment :

  • La fonction read.csv() pour importer des fichiers CSV
  • La fonction read.table() pour importer des fichiers texte
  • La fonction read.xlsx() du package "readxl" pour importer des fichiers Excel

Voici un exemple d'importation d'un fichier CSV :

data <- read.csv("chemin/vers/fichier.csv")

Exploration des données

L'exploration des données est une étape cruciale pour comprendre la structure et les caractéristiques des données importées. R propose de nombreuses fonctions pour explorer les données, telles que :

  • head() : affiche les premières lignes du jeu de données
  • summary() : fournit un résumé statistique des variables
  • str() : affiche la structure des données
  • dim() : donne les dimensions (nombre de lignes et de colonnes) du jeu de données

Voici un exemple d'utilisation de ces fonctions :

# Afficher les premières lignes du jeu de données
head(data)

# Obtenir un résumé statistique des variables
summary(data)

# Afficher la structure des données
str(data)

# Obtenir les dimensions du jeu de données
dim(data)

Nettoyage des données

Le nettoyage des données est une étape cruciale pour éliminer les valeurs manquantes, les valeurs aberrantes et les erreurs dans les données importées. R propose plusieurs fonctions pour nettoyer les données, notamment :

  • is.na() : vérifie si une valeur est manquante
  • na.omit() : supprime les lignes contenant des valeurs manquantes
  • complete.cases() : identifie les lignes complètes (sans valeurs manquantes)
  • outlier() : détecte les valeurs aberrantes

Voici un exemple d'utilisation de ces fonctions :

# Vérifier si une valeur est manquante
is.na(data$variable)

# Supprimer les lignes contenant des valeurs manquantes
data_clean <- na.omit(data)

# Identifier les lignes complètes
complete_rows <- complete.cases(data)

# Détecter les valeurs aberrantes
outliers <- outlier(data$variable)

Transformation des données

La transformation des données permet de modifier la structure ou les valeurs des données pour les rendre plus adaptées à l'analyse. R propose de nombreuses fonctions pour transformer les données, telles que :

  • mutate() : ajoute de nouvelles variables calculées à partir des variables existantes
  • filter() : filtre les lignes en fonction de certaines conditions
  • select() : sélectionne les variables à inclure dans le jeu de données
  • arrange() : trie les lignes en fonction d'une ou plusieurs variables

Voici un exemple d'utilisation de ces fonctions :

# Ajouter une nouvelle variable calculée
data_transformed <- mutate(data, nouvelle_variable = variable1 + variable2)

# Filtrer les lignes en fonction d'une condition
data_filtered <- filter(data, variable > 10)

# Sélectionner les variables à inclure dans le jeu de données
data_selected <- select(data, variable1, variable2)

# Trier les lignes en fonction d'une variable
data_sorted <- arrange(data, variable)

Visualisation des données

La visualisation des données est une étape essentielle pour comprendre les relations et les tendances dans les données. R propose de nombreuses bibliothèques pour créer des graphiques et des visualisations, notamment :

  • ggplot2 : une bibliothèque populaire pour créer des graphiques élégants
  • plotly : une bibliothèque interactive pour créer des graphiques interactifs
  • ggvis : une bibliothèque pour créer des graphiques interactifs basés sur ggplot2

Voici un exemple de création d'un graphique à l'aide de ggplot2 :

# Charger la bibliothèque ggplot2
library(ggplot2)

# Créer un graphique à barres
ggplot(data, aes(x = variable1, y = variable2)) +
  geom_bar(stat = "identity")
