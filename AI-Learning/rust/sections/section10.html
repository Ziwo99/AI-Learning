Création d'une application de ligne de commande

Dans cette sous-section, nous allons apprendre à créer une application de ligne de commande en utilisant Rust. Une application de ligne de commande est un programme qui s'exécute dans le terminal et permet à l'utilisateur d'interagir avec le programme en entrant des commandes.

Pour commencer, nous allons créer un nouveau projet Rust en utilisant la commande suivante :

cargo new ligne-de-commande
cd ligne-de-commande

Ensuite, nous allons créer un fichier source principal pour notre application en utilisant la commande suivante :

touch src/main.rs

Ouvrez le fichier src/main.rs dans votre éditeur de code préféré et ajoutez le code suivant :

use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();

    println!("Les arguments passés sont : {:?}", args);
}

Le code ci-dessus utilise le module std::env pour récupérer les arguments passés à l'application de ligne de commande. La fonction env::args() renvoie un itérable contenant les arguments, que nous collectons dans un vecteur de chaînes de caractères.

Pour exécuter notre application, utilisez la commande suivante :

cargo run -- arg1 arg2 arg3

Remplacez arg1, arg2 et arg3 par les arguments que vous souhaitez passer à l'application. Vous devriez voir les arguments affichés dans le terminal.

Développement d'une application web

Dans cette sous-section, nous allons apprendre à développer une application web en utilisant Rust. Rust offre plusieurs frameworks web populaires tels que Rocket, Actix et Warp, qui facilitent le développement d'applications web rapides et sécurisées.

Pour commencer, nous allons créer un nouveau projet Rust en utilisant la commande suivante :

cargo new application-web
cd application-web

Ensuite, nous allons ajouter les dépendances nécessaires à notre projet en modifiant le fichier Cargo.toml comme suit :

[dependencies]
rocket = "0.5.0"

Ensuite, nous allons créer un fichier source principal pour notre application en utilisant la commande suivante :

touch src/main.rs

Ouvrez le fichier src/main.rs dans votre éditeur de code préféré et ajoutez le code suivant :

#[macro_use]
extern crate rocket;

#[get("/")]
fn index() -> &'static str {
    "Hello, world!"
}

#[launch]
fn rocket() -> _ {
    rocket::build().mount("/", routes![index])
}

Le code ci-dessus utilise le framework Rocket pour créer une application web simple. La fonction index() est une route qui renvoie la chaîne de caractères "Hello, world!" lorsque l'utilisateur accède à la racine de l'application. La fonction rocket() initialise l'application Rocket et monte la route index() à la racine de l'application.

Pour exécuter notre application, utilisez la commande suivante :

cargo run

Ouvrez votre navigateur et accédez à l'adresse http://localhost:8000. Vous devriez voir le message "Hello, world!" affiché dans votre navigateur.

Création d'une bibliothèque réutilisable

Dans cette sous-section, nous allons apprendre à créer une bibliothèque réutilisable en utilisant Rust. Une bibliothèque est un ensemble de fonctionnalités qui peuvent être utilisées par d'autres programmes.

Pour commencer, nous allons créer un nouveau projet Rust en utilisant la commande suivante :

cargo new bibliotheque
cd bibliotheque

Ensuite, nous allons créer un fichier source principal pour notre bibliothèque en utilisant la commande suivante :

touch src/lib.rs

Ouvrez le fichier src/lib.rs dans votre éditeur de code préféré et ajoutez le code suivant :

pub fn add(a: i32, b: i32) -> i32 {
    a + b
}

Le code ci-dessus définit une fonction add() qui prend deux entiers en entrée et renvoie leur somme.

Pour utiliser notre bibliothèque dans un autre projet, nous devons l'ajouter en tant que dépendance dans le fichier Cargo.toml du projet. Ajoutez la ligne suivante :

[dependencies]
bibliotheque = { path = "../bibliotheque" }

Ensuite, dans le fichier source principal du projet, vous pouvez utiliser la bibliothèque en ajoutant la ligne suivante :

use bibliotheque::add;

Vous pouvez maintenant utiliser la fonction add() dans votre projet.

Pour tester notre bibliothèque, nous allons créer un fichier source principal pour notre projet en utilisant la commande suivante :

touch src/main.rs

Ouvrez le fichier src/main.rs dans votre éditeur de code préféré et ajoutez le code suivant :

use bibliotheque::add;

fn main() {
    let result = add(2, 3);
    println!("Le résultat de l'addition est : {}", result);
}

Pour exécuter notre projet, utilisez la commande suivante :

cargo run

Vous devriez voir le résultat de l'addition affiché dans le terminal.
