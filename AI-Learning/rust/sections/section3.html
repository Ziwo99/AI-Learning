Variables et types de données

En Rust, les variables sont déclarées en utilisant le mot-clé "let". La syntaxe générale pour déclarer une variable est la suivante :

let nom_de_variable = valeur;

Par exemple, pour déclarer une variable entière nommée "x" avec une valeur de 5, on peut écrire :

let x = 5;

Rust est un langage à typage statique, ce qui signifie que le type de chaque variable doit être spécifié lors de sa déclaration. Rust offre plusieurs types de données primitifs, tels que :

  • Entiers : i8, i16, i32, i64, u8, u16, u32, u64
  • Flottants : f32, f64
  • Booléens : bool
  • Caractères : char

Il est également possible de déclarer des variables avec des types personnalisés en utilisant des structures ou des énumérations.

Opérations et expressions

En Rust, les opérations arithmétiques de base sont les suivantes :

  • Addition : "+". Par exemple, "let sum = 2 + 3;"
  • Soustraction : "-". Par exemple, "let difference = 5 - 2;"
  • Multiplication : "*". Par exemple, "let product = 4 * 3;"
  • Division : "/". Par exemple, "let quotient = 10 / 2;"
  • Modulo : "%". Par exemple, "let remainder = 7 % 3;"

Les expressions en Rust sont évaluées et renvoient une valeur. Par exemple :

let x = 5;
  let y = 10;
  let sum = x + y; // sum vaut 15

Structures de contrôle (boucles, conditions)

En Rust, les structures de contrôle permettent de prendre des décisions et de répéter des instructions. Voici quelques exemples :

Conditions :

let x = 5;

  if x > 0 {
      println!("x est positif");
  } else if x < 0 {
      println!("x est négatif");
  } else {
      println!("x est nul");
  }

Boucles :

La boucle "while" permet de répéter des instructions tant qu'une condition est vraie :

let mut i = 0;

  while i < 5 {
      println!("Valeur de i : {}", i);
      i += 1;
  }

La boucle "for" permet de parcourir une collection d'éléments :

let numbers = [1, 2, 3, 4, 5];

  for number in numbers.iter() {
      println!("Valeur : {}", number);
  }

Fonctions et modules

En Rust, les fonctions sont déclarées en utilisant le mot-clé "fn". Voici un exemple de déclaration de fonction :

fn add(x: i32, y: i32) -> i32 {
      return x + y;
  }

Les modules permettent d'organiser le code en regroupant des fonctions, des structures et d'autres éléments. Voici un exemple de déclaration de module :

mod math {
      pub fn add(x: i32, y: i32) -> i32 {
          return x + y;
      }
  }
