Les bibliothèques standard de Rust

Les bibliothèques standard de Rust sont un ensemble de modules et de types prédéfinis qui font partie intégrante de l'installation de Rust. Elles fournissent des fonctionnalités de base pour la manipulation des chaînes de caractères, les opérations d'entrée/sortie, la gestion des fichiers, les collections de données, les opérations mathématiques, etc.

Voici quelques exemples de bibliothèques standard couramment utilisées en Rust :

use std::fs::File;
use std::io::prelude::*;
use std::collections::HashMap;

Dans cet exemple, nous importons les modules File et prelude du module io de la bibliothèque standard, ainsi que le module HashMap du module collections.

Les bibliothèques tierces populaires

En plus des bibliothèques standard, il existe de nombreuses bibliothèques tierces populaires disponibles pour Rust. Ces bibliothèques offrent des fonctionnalités supplémentaires et des outils pour faciliter le développement en Rust.

Voici quelques exemples de bibliothèques tierces populaires en Rust :

extern crate reqwest;
extern crate serde;
extern crate actix_web;

Dans cet exemple, nous importons les bibliothèques tierces reqwest, serde et actix_web. Ces bibliothèques sont utilisées respectivement pour effectuer des requêtes HTTP, la sérialisation/désérialisation de données et le développement d'applications web.

Les frameworks web en Rust

Les frameworks web en Rust sont des bibliothèques qui fournissent une infrastructure pour développer des applications web. Ils facilitent la gestion des routes, des requêtes HTTP, des modèles de données, etc.

Voici quelques exemples de frameworks web populaires en Rust :

use actix_web::{web, App, HttpResponse, HttpServer};

async fn index() -> HttpResponse {
    HttpResponse::Ok().body("Hello, world!")
}

#[actix_web::main]
async fn main() -> std::io::Result<()> {
    HttpServer::new(|| {
        App::new()
            .route("/", web::get().to(index))
    })
    .bind("127.0.0.1:8080")?
    .run()
    .await
}

Dans cet exemple, nous utilisons le framework web actix_web pour créer une application web simple qui renvoie "Hello, world!" en réponse à une requête GET sur la route principale ("/").

Il est important de noter que les bibliothèques et les frameworks en Rust évoluent rapidement, il est donc recommandé de consulter la documentation officielle et les ressources communautaires pour obtenir les dernières informations sur les bibliothèques et les frameworks disponibles.
