Les structures et les méthodes

En Rust, les structures sont utilisées pour définir des types de données personnalisés. Elles permettent de regrouper plusieurs valeurs et de définir des méthodes associées à ces valeurs. Une méthode est une fonction qui peut être appelée sur une instance d'une structure.

Voici un exemple de déclaration d'une structure en Rust :

struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
}

Dans cet exemple, nous définissons une structure appelée "Rectangle" avec deux champs : "width" et "height". Ensuite, nous utilisons l'implémentation ("impl") pour définir une méthode "area" qui calcule et renvoie l'aire du rectangle.

Pour créer une instance d'une structure et appeler sa méthode, voici comment cela peut être fait :

let rect = Rectangle { width: 10, height: 20 };
let area = rect.area();
println!("L'aire du rectangle est : {}", area);

Ce code crée une instance de la structure "Rectangle" avec une largeur de 10 et une hauteur de 20. Ensuite, il appelle la méthode "area" sur cette instance et affiche le résultat.

L'héritage et le polymorphisme

En Rust, il n'y a pas de concept d'héritage de classe comme dans d'autres langages orientés objet. Cependant, Rust offre une fonctionnalité similaire appelée "trait" qui permet de définir des comportements communs à plusieurs types.

Un trait est une collection de méthodes qui peuvent être implémentées par différents types. Voici un exemple de déclaration d'un trait en Rust :

trait Shape {
    fn area(&self) -> u32;
    fn perimeter(&self) -> u32;
}

Dans cet exemple, nous définissons un trait appelé "Shape" avec deux méthodes : "area" et "perimeter". Tous les types qui implémentent ce trait doivent fournir une implémentation de ces méthodes.

Pour implémenter un trait pour une structure, nous utilisons l'implémentation ("impl") comme suit :

struct Rectangle {
    width: u32,
    height: u32,
}

impl Shape for Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
    
    fn perimeter(&self) -> u32 {
        2 * (self.width + self.height)
    }
}

Dans cet exemple, nous implémentons le trait "Shape" pour la structure "Rectangle" en fournissant des implémentations des méthodes "area" et "perimeter".

En utilisant le polymorphisme, nous pouvons traiter différentes instances de structures qui implémentent le même trait de manière uniforme. Voici comment cela peut être fait :

fn print_shape_info(shape: &dyn Shape) {
    println!("L'aire du shape est : {}", shape.area());
    println!("Le périmètre du shape est : {}", shape.perimeter());
}

let rect = Rectangle { width: 10, height: 20 };
print_shape_info(&rect);

Ce code définit une fonction "print_shape_info" qui prend une référence à un objet qui implémente le trait "Shape". Ensuite, il appelle cette fonction en passant une instance de la structure "Rectangle". La fonction peut accéder aux méthodes "area" et "perimeter" de l'objet de manière polymorphe.

Les traits et les implémentations

En Rust, les traits sont utilisés pour définir des comportements communs à plusieurs types. Une implémentation de trait est une façon de fournir une implémentation des méthodes définies dans le trait pour un type spécifique.

Voici un exemple de déclaration d'un trait en Rust :

trait Printable {
    fn print(&self);
}

Dans cet exemple, nous définissons un trait appelé "Printable" avec une méthode "print". Tous les types qui implémentent ce trait doivent fournir une implémentation de cette méthode.

Pour implémenter un trait pour un type, nous utilisons l'implémentation ("impl") comme suit :

struct Rectangle {
    width: u32,
    height: u32,
}

impl Printable for Rectangle {
    fn print(&self) {
        println!("Rectangle de largeur {} et de hauteur {}", self.width, self.height);
    }
}

Dans cet exemple, nous implémentons le trait "Printable" pour la structure "Rectangle" en fournissant une implémentation de la méthode "print".

Une fois qu'un type implémente un trait, nous pouvons utiliser les méthodes du trait sur des instances de ce type. Voici comment cela peut être fait :

let rect = Rectangle { width: 10, height: 20 };
rect.print();

Ce code crée une instance de la structure "Rectangle" et appelle la méthode "print" du trait "Printable" sur cette instance.

Conclusion

Dans cette section, nous avons exploré la programmation orientée objet en Rust. Nous avons appris à définir des structures et des méthodes, à utiliser le polymorphisme avec les traits et les implémentations, et à fournir des comportements communs à plusieurs types. La programmation orientée objet en Rust offre une approche flexible et puissante pour organiser et réutiliser du code.
