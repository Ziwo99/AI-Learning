Syntaxe de base

La syntaxe de base en C# est assez similaire à celle des autres langages de programmation. Voici un exemple de code simple en C# :

using System;

class Program {
    static void Main(string[] args) {
        Console.WriteLine("Hello, World!");
    }
}

Dans cet exemple, nous utilisons la classe Program avec une méthode Main qui est le point d'entrée de notre programme. La méthode Main affiche simplement le message "Hello, World!" à l'aide de la méthode WriteLine de la classe Console.

Variables et types de données

En C#, les variables sont utilisées pour stocker des données. Chaque variable a un type de données qui détermine le type de valeurs qu'elle peut contenir. Voici quelques exemples de types de données couramment utilisés en C# :

int age = 25;
double salary = 2500.50;
char grade = 'A';
string name = "John Doe";
bool isEmployed = true;

Dans cet exemple, nous avons déclaré des variables avec différents types de données tels que int pour les entiers, double pour les nombres à virgule flottante, char pour les caractères, string pour les chaînes de caractères et bool pour les valeurs booléennes.

Opérateurs et expressions

Les opérateurs en C# sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques exemples d'opérateurs couramment utilisés en C# :

int a = 10;
int b = 5;
int sum = a + b;
int difference = a - b;
int product = a * b;
double quotient = (double)a / b;
int remainder = a % b;

Dans cet exemple, nous utilisons des opérateurs tels que + pour l'addition, - pour la soustraction, * pour la multiplication, / pour la division et % pour le reste de la division.

Structures de contrôle (boucles, conditions)

Les structures de contrôle en C# sont utilisées pour contrôler le flux d'exécution du programme. Voici quelques exemples de structures de contrôle couramment utilisées en C# :

int age = 18;

if (age >= 18) {
    Console.WriteLine("Vous êtes majeur.");
} else {
    Console.WriteLine("Vous êtes mineur.");
}

for (int i = 0; i < 5; i++) {
    Console.WriteLine("Itération " + i);
}

while (age >= 18) {
    Console.WriteLine("Vous êtes majeur.");
    age--;
}

Dans cet exemple, nous utilisons une structure de contrôle if-else pour effectuer une action en fonction d'une condition. Nous utilisons également une boucle for pour répéter une action un certain nombre de fois et une boucle while pour répéter une action tant qu'une condition est vraie.

Fonctions et méthodes

Les fonctions et les méthodes en C# sont utilisées pour regrouper des instructions et les réutiliser à plusieurs endroits dans le programme. Voici un exemple de fonction en C# :

int Add(int a, int b) {
    return a + b;
}

int sum = Add(5, 3);
Console.WriteLine(sum); // Affiche 8

Dans cet exemple, nous avons défini une fonction Add qui prend deux paramètres a et b et renvoie leur somme. Nous appelons ensuite cette fonction avec les arguments 5 et 3 et stockons le résultat dans la variable sum.

Gestion des erreurs (exceptions)

La gestion des erreurs en C# est réalisée à l'aide de mécanismes d'exceptions. Voici un exemple de gestion d'exception en C# :

try {
    int result = 10 / 0;
} catch (DivideByZeroException ex) {
    Console.WriteLine("Erreur : division par zéro.");
}

Dans cet exemple, nous essayons de diviser 10 par 0, ce qui provoque une exception de division par zéro. Nous utilisons ensuite une structure try-catch pour capturer cette exception et afficher un message d'erreur approprié.

Ceci conclut la section sur les bases de la programmation en C#. Nous avons couvert la syntaxe de base, les variables et les types de données, les opérateurs et les expressions, les structures de contrôle, les fonctions et les méthodes, ainsi que la gestion des erreurs.
