Section 5: Manipulation des fichiers et des dossiers

Retour au cours

Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en programmation. En C#, vous pouvez utiliser la classe File du namespace System.IO pour effectuer ces opérations.

Pour lire le contenu d'un fichier, vous pouvez utiliser la méthode ReadAllText de la classe File. Cette méthode prend en paramètre le chemin du fichier et renvoie une chaîne de caractères contenant le contenu du fichier.

string content = File.ReadAllText("chemin/vers/le/fichier.txt");
Console.WriteLine(content);

Pour écrire dans un fichier, vous pouvez utiliser la méthode WriteAllText de la classe File. Cette méthode prend en paramètre le chemin du fichier et la chaîne de caractères à écrire.

string content = "Contenu à écrire dans le fichier";
File.WriteAllText("chemin/vers/le/fichier.txt", content);

Manipulation des dossiers et des chemins de fichiers

En plus de la manipulation des fichiers, vous pouvez également manipuler les dossiers et les chemins de fichiers en utilisant la classe Directory du namespace System.IO.

Pour créer un nouveau dossier, vous pouvez utiliser la méthode CreateDirectory de la classe Directory. Cette méthode prend en paramètre le chemin du dossier à créer.

Directory.CreateDirectory("chemin/vers/le/nouveau/dossier");

Pour vérifier si un dossier existe, vous pouvez utiliser la méthode Exists de la classe Directory. Cette méthode renvoie une valeur booléenne indiquant si le dossier existe ou non.

bool folderExists = Directory.Exists("chemin/vers/le/dossier");
if (folderExists)
{
    Console.WriteLine("Le dossier existe.");
}
else
{
    Console.WriteLine("Le dossier n'existe pas.");
}

Vous pouvez également obtenir la liste des fichiers et des dossiers présents dans un dossier en utilisant la méthode GetFiles et la méthode GetDirectories de la classe Directory. Ces méthodes renvoient respectivement un tableau de chaînes de caractères contenant les chemins des fichiers et des dossiers.

string[] files = Directory.GetFiles("chemin/vers/le/dossier");
string[] folders = Directory.GetDirectories("chemin/vers/le/dossier");

Gestion des erreurs lors de la manipulation des fichiers

Lors de la manipulation des fichiers, il est important de prendre en compte les erreurs qui peuvent survenir. En C#, vous pouvez utiliser des blocs try-catch pour gérer les exceptions.

Par exemple, si vous essayez de lire le contenu d'un fichier qui n'existe pas, une exception de type FileNotFoundException sera levée. Vous pouvez capturer cette exception et afficher un message d'erreur approprié.

try
{
    string content = File.ReadAllText("chemin/vers/le/fichier.txt");
    Console.WriteLine(content);
}
catch (FileNotFoundException e)
{
    Console.WriteLine("Le fichier n'existe pas : " + e.Message);
}

Vous pouvez également utiliser d'autres types d'exceptions, tels que IOException pour gérer les erreurs liées à l'écriture ou à la lecture des fichiers.

Il est recommandé de gérer les exceptions de manière appropriée afin de fournir des messages d'erreur clairs et de prendre les mesures nécessaires pour résoudre les problèmes.
