Introduction aux bases de données

Une base de données est un système de stockage organisé et structuré qui permet de stocker, gérer et récupérer des données de manière efficace. Les bases de données sont utilisées dans de nombreux domaines, tels que les applications web, les systèmes de gestion de contenu, les systèmes de gestion de stock, etc. En utilisant une base de données, vous pouvez stocker et manipuler de grandes quantités de données de manière efficace et sécurisée.

Connexion à une base de données

Pour interagir avec une base de données dans C#, vous devez établir une connexion avec celle-ci. Cela implique de fournir les informations de connexion, telles que le nom du serveur, le nom de la base de données, le nom d'utilisateur et le mot de passe. Vous pouvez utiliser la classe SqlConnection de l'espace de noms System.Data.SqlClient pour établir une connexion à une base de données SQL Server. 

string connectionString = "Data Source=nom_du_serveur;Initial Catalog=nom_de_la_base_de_données;User ID=nom_d'utilisateur;Password=mot_de_passe";
SqlConnection connection = new SqlConnection(connectionString);
connection.Open();

Requêtes SQL en C#

Une fois que vous avez établi une connexion à une base de données, vous pouvez exécuter des requêtes SQL pour récupérer, insérer, mettre à jour ou supprimer des données. Vous pouvez utiliser la classe SqlCommand de l'espace de noms System.Data.SqlClient pour exécuter des requêtes SQL. 

string query = "SELECT * FROM table";
SqlCommand command = new SqlCommand(query, connection);
SqlDataReader reader = command.ExecuteReader();

while (reader.Read())
{
    // Traitez les données récupérées ici
}

reader.Close();

Utilisation d'ORM (Object-Relational Mapping)

L'ORM (Object-Relational Mapping) est une technique qui permet de mapper les objets d'une application aux tables d'une base de données relationnelle. Cela facilite la manipulation des données en utilisant des objets plutôt que des requêtes SQL. Entity Framework est un ORM populaire pour le développement d'applications C#. 

// Exemple d'utilisation d'Entity Framework
using (var context = new MyDbContext())
{
    var customers = context.Customers.ToList();

    foreach (var customer in customers)
    {
        // Traitez les objets Customer ici
    }
}

Manipulation de fichiers et dossiers

En plus de manipuler des données dans une base de données, vous pouvez également manipuler des fichiers et des dossiers dans vos applications C#. Vous pouvez utiliser les classes de l'espace de noms System.IO pour effectuer des opérations telles que la création, la suppression, la lecture et l'écriture de fichiers et de dossiers. 

// Exemple de lecture d'un fichier
string filePath = "chemin_du_fichier";
string content = File.ReadAllText(filePath);
Console.WriteLine(content);

Sérialisation et désérialisation de données

La sérialisation est le processus de conversion d'un objet en une représentation qui peut être stockée ou transmise, telle qu'un fichier XML ou JSON. La désérialisation est le processus inverse, qui consiste à convertir une représentation stockée ou transmise en un objet. C# offre des fonctionnalités intégrées pour la sérialisation et la désérialisation d'objets en utilisant les classes de l'espace de noms System.Xml.Serialization et System.Runtime.Serialization.Json. 

// Exemple de sérialisation d'un objet en XML
Person person = new Person { Name = "John", Age = 30 };
XmlSerializer serializer = new XmlSerializer(typeof(Person));
using (TextWriter writer = new StreamWriter("person.xml"))
{
    serializer.Serialize(writer, person);
}

// Exemple de désérialisation d'un fichier XML en objet
using (TextReader reader = new StreamReader("person.xml"))
{
    Person deserializedPerson = (Person)serializer.Deserialize(reader);
    Console.WriteLine(deserializedPerson.Name);
    Console.WriteLine(deserializedPerson.Age);
}
