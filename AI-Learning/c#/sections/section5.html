Section 5: Manipulation des fichiers et des dossiers en C#

Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes lors du développement d'applications en C#. Pour lire le contenu d'un fichier, vous pouvez utiliser la classe StreamReader. Voici un exemple de code qui lit le contenu d'un fichier texte :

using System;
using System.IO;

class Program {
    static void Main() {
        string filePath = "example.txt";

        using (StreamReader reader = new StreamReader(filePath)) {
            string line;
            while ((line = reader.ReadLine()) != null) {
                Console.WriteLine(line);
            }
        }
    }
}

Dans cet exemple, nous utilisons la classe StreamReader pour ouvrir le fichier spécifié par le chemin filePath. Ensuite, nous utilisons une boucle while pour lire chaque ligne du fichier et l'afficher à la console.

Pour écrire dans un fichier, vous pouvez utiliser la classe StreamWriter. Voici un exemple de code qui écrit du texte dans un fichier :

using System;
using System.IO;

class Program {
    static void Main() {
        string filePath = "example.txt";

        using (StreamWriter writer = new StreamWriter(filePath)) {
            writer.WriteLine("Hello, world!");
        }
    }
}

Dans cet exemple, nous utilisons la classe StreamWriter pour ouvrir le fichier spécifié par le chemin filePath. Ensuite, nous utilisons la méthode WriteLine pour écrire la chaîne de caractères "Hello, world!" dans le fichier.

Manipulation des dossiers et des chemins

La manipulation des dossiers et des chemins est également une tâche courante lors du développement d'applications en C#. La classe Directory fournit des méthodes utiles pour créer, supprimer et parcourir des dossiers.

Voici un exemple de code qui crée un nouveau dossier :

using System;
using System.IO;

class Program {
    static void Main() {
        string folderPath = "exampleFolder";

        Directory.CreateDirectory(folderPath);
    }
}

Dans cet exemple, nous utilisons la méthode CreateDirectory de la classe Directory pour créer un nouveau dossier avec le chemin spécifié par folderPath.

La classe Path fournit des méthodes pour manipuler les chemins de fichiers et de dossiers de manière portable. Voici un exemple de code qui combine deux chemins :

using System;
using System.IO;

class Program {
    static void Main() {
        string folderPath = "exampleFolder";
        string filePath = Path.Combine(folderPath, "example.txt");

        Console.WriteLine(filePath);
    }
}

Dans cet exemple, nous utilisons la méthode Combine de la classe Path pour combiner le chemin du dossier folderPath avec le nom du fichier example.txt. Le résultat est stocké dans la variable filePath et affiché à la console.
