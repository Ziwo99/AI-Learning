Section 7: Développement d'applications Windows avec Windows Forms

Introduction à Windows Forms

Windows Forms est un framework de développement d'interfaces utilisateur graphiques pour les applications Windows. Il permet de créer des applications Windows avec une interface utilisateur riche et interactive.

Les applications Windows Forms sont basées sur le concept de fenêtres et de contrôles. Une fenêtre représente une interface utilisateur et peut contenir différents contrôles tels que des boutons, des zones de texte, des listes déroulantes, etc.

Voici un exemple de code qui crée une fenêtre Windows Forms simple :

using System;
using System.Windows.Forms;

namespace WindowsFormsApp
{
    public class MainForm : Form
    {
        public MainForm()
        {
            InitializeComponent();
        }

        private void InitializeComponent()
        {
            this.Text = "Ma première application Windows Forms";
            this.Size = new Size(400, 300);
        }

        [STAThread]
        static void Main()
        {
            Application.Run(new MainForm());
        }
    }
}

Création d'interfaces utilisateur

La création d'interfaces utilisateur dans Windows Forms se fait en utilisant les différents contrôles disponibles. Ces contrôles peuvent être placés sur une fenêtre et configurés pour répondre aux besoins de l'application.

Voici un exemple de code qui crée une interface utilisateur avec un bouton et une zone de texte :

using System;
using System.Windows.Forms;

namespace WindowsFormsApp
{
    public class MainForm : Form
    {
        private Button myButton;
        private TextBox myTextBox;

        public MainForm()
        {
            InitializeComponent();
        }

        private void InitializeComponent()
        {
            this.Text = "Ma première application Windows Forms";
            this.Size = new Size(400, 300);

            myButton = new Button();
            myButton.Text = "Cliquez-moi !";
            myButton.Location = new Point(100, 100);
            myButton.Click += MyButton_Click;
            this.Controls.Add(myButton);

            myTextBox = new TextBox();
            myTextBox.Location = new Point(100, 150);
            this.Controls.Add(myTextBox);
        }

        private void MyButton_Click(object sender, EventArgs e)
        {
            myTextBox.Text = "Bonjour !";
        }

        [STAThread]
        static void Main()
        {
            Application.Run(new MainForm());
        }
    }
}

Gestion des événements

La gestion des événements dans Windows Forms permet de répondre aux actions de l'utilisateur, telles que le clic sur un bouton ou la saisie de texte dans une zone de texte. Les événements sont associés à des méthodes qui seront exécutées lorsque l'événement se produit.

Voici un exemple de code qui gère l'événement de clic sur un bouton :

using System;
using System.Windows.Forms;

namespace WindowsFormsApp
{
    public class MainForm : Form
    {
        private Button myButton;

        public MainForm()
        {
            InitializeComponent();
        }

        private void InitializeComponent()
        {
            this.Text = "Ma première application Windows Forms";
            this.Size = new Size(400, 300);

            myButton = new Button();
            myButton.Text = "Cliquez-moi !";
            myButton.Location = new Point(100, 100);
            myButton.Click += MyButton_Click;
            this.Controls.Add(myButton);
        }

        private void MyButton_Click(object sender, EventArgs e)
        {
            MessageBox.Show("Vous avez cliqué sur le bouton !");
        }

        [STAThread]
        static void Main()
        {
            Application.Run(new MainForm());
        }
    }
}

Utilisation des contrôles Windows Forms

Les contrôles Windows Forms offrent une large gamme de fonctionnalités pour créer des interfaces utilisateur interactives. Ils peuvent être utilisés pour afficher du texte, des images, des listes, des tableaux, etc.

Voici un exemple de code qui utilise différents contrôles Windows Forms :

using System;
using System.Windows.Forms;

namespace WindowsFormsApp
{
    public class MainForm : Form
    {
        private Label myLabel;
        private ListBox myListBox;
        private DataGridView myDataGridView;

        public MainForm()
        {
            InitializeComponent();
        }

        private void InitializeComponent()
        {
            this.Text = "Ma première application Windows Forms";
            this.Size = new Size(400, 300);

            myLabel = new Label();
            myLabel.Text = "Exemple d'utilisation des contrôles Windows Forms";
            myLabel.Location = new Point(100, 50);
            this.Controls.Add(myLabel);

            myListBox = new ListBox();
            myListBox.Items.Add("Élément 1");
            myListBox.Items.Add("Élément 2");
            myListBox.Items.Add("Élément 3");
            myListBox.Location = new Point(100, 100);
            this.Controls.Add(myListBox);

            myDataGridView = new DataGridView();
            myDataGridView.Columns.Add("Column1", "Colonne 1");
            myDataGridView.Columns.Add("Column2", "Colonne 2");
            myDataGridView.Rows.Add("Donnée 1", "Donnée 2");
            myDataGridView.Rows.Add("Donnée 3", "Donnée 4");
            myDataGridView.Location = new Point(100, 150);
            this.Controls.Add(myDataGridView);
        }

        [STAThread]
        static void Main()
        {
            Application.Run(new MainForm());
        }
    }
}
