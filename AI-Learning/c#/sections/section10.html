Réalisation d'un projet de gestion de contacts

Dans cette sous-section, nous allons réaliser un projet de gestion de contacts en utilisant C#. Ce projet permettra de créer, afficher, modifier et supprimer des contacts.

Voici les fonctionnalités principales du projet :

  • Création d'un nouveau contact en saisissant les informations telles que le nom, le prénom, l'adresse, le numéro de téléphone, etc.
  • Affichage de la liste des contacts existants avec la possibilité de les trier par ordre alphabétique ou par date d'ajout.
  • Modification des informations d'un contact existant.
  • Suppression d'un contact de la liste.

Voici un exemple de code pour la classe Contact :

public class Contact
{
    public string FirstName { get; set; }
    public string LastName { get; set; }
    public string Address { get; set; }
    public string PhoneNumber { get; set; }
    
    public Contact(string firstName, string lastName, string address, string phoneNumber)
    {
        FirstName = firstName;
        LastName = lastName;
        Address = address;
        PhoneNumber = phoneNumber;
    }
}

Voici un exemple de code pour la classe ContactManager :

public class ContactManager
{
    private List contacts;
    
    public ContactManager()
    {
        contacts = new List();
    }
    
    public void AddContact(Contact contact)
    {
        contacts.Add(contact);
    }
    
    public void RemoveContact(Contact contact)
    {
        contacts.Remove(contact);
    }
    
    public List GetContacts()
    {
        return contacts;
    }
}

Vous pouvez utiliser ces classes comme point de départ pour implémenter le projet de gestion de contacts. N'hésitez pas à ajouter d'autres fonctionnalités ou à personnaliser le projet selon vos besoins.

Réalisation d'un projet de gestion de stock

Dans cette sous-section, nous allons réaliser un projet de gestion de stock en utilisant C#. Ce projet permettra de gérer les produits d'un magasin, y compris leur ajout, leur modification, leur suppression et leur affichage.

Voici les fonctionnalités principales du projet :

  • Ajout d'un nouveau produit en saisissant les informations telles que le nom, la quantité, le prix, etc.
  • Affichage de la liste des produits existants avec la possibilité de les trier par nom, par quantité ou par prix.
  • Modification des informations d'un produit existant.
  • Suppression d'un produit de la liste.

Voici un exemple de code pour la classe Product :

public class Product
{
    public string Name { get; set; }
    public int Quantity { get; set; }
    public decimal Price { get; set; }
    
    public Product(string name, int quantity, decimal price)
    {
        Name = name;
        Quantity = quantity;
        Price = price;

}

Réalisation d'un projet de gestion de tâches

Dans cette sous-section, nous allons réaliser un projet de gestion de tâches en utilisant C#. Ce projet permettra de créer, afficher, modifier et supprimer des tâches.

Voici les fonctionnalités principales du projet :

  • Création d'une nouvelle tâche en saisissant les informations telles que le titre, la description, la date d'échéance, etc.
  • Affichage de la liste des tâches existantes avec la possibilité de les trier par date d'échéance ou par statut (en cours, terminée, en attente).
  • Modification des informations d'une tâche existante.
  • Suppression d'une tâche de la liste.

Voici un exemple de code pour la classe Task :

public class Task
{
    public string Title { get; set; }
    public string Description { get; set; }
    public DateTime DueDate { get; set; }
    public TaskStatus Status { get; set; }
    
    public Task(string title, string description, DateTime dueDate)
    {
        Title = title;
        Description = description;
        DueDate = dueDate;
        Status = TaskStatus.InProgress;
    }
}

public enum TaskStatus
{
    InProgress,
    Completed,
    Pending
}

Voici un exemple de code pour la classe TaskManager :

public class TaskManager
{
    private List tasks;
    
    public TaskManager()
    {
        tasks = new List();
    }
    
    public void AddTask(Task task)
    {
        tasks.Add(task);
    }
    
    public void RemoveTask(Task task)
    {
        tasks.Remove(task);
    }
    
    public List GetTasks()
    {
        return tasks;
    }
}

Vous pouvez utiliser ces classes comme point de départ pour implémenter le projet de gestion de tâches. N'hésitez pas à ajouter d'autres fonctionnalités ou à personnaliser le projet selon vos besoins.
