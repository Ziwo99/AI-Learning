Création d'une application de gestion de contacts

Dans cette sous-section, nous allons créer une application de gestion de contacts en utilisant C#. Cette application permettra aux utilisateurs de stocker et de gérer leurs contacts, y compris leurs noms, numéros de téléphone et adresses e-mail.

Voici un exemple de code pour créer une classe Contact :

public class Contact
{
    public string Name { get; set; }
    public string PhoneNumber { get; set; }
    public string Email { get; set; }
}

Ensuite, nous allons créer une classe principale pour notre application de gestion de contacts :

public class ContactManager
{
    private List<Contact> contacts;

    public ContactManager()
    {
        contacts = new List<Contact>();
    }

    public void AddContact(Contact contact)
    {
        contacts.Add(contact);
    }

    public void RemoveContact(Contact contact)
    {
        contacts.Remove(contact);
    }

    public void PrintContacts()
    {
        foreach (Contact contact in contacts)
        {
            Console.WriteLine("Name: " + contact.Name);
            Console.WriteLine("Phone Number: " + contact.PhoneNumber);
            Console.WriteLine("Email: " + contact.Email);
            Console.WriteLine("--------------------");
        }
    }
}

public class Program
{
    public static void Main(string[] args)
    {
        ContactManager contactManager = new ContactManager();

        Contact contact1 = new Contact();
        contact1.Name = "John Doe";
        contact1.PhoneNumber = "1234567890";
        contact1.Email = "john.doe@example.com";
        contactManager.AddContact(contact1);

        Contact contact2 = new Contact();
        contact2.Name = "Jane Smith";
        contact2.PhoneNumber = "0987654321";
        contact2.Email = "jane.smith@example.com";
        contactManager.AddContact(contact2);

        contactManager.PrintContacts();
    }
}

Ce code crée une instance de la classe ContactManager, ajoute deux contacts à la liste de contacts et imprime les détails de chaque contact.

Développement d'un jeu en C#

Dans cette sous-section, nous allons développer un jeu simple en utilisant C#. Le jeu consistera à deviner un nombre aléatoire généré par l'ordinateur.

Voici un exemple de code pour développer ce jeu :

public class GuessingGame
{
    private int randomNumber;
    private bool isGameOver;

    public GuessingGame()
    {
        Random random = new Random();
        randomNumber = random.Next(1, 101);
        isGameOver = false;
    }

    public void Play()
    {
        while (!isGameOver)
        {
            Console.Write("Enter your guess: ");
            int guess = Convert.ToInt32(Console.ReadLine());

            if (guess == randomNumber)
            {
                Console.WriteLine("Congratulations! You guessed the correct number.");
                isGameOver = true;
            }
            else if (guess < randomNumber)
            {
                Console.WriteLine("Too low! Try again.");
            }
            else
            {
                Console.WriteLine("Too high! Try again.");
            }
        }
    }
}

public class Program
{
    public static void Main(string[] args)
    {
        GuessingGame game = new GuessingGame();
        game.Play();
    }
}

Ce code crée une instance de la classe GuessingGame, génère un nombre aléatoire entre 1 et 100, puis demande à l'utilisateur de deviner le nombre. Le jeu continue jusqu'à ce que l'utilisateur devine le bon nombre.

Création d'un site web dynamique avec C#

Dans cette sous-section, nous allons créer un site web dynamique en utilisant C#. Ce site web permettra aux utilisateurs de s'inscrire, de se connecter et de gérer leurs informations personnelles.

Voici un exemple de code pour créer une page d'inscription :

<%@ Page Language="C#" %>
<!DOCTYPE html>
<html>
<head>
    <title>Registration Page</title>
</head>
<body>
    <h1>Registration Page</h1>
    <form runat="server">
        <label for="username">Username:</label>
        <input type="text" id="username" runat="server" />
        <br />
        <label for="password">Password:</label>
        <input type="password" id="password" runat="server" />
        <br />
        <label for="email">Email:</label>
        <input type="email" id="email" runat="server" />
        <br />
        <asp:Button ID="registerButton" runat="server" Text="Register" OnClick="RegisterButton_Click" />
    </form>
</body>
</html>

Ce code crée une page d'inscription avec des champs pour le nom d'utilisateur, le mot de passe et l'e-mail. Lorsque l'utilisateur clique sur le bouton "Register", la méthode RegisterButton_Click est appelée pour traiter les données d'inscription.

Ce sont quelques exemples de projets pratiques que vous pouvez réaliser en utilisant C#. N'hésitez pas à explorer davantage et à créer vos propres projets pour renforcer vos compétences en programmation C#.
