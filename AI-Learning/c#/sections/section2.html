Structure d'un programme C#

Un programme C# est composé de plusieurs éléments essentiels :

  • Un point d'entrée, qui est la méthode Main.
  • Des instructions, qui sont des lignes de code qui indiquent à l'ordinateur quoi faire.
  • Des commentaires, qui sont des annotations pour expliquer le code aux développeurs.

Voici un exemple de structure de base d'un programme C# :

using System;

namespace MonProgramme
{
    class Program
    {
        static void Main(string[] args)
        {
            // Instructions du programme
        }
    }
}

Variables et types de données

En C#, les variables sont utilisées pour stocker des données. Chaque variable a un type de données qui détermine le type de valeurs qu'elle peut contenir. Voici quelques types de données couramment utilisés en C# :

  • int : pour stocker des nombres entiers.
  • double : pour stocker des nombres à virgule flottante.
  • string : pour stocker du texte.
  • bool : pour stocker des valeurs booléennes (vrai ou faux).

Voici un exemple de déclaration de variables :

int age = 25;
double prix = 19.99;
string nom = "John Doe";
bool estActif = true;

Opérations arithmétiques et opérateurs logiques

En C#, vous pouvez effectuer des opérations arithmétiques telles que l'addition, la soustraction, la multiplication et la division sur des variables numériques. Voici quelques exemples :

int a = 10;
int b = 5;

int somme = a + b; // somme vaut 15
int difference = a - b; // difference vaut 5
int produit = a * b; // produit vaut 50
int quotient = a / b; // quotient vaut 2

Les opérateurs logiques, tels que && (et), || (ou) et ! (non), sont utilisés pour évaluer des expressions booléennes. Voici un exemple :

bool estVrai = true;
bool estFaux = false;

bool resultatEt = estVrai && estFaux; // resultatEt vaut false
bool resultatOu = estVrai || estFaux; // resultatOu vaut true
bool resultatNon = !estVrai; // resultatNon vaut false

Les boucles et les conditions

Les boucles permettent d'exécuter un bloc de code plusieurs fois. En C#, nous avons plusieurs types de boucles :

  • while : exécute un bloc de code tant qu'une condition est vraie.
  • do-while : exécute un bloc de code au moins une fois, puis tant qu'une condition est vraie.
  • for : exécute un bloc de code un nombre spécifié de fois.
  • foreach : itère sur les éléments d'une collection.

Voici un exemple d'utilisation de la boucle while :

int i = 0;

while (i < 10)
{
    Console.WriteLine(i);
    i++;
}

Les conditions permettent d'exécuter un bloc de code uniquement si une condition est vraie. En C#, nous avons plusieurs types de conditions :

  • if : exécute un bloc de code si une condition est vraie.
  • else : exécute un bloc de code si la condition de l'if est fausse.
  • else if : exécute un bloc de code si une autre condition est vraie.
  • switch : exécute un bloc de code en fonction de la valeur d'une variable.

Voici un exemple d'utilisation de la condition if :

int age = 18;

if (age >= 18)
{
    Console.WriteLine("Vous êtes majeur.");
}
else
{
    Console.WriteLine("Vous êtes mineur.");
}

Les tableaux et les collections

Un tableau est une structure de données qui permet de stocker plusieurs valeurs du même type. En C#, nous pouvons déclarer un tableau de la manière suivante :

int[] nombres = new int[5];

Les collections sont des structures de données plus flexibles que les tableaux, car elles peuvent contenir des éléments de différents types. En C#, nous avons plusieurs types de collections :

  • List<T> : une liste générique qui peut contenir des éléments de n'importe quel type.
  • Dictionary<TKey, TValue> : un dictionnaire générique qui associe des clés à des valeurs.
  • HashSet<T> : un ensemble générique qui ne contient pas de doublons.

Voici un exemple d'utilisation d'une liste :

List<string> fruits = new List<string>();
fruits.Add("Pomme");
fruits.Add("Banane");
fruits.Add("Orange");

foreach (string fruit in fruits)
{
    Console.WriteLine(fruit);
}

Ceci conclut la section 2 sur les bases de C#. Nous avons couvert la structure d'un programme C#, les variables et les types de données, les opérations arithmétiques et les opérateurs logiques, les boucles et les conditions, ainsi que les tableaux et les collections.
