Structure d'un programme C#

Un programme C# est composé de plusieurs éléments, notamment des classes, des méthodes et des instructions. Voici une structure de base d'un programme C# :

using System;

namespace MonProgramme
{
    class Program
    {
        static void Main(string[] args)
        {
            // Instructions du programme
        }
    }
}

La déclaration using System; permet d'inclure l'espace de noms System, qui contient des classes et des méthodes prédéfinies.

La déclaration namespace MonProgramme définit un espace de noms pour regrouper les classes et les méthodes.

La classe Program est la classe principale du programme.

La méthode Main est la méthode d'entrée du programme, c'est là que l'exécution commence.

Variables et types de données

En C#, les variables sont utilisées pour stocker des données. Voici les principaux types de données disponibles en C# :

  • int : pour stocker des nombres entiers.
  • double : pour stocker des nombres à virgule flottante.
  • bool : pour stocker des valeurs booléennes (vrai ou faux).
  • string : pour stocker des chaînes de caractères.

Voici comment déclarer et initialiser une variable en C# :

int age = 25;
double prix = 19.99;
bool estActif = true;
string nom = "John Doe";

Opérateurs et expressions

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques-uns des opérateurs les plus couramment utilisés en C# :

  • + : pour l'addition.
  • - : pour la soustraction.
  • * : pour la multiplication.
  • / : pour la division.
  • % : pour le modulo (reste de la division).

Voici quelques exemples d'expressions en utilisant ces opérateurs :

int a = 10;
int b = 5;

int somme = a + b; // somme vaut 15
int difference = a - b; // difference vaut 5
int produit = a * b; // produit vaut 50
int quotient = a / b; // quotient vaut 2
int reste = a % b; // reste vaut 0

Structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. Voici quelques-unes des structures de contrôle les plus couramment utilisées en C# :

Les boucles :

La boucle while permet d'exécuter une série d'instructions tant qu'une condition est vraie :

int i = 0;

while (i < 10)
{
    Console.WriteLine(i);
    i++;
}

La boucle for permet d'exécuter une série d'instructions un certain nombre de fois :

for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}

Les conditions :

La structure if permet d'exécuter des instructions conditionnellement :

int age = 18;

if (age >= 18)
{
    Console.WriteLine("Vous êtes majeur.");
}
else
{
    Console.WriteLine("Vous êtes mineur.");
}

La structure switch permet d'exécuter différentes instructions en fonction de la valeur d'une variable :

int jour = 1;

switch (jour)
{
    case 1:
        Console.WriteLine("Lundi");
        break;
    case 2:
        Console.WriteLine("Mardi");
        break;
    // ...
    default:
        Console.WriteLine("Jour inconnu");
        break;
}

Fonctions et méthodes

Les fonctions et les méthodes permettent de regrouper des instructions et de les réutiliser. Voici comment déclarer et appeler une fonction en C# :

int Somme(int a, int b)
{
    return a + b;
}

int resultat = Somme(5, 3); // resultat vaut 8

Les méthodes sont similaires aux fonctions, mais elles sont définies à l'intérieur d'une classe :

class Calculatrice
{
    public int Somme(int a, int b)
    {
        return a + b;
    }
}

Calculatrice calc = new Calculatrice();
int resultat = calc.Somme(5, 3); // resultat vaut 8

Tableaux et collections

Les tableaux et les collections permettent de stocker plusieurs valeurs dans une seule variable. Voici comment déclarer et utiliser un tableau en C# :

int[] nombres = new int[5];

nombres[0] = 1;
nombres[1] = 2;
nombres[2] = 3;
nombres[3] = 4;
nombres[4] = 5;

Console.WriteLine(nombres[2]); // Affiche 3

Les collections, telles que les listes et les dictionnaires, offrent des fonctionnalités supplémentaires par rapport aux tableaux :

List<string> fruits = new List<string>();

fruits.Add("Pomme");
fruits.Add("Banane");
fruits.Add("Orange");

Console.WriteLine(fruits[1]); // Affiche "Banane"

Les tableaux et les collections peuvent être parcourus à l'aide de boucles :

foreach (int nombre in nombres)
{
    Console.WriteLine(nombre);
}

foreach (string fruit in fruits)
{
    Console.WriteLine(fruit);
}
