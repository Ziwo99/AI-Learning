Section 2: Les bases de la programmation en C#

Variables et types de données

Les variables sont des emplacements de mémoire utilisés pour stocker des valeurs. En C#, vous devez déclarer le type de données d'une variable avant de l'utiliser. Voici quelques types de données couramment utilisés en C# :

int myInt = 10;
double myDouble = 3.14;
char myChar = 'A';
string myString = "Hello World";
bool myBool = true;

Opérateurs et expressions

Les opérateurs sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques opérateurs couramment utilisés en C# :

// Opérateurs arithmétiques
int sum = 10 + 5;
int difference = 10 - 5;
int product = 10 * 5;
double quotient = 10 / 5;

// Opérateurs de comparaison
bool isEqual = (10 == 5);
bool isNotEqual = (10 != 5);
bool isGreater = (10 > 5);
bool isLess = (10 < 5);

// Opérateurs logiques
bool logicalAnd = (true && false);
bool logicalOr = (true || false);
bool logicalNot = !true;

Structures de contrôle (boucles et conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme. Voici quelques structures de contrôle couramment utilisées en C# :

// Boucle while
int i = 0;
while (i < 10) {
    Console.WriteLine(i);
    i++;
}

// Boucle for
for (int j = 0; j < 10; j++) {
    Console.WriteLine(j);
}

// Condition if-else
int age = 18;
if (age >= 18) {
    Console.WriteLine("Vous êtes majeur.");
} else {
    Console.WriteLine("Vous êtes mineur.");
}

Fonctions et méthodes

Les fonctions et les méthodes sont des blocs de code réutilisables. Voici comment définir et utiliser une fonction en C# :

// Définition d'une fonction
int Add(int a, int b) {
    return a + b;
}

// Utilisation de la fonction
int result = Add(5, 3);
Console.WriteLine(result);

Tableaux et collections

Les tableaux et les collections permettent de stocker plusieurs valeurs dans une seule variable. Voici comment déclarer et utiliser un tableau en C# :

// Déclaration d'un tableau
int[] numbers = new int[5];

// Initialisation des valeurs du tableau
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

// Accès aux valeurs du tableau
Console.WriteLine(numbers[0]); // Affiche 1
Console.WriteLine(numbers[2]); // Affiche 3

Ce sont les bases de la programmation en C#. En comprenant ces concepts, vous serez en mesure de créer des programmes simples en C#.
