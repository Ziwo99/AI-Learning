Section 9: Développement d'applications web avec ASP.NET

Retour au cours

Introduction à ASP.NET

ASP.NET est un framework de développement web créé par Microsoft. Il permet de construire des applications web robustes et évolutives en utilisant le langage de programmation C#. ASP.NET offre une grande flexibilité et de nombreuses fonctionnalités pour le développement d'applications web.

Création de pages web avec ASP.NET

La création de pages web avec ASP.NET se fait en utilisant le modèle de programmation basé sur les pages. Dans ce modèle, chaque page web est représentée par un fichier avec l'extension .aspx. Ce fichier contient du code HTML et du code C# qui est exécuté côté serveur.

Voici un exemple de code ASP.NET pour une page web simple :

<%@ Page Language="C#" %>
<!DOCTYPE html>
<html>
<head>
    <title>Ma première page ASP.NET</title>
</head>
<body>
    <h1>Bienvenue sur ma page ASP.NET</h1>
    <p>Cette page a été créée en utilisant ASP.NET.</p>
</body>
</html>

Utilisation des contrôles web

ASP.NET offre une large gamme de contrôles web qui permettent d'ajouter des fonctionnalités interactives aux pages web. Ces contrôles peuvent être utilisés pour afficher du texte, des images, des formulaires, des boutons, des listes déroulantes, etc.

Voici un exemple de code ASP.NET utilisant un contrôle de bouton :

<%@ Page Language="C#" %>
<!DOCTYPE html>
<html>
<head>
    <title>Utilisation des contrôles web</title>
</head>
<body>
    <h1>Utilisation des contrôles web</h1>
    <asp:Button ID="btnSubmit" runat="server" Text="Cliquez ici" OnClick="btnSubmit_Click" />
    <asp:Label ID="lblMessage" runat="server" />
</body>
</html>

Dans cet exemple, un bouton est affiché sur la page et lorsqu'il est cliqué, le code C# associé à l'événement OnClick est exécuté.

Gestion des sessions et des cookies

ASP.NET permet de gérer les sessions et les cookies pour stocker des informations spécifiques à un utilisateur. Les sessions permettent de stocker des données côté serveur, tandis que les cookies sont utilisés pour stocker des données côté client.

Voici un exemple de code ASP.NET pour stocker une valeur dans une session :

<%@ Page Language="C#" %>
<!DOCTYPE html>
<html>
<head>
    <title>Gestion des sessions et des cookies</title>
</head>
<body>
    <h1>Gestion des sessions et des cookies</h1>
    <asp:Button ID="btnSetSession" runat="server" Text="Définir la session" OnClick="btnSetSession_Click" />
    <asp:Label ID="lblSessionValue" runat="server" />
</body>
</html>

Dans cet exemple, lorsque le bouton est cliqué, le code C# associé à l'événement OnClick définit une valeur dans la session. Cette valeur peut ensuite être récupérée et affichée sur la page.
