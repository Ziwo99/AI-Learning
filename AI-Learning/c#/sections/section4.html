Gestion des fichiers et des dossiers

La gestion des fichiers et des dossiers est une fonctionnalité essentielle dans de nombreuses applications C#. Elle permet de créer, lire, écrire et supprimer des fichiers, ainsi que de manipuler des dossiers.

Voici un exemple de code qui montre comment créer un fichier texte et y écrire du contenu :

string filePath = "C:\\example.txt";
string content = "Ceci est un exemple de contenu.";

File.WriteAllText(filePath, content);

Dans cet exemple, nous utilisons la classe File du namespace System.IO pour écrire le contenu dans le fichier spécifié par le chemin filePath.

Il est également possible de lire le contenu d'un fichier :

string fileContent = File.ReadAllText(filePath);
Console.WriteLine(fileContent);

Ce code lit le contenu du fichier spécifié par le chemin filePath et l'affiche dans la console.

Pour supprimer un fichier, vous pouvez utiliser la méthode File.Delete() :

File.Delete(filePath);

Ce code supprime le fichier spécifié par le chemin filePath.

Pour manipuler des dossiers, vous pouvez utiliser la classe Directory du namespace System.IO. Par exemple, pour créer un nouveau dossier :

string folderPath = "C:\\exampleFolder";

Directory.CreateDirectory(folderPath);

Ce code crée un nouveau dossier avec le chemin spécifié par folderPath.

Il est également possible de lister tous les fichiers et dossiers dans un dossier :

string[] files = Directory.GetFiles(folderPath);
string[] directories = Directory.GetDirectories(folderPath);

foreach (string file in files)
{
    Console.WriteLine(file);
}

foreach (string directory in directories)
{
    Console.WriteLine(directory);
}

Ce code récupère tous les fichiers et dossiers dans le dossier spécifié par folderPath et les affiche dans la console.

Manipulation des chaînes de caractères

La manipulation des chaînes de caractères est une tâche courante dans le développement d'applications C#. C# offre de nombreuses fonctionnalités pour manipuler et formater les chaînes de caractères.

Voici un exemple de code qui montre comment concaténer des chaînes de caractères :

string firstName = "John";
string lastName = "Doe";

string fullName = firstName + " " + lastName;
Console.WriteLine(fullName);

Ce code concatène les variables firstName et lastName pour former la chaîne de caractères fullName.

Il est également possible de formater les chaînes de caractères en utilisant la méthode string.Format() :

string formattedString = string.Format("Bonjour, je m'appelle {0} {1}.", firstName, lastName);
Console.WriteLine(formattedString);

Ce code utilise la méthode string.Format() pour insérer les valeurs des variables firstName et lastName dans la chaîne de format spécifiée.

Il existe également d'autres méthodes utiles pour manipuler les chaînes de caractères, telles que string.Length pour obtenir la longueur d'une chaîne, string.ToUpper() pour convertir une chaîne en majuscules, et string.ToLower() pour convertir une chaîne en minuscules.

Gestion des événements

La gestion des événements est une fonctionnalité clé de C# qui permet de réagir aux actions de l'utilisateur ou à d'autres événements dans une application.

Voici un exemple de code qui montre comment définir un gestionnaire d'événements pour un bouton :

Button button = new Button();
button.Click += Button_Click;

private void Button_Click(object sender, EventArgs e)
{
    Console.WriteLine("Le bouton a été cliqué !");
}

Ce code crée un nouvel objet Button et définit un gestionnaire d'événements pour l'événement Click. Lorsque le bouton est cliqué, la méthode Button_Click est appelée et affiche un message dans la console.

Il est également possible de définir des gestionnaires d'événements pour d'autres types d'événements, tels que MouseClick, KeyPress, etc.

Programmation asynchrone

La programmation asynchrone est une fonctionnalité avancée de C# qui permet d'exécuter des tâches de manière asynchrone, sans bloquer le thread principal de l'application.

Voici un exemple de code qui montre comment exécuter une tâche asynchrone :

private async Task DoSomethingAsync()
{
    await Task.Delay(1000);
    Console.WriteLine("La tâche asynchrone est terminée !");
}

private async void Button_Click(object sender, EventArgs e)
{
    await DoSomethingAsync();
}

Ce code définit une méthode asynchrone DoSomethingAsync qui attend une seconde en utilisant la méthode Task.Delay() et affiche un message dans la console. La méthode Button_Click est également marquée comme asynchrone et appelle la méthode DoSomethingAsync en utilisant l'opérateur await.

La programmation asynchrone est particulièrement utile pour les opérations longues, telles que les appels réseau ou les accès à la base de données, car elle permet à l'application de rester réactive pendant l'exécution de ces tâches.

LINQ (Language Integrated Query)

Language Integrated Query (LINQ) est une fonctionnalité puissante de C# qui permet d'effectuer des requêtes sur des collections de données, telles que des tableaux ou des listes, de manière simple et expressive.

Voici un exemple de code qui montre comment utiliser LINQ pour filtrer une liste d'objets :

List<Person> people = GetPeople();

var adults = from person in people
             where person.Age >= 18
             select person;

foreach (Person adult in adults)
{
    Console.WriteLine(adult.Name);
}

Ce code utilise la syntaxe de requête de LINQ pour filtrer les personnes de la liste people dont l'âge est supérieur ou égal à 18. Les personnes adultes sont ensuite affichées dans la console.

Il est également possible d'utiliser des opérations de tri, de regroupement et d'autres opérations de manipulation des données avec LINQ.

Gestion des erreurs et débogage

La gestion des erreurs et le débogage sont des aspects importants du développement d'applications C#. C# offre plusieurs fonctionnalités pour gérer les erreurs et faciliter le processus de débogage.

Voici un exemple de code qui montre comment gérer une exception :

try
{
    int result = 10 / 0;
}
catch (DivideByZeroException ex)
{
    Console.WriteLine("Une erreur s'est produite : " + ex.Message);
}

Ce code tente de diviser le nombre 10 par zéro, ce qui provoque une exception DivideByZeroException. La clause catch capture l'exception et affiche un message d'erreur dans la console.

Il est également possible d'utiliser des points d'arrêt dans un environnement de développement intégré (IDE) tel que Visual Studio pour déboguer une application pas à pas et inspecter les valeurs des variables à différents points d'exécution.

En utilisant des outils de débogage, il est possible de détecter et de corriger les erreurs plus rapidement, ce qui facilite le processus de développement.

Cette section a couvert les fonctionnalités avancées de C#, telles que la gestion des fichiers et des dossiers, la manipulation des chaînes de caractères, la gestion des événements, la programmation asynchrone, LINQ, la gestion des erreurs et le débogage. Ces fonctionnalités sont essentielles pour développer des applications C# robustes et performantes.
