Section 6 : Utilisation des frameworks et bibliothèques Scala

Section 6 : Utilisation des frameworks et bibliothèques Scala

Utilisation de sbt (Simple Build Tool)

sbt est un outil de construction (build tool) populaire utilisé pour gérer les projets Scala. Il facilite la compilation, le test et le déploiement de votre code Scala. Voici comment utiliser sbt pour créer et gérer un projet Scala : 


// Créer un nouveau projet Scala avec sbt
sbt new scala/scala-seed.g8

// Compiler le projet
sbt compile

// Exécuter les tests
sbt test

// Générer un exécutable JAR
sbt assembly

Utilisation de Play Framework

Play Framework est un framework web léger et réactif qui facilite le développement d'applications web en Scala. Voici comment utiliser Play Framework pour créer une application web simple : 


// Créer un nouveau projet Play
sbt new playframework/play-scala-seed.g8

// Démarrer le serveur de développement
sbt run

// Créer un contrôleur
// app/controllers/HomeController.scala
class HomeController @Inject()(cc: ControllerComponents) extends AbstractController(cc) {
    def index() = Action { implicit request: Request[AnyContent] =>
        Ok("Hello, World!")
    }
}

// Définir une route
// conf/routes
GET     /       controllers.HomeController.index

// Accéder à l'application dans le navigateur
http://localhost:9000/

Utilisation de Akka

Akka est une bibliothèque puissante pour la programmation concurrente et distribuée en Scala. Voici comment utiliser Akka pour créer un système d'acteurs simple : 


// Créer un acteur
class MyActor extends Actor {
    def receive = {
        case "Hello" => println("Hello, World!")
        case _ => println("Unknown message")
    }
}

// Créer un système d'acteurs
val system = ActorSystem("MySystem")

// Créer une instance de l'acteur
val myActor = system.actorOf(Props[MyActor], "myActor")

// Envoyer un message à l'acteur
myActor ! "Hello"

Utilisation de Spark

Spark est un framework de traitement des données rapide et distribué qui prend en charge le traitement de gros volumes de données en Scala. Voici comment utiliser Spark pour effectuer un simple traitement de données : 


// Créer un contexte Spark
val conf = new SparkConf().setAppName("MyApp").setMaster("local")
val sc = new SparkContext(conf)

// Charger les données à partir d'un fichier
val data = sc.textFile("data.txt")

// Effectuer un traitement sur les données
val result = data.flatMap(line => line.split(" ")).map(word => (word, 1)).reduceByKey(_ + _)

// Afficher le résultat
result.foreach(println)
