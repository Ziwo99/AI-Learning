Création d'une application de gestion de tâches

Dans cette sous-section, nous allons créer une application de gestion de tâches en utilisant Scala. Cette application permettra aux utilisateurs de créer, modifier et supprimer des tâches, ainsi que de les marquer comme terminées.

Voici un exemple de code pour créer une classe Task :

class Task(val id: Int, val description: String, var isCompleted: Boolean) {
  def complete(): Unit = {
    isCompleted = true
  }
}

val task1 = new Task(1, "Faire les courses", false)
val task2 = new Task(2, "Appeler le médecin", false)

Dans cet exemple, nous avons créé une classe Task avec trois propriétés : id, description et isCompleted. La méthode complete() permet de marquer une tâche comme terminée en modifiant la valeur de isCompleted à true. Nous avons ensuite créé deux instances de la classe Task.

En utilisant cette classe Task, nous pouvons créer une liste de tâches :

val taskList = List(task1, task2)

Nous pouvons ensuite afficher la liste des tâches non terminées :

val incompleteTasks = taskList.filter(!_.isCompleted)
incompleteTasks.foreach(task => println(task.description))

Ce code utilise la méthode filter() pour filtrer les tâches non terminées, puis la méthode foreach() pour afficher la description de chaque tâche.

Développement d'un système de recommandation

Dans cette sous-section, nous allons développer un système de recommandation en utilisant Scala. Ce système permettra de recommander des produits aux utilisateurs en fonction de leurs préférences.

Voici un exemple de code pour créer une classe User :

class User(val id: Int, val name: String) {
  var preferences: Map[String, Int] = Map()

  def addPreference(item: String, rating: Int): Unit = {
    preferences += (item -> rating)
  }
}

val user1 = new User(1, "Alice")
val user2 = new User(2, "Bob")

Dans cet exemple, nous avons créé une classe User avec deux propriétés : id et name. La propriété preferences est une carte qui stocke les préférences de l'utilisateur pour différents articles, avec les notes correspondantes. La méthode addPreference() permet d'ajouter une préférence à la carte.

En utilisant cette classe User, nous pouvons ajouter des préférences pour chaque utilisateur :

user1.addPreference("Livre", 5)
user1.addPreference("Film", 4)
user2.addPreference("Livre", 3)
user2.addPreference("Film", 5)

Nous pouvons ensuite recommander des articles à un utilisateur en fonction de ses préférences :

def recommend(user: User): List[String] = {
  val maxRating = user.preferences.values.max
  val topItems = user.preferences.filter(_._2 == maxRating).keys.toList
  topItems
}

val recommendedItems = recommend(user1)
recommendedItems.foreach(item => println(item))

Ce code utilise la méthode recommend() pour recommander les articles ayant la note maximale pour un utilisateur donné. Les articles recommandés sont ensuite affichés à l'aide de la méthode foreach().

Mise en place d'un service web avec Play Framework

Dans cette sous-section, nous allons mettre en place un service web en utilisant le framework Play. Ce service web permettra aux utilisateurs d'interagir avec notre application de gestion de tâches via des requêtes HTTP.

Voici un exemple de code pour créer une route dans Play :

GET     /tasks                  controllers.TaskController.getTasks
POST    /tasks                  controllers.TaskController.createTask
PUT     /tasks/:id              controllers.TaskController.updateTask(id: Int)
DELETE  /tasks/:id              controllers.TaskController.deleteTask(id: Int)

Dans cet exemple, nous avons défini quatre routes pour notre application de gestion de tâches.
