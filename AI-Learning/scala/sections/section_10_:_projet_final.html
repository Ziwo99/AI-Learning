Section 10 : Projet final

Réalisation d'un projet complet en Scala

Dans cette sous-section, nous allons mettre en pratique tout ce que nous avons appris jusqu'à présent en réalisant un projet complet en Scala. Ce projet consistera en la création d'une application de gestion de tâches.

Pour commencer, nous allons créer une classe Task qui représentera une tâche. Cette classe aura les attributs suivants :

class Task(val id: Int, val description: String, val priority: Int) {
  // Méthodes et fonctionnalités de la classe Task
}

Nous allons ensuite créer une classe TaskManager qui sera responsable de la gestion des tâches. Cette classe aura les fonctionnalités suivantes :

class TaskManager {
  private var tasks: List[Task] = List()

  def addTask(task: Task): Unit = {
    tasks = tasks :+ task
  }

  def removeTask(task: Task): Unit = {
    tasks = tasks.filterNot(_ == task)
  }

  def getTasks: List[Task] = {
    tasks
  }

  // Autres fonctionnalités de la classe TaskManager
}

Une fois que nous avons défini ces classes de base, nous pouvons commencer à implémenter les fonctionnalités de notre application de gestion de tâches. Par exemple, nous pouvons ajouter une fonctionnalité permettant d'afficher toutes les tâches :

def displayTasks(taskManager: TaskManager): Unit = {
  val tasks = taskManager.getTasks
  tasks.foreach(task => println(s"ID: ${task.id}, Description: ${task.description}, Priority: ${task.priority}"))
}

Nous pouvons également ajouter une fonctionnalité permettant de trier les tâches par priorité :

def sortTasksByPriority(taskManager: TaskManager): Unit = {
  val sortedTasks = taskManager.getTasks.sortBy(_.priority)
  sortedTasks.foreach(task => println(s"ID: ${task.id}, Description: ${task.description}, Priority: ${task.priority}"))
}

En utilisant ces exemples, vous pouvez continuer à développer votre application de gestion de tâches en ajoutant d'autres fonctionnalités telles que la modification d'une tâche, la recherche de tâches par mot-clé, etc.

Une fois que vous avez terminé de développer votre application, vous pouvez la tester en créant une instance de la classe TaskManager et en ajoutant quelques tâches :

val taskManager = new TaskManager()

val task1 = new Task(1, "Faire les courses", 2)
val task2 = new Task(2, "Appeler le client", 1)
val task3 = new Task(3, "Préparer la réunion", 3)

taskManager.addTask(task1)
taskManager.addTask(task2)
taskManager.addTask(task3)

displayTasks(taskManager)

Ce projet final vous permettra de mettre en pratique toutes les connaissances que vous avez acquises sur Scala et de créer une application fonctionnelle de gestion de tâches. N'hésitez pas à explorer d'autres fonctionnalités et à personnaliser votre application selon vos besoins.
