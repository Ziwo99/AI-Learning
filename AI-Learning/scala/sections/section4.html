Fonctions de haut niveau

Les fonctions de haut niveau en Scala sont des fonctions qui peuvent être assignées à des variables, passées en tant que paramètres à d'autres fonctions et retournées en tant que résultats d'autres fonctions. Elles sont considérées comme des citoyens de première classe dans le langage Scala.

Voici un exemple de déclaration d'une fonction de haut niveau en Scala :

val addition: (Int, Int) => Int = (a, b) => a + b

Dans cet exemple, la fonction "addition" prend deux entiers en paramètres et retourne leur somme.

Fonctions anonymes

Les fonctions anonymes en Scala sont des fonctions qui n'ont pas de nom et peuvent être définies directement là où elles sont utilisées. Elles sont souvent utilisées comme arguments de fonctions de haut niveau ou dans des opérations de transformation de collections.

Voici un exemple de déclaration d'une fonction anonyme en Scala :

val carre: Int => Int = (x) => x * x

Dans cet exemple, la fonction anonyme "carre" prend un entier en paramètre et retourne son carré.

Fonctions récursives

Les fonctions récursives en Scala sont des fonctions qui s'appellent elles-mêmes pendant leur exécution. Elles sont souvent utilisées pour résoudre des problèmes qui peuvent être décomposés en sous-problèmes plus petits.

Voici un exemple de déclaration d'une fonction récursive en Scala :

def factorielle(n: Int): Int = {
  if (n == 0) 1
  else n * factorielle(n - 1)
}

Dans cet exemple, la fonction récursive "factorielle" calcule la factorielle d'un nombre en multipliant ce nombre par la factorielle du nombre précédent, jusqu'à atteindre 0.

Immutabilité et immutabilité des données

En Scala, les données peuvent être déclarées comme immuables ou mutables. Les données immuables ne peuvent pas être modifiées une fois qu'elles ont été assignées, tandis que les données mutables peuvent être modifiées.

Voici un exemple de déclaration d'une variable immuable en Scala :

val x: Int = 5

Dans cet exemple, la variable "x" est déclarée comme immuable et ne peut pas être modifiée après avoir été assignée.

Traitement des collections avec les fonctions de la bibliothèque standard

La bibliothèque standard de Scala offre de nombreuses fonctions pour le traitement des collections, telles que les listes, les ensembles et les tableaux. Ces fonctions permettent de réaliser des opérations de transformation, de filtrage et d'agrégation sur les éléments des collections.

Voici un exemple d'utilisation de la fonction "map" pour transformer une liste en doublant chaque élément :

val liste = List(1, 2, 3, 4, 5)
val listeDoublee = liste.map(x => x * 2)

Dans cet exemple, la fonction "map" est utilisée pour appliquer la transformation "x => x * 2" à chaque élément de la liste, ce qui donne une nouvelle liste avec les éléments doublés.
