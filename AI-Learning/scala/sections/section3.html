Variables et types de données

En Scala, les variables sont déclarées en utilisant le mot-clé var suivi du nom de la variable et de son type. Par exemple :

var x: Int = 5

Ici, nous avons déclaré une variable x de type Int et lui avons attribué la valeur 5.

Scala prend également en charge l'inférence de type, ce qui signifie que vous pouvez omettre le type de la variable si le compilateur peut le déduire. Par exemple :

var y = "Hello"

Ici, la variable y est automatiquement déduite comme étant de type String.

Opérateurs et expressions

Scala prend en charge les opérateurs arithmétiques de base tels que l'addition, la soustraction, la multiplication et la division. Par exemple :

val a = 10
val b = 5
val sum = a + b
val difference = a - b
val product = a * b
val quotient = a / b

Ici, nous avons déclaré les variables a et b et effectué des opérations arithmétiques pour calculer la somme, la différence, le produit et le quotient.

Structures de contrôle (boucles, conditions)

En Scala, vous pouvez utiliser les structures de contrôle classiques telles que les boucles while et for, ainsi que les instructions conditionnelles if et else. Par exemple :

var i = 0

while (i < 10) {
  println("Iteration " + i)
  i += 1
}

for (j <- 1 to 5) {
  println("Value of j: " + j)
}

val x = 10

if (x > 5) {
  println("x is greater than 5")
} else {
  println("x is less than or equal to 5")
}

Ici, nous avons utilisé une boucle while pour afficher les itérations, une boucle for pour afficher les valeurs de j de 1 à 5, et une instruction if pour vérifier si x est supérieur à 5.

Fonctions et méthodes

En Scala, les fonctions sont définies en utilisant le mot-clé def. Par exemple :

def add(x: Int, y: Int): Int = {
  return x + y
}

val result = add(5, 10)
println("Result: " + result)

Ici, nous avons défini une fonction add qui prend deux paramètres x et y de type Int et renvoie leur somme. Nous avons ensuite appelé cette fonction avec les arguments 5 et 10 et stocké le résultat dans la variable result.

Classes et objets

En Scala, vous pouvez définir des classes en utilisant le mot-clé class. Par exemple :

class Person(name: String, age: Int) {
  def greet(): Unit = {
    println("Hello, my name is " + name + " and I am " + age + " years old.")
  }
}

val person = new Person("John", 30)
person.greet()

Ici, nous avons défini une classe Person avec deux paramètres name et age. La classe a une méthode greet qui affiche un message de salutation. Nous avons ensuite créé une instance de la classe Person avec les valeurs "John" et 30, et appelé la méthode greet sur cette instance.

Héritage et polymorphisme

En Scala, vous pouvez hériter d'une classe existante en utilisant le mot-clé extends. Par exemple :

class Animal(name: String) {
  def sound(): Unit = {
    println("The animal makes a sound.")
  }
}

class Dog(name: String) extends Animal(name) {
  override def sound(): Unit = {
    println("The dog barks.")
  }
}

val animal: Animal = new Dog("Buddy")
animal.sound()

Ici, nous avons défini une classe de base Animal avec une méthode sound. Nous avons ensuite défini une classe dérivée Dog qui hérite de la classe Animal et remplace la méthode sound pour afficher un aboiement de chien. Nous avons créé une instance de la classe Dog et l'avons assignée à une variable de type Animal, puis appelé la méthode sound sur cette variable.
