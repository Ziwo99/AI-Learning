Section 2 : Les fondamentaux de Scala

Section 2 : Les fondamentaux de Scala

Variables et types de données

Les variables en Scala sont déclarées en utilisant le mot-clé var suivi du nom de la variable et du type de données. Par exemple :

var age: Int = 25

Ici, nous avons déclaré une variable age de type entier (Int) et lui avons attribué une valeur de 25.

Scala prend également en charge l'inférence de type, ce qui signifie que vous pouvez omettre le type de données lors de la déclaration d'une variable si le compilateur peut déduire le type. Par exemple :

var name = "John Doe"

Ici, la variable name est automatiquement déduite comme étant de type String en raison de la valeur assignée ("John Doe").

Opérations et expressions

Scala prend en charge les opérations arithmétiques de base telles que l'addition, la soustraction, la multiplication et la division. Par exemple :

val a = 10
val b = 5
val sum = a + b
val product = a * b

Ici, nous avons déclaré deux variables a et b et effectué des opérations d'addition et de multiplication pour obtenir les résultats dans les variables sum et product.

Structures de contrôle

En Scala, vous pouvez utiliser des structures de contrôle telles que les boucles if, else et else if pour prendre des décisions conditionnelles. Par exemple :

val age = 18

if (age >= 18) {
    println("Vous êtes majeur.")
} else {
    println("Vous êtes mineur.")
}

Ici, nous avons utilisé une structure if pour vérifier si l'âge est supérieur ou égal à 18, puis affiché un message approprié en fonction de la condition.

Fonctions et méthodes

En Scala, vous pouvez définir des fonctions et des méthodes pour regrouper un ensemble d'instructions. Par exemple :

def greet(name: String): Unit = {
    println("Bonjour, " + name + "!")
}

greet("Alice")

Ici, nous avons défini une fonction greet qui prend un paramètre name de type String et affiche un message de salutation. Nous avons ensuite appelé cette fonction en lui passant le nom "Alice".

Collections et itérations

Scala offre une variété de collections telles que les listes, les ensembles et les tableaux, ainsi que des méthodes d'itération pour parcourir ces collections. Par exemple :

val numbers = List(1, 2, 3, 4, 5)

for (num <- numbers) {
    println(num)
}

Ici, nous avons créé une liste de nombres et utilisé une boucle for pour itérer sur chaque élément de la liste et l'afficher.

Ceci conclut la section sur les fondamentaux de Scala. Vous avez appris les bases de la déclaration de variables, les opérations et expressions, les structures de contrôle, les fonctions et méthodes, ainsi que les collections et itérations.
