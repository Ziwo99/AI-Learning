Section 2: Les bases de Scala

Syntaxe de base et structure du code Scala

Scala est un langage de programmation polyvalent qui combine les paradigmes de programmation orientée objet et fonctionnelle. La syntaxe de base de Scala est similaire à celle de Java, mais avec des fonctionnalités supplémentaires qui le rendent plus expressif et concis.

Voici un exemple de code Scala :

object HelloWorld {
  def main(args: Array[String]) {
    println("Hello, World!")
  }
}

Dans cet exemple, nous avons une classe HelloWorld avec une méthode main qui affiche le message "Hello, World!".

Types de données et variables en Scala

Scala prend en charge différents types de données, tels que les entiers, les flottants, les chaînes de caractères, les booléens, etc. Voici quelques exemples :

val x: Int = 10
val y: Double = 3.14
val name: String = "John"
val isTrue: Boolean = true

Dans cet exemple, nous déclarons des variables x, y, name et isTrue avec leurs types respectifs.

Opérations arithmétiques et opérateurs logiques en Scala

Scala prend en charge les opérations arithmétiques de base telles que l'addition, la soustraction, la multiplication et la division. Voici quelques exemples :

val sum = 10 + 5
val difference = 10 - 5
val product = 10 * 5
val quotient = 10 / 5

Scala prend également en charge les opérateurs logiques tels que && (ET), || (OU) et ! (NON). Voici un exemple :

val x = true
val y = false
val result = x && y

Dans cet exemple, la variable result aura la valeur false car x est vrai et y est faux.

Structures de contrôle (boucles, conditions) en Scala

Scala prend en charge les structures de contrôle telles que les boucles et les conditions.

Voici un exemple d'une boucle while en Scala :

var i = 0

while (i < 10) {
  println(i)
  i += 1
}

Cet exemple affiche les nombres de 0 à 9 en utilisant une boucle while.

Voici un exemple d'une condition if-else en Scala :

val x = 10

if (x > 5) {
  println("x is greater than 5")
} else {
  println("x is less than or equal to 5")
}

Cet exemple affiche "x is greater than 5" si la valeur de x est supérieure à 5, sinon il affiche "x is less than or equal to 5".
