Syntaxe de base et conventions de codage en Scala

Scala est un langage de programmation polyvalent qui combine les fonctionnalités de la programmation orientée objet et de la programmation fonctionnelle. La syntaxe de base de Scala est similaire à celle de Java, mais avec des fonctionnalités supplémentaires qui le rendent plus expressif et concis.

Voici quelques conventions de codage courantes en Scala :

  • Utilisez des noms de variables et de fonctions significatifs et descriptifs.
  • Utilisez des majuscules pour les noms de classes et de traits.
  • Utilisez des minuscules pour les noms de variables et de fonctions.
  • Utilisez des underscores (_) pour séparer les mots dans les noms de variables et de fonctions.
  • Utilisez des espaces pour améliorer la lisibilité du code.

Types de données et variables en Scala

Scala prend en charge différents types de données, y compris les types primitifs tels que Int, Double, Boolean, ainsi que les types de données plus complexes tels que les chaînes de caractères (String) et les listes (List).

Voici quelques exemples de déclaration de variables en Scala :

val age: Int = 25
var name: String = "John"
val pi: Double = 3.14

La déclaration val crée une variable immuable, tandis que la déclaration var crée une variable mutable.

Opérations arithmétiques et opérateurs logiques en Scala

Scala prend en charge les opérations arithmétiques de base telles que l'addition (+), la soustraction (-), la multiplication (*), la division (/) et le modulo (%). Il prend également en charge les opérateurs logiques tels que && (ET logique), || (OU logique) et ! (NON logique).

Voici quelques exemples d'opérations arithmétiques et logiques en Scala :

val x = 10
val y = 5

val sum = x + y
val difference = x - y
val product = x * y
val quotient = x / y
val remainder = x % y

val isTrue = true
val isFalse = false

val andResult = isTrue && isFalse
val orResult = isTrue || isFalse
val notResult = !isTrue

Structures de contrôle (boucles, conditions) en Scala

Scala prend en charge les structures de contrôle courantes telles que les boucles for et while, ainsi que les instructions conditionnelles if, else if et else.

Voici quelques exemples d'utilisation des structures de contrôle en Scala :

val numbers = List(1, 2, 3, 4, 5)

for (number <- numbers) {
  println(number)
}

var i = 0
while (i < 5) {
  println(i)
  i += 1
}

val x = 10

if (x > 5) {
  println("x is greater than 5")
} else if (x < 5) {
  println("x is less than 5")
} else {
  println("x is equal to 5")
}

Ces exemples illustrent comment utiliser les boucles for et while pour itérer sur une liste de nombres, ainsi que l'utilisation des instructions conditionnelles if, else if et else pour effectuer des actions en fonction d'une condition.
