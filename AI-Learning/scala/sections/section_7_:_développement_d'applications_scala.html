Section 7 : Développement d'applications Scala

Section 7 : Développement d'applications Scala

Création d'une application console

Lorsque vous développez une application Scala pour une utilisation en mode console, vous pouvez utiliser la classe App fournie par Scala pour simplifier le processus. Voici un exemple de création d'une application console Scala : 

object HelloWorld extends App {
  println("Hello, World!")
}

Dans cet exemple, nous avons créé un objet HelloWorld qui étend la classe App. La méthode main est automatiquement générée et exécutée lors de l'exécution de l'application. Dans ce cas, nous utilisons simplement la fonction println pour afficher le message "Hello, World!" dans la console.

Création d'une application web

Pour créer une application web en Scala, vous pouvez utiliser le framework Play. Voici un exemple de création d'une application web Scala avec Play : 

import play.api.mvc._

object HomeController extends Controller {
  def index = Action {
    Ok("Hello, World!")
  }
}

Dans cet exemple, nous avons créé un objet HomeController qui étend la classe Controller de Play. La méthode index est une action qui renvoie une réponse HTTP avec le message "Hello, World!" en utilisant la classe Ok. Cette action peut être associée à une URL spécifique dans le fichier de routage de l'application Play.

Création d'une application distribuée

Pour créer une application Scala distribuée, vous pouvez utiliser le framework Akka. Voici un exemple de création d'une application distribuée Scala avec Akka : 

import akka.actor._

object HelloWorld extends App {
  val system = ActorSystem("HelloWorldSystem")
  val helloActor = system.actorOf(Props[HelloActor], name = "helloActor")

  helloActor ! "Hello, World!"
}

class HelloActor extends Actor {
  def receive = {
    case message: String => println(message)
  }
}

Dans cet exemple, nous avons créé un système d'acteurs Akka avec un acteur HelloActor. L'acteur reçoit un message de type String et l'affiche dans la console. L'objet HelloWorld crée une instance du système d'acteurs, puis envoie le message "Hello, World!" à l'acteur helloActor.
