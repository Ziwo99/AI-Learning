Structure d'un programme Objective-C

Un programme Objective-C est composé de plusieurs éléments, notamment des classes, des méthodes et des variables. Voici un exemple de structure de base d'un programme Objective-C :

#import <Foundation/Foundation.h>

// Déclaration de la classe
@interface MaClasse : NSObject

// Déclaration des propriétés et des méthodes

@end

// Implémentation de la classe
@implementation MaClasse

// Implémentation des méthodes

@end

// Fonction principale
int main(int argc, const char * argv[]) {
    @autoreleasepool {
        // Code principal
        return 0;
    }
}

Variables et types de données

En Objective-C, les variables doivent être déclarées avec un type spécifique. Voici quelques types de données couramment utilisés :

  • int : pour les entiers
  • float : pour les nombres à virgule flottante
  • double : pour les nombres à virgule flottante avec une précision supérieure
  • BOOL : pour les valeurs booléennes (true ou false)
  • NSString : pour les chaînes de caractères
  • NSArray : pour les tableaux
  • NSDictionary : pour les dictionnaires

Voici un exemple de déclaration de variables :

int age = 25;
float pi = 3.14;
BOOL isTrue = YES;
NSString *name = @"John Doe";
NSArray *myArray = @[@"apple", @"banana", @"orange"];
NSDictionary *myDictionary = @{@"key1": @"value1", @"key2": @"value2"};

Opérateurs et expressions

Objective-C prend en charge différents types d'opérateurs, tels que les opérateurs arithmétiques, les opérateurs de comparaison et les opérateurs logiques. Voici quelques exemples d'opérateurs couramment utilisés :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
  • == : égalité
  • != : inégalité
  • < : inférieur
  • > : supérieur
  • && : ET logique
  • || : OU logique

Voici un exemple d'utilisation d'opérateurs :

int a = 5;
int b = 10;
int c = a + b; // c vaut 15

BOOL isTrue = (a > b) || (b > 0); // isTrue vaut YES

Les boucles et les conditions

Les boucles et les conditions permettent de contrôler le flux d'exécution d'un programme. Voici quelques exemples de boucles et de conditions couramment utilisées en Objective-C :

La boucle for :

for (int i = 0; i < 10; i++) {
    NSLog(@"%d", i);
}

La boucle while :

int i = 0;
while (i < 10) {
    NSLog(@"%d", i);
    i++;
}

La condition if-else :

int age = 25;
if (age >= 18) {
    NSLog(@"Vous êtes majeur");
} else {
    NSLog(@"Vous êtes mineur");
}

Les tableaux et les collections

Les tableaux et les collections permettent de stocker et de manipuler des ensembles de données. Voici quelques exemples d'utilisation de tableaux et de collections en Objective-C :

Les tableaux :

NSArray *myArray = @[@"apple", @"banana", @"orange"];
NSLog(@"%@", myArray[0]); // Affiche "apple"

NSMutableArray *mutableArray = [NSMutableArray arrayWithArray:myArray];
[mutableArray addObject:@"grape"];
NSLog(@"%@", mutableArray); // Affiche "apple, banana, orange, grape"

Les dictionnaires :

NSDictionary *myDictionary = @{@"key1": @"value1", @"key2": @"value2"};
NSLog(@"%@", myDictionary[@"key1"]); // Affiche "value1"

NSMutableDictionary *mutableDictionary = [NSMutableDictionary dictionaryWithDictionary:myDictionary];
[mutableDictionary setObject:@"value3" forKey:@"key3"];
NSLog(@"%@", mutableDictionary); // Affiche "key1=value1, key2=value2, key3=value3"

Les fonctions et les méthodes

Les fonctions et les méthodes permettent d'organiser et de réutiliser du code. Voici un exemple de déclaration et d'utilisation d'une méthode en Objective-C :

// Déclaration de la méthode
- (void)sayHello {
    NSLog(@"Hello!");
}

// Utilisation de la méthode
MaClasse *myObject = [[MaClasse alloc] init];
[myObject sayHello]; // Affiche "Hello!"

Les classes et les objets

Les classes et les objets sont les éléments fondamentaux de la programmation orientée objet en Objective-C. Voici un exemple de déclaration et d'utilisation d'une classe et d'un objet :

// Déclaration de la classe
@interface MaClasse : NSObject

@property (nonatomic, strong) NSString *name;

- (void)sayHello;

@end

// Implémentation de la classe
@implementation MaClasse

- (void)sayHello {
    NSLog(@"Hello, %@", self.name);
}

@end

// Utilisation de la classe et de l'objet
MaClasse *myObject = [[MaClasse alloc] init];
myObject.name = @"John Doe";
[myObject sayHello]; // Affiche "Hello, John Doe"

L'héritage et la polymorphie

L'héritage permet de créer de nouvelles classes basées sur des classes existantes, tandis que la polymorphie permet à un objet d'être traité comme un objet de n'importe quelle classe parente. Voici un exemple d'utilisation de l'héritage et de la polymorphie en Objective-C :

// Classe parente
@interface Animal : NSObject

- (void)sayHello;

@end

@implementation Animal

- (void)sayHello {
    NSLog(@"Hello, I'm an animal");
}

@end

// Classe enfant
@interface Dog : Animal

@end

@implementation Dog

- (void)sayHello {
    NSLog(@"Hello, I'm a dog");
}

@end

// Utilisation de l'héritage et de la polymorphie
Animal *animal = [[Animal alloc] init];
[animal sayHello]; // Affiche "Hello, I'm an animal"

Dog *dog = [[Dog alloc] init];
[dog sayHello]; // Affiche "Hello, I'm a dog"
