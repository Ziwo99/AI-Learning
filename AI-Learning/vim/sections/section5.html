Utilisation des raccourcis clavier

Les raccourcis clavier sont essentiels pour augmenter votre productivité avec Vim. Voici quelques raccourcis clavier couramment utilisés :

Ctrl + f : Fait défiler vers le bas d'une page
Ctrl + b : Fait défiler vers le haut d'une page
gg : Se déplace au début du fichier
G : Se déplace à la fin du fichier
yy : Copie la ligne actuelle
dd : Supprime la ligne actuelle
p : Colle le contenu copié ou supprimé après le curseur
u : Annule la dernière action
Ctrl + r : Rétablit la dernière action annulée

Utilisation des commandes en mode ligne

Les commandes en mode ligne vous permettent d'exécuter des actions spécifiques en utilisant des commandes textuelles. Voici quelques commandes en mode ligne utiles :

:w : Enregistre le fichier
:q : Quitte Vim
:wq : Enregistre le fichier et quitte Vim
:q! : Quitte Vim sans enregistrer les modifications
:set number : Active l'affichage des numéros de ligne
:set syntax=python : Active la coloration syntaxique pour le langage Python

Utilisation des commandes externes

Vim vous permet d'exécuter des commandes externes directement depuis l'éditeur. Voici comment utiliser les commandes externes :

:!ls : Exécute la commande "ls" pour lister les fichiers du répertoire courant
:!make : Exécute la commande "make" pour compiler votre projet
:r !date : Insère la date actuelle à l'emplacement du curseur

Il est important de noter que les commandes externes peuvent varier en fonction de votre système d'exploitation.
