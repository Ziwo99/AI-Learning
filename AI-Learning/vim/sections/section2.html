Navigation dans un fichier

La navigation dans un fichier est une compétence essentielle pour travailler efficacement avec Vim. Voici quelques commandes de navigation couramment utilisées :

h : Déplace le curseur d'un caractère vers la gauche
j : Déplace le curseur d'une ligne vers le bas
k : Déplace le curseur d'une ligne vers le haut
l : Déplace le curseur d'un caractère vers la droite
gg : Déplace le curseur en haut du fichier
G : Déplace le curseur en bas du fichier
0 : Déplace le curseur au début de la ligne
$ : Déplace le curseur à la fin de la ligne

Utilisez ces commandes de navigation pour vous déplacer rapidement dans un fichier et accéder aux parties spécifiques dont vous avez besoin.

Édition de texte

Vim offre de nombreuses fonctionnalités d'édition de texte pour vous aider à modifier et à manipuler votre contenu. Voici quelques commandes d'édition couramment utilisées :

i : Passe en mode insertion avant le curseur
a : Passe en mode insertion après le curseur
o : Insère une nouvelle ligne sous le curseur et passe en mode insertion
dd : Supprime la ligne entière sous le curseur
yy : Copie la ligne entière sous le curseur
p : Colle le contenu copié ou supprimé après le curseur
u : Annule la dernière action effectuée

Utilisez ces commandes d'édition pour modifier votre texte de manière précise et efficace.

Enregistrement et exécution de macros

Les macros sont un moyen puissant d'automatiser des tâches répétitives dans Vim. Voici comment enregistrer et exécuter une macro :

q{lettre} : Commence l'enregistrement d'une macro dans la lettre spécifiée
q : Arrête l'enregistrement de la macro
@{lettre} : Exécute la macro enregistrée dans la lettre spécifiée

Enregistrez vos macros pour automatiser des séquences d'actions et gagner du temps lors de l'édition de texte.
