Section 3 : Utilisation de JavaScript avec Ajax

Section 3 : Utilisation de JavaScript avec Ajax

Manipulation du DOM avec JavaScript

La manipulation du DOM (Document Object Model) est une étape essentielle dans le développement d'applications web interactives. Avec JavaScript, vous pouvez facilement accéder et modifier les éléments HTML d'une page.

Voici un exemple de code JavaScript qui modifie le contenu d'un élément HTML :

var element = document.getElementById('myElement');
element.innerHTML = 'Nouveau contenu';

Dans cet exemple, nous utilisons la méthode getElementById pour sélectionner l'élément avec l'ID "myElement". Ensuite, nous utilisons la propriété innerHTML pour modifier le contenu de cet élément.

Gestion des événements avec JavaScript

La gestion des événements est une autre fonctionnalité importante de JavaScript. Elle permet de réagir aux actions de l'utilisateur, telles que le clic sur un bouton ou la soumission d'un formulaire.

Voici un exemple de code JavaScript qui ajoute un gestionnaire d'événements à un bouton :

var button = document.getElementById('myButton');
button.addEventListener('click', function() {
    alert('Le bouton a été cliqué');
});

Dans cet exemple, nous utilisons la méthode addEventListener pour ajouter un gestionnaire d'événements au bouton avec l'ID "myButton". Lorsque le bouton est cliqué, la fonction anonyme est exécutée et affiche une boîte de dialogue avec le message "Le bouton a été cliqué".

Utilisation des objets XMLHttpRequest

L'objet XMLHttpRequest est utilisé pour effectuer des requêtes HTTP asynchrones, ce qui permet de charger des données à partir d'un serveur sans recharger la page entière.

Voici un exemple de code JavaScript qui utilise l'objet XMLHttpRequest pour charger des données depuis un fichier JSON :

var xhr = new XMLHttpRequest();
xhr.open('GET', 'data.json', true);
xhr.onreadystatechange = function() {
    if (xhr.readyState === 4 && xhr.status === 200) {
        var data = JSON.parse(xhr.responseText);
        console.log(data);
    }
};
xhr.send();

Dans cet exemple, nous créons un nouvel objet XMLHttpRequest en utilisant le constructeur XMLHttpRequest. Ensuite, nous utilisons la méthode open pour spécifier le type de requête (GET), l'URL du fichier JSON et si la requête doit être asynchrone (true). Enfin, nous utilisons la propriété onreadystatechange pour définir une fonction qui sera appelée chaque fois que l'état de la requête change. Lorsque la requête est terminée (readyState = 4) et que le statut est OK (status = 200), nous utilisons la méthode JSON.parse pour convertir la réponse en un objet JavaScript.

Ce sont là quelques-uns des concepts clés de l'utilisation de JavaScript avec Ajax. En comprenant ces concepts et en les appliquant dans vos projets, vous pourrez créer des applications web interactives et dynamiques.
