Introduction à JavaFX

JavaFX est un framework graphique moderne et puissant pour la création d'interfaces utilisateur riches dans les applications Java. Il offre une grande variété de fonctionnalités pour la conception d'interfaces utilisateur attrayantes et interactives.

Voici un exemple de code JavaFX qui crée une fenêtre simple avec un bouton :

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class HelloWorld extends Application {
    public static void main(String[] args) {
        launch(args);
    }

    @Override
    public void start(Stage primaryStage) {
        Button button = new Button("Hello World!");
        StackPane root = new StackPane();
        root.getChildren().add(button);
        primaryStage.setScene(new Scene(root, 300, 200));
        primaryStage.show();
    }
}

Utilisation de JDBC pour accéder à une base de données

JDBC (Java Database Connectivity) est une API Java standard pour interagir avec des bases de données relationnelles. Il permet d'exécuter des requêtes SQL, de récupérer des résultats et de mettre à jour les données dans la base de données.

Voici un exemple de code qui utilise JDBC pour se connecter à une base de données MySQL et exécuter une requête SELECT :

import java.sql.*;

public class DatabaseExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "root";
        String password = "password";

        try (Connection connection = DriverManager.getConnection(url, username, password)) {
            String query = "SELECT * FROM users";
            Statement statement = connection.createStatement();
            ResultSet resultSet = statement.executeQuery(query);

            while (resultSet.next()) {
                int id = resultSet.getInt("id");
                String name = resultSet.getString("name");
                System.out.println("ID: " + id + ", Name: " + name);
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

Introduction à Spring Framework

Spring Framework est un framework d'application Java populaire qui facilite le développement d'applications d'entreprise robustes et évolutives. Il fournit une infrastructure complète pour la gestion des dépendances, l'inversion de contrôle, l'injection de dépendances et bien plus encore.

Voici un exemple de code qui utilise Spring Framework pour créer une application simple avec une classe de service et une classe de contrôleur :

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@Service
class GreetingService {
    public String getGreeting() {
        return "Hello, World!";
    }
}

@RestController
class GreetingController {
    @Autowired
    private GreetingService greetingService;

    @GetMapping("/greeting")
    public String greeting() {
        return greetingService.getGreeting();
    }
}

Utilisation de Hibernate pour la persistance des données

Hibernate est un framework de persistance des données qui facilite l'interaction avec les bases de données relationnelles en utilisant des objets Java. Il simplifie le processus de mapping objet-relationnel et permet de manipuler les données de manière transparente.

Voici un exemple de code qui utilise Hibernate pour sauvegarder un objet dans une base de données :

import javax.persistence.*;

@Entity
@Table(name = "users")
class User {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private int id;

    private String name;

    // Getters and setters
}

public class HibernateExample {
    public static void main(String[] args) {
        EntityManagerFactory entityManagerFactory = Persistence.createEntityManagerFactory("myPersistenceUnit");
        EntityManager entityManager = entityManagerFactory.createEntityManager();

        User user = new User();
        user.setName("John Doe");

        entityManager.getTransaction().begin();
        entityManager.persist(user);
        entityManager.getTransaction().commit();

        entityManager.close();
        entityManagerFactory.close();
    }
}

Introduction à JUnit pour les tests unitaires

JUnit est un framework de tests unitaires pour Java. Il permet de créer et d'exécuter des tests unitaires pour vérifier le bon fonctionnement des différentes parties d'une application Java.

Voici un exemple de code qui utilise JUnit pour tester une méthode simple :

import org.junit.Test;
import static org.junit.Assert.*;

public class MathUtilsTest {
    @Test
    public void testAdd() {
        MathUtils mathUtils = new MathUtils();
        int result = mathUtils.add(5, 10);
        assertEquals(15, result);
    }
}

class MathUtils {
    public int add(int a, int b) {
        return a + b;
    }
}

Utilisation de Maven pour la gestion des dépendances

Maven est un outil de gestion de projet qui facilite la création, la compilation et la gestion des dépendances d'un projet Java. Il permet de gérer efficacement les bibliothèques externes et de simplifier le processus de construction d'une application Java.

Voici un exemple de fichier pom.xml qui configure Maven pour un projet Java :

<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>myproject</artifactId>
    <version>1.0.0</version>

    <dependencies>
        <dependency>
            <groupId>org.apache.commons</groupId>
            <artifactId>commons-lang3</artifactId>
            <version>3.12.0</version>
        </dependency>
    </dependencies>
</project>
