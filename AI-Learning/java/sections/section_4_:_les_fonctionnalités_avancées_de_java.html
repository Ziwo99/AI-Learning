Section 4 : Les fonctionnalités avancées de Java

Section 4 : Les fonctionnalités avancées de Java

Gestion des fichiers et des flux

La gestion des fichiers et des flux est une fonctionnalité avancée de Java qui permet de manipuler des fichiers et de lire/écrire des données à partir/depuis ces fichiers. Cette fonctionnalité est essentielle pour de nombreuses applications, notamment celles qui nécessitent le stockage et la récupération de données.

Voici un exemple de code qui illustre comment lire le contenu d'un fichier texte :

try {
    File file = new File("example.txt");
    BufferedReader br = new BufferedReader(new FileReader(file));
    String line;
    while ((line = br.readLine()) != null) {
        System.out.println(line);
    }
    br.close();
} catch (IOException e) {
    e.printStackTrace();
}

Et voici un exemple de code qui montre comment écrire dans un fichier texte :

try {
    FileWriter writer = new FileWriter("example.txt");
    writer.write("Hello, world!");
    writer.close();
} catch (IOException e) {
    e.printStackTrace();
}

Les exceptions avancées

Les exceptions avancées sont utilisées pour gérer les erreurs et les situations exceptionnelles dans un programme Java. Elles permettent de détecter et de traiter les erreurs de manière appropriée, ce qui améliore la robustesse et la fiabilité du code.

Voici un exemple de code qui montre comment gérer une exception :

try {
    int result = divide(10, 0);
    System.out.println("Result: " + result);
} catch (ArithmeticException e) {
    System.out.println("Error: Division by zero");
}

public static int divide(int a, int b) throws ArithmeticException {
    if (b == 0) {
        throw new ArithmeticException("Division by zero");
    }
    return a / b;
}

Les annotations

Les annotations sont des métadonnées qui fournissent des informations supplémentaires sur les éléments du code Java. Elles peuvent être utilisées pour spécifier des contraintes, des configurations ou des comportements particuliers.

Voici un exemple de code qui utilise une annotation personnalisée :

@MyAnnotation(name = "John", age = 30)
public class Person {
    // Code de la classe Person
}

Les génériques

Les génériques permettent de créer des classes, des interfaces et des méthodes qui peuvent être paramétrées par un type. Cela permet de réutiliser du code et d'améliorer la sécurité et la lisibilité du code.

Voici un exemple de code qui montre comment utiliser des génériques :

public class Box<T> {
    private T content;
    
    public void setContent(T content) {
        this.content = content;
    }
    
    public T getContent() {
        return content;
    }
}

Box<String> box = new Box<>();
box.setContent("Hello, world!");
String content = box.getContent();

Les threads et la concurrence

Les threads permettent d'exécuter plusieurs tâches simultanément dans un programme Java. La concurrence est la gestion de l'accès simultané aux ressources partagées par plusieurs threads. Ces fonctionnalités sont essentielles pour développer des applications réactives et performantes.

Voici un exemple de code qui montre comment créer et exécuter un thread :

public class MyThread extends Thread {
    public void run() {
        System.out.println("Thread is running");
    }
}

MyThread thread = new MyThread();
thread.start();

Les expressions lambda et les streams

Les expressions lambda et les streams sont des fonctionnalités introduites dans Java 8 qui simplifient le traitement des collections de données. Les expressions lambda permettent de définir des fonctions anonymes de manière concise, tandis que les streams permettent d'effectuer des opérations de traitement de données de manière fluide et parallèle.

Voici un exemple de code qui montre comment utiliser une expression lambda et un stream :

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
int sum = numbers.stream()
                 .filter(n -> n % 2 == 0)
                 .mapToInt(n -> n)
                 .sum();
System.out.println("Sum of even numbers: " + sum);
