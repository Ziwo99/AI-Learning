Section 1 : Introduction à Java

Histoire et évolution de Java

Java est un langage de programmation orienté objet créé par James Gosling et son équipe chez Sun Microsystems en 1995. Il a été conçu pour être portable, sécurisé et robuste. Depuis sa création, Java a connu plusieurs versions et évolutions, avec des améliorations constantes de ses fonctionnalités et de sa performance.

Installation et configuration de l'environnement de développement Java

Pour commencer à développer en Java, vous devez installer et configurer l'environnement de développement Java (JDK - Java Development Kit) sur votre machine. Voici les étapes à suivre :

  1. Téléchargez le JDK à partir du site officiel d'Oracle.
  2. Exécutez le programme d'installation et suivez les instructions.
  3. Configurez les variables d'environnement JAVA_HOME et PATH pour accéder aux outils Java depuis n'importe quel répertoire.
  4. Vérifiez l'installation en ouvrant une fenêtre de terminal et en exécutant la commande 
    java -version

Structure d'un programme Java

Un programme Java est composé de classes, de méthodes et d'instructions. Voici une structure de base d'un programme Java : 

public class HelloWorld {
    public static void main(String[] args) {
        // Instructions du programme
        System.out.println("Hello, World!");
    }
}

Variables et types de données

En Java, les variables sont utilisées pour stocker des données. Chaque variable a un type de données spécifique. Voici quelques types de données couramment utilisés en Java :

  • int : pour stocker des nombres entiers.
  • double : pour stocker des nombres décimaux.
  • boolean : pour stocker des valeurs booléennes (true ou false).
  • String : pour stocker des chaînes de caractères.

Voici un exemple d'utilisation de variables en Java : 

int age = 25;
double height = 1.75;
boolean isStudent = true;
String name = "John Doe";

Opérateurs et expressions

Les opérateurs en Java sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques opérateurs couramment utilisés en Java :

  • Opérateurs arithmétiques : +, -, *, /, %
  • Opérateurs de comparaison : ==, !=, >, <, >=, <=
  • Opérateurs logiques : &&, ||, !

Voici un exemple d'utilisation d'opérateurs en Java : 

int a = 5;
int b = 3;
int sum = a + b;
boolean isGreater = (a > b);

Entrées et sorties

En Java, vous pouvez interagir avec l'utilisateur en utilisant les entrées et les sorties. La classe Scanner est couramment utilisée pour lire les entrées de l'utilisateur, tandis que la classe System.out est utilisée pour afficher des sorties à l'utilisateur. Voici un exemple d'utilisation des entrées et sorties en Java : 

import java.util.Scanner;

public class InputOutputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();
        
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();
        
        System.out.println("Hello, " + name + "! You are " + age + " years old.");
    }
}
Ceci conclut la section d'introduction à Java. Vous avez maintenant une compréhension de base de Java, de son histoire, de son installation, de sa structure de programme, des variables et types de données, des opérateurs et expressions, ainsi que des entrées et sorties.
