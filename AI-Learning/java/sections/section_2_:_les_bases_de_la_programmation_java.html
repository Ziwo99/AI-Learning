Section 2 : Les bases de la programmation Java

Section 2 : Les bases de la programmation Java

Structures de contrôle conditionnelles (if, else, switch)

Les structures de contrôle conditionnelles permettent d'exécuter des blocs de code en fonction de certaines conditions. En Java, nous avons trois principales structures de contrôle conditionnelles : if, else et switch.

Boucles (for, while, do-while)

Les boucles permettent de répéter l'exécution d'un bloc de code tant qu'une condition est vérifiée. En Java, nous avons trois principales boucles : for, while et do-while.

Tableaux et collections

Les tableaux et les collections sont des structures de données utilisées pour stocker des éléments de même type. Les tableaux sont de taille fixe, tandis que les collections peuvent être de taille variable. En Java, nous avons plusieurs types de tableaux et de collections.

Fonctions et méthodes

Les fonctions et les méthodes sont des blocs de code réutilisables qui effectuent une tâche spécifique. Les fonctions sont définies en dehors des classes, tandis que les méthodes sont définies à l'intérieur des classes. En Java, nous pouvons définir nos propres fonctions et méthodes.

Gestion des exceptions

La gestion des exceptions permet de gérer les erreurs qui se produisent pendant l'exécution d'un programme. En Java, nous utilisons les blocs try-catch pour capturer et traiter les exceptions.

Manipulation des chaînes de caractères

La manipulation des chaînes de caractères est une opération courante en programmation. En Java, nous avons de nombreuses méthodes pour manipuler les chaînes de caractères, telles que la concaténation, la recherche de sous-chaînes, le remplacement, etc.

Voici un exemple de code utilisant les boucles et les structures de contrôle conditionnelles :

int[] numbers = {1, 2, 3, 4, 5};

for (int i = 0; i < numbers.length; i++) {
    if (numbers[i] % 2 == 0) {
        System.out.println(numbers[i] + " est pair");
    } else {
        System.out.println(numbers[i] + " est impair");
    }
}
