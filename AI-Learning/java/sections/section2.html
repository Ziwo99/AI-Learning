Les opérateurs en Java

Les opérateurs en Java sont des symboles spéciaux qui effectuent des opérations sur les variables et les valeurs. Ils peuvent être utilisés pour effectuer des calculs mathématiques, des opérations logiques et des manipulations de bits.

Voici quelques exemples d'opérateurs couramment utilisés en Java :

// Opérateurs arithmétiques
int a = 10 + 5; // Addition
int b = 10 - 5; // Soustraction
int c = 10 * 5; // Multiplication
int d = 10 / 5; // Division
int e = 10 % 5; // Modulo

// Opérateurs de comparaison
boolean f = 10 == 5; // Égal à
boolean g = 10 != 5; // Différent de
boolean h = 10 > 5; // Supérieur à
boolean i = 10 < 5; // Inférieur à
boolean j = 10 >= 5; // Supérieur ou égal à
boolean k = 10 <= 5; // Inférieur ou égal à

// Opérateurs logiques
boolean l = true && false; // ET logique
boolean m = true || false; // OU logique
boolean n = !true; // NON logique

// Opérateurs de manipulation de bits
int o = 10 & 5; // ET binaire
int p = 10 | 5; // OU binaire
int q = 10 ^ 5; // OU exclusif binaire
int r = ~10; // Complément à un binaire
int s = 10 << 2; // Décalage à gauche
int t = 10 >> 2; // Décalage à droite

Les structures de contrôle en Java (boucles, conditions)

Les structures de contrôle en Java permettent de contrôler le flux d'exécution d'un programme. Elles permettent d'exécuter certaines parties du code en fonction de conditions ou de répéter des blocs de code plusieurs fois.

Voici quelques exemples de structures de contrôle couramment utilisées en Java :

// Structure de contrôle if-else
int age = 18;
if (age >= 18) {
    System.out.println("Vous êtes majeur.");
} else {
    System.out.println("Vous êtes mineur.");
}

// Structure de contrôle switch-case
int day = 1;
switch (day) {
    case 1:
        System.out.println("Lundi");
        break;
    case 2:
        System.out.println("Mardi");
        break;
    case 3:
        System.out.println("Mercredi");
        break;
    default:
        System.out.println("Autre jour de la semaine");
}

// Boucle for
for (int i = 0; i < 5; i++) {
    System.out.println("Itération " + i);
}

// Boucle while
int i = 0;
while (i < 5) {
    System.out.println("Itération " + i);
    i++;
}

// Boucle do-while
int j = 0;
do {
    System.out.println("Itération " + j);
    j++;
} while (j < 5);

Les tableaux en Java

Un tableau en Java est une structure de données qui permet de stocker plusieurs valeurs de même type dans une seule variable. Les éléments d'un tableau sont accessibles à l'aide d'un index.

Voici comment déclarer et utiliser un tableau en Java :

// Déclaration d'un tableau d'entiers
int[] numbers = new int[5];

// Initialisation des éléments du tableau
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

// Accès aux éléments du tableau
int firstNumber = numbers[0];
int lastNumber = numbers[4];

// Parcourir un tableau avec une boucle for
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// Parcourir un tableau avec une boucle foreach
for (int number : numbers) {
    System.out.println(number);
}

Les fonctions et les méthodes en Java

Les fonctions et les méthodes en Java permettent d'organiser le code en le divisant en blocs réutilisables. Ils permettent d'effectuer des actions spécifiques et de retourner des valeurs si nécessaire.

Voici comment déclarer et utiliser une fonction en Java :

// Déclaration d'une fonction qui retourne la somme de deux nombres
public static int sum(int a, int b) {
    return a + b;
}

// Appel de la fonction
int result = sum(10, 20);
System.out.println(result);

// Déclaration d'une méthode dans une classe
public class MyClass {
    public void myMethod() {
        System.out.println("Hello, World!");
    }
}

// Appel de la méthode
MyClass myObject = new MyClass();
myObject.myMethod();

La gestion des exceptions en Java

Les exceptions en Java sont des événements qui se produisent pendant l'exécution d'un programme et qui interrompent le flux normal d'exécution. La gestion des exceptions permet de détecter et de gérer ces erreurs de manière appropriée.

Voici comment gérer les exceptions en Java :

// Utilisation d'un bloc try-catch
try {
    // Code susceptible de générer une exception
    int result = 10 / 0;
} catch (ArithmeticException e) {
    // Gestion de l'exception
    System.out.println("Une erreur s'est produite : " + e.getMessage());
}

// Utilisation d'un bloc try-catch-finally
try {
    // Code susceptible de générer une exception
    int result = 10 / 0;
} catch (ArithmeticException e) {
    // Gestion de l'exception
    System.out.println("Une erreur s'est produite : " + e.getMessage());
} finally {
    // Code exécuté toujours, qu'il y ait eu une exception ou non
    System.out.println("Fin de l'exécution");
}
