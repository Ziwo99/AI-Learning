Les opérateurs en Java

Les opérateurs en Java sont des symboles spéciaux qui effectuent des opérations sur les variables et les valeurs. Ils peuvent être utilisés pour effectuer des calculs mathématiques, des opérations logiques et des manipulations de bits.

Voici quelques exemples d'opérateurs couramment utilisés en Java :

int x = 10;
int y = 5;
int z;

// Opérateurs arithmétiques
z = x + y; // Addition
z = x - y; // Soustraction
z = x * y; // Multiplication
z = x / y; // Division
z = x % y; // Modulo

// Opérateurs de comparaison
boolean result;
result = x == y; // Égal à
result = x != y; // Différent de
result = x > y; // Supérieur à
result = x < y; // Inférieur à
result = x >= y; // Supérieur ou égal à
result = x <= y; // Inférieur ou égal à

// Opérateurs logiques
boolean a = true;
boolean b = false;
boolean c;
c = a && b; // ET logique
c = a || b; // OU logique
c = !a; // NON logique

// Opérateurs de manipulation de bits
int num = 5;
int result;
result = num & 3; // ET bit à bit
result = num | 3; // OU bit à bit
result = num ^ 3; // OU exclusif bit à bit
result = ~num; // Complément à un bit à bit
result = num << 2; // Décalage à gauche
result = num >> 2; // Décalage à droite

Les structures de contrôle (boucles, conditions) en Java

Les structures de contrôle en Java permettent de contrôler le flux d'exécution d'un programme. Elles permettent d'exécuter des blocs de code conditionnellement ou de manière répétitive.

Voici quelques exemples de structures de contrôle couramment utilisées en Java :

Les conditions avec l'instruction if

L'instruction if permet d'exécuter un bloc de code si une condition est vraie. Si la condition est fausse, le bloc de code est ignoré.

int x = 10;

if (x > 5) {
    System.out.println("x est supérieur à 5");
} else {
    System.out.println("x est inférieur ou égal à 5");
}

Les boucles avec l'instruction while

L'instruction while permet d'exécuter un bloc de code tant qu'une condition est vraie. Le bloc de code est répété jusqu'à ce que la condition devienne fausse.

int i = 0;

while (i < 10) {
    System.out.println("Iteration " + i);
    i++;
}

Les boucles avec l'instruction for

L'instruction for permet d'exécuter un bloc de code un certain nombre de fois. Elle est souvent utilisée lorsque le nombre d'itérations est connu à l'avance.

for (int i = 0; i < 10; i++) {
    System.out.println("Iteration " + i);
}

Les tableaux en Java

Un tableau en Java est une structure de données qui permet de stocker plusieurs valeurs du même type dans une seule variable. Chaque valeur est associée à un index, qui permet d'accéder à cette valeur dans le tableau.

Voici comment déclarer et initialiser un tableau en Java :

// Déclaration d'un tableau d'entiers
int[] numbers;

// Initialisation du tableau avec 5 éléments
numbers = new int[5];

// Accès aux éléments du tableau
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

// Parcours du tableau avec une boucle for
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

Les fonctions et les méthodes en Java

Les fonctions et les méthodes en Java sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent d'organiser le code en le divisant en petites parties logiques.

Voici comment déclarer et appeler une fonction en Java :

// Déclaration d'une fonction qui calcule la somme de deux nombres
public int sum(int a, int b) {
    return a + b;
}

// Appel de la fonction
int result = sum(10, 20);
System.out.println(result);

La gestion des exceptions en Java

Les exceptions en Java sont des événements qui se produisent lorsqu'une erreur se produit pendant l'exécution d'un programme. Elles permettent de gérer les erreurs de manière contrôlée.

Voici comment gérer les exceptions en Java :

try {
    // Code susceptible de générer une exception
    int result = 10 / 0;
} catch (ArithmeticException e) {
    // Gestion de l'exception
    System.out.println("Une erreur s'est produite : " + e.getMessage());
}
