Section 2 : Les bases du langage Java

Opérateurs et expressions

Les opérateurs en Java permettent de manipuler les valeurs et d'effectuer des opérations sur elles. Les expressions sont des combinaisons d'opérateurs, de variables et de constantes qui produisent une valeur.

Voici quelques exemples d'opérateurs couramment utilisés en Java :

int a = 10;
int b = 5;
int c = a + b; // Addition
int d = a - b; // Soustraction
int e = a * b; // Multiplication
int f = a / b; // Division
int g = a % b; // Modulo

Structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de contrôler le flux d'exécution d'un programme en fonction de certaines conditions.

Voici quelques exemples de structures de contrôle couramment utilisées en Java :

// Boucle while
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

// Boucle for
for (int j = 0; j < 10; j++) {
    System.out.println(j);
}

// Condition if-else
int age = 18;
if (age >= 18) {
    System.out.println("Vous êtes majeur.");
} else {
    System.out.println("Vous êtes mineur.");
}

Tableaux et collections

Les tableaux et les collections permettent de stocker et de manipuler des ensembles de données en Java.

Voici quelques exemples d'utilisation des tableaux et des collections en Java :

// Tableau
int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // Affiche 1

// ArrayList
ArrayList<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add("Charlie");
System.out.println(names.get(0)); // Affiche "Alice"

Manipulation des chaînes de caractères

Les chaînes de caractères sont utilisées pour représenter du texte en Java. Il existe de nombreuses méthodes pour manipuler et formater les chaînes de caractères.

Voici quelques exemples de manipulation des chaînes de caractères en Java :

String message = "Hello, World!";
System.out.println(message.length()); // Affiche 13

String name = "Alice";
String greeting = "Hello, " + name + "!";
System.out.println(greeting); // Affiche "Hello, Alice!"

Ces exemples couvrent les bases du langage Java. Ils vous permettront de comprendre et d'utiliser les concepts fondamentaux du langage.
