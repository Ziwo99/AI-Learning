Types de données primitifs

En Java, les types de données primitifs sont les types de base intégrés dans le langage. Ils représentent les valeurs de base et ne sont pas des objets. Voici les types de données primitifs en Java :

  • byte : un entier signé sur 8 bits
  • short : un entier signé sur 16 bits
  • int : un entier signé sur 32 bits
  • long : un entier signé sur 64 bits
  • float : un nombre à virgule flottante sur 32 bits
  • double : un nombre à virgule flottante sur 64 bits
  • boolean : une valeur booléenne (vrai ou faux)
  • char : un caractère Unicode sur 16 bits

Chaque type de données primitif a une taille fixe en mémoire et des limites de valeurs spécifiques.

Variables et constantes

En Java, une variable est un espace de stockage nommé qui peut contenir une valeur. Pour déclarer une variable, vous devez spécifier son type et son nom. Voici un exemple de déclaration de variable :

int age;

Une constante, en revanche, est une valeur qui ne peut pas être modifiée une fois qu'elle a été définie. En Java, vous pouvez déclarer une constante en utilisant le mot-clé final. Voici un exemple de déclaration de constante :

final double PI = 3.14159;

Les variables et les constantes peuvent être utilisées pour stocker des valeurs et effectuer des opérations dans un programme Java.

Opérations arithmétiques et opérateurs de comparaison

En Java, vous pouvez effectuer des opérations arithmétiques telles que l'addition, la soustraction, la multiplication et la division sur les types de données numériques. Voici quelques exemples d'opérations arithmétiques en Java :

int a = 5;
int b = 3;

int sum = a + b; // Addition
int difference = a - b; // Soustraction
int product = a * b; // Multiplication
int quotient = a / b; // Division
int remainder = a % b; // Reste de la division

Les opérateurs de comparaison permettent de comparer des valeurs et de renvoyer un résultat booléen (vrai ou faux). Voici quelques exemples d'opérateurs de comparaison en Java :

int a = 5;
int b = 3;

boolean isEqual = (a == b); // Égalité
boolean isNotEqual = (a != b); // Différence
boolean isGreater = (a > b); // Supérieur
boolean isLess = (a < b); // Inférieur
boolean isGreaterOrEqual = (a >= b); // Supérieur ou égal
boolean isLessOrEqual = (a <= b); // Inférieur ou égal

Conversion de types de données

En Java, vous pouvez convertir un type de données en un autre type de données compatible. Il existe deux types de conversions : la conversion implicite et la conversion explicite.

La conversion implicite se produit lorsque vous affectez une valeur d'un type de données plus petit à un type de données plus grand. Par exemple :

int a = 5;
double b = a; // Conversion implicite de int à double

La conversion explicite, également appelée "cast", se produit lorsque vous forcez une conversion d'un type de données à un autre type de données. Par exemple :

double a = 3.14;
int b = (int) a; // Conversion explicite de double à int

Il est important de noter que certaines conversions peuvent entraîner une perte de précision ou une erreur si les types de données ne sont pas compatibles.
