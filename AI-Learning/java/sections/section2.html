Les opérateurs en Java

Les opérateurs en Java sont des symboles spéciaux qui effectuent des opérations sur les variables et les valeurs. Ils peuvent être utilisés pour effectuer des calculs mathématiques, des opérations logiques et des manipulations de bits.

Voici quelques exemples d'opérateurs couramment utilisés en Java :

  • L'opérateur d'addition (+) : utilisé pour ajouter deux valeurs.
  • L'opérateur de soustraction (-) : utilisé pour soustraire une valeur d'une autre.
  • L'opérateur de multiplication (*) : utilisé pour multiplier deux valeurs.
  • L'opérateur de division (/) : utilisé pour diviser une valeur par une autre.
  • L'opérateur de modulo (%) : utilisé pour obtenir le reste d'une division.
  • L'opérateur d'incrémentation (++) : utilisé pour augmenter la valeur d'une variable de 1.
  • L'opérateur de décrémentation (--) : utilisé pour diminuer la valeur d'une variable de 1.

Il existe également des opérateurs de comparaison (==, !=, >, <, >=, <=) qui permettent de comparer des valeurs, ainsi que des opérateurs logiques (&&, ||, !) qui permettent de combiner des conditions.

Les structures de contrôle en Java (boucles, conditions)

Les structures de contrôle en Java permettent de contrôler le flux d'exécution d'un programme. Elles permettent d'exécuter certaines instructions uniquement si une condition est remplie, ou de répéter un bloc d'instructions plusieurs fois.

Les structures de contrôle les plus couramment utilisées en Java sont :

  • La structure conditionnelle if : permet d'exécuter un bloc d'instructions uniquement si une condition est vraie.
  • La structure conditionnelle if-else : permet d'exécuter un bloc d'instructions si une condition est vraie, et un autre bloc d'instructions si la condition est fausse.
  • La structure conditionnelle switch : permet d'exécuter différentes actions en fonction de la valeur d'une variable.
  • Les boucles for, while et do-while : permettent de répéter un bloc d'instructions plusieurs fois.

Voici un exemple d'utilisation d'une boucle for :

for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

Ce code affiche les nombres de 0 à 9 en utilisant une boucle for.

Les tableaux en Java

Un tableau en Java est une structure de données qui permet de stocker plusieurs valeurs de même type dans une seule variable. Chaque valeur dans le tableau est identifiée par un index, qui commence à 0 pour le premier élément.

Voici comment déclarer et initialiser un tableau en Java :

int[] numbers = new int[5];
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;

Ce code crée un tableau d'entiers de taille 5 et y stocke les valeurs de 1 à 5.

Il est également possible d'initialiser un tableau lors de sa déclaration :

int[] numbers = {1, 2, 3, 4, 5};

Ce code crée un tableau d'entiers et y stocke les mêmes valeurs que précédemment.

Les fonctions et les méthodes en Java

Les fonctions et les méthodes en Java sont des blocs de code réutilisables qui effectuent une tâche spécifique. Ils permettent d'organiser le code en le divisant en petites parties, ce qui facilite sa compréhension et sa maintenance.

Voici comment déclarer et appeler une fonction en Java :

public int add(int a, int b) {
    return a + b;
}

int result = add(3, 4);
System.out.println(result);

Ce code déclare une fonction add qui prend deux entiers en paramètres et renvoie leur somme. La fonction est ensuite appelée avec les valeurs 3 et 4, et le résultat est affiché.

La gestion des exceptions en Java

Les exceptions en Java sont des événements qui se produisent pendant l'exécution d'un programme et qui interrompent le flux normal d'exécution. Elles permettent de gérer les erreurs et les situations exceptionnelles de manière contrôlée.

Voici comment gérer une exception en Java :

try {
    // Code susceptible de générer une exception
} catch (Exception e) {
    // Code exécuté en cas d'exception
}

Ce code entoure le bloc de code susceptible de générer une exception avec une instruction try. Si une exception se produit, le code à l'intérieur du bloc catch est exécuté.

Il est également possible de gérer plusieurs types d'exceptions différentes en utilisant plusieurs blocs catch :

try {
    // Code susceptible de générer une exception
} catch (IOException e) {
    // Code exécuté en cas d'exception de type IOException
} catch (SQLException e) {
    // Code exécuté en cas d'exception de type SQLException
}

Ce code gère à la fois les exceptions de type IOException et SQLException.
