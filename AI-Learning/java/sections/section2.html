Section 2 : Les fondamentaux de la programmation Java

Les structures de contrôle (boucles, conditions)

Les structures de contrôle sont essentielles en programmation Java pour contrôler le flux d'exécution du programme. Les boucles permettent de répéter des instructions plusieurs fois, tandis que les conditions permettent de prendre des décisions en fonction de certaines conditions.

Voici un exemple d'utilisation de la boucle while en Java :

i = 0;

while (i < 10) {
    System.out.println("Iteration " + i);
    i++;
}

Dans cet exemple, la boucle while répète les instructions à l'intérieur tant que la condition "i < 10" est vraie. À chaque itération, la variable i est incrémentée de 1.

Les conditions sont utilisées pour exécuter des blocs de code en fonction de certaines conditions. Voici un exemple d'utilisation de la condition if en Java :

int age = 18;

if (age >= 18) {
    System.out.println("Vous êtes majeur.");
} else {
    System.out.println("Vous êtes mineur.");
}

Dans cet exemple, le bloc de code à l'intérieur de la condition if est exécuté si la condition "age >= 18" est vraie. Sinon, le bloc de code à l'intérieur de la condition else est exécuté.

Les tableaux en Java

Les tableaux en Java permettent de stocker plusieurs valeurs de même type dans une seule variable. Ils sont utiles lorsque vous avez besoin de manipuler un ensemble de données.

Voici un exemple de déclaration et d'utilisation d'un tableau en Java :

int[] numbers = {1, 2, 3, 4, 5};

for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

Dans cet exemple, nous déclarons un tableau d'entiers appelé "numbers" et lui attribuons les valeurs 1, 2, 3, 4 et 5. Ensuite, nous utilisons une boucle for pour parcourir chaque élément du tableau et l'afficher.

Les méthodes et les fonctions en Java

Les méthodes et les fonctions en Java permettent d'organiser le code en le divisant en blocs réutilisables. Ils sont utilisés pour effectuer des actions spécifiques et peuvent prendre des paramètres en entrée et renvoyer des valeurs en sortie.

Voici un exemple de déclaration et d'appel d'une méthode en Java :

public int addNumbers(int a, int b) {
    return a + b;
}

int result = addNumbers(5, 3);
System.out.println(result);

Dans cet exemple, nous déclarons une méthode appelée "addNumbers" qui prend deux paramètres entiers "a" et "b" et renvoie leur somme. Ensuite, nous appelons cette méthode avec les valeurs 5 et 3, et stockons le résultat dans la variable "result" que nous affichons ensuite.

La gestion des exceptions en Java

La gestion des exceptions en Java permet de gérer les erreurs et les situations exceptionnelles qui peuvent survenir pendant l'exécution d'un programme. Elle permet de capturer et de traiter ces exceptions de manière appropriée.

Voici un exemple de gestion d'exception en Java :

try {
    int result = 10 / 0;
    System.out.println(result);
} catch (ArithmeticException e) {
    System.out.println("Une erreur de division par zéro s'est produite.");
}

Dans cet exemple, nous essayons de diviser 10 par 0, ce qui provoque une exception de type ArithmeticException. Nous capturons cette exception à l'aide du bloc try-catch et affichons un message d'erreur approprié.

Les classes et les objets en Java

Les classes et les objets sont les éléments fondamentaux de la programmation orientée objet en Java. Les classes sont des modèles ou des plans pour créer des objets, qui sont des instances de ces classes.

Voici un exemple de déclaration d'une classe et de création d'un objet en Java :

public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void sayHello() {
        System.out.println("Bonjour, je m'appelle " + name + " et j'ai " + age + " ans.");
    }
}

Person person = new Person("Alice", 25);
person.sayHello();

Dans cet exemple, nous déclarons une classe appelée "Person" avec deux attributs privés "name" et "age". Nous avons également un constructeur pour initialiser ces attributs et une méthode "sayHello" pour afficher un message de salutation. Ensuite, nous créons un objet de type "Person" avec le nom "Alice" et l'âge 25, et appelons la méthode "sayHello" sur cet objet.
