Introduction à Swing

Swing est une bibliothèque graphique pour Java qui permet de créer des interfaces graphiques (GUI) pour les applications Java. Elle offre une grande variété de composants graphiques tels que des boutons, des champs de texte, des listes déroulantes, des tableaux, etc. Swing est basé sur le modèle de conception MVC (Modèle-Vue-Contrôleur) et est inclus dans le package javax.swing.

Création d'interfaces graphiques

Pour créer une interface graphique avec Swing, vous devez utiliser des conteneurs tels que JFrame, JPanel, JDialog, etc. Ces conteneurs peuvent contenir d'autres composants graphiques tels que des boutons, des champs de texte, des listes, etc. Vous pouvez utiliser des gestionnaires de disposition (layout managers) pour organiser les composants à l'intérieur des conteneurs.

Voici un exemple de création d'une fenêtre avec un bouton en utilisant JFrame : 


import javax.swing.*;

public class MaFenetre extends JFrame {
    public MaFenetre() {
        JButton bouton = new JButton("Cliquez ici");
        getContentPane().add(bouton);
        
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        pack();
        setVisible(true);
    }
    
    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> new MaFenetre());
    }
}

Gestion des événements

Swing permet de gérer les événements générés par les composants graphiques. Vous pouvez associer des écouteurs d'événements (event listeners) à des composants spécifiques pour réagir aux actions de l'utilisateur, telles que le clic sur un bouton ou la saisie de texte dans un champ de texte.

Voici un exemple de gestion d'événements avec un bouton en utilisant ActionListener : 


import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class MaFenetre extends JFrame {
    public MaFenetre() {
        JButton bouton = new JButton("Cliquez ici");
        bouton.addActionListener(new ActionListener() {
            public void actionPerformed(ActionEvent e) {
                JOptionPane.showMessageDialog(null, "Vous avez cliqué sur le bouton !");
            }
        });
        
        getContentPane().add(bouton);
        
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        pack();
        setVisible(true);
    }
    
    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> new MaFenetre());
    }
}

Dans cet exemple, nous avons créé un ActionListener anonyme qui affiche une boîte de dialogue lorsque le bouton est cliqué.

Ceci conclut la section sur la programmation graphique avec Swing. Vous avez appris comment créer des interfaces graphiques, organiser les composants et gérer les événements. Continuez à pratiquer et à explorer les fonctionnalités avancées de Swing pour créer des applications Java avec des interfaces utilisateur attrayantes et interactives.
