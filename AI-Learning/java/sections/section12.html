Projet 1 : Création d'une application de gestion de bibliothèque

Dans ce projet, vous allez créer une application de gestion de bibliothèque en Java. Cette application permettra aux utilisateurs de gérer les livres de leur bibliothèque, d'ajouter de nouveaux livres, de les emprunter et de les rendre. Voici les fonctionnalités principales de l'application :

  • Affichage de la liste des livres disponibles
  • Ajout d'un nouveau livre à la bibliothèque
  • Emprunt d'un livre
  • Retour d'un livre emprunté

Pour implémenter cette application, vous allez utiliser les concepts de base de la programmation orientée objet en Java. Vous créerez une classe Bibliothèque qui représentera la bibliothèque et une classe Livre qui représentera un livre. public class Bibliothèque {

    }

    public void emprunterLivre(int index) {
        // Emprunter le livre à l'index spécifié
    }

    public void rendreLivre(int index) {
        // Rendre le livre à l'index spécifié
    }

    public void afficherLivresDisponibles() {
        // Afficher la liste des livres disponibles
    }
}

Vous devrez également implémenter la classe Livre avec les attributs appropriés tels que le titre, l'auteur, etc., ainsi que les méthodes nécessaires pour accéder à ces attributs.

Projet 2 : Développement d'un jeu en Java

Dans ce projet, vous allez développer un jeu en Java. Le jeu sera un jeu de devinettes où l'utilisateur devra deviner un nombre choisi aléatoirement par l'ordinateur. Voici les fonctionnalités principales du jeu :

  • Génération d'un nombre aléatoire
  • Demande à l'utilisateur de deviner le nombre
  • Comparaison de la réponse de l'utilisateur avec le nombre généré
  • Affichage d'un message indiquant si la réponse est correcte ou non

Pour implémenter ce jeu, vous utiliserez les structures de contrôle en Java telles que les boucles et les conditions. Vous utiliserez également des fonctions pour générer le nombre aléatoire et pour comparer la réponse de l'utilisateur avec le nombre généré.

Voici un exemple de code pour la fonction de comparaison :

public void comparerReponse(int reponseUtilisateur, int nombreGenere) {
    if (reponseUtilisateur == nombreGenere) {
        System.out.println("Bravo, vous avez deviné le nombre !");
    } else if (reponseUtilisateur < nombreGenere) {
        System.out.println("Le nombre à deviner est plus grand.");
    } else {
        System.out.println("Le nombre à deviner est plus petit.");
    }
}

Vous devrez également ajouter une boucle pour permettre à l'utilisateur de continuer à deviner jusqu'à ce qu'il trouve la bonne réponse.
