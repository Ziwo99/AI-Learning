Section 6 : Les entrées/sorties en Java

Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en programmation Java. Pour lire un fichier, vous pouvez utiliser la classe FileReader et pour écrire dans un fichier, vous pouvez utiliser la classe FileWriter. Voici un exemple de code qui lit le contenu d'un fichier :

import java.io.FileReader;
import java.io.IOException;

public class FileExample {
    public static void main(String[] args) {
        try {
            FileReader reader = new FileReader("example.txt");
            int character;
            while ((character = reader.read()) != -1) {
                System.out.print((char) character);
            }
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons la méthode read() de la classe FileReader pour lire chaque caractère du fichier. La boucle se poursuit tant qu'il y a des caractères à lire. Une fois la lecture terminée, nous fermons le lecteur en utilisant la méthode close().

Pour écrire dans un fichier, vous pouvez utiliser la classe FileWriter. Voici un exemple de code qui écrit du texte dans un fichier :

import java.io.FileWriter;
import java.io.IOException;

public class FileExample {
    public static void main(String[] args) {
        try {
            FileWriter writer = new FileWriter("example.txt");
            writer.write("Hello, world!");
            writer.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons la méthode write() de la classe FileWriter pour écrire le texte "Hello, world!" dans le fichier. Ensuite, nous fermons le writer en utilisant la méthode close().

Utilisation des flux d'entrée/sortie

En plus des classes FileReader et FileWriter, Java fournit également des classes de flux d'entrée/sortie pour lire et écrire des données à partir de différentes sources, telles que le clavier, les fichiers, les sockets réseau, etc. Voici quelques-unes des classes de flux d'entrée/sortie les plus couramment utilisées :

  • InputStream : utilisé pour lire des données binaires à partir d'une source.
  • OutputStream : utilisé pour écrire des données binaires vers une destination.
  • Reader : utilisé pour lire des données caractère à partir d'une source.
  • Writer : utilisé pour écrire des données caractère vers une destination.

Voici un exemple de code qui utilise la classe BufferedReader pour lire une ligne de texte à partir de l'entrée standard (clavier) :

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class ConsoleExample {
    public static void main(String[] args) {
        try {
            BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
            System.out.print("Enter your name: ");
            String name = reader.readLine();
            System.out.println("Hello, " + name + "!");
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons la classe BufferedReader pour lire une ligne de texte à partir de l'entrée standard (clavier). La méthode readLine() lit une ligne de texte et la retourne en tant que chaîne de caractères. Ensuite, nous affichons un message de salutation avec le nom saisi.

Sérialisation des objets

La sérialisation des objets en Java permet de convertir un objet en une séquence d'octets, qui peut ensuite être enregistrée dans un fichier ou transmise sur le réseau. Pour sérialiser un objet, vous pouvez utiliser la classe ObjectOutputStream, et pour désérialiser un objet, vous pouvez utiliser la classe ObjectInputStream. Voici un exemple de code :

import java.io.*;

public class SerializationExample {
    public static void main(String[] args) {
        try {
            // Sérialisation
            ObjectOutputStream outputStream = new ObjectOutputStream(new FileOutputStream("object.ser"));
            MyObject object = new MyObject("Hello, world!");
            outputStream.writeObject(object);
            outputStream.close();

            // Désérialisation
            ObjectInputStream inputStream = new ObjectInputStream(new FileInputStream("object.ser"));
            MyObject deserializedObject = (MyObject) inputStream.readObject();
            inputStream.close();

            System.out.println(deserializedObject.getMessage());
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }
    }
}

class MyObject implements Serializable {
    private String message;

    public MyObject(String message) {
        this.message = message;
    }

    public String getMessage() {
        return message;
    }
}

Dans cet exemple, nous sérialisons un objet de la classe MyObject en utilisant la classe ObjectOutputStream. Nous écrivons ensuite l'objet sérialisé dans un fichier. Pour désérialiser l'objet, nous utilisons la classe ObjectInputStream et nous effectuons un cast vers la classe MyObject. Enfin, nous affichons le message de l'objet désérialisé.

Il est important de noter que les classes que vous souhaitez sérialiser doivent implémenter l'interface Serializable.
