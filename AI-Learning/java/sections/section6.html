Section 6 : Les bases de données en Java

Les concepts de base des bases de données

Les bases de données sont des outils essentiels pour stocker et gérer de grandes quantités de données de manière structurée. En Java, nous utilisons le langage SQL (Structured Query Language) pour interagir avec les bases de données. Voici quelques concepts de base à connaître :

  • Tables : Les données sont stockées dans des tables, qui sont des structures organisées en lignes et colonnes.
  • Colonnes : Chaque colonne représente un attribut spécifique des données, tel que le nom, l'âge ou l'adresse.
  • Lignes : Chaque ligne dans une table représente un enregistrement unique contenant les valeurs des attributs.
  • Clés primaires : Chaque table a une clé primaire, qui est une colonne unique utilisée pour identifier de manière unique chaque enregistrement dans la table.
  • Clés étrangères : Les clés étrangères sont utilisées pour établir des relations entre les tables en reliant les clés primaires et étrangères.

La connexion à une base de données en Java

Pour se connecter à une base de données en Java, nous utilisons la bibliothèque JDBC (Java Database Connectivity). Voici les étapes pour établir une connexion :

  1. Charger le pilote JDBC approprié en utilisant la méthode Class.forName().
  2. Créer une instance de la classe Connection en utilisant la méthode DriverManager.getConnection() et en fournissant les informations de connexion telles que l'URL de la base de données, le nom d'utilisateur et le mot de passe.
  3. Utiliser la connexion pour exécuter des requêtes SQL et récupérer les résultats.
  4. Fermer la connexion en utilisant la méthode Connection.close() une fois que vous avez terminé.

Voici un exemple de code pour se connecter à une base de données MySQL :

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DatabaseConnection {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "root";
        String password = "password";

        try {
            Class.forName("com.mysql.jdbc.Driver");
            Connection connection = DriverManager.getConnection(url, username, password);
            System.out.println("Connected to the database!");
            connection.close();
        } catch (ClassNotFoundException | SQLException e) {
            e.printStackTrace();
        }
    }
}

Les requêtes SQL en Java

Les requêtes SQL sont utilisées pour interagir avec les bases de données et récupérer des données spécifiques. En Java, nous utilisons la classe Statement pour exécuter des requêtes SQL. Voici quelques exemples de requêtes couramment utilisées :

  • SELECT : Utilisé pour récupérer des données spécifiques à partir d'une table.
  • INSERT : Utilisé pour insérer de nouvelles lignes dans une table.
  • UPDATE : Utilisé pour mettre à jour les valeurs des attributs dans une table.
  • DELETE : Utilisé pour supprimer des lignes d'une table.

Voici un exemple de code pour exécuter une requête SELECT :

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.SQLException;
import java.sql.Statement;

public class DatabaseQuery {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "root";
        String password = "password";

        try {
            Class.forName("com.mysql.jdbc.Driver");
            Connection connection = DriverManager.getConnection(url, username, password);
            Statement statement = connection.createStatement();
            String query = "SELECT * FROM employees";
            ResultSet resultSet = statement.executeQuery(query);

            while (resultSet.next()) {
                int id = resultSet.getInt("id");
                String name = resultSet.getString("name");
                int age = resultSet.getInt("age");
                System.out.println("ID: " + id + ", Name: " + name + ", Age: " + age);
            }

            resultSet.close();
            statement.close();
            connection.close();
        } catch (ClassNotFoundException | SQLException e) {
            e.printStackTrace();
        }
    }
}

La manipulation des données en Java

En Java, nous pouvons manipuler les données dans une base de données en utilisant des requêtes SQL. Voici quelques exemples de manipulations courantes :

  • Ajouter une nouvelle ligne à une table en utilisant une requête INSERT.
  • Mettre à jour les valeurs des attributs dans une table en utilisant une requête UPDATE.
  • Supprimer des lignes d'une table en utilisant une requête DELETE.

Voici un exemple de code pour insérer une nouvelle ligne dans une table :

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import java.sql.Statement;

public class DatabaseInsert {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "root";
        String password = "password";

        try {
            Class.forName("com.mysql.jdbc.Driver");
            Connection connection = DriverManager.getConnection(url, username, password);
            Statement statement = connection.createStatement();
            String query = "INSERT INTO employees (name, age) VALUES ('John Doe', 30)";
            int rowsAffected = statement.executeUpdate(query);
            System.out.println(rowsAffected + " row(s) inserted.");

            statement.close();
            connection.close();
        } catch (ClassNotFoundException | SQLException e) {
            e.printStackTrace();
        }
    }
}

La gestion des transactions en Java

Les transactions sont utilisées pour garantir l'intégrité des données dans une base de données. En Java, nous pouvons gérer les transactions en utilisant les méthodes commit() et rollback() de la classe Connection. Voici comment utiliser les transactions :

  1. Définir le mode de transaction en utilisant la méthode setAutoCommit(false) de la classe Connection.
  2. Exécuter les requêtes SQL dans la transaction.
  3. Appeler la méthode commit() pour valider les modifications ou rollback() pour annuler les modifications.
  4. Activer le mode de transaction automatique en utilisant la méthode setAutoCommit(true).

Voici un exemple de code pour gérer une transaction :

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import java.sql.Statement;

public class DatabaseTransaction {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "root";
        String password = "password";

        try {
            Class.forName("com.mysql.jdbc.Driver");
            Connection connection = DriverManager.getConnection(url, username, password);
            connection.setAutoCommit(false);

            Statement statement = connection.createStatement();
            String query1 = "INSERT INTO employees (name, age) VALUES ('John Doe', 30)";
            String query2 = "UPDATE employees SET age = 31 WHERE name = 'John Doe'";

            statement.executeUpdate(query1);
            statement.executeUpdate(query2);

            connection.commit();
            System.out.println("Transaction committed successfully.");

            statement.close();
            connection.setAutoCommit(true);
            connection.close();
        } catch (ClassNotFoundException | SQLException e) {
            e.printStackTrace();
        }
    }
}
