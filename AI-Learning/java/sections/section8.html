Lecture et écriture de fichiers

La lecture et l'écriture de fichiers sont des opérations courantes en programmation Java. Pour lire ou écrire des données à partir d'un fichier, vous devez d'abord créer un objet de type File pour représenter le fichier lui-même. Ensuite, vous pouvez utiliser des classes telles que FileReader, FileWriter, BufferedReader et BufferedWriter pour effectuer les opérations de lecture et d'écriture.

Voici un exemple de code qui lit le contenu d'un fichier ligne par ligne :

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReaderExample {
    public static void main(String[] args) {
        try {
            FileReader fileReader = new FileReader("example.txt");
            BufferedReader bufferedReader = new BufferedReader(fileReader);

            String line;
            while ((line = bufferedReader.readLine()) != null) {
                System.out.println(line);
            }

            bufferedReader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons FileReader pour lire le fichier "example.txt" et BufferedReader pour lire les lignes une par une à l'aide de la méthode readLine(). La boucle while se poursuit tant que la méthode readLine() renvoie une ligne non nulle.

Voici un exemple de code qui écrit du contenu dans un fichier :

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class FileWriterExample {
    public static void main(String[] args) {
        try {
            FileWriter fileWriter = new FileWriter("example.txt");
            BufferedWriter bufferedWriter = new BufferedWriter(fileWriter);

            bufferedWriter.write("Hello, world!");
            bufferedWriter.newLine();
            bufferedWriter.write("This is an example.");

            bufferedWriter.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons FileWriter pour écrire dans le fichier "example.txt" et BufferedWriter pour écrire les lignes une par une à l'aide de la méthode write(). La méthode newLine() est utilisée pour insérer une nouvelle ligne entre les lignes écrites.

Flux d'entrée/sortie

Les flux d'entrée/sortie sont utilisés pour lire et écrire des données binaires à partir de fichiers ou d'autres sources/sinks de données. En Java, les flux d'entrée/sortie sont représentés par les classes InputStream et OutputStream.

Voici un exemple de code qui lit des données binaires à partir d'un fichier :

import java.io.FileInputStream;
import java.io.IOException;

public class FileInputStreamExample {
    public static void main(String[] args) {
        try {
            FileInputStream fileInputStream = new FileInputStream("example.bin");

            int data;
            while ((data = fileInputStream.read()) != -1) {
                System.out.println(data);
            }

            fileInputStream.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons FileInputStream pour lire le fichier binaire "example.bin". La méthode read() renvoie un entier représentant le prochain octet de données lu à partir du fichier. La boucle while se poursuit tant que la méthode read() renvoie une valeur différente de -1, indiquant la fin du fichier.

Voici un exemple de code qui écrit des données binaires dans un fichier :

import java.io.FileOutputStream;
import java.io.IOException;

public class FileOutputStreamExample {
    public static void main(String[] args) {
        try {
            FileOutputStream fileOutputStream = new FileOutputStream("example.bin");

            byte[] data = { 0x48, 0x65, 0x6C, 0x6C, 0x6F, 0x2C, 0x20, 0x77, 0x6F, 0x72, 0x6C, 0x64 };
            fileOutputStream.write(data);

            fileOutputStream.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons FileOutputStream pour écrire des données binaires dans le fichier "example.bin". La méthode write() est utilisée pour écrire les octets de données spécifiés dans le tableau byte[].

Sérialisation d'objets

La sérialisation d'objets est le processus de conversion d'un objet en une séquence d'octets, qui peut ensuite être enregistrée dans un fichier ou transmise sur un réseau. En Java, la sérialisation d'objets est prise en charge par les classes ObjectOutputStream et ObjectInputStream.

Voici un exemple de code qui sérialise un objet dans un fichier :

import java.io.FileOutputStream;
import java.io.IOException;
import java.io.ObjectOutputStream;
import java.io.Serializable;

class Person implements Serializable {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }
}

public class ObjectSerializationExample {
    public static void main(String[] args) {
        try {
            FileOutputStream fileOutputStream = new FileOutputStream("person.ser");
            ObjectOutputStream objectOutputStream = new ObjectOutputStream(fileOutputStream);

            Person person = new Person("John Doe", 30);
            objectOutputStream.writeObject(person);

            objectOutputStream.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons ObjectOutputStream pour sérialiser l'objet Person dans le fichier "person.ser". La méthode writeObject() est utilisée pour écrire l'objet sérialisé dans le fichier.

Voici un exemple de code qui désérialise un objet à partir d'un fichier :

import java.io.FileInputStream;
import java.io.IOException;
import java.io.ObjectInputStream;

public class ObjectDeserializationExample {
    public static void main(String[] args) {
        try {
            FileInputStream fileInputStream = new FileInputStream("person.ser");
            ObjectInputStream objectInputStream = new ObjectInputStream(fileInputStream);

            Person person = (Person) objectInputStream.readObject();
            System.out.println("Name: " + person.getName());
            System.out.println("Age: " + person.getAge());

            objectInputStream.close();
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons ObjectInputStream pour désérialiser l'objet Person à partir du fichier "person.ser". La méthode readObject() est utilisée pour lire l'objet sérialisé à partir du fichier. Nous devons ensuite effectuer un cast de l'objet lu en tant qu'instance de la classe Person.
