Les threads en Java

Les threads en Java permettent d'exécuter plusieurs tâches simultanément. Un thread est une unité d'exécution indépendante qui peut être exécutée en parallèle avec d'autres threads.

Voici un exemple de création et d'exécution d'un thread en Java :

public class MyThread extends Thread {
    public void run() {
        // Code à exécuter dans le thread
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread thread = new MyThread();
        thread.start(); // Démarre le thread
    }
}

La concurrence en Java

La concurrence en Java concerne la gestion de l'accès simultané à des ressources partagées par plusieurs threads. Java fournit des mécanismes tels que les verrous et les sémaphores pour garantir la cohérence des données partagées.

Voici un exemple d'utilisation d'un verrou (lock) en Java :

import java.util.concurrent.locks.Lock;
import java.util.concurrent.locks.ReentrantLock;

public class Counter {
    private int count = 0;
    private Lock lock = new ReentrantLock();

    public void increment() {
        lock.lock();
        try {
            count++;
        } finally {
            lock.unlock();
        }
    }
}

Les annotations en Java

Les annotations en Java permettent d'ajouter des métadonnées à du code source. Elles peuvent être utilisées pour spécifier des informations supplémentaires sur des classes, des méthodes, des variables, etc. Les annotations peuvent être utilisées par le compilateur ou par d'autres outils de développement.

Voici un exemple d'utilisation de l'annotation @Override en Java :

public class MyClass {
    @Override
    public String toString() {
        return "This is my class";
    }
}

Les expressions régulières en Java

Les expressions régulières en Java permettent de rechercher et de manipuler des chaînes de caractères en utilisant des motifs spécifiques. Elles sont très utiles pour valider des formats de données, extraire des informations d'une chaîne, etc.

Voici un exemple d'utilisation d'une expression régulière en Java :

import java.util.regex.Pattern;
import java.util.regex.Matcher;

public class RegexExample {
    public static void main(String[] args) {
        String input = "Hello, my email is example@example.com";
        String pattern = "\\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\\.[A-Za-z]{2,}\\b";

        Pattern regex = Pattern.compile(pattern);
        Matcher matcher = regex.matcher(input);

        while (matcher.find()) {
            System.out.println("Email found: " + matcher.group());
        }
    }
}

La programmation fonctionnelle en Java

La programmation fonctionnelle en Java permet d'utiliser des fonctions comme des objets de première classe. Elle introduit des concepts tels que les fonctions lambda, les interfaces fonctionnelles et les méthodes de référence.

Voici un exemple d'utilisation d'une fonction lambda en Java :

import java.util.Arrays;
import java.util.List;

public class FunctionalExample {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

        numbers.forEach((Integer number) -> {
            System.out.println(number * 2);
        });
    }
}

Ce code affichera le double de chaque nombre de la liste.
