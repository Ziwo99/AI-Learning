Les flux d'entrée/sortie en Java

Les flux d'entrée/sortie en Java sont utilisés pour lire et écrire des données à partir de différentes sources et destinations. Les flux d'entrée sont utilisés pour lire des données à partir d'une source, tandis que les flux de sortie sont utilisés pour écrire des données vers une destination.

En Java, les flux d'entrée/sortie sont représentés par les classes InputStream et OutputStream. Ces classes abstraites fournissent les méthodes de base pour lire et écrire des données.

Voici un exemple de lecture d'un fichier texte à l'aide d'un flux d'entrée :

try {
    InputStream input = new FileInputStream("fichier.txt");
    int data = input.read();
    while (data != -1) {
        System.out.print((char) data);
        data = input.read();
    }
    input.close();
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous créons un objet FileInputStream en spécifiant le nom du fichier à lire. Ensuite, nous utilisons la méthode read() pour lire les données du fichier octet par octet. La méthode read() renvoie -1 lorsque la fin du fichier est atteinte.

Il est important de noter que les flux d'entrée/sortie doivent être fermés après utilisation à l'aide de la méthode close() pour libérer les ressources système.

La sérialisation en Java

La sérialisation en Java est le processus de conversion d'un objet en une séquence d'octets, qui peut être ensuite enregistrée dans un fichier ou transmise via un réseau. La désérialisation est le processus inverse, où la séquence d'octets est utilisée pour recréer l'objet d'origine.

En Java, la sérialisation est prise en charge par l'interface Serializable. Pour qu'un objet puisse être sérialisé, il doit implémenter cette interface. Voici un exemple :

import java.io.*;

public class Personne implements Serializable {
    private String nom;
    private int age;

    // constructeur, getters, setters...

    public static void main(String[] args) {
        Personne personne = new Personne("John Doe", 30);

        try {
            FileOutputStream fileOut = new FileOutputStream("personne.ser");
            ObjectOutputStream out = new ObjectOutputStream(fileOut);
            out.writeObject(personne);
            out.close();
            fileOut.close();
            System.out.println("L'objet a été sérialisé avec succès.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous créons un objet Personne et le sérialisons en l'enregistrant dans un fichier "personne.ser". Nous utilisons les classes FileOutputStream et ObjectOutputStream pour écrire l'objet sérialisé dans le fichier.

Pour désérialiser un objet, nous utilisons les classes FileInputStream et ObjectInputStream :

try {
    FileInputStream fileIn = new FileInputStream("personne.ser");
    ObjectInputStream in = new ObjectInputStream(fileIn);
    Personne personne = (Personne) in.readObject();
    in.close();
    fileIn.close();
    System.out.println("L'objet a été désérialisé avec succès.");
} catch (IOException e) {
    e.printStackTrace();
} catch (ClassNotFoundException e) {
    e.printStackTrace();
}

Dans cet exemple, nous lisons l'objet sérialisé à partir du fichier "personne.ser" et le convertissons en un objet de type Personne en utilisant la méthode readObject().

La gestion des fichiers en Java

En Java, la gestion des fichiers est effectuée à l'aide des classes File et FileWriter/FileReader. La classe File est utilisée pour représenter un fichier ou un répertoire, tandis que les classes FileWriter et FileReader sont utilisées pour écrire et lire des données à partir d'un fichier texte.

Voici un exemple de création d'un fichier et d'écriture de données :

try {
    File file = new File("fichier.txt");
    FileWriter writer = new FileWriter(file);
    writer.write("Hello, World!");
    writer.close();
    System.out.println("Les données ont été écrites avec succès dans le fichier.");
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous créons un objet File en spécifiant le nom du fichier à créer. Ensuite, nous utilisons la classe FileWriter pour écrire les données dans le fichier à l'aide de la méthode write(). Enfin, nous fermons le flux d'écriture à l'aide de la méthode close().

Voici un exemple de lecture des données à partir d'un fichier :

try {
    File file = new File("fichier.txt");
    FileReader reader = new FileReader(file);
    int data = reader.read();
    while (data != -1) {
        System.out.print((char) data);
        data = reader.read();
    }
    reader.close();
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous créons un objet FileReader en spécifiant le nom du fichier à lire. Ensuite, nous utilisons la méthode read() pour lire les données du fichier caractère par caractère.

Les entrées/sorties réseau en Java

En Java, les entrées/sorties réseau sont gérées à l'aide des classes Socket et ServerSocket. La classe Socket est utilisée pour établir une connexion avec un serveur distant, tandis que la classe ServerSocket est utilisée pour écouter les connexions entrantes sur un port spécifique.

Voici un exemple de client TCP qui se connecte à un serveur distant :

try {
    Socket socket = new Socket("localhost", 8080);
    OutputStream output = socket.getOutputStream();
    output.write("Hello, Server!".getBytes());
    output.close();
    socket.close();
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous créons un objet Socket en spécifiant l'adresse IP et le port du serveur distant. Ensuite, nous utilisons la méthode getOutputStream() pour obtenir le flux de sortie et écrire les données vers le serveur à l'aide de la méthode write(). Enfin, nous fermons le flux de sortie et la connexion à l'aide des méthodes close().

Voici un exemple de serveur TCP qui écoute les connexions entrantes :

try {
    ServerSocket serverSocket = new ServerSocket(8080);
    Socket socket = serverSocket.accept();
    InputStream input = socket.getInputStream();
    int data = input.read();
    while (data != -1) {
        System.out.print((char) data);
        data = input.read();
    }
    input.close();
    socket.close();
    serverSocket.close();
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous créons un objet ServerSocket en spécifiant le port sur lequel le serveur doit écouter les connexions entrantes. Ensuite, nous utilisons la méthode accept() pour attendre une connexion entrante. Une fois la connexion établie, nous utilisons la méthode getInputStream() pour obtenir le flux d'entrée et lire les données du client à l'aide de la méthode read(). Enfin, nous fermons le flux d'entrée, la connexion et le serveur à l'aide des méthodes close().
