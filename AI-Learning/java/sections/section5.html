Section 5 : Les entrées/sorties en Java

Les flux d'entrée/sortie en Java

Les flux d'entrée/sortie en Java permettent de lire et d'écrire des données à partir de différentes sources, telles que des fichiers, des sockets réseau, etc. Ils sont essentiels pour la manipulation des données dans les programmes Java.

Pour lire des données à partir d'une source, on utilise un flux d'entrée. Pour écrire des données vers une destination, on utilise un flux de sortie. Les flux d'entrée/sortie sont représentés par les classes InputStream et OutputStream respectivement.

Voici un exemple de lecture de données à partir d'un fichier en utilisant un flux d'entrée :

try {
    InputStream inputStream = new FileInputStream("fichier.txt");
    int data = inputStream.read();
    while (data != -1) {
        // Traiter les données lues
        System.out.print((char) data);
        data = inputStream.read();
    }
    inputStream.close();
} catch (IOException e) {
    e.printStackTrace();
}

Et voici un exemple d'écriture de données vers un fichier en utilisant un flux de sortie :

try {
    OutputStream outputStream = new FileOutputStream("fichier.txt");
    String data = "Contenu à écrire dans le fichier";
    byte[] bytes = data.getBytes();
    outputStream.write(bytes);
    outputStream.close();
} catch (IOException e) {
    e.printStackTrace();
}

La sérialisation en Java

La sérialisation en Java permet de convertir un objet en une séquence d'octets, afin de pouvoir le stocker dans un fichier ou le transmettre sur un réseau. La désérialisation permet de reconstituer l'objet à partir de la séquence d'octets.

Pour sérialiser un objet, il doit implémenter l'interface Serializable. Voici un exemple :

import java.io.*;

public class Personne implements Serializable {
    private String nom;
    private int age;

    // Constructeur, getters, setters, etc.

    public static void main(String[] args) {
        Personne personne = new Personne("John Doe", 30);

        try {
            FileOutputStream fileOutputStream = new FileOutputStream("personne.ser");
            ObjectOutputStream objectOutputStream = new ObjectOutputStream(fileOutputStream);
            objectOutputStream.writeObject(personne);
            objectOutputStream.close();
            fileOutputStream.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Pour désérialiser un objet, on utilise le code suivant :

try {
    FileInputStream fileInputStream = new FileInputStream("personne.ser");
    ObjectInputStream objectInputStream = new ObjectInputStream(fileInputStream);
    Personne personne = (Personne) objectInputStream.readObject();
    objectInputStream.close();
    fileInputStream.close();
} catch (IOException | ClassNotFoundException e) {
    e.printStackTrace();
}

La gestion des fichiers en Java

La gestion des fichiers en Java permet de créer, supprimer, renommer, copier, déplacer, etc. des fichiers et des répertoires. Elle est réalisée en utilisant la classe File.

Voici un exemple de création d'un fichier :

File file = new File("fichier.txt");
try {
    if (file.createNewFile()) {
        System.out.println("Le fichier a été créé avec succès.");
    } else {
        System.out.println("Le fichier existe déjà.");
    }
} catch (IOException e) {
    e.printStackTrace();
}

Et voici un exemple de suppression d'un fichier :

File file = new File("fichier.txt");
if (file.delete()) {
    System.out.println("Le fichier a été supprimé avec succès.");
} else {
    System.out.println("Échec de la suppression du fichier.");
}

Les opérations sur les fichiers en Java

Les opérations sur les fichiers en Java permettent de lire, écrire, copier, déplacer, renommer, etc. des fichiers. Elles sont réalisées en utilisant les classes FileReader, FileWriter, BufferedReader, BufferedWriter, etc.

Voici un exemple de lecture d'un fichier ligne par ligne :

try {
    BufferedReader bufferedReader = new BufferedReader(new FileReader("fichier.txt"));
    String line = bufferedReader.readLine();
    while (line != null) {
        // Traiter la ligne lue
        System.out.println(line);
        line = bufferedReader.readLine();
    }
    bufferedReader.close();
} catch (IOException e) {
    e.printStackTrace();
}

Et voici un exemple d'écriture dans un fichier :

try {
    BufferedWriter bufferedWriter = new BufferedWriter(new FileWriter("fichier.txt"));
    bufferedWriter.write("Contenu à écrire dans le fichier");
    bufferedWriter.close();
} catch (IOException e) {
    e.printStackTrace();
}

Les flux de données en Java

Les flux de données en Java permettent de lire et d'écrire des données de types primitifs (int, double, etc.) et d'objets. Ils sont réalisés en utilisant les classes DataInputStream et DataOutputStream.

Voici un exemple de lecture de données à partir d'un fichier en utilisant un flux de données :

try {
    DataInputStream dataInputStream = new DataInputStream(new FileInputStream("fichier.dat"));
    int intValue = dataInputStream.readInt();
    double doubleValue = dataInputStream.readDouble();
    String stringValue = dataInputStream.readUTF();
    dataInputStream.close();
} catch (IOException e) {
    e.printStackTrace();
}

Et voici un exemple d'écriture de données vers un fichier en utilisant un flux de données :

try {
    DataOutputStream dataOutputStream = new DataOutputStream(new FileOutputStream("fichier.dat"));
    int intValue = 42;
    double doubleValue = 3.14;
    String stringValue = "Hello World";
    dataOutputStream.writeInt(intValue);
    dataOutputStream.writeDouble(doubleValue);
    dataOutputStream.writeUTF(stringValue);
    dataOutputStream.close();
} catch (IOException e) {
    e.printStackTrace();
}
