Les flux d'entrée/sortie en Java

En Java, les flux d'entrée/sortie (I/O) sont utilisés pour lire des données à partir d'une source (entrée) ou écrire des données vers une destination (sortie). Les flux d'entrée/sortie sont représentés par les classes InputStream et OutputStream.

Voici un exemple de code qui utilise un flux d'entrée pour lire des données à partir d'un fichier :

import java.io.FileInputStream;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try {
            FileInputStream inputStream = new FileInputStream("example.txt");
            int data;
            while ((data = inputStream.read()) != -1) {
                System.out.print((char) data);
            }
            inputStream.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons la classe FileInputStream pour créer un flux d'entrée à partir du fichier "example.txt". Nous utilisons ensuite la méthode read() pour lire les données du fichier, caractère par caractère, jusqu'à ce que la méthode renvoie -1, indiquant la fin du fichier.

La sérialisation en Java

La sérialisation en Java est le processus de conversion d'un objet en une séquence d'octets, qui peut être ensuite enregistrée dans un fichier ou transmise via un réseau. La sérialisation est utilisée pour la persistance des objets et la communication entre les applications.

Voici un exemple de code qui utilise la sérialisation pour enregistrer un objet dans un fichier :

import java.io.FileOutputStream;
import java.io.IOException;
import java.io.ObjectOutputStream;

public class SerializeExample {
    public static void main(String[] args) {
        try {
            FileOutputStream outputStream = new FileOutputStream("object.ser");
            ObjectOutputStream objectOutputStream = new ObjectOutputStream(outputStream);
            MyClass myObject = new MyClass();
            objectOutputStream.writeObject(myObject);
            objectOutputStream.close();
            outputStream.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

class MyClass implements Serializable {
    // code de la classe
}

Dans cet exemple, nous utilisons la classe ObjectOutputStream pour créer un flux de sortie et la méthode writeObject() pour sérialiser l'objet myObject et l'enregistrer dans le fichier "object.ser". La classe MyClass doit implémenter l'interface Serializable pour être sérialisée.

La gestion des fichiers en Java

En Java, la gestion des fichiers est réalisée à l'aide des classes File et Path. La classe File est utilisée pour représenter un fichier ou un répertoire, tandis que la classe Path est utilisée pour représenter un chemin d'accès à un fichier ou à un répertoire.

Voici un exemple de code qui utilise la classe File pour créer un nouveau fichier :

import java.io.File;
import java.io.IOException;

public class CreateFileExample {
    public static void main(String[] args) {
        try {
            File file = new File("example.txt");
            if (file.createNewFile()) {
                System.out.println("File created successfully.");
            } else {
                System.out.println("File already exists.");
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons la méthode createNewFile() de la classe File pour créer un nouveau fichier "example.txt". Si le fichier existe déjà, la méthode renvoie false.

Les entrées/sorties réseau en Java

En Java, les entrées/sorties réseau sont utilisées pour communiquer avec d'autres applications via un réseau. Les classes Socket et ServerSocket sont utilisées pour établir des connexions réseau et échanger des données.

Voici un exemple de code qui utilise la classe Socket pour se connecter à un serveur distant :

import java.io.IOException;
import java.io.InputStream;
import java.io.OutputStream;
import java.net.Socket;

public class ClientExample {
    public static void main(String[] args) {
        try {
            Socket socket = new Socket("localhost", 8080);
            OutputStream outputStream = socket.getOutputStream();
            outputStream.write("Hello, server!".getBytes());
            outputStream.flush();
            InputStream inputStream = socket.getInputStream();
            byte[] buffer = new byte[1024];
            int bytesRead = inputStream.read(buffer);
            String response = new String(buffer, 0, bytesRead);
            System.out.println("Server response: " + response);
            socket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Dans cet exemple, nous utilisons la classe Socket pour créer une connexion avec un serveur distant sur le port 8080. Nous utilisons ensuite les flux d'entrée/sortie pour envoyer des données au serveur et recevoir sa réponse.
