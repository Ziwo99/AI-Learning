Les flux d'entrée/sortie en Java

Les flux d'entrée/sortie en Java sont utilisés pour lire et écrire des données à partir de sources externes, telles que des fichiers ou des connexions réseau. Les flux d'entrée sont utilisés pour lire des données à partir de sources externes, tandis que les flux de sortie sont utilisés pour écrire des données vers des destinations externes.

En Java, les flux d'entrée/sortie sont représentés par les classes InputStream et OutputStream. Ces classes abstraites fournissent des méthodes pour lire et écrire des données de manière générique, indépendamment de la source ou de la destination réelle des données.

Voici un exemple de lecture d'un fichier texte à l'aide d'un flux d'entrée en Java : 


try (InputStream inputStream = new FileInputStream("fichier.txt")) {
    int data;
    while ((data = inputStream.read()) != -1) {
        System.out.print((char) data);
    }
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous utilisons la classe FileInputStream pour créer un flux d'entrée à partir du fichier "fichier.txt". Ensuite, nous utilisons la méthode read() pour lire les données du flux d'entrée, un octet à la fois. La méthode read() renvoie -1 lorsque la fin du flux est atteinte.

Notez que nous utilisons un bloc try-with-resources pour nous assurer que le flux d'entrée est correctement fermé, même en cas d'exception.

La sérialisation en Java

La sérialisation en Java est le processus de conversion d'un objet en une séquence d'octets, qui peut ensuite être enregistrée dans un fichier ou transmise via un réseau. La désérialisation est le processus inverse, qui consiste à reconstituer un objet à partir de sa représentation sérialisée.

En Java, la sérialisation est prise en charge par les classes ObjectOutputStream et ObjectInputStream. Ces classes fournissent des méthodes pour sérialiser et désérialiser des objets, respectivement.

Voici un exemple de sérialisation d'un objet en Java : 


try (OutputStream outputStream = new FileOutputStream("objet.ser");
     ObjectOutputStream objectOutputStream = new ObjectOutputStream(outputStream)) {
    MyClass myObject = new MyClass();
    objectOutputStream.writeObject(myObject);
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous utilisons la classe FileOutputStream pour créer un flux de sortie vers le fichier "objet.ser". Ensuite, nous utilisons la classe ObjectOutputStream pour sérialiser l'objet myObject et l'écrire dans le flux de sortie.

Notez que la classe de l'objet que nous sérialisons doit implémenter l'interface Serializable pour être sérialisable.

La gestion des fichiers en Java

La gestion des fichiers en Java permet de créer, lire, écrire et supprimer des fichiers sur le système de fichiers local. Java fournit plusieurs classes pour effectuer ces opérations, telles que File, FileReader, FileWriter, etc.

Voici un exemple de création d'un fichier texte en Java : 


try (FileWriter fileWriter = new FileWriter("fichier.txt")) {
    fileWriter.write("Contenu du fichier");
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous utilisons la classe FileWriter pour créer un flux de sortie vers le fichier "fichier.txt". Ensuite, nous utilisons la méthode write() pour écrire le contenu du fichier.

Voici un exemple de lecture d'un fichier texte en Java : 


try (FileReader fileReader = new FileReader("fichier.txt")) {
    int data;
    while ((data = fileReader.read()) != -1) {
        System.out.print((char) data);
    }
} catch (IOException e) {
    e.printStackTrace();
}

Dans cet exemple, nous utilisons la classe FileReader pour créer un flux d'entrée à partir du fichier "fichier.txt". Ensuite, nous utilisons la méthode read() pour lire les données du flux d'entrée, un caractère à la fois.

Notez que nous utilisons un bloc try-with-resources pour nous assurer que les flux de fichier sont correctement fermés, même en cas d'exception.
