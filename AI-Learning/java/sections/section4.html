Les listes en Java

Une liste en Java est une collection ordonnée d'éléments. Chaque élément est identifié par un index, qui commence à zéro pour le premier élément. Les listes en Java sont implémentées par l'interface List et ont plusieurs classes d'implémentation, telles que ArrayList et LinkedList.

Voici un exemple de création d'une liste en utilisant la classe ArrayList :

List<String> fruits = new ArrayList<>();
fruits.add("pomme");
fruits.add("banane");
fruits.add("orange");

Dans cet exemple, nous avons créé une liste appelée fruits et ajouté trois éléments : "pomme", "banane" et "orange".

Il est également possible d'accéder aux éléments d'une liste en utilisant leur index :

String premierFruit = fruits.get(0);
System.out.println(premierFruit); // Affiche "pomme"

La méthode get() permet d'obtenir l'élément à l'index spécifié.

Les ensembles en Java

Un ensemble en Java est une collection qui ne permet pas d'avoir des éléments en double. Les ensembles en Java sont implémentés par l'interface Set et ont plusieurs classes d'implémentation, telles que HashSet et TreeSet.

Voici un exemple de création d'un ensemble en utilisant la classe HashSet :

Set<String> ensemble = new HashSet<>();
ensemble.add("rouge");
ensemble.add("vert");
ensemble.add("bleu");
ensemble.add("rouge"); // Cet élément sera ignoré car il est en double

Dans cet exemple, nous avons créé un ensemble appelé ensemble et ajouté trois éléments : "rouge", "vert" et "bleu". Comme l'élément "rouge" est en double, il est ignoré.

Il est également possible de vérifier si un ensemble contient un certain élément :

boolean contientRouge = ensemble.contains("rouge");
System.out.println(contientRouge); // Affiche "true"

La méthode contains() renvoie true si l'ensemble contient l'élément spécifié.

Les cartes en Java

Une carte en Java est une collection d'éléments qui sont stockés sous forme de paires clé-valeur. Chaque élément est identifié par une clé unique. Les cartes en Java sont implémentées par l'interface Map et ont plusieurs classes d'implémentation, telles que HashMap et TreeMap.

Voici un exemple de création d'une carte en utilisant la classe HashMap :

Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 10);
scores.put("Bob", 15);
scores.put("Charlie", 20);

Dans cet exemple, nous avons créé une carte appelée scores et ajouté trois paires clé-valeur : "Alice" avec une valeur de 10, "Bob" avec une valeur de 15 et "Charlie" avec une valeur de 20.

Il est également possible d'accéder à la valeur d'une clé spécifique :

int scoreAlice = scores.get("Alice");
System.out.println(scoreAlice); // Affiche "10"

La méthode get() permet d'obtenir la valeur associée à la clé spécifiée.

Les itérations sur les collections en Java

En Java, il est possible d'itérer sur les éléments d'une collection en utilisant différentes méthodes. Voici deux exemples couramment utilisés :

1. Utilisation d'une boucle for-each :

for (String fruit : fruits) {
    System.out.println(fruit);
}

Cette boucle permet d'itérer sur tous les éléments de la liste fruits et d'afficher chaque élément.

2. Utilisation d'un itérateur :

Iterator<String> iterator = fruits.iterator();
while (iterator.hasNext()) {
    String fruit = iterator.next();
    System.out.println(fruit);
}

Cet exemple utilise un itérateur pour parcourir la liste fruits et afficher chaque élément. La méthode hasNext() vérifie s'il y a un élément suivant, et la méthode next() renvoie l'élément suivant.

Il est important de noter que ces exemples peuvent être appliqués à d'autres types de collections en Java, tels que les ensembles et les cartes.
