Section 4: Les collections en Java

Les listes en Java

Les listes en Java sont des structures de données qui permettent de stocker et de manipuler des éléments de manière ordonnée. Elles sont implémentées par l'interface List et offrent des fonctionnalités telles que l'ajout, la suppression et la recherche d'éléments.

Voici un exemple de création d'une liste en Java :

List<String> liste = new ArrayList<>();
liste.add("élément 1");
liste.add("élément 2");
liste.add("élément 3");

Les ensembles en Java

Les ensembles en Java sont des structures de données qui permettent de stocker des éléments uniques, c'est-à-dire qu'ils ne peuvent pas contenir de doublons. Ils sont implémentés par l'interface Set et offrent des fonctionnalités telles que l'ajout, la suppression et la recherche d'éléments.

Voici un exemple de création d'un ensemble en Java :

Set<Integer> ensemble = new HashSet<>();
ensemble.add(1);
ensemble.add(2);
ensemble.add(3);

Les cartes en Java

Les cartes en Java sont des structures de données qui permettent de stocker des paires clé-valeur. Chaque clé est unique et associée à une valeur. Elles sont implémentées par l'interface Map et offrent des fonctionnalités telles que l'ajout, la suppression et la recherche d'éléments par clé.

Voici un exemple de création d'une carte en Java :

Map<String, Integer> carte = new HashMap<>();
carte.put("clé 1", 1);
carte.put("clé 2", 2);
carte.put("clé 3", 3);

Les itérations sur les collections en Java

Les itérations sur les collections en Java permettent de parcourir les éléments d'une collection et d'effectuer des opérations sur chacun d'eux. Il existe plusieurs façons de réaliser des itérations, notamment à l'aide de boucles for ou de l'interface Iterator.

Voici un exemple d'itération sur une liste en Java :

for (String element : liste) {
    System.out.println(element);
}

Les algorithmes de tri et de recherche en Java

Les algorithmes de tri et de recherche en Java permettent de trier une collection selon un ordre spécifié ou de rechercher un élément dans une collection. Java offre des méthodes prédéfinies pour effectuer ces opérations, telles que sort() pour le tri et binarySearch() pour la recherche binaire.

Voici un exemple de tri d'une liste en Java :

Collections.sort(liste);

Et voici un exemple de recherche d'un élément dans une liste triée en Java :

int index = Collections.binarySearch(liste, "élément 2");

Ce sont là quelques-unes des fonctionnalités offertes par les collections en Java. Elles sont très utiles dans de nombreux cas de programmation et permettent de manipuler efficacement des ensembles de données.
