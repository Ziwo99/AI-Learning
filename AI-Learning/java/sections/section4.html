Section 4 : Les collections en Java

Les listes en Java

Les listes en Java sont des structures de données qui permettent de stocker et de manipuler une collection ordonnée d'éléments. Elles sont implémentées par l'interface List et offrent des fonctionnalités telles que l'ajout, la suppression et la recherche d'éléments.

Voici un exemple d'utilisation d'une liste en Java :

List<String> fruits = new ArrayList<>();
fruits.add("pomme");
fruits.add("banane");
fruits.add("orange");

for (String fruit : fruits) {
    System.out.println(fruit);
}

Les ensembles en Java

Les ensembles en Java sont des structures de données qui permettent de stocker une collection d'éléments uniques, sans ordre spécifique. Ils sont implémentés par l'interface Set et offrent des fonctionnalités telles que l'ajout, la suppression et la recherche d'éléments.

Voici un exemple d'utilisation d'un ensemble en Java :

Set<String> colors = new HashSet<>();
colors.add("rouge");
colors.add("vert");
colors.add("bleu");

System.out.println(colors.contains("rouge")); // true
System.out.println(colors.contains("jaune")); // false

Les cartes en Java

Les cartes en Java sont des structures de données qui permettent de stocker des paires clé-valeur. Chaque clé est unique et associée à une valeur spécifique. Les cartes sont implémentées par l'interface Map et offrent des fonctionnalités telles que l'ajout, la suppression et la recherche d'éléments par clé.

Voici un exemple d'utilisation d'une carte en Java :

Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 95);
scores.put("Bob", 80);
scores.put("Charlie", 75);

System.out.println(scores.get("Alice")); // 95
System.out.println(scores.containsKey("Bob")); // true

Les itérations et les opérations sur les collections en Java

En Java, il est possible d'itérer sur les collections à l'aide de boucles telles que la boucle for-each ou la boucle for. Cela permet de parcourir tous les éléments d'une collection et d'effectuer des opérations sur chacun d'eux.

Voici un exemple d'itération sur une liste en Java :

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

for (int number : numbers) {
    System.out.println(number);
}

Les comparateurs et les triages en Java

En Java, il est possible de trier les collections en utilisant des comparateurs. Un comparateur est une interface fonctionnelle qui définit une méthode de comparaison entre deux éléments. Les collections peuvent être triées en utilisant des méthodes telles que sort de la classe Collections ou sort de la classe Arrays.

Voici un exemple de tri d'une liste en Java :

List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

Collections.sort(names);

for (String name : names) {
    System.out.println(name);
}
