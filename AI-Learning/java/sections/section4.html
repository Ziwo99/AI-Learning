Les listes en Java

Une liste en Java est une collection ordonnée d'éléments. Elle permet de stocker des objets de manière séquentielle et d'accéder à ces objets par leur position dans la liste.

En Java, la classe List est une interface qui définit les méthodes de base pour manipuler une liste. Il existe plusieurs implémentations de cette interface, telles que ArrayList et LinkedList.

Voici un exemple de création et de manipulation d'une liste en Java :

List<String> fruits = new ArrayList<>();
fruits.add("pomme");
fruits.add("banane");
fruits.add("orange");

System.out.println(fruits.get(0)); // Affiche "pomme"
System.out.println(fruits.size()); // Affiche 3

Les ensembles en Java

Un ensemble en Java est une collection d'éléments uniques, c'est-à-dire qu'il ne peut pas contenir de doublons. Il permet de stocker des objets sans se soucier de leur ordre.

En Java, la classe Set est une interface qui définit les méthodes de base pour manipuler un ensemble. Les implémentations courantes de cette interface sont HashSet et TreeSet.

Voici un exemple de création et de manipulation d'un ensemble en Java :

Set<String> villes = new HashSet<>();
villes.add("Paris");
villes.add("Londres");
villes.add("New York");

System.out.println(villes.contains("Paris")); // Affiche true
System.out.println(villes.size()); // Affiche 3

Les cartes en Java

Une carte en Java est une collection d'associations clé-valeur. Chaque élément de la carte est composé d'une clé unique et d'une valeur associée.

En Java, la classe Map est une interface qui définit les méthodes de base pour manipuler une carte. Les implémentations courantes de cette interface sont HashMap et TreeMap.

Voici un exemple de création et de manipulation d'une carte en Java :

Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 10);
scores.put("Bob", 15);
scores.put("Charlie", 12);

System.out.println(scores.get("Bob")); // Affiche 15
System.out.println(scores.containsKey("Alice")); // Affiche true

Les itérations et les filtres en Java

En Java, il est possible de parcourir les éléments d'une collection à l'aide de boucles ou d'itérateurs. Cela permet d'effectuer des opérations sur chaque élément de la collection.

Voici un exemple d'itération sur une liste en Java :

List<Integer> nombres = new ArrayList<>();
nombres.add(1);
nombres.add(2);
nombres.add(3);

for (int nombre : nombres) {
    System.out.println(nombre);
}

Il est également possible de filtrer les éléments d'une collection en utilisant des prédicats ou des expressions lambda. Cela permet de sélectionner uniquement les éléments qui satisfont une condition spécifique.

Voici un exemple de filtrage d'une liste en Java :

List<Integer> nombres = new ArrayList<>();
nombres.add(1);
nombres.add(2);
nombres.add(3);

List<Integer> nombresPairs = nombres.stream()
                                     .filter(nombre -> nombre % 2 == 0)
                                     .collect(Collectors.toList());

System.out.println(nombresPairs); // Affiche [2]

Les comparateurs en Java

En Java, il est possible de trier les éléments d'une collection en utilisant des comparateurs. Un comparateur permet de définir l'ordre des éléments en fonction de critères spécifiques.

Voici un exemple de tri d'une liste en Java :

List<String> fruits = new ArrayList<>();
fruits.add("pomme");
fruits.add("banane");
fruits.add("orange");

Collections.sort(fruits);

System.out.println(fruits); // Affiche [banane, orange, pomme]

Il est également possible de définir des comparateurs personnalisés pour trier les éléments selon des critères spécifiques.

Voici un exemple de tri avec un comparateur personnalisé en Java :

List<String> fruits = new ArrayList<>();
fruits.add("pomme");
fruits.add("banane");
fruits.add("orange");

Collections.sort(fruits, (fruit1, fruit2) -> fruit1.length() - fruit2.length());

System.out.println(fruits); // Affiche [pomme, orange, banane]
