Les listes en Java

En Java, une liste est une collection ordonnée d'éléments. Chaque élément est identifié par un index, qui commence à 0 pour le premier élément. Les listes en Java sont implémentées par l'interface List et ont plusieurs classes d'implémentation, telles que ArrayList et LinkedList.

Voici un exemple de création d'une liste en Java :

List<String> fruits = new ArrayList<>();
fruits.add("pomme");
fruits.add("banane");
fruits.add("orange");

Vous pouvez accéder aux éléments d'une liste en utilisant leur index :

String premierFruit = fruits.get(0);
System.out.println(premierFruit); // Affiche "pomme"

Les listes en Java offrent également des méthodes pour ajouter, supprimer et rechercher des éléments :

fruits.add("kiwi"); // Ajoute "kiwi" à la fin de la liste
fruits.remove(1); // Supprime l'élément à l'index 1 (ici "banane")
int indexOrange = fruits.indexOf("orange"); // Recherche l'index de l'élément "orange"

Les ensembles en Java

En Java, un ensemble est une collection d'éléments uniques, sans ordre spécifique. Les ensembles en Java sont implémentés par l'interface Set et ont plusieurs classes d'implémentation, telles que HashSet et TreeSet.

Voici un exemple de création d'un ensemble en Java :

Set<String> pays = new HashSet<>();
pays.add("France");
pays.add("Espagne");
pays.add("Italie");

Les ensembles en Java offrent des méthodes pour ajouter, supprimer et rechercher des éléments, tout comme les listes :

pays.add("Allemagne"); // Ajoute "Allemagne" à l'ensemble
pays.remove("Espagne"); // Supprime l'élément "Espagne"
boolean contientItalie = pays.contains("Italie"); // Vérifie si l'ensemble contient "Italie"

Les cartes en Java

En Java, une carte est une collection d'associations clé-valeur. Chaque élément de la carte est identifié par une clé unique. Les cartes en Java sont implémentées par l'interface Map et ont plusieurs classes d'implémentation, telles que HashMap et TreeMap.

Voici un exemple de création d'une carte en Java :

Map<String, Integer> population = new HashMap<>();
population.put("Paris", 2200000);
population.put("Madrid", 3200000);
population.put("Rome", 2800000);

Vous pouvez accéder aux valeurs d'une carte en utilisant leur clé :

int populationParis = population.get("Paris");
System.out.println(populationParis); // Affiche 2200000

Les cartes en Java offrent des méthodes pour ajouter, supprimer et rechercher des éléments, tout comme les listes et les ensembles :

population.put("Berlin", 3500000); // Ajoute l'association clé-valeur "Berlin" - 3500000 à la carte
population.remove("Madrid"); // Supprime l'association avec la clé "Madrid"
boolean contientRome = population.containsKey("Rome"); // Vérifie si la carte contient la clé "Rome"

Les itérations et les filtres en Java

En Java, les collections offrent des fonctionnalités d'itération et de filtrage pour parcourir et manipuler les éléments de manière efficace.

Voici un exemple d'itération sur une liste en Java :

for (String fruit : fruits) {
    System.out.println(fruit);
}

Vous pouvez également utiliser des filtres pour sélectionner uniquement les éléments qui répondent à certaines conditions :

List<String> fruitsAvecA = fruits.stream()
                                .filter(fruit -> fruit.contains("a"))
                                .collect(Collectors.toList());

Cet exemple crée une nouvelle liste contenant uniquement les fruits qui contiennent la lettre "a".

Les itérations et les filtres en Java sont des fonctionnalités puissantes pour manipuler les collections de manière flexible et efficace.
