Tableaux unidimensionnels et multidimensionnels

Un tableau est une structure de données qui permet de stocker plusieurs valeurs de même type dans une seule variable. En Java, les tableaux peuvent être unidimensionnels ou multidimensionnels.

Un tableau unidimensionnel est une liste ordonnée d'éléments du même type. Pour déclarer et initialiser un tableau unidimensionnel, vous pouvez utiliser la syntaxe suivante :

type[] nomDuTableau = new type[taille];

Par exemple, pour créer un tableau unidimensionnel de nombres entiers de taille 5 :

int[] tableauEntiers = new int[5];

Un tableau multidimensionnel est un tableau qui contient d'autres tableaux. Il peut être considéré comme une matrice ou un tableau de tableaux. Pour déclarer et initialiser un tableau multidimensionnel, vous pouvez utiliser la syntaxe suivante :

type[][] nomDuTableau = new type[taille1][taille2];

Par exemple, pour créer un tableau multidimensionnel de nombres entiers de taille 3x3 :

int[][] tableauEntiers = new int[3][3];

Listes, ensembles et maps

En plus des tableaux, Java propose des structures de données plus avancées telles que les listes, les ensembles et les maps.

Une liste est une collection ordonnée d'éléments. En Java, vous pouvez utiliser l'interface List pour créer une liste. Voici comment déclarer et initialiser une liste :

List<type> nomDeLaListe = new ArrayList<>();

Par exemple, pour créer une liste de nombres entiers :

List<Integer> listeEntiers = new ArrayList<>();

Un ensemble est une collection d'éléments uniques, sans ordre spécifique. En Java, vous pouvez utiliser l'interface Set pour créer un ensemble. Voici comment déclarer et initialiser un ensemble :

Set<type> nomDeLensemble = new HashSet<>();

Par exemple, pour créer un ensemble de chaînes de caractères :

Set<String> ensembleChaines = new HashSet<>();

Une map est une collection d'associations clé-valeur. En Java, vous pouvez utiliser l'interface Map pour créer une map. Voici comment déclarer et initialiser une map :

Map<typeCle, typeValeur> nomDeLaMap = new HashMap<>();

Par exemple, pour créer une map de chaînes de caractères associées à des nombres entiers :

Map<String, Integer> mapChainesEntiers = new HashMap<>();

Manipulation de tableaux et collections

Une fois que vous avez déclaré et initialisé un tableau ou une collection, vous pouvez effectuer différentes opérations de manipulation.

Pour accéder à un élément spécifique d'un tableau unidimensionnel, vous pouvez utiliser l'indice de l'élément. Par exemple, pour accéder au premier élément du tableauEntiers :

int premierElement = tableauEntiers[0];

Pour accéder à un élément spécifique d'un tableau multidimensionnel, vous devez spécifier l'indice de chaque dimension. Par exemple, pour accéder à l'élément à la deuxième ligne et à la troisième colonne du tableauEntiers :

int element = tableauEntiers[1][2];

Pour ajouter un élément à une liste, vous pouvez utiliser la méthode add(). Par exemple, pour ajouter un nombre entier à la listeEntiers :

listeEntiers.add(10);

Pour supprimer un élément d'une liste, vous pouvez utiliser la méthode remove(). Par exemple, pour supprimer le premier élément de la listeEntiers :

listeEntiers.remove(0);

Pour ajouter un élément à un ensemble, vous pouvez utiliser la méthode add(). Par exemple, pour ajouter une chaîne de caractères à l'ensembleChaines :

ensembleChaines.add("exemple");

Pour supprimer un élément d'un ensemble, vous pouvez utiliser la méthode remove(). Par exemple, pour supprimer une chaîne de caractères de l'ensembleChaines :

ensembleChaines.remove("exemple");

Pour ajouter une association clé-valeur à une map, vous pouvez utiliser la méthode put(). Par exemple, pour ajouter une chaîne de caractères associée à un nombre entier à la mapChainesEntiers :

mapChainesEntiers.put("exemple", 10);

Pour supprimer une association clé-valeur d'une map, vous pouvez utiliser la méthode remove(). Par exemple, pour supprimer la chaîne de caractères associée à la clé "exemple" de la mapChainesEntiers :

mapChainesEntiers.remove("exemple");

Ces exemples ne couvrent qu'une petite partie des opérations de manipulation possibles avec les tableaux et les collections en Java. Il existe de nombreuses autres méthodes et fonctionnalités disponibles pour répondre à vos besoins spécifiques.
