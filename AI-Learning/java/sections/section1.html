Section 1 : Introduction à Java

Histoire et évolution de Java

Java a été développé par James Gosling et son équipe chez Sun Microsystems dans les années 1990. Il a été conçu pour être un langage de programmation polyvalent, portable et orienté objet. L'objectif principal de Java était de permettre aux développeurs de créer des applications qui pourraient fonctionner sur n'importe quel appareil, indépendamment de son système d'exploitation.

Depuis sa création, Java a connu de nombreuses évolutions et mises à jour. La version la plus récente est Java 15, qui a été publiée en septembre 2020. Chaque nouvelle version apporte des fonctionnalités et des améliorations pour rendre le développement en Java plus efficace et plus facile.

Installation et configuration de l'environnement de développement Java

Avant de commencer à programmer en Java, vous devez installer et configurer l'environnement de développement Java sur votre machine. Voici les étapes à suivre :

  1. Téléchargez et installez la dernière version du kit de développement Java (JDK) à partir du site officiel d'Oracle.
  2. Configurez les variables d'environnement JAVA_HOME et PATH pour que votre système puisse trouver les outils Java.
  3. Vérifiez l'installation en ouvrant une fenêtre de terminal et en exécutant la commande "java -version". Vous devriez voir la version de Java installée.

Structure d'un programme Java

Un programme Java est composé de classes, de méthodes et d'instructions. Voici une structure de base d'un programme Java : 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Dans cet exemple, nous avons une classe appelée "HelloWorld" avec une méthode principale "main". La méthode "main" est le point d'entrée de notre programme Java. À l'intérieur de cette méthode, nous utilisons l'instruction "System.out.println" pour afficher le message "Hello, World!" à la console.

Variables et types de données en Java

En Java, les variables sont utilisées pour stocker des données. Chaque variable a un type de données qui détermine le type de valeurs qu'elle peut contenir. Voici quelques types de données couramment utilisés en Java :

  • int : pour stocker des nombres entiers.
  • double : pour stocker des nombres décimaux.
  • boolean : pour stocker des valeurs booléennes (true ou false).
  • String : pour stocker des chaînes de caractères.

Voici un exemple de déclaration et d'utilisation de variables en Java : 

int age = 25;
double height = 1.75;
boolean isStudent = true;
String name = "John Doe";

System.out.println("Name: " + name);
System.out.println("Age: " + age);
System.out.println("Height: " + height);
System.out.println("Is student: " + isStudent);

Dans cet exemple, nous déclarons et initialisons des variables pour stocker l'âge, la taille, le statut d'étudiant et le nom. Ensuite, nous utilisons l'instruction "System.out.println" pour afficher les valeurs de ces variables à la console.
