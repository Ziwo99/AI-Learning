Historique de Java

Java est un langage de programmation créé par James Gosling et son équipe chez Sun Microsystems en 1995. Il a été conçu pour être portable, c'est-à-dire qu'il peut être exécuté sur différents systèmes d'exploitation sans nécessiter de modifications importantes. Java a été initialement développé pour les appareils électroniques grand public, mais il est rapidement devenu populaire dans le domaine du développement web et des applications d'entreprise.

Installation et configuration de l'environnement de développement Java

Pour commencer à programmer en Java, vous devez installer et configurer l'environnement de développement Java (JDK - Java Development Kit) sur votre machine. Le JDK comprend le compilateur Java, les bibliothèques standard et les outils nécessaires pour développer des applications en Java. Vous pouvez télécharger le JDK à partir du site officiel d'Oracle et suivre les instructions d'installation spécifiques à votre système d'exploitation.

Structure d'un programme Java

Un programme Java est composé de classes, qui sont les blocs de construction fondamentaux de Java. Chaque classe est définie dans un fichier source Java avec l'extension ".java". Un programme Java commence par une classe principale, qui est la classe contenant la méthode "main". La méthode "main" est le point d'entrée du programme, c'est là que l'exécution commence.

public class Main {
    public static void main(String[] args) {
        // Code à exécuter
    }
}

Variables et types de données en Java

En Java, vous devez déclarer une variable avant de l'utiliser. Une variable est un espace de stockage nommé qui peut contenir une valeur. Java prend en charge différents types de données, tels que les entiers, les nombres à virgule flottante, les caractères, les chaînes de caractères, les booléens, etc. Chaque type de données a une taille et une plage de valeurs spécifiques.

int age = 25;
double price = 19.99;
char grade = 'A';
String name = "John Doe";
boolean isStudent = true;

Opérateurs et expressions en Java

Les opérateurs en Java sont utilisés pour effectuer des opérations sur les variables et les valeurs. Les opérateurs arithmétiques sont utilisés pour effectuer des calculs mathématiques, les opérateurs de comparaison sont utilisés pour comparer des valeurs, les opérateurs logiques sont utilisés pour combiner des expressions booléennes, etc. Les expressions en Java sont des combinaisons d'opérateurs, de variables et de valeurs qui produisent un résultat.

int sum = 10 + 5;
boolean isGreater = 10 > 5;
boolean isTrue = true && false;
