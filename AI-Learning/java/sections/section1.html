Section 1 : Introduction à Java

Histoire et principes de base de Java

Java est un langage de programmation orienté objet créé par James Gosling et son équipe chez Sun Microsystems en 1995. Il a été conçu pour être portable, sécurisé et robuste, ce qui signifie qu'il peut fonctionner sur différentes plateformes sans nécessiter de modifications majeures.

Les principes de base de Java comprennent :

  • L'encapsulation : permet de regrouper des données et des méthodes dans une classe.
  • L'héritage : permet à une classe d'hériter des propriétés et des méthodes d'une autre classe.
  • Le polymorphisme : permet à une classe d'avoir plusieurs formes, en utilisant l'héritage et les interfaces.
  • Les classes et les objets : les objets sont des instances de classes, qui sont des modèles pour créer des objets.

Installation et configuration de l'environnement de développement Java

Pour commencer à programmer en Java, vous devez installer et configurer l'environnement de développement Java (JDK - Java Development Kit) sur votre machine. Voici les étapes à suivre :

  1. Téléchargez le JDK à partir du site officiel d'Oracle.
  2. Installez le JDK en suivant les instructions spécifiques à votre système d'exploitation.
  3. Configurez les variables d'environnement JAVA_HOME et PATH pour pouvoir exécuter les commandes Java depuis n'importe quel répertoire.
  4. Vérifiez l'installation en ouvrant une fenêtre de terminal et en exécutant la commande 
    java -version

Structure d'un programme Java

Un programme Java est composé de classes, de méthodes et d'instructions. Voici une structure de base d'un programme Java :

public class HelloWorld {
    public static void main(String[] args) {
        // Instructions du programme
        System.out.println("Hello, World!");
    }
}

Explication :

  • public class HelloWorld : définit une classe publique nommée HelloWorld.
  • public static void main(String[] args) : définit la méthode principale du programme.
  • // Instructions du programme : représente les instructions du programme.
  • System.out.println("Hello, World!"); : affiche le message "Hello, World!" à l'écran.

Variables et types de données en Java

En Java, les variables sont utilisées pour stocker des données. Voici les types de données les plus couramment utilisés en Java :

  • int : pour stocker des nombres entiers.
  • double : pour stocker des nombres décimaux.
  • boolean : pour stocker des valeurs booléennes (vrai ou faux).
  • String : pour stocker des chaînes de caractères.

Exemple :

int age = 25;
double height = 1.75;
boolean isStudent = true;
String name = "John Doe";

Opérateurs et expressions en Java

Les opérateurs en Java sont utilisés pour effectuer des opérations sur les variables et les valeurs. Voici quelques exemples d'opérateurs couramment utilisés :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
  • % : modulo (reste de la division)

Exemple :

int a = 10;
int b = 5;
int sum = a + b;
int difference = a - b;
int product = a * b;
int quotient = a / b;
int remainder = a % b;

Les expressions en Java combinent des variables, des valeurs et des opérateurs pour effectuer des calculs. Par exemple :

int result = (a + b) * (a - b);

Cette expression calcule le carré de la différence entre a et b.
