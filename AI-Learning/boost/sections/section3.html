Boost.Array

Boost.Array est une bibliothèque qui fournit un conteneur de tableau fixe en C++. Elle permet de manipuler des tableaux de taille fixe avec des fonctionnalités supplémentaires par rapport aux tableaux natifs de C++. Boost.Array garantit la sécurité des accès aux éléments du tableau et facilite la gestion de la mémoire.

#include <boost/array.hpp>
#include <iostream>

int main() {
    boost::array<int, 5> arr = {1, 2, 3, 4, 5};

    for (const auto& element : arr) {
        std::cout << element << " ";
    }

    return 0;
}

Dans cet exemple, nous utilisons Boost.Array pour créer un tableau de taille fixe contenant les entiers de 1 à 5. Nous parcourons ensuite le tableau et affichons chaque élément à l'aide d'une boucle for-each.

Boost.Bind

Boost.Bind est une bibliothèque qui permet de créer des fonctions ou des objets fonctionnels à partir de fonctions existantes en spécifiant certains arguments. Cela facilite la création de fonctions partielles ou de fonctions adaptées à des interfaces spécifiques.

#include <boost/bind.hpp>
#include <iostream>

void printSum(int a, int b) {
    std::cout << "Sum: " << a + b << std::endl;
}

int main() {
    auto sumFunction = boost::bind(printSum, 10, _1);
    sumFunction(5);

    return 0;
}

Dans cet exemple, nous utilisons Boost.Bind pour créer une fonction partiellement appliquée à partir de la fonction printSum. La fonction sumFunction prend un seul argument et l'utilise comme deuxième argument pour la fonction printSum, avec le premier argument fixé à 10. Ainsi, lorsque nous appelons sumFunction avec l'argument 5, la fonction printSum est appelée avec les arguments 10 et 5, affichant ainsi la somme 15.

Boost.Date_Time

Boost.Date_Time est une bibliothèque qui fournit des fonctionnalités avancées pour la manipulation des dates et du temps en C++. Elle facilite la gestion des calendriers, des fuseaux horaires, des durées et des opérations sur les dates et les heures.

#include <boost/date_time.hpp>
#include <iostream>

int main() {
    boost::posix_time::ptime now = boost::posix_time::second_clock::local_time();
    std::cout << "Current time: " << now << std::endl;

    return 0;
}

Dans cet exemple, nous utilisons Boost.Date_Time pour obtenir l'heure locale actuelle à l'aide de la classe ptime et de la fonction second_clock::local_time(). Nous affichons ensuite cette heure à l'aide de std::cout.

Boost.Filesystem

Boost.Filesystem est une bibliothèque qui fournit des fonctionnalités pour la manipulation des fichiers et des répertoires en C++. Elle facilite la création, la suppression, le déplacement et la recherche de fichiers et de répertoires, ainsi que la manipulation des chemins d'accès.

#include <boost/filesystem.hpp>
#include <iostream>

int main() {
    boost::filesystem::path filePath("path/to/file.txt");

    if (boost::filesystem::exists(filePath)) {
        std::cout << "File exists!" << std::endl;
    } else {
        std::cout << "File does not exist." << std::endl;
    }

    return 0;
}

Dans cet exemple, nous utilisons Boost.Filesystem pour vérifier si un fichier spécifié par le chemin d'accès "path/to/file.txt" existe. Nous utilisons la fonction exists() pour effectuer cette vérification et affichons un message approprié en fonction du résultat.

Boost.Regex

Boost.Regex est une bibliothèque qui fournit des fonctionnalités pour la manipulation des expressions régulières en C++. Elle permet de rechercher, de valider et de manipuler des chaînes de caractères en utilisant des motifs spécifiques.

#include <boost/regex.hpp>
#include <iostream>
#include <string>

int main() {
    std::string input = "Hello, Boost!";
    boost::regex pattern("Boost");

    if (boost::regex_search(input, pattern)) {
        std::cout << "Pattern found!" << std::endl;
    } else {
        std::cout << "Pattern not found." << std::endl;
    }

    return 0;
}

Dans cet exemple, nous utilisons Boost.Regex pour rechercher le motif "Boost" dans une chaîne de caractères. Nous utilisons la fonction regex_search() pour effectuer cette recherche et affichons un message approprié en fonction du résultat.

Boost.Serialization

Boost.Serialization est une bibliothèque qui fournit des fonctionnalités pour la sérialisation d'objets en C++. Elle permet de convertir des objets en une représentation binaire ou textuelle, ce qui facilite leur stockage ou leur transmission.

#include <boost/serialization/serialization.hpp>
#include <boost/archive/text_oarchive.hpp>
#include <boost/archive/text_iarchive.hpp>
#include <iostream>
#include <fstream>

class MyClass {
public:
    int data;

    template <class Archive>
    void serialize(Archive& ar, const unsigned int version) {
        ar & data;
    }
};

int main() {
    MyClass obj;
    obj.data = 42;

    std::ofstream file("data.txt");
    boost::archive::text_oarchive archive(file);
    archive << obj;
    file.close();

    std::ifstream file2("data.txt");
    boost::archive::text_iarchive archive2(file2);
    MyClass obj2;
    archive2 >> obj2;
    file2.close();

    std::cout << "Deserialized data: " << obj2.data << std::endl;

    return 0;
}

Dans cet exemple, nous utilisons Boost.Serialization pour sérialiser un objet de la classe MyClass en utilisant un archiveur textuel. Nous stockons ensuite la représentation sérialisée dans un fichier "data.txt". Nous pouvons ensuite désérialiser l'objet à partir du fichier et afficher la valeur de son membre "data".

Boost.Smart_Ptr

Boost.Smart_Ptr est une bibliothèque qui fournit des pointeurs intelligents en C++. Elle facilite la gestion de la mémoire en automatisant la libération des ressources allouées dynamiquement lorsque les pointeurs intelligents sortent de portée.

#include <boost/smart_ptr.hpp>
#include <iostream>

int main() {
    boost::shared_ptr<int> ptr(new int(42));
    std::cout << "Value: " << *ptr << std::endl;

    return 0;
}

Dans cet exemple, nous utilisons Boost.Smart_Ptr pour créer un pointeur intelligent partagé vers un entier de valeur 42. Nous utilisons ensuite l'opérateur de déréférencement (*) pour accéder à la valeur pointée et l'afficher à l'aide de std::cout.

Boost.Thread

Boost.Thread est une bibliothèque qui fournit des fonctionnalités pour la programmation multi-threading en C++. Elle facilite la création, la synchronisation et la communication entre les threads, ainsi que la gestion des verrous et des variables conditionnelles.

#include <boost/thread.hpp>
#include <iostream>

void printMessage() {
    std::cout << "Hello from thread!" << std::endl;
}

int main() {
    boost::thread thread(printMessage);
    thread.join();

    return 0;
}

Dans cet exemple, nous utilisons Boost.Thread pour créer un thread qui exécute la fonction printMessage. Nous utilisons la fonction join() pour attendre que le thread se termine avant de terminer le programme principal.

Boost.Units

Boost.Units est une bibliothèque qui fournit des fonctionnalités pour la manipulation des unités de mesure en C++. Elle facilite la conversion entre différentes unités, la réalisation d'opérations mathématiques avec des quantités de différentes unités et la vérification de la cohérence des unités.

#include <boost/units.hpp>
#include <iostream>

int main() {
    typedef boost::units::quantity<boost::units::si::length> Length;
    Length distance = 10.0 * boost::units::si::meters;

    std::cout << "Distance: " << distance << std::endl;

    return 0;
}

Dans cet exemple, nous utilisons Boost.Units pour définir une quantité de longueur en mètres à l'aide du type Length. Nous utilisons ensuite cette quantité pour afficher la distance à l'aide de std::cout.

Boost.Utility

Boost.Utility est une bibliothèque qui fournit diverses fonctionnalités utilitaires en C++. Elle comprend des fonctions pour la manipulation des chaînes de caractères, la génération de nombres aléatoires, la manipulation des types, la gestion des exceptions et bien plus encore.

#include <boost/utility.hpp>
#include <iostream>

int main() {
    std::string str = "Boost.Utility";
    std::cout << "Length: " << boost::size(str) << std::endl;

    return 0;
}

Dans cet exemple, nous utilisons Boost.Utility pour obtenir la taille de la chaîne de caractères "Boost.Utility" à l'aide de la fonction size(). Nous affichons ensuite cette taille à l'aide de std::cout.
