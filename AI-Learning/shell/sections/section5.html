Les expressions régulières

Les expressions régulières sont des motifs de recherche utilisés pour trouver des correspondances dans du texte. Elles sont très puissantes et flexibles, permettant de rechercher des motifs spécifiques, de valider des formats de données et d'effectuer des remplacements.

Voici un exemple d'utilisation d'une expression régulière pour trouver tous les mots commençant par "cat" dans un texte :

import re

text = "I have a cat, a dog, and a parrot."
pattern = r"cat\w+"

matches = re.findall(pattern, text)
for match in matches:
    print(match)

Ce code affichera tous les mots commençant par "cat" dans le texte, dans cet exemple "cat" sera affiché.

Les commandes de gestion des fichiers et des répertoires

Le Shell offre de nombreuses commandes pour gérer les fichiers et les répertoires. Voici quelques-unes des commandes les plus couramment utilisées :

  • ls : liste les fichiers et les répertoires dans le répertoire courant
  • cd : change le répertoire courant
  • mkdir : crée un nouveau répertoire
  • rm : supprime un fichier ou un répertoire
  • cp : copie un fichier ou un répertoire
  • mv : déplace ou renomme un fichier ou un répertoire

Voici un exemple d'utilisation de la commande ls :

ls -l

Cette commande affiche une liste détaillée des fichiers et des répertoires dans le répertoire courant, y compris les permissions, les propriétaires et les dates de modification.

Les commandes de gestion des réseaux

Le Shell offre également des commandes pour gérer les réseaux. Ces commandes permettent de vérifier la connectivité, de résoudre des noms de domaine, de transférer des fichiers et bien plus encore.

Voici quelques-unes des commandes de gestion des réseaux les plus couramment utilisées :

  • ping : vérifie la connectivité avec une adresse IP ou un nom de domaine
  • nslookup : résout un nom de domaine en une adresse IP
  • scp : transfère des fichiers entre des machines distantes de manière sécurisée
  • ssh : se connecte à une machine distante via SSH

Voici un exemple d'utilisation de la commande ping :

ping google.com

Cette commande vérifie la connectivité avec le site web de Google en envoyant des paquets ICMP et en affichant les résultats.

Les commandes de gestion des archives

Les commandes de gestion des archives permettent de créer, extraire et manipuler des fichiers compressés. Elles sont très utiles pour économiser de l'espace de stockage et pour transférer des fichiers plus rapidement.

Voici quelques-unes des commandes de gestion des archives les plus couramment utilisées :

  • tar : crée une archive ou extrait des fichiers d'une archive
  • gzip : compresse un fichier
  • gunzip : décompresse un fichier compressé avec gzip
  • zip : crée une archive ZIP
  • unzip : extrait des fichiers d'une archive ZIP

Voici un exemple d'utilisation de la commande tar :

tar -cvf archive.tar file1 file2

Cette commande crée une archive tar contenant les fichiers "file1" et "file2".
