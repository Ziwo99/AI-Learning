Les commandes de recherche et de filtrage

Les commandes de recherche et de filtrage permettent de rechercher des informations spécifiques dans les fichiers ou les résultats de commandes, et de filtrer ces informations selon certains critères.

La commande grep est l'une des commandes les plus utilisées pour la recherche de motifs dans les fichiers. Elle permet de rechercher une expression régulière dans un ou plusieurs fichiers. Par exemple, pour rechercher le mot "hello" dans un fichier nommé "file.txt", on peut utiliser la commande suivante :

grep "hello" file.txt

La commande sed est une commande de traitement de texte qui permet de modifier le contenu d'un fichier en utilisant des expressions régulières. Par exemple, pour remplacer toutes les occurrences du mot "apple" par le mot "orange" dans un fichier nommé "file.txt", on peut utiliser la commande suivante :

sed 's/apple/orange/g' file.txt

La commande awk est une commande puissante pour le traitement de fichiers texte. Elle permet de rechercher et de manipuler des données en utilisant des motifs et des actions spécifiques. Par exemple, pour afficher la deuxième colonne d'un fichier CSV nommé "data.csv", on peut utiliser la commande suivante :

awk -F ',' '{print $2}' data.csv

Les commandes de manipulation de texte

Les commandes de manipulation de texte permettent de manipuler le contenu des fichiers texte de différentes manières, comme la suppression de lignes vides, la suppression de caractères spécifiques, la concaténation de fichiers, etc.

La commande cut permet de découper les lignes d'un fichier en utilisant un délimiteur spécifique. Par exemple, pour extraire la deuxième colonne d'un fichier CSV nommé "data.csv", en utilisant la virgule comme délimiteur, on peut utiliser la commande suivante :

cut -d ',' -f 2 data.csv

La commande sort permet de trier les lignes d'un fichier selon un ordre spécifique. Par exemple, pour trier les lignes d'un fichier nommé "file.txt" par ordre alphabétique, on peut utiliser la commande suivante :

sort file.txt

La commande uniq permet de supprimer les lignes en double d'un fichier. Par exemple, pour supprimer les lignes en double d'un fichier nommé "file.txt", on peut utiliser la commande suivante :

uniq file.txt

Les commandes de gestion des processus

Les commandes de gestion des processus permettent de gérer les processus en cours d'exécution sur le système, comme les démarrer, les arrêter, les mettre en pause, etc.

La commande ps permet d'afficher les processus en cours d'exécution sur le système. Par exemple, pour afficher tous les processus en cours d'exécution, on peut utiliser la commande suivante :

ps aux

La commande kill permet d'arrêter un processus en utilisant son identifiant de processus (PID). Par exemple, pour arrêter un processus avec le PID 1234, on peut utiliser la commande suivante :

kill 1234

La commande top permet d'afficher les processus en cours d'exécution en temps réel, triés par utilisation des ressources. Elle est souvent utilisée pour surveiller les performances du système. Pour lancer la commande top, il suffit de l'exécuter dans le terminal :

top

Les commandes de gestion des utilisateurs et des permissions

Les commandes de gestion des utilisateurs et des permissions permettent de gérer les utilisateurs et les groupes sur le système, ainsi que les permissions d'accès aux fichiers et aux répertoires.

La commande useradd permet de créer un nouvel utilisateur sur le système. Par exemple, pour créer un utilisateur nommé "john", on peut utiliser la commande suivante :

useradd john

La commande passwd permet de changer le mot de passe d'un utilisateur. Par exemple, pour changer le mot de passe de l'utilisateur "john", on peut utiliser la commande suivante :

passwd john

La commande chmod permet de modifier les permissions d'accès aux fichiers et aux répertoires. Par exemple, pour donner les permissions de lecture, d'écriture et d'exécution à l'utilisateur, et les permissions de lecture aux autres utilisateurs, sur un fichier nommé "file.txt", on peut utiliser la commande suivante :

chmod u=rwx,g=r,o=r file.txt

La commande chown permet de changer le propriétaire d'un fichier ou d'un répertoire. Par exemple, pour changer le propriétaire d'un fichier nommé "file.txt" en "john", on peut utiliser la commande suivante :

chown john file.txt

La commande chgrp permet de changer le groupe d'un fichier ou d'un répertoire. Par exemple, pour changer le groupe d'un fichier nommé "file.txt" en "users", on peut utiliser la commande suivante :

chgrp users file.txt
