Les commandes de base

Les commandes de base sont les commandes essentielles pour interagir avec le Shell. Elles permettent d'exécuter des actions simples et courantes.

Voici quelques exemples de commandes de base :

ls : Affiche la liste des fichiers et des répertoires dans le répertoire courant.
cd : Change le répertoire courant.
mkdir : Crée un nouveau répertoire.
rm : Supprime un fichier ou un répertoire.
cp : Copie un fichier ou un répertoire.
mv : Déplace ou renomme un fichier ou un répertoire.

La navigation dans le système de fichiers

La navigation dans le système de fichiers permet de se déplacer entre les répertoires et d'accéder aux fichiers.

Voici quelques commandes pour naviguer dans le système de fichiers :

pwd : Affiche le chemin absolu du répertoire courant.
cd : Change le répertoire courant.
ls : Affiche la liste des fichiers et des répertoires dans le répertoire courant.

La manipulation des fichiers et des répertoires

La manipulation des fichiers et des répertoires permet d'effectuer des actions telles que la création, la suppression, la copie et le déplacement de fichiers et de répertoires.

Voici quelques commandes pour manipuler les fichiers et les répertoires :

mkdir : Crée un nouveau répertoire.
rm : Supprime un fichier ou un répertoire.
cp : Copie un fichier ou un répertoire.
mv : Déplace ou renomme un fichier ou un répertoire.

Les variables d'environnement

Les variables d'environnement sont des variables spéciales qui stockent des informations sur le système et sur la configuration du Shell.

Voici quelques commandes pour manipuler les variables d'environnement :

env : Affiche les variables d'environnement.
export : Définit une nouvelle variable d'environnement.
echo : Affiche la valeur d'une variable d'environnement.
